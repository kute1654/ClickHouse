<Info>
  **호환성**

  자체 호스팅 ClickHouse 배포와 ClickHouse Cloud 서비스 사이에 동작 차이가 있다면,
  이는 [호환성 설정](/ko/reference/settings/session-settings#compatibility)과 관련이 있을 수 있습니다.
  Cloud에서는 서비스 생성 시 호환성이 설정되며, 서비스가 업그레이드되더라도 클라이언트가 일관된 동작을 유지할 수 있도록 서비스 수준에서는 이 값이 변경되지 않습니다.
  호환성을 변경하려면 [support](https://clickhouse.com/support/program)를 통해 요청하십시오.
</Info>
