---
title: 원격 IP 액세스 목록 세부 정보 추가
---

<Accordion title="IP 액세스 목록 관리">
  ClickHouse Cloud 서비스 목록에서 작업할 서비스를 선택한 다음 **설정**으로 이동합니다. IP 액세스 목록에 ClickHouse Cloud 서비스에 연결해야 하는 원격 시스템의 IP 주소 또는 IP 주소 범위가 포함되어 있지 않으면 **Add IPs**로 이 문제를 해결할 수 있습니다.

  <Image size="md" img="/images/_snippets/ip-allow-list-check-list.webp" alt="IP 액세스 목록에서 서비스가 해당 IP 주소의 트래픽을 허용하는지 확인" border />

  ClickHouse Cloud 서비스에 연결해야 하는 개별 IP 주소 또는 IP 주소 범위를 추가합니다. 필요에 맞게 양식을 수정한 다음 **Save**를 클릭하십시오.

  <Image size="md" img="/images/_snippets/ip-allow-list-add-current-ip.webp" alt="ClickHouse Cloud의 IP 액세스 목록에 현재 IP 주소 추가" border />
</Accordion>
