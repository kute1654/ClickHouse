---
title: S3 설정
---

| Setting                                   | 설명                                                     | 기본값 |
| ----------------------------------------- | ------------------------------------------------------ | --- |
| `use_same_s3_credentials_for_base_backup` | S3로 기준 백업을 수행할 때 쿼리의 자격 증명을 상속할지 여부입니다. `S3`에서만 작동합니다. |     |
| `s3_storage_class`                        | S3 백업에 사용할 스토리지 클래스입니다. 예시: `STANDARD`                 |     |
