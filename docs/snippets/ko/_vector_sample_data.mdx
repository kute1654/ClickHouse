---
title: Vector 샘플 데이터
---

샘플 데이터셋으로 실험해 보려면 다음 nginx 예시 샘플을 다운로드하십시오.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

<Note>
  이 데이터는 구문 분석을 쉽게 하기 위해 JSON 포맷으로 로그를 출력하도록 구성된 Nginx 인스턴스에서 수집되었습니다. 해당 로그에 대한 Nginx 구성은 [&quot;ClickStack으로 Nginx 로그 모니터링하기&quot;](/ko/clickstack/integration-examples/nginx-logs#configure-nginx)를 참조하십시오.
</Note>
