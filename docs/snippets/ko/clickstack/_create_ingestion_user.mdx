---
title: 수집용 사용자 생성
---

`default`를 재사용하는 대신 collector 전용 사용자를 생성하는 것이 좋습니다. SQL 콘솔을 통해 서비스에 연결한 다음, 다음을 실행하십시오:

```sql
CREATE USER hyperdx_ingest IDENTIFIED WITH sha256_password BY 'ClickH0u3eRocks123!';
GRANT SELECT, INSERT, CREATE DATABASE, CREATE TABLE, CREATE VIEW ON otel.* TO hyperdx_ingest;
```

<Tip>
  위 스니펫의 비밀번호를 강력한 값으로 변경하십시오.
</Tip>

collector는 처음 사용할 때 `otel` 데이터베이스에 로그, 트레이스, 메트릭용 스키마를 생성합니다. 프로덕션 환경용 사용자 설정에 대한 자세한 내용은 [프로덕션 환경으로 전환](/ko/clickstack/managing/production#create-a-database-ingestion-user-managed)을 참조하십시오.
