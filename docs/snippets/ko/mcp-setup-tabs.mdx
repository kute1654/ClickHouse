    <p style={{ marginBottom: '8px', color: '#9ca3af', fontSize: '13px' }}>터미널에서 다음 명령을 실행하세요:</p>

    <CodeBlock language="bash">
      {`claude mcp add clickhouse-docs -- npx -y mintlify-mcp --doc-url https://clickhouse.com/docs`}
    </CodeBlock>
  </Tab>

  <Tab title="Claude Desktop" icon="/images/logo-claude.svg">
    <p style={{ marginBottom: '8px', color: '#9ca3af', fontSize: '13px' }}><code>claude&#95;desktop&#95;config.json</code>에 다음을 추가하세요:</p>

    <CodeBlock language="json">
      {`{
              "mcpServers": {
                "clickhouse-docs": {
                  "command": "npx",
                  "args": ["-y", "mintlify-mcp", "--doc-url", "https://clickhouse.com/docs"]
                }
              }
              }`}
    </CodeBlock>
  </Tab>
