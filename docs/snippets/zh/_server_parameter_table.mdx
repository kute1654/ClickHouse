---
title: 服务器参数表
---

对于每台服务器，需要指定以下参数：

| 参数     | 说明                                                                                                                         | 默认值 |
| ------ | -------------------------------------------------------------------------------------------------------------------------- | --- |
| `host` | 远程服务器的地址。可以使用域名、IPv4 地址或 IPv6 地址。如果指定的是域名，服务器会在启动时发起 DNS 请求，并在服务器运行期间一直使用该解析结果。如果 DNS 请求失败，服务器将无法启动。如果更改了 DNS 记录，则需要重启服务器。 | -   |
| `port` | 用于消息通信的 TCP 端口 (配置中的 `tcp_port`，通常设为 9000) 。不要将其与 `http_port` 混淆。                                                          | -   |
