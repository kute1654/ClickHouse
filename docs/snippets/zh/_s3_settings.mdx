---
title: S3 设置
---

| 设置                                        | 描述                                  | 默认值 |
| ----------------------------------------- | ----------------------------------- | --- |
| `use_same_s3_credentials_for_base_backup` | 执行到 S3 的基准备份时，是否从查询继承凭证。仅对 `S3` 有效。 |     |
| `s3_storage_class`                        | S3 备份使用的存储类。例如：`STANDARD`           |     |
