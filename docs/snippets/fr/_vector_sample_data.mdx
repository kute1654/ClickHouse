title: Données d’exemple Vector
---

Si vous souhaitez faire des essais avec un jeu de données d’exemple, téléchargez l’exemple nginx ci-dessous.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

<Note>
  Ces données ont été collectées à partir d’une instance Nginx configurée pour écrire les logs au format JSON afin d’en faciliter l’analyse. Pour la configuration Nginx correspondante, consultez [&quot;Surveillance des logs Nginx avec ClickStack&quot;](/fr/clickstack/integration-examples/nginx-logs#configure-nginx).
</Note>
