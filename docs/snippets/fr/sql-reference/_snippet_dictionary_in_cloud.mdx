<Tip>
  Si vous utilisez un dictionnaire avec ClickHouse Cloud, utilisez l’option DDL query pour créer vos dictionnaires, et créez votre dictionnaire en tant qu’utilisateur `default`.
  Vérifiez également la liste des sources de dictionnaire prises en charge dans le [guide de compatibilité Cloud](/fr/products/cloud/guides/cloud-compatibility).
</Tip>
