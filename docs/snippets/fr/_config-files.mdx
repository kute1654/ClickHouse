---
title: Fichiers de configuration
---

<Warning>
  **bonnes pratiques**

  Pour configurer ClickHouse Server en ajoutant ou en modifiant des fichiers de configuration, vous devez :

  * Ajouter des fichiers dans le répertoire `/etc/clickhouse-server/config.d/`
  * Ajouter des fichiers dans le répertoire `/etc/clickhouse-server/users.d/`
  * Laisser le fichier `/etc/clickhouse-server/config.xml` tel quel
  * Laisser le fichier `/etc/clickhouse-server/users.xml` tel quel
</Warning>
