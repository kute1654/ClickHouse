---
title: Fichiers de configuration de Keeper
---

<Warning>
  **bonnes pratiques**

  Lorsque vous configurez ClickHouse Keeper en modifiant le fichier de configuration, vous devez :

  * Faire une sauvegarde de `/etc/clickhouse-keeper/keeper_config.xml`
  * Modifier le fichier `/etc/clickhouse-keeper/keeper_config.xml`
</Warning>
