---
title: Obtener credenciales
---

Necesitarás:

* El endpoint HTTPS de tu servicio de ClickHouse Cloud, incluido el protocolo y el puerto; por ejemplo, `https://abc123xyz.us-central1.gcp.clickhouse.cloud:8443`.
* Un nombre de usuario y una contraseña de ClickHouse para la ingestión.

Si no los tienes anotados, abre tu servicio en la [ClickHouse Cloud console](https://console.clickhouse.cloud) y selecciona **Conectar**. Anota la URL del cuadro de diálogo que aparece a continuación. Más abajo crearemos un usuario específico para la ingestión.
