Si tiene problemas con un servicio que no aparece en [fuentes de datos compatibles](/es/integrations/clickpipes/object-storage/amazon-s3/overview#supported-data-sources), [póngase en contacto con nuestro equipo](https://clickhouse.com/company/contact?loc=clickpipes).
