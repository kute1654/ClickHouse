---
title: Archivos de configuración
---

<Warning>
  **prácticas recomendadas**

  Al configurar ClickHouse Server añadiendo o editando archivos de configuración, debe:

  * Añadir archivos al directorio `/etc/clickhouse-server/config.d/`
  * Añadir archivos al directorio `/etc/clickhouse-server/users.d/`
  * Dejar el archivo `/etc/clickhouse-server/config.xml` tal como está
  * Dejar el archivo `/etc/clickhouse-server/users.xml` tal como está
</Warning>
