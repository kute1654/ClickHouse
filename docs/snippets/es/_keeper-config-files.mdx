<Warning>
  **mejores prácticas**

  Al configurar ClickHouse Keeper editando archivos de configuración, debe:

  * Hacer una copia de seguridad del archivo `/etc/clickhouse-keeper/keeper_config.xml`
  * Editar el archivo `/etc/clickhouse-keeper/keeper_config.xml`
</Warning>
