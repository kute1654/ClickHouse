<Tip>
  **ClickHouse Cloud なら管理がシンプルになります**

  [ClickHouse Cloud](/ja/products/cloud/getting-started/intro)
  を利用すると、分片やレプリカの管理に伴う運用負荷を軽減できます。この
  プラットフォームでは、高可用性、レプリケーション、スケーリングに関する処理が自動的に行われます。
  コンピュートとストレージは分離されており、手動での
  設定や継続的な保守を必要とせず、需要に応じてスケールします。

  [詳細はこちら](/ja/products/cloud/features/autoscaling/overview)
</Tip>
