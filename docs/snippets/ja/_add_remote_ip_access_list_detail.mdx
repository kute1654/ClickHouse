---
title: リモート IP Access List の詳細を追加
---

<Accordion title="IP Access List の管理">
  ClickHouse Cloud のサービス一覧から対象のサービスを選択し、**Settings** に切り替えます。接続元となるリモートシステムの IP アドレスまたはアドレス範囲が IP Access List に含まれていない場合は、**Add IPs** で追加できます。

  <Image size="md" img="/images/_snippets/ip-allow-list-check-list.webp" alt="IP Access List で、サービスが自分の IP アドレスからのトラフィックを許可しているか確認する" border />

  ClickHouse Cloud サービスへの接続に必要な個別の IP アドレス、またはアドレス範囲を追加します。必要に応じてフォームを編集し、**Save** します。

  <Image size="md" img="/images/_snippets/ip-allow-list-add-current-ip.webp" alt="ClickHouse Cloud の IP Access List に現在の IP アドレスを追加する" border />
</Accordion>
