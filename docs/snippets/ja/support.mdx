ClickHouse は、ClickHouse Cloud のユーザーおよびお客様向けにサポートサービスを提供しています。私たちの目標は、比類のないパフォーマンス、使いやすさ、そして卓越したスピードと高品質な成果という ClickHouse 製品の特長を体現するサポートチームを提供することです。詳細については、[ClickHouse Support Program](https://clickhouse.com/support/program/) ページをご覧ください。

[Cloud Console にログイン](https://console.clickhouse.cloud/support)し、メニューから **Help -&gt; Support** を選択すると、新しいサポートケースを作成したり、送信済みケースのステータスを確認したりできます。

また、[status page](https://status.clickhouse.com) を購読すると、当社のプラットフォームに影響するインシデントの通知をすばやく受け取れます。

<Note>
  サポートインシデントに関するサービスレベル契約は、サブスクリプションをご利用のお客様にのみ適用されます。現在 ClickHouse Cloud のユーザーでない場合も、できる限りご質問には回答いたしますが、代わりに以下のコミュニティリソースのご利用をお勧めします。

  * [ClickHouse コミュニティ Slack チャンネル](https://clickhouse.com/slack)
  * [その他のコミュニティオプション](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
</Note>
