---
title: サーバーパラメータテーブル
---

各サーバーに対して、次のパラメーターを指定します。

| Parameter | Description                                                                                                                                                                     | Default Value |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------- |
| `host`    | リモートサーバーのアドレスです。ドメイン名、IPv4 アドレス、IPv6 アドレスのいずれかを使用できます。ドメイン名を指定すると、サーバーは起動時に DNS ルックアップを実行し、その結果はサーバーの稼働中保持されます。DNS ルックアップに失敗すると、サーバーは起動しません。DNS レコードを変更した場合は、サーバーを再起動する必要があります。 | -             |
| `port`    | メッセンジャー通信用の TCP ポートです (設定ファイルでは `tcp_port`、通常は 9000 に設定されています) 。`http_port` と混同しないでください。                                                                                        | -             |
