---
title: Vector サンプルデータ
---

サンプルデータセットで試す場合は、次の nginx サンプルをダウンロードしてください。

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

<Note>
  このデータは、パースしやすいようにログを JSON フォーマットで出力するよう設定された Nginx インスタンスから収集されたものです。これらのログに対応する Nginx の設定については、[&quot;ClickStack で Nginx ログを監視する&quot;](/ja/clickstack/integration-examples/nginx-logs#configure-nginx)を参照してください。
</Note>
