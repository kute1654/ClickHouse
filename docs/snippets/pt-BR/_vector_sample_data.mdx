title: Dados de exemplo do Vector
---

Se quiser testar com um conjunto de dados de exemplo, baixe o exemplo do nginx a seguir.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

<Note>
  Estes dados foram coletados de uma instância do Nginx configurada para gerar logs em formato JSON, para facilitar o parsing. Para ver a configuração do Nginx para esses logs, consulte [&quot;Monitorando logs do Nginx com ClickStack&quot;](/pt-BR/clickstack/integration-examples/nginx-logs#configure-nginx).
</Note>
