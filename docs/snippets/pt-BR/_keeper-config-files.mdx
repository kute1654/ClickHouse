title: Arquivos de configuração do Keeper
---

<Warning>
  **melhores práticas**

  Ao configurar o ClickHouse Keeper editando os arquivos de configuração, você deve:

  * Fazer backup de `/etc/clickhouse-keeper/keeper_config.xml`
  * Editar o arquivo `/etc/clickhouse-keeper/keeper_config.xml`
</Warning>
