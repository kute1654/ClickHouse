---
title: Arquivos de Configuração
---

<Warning>
  **melhores práticas**

  Ao configurar o ClickHouse Server adicionando ou editando arquivos de configuração, você deve:

  * Adicionar arquivos ao diretório `/etc/clickhouse-server/config.d/`
  * Adicionar arquivos ao diretório `/etc/clickhouse-server/users.d/`
  * Deixar o arquivo `/etc/clickhouse-server/config.xml` como está
  * Deixar o arquivo `/etc/clickhouse-server/users.xml` como está
</Warning>
