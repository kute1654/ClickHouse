---
title: Adicionar detalhes remotos à IP Access List
---

<Accordion title="Gerenciar sua IP Access List">
  Na lista de serviços do ClickHouse Cloud, escolha o serviço com o qual você vai trabalhar e vá para **Configurações**. Se a IP Access List não incluir o endereço IP ou o intervalo de IPs do sistema remoto que precisa se conectar ao seu serviço do ClickHouse Cloud, você pode resolver o problema com **Adicionar IPs**:

  <Image size="md" img="/images/_snippets/ip-allow-list-check-list.webp" alt="Verifique se o serviço permite tráfego do seu endereço IP na IP Access List" border />

  Adicione o endereço IP individual ou o intervalo de endereços que precisam se conectar ao seu serviço do ClickHouse Cloud. Ajuste o formulário conforme necessário e depois clique em **Salvar**.

  <Image size="md" img="/images/_snippets/ip-allow-list-add-current-ip.webp" alt="Adicione seu endereço IP atual à IP Access List no ClickHouse Cloud" border />
</Accordion>
