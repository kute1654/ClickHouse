---
title: "مقتطف: القاموس في Cloud"
---

<Tip>
  إذا كنت تستخدم قاموسًا مع ClickHouse Cloud، فيُرجى استخدام خيار DDL query لإنشاء قواميسك، وإنشاء القاموس باستخدام المستخدم `default`.
  وتحقق أيضًا من قائمة مصادر القواميس المدعومة في [دليل التوافق مع Cloud](/ar/products/cloud/guides/cloud-compatibility).
</Tip>
