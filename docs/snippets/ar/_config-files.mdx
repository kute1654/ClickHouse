---
title: ملفات الإعداد
---

<Warning>
  **أفضل الممارسات**

  عند إعداد ClickHouse Server عبر إضافة ملفات الإعداد أو تعديلها، ينبغي عليك:

  * إضافة الملفات إلى الدليل `/etc/clickhouse-server/config.d/`
  * إضافة الملفات إلى الدليل `/etc/clickhouse-server/users.d/`
  * ترك الملف `/etc/clickhouse-server/config.xml` كما هو
  * ترك الملف `/etc/clickhouse-server/users.xml` كما هو
</Warning>
