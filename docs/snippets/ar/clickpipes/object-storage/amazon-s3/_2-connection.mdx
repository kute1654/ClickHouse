title: 2 الاتصال
---

<Image img="/images/integrations/data-ingestion/clickpipes/object-storage/amazon-s3/cp_step2.webp" alt="املأ تفاصيل الاتصال" size="lg" border />

* **طريقة المصادقة**: يدعم S3 ClickPipe [بيانات اعتماد IAM](/ar/integrations/clickpipes/object-storage/amazon-s3/overview#iam-credentials) (`Credentials`) و[المصادقة المستندة إلى دور IAM](/ar/integrations/clickpipes/object-storage/amazon-s3/overview#iam-role) (`IAM role`). راجع [الوثائق المرجعية](/ar/integrations/clickpipes/object-storage/amazon-s3/overview#access-control) للحصول على إرشادات حول المصادقة والأذونات.

  * **مسار ملف S3**: يتوقع S3 ClickPipe [URI بأسلوب الاستضافة الافتراضية](https://docs.aws.amazon.com/AmazonS3/latest/userguide/VirtualHosting.html#virtual-hosted-style-access).

    ```bash
    https://bucket-name.s3.region-code.amazonaws.com/key-name
    ```

    يمكنك استخدام أحرف البدل في POSIX لمطابقة عدة ملفات أو بادئات. راجع [الوثائق المرجعية](/ar/integrations/clickpipes/object-storage/amazon-s3/overview#file-pattern-matching) للحصول على إرشادات حول الأنماط المدعومة.
