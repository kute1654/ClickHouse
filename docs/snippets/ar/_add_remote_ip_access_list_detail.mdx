---
title: إضافة تفاصيل قائمة الوصول إلى IP البعيد
---

<Accordion title="إدارة قائمة الوصول إلى IP">
  من قائمة خدمات ClickHouse Cloud، اختر الخدمة التي ستعمل عليها ثم انتقل إلى **Settings**. إذا كانت قائمة الوصول إلى IP لا تتضمن عنوان IP أو نطاق العناوين الخاص بالنظام البعيد الذي يحتاج إلى الاتصال بخدمة ClickHouse Cloud، يمكنك حل المشكلة باستخدام **Add IPs**:

  <Image size="md" img="/images/_snippets/ip-allow-list-check-list.webp" alt="تحقق مما إذا كانت الخدمة تسمح بمرور الزيارات من عنوان IP الخاص بك ضمن قائمة الوصول إلى IP" border />

  أضف عنوان IP الفردي أو نطاق العناوين الذي يحتاج إلى الاتصال بخدمة ClickHouse Cloud. عدّل النموذج بما يناسبك ثم انقر على **Save**.

  <Image size="md" img="/images/_snippets/ip-allow-list-add-current-ip.webp" alt="أضف عنوان IP الحالي إلى قائمة الوصول إلى IP في ClickHouse Cloud" border />
</Accordion>
