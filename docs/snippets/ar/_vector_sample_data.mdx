title: بيانات عينة Vector
---

إذا كنت ترغب في تجربة مجموعة بيانات نموذجية، فنزّل عينة nginx التالية.

```bash
curl -O https://datasets-documentation.s3.eu-west-3.amazonaws.com/clickstack-integrations/access.log
```

<Note>
  جُمعت هذه البيانات من مثيل Nginx مُهيّأ لإخراج السجلات بتنسيق JSON لتسهيل تحليلها. للاطلاع على إعداد Nginx لهذه السجلات، راجع [&quot;مراقبة سجلات Nginx باستخدام ClickStack&quot;](/ar/clickstack/integration-examples/nginx-logs#configure-nginx).
</Note>
