QStudio 是一款免费的 SQL 图形界面工具，可用于运行 SQL 脚本、轻松浏览表、生成图表并导出结果。它支持所有操作系统和各种数据库。

QStudio 通过 JDBC 连接到 ClickHouse。

<div id="1-gather-your-clickhouse-details">
  ## 1. 获取 ClickHouse 连接信息
</div>

QStudio 使用基于 HTTP(S) 的 JDBC 连接 ClickHouse；你需要准备以下信息：

* 端点
* 端口号
* 用户名
* 密码

<ConnectionDetails />

<div id="2-download-qstudio">
  ## 2. 下载 QStudio
</div>

QStudio 可在 https://www.timestored.com/qstudio/download/ 下载。

<div id="3-add-a-database">
  ## 3. 添加数据库
</div>

* 首次打开 QStudio 时，点击菜单中的 **Server-&gt;Add Server**，或点击工具栏上的添加服务器按钮。
* 然后设置以下信息：

<Image img="/images/integrations/sql-clients/qstudio-add-connection.webp" size="lg" border alt="显示 ClickHouse 连接设置的 QStudio 数据库连接配置界面" />

1. 服务器类型：Clickhouse.com
2. 注意：Host 必须包含 https://
   Host: https://abc.def.clickhouse.cloud
   Port: 8443
3. 用户名：default
   密码：`XXXXXXXXXXX`
4. 点击 Add

如果 QStudio 检测到你尚未安装 ClickHouse JDBC 驱动，它会提示你下载并安装：
