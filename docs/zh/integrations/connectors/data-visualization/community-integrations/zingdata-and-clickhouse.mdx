<a href="https://www.zingdata.com/" target="_blank">Zing Data</a> 是一个数据探索与可视化平台。Zing Data 通过 ClickHouse 提供的 JS driver 连接到 ClickHouse。

<div id="how-to-connect">
  ## 如何连接
</div>

1. 收集连接信息。

<ConnectionDetails />

2. 下载或访问 Zing Data

   * 如需在移动设备上将 ClickHouse 与 Zing Data 配合使用，请从 [Google Play Store](https://play.google.com/store/apps/details?id=com.getzingdata.android) 或 [Apple App Store](https://apps.apple.com/us/app/zing-data-collaborative-bi/id1563294091) 下载 Zing Data 应用。

   * 如需在 Web 端将 ClickHouse 与 Zing Data 配合使用，请访问 [Zing web console](https://console.getzingdata.com/) 并创建账户。

3. 添加数据源

   * 如需通过 Zing Data 与 ClickHouse 数据交互，你需要先定义一个 ***数据源***。在 Zing Data 移动应用的菜单中，选择 **Sources**，然后点击 **Add a Datasource**。

   * 如需在 Web 端添加数据源，点击顶部菜单中的 **Data Sources**，再点击 **New Datasource**，然后从下拉菜单中选择 **ClickHouse**

   <Image size="md" img="/images/integrations/data-visualization/zing_01.webp" alt="Zing Data 界面，显示“New Datasource”按钮以及下拉菜单中的 ClickHouse 选项" border />

   <br />

4. 填写连接信息，然后点击 **Check Connection**。

   <Image size="md" img="/images/integrations/data-visualization/zing_02.webp" alt="Zing Data 中的 ClickHouse 连接配置表单，包含 server、port、database、username 和 password 字段" border />

   <br />

5. 如果连接成功，Zing 会引导你进入表选择页面。选择所需的表，然后点击 **Save**。如果 Zing 无法连接到你的数据源，你会看到一条消息，提示你检查凭据并重试。如果检查凭据并重试后问题仍然存在，<a id="contact_link" href="mailto:hello@getzingdata.com">请在此联系 Zing 支持团队。</a>

   <Image size="md" img="/images/integrations/data-visualization/zing_03.webp" alt="Zing Data 的表选择界面，显示带复选框的可用 ClickHouse 表" border />

   <br />

6. 添加 ClickHouse 数据源后，它将在你的 Zing 组织中对所有人可见，可在 **Data Sources** / **Sources** 选项卡下找到。

<div id="creating-charts-and-dashboards-in-zing-data">
  ## 在 Zing Data 中创建图表和仪表盘
</div>

1. 添加 ClickHouse 数据源后，在网页端点击 **Zing App**，或在移动端点击该数据源，即可开始创建图表。

2. 点击表列表中的某个表即可创建图表。

   <Image size="sm" img="/images/integrations/data-visualization/zing_04.webp" alt="Zing Data 界面，显示包含可用 ClickHouse 表的表列表" border />

   <br />

3. 使用可视化查询构建器选择所需字段、聚合等，然后点击 **Run Question**。

   <Image size="md" img="/images/integrations/data-visualization/zing_05.webp" alt="Zing Data 可视化查询构建器界面，显示字段选择和聚合选项" border />

   <br />

4. 如果你熟悉 SQL，也可以编写自定义 SQL 来运行查询并创建图表。

   <Image size="md" img="/images/integrations/data-visualization/zing_06.webp" alt="Zing Data 中的 SQL 编辑器模式，显示 SQL 查询编写界面" border />

   <Image size="md" img="/images/integrations/data-visualization/zing_07.webp" alt="Zing Data 中的 SQL 查询结果，以表格形式显示数据" border />

5. 示例图表如下所示。你可以通过三点菜单保存该问题。你还可以对图表添加评论、标记团队成员、创建实时告警、更改图表类型等。

   <Image size="md" img="/images/integrations/data-visualization/zing_08.webp" alt="Zing Data 中的示例图表可视化，显示来自 ClickHouse 的数据和选项菜单" border />

   <br />

6. 可以在 Home 屏幕中 **仪表盘** 下方点击 “+” 图标来创建仪表盘。还可以将现有问题拖入其中，显示在仪表盘上。

   <Image size="md" img="/images/integrations/data-visualization/zing_09.webp" alt="Zing Data 仪表盘视图，显示以仪表盘布局排列的多个可视化内容" border />

   <br />
