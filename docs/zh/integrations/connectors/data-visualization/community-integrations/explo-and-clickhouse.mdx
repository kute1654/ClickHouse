<div id="2--connect-explo-to-clickhouse">
  ## 2.  将 Explo 连接到 ClickHouse
</div>

1. 注册 Explo 账户。

2. 点击左侧边栏中的 Explo **数据**选项卡。

<Image img="/images/integrations/data-visualization/explo_01.webp" size="sm" alt="数据选项卡" border />

3. 点击右上角的 **连接数据源**。

<Image img="/images/integrations/data-visualization/explo_02.webp" size="sm" alt="连接数据源" border />

4. 在 **入门** 页面填写相关信息

<Image img="/images/integrations/data-visualization/explo_03.webp" size="md" alt="入门" border />

5. 选择 **ClickHouse**

<Image img="/images/integrations/data-visualization/explo_04.webp" size="md" alt="ClickHouse" border />

6. 输入你的 **ClickHouse 凭据**。

<Image img="/images/integrations/data-visualization/explo_05.webp" size="md" alt="凭据" border />

7. 配置 **安全**

<Image img="/images/integrations/data-visualization/explo_06.webp" size="md" alt="安全" border />

8. 在 ClickHouse 中，**将 Explo 的 IP 地址加入白名单**。
   `54.211.43.19, 52.55.98.121, 3.214.169.94, and 54.156.141.148`

<div id="3-create-a-dashboard">
  ## 3. 创建仪表盘
</div>

1. 前往左侧导航栏中的 **Dashboard** 选项卡。

<Image img="/images/integrations/data-visualization/explo_07.webp" size="sm" alt="仪表盘" border />

2. 点击右上角的 **Create Dashboard**，然后为仪表盘命名。至此，你已创建好一个仪表盘。

<Image img="/images/integrations/data-visualization/explo_08.webp" size="sm" alt="创建仪表盘" border />

3. 现在，你应该会看到一个类似这样的界面：

<Image img="/images/integrations/data-visualization/explo_09.webp" size="md" alt="Explo 仪表盘" border />

<div id="4-run-a-sql-query">
  ## 4. 运行 SQL 查询
</div>

1. 在右侧边栏中 schema 标题下找到你的表名。然后将以下命令输入到数据集编辑器中：
   `SELECT * FROM YOUR_TABLE_NAME
   LIMIT 100`

<Image img="/images/integrations/data-visualization/explo_10.webp" size="md" alt="Explo Dashboard" border />

2. 现在点击“运行”，然后前往“预览”选项卡查看数据。

<Image img="/images/integrations/data-visualization/explo_11.webp" size="md" alt="Explo Dashboard" border />

<div id="5-build-a-chart">
  ## 5. 创建图表
</div>

1. 从左侧将柱状图图标拖到画布上。

<Image img="/images/integrations/data-visualization/explo_16.webp" size="sm" alt="Explo Dashboard" border />

2. 选择数据集。此时你应该会看到如下界面：

<Image img="/images/integrations/data-visualization/explo_12.webp" size="sm" alt="Explo Dashboard" border />

3. 按如下方式在 X 轴中填入 **county**，并在 Y 轴部分填入 **Price**：

<Image img="/images/integrations/data-visualization/explo_13.webp" size="sm" alt="Explo Dashboard" border />

4. 现在，将聚合方式更改为 **AVG**。

<Image img="/images/integrations/data-visualization/explo_14.webp" size="sm" alt="Explo Dashboard" border />

5. 现在，我们已经得到了按县划分的房屋平均价格！

<Image img="/images/integrations/data-visualization/explo_15.webp" size="md" alt="Explo Dashboard" />
