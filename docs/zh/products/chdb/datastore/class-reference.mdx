本参考文档介绍了 DataStore API 的核心类。

<div id="datastore">
  ## DataStore
</div>

用于数据处理的主要类，类似于 DataFrame。

```python
from chdb.datastore import DataStore
```

<div id="datastore-constructor">
  ### 构造函数
</div>

```python
DataStore(data=None, columns=None, index=None, dtype=None, copy=None)
```

**参数：**

| 参数        | 类型                            | 说明    |
| --------- | ----------------------------- | ----- |
| `data`    | dict/list/DataFrame/DataStore | 输入数据  |
| `columns` | list                          | 列名    |
| `index`   | Index                         | 行索引   |
| `dtype`   | dict                          | 列数据类型 |
| `copy`    | bool                          | 复制数据  |

**示例：**

```python
# 从字典创建
ds = DataStore({'a': [1, 2, 3], 'b': ['x', 'y', 'z']})

# 从 pandas DataFrame 创建
import pandas as pd
ds = DataStore(pd.DataFrame({'a': [1, 2, 3]}))

# 空 DataStore
ds = DataStore()
```

</div>

```python
ds[ds['age'] <= 25]          # 小于等于
```

<div id="columnexpr-logical">

</div>

```python

ds[(ds['age'] > 25) | (ds['city'] == 'NYC')]    # 或
```

<div id="f-class">
  ### F (函数)
</div>

ClickHouse 函数所在的命名空间。

```python
from chdb.datastore import F, Field

# 聚合
F.sum(Field('amount'))
F.avg(Field('price'))
F.count(Field('id'))
F.quantile(Field('value'), 0.95)

# 条件
F.sum_if(Field('amount'), Field('status') == 'completed')
F.count_if(Field('active'))

# Window
F.row_number().over(order_by='date')
F.lag('price', 1).over(partition_by='product', order_by='date')
```

详情请参阅[聚合](/zh/products/chdb/datastore/aggregation#f-namespace)。

<div id="field-class">
  ### 字段
</div>

通过名称引用列。

```python
from chdb.datastore import Field

# 创建字段引用
amount = Field('amount')
price = Field('price')

# 在表达式中使用
F.sum(Field('amount'))
F.avg(Field('price'))
```

<div id="casewhen-class">
  ### CaseWhen
</div>

CASE WHEN 表达式构建器。

```python
# 创建 case-when 表达式
result = (ds
    .when(ds['score'] >= 90, 'A')
    .when(ds['score'] >= 80, 'B')
    .when(ds['score'] >= 70, 'C')
    .otherwise('F')
)

# 赋值给列
ds['grade'] = result
```

<div id="window-class">
  ### Window
</div>

窗口函数的窗口规范。

```python
from chdb.datastore import F

# 创建窗口
window = F.window(
    partition_by='category',
    order_by='date',
    rows_between=(-7, 0)
)

# 与聚合函数配合使用
ds['rolling_avg'] = F.avg('price').over(window)
```
