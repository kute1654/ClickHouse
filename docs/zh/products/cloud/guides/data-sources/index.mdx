<div id="clickpipes-for-data-ingestion">
  ### 用于数据摄取的 ClickPipes
</div>

ClickPipes 可帮助客户轻松接入来自多种来源的流式数据。更多信息请参阅我们 Integrations 文档中的 [ClickPipes](/zh/integrations/clickpipes/home)。
