<div id="notes">
  ## 说明
</div>

* **operator 的 Helm 图表版本** (`1.18587.1`) 与 operator **镜像标签** (`main-1.18587.1`) 不同。通过 `helm install` 安装时请使用 Helm 图表版本，复制容器镜像时请使用镜像标签。
* **FIPS 构建**会在 server 和 Keeper 的标准镜像标签后追加 `-fips`。
* 构建这些文档时，版本标签会自动自动替换。如果你看到原始模板变量 (例如 `25.12.1.1497`) ，则说明构建步骤尚未运行。
