ClickHouse BYOC 支持多种私有网络方案，可增强安全性并为您的服务提供直接连接能力。本指南将介绍推荐的连接方式，帮助您将部署在自有 AWS 或 GCP 账户中的 ClickHouse Cloud 安全地连接到其他网络或服务，例如内部应用或分析工具。我们将涵盖 VPC Peering、AWS PrivateLink 和 GCP Private Service Connect 等选项，并概述每种方案的主要步骤和注意事项。

如果您需要为 ClickHouse BYOC 部署配置私有网络连接，请按照相关指南中的步骤操作；如需处理更高级的场景，也可联系 ClickHouse Support 获取帮助。
