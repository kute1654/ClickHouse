ClickHouse Cloud 是由广受欢迎的开源 OLAP 数据库 ClickHouse 的创建者打造的云服务。
你可以通过[注册免费试用](https://console.clickhouse.cloud/signUp)体验 ClickHouse Cloud。

<div id="clickhouse-cloud-benefits">
  ## ClickHouse Cloud 的优势
</div>

下面介绍使用 ClickHouse Cloud 的一些优势：

* **快速实现价值**：无需规划集群规模或进行扩缩容，即可立即开始构建。
* **无缝扩缩容**：自动扩缩容可适应不断变化的工作负载，因此您无需为峰值用量过度预配资源。
* **无服务器运维**：集群规模、扩缩容、安全性、可靠性和升级均由我们负责，您可以高枕无忧。
* **透明定价**：只为实际使用的资源付费，并提供资源预留和扩缩容控制功能。
* **总体拥有成本低**：兼具出色的性价比和较低的管理开销。
* **广泛的生态系统**：您可以继续使用自己喜爱的数据库连接器、可视化工具、SQL 以及各种语言客户端。

<div id="oss-vs-clickhouse-cloud">
  ## ClickHouse Cloud 使用哪个版本的 ClickHouse？
</div>

ClickHouse Cloud 会定期将您的服务升级到较新版本，其中包含修复、新功能和性能提升。在核心数据库版本于开源版本中发布后，我们还会在云端暂存环境中进行额外验证，通常需要 6–8 周，之后才会逐步发布到生产环境。发布会按云服务提供商、服务类型和区域分阶段进行。

您可以通过订阅特定的发布渠道，为您的 ClickHouse Cloud 服务指定升级计划。例如，我们提供[“快速”发布渠道](/zh/products/cloud/features/admin-features/upgrades#fast-release-channel-early-upgrades)，让您能够在常规发布计划之前接收更新；同时也提供[“慢速”发布渠道](/zh/products/cloud/features/admin-features/upgrades#slow-release-channel-deferred-upgrades)以及其他更细粒度的延期升级选项。

有关 ClickHouse Cloud 升级流程的概览 (包括向后兼容性保证) ，请参阅[Upgrades](/zh/products/cloud/features/admin-features/upgrades)参考文档。
