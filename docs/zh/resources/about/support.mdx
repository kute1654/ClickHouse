ClickHouse 为 ClickHouse Cloud 用户和客户提供支持服务。我们的目标是打造一支体现 ClickHouse 产品特质的支持团队——性能卓越、易于使用，并能以极快的速度提供高质量的支持结果。更多详细信息，请[访问我们的 ClickHouse Support Program](https://clickhouse.com/support/program/) 页面。

[登录 Cloud Console](https://console.clickhouse.cloud/support)，然后在菜单中选择 **Help -&gt; Support**，即可创建新的支持工单并查看已提交工单的状态。

您还可以订阅我们的[状态页面](https://status.clickhouse.com)，以便在平台发生任何事件时及时收到通知。

<Note>
  请注意，只有订阅客户的支持事件才享有服务级别协议。如果您目前不是 ClickHouse Cloud 用户，虽然我们会尽力回答您的问题，但仍建议您优先使用以下社区资源之一：

  * [ClickHouse 社区 Slack 频道](https://clickhouse.com/slack)
  * [其他社区选项](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
</Note>
