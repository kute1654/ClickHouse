<div id="question">
  ## 问题
</div>

如何更改我在 ClickHouse Cloud 中的账单联系人？

<div id="answer">
  ## 答案
</div>

要以 Admin 身份更改账单联系人，请按以下步骤操作：

1. 邀请一位新用户以 Admin 身份加入 Cloud 组织。
2. 对方接受邀请后，进入 ClickHouse Cloud 控制台的账单页面 (Admin-&gt;Billing) ，找到“账单联系人”部分。
3. 点击 Edit 按钮，将新的 Admin 用户设为账单联系人。
