在 Aarch64 机器上为 Aarch64 构建 ClickHouse 不需要任何特殊步骤。

如果要在 x86 Linux 机器上为 AArch64 交叉编译 ClickHouse，请向 `cmake` 传递以下参数：`-DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-aarch64.cmake`
