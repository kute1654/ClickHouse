[Model Context Protocol](https://www.anthropic.com/news/model-context-protocol) (MCP) 是由 Anthropic 开发的一项标准，使 AI 助手能够与外部系统无缝集成。
该协议允许 AI 助手以安全且标准化的方式连接到数据源、API、数据库等。

MCP 在 AI 模型与各种服务之间建立了一个通用接口，无需再为每种集成单独实现自定义工具。
你可以将它理解为专为 AI 系统设计的一套通用 API 标准。

MCP 的关键优势在于，AI 库只需针对该协议实现一次支持。
之后，所有兼容 MCP 的服务都可立即接入，从而为 AI 库维护者节省大量时间。

<div id="mcp-architecture">
  ## MCP 的架构是什么？
</div>

MCP 采用客户端-服务器架构：

* 客户端 (如 Claude Desktop、Cursor 或 VS Code) 会与 MCP 服务器建立连接。你可以在 GitHub 仓库 [awesome-mcp-clients](https://github.com/punkpeye/awesome-mcp-clients?tab=readme-ov-file#windsurf) 中查看客户端汇总。
* 服务器通过标准化接口提供工具和能力。你可以在 GitHub 仓库 [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) 中查看服务器汇总。
* 随后，AI 模型便可在需要时使用这些工具来访问外部数据和功能。

<div id="clickhouse-mcp-server">
  ## ClickHouse 有 MCP 服务器吗？
</div>

有！
[ClickHouse MCP 服务器](https://github.com/ClickHouse/mcp-clickhouse)提供以下工具：

* `run_select_query` - 在你的 ClickHouse 集群上执行 SQL 查询。
* `list_databases` - 列出你的 ClickHouse 集群中的所有数据库。
* `list_tables` - 列出某个数据库中的所有表。

如果你在寻找 ClickHouse Cloud 中的远程 MCP 服务器，请参阅[&quot;Cloud 中的远程 MCP 服务器&quot;](/zh/products/cloud/features/ai-ml/remote-mcp)页面

<Tip>
  **Agent 技能**

  [ClickHouse Agent Skills 仓库](https://github.com/ClickHouse/agent-skills)提供了一套封装好的说明，可为 AI 编码智能体 (Claude Code、Cursor、Copilot 等) 补充特定领域的专业能力。该仓库为 ClickHouse 数据库提供相关技能，涵盖 schema 设计、查询优化和数据摄取模式。
</Tip>
