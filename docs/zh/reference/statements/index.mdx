用户通过 SQL 语句与 ClickHouse 交互。ClickHouse 支持常见的 SQL 语句，如 [SELECT](/zh/reference/statements/select/index) 和 [CREATE](/zh/reference/statements/create/index)，也提供 [KILL](/zh/reference/statements/kill) 和 [OPTIMIZE](/zh/reference/statements/optimize) 等专用语句。
