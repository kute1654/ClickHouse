```sql
ALTER TABLE [db].name [ON CLUSTER cluster] APPLY DELETED MASK [IN PARTITION partition_id]
```

该命令会应用由[轻量级删除](/zh/reference/statements/delete)生成的掩码，并强制将标记为已删除的行从磁盘中移除。此命令属于重量级变更，其语义等同于查询 `ALTER TABLE [db].name DELETE WHERE _row_exists = 0`。

<Note>
  它仅适用于 [`MergeTree`](/zh/reference/engines/table-engines/mergetree-family/mergetree) 家族中的表 (包括[复制](/zh/reference/engines/table-engines/mergetree-family/replication)表) 。
</Note>
