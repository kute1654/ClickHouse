<div id="description">
  ## 描述
</div>

包含有关已缓存 DNS 记录的信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/zh/reference/data-types/index)) — 主机名。
* `ip_address` ([String](/zh/reference/data-types/index)) — IP 地址。
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/zh/reference/data-types/index)) — IP 地址族。
* `cached_at` ([DateTime](/zh/reference/data-types/index)) — 记录被缓存时的时间戳。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| 主机名       | IP 地址     | IP 地址族 | cached&#95;at           |
| :-------- | :-------- | :----- | :------------------ |
| localhost | ::1       | IPv6   | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1 | IPv4   | 2024-02-11 17:04:40 |
