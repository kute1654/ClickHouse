<div id="description">
  ## 描述
</div>

包含配置文件中可用集群及其服务器的信息。

<div id="Columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `cluster` ([String](/zh/reference/data-types/index)) — 集群名称。
* `shard_num` ([UInt32](/zh/reference/data-types/index)) — 集群中的分片编号，从 1 开始。
* `shard_name` ([String](/zh/reference/data-types/index)) — 集群中分片的名称。
* `shard_weight` ([UInt32](/zh/reference/data-types/index)) — 写入数据时该分片的相对权重。
* `internal_replication` ([UInt8](/zh/reference/data-types/index)) — 用于指示该主机是否属于某个可自行复制数据的节点组的标志。
* `replica_num` ([UInt32](/zh/reference/data-types/index)) — 分片中的副本编号，从 1 开始。
* `host_name` ([String](/zh/reference/data-types/index)) — 配置中指定的主机名。
* `host_address` ([String](/zh/reference/data-types/index)) — 从 DNS 获取的主机 IP 地址。
* `port` ([UInt16](/zh/reference/data-types/index)) — 用于连接到服务器的端口。
* `is_local` ([UInt8](/zh/reference/data-types/index)) — 用于指示该主机是否为本地主机的标志。
* `user` ([String](/zh/reference/data-types/index)) — 用于连接到服务器的用户名。
* `default_database` ([String](/zh/reference/data-types/index)) — 默认数据库名称。
* `errors_count` ([UInt32](/zh/reference/data-types/index)) — 该主机连接副本失败的次数。
* `slowdowns_count` ([UInt32](/zh/reference/data-types/index)) — 使用对冲请求建立连接时，因响应变慢而导致切换副本的次数。
* `estimated_recovery_time` ([UInt32](/zh/reference/data-types/index)) — 距离副本错误计数清零并恢复为正常状态的剩余秒数。
* `database_shard_name` ([String](/zh/reference/data-types/index)) — `Replicated` 数据库分片的名称 (适用于属于 `Replicated` 数据库的集群) 。
* `database_replica_name` ([String](/zh/reference/data-types/index)) — `Replicated` 数据库副本的名称 (适用于属于 `Replicated` 数据库的集群) 。
* `is_shared_catalog_cluster` ([UInt8](/zh/reference/data-types/index)) — Bool，表示该集群是否属于 Shared Catalog。
* `is_active` ([Nullable(UInt8)](/zh/reference/data-types/index)) — Replicated 数据库副本的状态 (适用于属于 Replicated 数据库的集群) ：1 表示&#39;副本在线&#39;，0 表示&#39;副本离线&#39;，NULL 表示&#39;未知&#39;。
* `unsynced_after_recovery` ([Nullable(UInt8)](/zh/reference/data-types/index)) — 指示某个 Replicated 数据库副本在创建或恢复后，其复制延迟是否大于 max&#95;replication&#95;lag&#95;to&#95;enqueue。
* `replication_lag` ([Nullable(UInt32)](/zh/reference/data-types/index)) — `Replicated` 数据库副本的复制延迟 (适用于属于 Replicated 数据库的集群) 。
* `recovery_time` ([Nullable(UInt64)](/zh/reference/data-types/index)) — `Replicated` 数据库副本的恢复时间 (适用于属于 `Replicated` 数据库的集群) ，单位为毫秒。

{/*AUTOGENERATED_END*/}
