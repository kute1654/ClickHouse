<div id="description">
  ## 描述
</div>

包含当前正在运行的后台拉取的信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 表名称。
* `elapsed` ([Float64](/zh/reference/data-types/index)) — 自开始显示当前正在运行的后台拉取以来经过的时间 (以秒为单位) 。
* `progress` ([Float64](/zh/reference/data-types/index)) — 已完成工作的比例，范围为 0 到 1。
* `result_part_name` ([String](/zh/reference/data-types/index)) — 显示当前正在运行的后台拉取时，将形成的结果分片名称。
* `result_part_path` ([String](/zh/reference/data-types/index)) — 显示当前正在运行的后台拉取时，将形成的结果分片的绝对路径。
* `partition_id` ([String](/zh/reference/data-types/index)) — 分区 ID。
* `total_size_bytes_compressed` ([UInt64](/zh/reference/data-types/index)) — 结果分片中压缩数据的总大小 (以字节为单位) 。
* `bytes_read_compressed` ([UInt64](/zh/reference/data-types/index)) — 从结果分片中读取的压缩字节数。
* `source_replica_path` ([String](/zh/reference/data-types/index)) — 源副本的绝对路径。
* `source_replica_hostname` ([String](/zh/reference/data-types/index)) — 源副本的主机名。
* `source_replica_port` ([UInt16](/zh/reference/data-types/index)) — 源副本的端口号。
* `interserver_scheme` ([String](/zh/reference/data-types/index)) — 服务器间 scheme 的名称。
* `URI` ([String](/zh/reference/data-types/index)) — 统一资源标识符。
* `to_detached` ([UInt8](/zh/reference/data-types/index)) — 该 flag 表示当前正在运行的后台拉取是否使用 TO DETACHED 表达式执行。
* `thread_id` ([UInt64](/zh/reference/data-types/index)) — 线程标识符。

{/*AUTOGENERATED_END*/}
