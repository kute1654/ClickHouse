<div id="description">
  ## 描述
</div>

包含[字典](/zh/reference/statements/create/dictionary)的相关信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 包含通过 DDL 查询创建的字典的数据库名称。对于其他字典，该值为空字符串。
* `name` ([String](/zh/reference/data-types/index)) — 字典名称。
* `uuid` ([UUID](/zh/reference/data-types/index)) — 字典 UUID。
* `status` ([Enum8(&#39;NOT&#95;LOADED&#39; = 0, &#39;LOADED&#39; = 1, &#39;FAILED&#39; = 2, &#39;LOADING&#39; = 3, &#39;FAILED&#95;AND&#95;RELOADING&#39; = 4, &#39;LOADED&#95;AND&#95;RELOADING&#39; = 5, &#39;NOT&#95;EXIST&#39; = 6)](/zh/reference/data-types/index)) — 字典状态。可能的值：
  * **NOT&#95;LOADED** — 字典未加载，因为尚未被使用
  * **LOADED** — 字典已成功加载
  * **FAILED** — 由于发生错误，无法加载字典
  * **LOADING** — 字典当前正在加载
  * **LOADED&#95;AND&#95;RELOADING** — 字典已成功加载
  * **and is being reloaded right now (frequent reasons: SYSTEM RELOAD DICTIONARY query** —
  * **timeout** —
  * **dictionary config has changed)** —
  * **FAILED&#95;AND&#95;RELOADING** — 由于发生错误，字典未能加载，当前正在重新加载。
* `origin` ([String](/zh/reference/data-types/index)) — 描述该字典的配置文件路径。
* `type` ([String](/zh/reference/data-types/index)) — 字典分配类型。在内存中存储字典。
* `key.names` ([Array(String)](/zh/reference/data-types/index)) — 字典提供的键名称数组。
* `key.types` ([Array(String)](/zh/reference/data-types/index)) — 字典提供的对应键类型数组。
* `attribute.names` ([Array(String)](/zh/reference/data-types/index)) — 字典提供的 attribute 名称数组。
* `attribute.types` ([Array(String)](/zh/reference/data-types/index)) — 字典提供的对应 attribute 类型数组。
* `bytes_allocated` ([UInt64](/zh/reference/data-types/index)) — 为字典分配的 RAM 量。
* `hierarchical_index_bytes_allocated` ([UInt64](/zh/reference/data-types/index)) — 为层次化索引分配的 RAM 量。
* `query_count` ([UInt64](/zh/reference/data-types/index)) — 自字典加载后或自上次成功重启后产生的查询数量。
* `hit_rate` ([Float64](/zh/reference/data-types/index)) — 对于 cache 字典，值命中缓存的使用占比。
* `found_rate` ([Float64](/zh/reference/data-types/index)) — 找到值的使用占比。
* `element_count` ([UInt64](/zh/reference/data-types/index)) — 字典中存储的条目数。
* `load_factor` ([Float64](/zh/reference/data-types/index)) — 字典的填充率 (对于哈希字典，则表示哈希表的填充率) 。
* `source` ([String](/zh/reference/data-types/index)) — 描述该字典字典源的文本。
* `lifetime_min` ([UInt64](/zh/reference/data-types/index)) — 字典在内存中的最小 lifetime，超过该时间后 ClickHouse 会尝试重新加载字典 (如果设置了 invalidate&#95;query，则仅在字典发生变化时重新加载) 。单位为秒。
* `lifetime_max` ([UInt64](/zh/reference/data-types/index)) — 字典在内存中的最大 lifetime，超过该时间后 ClickHouse 会尝试重新加载字典 (如果设置了 invalidate&#95;query，则仅在字典发生变化时重新加载) 。单位为秒。
* `loading_start_time` ([DateTime](/zh/reference/data-types/index)) — 开始加载字典的时间。
* `last_successful_update_time` ([DateTime](/zh/reference/data-types/index)) — 字典上次成功加载或更新完成的时间。可用于监控字典源的一些问题并排查原因。
* `error_count` ([UInt64](/zh/reference/data-types/index)) — 自上次成功加载以来的错误次数。可用于监控字典源的一些问题并排查原因。
* `loading_duration` ([Float32](/zh/reference/data-types/index)) — 字典加载耗时。
* `last_exception` ([String](/zh/reference/data-types/index)) — 如果字典未能创建，则为创建或重新加载字典时发生的错误文本。
* `comment` ([String](/zh/reference/data-types/index)) — 字典的注释文本。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

配置字典：

```sql
CREATE DICTIONARY dictionary_with_comment
(
    id UInt64,
    value String
)
PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() TABLE 'source_table'))
LAYOUT(FLAT())
LIFETIME(MIN 0 MAX 1000)
COMMENT 'The temporary dictionary';
```

确保字典已加载。

```sql
SELECT * FROM system.dictionaries LIMIT 1 FORMAT Vertical;
```
