<div id="description">
  ## 描述
</div>

有关[可刷新materialized view](/zh/reference/statements/create/view#refreshable-materialized-view)的信息。包含所有可刷新materialized view，无论当前是否有刷新正在进行。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 该表所在数据库的名称。
* `view` ([String](/zh/reference/data-types/index)) — 表名。
* `uuid` ([UUID](/zh/reference/data-types/index)) — 表的 UUID (Atomic 数据库) 。
* `status` ([String](/zh/reference/data-types/index)) — 刷新的当前状态。
* `last_success_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 最近一次成功刷新开始的时间。如果自服务器启动或表创建以来未发生过成功刷新，则为 NULL。
* `last_success_duration_ms` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 最近一次刷新所耗费的时间。
* `last_refresh_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 最近一次刷新尝试完成的时间 (如果已知) ，或开始的时间 (如果未知或仍在运行) 。如果自服务器启动或表创建以来未发生过刷新尝试，则为 NULL。
* `last_refresh_replica` ([String](/zh/reference/data-types/index)) — 如果已启用协调，则为执行当前 (如果正在运行) 或上一次 (如果未运行) 刷新尝试的副本名称。
* `next_refresh_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 计划开始下一次刷新的时间。如果当前无法确定下一次刷新时间，则为 NULL，例如在等待依赖项时 (`status` 为 `WaitingForDependencies` 或 `MissingDependencies`) 。
* `exception` ([String](/zh/reference/data-types/index)) — 如果上一次尝试失败，则为该次尝试的错误消息。
* `retry` ([UInt64](/zh/reference/data-types/index)) — 当前刷新截至目前的失败重试次数。如果 `status` 为 `RunningOnAnotherReplica`，则不可用。
* `progress` ([Nullable(Float64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新的进度，取值范围为 0 到 1。如果 `status` 为 `RunningOnAnotherReplica` 或刷新未在运行，则为 NULL。
* `read_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新所读取的行数。如果 `status` 为 `RunningOnAnotherReplica`，则为 NULL。
* `read_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新所读取的字节数。如果 `status` 为 `RunningOnAnotherReplica`，则为 NULL
* `total_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新预计需要读取的总行数。如果 `status` 为 `RunningOnAnotherReplica`，则为 NULL
* `written_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新所写入的行数。如果 `status` 为 `RunningOnAnotherReplica`，则为 NULL
* `written_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 给定副本上当前正在运行或最近一次完成的刷新所写入的字节数。如果 `status` 为 `RunningOnAnotherReplica`，则为 NULL

{/*AUTOGENERATED_END*/}

    next_refresh_time
