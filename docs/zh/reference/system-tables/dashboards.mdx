<div id="description">
  ## 描述
</div>

包含可通过 [HTTP 接口](/zh/concepts/features/interfaces/http) 访问的 `/dashboard` 页面所使用的查询。
该表可用于监控和故障排查。仪表盘中的每个图表在该表中都对应一行。

<Note>
  `/dashboard` 页面不仅可以渲染来自 `system.dashboards` 的查询，还可以渲染来自任何具有相同 schema 的表的查询。
  这可用于创建自定义仪表盘。
</Note>

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `dashboard` ([String](/zh/reference/data-types/index)) — 仪表盘名称。
* `title` ([String](/zh/reference/data-types/index)) — 图表标题。
* `query` ([String](/zh/reference/data-types/index)) — 用于获取要显示数据的查询。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT *
FROM system.dashboards
WHERE title ILIKE '%CPU%'
```
