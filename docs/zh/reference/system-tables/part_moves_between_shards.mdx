<div id="description">
  ## 描述
</div>

包含当前正在分片间移动的 分片 及其进度信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 执行移动操作的数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 执行移动操作的表名称。
* `task_name` ([String](/zh/reference/data-types/index)) — 移动任务的名称。
* `task_uuid` ([UUID](/zh/reference/data-types/index)) — 移动任务的标识符。
* `create_time` ([日期时间](/zh/reference/data-types/index)) — 任务的创建时间。
* `part_name` ([String](/zh/reference/data-types/index)) — 正在移动的分片名称。
* `part_uuid` ([UUID](/zh/reference/data-types/index)) — 正在移动的分片 UUID。
* `to_shard` ([String](/zh/reference/data-types/index)) — 目标分片的名称。
* `dst_part_name` ([String](/zh/reference/data-types/index)) — 结果分片名称。
* `update_time` ([日期时间](/zh/reference/data-types/index)) — 最近一次更新的时间。
* `state` ([String](/zh/reference/data-types/index)) — 此次移动的当前状态。
* `rollback` ([UInt8](/zh/reference/data-types/index)) — 指示该操作是否已回滚的标志。
* `num_tries` ([UInt32](/zh/reference/data-types/index)) — 完成该操作的尝试次数。
* `last_exception` ([String](/zh/reference/data-types/index)) — 最后一次异常的名称 (如有) 。

{/*AUTOGENERATED_END*/}
