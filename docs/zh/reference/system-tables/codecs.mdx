<div id="description">
  ## 描述
</div>

包含有关压缩和加密编解码器的信息。

你可以使用此表查看可用的压缩和加密编解码器信息

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 编解码器名称。
* `method_byte` ([UInt8](/zh/reference/data-types/index)) — 在压缩文件中标识编解码器的字节。
* `is_compression` ([UInt8](/zh/reference/data-types/index)) — 如果此编解码器执行压缩，则为 true；否则，它也可能只是有助于压缩的某种转换。
* `is_generic_compression` ([UInt8](/zh/reference/data-types/index)) — 该编解码器是通用压缩算法，例如 lz4、zstd。
* `is_encryption` ([UInt8](/zh/reference/data-types/index)) — 该编解码器用于加密。
* `is_timeseries_codec` ([UInt8](/zh/reference/data-types/index)) — 该编解码器用于浮点时间序列。
* `is_experimental` ([UInt8](/zh/reference/data-types/index)) — 该编解码器为 Experimental。
* `description` ([String](/zh/reference/data-types/index)) — 编解码器的概要说明。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```
