<div id="description">
  ## 说明
</div>

包含所有QUOTA在各个时间间隔上的最大值信息。一个QUOTA可以对应任意数量的行，也可以不对应任何行。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/zh/reference/data-types/index)) — QUOTA 名称。
* `duration` ([UInt32](/zh/reference/data-types/index)) — 用于计算资源消耗的时间间隔长度，单位为秒。
* `is_randomized_interval` ([UInt8](/zh/reference/data-types/index)) — 布尔值。表示该时间间隔是否随机化。若未随机化，则时间间隔始终在固定时间开始。例如，1 分钟的时间间隔总是从整分钟开始 (即可以从 11:20:00 开始，但绝不会从 11:20:01 开始) ，1 天的时间间隔则总是从 UTC 午夜开始。如果时间间隔已随机化，则第一个时间间隔会在随机时间开始，后续时间间隔依次连续开始。取值：0 — 时间间隔未随机化，1 — 时间间隔已随机化。
* `max_queries` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许执行的所有类型查询的最大数量。
* `max_query_selects` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许执行的 SELECT 查询的最大数量。
* `max_query_inserts` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许执行的 INSERT 查询的最大数量。
* `max_errors` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许返回错误的查询最大数量。
* `max_result_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询结果集的最大总行数。
* `max_result_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询结果集的最大总字节数。
* `max_read_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询在执行期间可读取的最大行数。
* `max_read_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询在执行期间可读取的最大字节数。
* `max_execution_time` ([Nullable(Float64)](/zh/reference/data-types/index)) — 在指定时间段内允许所有查询执行的最长时间 (以纳秒为单位)
* `max_written_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询在执行期间可写入的最大字节数。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的连续身份验证失败的最大次数。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许任意单个归一化查询的最大执行次数。
