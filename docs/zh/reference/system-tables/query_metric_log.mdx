<div id="description">
  ## 描述
</div>

包含来自表 `system.events` 的各个查询的内存和指标值的历史记录，并定期写入磁盘。

查询一旦开始，系统就会按 `query_metric_log_interval` 毫秒的固定间隔收集数据 (默认值为 1000
) 。如果查询耗时超过 `query_metric_log_interval`，则在查询结束时也会收集数据。
