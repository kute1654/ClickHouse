<div id="description">
  ## 描述
</div>

包含用于检查 `clickhouse` 二进制文件的内部信息。访问时需要 `内部信息` 权限。
此表仅对 C++ 专家和 ClickHouse 工程师有用。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/zh/reference/data-types/index)) — 二进制文件中的符号名称。该名称经过名称修饰。你可以使用 demangle(symbol) 获取可读名称。
* `address_begin` ([UInt64](/zh/reference/data-types/index)) — 二进制文件中该符号的起始地址。
* `address_end` ([UInt64](/zh/reference/data-types/index)) — 二进制文件中该符号的结束地址。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
