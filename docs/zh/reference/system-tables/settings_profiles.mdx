<div id="description">
  ## 描述
</div>

包含已配置 profile 的各项属性。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — profile 名称。
* `id` ([UUID](/zh/reference/data-types/index)) — profile ID。
* `storage` ([String](/zh/reference/data-types/index)) — setting profile 存储的路径。在 `access_control_path` 参数中配置。
* `num_elements` ([UInt64](/zh/reference/data-types/index)) — `system.settings_profile_elements` 表中此 profile 的元素数量。
* `apply_to_all` ([UInt8](/zh/reference/data-types/index)) — 表示该 profile 已应用于所有角色和/或用户。
* `apply_to_list` ([Array(String)](/zh/reference/data-types/index)) — 应用此 profile 的角色和/或用户列表。
* `apply_to_except` ([Array(String)](/zh/reference/data-types/index)) — 此 profile 将应用于除所列角色和/或用户之外的所有角色和/或用户。

{/*AUTOGENERATED_END*/}
