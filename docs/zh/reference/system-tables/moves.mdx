<div id="description">
  ## 描述
</div>

该表包含 [MergeTree](/zh/reference/engines/table-engines/mergetree-family/mergetree) 表中正在进行的[数据分区片段移动](/zh/reference/statements/alter/partition#move-partitionpart)相关信息。每次数据分区片段移动对应一行。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 包含正在移动的数据分区片段的表名称。
* `elapsed` ([Float64](/zh/reference/data-types/index)) — 自数据分区片段开始移动以来经过的时间 (以秒为单位) 。
* `target_disk_name` ([String](/zh/reference/data-types/index)) — 数据分区片段将要移动到的磁盘名称。
* `target_disk_path` ([String](/zh/reference/data-types/index)) — 该磁盘在文件系统中的挂载点路径。
* `part_name` ([String](/zh/reference/data-types/index)) — 正在移动的数据分区片段名称。
* `part_size` ([UInt64](/zh/reference/data-types/index)) — 数据分区片段的大小。
* `thread_id` ([UInt64](/zh/reference/data-types/index)) — 执行移动操作的线程标识符。

{/*AUTOGENERATED_END*/}
