<div id="description">
  ## 说明
</div>

此表包含可即时计算并以 Prometheus 格式导出的直方图指标，并且始终保持最新。用于替代已弃用的 `system.latency_log`。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/zh/reference/data-types/index)) — 指标名称。
* `value` ([Float64](/zh/reference/data-types/index)) — 指标值。
* `description` ([String](/zh/reference/data-types/index)) — 指标说明。
* `labels` ([Map(String, String)](/zh/reference/data-types/index)) — 标记。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

您可以使用如下查询，将所有直方图指标导出为 Prometheus 格式。

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
