<div id="description">
  ## 描述
</div>

包含每个已分离表的相关信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 该表所在的数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 表名。
* `uuid` ([UUID](/zh/reference/data-types/index)) — 表的 uuid (Atomic 数据库) 。
* `metadata_path` ([String](/zh/reference/data-types/index)) — 文件系统中该表元数据的路径。
* `is_permanently` ([UInt8](/zh/reference/data-types/index)) — 该表已被永久分离。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
