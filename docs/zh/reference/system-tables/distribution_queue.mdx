<div id="description">
  ## 描述
</div>

包含有关位于队列中、等待发送到各个分片的本地文件的信息。这些本地文件包含新的 parts，这些 parts 是在以异步模式向 Distributed 表插入新数据时创建的。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 表名称。
* `data_path` ([String](/zh/reference/data-types/index)) — 本地文件所在文件夹的路径。
* `is_blocked` ([UInt8](/zh/reference/data-types/index)) — 该标志表示是否禁止向服务器发送本地文件。
* `error_count` ([UInt64](/zh/reference/data-types/index)) — 错误次数。
* `data_files` ([UInt64](/zh/reference/data-types/index)) — 文件夹中的本地文件数。
* `data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 本地文件中压缩数据的大小，单位为字节。
* `broken_data_files` ([UInt64](/zh/reference/data-types/index)) — 因错误而被标记为损坏的文件数。
* `broken_data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 损坏文件中压缩数据的大小，单位为字节。
* `last_exception` ([String](/zh/reference/data-types/index)) — 最近一次发生的错误的文本消息 (如有) 。
* `last_exception_time` ([DateTime](/zh/reference/data-types/index)) — 最近一次发生异常的时间。

{/*AUTOGENERATED_END*/}
