<div id="description">
  ## 描述
</div>

包含已配置[角色](/zh/concepts/features/security/access-rights#role-management)的信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 角色名称。
* `id` ([UUID](/zh/reference/data-types/index)) — 角色 ID。
* `storage` ([String](/zh/reference/data-types/index)) — 角色存储路径。在 `access_control_path` 参数中配置。

{/*AUTOGENERATED_END*/}
