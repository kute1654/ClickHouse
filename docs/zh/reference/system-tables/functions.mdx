<div id="description">
  ## 描述
</div>

包含普通函数和聚合函数的相关信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 函数名称。
* `is_aggregate` ([UInt8](/zh/reference/data-types/index)) — 函数是否为 聚合函数。
* `case_insensitive` ([UInt8](/zh/reference/data-types/index)) — 函数名是否可不区分大小写使用。
* `alias_to` ([String](/zh/reference/data-types/index)) — 如果该函数名是别名，则为其原始函数名。
* `create_query` ([String](/zh/reference/data-types/index)) — 已废弃。
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/zh/reference/data-types/index)) — 已废弃。
* `description` ([String](/zh/reference/data-types/index)) — 对函数功能的概述。
* `syntax` ([String](/zh/reference/data-types/index)) — 函数签名。
* `arguments` ([String](/zh/reference/data-types/index)) — 函数的 argument。
* `parameters` ([String](/zh/reference/data-types/index)) — 函数的 parameter (仅适用于 聚合函数) 。
* `returned_value` ([String](/zh/reference/data-types/index)) — 函数的返回值。
* `examples` ([String](/zh/reference/data-types/index)) — 使用示例。
* `introduced_in` ([String](/zh/reference/data-types/index)) — 首次引入该函数的 ClickHouse 版本。
* `categories` ([String](/zh/reference/data-types/index)) — 函数所属类别。
* `deterministic` ([Nullable(UInt8)](/zh/reference/data-types/index)) — 对于相同的参数，函数是否总是返回相同结果。未知时为 NULL (例如 聚合函数 或用户自定义函数) 。
* `higher_order` ([Nullable(UInt8)](/zh/reference/data-types/index)) — 该函数是否为高阶函数，即至少接受一个 lambda expression 作为 argument (例如 arrayMap、arrayFilter、mapApply) 。未知时为 NULL。

{/*AUTOGENERATED_END*/}
