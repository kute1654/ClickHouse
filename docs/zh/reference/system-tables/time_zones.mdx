<div id="description">
  ## 描述
</div>

列出了 ClickHouse 服务器 支持的时区。该列表可能会因 ClickHouse 版本而异。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/zh/reference/data-types/index)) — 支持的时区列表。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
