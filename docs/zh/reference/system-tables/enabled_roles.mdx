<div id="description">
  ## 描述
</div>

包含当前所有已启用角色，包括当前用户当前启用的角色，以及授予当前角色的其他角色。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/zh/reference/data-types/index)) — 角色名称。
* `with_admin_option` ([UInt8](/zh/reference/data-types/index)) — 若该角色具有 ADMIN OPTION 权限，则为 1。
* `is_current` ([UInt8](/zh/reference/data-types/index)) — 标志位，表示 `enabled_role` 是否为当前用户的当前角色。
* `is_default` ([UInt8](/zh/reference/data-types/index)) — 标志位，表示 `enabled_role` 是否为默认角色。

{/*AUTOGENERATED_END*/}
