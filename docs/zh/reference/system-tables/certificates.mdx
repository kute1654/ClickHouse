<div id="description">
  ## 描述
</div>

包含可用证书及其来源的相关信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/zh/reference/data-types/index)) — 证书版本。v1 的值为 0，v2 为 1，v3 为 2。
* `serial_number` ([Nullable(String)](/zh/reference/data-types/index)) — 由签发方分配的证书序列号。
* `signature_algo` ([Nullable(String)](/zh/reference/data-types/index)) — 签名算法——签发方用于签署此证书的算法。
* `issuer` ([Nullable(String)](/zh/reference/data-types/index)) — 签发方——签发此证书的证书颁发机构的唯一标识符。
* `not_before` ([Nullable(String)](/zh/reference/data-types/index)) — 此证书有效时间窗口的开始时间。
* `not_after` ([Nullable(String)](/zh/reference/data-types/index)) — 此证书有效时间窗口的结束时间。
* `subject` ([Nullable(String)](/zh/reference/data-types/index)) — 使用者——用于标识公钥所有者。
* `pkey_algo` ([Nullable(String)](/zh/reference/data-types/index)) — 公钥算法，定义该公钥可搭配使用的算法。
* `path` ([String](/zh/reference/data-types/index)) — 指向包含此证书的文件或目录的路径。
* `default` ([UInt8](/zh/reference/data-types/index)) — 证书位于默认证书位置。
* `protocol` ([String](/zh/reference/data-types/index)) — 如果证书来自按协议划分的 TLS 配置，则为协议名称；否则为空。

{/*AUTOGENERATED_END*/}
