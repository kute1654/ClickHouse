<div id="description">
  ## 描述
</div>

包含 Kafka 消费者的相关信息。
适用于 [Kafka 表引擎](/zh/reference/engines/table-engines/integrations/kafka) (ClickHouse 原生集成)

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 带有表引擎的表所在的数据库。
* `table` ([String](/zh/reference/data-types/index)) — 带有表引擎的表名称。
* `consumer_id` ([String](/zh/reference/data-types/index)) — Kafka 消费者标识符。请注意，一个表可以有多个消费者。由 `kafka_num_consumers` 参数指定。
* `assignments.topic` ([Array(String)](/zh/reference/data-types/index)) — Kafka topic。
* `assignments.partition_id` ([Array(Int32)](/zh/reference/data-types/index)) — Kafka 分区 ID。请注意，一个分区只能分配给一个消费者。
* `assignments.current_offset` ([Array(Int64)](/zh/reference/data-types/index)) — 当前 offset。
* `assignments.intent_size` ([Array(Nullable(Int64))](/zh/reference/data-types/index)) — 在新的 StorageKafka 中，已推送但尚未提交的消息数。
* `exceptions.time` ([Array(DateTime)](/zh/reference/data-types/index)) — 最近 10 个异常产生时的时间戳。
* `exceptions.text` ([Array(String)](/zh/reference/data-types/index)) — 最近 10 个异常的文本。
* `last_poll_time` ([DateTime](/zh/reference/data-types/index)) — 最近一次 poll 的时间戳。
* `num_messages_read` ([UInt64](/zh/reference/data-types/index)) — 该消费者读取的消息数。
* `last_commit_time` ([DateTime](/zh/reference/data-types/index)) — 最近一次 poll 的时间戳。
* `num_commits` ([UInt64](/zh/reference/data-types/index)) — 该消费者的提交总次数。
* `last_rebalance_time` ([DateTime](/zh/reference/data-types/index)) — 最近一次 Kafka rebalance 的时间戳。
* `num_rebalance_revocations` ([UInt64](/zh/reference/data-types/index)) — 该消费者的分区分配被撤销的次数。
* `num_rebalance_assignments` ([UInt64](/zh/reference/data-types/index)) — 该消费者被分配到 Kafka 集群的次数。
* `is_currently_used` ([UInt8](/zh/reference/data-types/index)) — 指示该消费者当前是否正在使用的标志。
* `last_used` ([DateTime64(6)](/zh/reference/data-types/index)) — 该消费者上次被使用的时间。
* `rdkafka_stat` ([String](/zh/reference/data-types/index)) — 库内部统计信息。将 statistics&#95;interval&#95;ms 设为 0 可禁用，默认值为 3000 (每三秒一次) 。
* `dependencies` ([Array(Array(String))](/zh/reference/data-types/index)) — 传递性数据库依赖项。
* `missing_dependencies` ([Array(Array(String))](/zh/reference/data-types/index)) — 缺失的传递性数据库依赖项。

{/*AUTOGENERATED_END*/}
