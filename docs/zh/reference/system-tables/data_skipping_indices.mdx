<div id="description">
  ## 说明
</div>

包含所有表中现有的数据跳过索引信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 数据库名。
* `table` ([String](/zh/reference/data-types/index)) — 表名。
* `name` ([String](/zh/reference/data-types/index)) — 索引名。
* `type` ([String](/zh/reference/data-types/index)) — 索引类型。
* `type_full` ([String](/zh/reference/data-types/index)) — CREATE 语句中的索引类型表达式。
* `expr` ([String](/zh/reference/data-types/index)) — 用于计算索引的表达式。
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/zh/reference/data-types/index)) — 索引是否为隐式创建 (通过 add&#95;minmax&#95;index&#95;for&#95;numeric&#95;columns 或类似方式) 。
* `granularity` ([UInt64](/zh/reference/data-types/index)) — 块中包含的粒度数量。
* `data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 压缩后数据的大小 (以字节为单位) 。
* `data_uncompressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 解压后数据的大小 (以字节为单位) 。
* `marks_bytes` ([UInt64](/zh/reference/data-types/index)) — 标记的大小 (以字节为单位) 。

{/*AUTOGENERATED_END*/}
