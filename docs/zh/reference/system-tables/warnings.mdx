<div id="description">
  ## 描述
</div>

包含有关服务器配置的警告信息，clickhouse-client 在连接到服务器后会立即显示这些信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `message` ([String](/zh/reference/data-types/index)) — 由 ClickHouse server 发出的警告消息。
* `message_format_string` ([LowCardinality(String)](/zh/reference/data-types/index)) — 用于格式化该消息的格式字符串。

{/*AUTOGENERATED_END*/}
