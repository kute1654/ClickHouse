<div id="description">
  ## 描述
</div>

包含贡献者的信息。查询执行时，其顺序是随机的。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — git log 中的贡献者 (作者) 姓名。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT * FROM system.contributors LIMIT 10
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
│ Max Vetrov       │
│ LiuYangkuan      │
│ svladykin        │
│ zamulla          │
│ Šimon Podlipský  │
│ BayoNet          │
│ Ilya Khomutov    │
│ Amy Krishnevsky  │
│ Loud_Scream      │
└──────────────────┘
```

若要在表中找到你自己的记录，请使用以下查询：

```sql
SELECT * FROM system.contributors WHERE name = 'Olga Khvostikova'
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
└──────────────────┘
```
