<div id="description">
  ## 描述
</div>

列出服务器支持的数据库引擎。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 数据库引擎的名称。
* `description` ([String](/zh/reference/data-types/index)) — 对数据库引擎功能的概述。
* `syntax` ([String](/zh/reference/data-types/index)) — 在 `CREATE DATABASE` 查询的 `ENGINE` 子句中指定数据库引擎的方式。
* `examples` ([String](/zh/reference/data-types/index)) — 使用示例。
* `introduced_in` ([String](/zh/reference/data-types/index)) — 首次引入该数据库引擎的 ClickHouse 版本，格式为 major.minor。
* `related` ([Array(String)](/zh/reference/data-types/index)) — 相关数据库引擎的名称。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```

```text title="Response"
┌─name─────┐
│ Atomic   │
│ Ordinary │
```
