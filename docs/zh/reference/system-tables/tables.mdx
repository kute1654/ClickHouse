<div id="description">
  ## 描述
</div>

包含服务器所知晓的每个表的元数据。

[Detached](/zh/reference/statements/detach) 表不会显示在 `system.tables` 中。

[临时表](/zh/reference/statements/create/table#temporary-tables) 仅在创建它们的那些会话的 `system.tables` 中可见。它们的 `database` field 为空，且 `is_temporary` 标志已启用。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 该表所在数据库的名称。
* `name` ([String](/zh/reference/data-types/index)) — 表名。
* `uuid` ([UUID](/zh/reference/data-types/index)) — 表的 UUID (Atomic 数据库) 。
* `engine` ([String](/zh/reference/data-types/index)) — 表引擎名称 (不含参数) 。
* `is_temporary` ([UInt8](/zh/reference/data-types/index)) — 用于指示该表是否为临时表的标志。
* `data_paths` ([Array(String)](/zh/reference/data-types/index)) — 文件系统中表数据的路径。
* `metadata_path` ([String](/zh/reference/data-types/index)) — 文件系统中表元数据的路径。
* `metadata_modification_time` ([DateTime](/zh/reference/data-types/index)) — 表元数据的最近修改时间。
* `metadata_version` ([Int32](/zh/reference/data-types/index)) — ReplicatedMergeTree 表的元数据版本；非 ReplicatedMergeTree 表为 0。
* `dependencies_database` ([Array(String)](/zh/reference/data-types/index)) — 数据库依赖项。
* `dependencies_table` ([Array(String)](/zh/reference/data-types/index)) — 表依赖项 (当前表的 materialized view) 。
* `create_table_query` ([String](/zh/reference/data-types/index)) — 用于创建该表的查询。
* `engine_full` ([String](/zh/reference/data-types/index)) — 表引擎参数。
* `as_select` ([String](/zh/reference/data-types/index)) — 视图的 SELECT 查询。
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](/zh/reference/data-types/index)) — parameterized view 的参数。
* `partition_key` ([String](/zh/reference/data-types/index)) — 表中指定的分区键表达式。
* `sorting_key` ([String](/zh/reference/data-types/index)) — 表中指定的排序键表达式。
* `primary_key` ([String](/zh/reference/data-types/index)) — 表中指定的主键表达式。
* `sampling_key` ([String](/zh/reference/data-types/index)) — 表中指定的采样键表达式。
* `unique_key` ([String](/zh/reference/data-types/index)) — 表中指定的唯一键表达式 (UNIQUE KEY 子句) 。
* `storage_policy` ([String](/zh/reference/data-types/index)) — 存储策略。适用于使用 MergeTree 和 Distributed 引擎的表。
* `total_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 总行数；如果可以快速确定表中的精确行数，则返回该值，否则返回 NULL (包括底层 Buffer table) 。
* `total_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 总字节数；如果可以快速确定该表在存储上的精确字节数，则返回该值，否则返回 NULL (不包括任何底层存储) 。如果表将数据存储在磁盘上，则返回已使用的磁盘空间 (即 compressed) 。如果表将数据存储在内存中，则返回内存中已使用字节数的估算值。
* `total_bytes_uncompressed` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 未压缩字节总数；如果可以根据该表在存储上的 part checksums 快速确定精确字节数，则返回该值，否则返回 NULL (不考虑底层存储 (如有) ) 。
* `parts` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 该表中的 parts 总数。
* `active_parts` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 此表中活跃 parts 的数量。
* `total_marks` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 此表中所有 parts 的标记总数。
* `active_on_fly_data_mutations` ([UInt64](/zh/reference/data-types/index)) — 可即时应用的活跃数据变更 (UPDATE 和 DELETE) 总数。
* `active_on_fly_alter_mutations` ([UInt64](/zh/reference/data-types/index)) — 可即时应用的活跃 ALTER 变更 (MODIFY COLUMN) 总数。
* `active_on_fly_metadata_mutations` ([UInt64](/zh/reference/data-types/index)) — 可即时应用的活跃元数据变更 (RENAME) 总数。
* `columns_descriptions_cache_size` ([UInt64](/zh/reference/data-types/index)) — *MergeTree 表的列描述缓存大小
* `lifetime_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 自服务器启动以来 INSERT 的总行数 (仅适用于 Buffer 表) 。
* `lifetime_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 自服务器启动以来 INSERT 的总字节数 (仅适用于 Buffer 表) 。
* `comment` ([String](/zh/reference/data-types/index)) — 该表的注释。
* `has_own_data` ([UInt8](/zh/reference/data-types/index)) — 指示该表自身是否在磁盘上存储数据，还是仅访问其他源的标志。
* `loading_dependencies_database` ([Array(String)](/zh/reference/data-types/index)) — 数据库加载依赖项 (应在当前对象之前加载的对象列表) 。
* `loading_dependencies_table` ([Array(String)](/zh/reference/data-types/index)) — 表加载依赖项 (应在当前对象之前加载的对象列表) 。
* `loading_dependent_database` ([Array(String)](/zh/reference/data-types/index)) — 依赖于加载的数据库。
* `loading_dependent_table` ([Array(String)](/zh/reference/data-types/index)) — 依赖于加载的表。
* `target_database` ([String](/zh/reference/data-types/index)) — 对于 materialized view，这是该视图写入的目标端表所在的数据库 (`TO` 目标，或隐式的 `.inner.*` 表) 。其他引擎则为空。
* `target_table` ([String](/zh/reference/data-types/index)) — 对于 materialized view，这是该视图写入的目标端表名称 (`TO` 目标，或隐式的 `.inner.*` 表) 。其他引擎则为空。
* `definer` ([String](/zh/reference/data-types/index)) — 用于该表的 SQL security definer 名称。

{/*AUTOGENERATED_END*/}
