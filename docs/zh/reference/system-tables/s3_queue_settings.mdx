<div id="description">
  ## 说明
</div>

包含有关 [S3Queue](/zh/reference/engines/table-engines/integrations/s3queue) 表设置的信息。自 server 版本 `24.10` 起可用。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 使用 S3Queue 引擎的表所在的数据库。
* `table` ([String](/zh/reference/data-types/index)) — 使用 S3Queue 引擎的表名。
* `name` ([String](/zh/reference/data-types/index)) — 设置名称。
* `value` ([String](/zh/reference/data-types/index)) — 设置值。
* `type` ([String](/zh/reference/data-types/index)) — 设置类型 (实现相关的字符串值) 。
* `changed` ([UInt8](/zh/reference/data-types/index)) — 如果该设置已在配置中显式定义或被显式修改，则为 1。
* `description` ([String](/zh/reference/data-types/index)) — 设置说明。
* `alterable` ([UInt8](/zh/reference/data-types/index)) — 显示当前用户是否可以通过 ALTER TABLE MODIFY SETTING 修改该设置：0 — 当前用户不能修改该设置，1 — 当前用户可以修改该设置。

{/*AUTOGENERATED_END*/}
