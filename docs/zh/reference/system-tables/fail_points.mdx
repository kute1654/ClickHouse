<div id="description">
  ## 描述
</div>

包含服务器中已注册的所有可用 failpoint 及其类型，以及当前是否启用。

可以在运行时使用 `SYSTEM ENABLE FAILPOINT` 和 `SYSTEM DISABLE FAILPOINT` 语句来启用或禁用 failpoint。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — failpoint 的名称。
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](/zh/reference/data-types/index)) — failpoint 的类型：&#39;once&#39; 触发一次后自动禁用，&#39;regular&#39; 每次都会触发，&#39;pauseable&#95;once&#39; 会阻塞执行一次，&#39;pauseable&#39; 会在每次执行时阻塞，直到恢复。
* `enabled` ([UInt8](/zh/reference/data-types/index)) — failpoint 当前是否启用：启用为 (1)，禁用为 (0)。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql title="Query"
SYSTEM ENABLE FAILPOINT replicated_merge_tree_insert_retry_pause;
SELECT * FROM system.fail_points WHERE enabled = 1
```

