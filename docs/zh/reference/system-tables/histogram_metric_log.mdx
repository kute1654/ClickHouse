<div id="description">
  ## 描述
</div>

`system.histogram_metrics` 的历史记录。每隔 `collect_interval_milliseconds` 采集一次快照，并写入磁盘。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 执行查询的服务器主机名。
* `event_date` ([Date](/zh/reference/data-types/date)) — 事件日期。
* `event_time` ([DateTime](/zh/reference/data-types/datetime)) — 事件时间。
* `event_time_microseconds` ([DateTime64(6)](/zh/reference/data-types/datetime64)) — 微秒精度的事件时间。
* `metric` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 指标名称。
* `labels` ([Map(LowCardinality(String), LowCardinality(String))](/zh/reference/data-types/map)) — 指标标记。
* `histogram` ([Map(Float64, UInt64)](/zh/reference/data-types/map)) — 累积直方图：将桶的上界映射到小于或等于该上界的观测值数量；包含 +inf 作为最后一个桶。
* `count` ([UInt64](/zh/reference/data-types/int-uint)) — 观测值总数，等于 histogram[+inf]。
* `sum` ([Float64](/zh/reference/data-types/float)) — 所有观测值之和。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT event_time, metric, labels, histogram
FROM system.histogram_metric_log
WHERE metric = 'keeper_response_time_ms'
ORDER BY event_time DESC
LIMIT 1
FORMAT Vertical;
```
