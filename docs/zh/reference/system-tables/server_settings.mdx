<div id="description">
  ## 说明
</div>

包含在 `config.xml` 中指定的服务器全局设置相关信息。
目前，该表仅显示 `config.xml` 第一层的设置，尚不支持嵌套配置 (例如 [logger](/zh/reference/settings/server-settings/settings#logger)) 。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 服务器设置的名称。
* `value` ([String](/zh/reference/data-types/index)) — 服务器设置的值。
* `default` ([String](/zh/reference/data-types/index)) — 服务器设置的默认值。
* `changed` ([UInt8](/zh/reference/data-types/index)) — 显示该设置是否在 config.xml 中指定。
* `description` ([String](/zh/reference/data-types/index)) — 服务器设置的简短描述。
* `type` ([String](/zh/reference/data-types/index)) — 服务器设置值的类型。
* `changeable_without_restart` ([Enum8(&#39;No&#39; = 0, &#39;IncreaseOnly&#39; = 1, &#39;DecreaseOnly&#39; = 2, &#39;Yes&#39; = 3)](/zh/reference/data-types/index)) — 显示该设置是否可在运行时更改。
* `is_obsolete` ([UInt8](/zh/reference/data-types/index)) — 显示该设置是否已废弃。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

```sql
SELECT *
FROM system.server_settings
WHERE name LIKE '%thread_pool%'
```
