<div id="description">
  ## 描述
</div>

包含所有可用聚合函数组合器的列表；这些组合器可应用于聚合函数，以改变其工作方式。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 组合器的名称。
* `is_internal` ([UInt8](/zh/reference/data-types/index)) — 该组合器是否仅供内部使用。
* `description` ([String](/zh/reference/data-types/index)) — 对该组合器作用的概述。
* `syntax` ([String](/zh/reference/data-types/index)) — 该组合器如何应用到聚合函数名称上。
* `examples` ([String](/zh/reference/data-types/index)) — 使用示例。
* `introduced_in` ([String](/zh/reference/data-types/index)) — 首次引入该组合器的 ClickHouse 版本，格式为 major.minor。
* `related` ([Array(String)](/zh/reference/data-types/index)) — 相关组合器的名称。

{/*AUTOGENERATED_END*/}
