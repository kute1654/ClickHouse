<div id="description">
  ## 描述
</div>

包含用于采用 [*GraphiteMergeTree](/zh/reference/engines/table-engines/mergetree-family/graphitemergetree) 引擎的表的参数 [graphite&#95;rollup](/zh/reference/settings/server-settings/settings#graphite) 的相关信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/zh/reference/data-types/index)) — graphite&#95;rollup 参数名称。
* `rule_type` ([String](/zh/reference/data-types/index)) — 规则类型。可能的值：RuleTypeAll = 0 - 默认，带正则表达式，与旧方案兼容；RuleTypePlain = 1 - 纯指标，带正则表达式，与旧方案兼容；RuleTypeTagged = 2 - 带标签的指标，带正则表达式，与旧方案兼容；RuleTypeTagList = 3 - 带标签的指标，带正则表达式 (由类似 &#39;retention=10min ; env=(staging|prod)&#39; 的字符串转换为 RuleTypeTagged)
* `regexp` ([String](/zh/reference/data-types/index)) — 指标名称的模式。
* `function` ([String](/zh/reference/data-types/index)) — 聚合函数的名称。
* `age` ([UInt64](/zh/reference/data-types/index)) — 数据的最小时间，以秒为单位。
* `precision` ([UInt64](/zh/reference/data-types/index)) — 以秒为单位定义数据时间的精度。
* `priority` ([UInt16](/zh/reference/data-types/index)) — 模式优先级。
* `is_default` ([UInt8](/zh/reference/data-types/index)) — 该模式是否为默认模式。
* `Tables.database` ([Array(String)](/zh/reference/data-types/index)) — 使用 `config_name` 参数的数据库表名称数组。
* `Tables.table` ([Array(String)](/zh/reference/data-types/index)) — 使用 `config_name` 参数的表名称数组。

{/*AUTOGENERATED_END*/}
