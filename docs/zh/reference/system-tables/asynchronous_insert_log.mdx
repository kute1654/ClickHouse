<div id="description">
  ## 说明
</div>

包含有关异步插入的信息。每个条目都表示一个已缓冲到异步插入查询中的插入查询。

要启用日志记录，请在 [asynchronous&#95;insert&#95;log](/zh/reference/settings/server-settings/settings#asynchronous_insert_log) 部分中配置相关参数。

数据的刷写周期通过 [asynchronous&#95;insert&#95;log](/zh/reference/settings/server-settings/settings#asynchronous_insert_log) 服务器设置部分中的 `flush_interval_milliseconds` 参数来设置。要强制刷写，请使用 [SYSTEM FLUSH LOGS](/zh/reference/statements/system#flush-logs) 查询。

ClickHouse 不会自动从表中删除数据。更多详情，请参阅 [简介](/zh/reference/system-tables/overview#system-tables-introduction)。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 执行该查询的服务器主机名。
* `event_date` ([Date](/zh/reference/data-types/date)) — 发生异步插入的日期。
* `event_time` ([DateTime](/zh/reference/data-types/datetime)) — 异步插入执行完成的日期和时间。
* `event_time_microseconds` ([DateTime64(6)](/zh/reference/data-types/datetime64)) — 异步插入执行完成的日期和时间，精度为微秒。
* `query` ([String](/zh/reference/data-types/string)) — 查询字符串。
* `database` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 该表所在数据库的名称。
* `table` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 表名。
* `format` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 格式名称。
* `query_id` ([String](/zh/reference/data-types/string)) — 初始查询的 ID。
* `bytes` ([UInt64](/zh/reference/data-types/int-uint)) — 已插入的字节数。
* `rows` ([UInt64](/zh/reference/data-types/int-uint)) — 已插入的行数。
* `exception` ([String](/zh/reference/data-types/string)) — 异常消息。
* `status` ([Enum8(&#39;Ok&#39; = 0, &#39;ParsingError&#39; = 1, &#39;FlushError&#39; = 2)](/zh/reference/data-types/enum)) — 插入状态。取值：&#39;Ok&#39; = 0 — 插入成功，&#39;ParsingError&#39; = 1 — 解析数据时发生异常，&#39;FlushError&#39; = 2 — 刷写数据时发生异常。
* `data_kind` ([Enum8(&#39;Parsed&#39; = 0, &#39;Preprocessed&#39; = 1)](/zh/reference/data-types/enum)) — 数据状态。取值：&#39;Parsed&#39; 和 &#39;Preprocessed&#39;。
* `flush_time` ([DateTime](/zh/reference/data-types/datetime)) — 发生刷写的日期和时间。
* `flush_time_microseconds` ([DateTime64(6)](/zh/reference/data-types/datetime64)) — 发生刷写的日期和时间，精度为微秒。
* `flush_query_id` ([String](/zh/reference/data-types/string)) — 刷写查询的 ID。
* `timeout_milliseconds` ([UInt64](/zh/reference/data-types/int-uint)) — 为此条目计算的自适应超时时间 (毫秒) 。

{/*AUTOGENERATED_END*/}
