<div id="description">
  ## 描述
</div>

此表包含 MergeTree 家族表的投影 parts 中各列的信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `partition` ([String](/zh/reference/data-types/index)) — 分区名称。
* `name` ([String](/zh/reference/data-types/index)) — 数据分区片段的名称。
* `part_type` ([String](/zh/reference/data-types/index)) — 数据分区片段的存储格式。
* `parent_name` ([String](/zh/reference/data-types/index)) — 源 (父) 数据分区片段的名称。
* `parent_uuid` ([UUID](/zh/reference/data-types/index)) — 源 (父) 数据分区片段的 UUID。
* `parent_part_type` ([String](/zh/reference/data-types/index)) — 源 (父) 数据分区片段的存储格式。
* `active` ([UInt8](/zh/reference/data-types/index)) — 指示数据分区片段是否处于活动状态的标志
* `marks` ([UInt64](/zh/reference/data-types/index)) — 标记数量。
* `rows` ([UInt64](/zh/reference/data-types/index)) — 行数。
* `bytes_on_disk` ([UInt64](/zh/reference/data-types/index)) — 所有数据分区片段文件的总大小 (以字节为单位) 。
* `data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 数据分区片段中压缩数据的总大小。不包括所有辅助文件 (例如标记文件) 。
* `data_uncompressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 数据分区片段中未压缩数据的总大小。不包括所有辅助文件 (例如标记文件) 。
* `marks_bytes` ([UInt64](/zh/reference/data-types/index)) — 标记文件的大小。
* `parent_marks` ([UInt64](/zh/reference/data-types/index)) — 源 (父) 数据分区片段中的标记数量。
* `parent_rows` ([UInt64](/zh/reference/data-types/index)) — 源 (父) 数据分区片段中的行数。
* `parent_bytes_on_disk` ([UInt64](/zh/reference/data-types/index)) — 所有源 (父) 数据分区片段文件的总大小 (以字节为单位) 。
* `parent_data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 源 (父) 数据分区片段中压缩数据的总大小。
* `parent_data_uncompressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 源 (父) 数据分区片段中未压缩数据的总大小。
* `parent_marks_bytes` ([UInt64](/zh/reference/data-types/index)) — 源 (父) 数据分区片段中标记文件的大小。
* `modification_time` ([日期时间](/zh/reference/data-types/index)) — 包含该数据分区片段的目录被修改的时间。这通常对应于数据分区片段的创建时间。
* `remove_time` ([日期时间](/zh/reference/data-types/index)) — 数据分区片段变为非活动状态的时间。
* `refcount` ([UInt32](/zh/reference/data-types/index)) — 数据分区片段被使用的位置数量。值大于 2 表示该数据分区片段被用于查询或合并。
* `min_date` ([Date](/zh/reference/data-types/index)) — 如果分区键中包含 Date 列，则该列的最小值。
* `max_date` ([Date](/zh/reference/data-types/index)) — 如果分区键中包含 Date 列，则该列的最大值。
* `min_time` ([日期时间](/zh/reference/data-types/index)) — 如果分区键中包含 DateTime 列，则该列的最小值。
* `max_time` ([DateTime](/zh/reference/data-types/index)) — 如果分区键中包含 DateTime 列，则为该列的最大值。
* `partition_id` ([String](/zh/reference/data-types/index)) — 分区的 ID。
* `min_block_number` ([Int64](/zh/reference/data-types/index)) — 合并后构成当前数据分区片段的最小数据分区片段编号。
* `max_block_number` ([Int64](/zh/reference/data-types/index)) — 合并后构成当前数据分区片段的最大数据分区片段编号。
* `level` ([UInt32](/zh/reference/data-types/index)) — 合并树的深度。零表示当前数据分区片段是通过插入操作创建的，而不是通过合并其他数据分区片段创建的。
* `data_version` ([UInt64](/zh/reference/data-types/index)) — 用于确定哪些变更应应用到数据分区片段的编号 (版本高于 data&#95;version 的变更) 。
* `primary_key_bytes_in_memory` ([UInt64](/zh/reference/data-types/index)) — 主键值使用的内存量 (以字节为单位) 。
* `primary_key_bytes_in_memory_allocated` ([UInt64](/zh/reference/data-types/index)) — 为主键值预留的内存量 (以字节为单位) 。
* `database` ([String](/zh/reference/data-types/index)) — 数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 表名称。
* `engine` ([String](/zh/reference/data-types/index)) — 不带参数的表引擎名称。
* `disk_name` ([String](/zh/reference/data-types/index)) — 存储该数据分区片段的磁盘名称。
* `path` ([String](/zh/reference/data-types/index)) — 数据分区片段文件所在文件夹的绝对路径。
* `column` ([String](/zh/reference/data-types/index)) — 列名称。
* `type` ([String](/zh/reference/data-types/index)) — 列类型。
* `column_position` ([UInt64](/zh/reference/data-types/index)) — 列在表中的序号位置，从 1 开始。
* `default_kind` ([String](/zh/reference/data-types/index)) — 默认值的表达式类型 (DEFAULT、MATERIALIZED、ALIAS) ；如果未定义，则为空字符串。
* `default_expression` ([String](/zh/reference/data-types/index)) — 默认值的表达式；如果未定义，则为空字符串。
* `column_bytes_on_disk` ([UInt64](/zh/reference/data-types/index)) — 列的总大小 (以字节为单位) 。
* `column_data_compressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 列中压缩数据的总大小 (以字节为单位) 。
* `column_data_uncompressed_bytes` ([UInt64](/zh/reference/data-types/index)) — 列中解压后数据的总大小 (以字节为单位) 。
* `column_marks_bytes` ([UInt64](/zh/reference/data-types/index)) — 列标记占用的大小 (以字节为单位) 。
* `column_modification_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 该列最后一次修改的时间。
