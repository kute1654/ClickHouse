<div id="description">
  ## 描述
</div>

包含队列中待处理异步插入的信息。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/zh/reference/data-types/index)) — 查询文本。
* `database` ([String](/zh/reference/data-types/index)) — 数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 表名称。
* `format` ([String](/zh/reference/data-types/index)) — 格式名称。
* `first_update` ([DateTime64(6)](/zh/reference/data-types/index)) — 首次写入时间，精度为微秒。
* `total_bytes` ([UInt64](/zh/reference/data-types/index)) — 队列中等待处理的总字节数。
* `entries.query_id` ([Array(String)](/zh/reference/data-types/index)) — 队列中等待处理的插入的查询 ID 数组。
* `entries.bytes` ([Array(UInt64)](/zh/reference/data-types/index)) — 队列中每个等待处理的插入查询的字节数数组。

{/*AUTOGENERATED_END*/}

  ## 示例
</div>

```sql title="Query"
SELECT * FROM system.asynchronous_inserts LIMIT 1 \G;

```
