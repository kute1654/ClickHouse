<div id="description">
  ## 描述
</div>

包含表 `system.errors` 中错误值的历史记录，并定期写入磁盘。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 执行该查询的服务器主机名。
* `event_date` ([Date](/zh/reference/data-types/date)) — 事件日期。
* `event_time` ([DateTime](/zh/reference/data-types/datetime)) — 事件时间。
* `code` ([Int32](/zh/reference/data-types/int-uint)) — 错误代码。
* `error` ([LowCardinality(String)](/zh/reference/data-types/lowcardinality)) — 错误名称。
* `value` ([UInt64](/zh/reference/data-types/int-uint)) — 在该时间间隔内发生的错误次数。
* `remote` ([UInt8](/zh/reference/data-types/int-uint)) — 远程异常 (即在某次分布式查询期间接收到的异常) 。
* `last_error_time` ([DateTime](/zh/reference/data-types/datetime)) — 最近一次错误发生的时间。
* `last_error_message` ([String](/zh/reference/data-types/string)) — 最近一次错误的消息。
* `last_error_query_id` ([String](/zh/reference/data-types/string)) — 导致最近一次错误的查询 ID (如果可用) 。
* `last_error_trace` ([Array(UInt64)](/zh/reference/data-types/array)) — 栈跟踪，表示被调用方法所存储的物理地址列表。

{/*AUTOGENERATED_END*/}
