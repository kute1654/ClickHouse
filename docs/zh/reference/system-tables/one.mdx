<div id="description">
  ## 描述
</div>

此表仅包含一行数据，且只有一个值为 0 的 `dummy` UInt8 列。

如果 `SELECT` 查询未指定 `FROM` 子句，则使用此表。

这类似于其他 DBMSs 中的 `DUAL` 表。
