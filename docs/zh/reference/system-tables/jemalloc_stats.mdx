  ## 说明
</div>

以单行单列的形式返回 jemalloc 统计信息。等同于 SYSTEM JEMALLOC STATS 命令。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `stats` ([String](/zh/reference/data-types/index)) — 由 malloc&#95;stats&#95;print 输出的 JEMalloc 统计信息。
