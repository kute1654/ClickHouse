<div id="description">
  ## 描述
</div>

包含[用户自定义函数 (UDF) ](/zh/reference/functions/regular-functions/udf)的加载状态、错误信息和配置元数据。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — UDF 名称。
* `load_status` ([Enum8(&#39;Success&#39; = 0, &#39;Failed&#39; = 1)](/zh/reference/data-types/index)) — 加载状态。可能的值：
  * **Success** — UDF 已加载并可使用
  * **Failed** — UDF 加载失败 (详见字段 &#39;loading&#95;error&#95;message&#39;) 。
* `loading_error_message` ([String](/zh/reference/data-types/index)) — 加载失败时的详细错误信息。若加载成功则为空。
* `last_successful_update_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 最近一次成功更新的时间戳。若从未成功则为 NULL。
* `loading_duration_ms` ([UInt64](/zh/reference/data-types/index)) — 加载 UDF 所耗费的时间，单位为毫秒。
* `type` ([Enum8(&#39;executable&#39; = 0, &#39;executable&#95;pool&#39; = 1)](/zh/reference/data-types/index)) — UDF 类型：&#39;executable&#39; (单进程) 或 &#39;executable&#95;pool&#39; (进程池) 。
* `command` ([String](/zh/reference/data-types/index)) — 此 UDF 要执行的脚本或命令。
* `format` ([String](/zh/reference/data-types/index)) — I/O 数据格式 (例如 &#39;TabSeparated&#39;、&#39;JSONEachRow&#39;) 。
* `return_type` ([String](/zh/reference/data-types/index)) — 函数返回类型 (例如 &#39;String&#39;、&#39;UInt64&#39;) 。
* `return_name` ([String](/zh/reference/data-types/index)) — 可选的返回值标识符。若未配置则为空。
* `argument_types` ([Array(String)](/zh/reference/data-types/index)) — 参数类型的 Array (例如 [&#39;String&#39;, &#39;UInt64&#39;]) 。
* `argument_names` ([Array(String)](/zh/reference/data-types/index)) — 参数名称的 Array。未命名参数对应空字符串。
* `max_command_execution_time` ([UInt64](/zh/reference/data-types/index)) — 处理一个数据块的最长时间 (秒) 。仅适用于 &#39;executable&#95;pool&#39; 类型。
* `command_termination_timeout` ([UInt64](/zh/reference/data-types/index)) — 向命令进程发送 SIGTERM 前等待的秒数。
* `command_read_timeout` ([UInt64](/zh/reference/data-types/index)) — 从命令 stdout 读取的超时时间，单位为毫秒。
* `command_write_timeout` ([UInt64](/zh/reference/data-types/index)) — 向命令 stdin 写入的超时时间，单位为毫秒。
* `pool_size` ([UInt64](/zh/reference/data-types/index)) — 命令进程实例数量。仅适用于 &#39;executable&#95;pool&#39; 类型。
* `send_chunk_header` ([UInt8](/zh/reference/data-types/index)) — 是否在每个数据 chunk 前发送行数 (布尔值) 。
* `execute_direct` ([UInt8](/zh/reference/data-types/index)) — 是否直接执行命令 (1) ，还是通过 /bin/bash 执行 (0) 。
* `lifetime` ([UInt64](/zh/reference/data-types/index)) — 重新加载间隔，单位为秒。0 表示禁用重新加载。
* `deterministic` ([UInt8](/zh/reference/data-types/index)) — 对于相同参数，函数是否返回相同结果 (布尔值) 。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 示例
</div>

查看所有 UDF 及其加载状态：

```sql
SELECT
    name,
    load_status,
    type,
    command,
    return_type,
    argument_types
FROM system.user_defined_functions
FORMAT Vertical;
```

```response
Row 1:
──────
name:           my_sum_udf
load_status:    Success
type:           executable
command:        /var/lib/clickhouse/user_scripts/sum.py
return_type:    UInt64
argument_types: ['UInt64','UInt64']
```

查找失败的 UDF：

```sql
SELECT
    name,
    loading_error_message
FROM system.user_defined_functions
WHERE load_status = 'Failed';
```
