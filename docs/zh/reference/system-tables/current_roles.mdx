<div id="description">
  ## 描述
</div>

包含当前用户的已启用角色。`SET ROLE` 会修改此表中的内容。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/zh/reference/data-types/index)) — 角色名称。
* `with_admin_option` ([UInt8](/zh/reference/data-types/index)) — 如果该角色具有 ADMIN OPTION 权限，则为 1。
* `is_default` ([UInt8](/zh/reference/data-types/index)) — 如果该角色被设置为默认角色，则为 1。

{/*AUTOGENERATED_END*/}
