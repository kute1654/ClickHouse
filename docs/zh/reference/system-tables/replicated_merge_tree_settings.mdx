<div id="description">
  ## 说明
</div>

包含所有 ReplicatedMergeTree 引擎专有设置及其当前值、默认值和相关说明。你可以在 CREATE 查询的 SETTINGS 部分修改其中任意一项。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 设置名称。
* `value` ([String](/zh/reference/data-types/index)) — 设置值。
* `default` ([String](/zh/reference/data-types/index)) — 设置默认值。
* `changed` ([UInt8](/zh/reference/data-types/index)) — 如果该设置已在配置中显式定义或被显式修改，则为 1。
* `description` ([String](/zh/reference/data-types/index)) — 设置说明。
* `min` ([Nullable(String)](/zh/reference/data-types/index)) — 设置的最小值 (如果已通过约束设置) 。如果该设置没有最小值，则为 NULL。
* `max` ([Nullable(String)](/zh/reference/data-types/index)) — 设置的最大值 (如果已通过约束设置) 。如果该设置没有最大值，则为 NULL。
* `disallowed_values` ([Array(String)](/zh/reference/data-types/index)) — 不允许的值列表
* `readonly` ([UInt8](/zh/reference/data-types/index)) — 表示当前用户是否可以更改该设置：0 — 当前用户可以更改该设置，1 — 当前用户不能更改该设置。
* `type` ([String](/zh/reference/data-types/index)) — 设置类型 (实现相关的字符串值) 。
* `is_obsolete` ([UInt8](/zh/reference/data-types/index)) — 表示该设置是否已废弃。
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/zh/reference/data-types/index)) —
  此功能的支持层级。ClickHouse 功能按层级划分，具体取决于其当前的
  开发状态以及使用时可预期的行为：

- PRODUCTION：该功能属于稳定版本，可安全使用，且与其他 PRODUCTION 功能交互时不会出现问题。
- BETA：该功能稳定且安全。与其他功能组合使用时的结果尚不明确，且不保证正确性。欢迎测试并提交反馈。
- EXPERIMENTAL：该功能仍在开发中。仅适用于开发者和 ClickHouse 爱好者。该功能可能有效，也可能无效，并且可能随时被移除。
- OBSOLETE：不再受支持。要么已被移除，要么将在未来版本中移除。

{/*AUTOGENERATED_END*/}
