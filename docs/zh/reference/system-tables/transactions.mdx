<div id="description">
  ## 描述
</div>

包含事务及其状态列表。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/zh/reference/data-types/tuple)) — 事务的标识符。
* `tid_hash` ([UInt64](/zh/reference/data-types/int-uint)) — 该标识符的哈希值。
* `elapsed` ([Float64](/zh/reference/data-types/float)) — 事务处理所用的时间。
* `is_readonly` ([UInt8](/zh/reference/data-types/int-uint)) — 该标志指示事务是否执行过任何写操作。
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/zh/reference/data-types/enum)) — 事务的状态。可能的值：RUNNING、COMMITTING、COMMITTED、ROLLED&#95;BACK。

{/*AUTOGENERATED_END*/}
