<div id="description">
  ## 描述
</div>

包含 [AzureQueue](/zh/reference/engines/table-engines/integrations/azure-queue) 表的设置相关信息。
自 `24.10` server 版本起可用。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/zh/reference/data-types/index)) — 使用 AzureQueue engine 的表所在的数据库名称。
* `table` ([String](/zh/reference/data-types/index)) — 使用 AzureQueue engine 的表名称。
* `name` ([String](/zh/reference/data-types/index)) — 设置名称。
* `value` ([String](/zh/reference/data-types/index)) — 设置值。
* `type` ([String](/zh/reference/data-types/index)) — 设置类型 (实现相关的字符串值) 。
* `changed` ([UInt8](/zh/reference/data-types/index)) — 如果该设置是在配置中显式定义的，或被显式修改过，则为 1。
* `description` ([String](/zh/reference/data-types/index)) — 设置说明。
* `alterable` ([UInt8](/zh/reference/data-types/index)) — 表示当前用户是否可通过 ALTER TABLE MODIFY SETTING 修改该设置：0 — 当前用户不能修改该设置，1 — 当前用户可以修改该设置。

{/*AUTOGENERATED_END*/}
