<div id="description">
  ## 描述
</div>

列出通过 SQL 查询创建或从配置文件解析得到的所有命名集合。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 集合名称。
* `collection` ([Map(String, String)](/zh/reference/data-types/index)) — 集合内部信息。
* `source` ([String](/zh/reference/data-types/index)) — 命名集合源。
* `create_query` ([String](/zh/reference/data-types/index)) — 命名集合的创建查询。

{/*AUTOGENERATED_END*/}
