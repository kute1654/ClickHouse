  ## 描述

所有用户的配额使用情况。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/zh/reference/data-types/index)) — 配额名称。
* `quota_key` ([String](/zh/reference/data-types/index)) — 键值。
* `is_current` ([UInt8](/zh/reference/data-types/index)) — 当前用户的配额使用情况。
* `start_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 用于计算资源消耗的开始时间。
* `end_time` ([Nullable(DateTime)](/zh/reference/data-types/index)) — 用于计算资源消耗的结束时间。
* `duration` ([Nullable(UInt32)](/zh/reference/data-types/index)) — 用于计算资源消耗的时间间隔长度，以秒为单位。
* `queries` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前已执行的查询数量。
* `max_queries` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 允许执行的各类查询的最大数量。
* `query_selects` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前已执行的 SELECT 查询数量。
* `max_query_selects` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 允许执行的 SELECT 查询最大数量。
* `query_inserts` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前已执行的 INSERT 查询数量。
* `max_query_inserts` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 允许执行的 INSERT 查询最大数量。
* `errors` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前出错的查询数量。
* `max_errors` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许出错的查询最大数量。
* `result_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内所有查询结果集的总行数。
* `max_result_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询结果集总行数上限。
* `result_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内所有查询结果集的总字节数。
* `max_result_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的所有查询结果集总字节数上限。
* `read_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内执行所有查询时读取的总行数。
* `max_read_rows` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内执行所有查询时允许读取的最大行数。
* `read_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内执行所有查询时读取的总字节数。
* `max_read_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内执行所有查询时允许读取的最大字节数。
* `execution_time` ([Nullable(Float64)](/zh/reference/data-types/index)) — 当前时间段内执行查询花费的总时间 (以纳秒为单位)
* `max_execution_time` ([Nullable(Float64)](/zh/reference/data-types/index)) — 在指定时间段内允许所有查询执行的最大总时间 (以纳秒为单位)
* `written_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内执行所有查询时写入的总字节数。
* `max_written_bytes` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内执行所有查询时允许写入的最大字节数。
* `failed_sequential_authentications` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内连续身份验证失败的次数。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许的连续身份验证失败最大次数。
* `queries_per_normalized_hash` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 当前时间段内任一单个归一化查询的最大执行次数。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/zh/reference/data-types/index)) — 在指定时间段内允许任一单个归一化查询的最大执行次数。
