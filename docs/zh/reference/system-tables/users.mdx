<div id="description">
  ## 说明
</div>

包含服务器上已配置的[用户账户](/zh/concepts/features/security/access-rights#user-account-management)列表。

<div id="columns">
  ## 列
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — 用户名。
* `id` ([UUID](/zh/reference/data-types/index)) — 用户 ID。
* `storage` ([String](/zh/reference/data-types/index)) — 用户存储的路径。在 `access&#95;control&#95;path` 参数中配置。
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/zh/reference/data-types/index)) — 显示身份验证类型。用户身份可以通过多种方式验证：无密码、明文密码、SHA256 编码密码、双重 SHA-1 编码密码或 bcrypt 编码密码。
* `auth_params` ([Array(String)](/zh/reference/data-types/index)) — JSON 格式的身份验证参数，具体取决于 `auth&#95;type`。
* `valid_until` ([Array(DateTime)](/zh/reference/data-types/index)) — 用户凭据的过期日期和时间。
* `host_ip` ([Array(String)](/zh/reference/data-types/index)) — 允许连接到 ClickHouse 服务器的主机 IP 地址。
* `host_names` ([Array(String)](/zh/reference/data-types/index)) — 允许连接到 ClickHouse 服务器的主机名。
* `host_names_regexp` ([Array(String)](/zh/reference/data-types/index)) — 允许连接到 ClickHouse 服务器的主机名正则表达式。
* `host_names_like` ([Array(String)](/zh/reference/data-types/index)) — 使用 LIKE 谓词设置的、允许连接到 ClickHouse 服务器的主机名。
* `default_roles_all` ([UInt8](/zh/reference/data-types/index)) — 表示默认已为该用户设置所有已授予角色。
* `default_roles_list` ([Array(String)](/zh/reference/data-types/index)) — 默认提供的已授予角色列表。
* `default_roles_except` ([Array(String)](/zh/reference/data-types/index)) — 除列出的角色外，其他所有已授予角色都设为默认角色。
* `grantees_any` ([UInt8](/zh/reference/data-types/index)) — 该标志表示具有任意授权选项的用户是否可以将其授予任何人。
* `grantees_list` ([Array(String)](/zh/reference/data-types/index)) — 该用户允许向其授予授权选项的用户或角色列表。
* `grantees_except` ([Array(String)](/zh/reference/data-types/index)) — 禁止该用户向其授予授权选项的用户或角色列表。
* `default_database` ([String](/zh/reference/data-types/index)) — 该用户的默认数据库名称。

{/*AUTOGENERATED_END*/}
