<div id="description">
  ## 描述
</div>

显示此 ClickHouse server 当前在 ZooKeeper 节点 (包括辅助 ZooKeeper) 上注册的活动 [watch](https://zookeeper.apache.org/doc/r3.3.3/zookeeperProgrammers.html#ch_zkWatches)。每一行表示一个 watch。

<div id="columns">
  ## 列
</div>

* `zookeeper_name` ([String](/zh/reference/data-types/string)) — ZooKeeper 连接的名称 (主连接为 `default`，或辅助连接名称) 。
* `create_time` ([DateTime](/zh/reference/data-types/datetime)) — watch 创建的时间。
* `create_time_microseconds` ([DateTime64](/zh/reference/data-types/datetime64)) — 以微秒精度记录的 watch 创建时间。
* `path` ([String](/zh/reference/data-types/string)) — 被 watch 监听的 ZooKeeper 路径。
* `session_id` ([Int64](/zh/reference/data-types/int-uint)) — 注册该 watch 的连接的会话 ID。
* `request_xid` ([Int64](/zh/reference/data-types/int-uint)) — 创建该 watch 的请求的 XID。
* `op_num` ([Enum](/zh/reference/data-types/enum)) — 创建该 watch 的请求类型。
* `watch_type` ([Enum8](/zh/reference/data-types/enum)) — watch 类型。可能的值：
  * `Children` — 监听子节点列表的变化 (由 `List` 操作设置) 。
  * `Exists` — 监听节点的创建或删除。
  * `Data` — 监听节点数据的变化 (由 `Get` 操作设置) 。

示例：

```sql
SELECT * FROM system.zookeeper_watches FORMAT Vertical;
```

```text
Row 1:
──────
zookeeper_name:           default
create_time:              2026-03-16 12:00:00
create_time_microseconds: 2026-03-16 12:00:00.123456
path:                     /clickhouse/task_queue/ddl
session_id:               106662742089334927
request_xid:              10858
op_num:                   List
watch_type:               Children
```
