ClickHouse дёӯзҡ„ CASE иЎЁиҫҫејҸжҸҗдҫӣдәҶзұ»дјј SQL CASE иҝҗз®—з¬Ұзҡ„жқЎд»¶йҖ»иҫ‘гҖӮе®ғдјҡеҜ№жқЎд»¶иҝӣиЎҢжұӮеҖјпјҢе№¶ж №жҚ®з¬¬дёҖдёӘеҢ№й…Қзҡ„жқЎд»¶иҝ”еӣһзӣёеә”зҡ„еҖјгҖӮ

1. `CASE WHEN ... THEN ... ELSE ... END`
   <br />
   иҝҷз§ҚеҪўејҸе…·жңүе®Ңе…Ёзҡ„зҒөжҙ»жҖ§пјҢеҶ…йғЁйҖҡиҝҮ [multiIf](/zh/reference/functions/regular-functions/conditional-functions#multiIf) еҮҪж•°е®һзҺ°гҖӮжҜҸдёӘжқЎд»¶йғҪдјҡзӢ¬з«ӢжұӮеҖјпјҢиЎЁиҫҫејҸдёӯд№ҹеҸҜд»ҘеҢ…еҗ«йқһеёёйҮҸеҖјгҖӮ

```sql
SELECT
    number,
    CASE
        WHEN number % 2 = 0 THEN number + 1
        WHEN number % 2 = 1 THEN number * 10
        ELSE number

FROM system.numbers
WHERE number < 5;
```

дҫӢеҰӮпјҢдёӢйқўжүҖжңүиЎҢйғҪдјҡиҝ”еӣһз¬¬дёҖдёӘеҢ№й…ҚеҲҶж”ҜжүҖеңЁж—¶еҢәзҡ„ж—¶й—ҙжҲіпјҢеҚі `Asia/Kolkata`

<div id="if">
  ## if
</div>

Introduced in: v1.1.0

жү§иЎҢжқЎд»¶еҲҶж”ҜгҖӮ

* еҰӮжһңжқЎд»¶ `cond` зҡ„жұӮеҖјз»“жһңдёәйқһйӣ¶еҖјпјҢеҲҷеҮҪж•°иҝ”еӣһиЎЁиҫҫејҸ `then` зҡ„з»“жһңгҖӮ
* еҰӮжһң `cond` зҡ„жұӮеҖјз»“жһңдёәйӣ¶жҲ– NULLпјҢеҲҷиҝ”еӣһ `else` иЎЁиҫҫејҸзҡ„з»“жһңгҖӮ

и®ҫзҪ® [`short_circuit_function_evaluation`](/zh/reference/settings/session-settings#short_circuit_function_evaluation) з”ЁдәҺжҺ§еҲ¶жҳҜеҗҰеҗҜз”Ёзҹӯи·ҜжұӮеҖјгҖӮ

еҗҜз”ЁеҗҺпјҢд»…еңЁ `cond` дёә true зҡ„иЎҢдёҠи®Ўз®— `then` иЎЁиҫҫејҸпјҢиҖҢд»…еңЁ `cond` дёә false зҡ„иЎҢдёҠи®Ўз®— `else` иЎЁиҫҫејҸгҖӮ

дҫӢеҰӮпјҢеҗҜз”Ёзҹӯи·ҜжұӮеҖјеҗҺпјҢжү§иЎҢд»ҘдёӢжҹҘиҜўж—¶дёҚдјҡеӣ йҷӨд»Ҙйӣ¶иҖҢжҠӣеҮәејӮеёёпјҡ

```sql
SELECT if(number = 0, 0, intDiv(42, number)) FROM numbers(10)
```

`then` е’Ң `else` еҝ…йЎ»дёәзӣёиҝ‘зұ»еһӢгҖӮ

**иҜӯжі•**

```sql
if(cond, then, else)
```

**еҸӮж•°**

* `cond` вҖ” иҰҒи®Ўз®—зҡ„жқЎд»¶гҖӮ[`UInt8`](/zh/reference/data-types/int-uint) жҲ– [`Nullable(UInt8)`](/zh/reference/data-types/nullable) жҲ– [`NULL`](/zh/reference/syntax#null)
* `then` вҖ” еҰӮжһң `cond` дёә `true`пјҢеҲҷиҝ”еӣһиҜҘиЎЁиҫҫејҸгҖӮ- `else` вҖ” еҰӮжһң `cond` дёә `false` жҲ– `NULL`пјҢеҲҷиҝ”еӣһиҜҘиЎЁиҫҫејҸгҖӮ

**иҝ”еӣһеҖј**

ж №жҚ®жқЎд»¶ `cond`пјҢиҝ”еӣһ `then` жҲ– `else` иЎЁиҫҫејҸзҡ„з»“жһңгҖӮ

**зӨәдҫӢ**

**з”Ёжі•зӨәдҫӢ**

```sql title=Query
SELECT if(1, 2 + 2, 2 + 6) AS res;
```

```response title=Response
в”Ңв”Җresв”Җв”җ
в”Ӯ   4 в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”ҳ
```

<div id="multiIf">
  ## multiIf
</div>

еј•е…ҘзүҲжң¬пјҡv1.1.0

е…Ғи®ёеңЁжҹҘиҜўдёӯд»Ҙжӣҙзҙ§еҮ‘зҡ„ж–№ејҸзј–еҶҷ [`CASE`](/zh/reference/operators/index#conditional-expression) иҝҗз®—з¬ҰгҖӮ
жҢүйЎәеәҸи®Ўз®—еҗ„дёӘжқЎд»¶гҖӮеҜ№дәҺз¬¬дёҖдёӘдёә true (йқһйӣ¶дё”дёҚдёә `NULL`) зҡ„жқЎд»¶пјҢиҝ”еӣһеҜ№еә”зҡ„еҲҶж”ҜеҖјгҖӮ
еҰӮжһңжІЎжңүд»»дҪ•жқЎд»¶дёә trueпјҢеҲҷиҝ”еӣһ `else` еҖјгҖӮ

и®ҫзҪ® [`short_circuit_function_evaluation`](/zh/reference/settings/session-settings#short_circuit_function_evaluation) з”ЁдәҺжҺ§еҲ¶
жҳҜеҗҰдҪҝз”Ёзҹӯи·ҜжұӮеҖјгҖӮеҰӮжһңеҗҜз”ЁпјҢ`then_i` иЎЁиҫҫејҸд»…дјҡеңЁ
`((NOT cond_1) AND ... AND (NOT cond_{i-1}) AND cond_i)` дёә true зҡ„иЎҢдёҠи®Ўз®—гҖӮ

дҫӢеҰӮпјҢдҪҝз”Ёзҹӯи·ҜжұӮеҖјж—¶пјҢжү§иЎҢд»ҘдёӢжҹҘиҜўдёҚдјҡжҠӣеҮәйҷӨд»Ҙйӣ¶ејӮеёёпјҡ

```sql
SELECT multiIf(number = 2, intDiv(1, number), number = 5) FROM numbers(10)
```

жүҖжңүеҲҶж”ҜиЎЁиҫҫејҸе’Ң else иЎЁиҫҫејҸйғҪеҝ…йЎ»е…·жңүе…ұеҗҢи¶…зұ»еһӢгҖӮ`NULL` жқЎд»¶дјҡиў«и§Ҷдёә falseгҖӮ

**иҜӯжі•**

```sql
multiIf(cond_1, then_1, cond_2, then_2, ..., else)
```

**еҲ«еҗҚ**: `caseWithoutExpression`, `caseWithoutExpr`

**еҸӮж•°**

* `cond_N` вҖ” з¬¬ N дёӘжұӮеҖјжқЎд»¶пјҢз”ЁдәҺжҺ§еҲ¶жҳҜеҗҰиҝ”еӣһ `then_N`гҖӮ[`UInt8`](/zh/reference/data-types/int-uint) жҲ– [`Nullable(UInt8)`](/zh/reference/data-types/nullable) жҲ– [`NULL`](/zh/reference/syntax#null)
* `then_N` вҖ” еҪ“ `cond_N` дёә true ж—¶пјҢеҮҪж•°иҝ”еӣһзҡ„з»“жһңгҖӮ - `else` вҖ” еҰӮжһңжүҖжңүжқЎд»¶йғҪдёҚдёә trueпјҢеҮҪж•°иҝ”еӣһзҡ„з»“жһңгҖӮ

**иҝ”еӣһеҖј**

иҝ”еӣһдёҺеҢ№й…Қзҡ„ `cond_N` еҜ№еә”зҡ„ `then_N` з»“жһңпјӣеҗҰеҲҷиҝ”еӣһ `else` зҡ„з»“жһңгҖӮ

**зӨәдҫӢ**

**дҪҝз”ЁзӨәдҫӢ**

```sql title=Query

INSERT INTO LEFT_RIGHT VALUES (NULL, 4), (1, 3), (2, 2), (3, 1), (4, NULL);

SELECT
    left,
    right,
    multiIf(left < right, 'left is smaller', left > right, 'left is greater', left = right, 'Both equal', 'Null value') AS result
FROM LEFT_RIGHT;
```
