<div id="anyHeavy">
  ## anyHeavy
</div>

引入版本：v1.1.0

使用 [heavy hitters](https://doi.org/10.1145/762471.762473) 算法选取一个高频出现的值。
如果某个值在查询的每个执行线程中出现的次数都超过半数，则返回该值。
通常，结果是非确定性的。

**语法**

```sql
anyHeavy(column)
```

**参数**

* `column` — 列名。[`String`](/zh/reference/data-types/string)

**返回值**

返回一个出现频率较高的值。该结果是非确定性的。[`Any`](/zh/reference/data-types/index)

**示例**

**使用示例**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
