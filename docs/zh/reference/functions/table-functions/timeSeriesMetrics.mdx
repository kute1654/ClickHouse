`timeSeriesMetrics(db_name.time_series_table)` - 返回表引擎为 [TimeSeries](/zh/reference/engines/table-engines/integrations/time-series) 的表 `db_name.time_series_table` 所使用的 [指标](/zh/reference/engines/table-engines/integrations/time-series#metrics-table) 表：

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS metrics_table
```

当 *metrics* 表为内表时，该函数同样适用：

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

以下查询是等价的：

```sql
SELECT * FROM timeSeriesMetrics(db_name.time_series_table);
SELECT * FROM timeSeriesMetrics('db_name.time_series_table');
SELECT * FROM timeSeriesMetrics('db_name', 'time_series_table');
```
