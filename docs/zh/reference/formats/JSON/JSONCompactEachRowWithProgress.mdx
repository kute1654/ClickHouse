<div id="description">
  ## 说明
</div>

此格式将 JSONCompactEachRow 紧凑的逐行输出与流式进度信息结合起来。
它会将元数据、各行、进度更新、
总计和异常分别输出为独立的 JSON 对象。值按其原生类型表示。

主要特性：

* 首先输出包含列名和类型的元数据
* 每一行都是一个独立的 JSON 对象，其中 `"row"` 键包含一个值数组
* 在查询执行期间包含进度更新 (以 `{"progress":...}` 对象的形式) 
* 支持总计和极值
* 值保留其原生类型 (数字仍为数字，字符串仍为字符串)

<div id="example-usage">
  ## 使用示例
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactEachRowWithProgress
```
