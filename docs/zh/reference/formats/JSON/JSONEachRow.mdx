<div id="description">
  ## 描述
</div>

在此格式中，ClickHouse 会将每一行输出为一个单独的、以换行符分隔的 JSON 对象。

<div id="example-usage">
  ## 示例用法
</div>

```sql

```

<div id="reading-data">
  ### 读取数据
</div>

使用 `JSONEachRow` 格式读取数据：

```sql
SELECT *
FROM football
FORMAT JSONEachRow
```
