<div id="description">
  ## 说明
</div>

<Tip>
  JSONColumns* 格式的输出会先给出 ClickHouse 字段名，然后列出该字段在表中各行的内容；
  直观来看，数据相当于向左旋转了 90 度。
</Tip>

在这种格式中，所有数据都表示为一个 JSON Object。

<Note>
  `JSONColumns` 格式会先在内存中缓冲所有数据，再将其作为单个块输出，因此可能导致较高的内存占用。
</Note>

<div id="reading-data">
  ### 读取数据
</div>

使用 `JSONColumns` 格式读取数据：

```sql
SELECT *
FROM football
FORMAT JSONColumns
```

<div id="format-settings">
  ## 格式设置
</div>

导入时，如果将 [`input_format_skip_unknown_fields`](/zh/reference/settings/formats#input_format_skip_unknown_fields) 设置为 `1`，则会跳过名称未知的列。
块中不存在的列将填充为默认值 (此处可使用 [`input_format_defaults_for_omitted_fields`](/zh/reference/settings/formats#input_format_defaults_for_omitted_fields) 设置)
