<div id="description">
  ## 描述
</div>

它与 [JSONEachRow](/zh/reference/formats/JSON/JSONEachRow) 的唯一区别在于，JSON 经过了美化格式化，使用换行符作分隔，并采用 4 个空格缩进。

<div id="example-usage">
  ## 使用示例
</div>

<div id="reading-data">
  ### 读取数据
</div>

使用 `PrettyJSONEachRow` 格式读取数据：

```sql
SELECT *
FROM football
FORMAT PrettyJSONEachRow
```
