<div id="description">
  ## 说明
</div>

在 `Null` 格式中，不会输出任何内容。
这听起来可能有些奇怪，但需要注意的是，尽管没有任何输出，查询仍会被处理；
而且在使用命令行客户端时，数据仍会传输到客户端。

<Tip>
  `Null` 格式可用于性能测试。
</Tip>
