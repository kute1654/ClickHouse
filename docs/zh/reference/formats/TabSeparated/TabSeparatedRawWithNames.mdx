<div id="description">
  ## 描述
</div>

与 [`TabSeparatedWithNames`](/zh/reference/formats/TabSeparated/TabSeparatedWithNames) 格式的不同之处在于，
该格式写入行时不会进行转义。

<Note>
  使用此格式解析时，每个字段中都不能包含制表符或换行符。
</Note>

```sql

FROM football
FORMAT TabSeparatedRawWithNames
```
