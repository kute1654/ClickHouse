该数据集包含 GitHub 上从 2011 年到 2020 年 12 月 6 日期间的所有事件，总计 31 亿条记录。下载大小为 75 GB；如果以启用 lz4 压缩的表形式存储，最多需要 200 GB 的磁盘空间。

完整的数据集说明、相关洞察、下载说明以及交互式查询见此[仓库](https://github.com/ClickHouse/github-explorer)
