<div id="what-youll-build">
  ## 你将构建的内容
</div>

在本快速入门中，你将创建首个 ClickHouse Cloud 服务，熟悉 Cloud 控制台和内置的 SQL 控制台，并加载一个示例数据集，从而可以基于真实数据进行查询。
完成后，你将拥有一个正在运行且已加载数据的 ClickHouse Cloud 服务，可作为后续快速入门系列的基础。

<Steps titleSize="h3">
  <Step>
    ### 注册 ClickHouse Cloud

    前往 [console.clickhouse.cloud](https://console.clickhouse.cloud/)。你会看到登录页面。点击底部“还没有账户？”旁边的 **Register**。

    <Frame caption="点击 Register 创建新账户">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="带有 Register 链接的 ClickHouse Cloud 登录页面" />
    </Frame>

    在注册页面中，你可以使用 Google、Microsoft 注册，或输入企业邮箱。无需信用卡——免费试用提供 300 美元额度和 30 天试用期，供你体验 ClickHouse Cloud。

    <Frame caption="使用 Google、Microsoft 或企业邮箱注册">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="ClickHouse Cloud 注册页面" />
    </Frame>

    如果你使用邮箱注册，系统会提示你设置密码。密码必须满足以下要求：

    * 至少 12 个字符
    * 以下 4 项中至少满足 3 项：小写字母、大写字母、数字、特殊字符 (例如 `!@#$%^&*`)

    <Frame caption="输入密码并接受服务条款">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="带有密码字段的 ClickHouse Cloud 邮箱注册页面" />
    </Frame>

    接受服务条款和隐私政策，然后点击 **Sign up**。检查邮箱中的确认链接，并点击 **Verify email** 激活账户。

    验证完成后，使用新的邮箱和密码登录。
  </Step>

  <Step>
    ### 创建新服务

    首次登录后，系统会直接将你带到服务配置页面。请完成以下配置：

    | 设置         | 建议                                                 |
    | ---------- | -------------------------------------------------- |
    | **服务名称**   | 为服务起一个便于识别的名称 (例如 `My first service`)              |
    | **云提供商**   | 根据你的偏好选择 AWS、GCP 或 Azure                           |
    | **区域**     | 选择离你最近的区域，以获得最低延迟                                  |
    | **内存和扩缩容** | 在本快速入门中，请选择 **Mini 12GB** (1 个副本) ——这是最经济、最适合学习的选项 |

    点击 **创建服务** 以预配服务。

    <Frame caption="配置你的云服务">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="ClickHouse Cloud 服务配置页面" />
    </Frame>

    随后你会进入服务主页，初始状态会显示为 **Provisioning**。此过程通常不到一分钟。完成后，状态将变为 **Running**。

    <Frame caption="你的服务正在运行">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="显示 Running 状态的 ClickHouse Cloud 服务主页" />
    </Frame>
  </Step>

  <Step>
    ### 加载示例数据集

    点击左侧边栏中的 **Data sources**。你会看到几种将数据导入到服务中的方式。

    <Frame caption="包含数据加载选项的数据源页面">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="ClickHouse Cloud 数据源页面" />
    </Frame>

    点击 **Add sample data**，浏览可用的示例数据集。你会看到三个选项：Cell Towers、UK Property Price Paid 和 New York Taxi Data。点击 **UK Property Price Paid** 数据集上的 **Get started**。

    <Frame caption="选择 UK Property Price Paid 数据集">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="ClickHouse Cloud 示例数据集选择" />
    </Frame>

    你会看到数据预览和表 schema。点击 **Import dataset**，将数据加载到你的服务中。

    <Frame caption="预览数据并点击 Import dataset">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="带英国房产数据的 ClickHouse Cloud 数据集导入预览" />
    </Frame>

    导入过程需要一两分钟。完成后，即可开始查询数据。
  </Step>

  <Step>
    ### 查询数据

    点击左侧边栏中的 **SQL 控制台**，打开内置查询编辑器。你可以直接在浏览器中编写并运行 SQL 查询，无需借助任何外部工具。

    尝试运行几个查询，探索英国房产价格数据。

    找出历史上成交价最高的房产：

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    按年份计算平均销售价格：

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    这些查询能在几秒内扫描数百万行——这正是 ClickHouse 的列式存储和压缩技术发挥作用的体现。
  </Step>
</Steps>
