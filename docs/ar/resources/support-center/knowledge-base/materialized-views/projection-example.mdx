<div id="question">
  ## السؤال
</div>

كيف يمكنني معرفة ما إذا كان يُستخدم الإسقاط؟

<div id="answer">
  ## الإجابة
</div>

1. أنشئ قاعدة بيانات نموذجية

```
CREATE database db1;
```

2. أنشئ جدولًا نموذجيًا يستخدم column1 كمفتاح أساسي

```
CREATE table db1.table1_projections
(
 column1 Int32,
 column2 Int32
)
engine = MergeTree()
order by column1;
```

3. أضِف إسقاطًا `for_column2` لجعل column2 المفتاح الأساسي

```
ALTER table db1.table1_projections add projection for_column2
(
  select * 
  order by column2
);
```

4. أدرج بيانات اختبار

*سيؤدي ذلك إلى إدراج 100000 صف بقيم عشوائية في column1 وcolumn2

```
INSERT INTO db1.table1_projections
select 
 floor(randNormal(50, 5)) as column1,
 floor(randUniform(1, 100)) as column2
from numbers(100000);
```

5. تحقّق من عيّنة من البيانات

```
clickhouse-cloud :) SELECT * from db1.table1_projections limit 5;

SELECT *
FROM db1.table1_projections
LIMIT 5

┌─column1─┬─column2─┐
│      28 │      41 │
│      29 │      12 │
│      30 │      73 │
│      30 │      75 │
│      30 │      70 │
└─────────┴─────────┘
```

6. تحقّق من أنه يستخدم الجدول الأصلي مع column1:

```
clickhouse-cloud :) explain indexes = 1 
                    SELECT count() from db1.table1_projections where column1 > 50;

EXPLAIN indexes = 1
SELECT count()
FROM db1.table1_projections
WHERE column1 > 50

```

*لاحظ أنه يقرأ من `db1.table1_projections`

7. اختبر القراءة من الإسقاط باستخدام column2 في عبارة WHERE

