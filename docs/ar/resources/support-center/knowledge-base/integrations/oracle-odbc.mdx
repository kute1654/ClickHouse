إذا كنت تستخدم Oracle كمصدر لقواميس ClickHouse الخارجية عبر Oracle ODBC Driver، فيجب ضبط القيمة الصحيحة لمتغير البيئة `NLS_LANG` في `/etc/default/clickhouse`. لمزيد من المعلومات، راجع [الأسئلة الشائعة حول Oracle NLS&#95;LANG](https://www.oracle.com/technetwork/products/globalization/nls-lang-099431.html).

**مثال**

```sql
NLS_LANG=RUSSIAN_RUSSIA.UTF8
```
