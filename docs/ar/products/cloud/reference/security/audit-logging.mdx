<div id="console-audit-log-events">
  ## أحداث سجل تدقيق Console
</div>

تُصنَّف الأنواع المختلفة من الأحداث التي يتم التقاطها على مستوى المؤسسة ضمن 3 فئات: **المؤسسة** و**الخدمة** و**المستخدم**. لمزيد من المعلومات حول سجل التدقيق وكيفية تصديره أو إضافة تكامل عبر API، راجع وثائق [سجل تدقيق Console](/ar/products/cloud/guides/security/audit-logging/console-audit-log) في قسم الأدلة أعلاه.

تُسجَّل الأحداث التالية في سجل التدقيق.

<div id="organization">
  ### المؤسسة
</div>

* تم إنشاء المؤسسة
* تم حذف المؤسسة
* تم تغيير اسم المؤسسة

<div id="service">
  ### الخدمة
</div>

* تم إنشاء الخدمة
* تم حذف الخدمة
* تم إيقاف الخدمة
* تم تشغيل الخدمة
* تم تغيير اسم الخدمة
* تم تغيير قائمة الوصول عبر IP الخاصة بالخدمة
* تمت إعادة تعيين كلمة مرور الخدمة

<div id="user">
  ### المستخدم
</div>

* تم تغيير دور المستخدم
* تمت إزالة المستخدم من المؤسسة
* تمت دعوة المستخدم إلى المؤسسة
* انضم المستخدم إلى المؤسسة
* تم حذف دعوة المستخدم
* غادر المستخدم المؤسسة
