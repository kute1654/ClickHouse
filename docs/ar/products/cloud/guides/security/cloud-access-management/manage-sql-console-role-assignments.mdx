<Warning>
  هذا الإعداد مُهمَل وقد استُبدل بإعداد جديد في شاشة إنشاء الأدوار في المؤسسات التي انتقلت إلى الأدوار المخصصة. لمزيد من المعلومات، راجع [Manage cloud users](/ar/products/cloud/guides/security/cloud-access-management/manage-cloud-users).
</Warning>

> يوضّح هذا الدليل كيفية تكوين تعيينات أدوار SQL Console، والتي
> تحدد أذونات الوصول على مستوى وحدة التحكم والميزات التي يمكن
> للمستخدم الوصول إليها داخل Cloud Console.

<Steps>
  <Step>
    ### الوصول إلى إعدادات الخدمة

    من صفحة الخدمات، انقر على القائمة في الزاوية العلوية اليمنى للخدمة التي تريد تعديل إعدادات الوصول إلى SQL Console لها.

    <Image img="/images/cloud/guides/sql_console/service_level_access/1_service_settings.webp" size="lg" />

    حدِّد `settings` من القائمة المنبثقة.

    <Image img="/images/cloud/guides/sql_console/service_level_access/2_service_settings.webp" size="lg" />
  </Step>

  <Step>
    ### ضبط الوصول إلى SQL Console

    ضمن قسم &quot;Security&quot;، ابحث عن منطقة &quot;SQL console access&quot;:

    <Image img="/images/cloud/guides/sql_console/service_level_access/3_service_settings.webp" size="md" />
  </Step>

  <Step>
    ### تحديث الإعدادات لـ Service Admin

    حدِّد القائمة المنسدلة الخاصة بـ Service Admin لتغيير إعدادات التحكم في الوصول لأدوار Service Admin:

    <Image img="/images/cloud/guides/sql_console/service_level_access/4_service_settings.webp" size="md" />

    يمكنك الاختيار من الأدوار التالية:

    | الدور         |
    | ------------- |
    | `No access`   |
    | `Read only`   |
    | `Full access` |
  </Step>

  <Step>
    ### تحديث الإعدادات لـ Service Read Only

    حدِّد القائمة المنسدلة الخاصة بـ Service Read Only لتغيير إعدادات التحكم في الوصول لأدوار Service Read Only:

    <Image img="/images/cloud/guides/sql_console/service_level_access/5_service_settings.webp" size="md" />

    يمكنك الاختيار من الأدوار التالية:

    | الدور         |
    | ------------- |
    | `No access`   |
    | `Read only`   |
    | `Full access` |
  </Step>

  <Step>
    ### مراجعة المستخدمين الذين لديهم حق الوصول

    يمكن عرض نظرة عامة على مستخدمي الخدمة من خلال تحديد عدد المستخدمين:

    <Image img="/images/cloud/guides/sql_console/service_level_access/6_service_settings.webp" size="md" />

    سيُفتح تبويب على يمين الصفحة يعرض العدد الإجمالي للمستخدمين وأدوارهم:

    <Image img="/images/cloud/guides/sql_console/service_level_access/7_service_settings.webp" size="md" />
  </Step>
</Steps>
