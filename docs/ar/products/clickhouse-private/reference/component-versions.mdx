الإصدارات المستقرة الحالية لجميع مكوّنات ClickHouse Private. استخدم هذه الوسوم عند نسخ الحِزم البرمجية إلى المستودع الخاص بك وتثبيتها عبر Helm.

  ## صور الحاويات
</div>

| المكوّن                  | وسم الصورة          | الوصف                                     |
| ------------------------ | ------------------- | ----------------------------------------- |
| clickhouse-server        | `25.12.1.1497`      | خادم ClickHouse (قياسي)                   |
| clickhouse-server (FIPS) | `25.12.1.1497-fips` | خادم ClickHouse المتوافق مع FIPS 140-2    |
| clickhouse-keeper        | `26.2.1.258`        | خدمة تنسيق موزعة (بديل لـ ZooKeeper)      |
| clickhouse-keeper (FIPS) | `26.2.1.258-fips`   | ClickHouse Keeper المتوافق مع FIPS 140-2  |
| clickhouse-operator      | `main-1.18587.1`    | مشغّل Kubernetes لإدارة دورة حياة العنقود |

<div id="notes">
  ## ملاحظات
</div>

* يختلف **إصدار مخطط Helm الخاص بالمشغّل** (`1.18587.1`) عن **وسم صورة** المشغّل (`main-1.18587.1`). استخدم إصدار مخطط Helm عند التثبيت عبر `helm install`، واستخدم وسم الصورة عند نسخ صور الحاويات.
* تُضيف **إصدارات FIPS** اللاحقة `-fips` إلى وسوم الصور القياسية لكلٍّ من خادم وKeeper.
* تُستبدل وسوم الإصدار تلقائيًا عند بناء هذه الوثائق. إذا رأيت متغيرات قالب خام (مثل `25.12.1.1497`)، فهذا يعني أن خطوة البناء لم تُنفَّذ.
