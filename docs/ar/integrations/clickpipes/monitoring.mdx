بالإضافة إلى المراقبة داخل Console، يوفّر ClickPipes مقاييس عبر [نقطة نهاية متوافق مع Prometheus](/ar/products/cloud/features/monitoring/prometheus) ليتم جمعها. تُنشر هذه المقاييس مع مقاييس خدمة ClickHouse Cloud الأخرى، وتتيح لك دمج مراقبة ClickPipes مع مكدس observability الحالي لديك (مثل [Grafana](/ar/products/cloud/features/monitoring/prometheus#integrating-with-grafana) و[Datadog](/ar/products/cloud/features/monitoring/prometheus#integrating-with-datadog)).

<div id="metric-labels">
  ## تسميات المقاييس
</div>

تستخدم مقاييس ClickPipes [التسميات القياسية على مستوى الخدمة](/ar/products/cloud/features/monitoring/prometheus#metric-labels) (`clickhouse_org`, `clickhouse_service`, `clickhouse_service_name`)، بالإضافة إلى تسميات ClickPipes التالية:

| التسمية            | الوصف                                                                      |
| ------------------ | -------------------------------------------------------------------------- |
| `clickpipe_id`     | المعرّف الفريد لـ ClickPipe                                                |
| `clickpipe_name`   | اسم ClickPipe                                                              |
| `clickpipe_source` | نوع المصدر (على سبيل المثال: `kafka` أو `postgres` أو `s3`)                |
| `clickpipe_state`  | حالة ClickPipe (على سبيل المثال: `Stopped` أو `Provisioning` أو `Running`) |

<div id="available-metrics">
  ## المقاييس المتاحة
</div>

<div id="metrics-data-transfer">
  ### نقل البيانات
</div>

| المقياس                                    | النوع | الوصف                                                                                                                             |
| ------------------------------------------ | ----- | --------------------------------------------------------------------------------------------------------------------------------- |
| `ClickPipes_FetchedBytes_Total`            | عداد  | إجمالي البايتات غير المضغوطة التي جرى جلبها من المصدر.                                                                            |
| `ClickPipes_FetchedBytesCompressed_Total`  | عداد  | إجمالي البايتات المضغوطة التي جرى جلبها من المصدر. إذا كانت بيانات المصدر غير مضغوطة، فهذا يعادل `ClickPipes_FetchedBytes_Total`. |
| `ClickPipes_FetchedBytesInitialLoad_Total` | عداد  | **CDC ClickPipes** إجمالي البايتات غير المضغوطة التي جرى جلبها خلال مرحلة التحميل الأولي.                                         |
| `ClickPipes_FetchedBytesResync_Total`      | عداد  | **CDC ClickPipes** إجمالي البايتات غير المضغوطة التي جرى جلبها أثناء عملية إعادة المزامنة.                                        |
| `ClickPipes_SentBytes_Total`               | عداد  | إجمالي البايتات غير المضغوطة المُرسلة إلى ClickHouse.                                                                             |
| `ClickPipes_SentBytesCompressed_Total`     | عداد  | إجمالي البايتات المضغوطة المُرسلة إلى ClickHouse.                                                                                 |

<div id="metrics-events">
  ### الأحداث والسجلات
</div>

| المقياس                             | النوع | الوصف                                                                                                                                |
| ----------------------------------- | ----- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `ClickPipes_FetchedEvents_Total`    | عداد  | إجمالي عدد السجلات التي تم جلبها من المصدر.                                                                                          |
| `ClickPipes_SentEvents_Total`       | عداد  | إجمالي عدد السجلات التي تم إرسالها إلى ClickHouse.                                                                                   |
| `ClickPipes_FetchedEvents_PerTable` | مقياس | **CDC ClickPipes** عدد السجلات التي تم جلبها لكل جدول وجهة ولكل نوع حدث. يستخدم تسميتي `destination_table` و`event_type` الإضافيتين. |

<div id="metrics-errors">
  ### الأخطاء
</div>

| المقياس                   | النوع | الوصف                                                                                             |
| ------------------------- | ----- | ------------------------------------------------------------------------------------------------- |
| `ClickPipes_Errors_Total` | عدّاد | إجمالي [الأخطاء](/ar/integrations/clickpipes/home#error-reporting) التي تمت مواجهتها أثناء الاستيعاب. |

<div id="metrics-latency">
  ### زمن الاستجابة
</div>

| المقياس                                   | النوع         | الوصف                                                                                                                        |
| ----------------------------------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `ClickPipes_Latency`                      | مقياس (*avg*) | الوقت، بالمللي ثانية، بين إنشاء سجل في المصدر وكتابته إلى ClickHouse.                                                        |
| `ClickPipes_SourceReplicationLatency_MiB` | مقياس (*avg*) | **CDC ClickPipes** تأخر النسخ المتماثل في المصدر بوحدة MiB. بالنسبة إلى Postgres ClickPipes، يعكس هذا تأخر replication slot. |

<div id="metrics-resources">
  ### استخدام الموارد
</div>

| مقياس                            | النوع          | الوصف                                                                  |
| -------------------------------- | -------------- | ---------------------------------------------------------------------- |
| `ClickPipes_Replica_CPUUsage`    | مقياس (*avg*)  | متوسط استخدام CPU للنسخ المتماثلة الخاصة بالاستيعاب، بوحدة millicores. |
| `ClickPipes_Replica_CPULimit`    | مقياس (*last*) | حد CPU المُعدّ للنسخ المتماثلة الخاصة بالاستيعاب، بوحدة millicores.    |
| `ClickPipes_Replica_MemoryUsage` | مقياس (*avg*)  | متوسط استخدام الذاكرة للنسخ المتماثلة الخاصة بالاستيعاب، بالبايت.      |
| `ClickPipes_Replica_MemoryLimit` | مقياس (*last*) | حد الذاكرة المُعدّ للنسخ المتماثلة الخاصة بالاستيعاب، بالبايت.         |

<div id="metrics-state">
  ### الحالة والتقدم
</div>

| المقياس                         | النوع | الوصف                                                                                            |
| ------------------------------- | ----- | ------------------------------------------------------------------------------------------------ |
| `ClickPipes_Info`               | مقياس | يساوي دائمًا `1`. استخدم التسمية `clickpipe_state` للتصفية أو التنبيه استنادًا إلى حالة الـ pipe. |
| `ClickPipes_LastFetchedBatchId` | مقياس | معرّف آخر دفعة جرى جلبها من المصدر.                                                              |
| `ClickPipes_LastSentBatchId`    | مقياس | معرّف آخر دفعة أُرسلت إلى ClickHouse.                                                            |
