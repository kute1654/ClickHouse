`timeSeriesMetrics(db_name.time_series_table)` - تعيد جدول [المقاييس](/ar/reference/engines/table-engines/integrations/time-series#metrics-table)
الذي يستخدمه الجدول `db_name.time_series_table` والذي يكون محرك جدوله هو محرك [TimeSeries](/ar/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS metrics_table
```

تعمل الدالة أيضًا إذا كان جدول *metrics* داخليًا:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

الاستعلامات التالية متكافئة:

```sql
SELECT * FROM timeSeriesMetrics(db_name.time_series_table);
SELECT * FROM timeSeriesMetrics('db_name.time_series_table');
SELECT * FROM timeSeriesMetrics('db_name', 'time_series_table');
```
