`timeSeriesTags(db_name.time_series_table)` - يعيد [جدول الوسوم](/ar/reference/engines/table-engines/integrations/time-series#tags-table) الذي
يستخدمه الجدول `db_name.time_series_table` والذي يكون محرك جدوله هو محرك [TimeSeries](/ar/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS tags_table
```

كما تعمل الدالة إذا كان جدول *tags* داخليًا:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

الاستعلامات التالية متكافئة:

```sql
SELECT * FROM timeSeriesTags(db_name.time_series_table);
SELECT * FROM timeSeriesTags('db_name.time_series_table');
SELECT * FROM timeSeriesTags('db_name', 'time_series_table');
```
