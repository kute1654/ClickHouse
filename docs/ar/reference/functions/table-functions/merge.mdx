ينشئ [Merge](/ar/reference/engines/table-engines/special/merge) جدولًا مؤقتًا.
يُشتق مخطط الجدول من الجداول الأساسية باستخدام اتحاد أعمدتها واستنتاج الأنواع المشتركة.
تتوفّر الأعمدة الظاهرية نفسها كما في محرك جدول [Merge](/ar/reference/engines/table-engines/special/merge).

<div id="syntax">
  ## البنية
</div>

```sql
merge(['db_name',] 'tables_regexp')
```

<div id="arguments">
  ## الوسيطات
</div>

| الوسيط          | الوصف                                                                                                                                                                                                                                                                                                      |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `db_name`       | القيم الممكنة (اختياري، والقيمة الافتراضية هي `currentDatabase()`):<br />    - اسم قاعدة البيانات،<br />    - تعبير ثابت يُرجع سلسلة نصية تتضمن اسم قاعدة بيانات، على سبيل المثال `currentDatabase()`،<br />    - `REGEXP(expression)`، حيث تكون `expression` تعبيرًا نمطيًا لمطابقة أسماء قواعد البيانات. |
| `tables_regexp` | تعبير نمطي لمطابقة أسماء الجداول في قاعدة البيانات أو قواعد البيانات المحددة.                                                                                                                                                                                                                              |
