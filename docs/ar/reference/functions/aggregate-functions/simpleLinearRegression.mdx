<div id="simpleLinearRegression">
  ## simpleLinearRegression
</div>

أُضيف في: v20.1.0

ينفّذ انحدارًا خطيًا بسيطًا (أحادي البعد).

**الصيغة**

```sql
simpleLinearRegression(x, y)
```

**الوسيطات**

* `x` — عمود يحتوي على قيم المتغير التفسيري. [`Float64`](/ar/reference/data-types/float)
* `y` — عمود يحتوي على قيم المتغير التابع. [`Float64`](/ar/reference/data-types/float)

**القيمة المُعادة**

يعيد الثابتين `(k, b)` للخط المستقيم الناتج `y = k*x + b`. [`Tuple(Float64, Float64)`](/ar/reference/data-types/tuple)
