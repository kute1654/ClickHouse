<div id="entropy">
  ## entropy
</div>

أُضيفت في: v20.1.0

تحسب [إنتروبيا شانون](https://en.wikipedia.org/wiki/Entropy_\(information_theory\)) لعمود من القيم.

**الصياغة**

```sql
entropy(val)
```

**الوسائط**

* `val` — عمود من القيم من أي نوع. [`Any`](/ar/reference/data-types/index)

**القيمة المُعادة**

تُرجِع إنتروبيا شانون. [`Float64`](/ar/reference/data-types/float)

**أمثلة**

**الحساب الأساسي لإنتروبيا شانون**

```sql title=Query
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy
```

```response title=Response
┌─entropy(vals)─┬─entropy(strings)─┐
│             1 │             1.75 │
└───────────────┴──────────────────┘
```
