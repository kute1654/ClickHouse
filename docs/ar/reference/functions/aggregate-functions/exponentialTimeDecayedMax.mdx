<div id="exponentialTimeDecayedMax">
  ## exponentialTimeDecayedMax
</div>

أُضيفت في: v21.12.0

تعيد القيمة العظمى بين المتوسط المتحرك المملّس أسيًا المحسوب عند الفهرس `t` وقيمته عند `t-1`.

**الصياغة**

```sql
exponentialTimeDecayedMax(x)(value, timeunit)
```

**المعلمات**

* `x` — فترة نصف العمر. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Decimal`](/ar/reference/data-types/decimal)

**الوسائط**

* `value` — القيمة. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Decimal`](/ar/reference/data-types/decimal)
* `timeunit` — الوحدة الزمنية. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Decimal`](/ar/reference/data-types/decimal) أو [`DateTime`](/ar/reference/data-types/datetime) أو [`DateTime64`](/ar/reference/data-types/datetime64)

**القيمة المعادة**

تعيد أكبر قيمة للمتوسط المتحرك المرجّح المملّس أسيًا عند `t` و`t-1`. [`Float64`](/ar/reference/data-types/float)

**أمثلة**

**استخدام دالة النافذة مع تمثيل مرئي**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedMax(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
