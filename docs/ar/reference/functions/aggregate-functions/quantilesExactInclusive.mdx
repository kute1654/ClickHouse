<div id="quantilesExactInclusive">
  ## quantilesExactInclusive
</div>

قُدِّمت في: v20.1.0

تحسب هذه الدالة بدقة عدة [كوانتيلات](https://en.wikipedia.org/wiki/Quantile) لتسلسل بيانات رقمية عند مستويات مختلفة في الوقت نفسه باستخدام الطريقة التضمينية.

تعادل هذه الدالة [`quantileExactInclusive`](/ar/reference/functions/aggregate-functions/quantileExactInclusive)، لكنها تتيح حساب عدة مستويات كوانتيل في مرور واحد، ما يجعلها أكثر كفاءة من استدعاء دوال الكوانتيل الفردية.

تستخدم هذه الدالة الطريقة التضمينية لحساب الكوانتيلات، كما هو موضح في [طريقة R-7](https://en.wikipedia.org/wiki/Quantile#Estimating_quantiles_from_a_sample).
وهذا يعادل الدالة [PERCENTILE.INC](https://support.microsoft.com/en-us/office/percentile-inc-function-680f9539-45eb-410b-9a5e-c1355e5fe2ed) في Excel.

للحصول على قيم دقيقة، تُجمَع جميع القيم المُمرَّرة في مصفوفة، ثم تُفرَز جزئيًا.
ويبلغ تعقيد خوارزمية الفرز `O(N·log(N))` من المقارنات، حيث `N = std::distance(first, last)`.

**البنية**

```sql
quantilesExactInclusive(level1, level2, ...)(expr)
```

**المعلمات**

* `level` — مستويات الكوانتيلات. أعداد فاصلة عائمة ثابتة من 0 إلى 1 (شاملًا). نوصي باستخدام قيم `level` ضمن النطاق `[0.01, 0.99]`. [`Float*`](/ar/reference/data-types/float)

**الوسائط**

* `expr` — تعبير يعتمد على قيم العمود ويُنتج أنواع بيانات رقمية، أو [`Date`](/ar/reference/data-types/date)، أو [`DateTime`](/ar/reference/data-types/datetime). [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Decimal*`](/ar/reference/data-types/decimal) أو [`Date`](/ar/reference/data-types/date) أو [`DateTime`](/ar/reference/data-types/datetime)

**القيمة المعادة**

مصفوفة من الكوانتيلات للمستويات المحددة، بالترتيب نفسه الذي حُدِّدت به هذه المستويات. [`Array(Float64)`](/ar/reference/data-types/array)

**أمثلة**

**حساب عدة كوانتيلات شاملة دقيقة**

```sql title=Query
CREATE TABLE num AS numbers(1000);
SELECT quantilesExactInclusive(0.25, 0.5, 0.75, 0.9, 0.95, 0.99, 0.999)(number) FROM num;
```

```response title=Response
┌─quantilesExactInclusive(0.25, 0.5, 0.75, 0.9, 0.95, 0.99, 0.999)(number)─┐
│ [249.75,499.5,749.25,899.1,949.05,989.01,998.001]                        │
└──────────────────────────────────────────────────────────────────────────┘
```
