<div id="groupBitOr">
  ## groupBitOr
</div>

قُدِّم في: v1.1.0

يطبّق `OR` على مستوى البت على سلسلة من الأرقام.

**الصيغة**

```sql
groupBitOr(expr)
```

**الأسماء البديلة**: `BIT_OR`

**الوسائط**

* `expr` — تعبير من النوع `(U)Int*`. [`(U)Int*`](/ar/reference/data-types/int-uint)

**القيمة المُعادة**

يعيد قيمة من النوع `(U)Int*`. [`(U)Int*`](/ar/reference/data-types/int-uint)

**أمثلة**

**مثال على `OR` على مستوى البت**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
