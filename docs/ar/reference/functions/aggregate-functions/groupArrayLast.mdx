<div id="groupArrayLast">
  ## groupArrayLast
</div>

أُضيفت في: v23.1.0

تنشئ مصفوفة من آخر قيم الوسيط.
على سبيل المثال، فإن `groupArrayLast(1)(x)` مكافئة لـ `[anyLast(x)]`.
في بعض الحالات، لا يزال بإمكانك الاعتماد على ترتيب التنفيذ.
وينطبق هذا على الحالات التي يأتي فيها SELECT من استعلام فرعي يستخدم ORDER BY إذا كانت نتيجة الاستعلام الفرعي صغيرة بما يكفي.

**الصياغة**

```sql
groupArrayLast(max_size)(x)
```

**المعلمات**

* `max_size` — الحجم الأقصى للمصفوفة الناتجة. [`UInt64`](/ar/reference/data-types/int-uint)

**الوسيطات**

* `max_size` — الحجم الأقصى للمصفوفة الناتجة. [`UInt64`](/ar/reference/data-types/int-uint)
* `x` — وسيطة (اسم العمود أو تعبير). [`Any`](/ar/reference/data-types/index)

**القيمة المُعادة**

تعيد مصفوفة من آخر قيم الوسيطة. [`Array(T)`](/ar/reference/data-types/array)

**أمثلة**

**مثال على الاستخدام**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**مقارنة بـ groupArray**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
