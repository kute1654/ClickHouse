<div id="timeSeriesGroupArray">
  ## timeSeriesGroupArray
</div>

أُضيف في: v25.8.0

يرتّب بيانات السلاسل الزمنية حسب الطابع الزمني بترتيب تصاعدي.

<Note>
  هذه الدالة تجريبية. فعِّلها بتعيين `allow_experimental_ts_to_grid_aggregate_function=true`.
</Note>

**البنية**

```sql
timeSeriesGroupArray(timestamp, value)
```

**المعاملات**

* `timestamp` — الطابع الزمني للعينة. [`DateTime`](/ar/reference/data-types/datetime) أو [`UInt32`](/ar/reference/data-types/int-uint) أو [`UInt64`](/ar/reference/data-types/int-uint)
* `value` — قيمة السلسلة الزمنية المقابلة للطابع الزمني. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Decimal`](/ar/reference/data-types/decimal)

**القيمة المُعادة**

تُرجِع مصفوفة من عناصر `Tuple` بالشكل `(timestamp, value)` مرتبة حسب الطابع الزمني ترتيبًا تصاعديًا. إذا وُجدت عدة قيم للطابع الزمني نفسه، فستختار الدالة أكبر هذه القيم. [`Array(Tuple(T1, T2))`](/ar/reference/data-types/array)

**أمثلة**

**الاستخدام الأساسي مع القيم الفردية**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamp, value)
FROM
(
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesGroupArray(timestamp, value)───────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]  │
└──────────────────────────────────────────────────────┘
```

**تمرير عينات متعددة من الطوابع الزمنية والقيم في صورة مصفوفات متساوية الحجم**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamps, values);
```

```response title=Response
┌─timeSeriesGroupArray(timestamps, values)──────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]   │
└───────────────────────────────────────────────────────┘
```
