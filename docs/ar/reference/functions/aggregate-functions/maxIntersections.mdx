<div id="maxIntersections">
  ## maxIntersections
</div>

أُضيفت في: v20.1.0

دالة تجميع تحسب الحد الأقصى لعدد الفواصل الزمنية التي تتقاطع معًا ضمن مجموعة من الفواصل الزمنية (إذا كانت جميع الفواصل الزمنية تتقاطع مرة واحدة على الأقل).

**الصيغة**

```sql
maxIntersections(start_column, end_column)
```

**المعاملات**

* `start_column` — عمود رقمي يمثّل بداية كل فاصل زمني. إذا كانت قيمة `start_column` هي `NULL` أو 0، فسيتم تجاهل هذا الفاصل الزمني. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float)
* `end_column` — عمود رقمي يمثّل نهاية كل فاصل زمني. إذا كانت قيمة `end_column` هي `NULL` أو 0، فسيتم تجاهل هذا الفاصل الزمني. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float)

**القيمة المُعادة**

يعيد الحد الأقصى لعدد الفواصل الزمنية المتداخلة. [`UInt64`](/ar/reference/data-types/int-uint)

**أمثلة**

**حساب الحد الأقصى لعدد التداخلات**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersections(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersections(start, end)─┐
│                            3 │
└──────────────────────────────┘
```
