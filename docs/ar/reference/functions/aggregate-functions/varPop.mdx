## varPop {#varPop}

مُقدَّم في: v1.1.0

يحسب تباين المجتمع.

يُحسب تباين المجتمع وفق الصيغة:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

حيث:
- $x$ هي كل قيمة في المجتمع
- $\bar{x}$ هو متوسط المجتمع
- $n$ هو حجم المجتمع

<Note>
تستخدم هذه الدالة خوارزمية غير مستقرة عدديًا. إذا كنت بحاجة إلى [الاستقرار العددي](https://en.wikipedia.org/wiki/Numerical_stability) في الحسابات، فاستخدم الدالة [`varPopStable`](/reference/functions/aggregate-functions/varPopStable)، إذ تعمل بشكل أبطأ لكنها تُقلّل من الخطأ الحسابي.
</Note>
    

**الصياغة**

```sql
varPop(x)
```

**الأسماء البديلة**: `VAR_POP`

**الوسائط**

- `x` — مجتمع القيم المراد حساب تباينه. [`(U)Int*`](/reference/data-types/int-uint) أو [`Float*`](/reference/data-types/float) أو [`Decimal*`](/reference/data-types/decimal)

**القيمة المُعادة**

يُعيد تباين المجتمع للقيمة `x`. [`Float64`](/reference/data-types/float)

**أمثلة**

**حساب تباين المجتمع**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
