<div id="skewPop">
  ## skewPop
</div>

استُحدثت في: v20.1.0

تحسب [معامل الالتواء](https://en.wikipedia.org/wiki/Skewness) لمتتالية من القيم.

**الصيغة**

```sql
skewPop(expr)
```

**الوسائط**

* `expr` — تعبير يُرجع عددًا. [`تعبير`](/ar/reference/data-types/special-data-types/expression)

**القيمة المُعادة**

يُرجع معامل الالتواء للتوزيع المُعطى. [`Float64`](/ar/reference/data-types/float)

**أمثلة**

**توزيع متماثل**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**توزيع منحرف إلى اليمين**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
