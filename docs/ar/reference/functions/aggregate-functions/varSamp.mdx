## varSamp {#varSamp}

مُقدَّمة في: v1.1.0

تحسب تباين العينة لمجموعة بيانات.

يُحسب تباين العينة باستخدام الصيغة:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

حيث:
- $x$ هي كل نقطة بيانات منفردة في مجموعة البيانات
- $\bar{x}$ هو المتوسط الحسابي لمجموعة البيانات
- $n$ هو عدد نقاط البيانات في مجموعة البيانات

تفترض الدالة أن مجموعة البيانات المُدخلة تمثّل عينةً من مجتمع إحصائي أكبر. إذا أردت حساب تباين المجتمع الإحصائي بأكمله (حين تمتلك مجموعة البيانات الكاملة)، فاستخدم [`varPop`](/reference/functions/aggregate-functions/varPop) عوضًا عن ذلك.

<Note>
تستخدم هذه الدالة خوارزمية غير مستقرة عدديًا. إذا كنت بحاجة إلى [الاستقرار العددي](https://en.wikipedia.org/wiki/Numerical_stability) في الحسابات، فاستخدم الدالة [`varSampStable`](/reference/functions/aggregate-functions/varSampStable). تعمل بصورة أبطأ، لكنها تُقلّل من الخطأ الحسابي.
</Note>
    

**الصيغة**

```sql
varSamp(x)
```

**الأسماء البديلة**: `VAR_SAMP`

**الوسيطات**

- `x` — المجتمع الإحصائي الذي تريد حساب تباين العينة له. [`(U)Int*`](/reference/data-types/int-uint) أو [`Float*`](/reference/data-types/float) أو [`Decimal*`](/reference/data-types/decimal)

**القيمة المُعادة**

تُعيد تباين العينة لمجموعة البيانات المُدخلة `x`. [`Float64`](/reference/data-types/float)

**أمثلة**

**حساب تباين العينة**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSamp(x),3) AS var_samp FROM test_data;
```

```response title=Response
┌─var_samp─┐
│    0.865 │
└──────────┘
```
