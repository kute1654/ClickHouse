<div id="groupBitAnd">
  ## groupBitAnd
</div>

أُضيف في: v1.1.0

يُطبّق عملية AND على مستوى البِتّات على مجموعة من الأرقام.

**الصيغة**

```sql
groupBitAnd(expr)
```

**الأسماء المستعارة**: `BIT_AND`

**الوسائط**

* `expr` — تعبير من النوع `(U)Int*`. [`(U)Int*`](/ar/reference/data-types/int-uint)

**القيمة المُعادة**

يُرجع قيمة من النوع `(U)Int*`. [`(U)Int*`](/ar/reference/data-types/int-uint)

**أمثلة**

**مثال على العملية AND على مستوى البتات**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
