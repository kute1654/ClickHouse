<div id="skewSamp">
  ## skewSamp
</div>

أُضيف في: v20.1.0

يحسب [التواء العينة](https://en.wikipedia.org/wiki/Skewness) لمتتالية.

ويمثل تقديرًا غير متحيز لالتواء متغير عشوائي إذا كانت القيم المُمرَّرة تُشكّل عيّنةً منه.

**الصيغة**

```sql
skewSamp(expr)
```

**الوسائط**

* `expr` — تعبير يُرجع عددًا. [`تعبير`](/ar/reference/data-types/special-data-types/expression)

**القيمة المعادة**

تُرجع هذه الدالة معامل الالتواء للتوزيع المُعطى. إذا كان `n <= 1` (حيث إن `n` هو حجم العينة)، فستُرجع الدالة `nan`. [`Float64`](/ar/reference/data-types/float)

**أمثلة**

**توزيع متماثل**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**توزيع منحرف لليمين**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
