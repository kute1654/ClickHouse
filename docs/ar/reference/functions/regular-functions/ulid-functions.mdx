<div id="ULIDStringToDateTime">
  ## ULIDStringToDateTime
</div>

طُرحت في: v23.3.0

تستخرج هذه الدالة الطابع الزمني من [ULID](https://github.com/ulid/spec).

**البنية**

```sql
ULIDStringToDateTime(ulid[, timezone])
```

**الوسيطات**

* `ulid` — قيمة ULID المُدخلة. [`String`](/ar/reference/data-types/string) أو [`FixedString(26)`](/ar/reference/data-types/fixedstring)
* `timezone` — اختياري. اسم المنطقة الزمنية للقيمة المُعادة. [`String`](/ar/reference/data-types/string)

**القيمة المُعادة**

طابع زمني بدقة الملي ثانية. [`DateTime64(3)`](/ar/reference/data-types/datetime64)

**أمثلة**

**مثال على الاستخدام**

```sql title=Query
SELECT ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')
```

```response title=Response
┌─ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')─┐
│                            2022-12-28 00:40:37.616 │
└────────────────────────────────────────────────────┘
```

<div id="generateULID">
  ## generateULID
</div>

أُضيف في: v23.2.0

ينشئ [معرّفًا فريدًا عالميًا قابلًا للفرز معجميًا (ULID)](https://github.com/ulid/spec).

**الصيغة**

```sql
generateULID([x])
```

**الوسيطات**

* `x` — اختياري. تعبير يُنتِج أيًّا من أنواع البيانات المدعومة. تُهمَل القيمة الناتجة، ولكن يُستخدم التعبير نفسه لتجاوز [إزالة التعبيرات الفرعية الشائعة](/ar/reference/functions/regular-functions/overview#common-subexpression-elimination) إذا جرى استدعاء الدالة عدة مرات ضمن استعلام واحد. [`Any`](/ar/reference/data-types/index)

**القيمة المُعادة**

تُرجِع ULID. [`FixedString(26)`](/ar/reference/data-types/fixedstring)

**أمثلة**

**مثال على الاستخدام**

```sql title=Query
SELECT generateULID()
```

```response title=Response
┌─generateULID()─────────────┐
│ 01GNB2S2FGN2P93QPXDNB4EN2R │
└────────────────────────────┘
```

**مثال على الاستخدام عند الحاجة إلى إنشاء قيم متعددة في صف واحد**

```sql title=Query
SELECT generateULID(1), generateULID(2)
```

```response title=Response
┌─generateULID(1)────────────┬─generateULID(2)────────────┐
│ 01GNB2SGG4RHKVNT9ZGA4FFMNP │ 01GNB2SGG4V0HMQVH4VBVPSSRB │
└────────────────────────────┴────────────────────────────┘
```
