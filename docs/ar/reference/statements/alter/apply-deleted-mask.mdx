```sql
ALTER TABLE [db].name [ON CLUSTER cluster] APPLY DELETED MASK [IN PARTITION partition_id]
```

يطبّق هذا الأمر القناع الذي أنشأته [عملية الحذف الخفيف](/ar/reference/statements/delete) ويزيل قسرًا الصفوف المعلَّمة على أنها محذوفة من القرص. هذا الأمر عبارة عن mutation ثقيلة، ويعادل من حيث الدلالة الاستعلام `ALTER TABLE [db].name DELETE WHERE _row_exists = 0`.

<Note>
  لا يعمل إلا مع الجداول ضمن عائلة [`MergeTree`](/ar/reference/engines/table-engines/mergetree-family/mergetree) (بما في ذلك الجداول [replicated](/ar/reference/engines/table-engines/mergetree-family/replication)).
</Note>
