```sql
USE [DATABASE] db
```

يتيح لك تعيين قاعدة البيانات الحالية للجلسة.

تُستخدم قاعدة البيانات الحالية للعثور على الجداول إذا لم تُحدَّد قاعدة البيانات صراحةً في الاستعلام بوضعها قبل اسم الجدول وفصلها عنه بنقطة.

لا يمكن تنفيذ هذا الاستعلام عند استخدام بروتوكول HTTP، إذ لا يوجد مفهوم للجلسة.
