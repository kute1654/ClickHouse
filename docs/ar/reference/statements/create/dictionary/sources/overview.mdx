<div id="dictionary-sources">
  ## الصياغة
</div>

<CloudDetails />

يمكن ربط القاموس بـ ClickHouse من مصادر مختلفة عديدة.
يُحدَّد المصدر في قسم `source` في ملف التكوين، وباستخدام العبارة `SOURCE` في تعليمة DDL.

<Tabs>
  <Tab title="DDL">
    ```sql
    CREATE DICTIONARY dict_name (...)
    ...
    SOURCE(SOURCE_TYPE(param1 val1 ... paramN valN)) -- تكوين المصدر
    ...
    ```
  </Tab>

  <Tab title="ملف التكوين">
    ```xml
    <clickhouse>
      <dictionary>
        ...
        <source>
          <source_type>
            <!-- تكوين المصدر -->
          </source_type>
        </source>
        ...
      </dictionary>
      ...
    </clickhouse>
    ```
  </Tab>
</Tabs>

<br />
