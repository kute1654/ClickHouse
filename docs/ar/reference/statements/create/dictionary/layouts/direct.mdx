<div id="direct">
  ## direct
</div>

لا يُخزَّن القاموس في الذاكرة، بل يصل مباشرةً إلى المصدر عند معالجة الطلب.

يكون مفتاح القاموس من النوع [UInt64](/ar/reference/data-types/int-uint).

جميع أنواع [المصادر](/ar/reference/statements/create/dictionary/sources/overview#dictionary-sources) مدعومة، باستثناء الملفات المحلية.

مثال على الإعداد:

<Tabs>
  <Tab title="DDL">
    ```sql
    LAYOUT(DIRECT())
    ```
  </Tab>

  <Tab title="ملف الإعداد">
    ```xml
    <layout>
      <direct />
    </layout>
    ```
  </Tab>
</Tabs>

<br />

  ## complex_key_direct
