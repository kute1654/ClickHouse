<div id="description">
  ## الوصف
</div>

تتضمن معلومات عن أدلة المستخدمين المُهيأة — وهي الأدلة الموجودة في نظام الملفات التي يُسمح لخادم ClickHouse بقراءة البيانات التي يوفّرها المستخدم منها.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم الدليل.
* `type` ([String](/ar/reference/data-types/index)) — نوع تخزين بيانات الوصول، مثل users.xml أو replicated أو memory وما إلى ذلك.
* `params` ([String](/ar/reference/data-types/index)) — JSON يتضمن معلمات تخزين بيانات الوصول.
* `precedence` ([UInt64](/ar/reference/data-types/index)) — الترتيب الذي يُصرَّح فيه بهذا الدليل ضمن config. ويُستخدَم الترتيب نفسه عندما يحاول ClickHouse العثور على مستخدم أو دور.

{/*AUTOGENERATED_END*/}
