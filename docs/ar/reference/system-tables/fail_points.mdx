<div id="description">
  ## الوصف
</div>

يتضمن قائمة بجميع نقاط الفشل المتاحة والمسجّلة في الخادم، إلى جانب نوع كلٍّ منها وما إذا كانت مفعّلة حاليًا.

يمكن تفعيل نقاط الفشل وتعطيلها في وقت التشغيل باستخدام التعليمتين `SYSTEM ENABLE FAILPOINT` و`SYSTEM DISABLE FAILPOINT`.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم نقطة الفشل.
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](/ar/reference/data-types/index)) — نوع نقطة الفشل: &#39;once&#39; تُفعَّل مرة واحدة ثم تُعطَّل تلقائيًا، و&#39;regular&#39; تُفعَّل في كل مرة، و&#39;pauseable&#95;once&#39; يعلّق التنفيذ مرة واحدة، و&#39;pauseable&#39; يعلّق التنفيذ في كل مرة إلى أن يُستأنف.
* `enabled` ([UInt8](/ar/reference/data-types/index)) — يحدّد ما إذا كانت نقطة الفشل مفعّلة حاليًا (1) أو معطّلة (0).

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql title="Query"
SYSTEM ENABLE FAILPOINT replicated_merge_tree_insert_retry_pause;
SELECT * FROM system.fail_points WHERE enabled = 1
```
