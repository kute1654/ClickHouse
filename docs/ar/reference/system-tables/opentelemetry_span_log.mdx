<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن [مقاطع التتبّع](https://opentracing.io/docs/overview/spans/) الخاصة بالاستعلامات المُنفَّذة.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم المضيف الذي جرى فيه التقاط هذا الـ span.
* `trace_id` ([UUID](/ar/reference/data-types/uuid)) — معرّف التتبّع للاستعلام المنفَّذ.
* `span_id` ([UInt64](/ar/reference/data-types/int-uint)) — معرّف span التتبّع.
* `parent_span_id` ([UInt64](/ar/reference/data-types/int-uint)) — معرّف span التتبّع الأب.
* `operation_name` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم العملية.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/ar/reference/data-types/enum)) — قيمة SpanKind الخاصة بالـ span. INTERNAL — يشير إلى أن الـ span يمثّل عملية داخلية داخل تطبيق. SERVER — يشير إلى أن الـ span يغطي المعالجة من جهة الخادم لنداء RPC متزامن أو أي طلب بعيد آخر. CLIENT — يشير إلى أن الـ span يصف طلبًا إلى خدمة بعيدة. PRODUCER — يشير إلى أن الـ span يصف الجهة البادئة لطلب غير متزامن. وغالبًا ما ينتهي هذا الـ span الأب قبل الـ CONSUMER span الابن المقابل، وربما حتى قبل أن يبدأ الـ span الابن. CONSUMER - يشير إلى أن الـ span يصف span ابنًا لطلب PRODUCER غير متزامن.
* `start_time_us` ([UInt64](/ar/reference/data-types/int-uint)) — وقت بدء span التتبّع (بالميكروثانية).
* `finish_time_us` ([UInt64](/ar/reference/data-types/int-uint)) — وقت انتهاء span التتبّع (بالميكروثانية).
* `finish_date` ([Date](/ar/reference/data-types/date)) — تاريخ انتهاء span التتبّع.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/ar/reference/data-types/enum)) — رمز حالة الـ span.
* `status_message` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — رسالة الخطأ.
* `attribute` ([Map(LowCardinality(String), String)](/ar/reference/data-types/map)) — السمة المرتبطة بـ span التتبّع. تُملأ وفقًا للتوصيات الواردة في معيار OpenTelemetry.

**الأسماء المستعارة:**

* `attribute.names` — اسم مستعار لـ `mapKeys(attribute)`.
* `attribute.values` — اسم مستعار لـ `mapValues(attribute)`.

{/*AUTOGENERATED_END*/}
