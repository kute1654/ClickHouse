<div id="description">
  ## الوصف
</div>

يُستخدم جدول نظامي هذا لتنفيذ الاستعلام `SHOW PROCESSLIST`.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `is_initial_query` ([UInt8](/ar/reference/data-types/index)) — ما إذا كان هذا الاستعلام صادرًا مباشرةً من المستخدم أو أصدره ClickHouse server ضمن نطاق distributed query execution.
* `user` ([String](/ar/reference/data-types/index)) — المستخدم الذي نفّذ الاستعلام. ضع في اعتبارك أنه في distributed processing، تُرسَل الاستعلامات إلى الخوادم البعيدة باستخدام المستخدم default. يحتوي هذا الحقل على اسم المستخدم الخاص باستعلام معيّن، وليس الاستعلام الذي بدأه هذا الاستعلام.
* `query_id` ([String](/ar/reference/data-types/index)) — Query id، إذا كان محددًا.
* `address` ([IPv6](/ar/reference/data-types/index)) — عنوان IP الذي أُجري منه الاستعلام. وينطبق الأمر نفسه على distributed processing. لتتبّع المصدر الأصلي الذي انطلق منه distributed query، راجع system.processes على خادم مُرسِل طلب الاستعلام.
* `port` ([UInt16](/ar/reference/data-types/index)) — منفذ client الذي أُجري منه الاستعلام.
* `initial_user` ([String](/ar/reference/data-types/index)) — اسم المستخدم الذي شغّل initial query (لـ distributed query execution).
* `initial_query_id` ([String](/ar/reference/data-types/index)) — معرّف initial query (لـ distributed query execution).
* `initial_address` ([IPv6](/ar/reference/data-types/index)) — عنوان IP الذي أُطلق منه استعلام parent.
* `initial_port` ([UInt16](/ar/reference/data-types/index)) — منفذ client الذي استُخدم لإجراء استعلام parent.
* `interface` ([UInt8](/ar/reference/data-types/index)) — الواجهة التي استُخدمت لإرسال الاستعلام. TCP = 1، HTTP = 2، GRPC = 3، MYSQL = 4، POSTGRESQL = 5، LOCAL = 6، TCP&#95;INTERSERVER = 7.
* `os_user` ([String](/ar/reference/data-types/index)) — اسم مستخدم نظام التشغيل الذي يشغّل clickhouse-client.
* `client_hostname` ([String](/ar/reference/data-types/index)) — hostname جهاز client الذي يعمل عليه clickhouse-client أو أي TCP client آخر.
* `client_name` ([String](/ar/reference/data-types/index)) — اسم clickhouse-client أو أي TCP client آخر.
* `client_agent` ([LowCardinality(String)](/ar/reference/data-types/index)) — agent برمجي معتمد على الذكاء الاصطناعي استدعى client (مثل `claude-code` و`cursor`)، ويُكتشف من متغيرات البيئة. تكون القيمة فارغة إذا لم يُكتشف أي agent.
* `client_revision` ([UInt64](/ar/reference/data-types/index)) — revision الخاص بـ clickhouse-client أو أي TCP client آخر.
* `client_version_major` ([UInt64](/ar/reference/data-types/index)) — الإصدار الرئيسي لـ clickhouse-client أو أي TCP client آخر.
* `client_version_minor` ([UInt64](/ar/reference/data-types/index)) — الإصدار الثانوي لـ clickhouse-client أو أي TCP client آخر.
* `client_version_patch` ([UInt64](/ar/reference/data-types/index)) — component التصحيح من إصدار clickhouse-client أو أي TCP client آخر.
* `http_method` ([UInt8](/ar/reference/data-types/index)) — طريقة HTTP التي بدأت الاستعلام. Possible values: 0 — أُطلق الاستعلام من واجهة TCP. 1 — استُخدمت طريقة GET. 2 — استُخدمت طريقة POST.
* `http_user_agent` ([String](/ar/reference/data-types/index)) — HTTP header ‏UserAgent المُمرَّر في استعلام HTTP.
* `http_referer` ([String](/ar/reference/data-types/index)) — HTTP header ‏Referer المُمرَّر في استعلام HTTP (ويحتوي على عنوان كامل أو جزئي للصفحة التي تُجري الاستعلام).
* `forwarded_for` ([String](/ar/reference/data-types/index)) — HTTP header ‏X-Forwarded-For المُمرَّر في استعلام HTTP.
* `quota_key` ([String](/ar/reference/data-types/index)) — quota key المحدَّد في إعداد quotas (راجع keyed).
* `distributed_depth` ([UInt64](/ar/reference/data-types/index)) — عدد المرات التي أُعيد فيها إرسال الاستعلام داخليًا بين Server nodes.
* `elapsed` ([Float64](/ar/reference/data-types/index)) — الوقت بالثواني منذ بدء تنفيذ الطلب.
* `is_cancelled` ([UInt8](/ar/reference/data-types/index)) — ما إذا كان الاستعلام قد أُلغي.
* `is_all_data_sent` ([UInt8](/ar/reference/data-types/index)) — ما إذا كانت جميع البيانات قد أُرسلت إلى العميل (أي أن الاستعلام قد اكتمل على الخادم).
* `read_rows` ([UInt64](/ar/reference/data-types/index)) — عدد الصفوف المقروءة من الجدول. في المعالجة الموزعة، يكون هذا على الخادم الطالب هو الإجمالي عبر جميع الخوادم البعيدة.
* `read_bytes` ([UInt64](/ar/reference/data-types/index)) — عدد البايتات غير المضغوطة المقروءة من الجدول. في المعالجة الموزعة، يكون هذا على الخادم الطالب هو الإجمالي عبر جميع الخوادم البعيدة.
* `total_rows_approx` ([UInt64](/ar/reference/data-types/index)) — تقدير العدد الإجمالي للصفوف التي ينبغي قراءتها. في المعالجة الموزعة، يكون هذا على الخادم الطالب هو الإجمالي عبر جميع الخوادم البعيدة. ويمكن تحديثه أثناء معالجة الطلب عند اكتشاف مصادر جديدة تحتاج إلى المعالجة.
* `written_rows` ([UInt64](/ar/reference/data-types/index)) — عدد الصفوف المكتوبة إلى التخزين.
* `written_bytes` ([UInt64](/ar/reference/data-types/index)) — عدد البايتات المكتوبة إلى التخزين.
* `memory_usage` ([Int64](/ar/reference/data-types/index)) — مقدار ذاكرة RAM التي يستخدمها الاستعلام. وقد لا يشمل ذلك بعض أنواع الذاكرة المخصصة.
* `peak_memory_usage` ([Int64](/ar/reference/data-types/index)) — الذروة الحالية لاستخدام الذاكرة.
* `query` ([String](/ar/reference/data-types/index)) — نص الاستعلام. بالنسبة إلى INSERT، لا يتضمن البيانات المراد إدراجها.
* `normalized_query_hash` ([UInt64](/ar/reference/data-types/index)) — قيمة hash رقمية تكون متطابقة، مثلًا، للاستعلامات التي لا تختلف إلا في قيم الثوابت.
* `query_kind` ([String](/ar/reference/data-types/index)) — نوع الاستعلام، مثل SELECT أو INSERT أو غير ذلك.
* `thread_ids` ([Array(UInt64)](/ar/reference/data-types/index)) — قائمة معرّفات جميع سلاسل التنفيذ التي شاركت في هذا الاستعلام.
* `peak_threads_usage` ([UInt64](/ar/reference/data-types/index)) — الحد الأقصى لعدد سلاسل التنفيذ المتزامنة التي نفّذت الاستعلام.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/ar/reference/data-types/index)) — قيم ProfileEvents المحسوبة لهذا الاستعلام.
* `Settings` ([Map(LowCardinality(String), LowCardinality(String))](/ar/reference/data-types/index)) — قائمة إعدادات المستخدم المعدّلة.
* `current_database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات الحالية.
* `is_internal` ([UInt8](/ar/reference/data-types/index)) — يشير إلى ما إذا كان هذا استعلامًا مساعدًا نُفِّذ داخليًا.

{/*AUTOGENERATED_END*/}
