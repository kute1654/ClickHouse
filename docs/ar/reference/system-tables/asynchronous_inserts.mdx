<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن عمليات الإدراج غير المتزامنة المعلّقة في قائمة الانتظار.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ar/reference/data-types/index)) — نص الاستعلام.
* `database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات.
* `table` ([String](/ar/reference/data-types/index)) — اسم الجدول.
* `format` ([String](/ar/reference/data-types/index)) — اسم التنسيق.
* `first_update` ([DateTime64(6)](/ar/reference/data-types/index)) — وقت أول عملية إدراج بدقة الميكروثانية.
* `total_bytes` ([UInt64](/ar/reference/data-types/index)) — إجمالي عدد البايتات المنتظرة في قائمة الانتظار.
* `entries.query_id` ([Array(String)](/ar/reference/data-types/index)) — مصفوفة معرّفات الاستعلام لعمليات الإدراج المنتظرة في قائمة الانتظار.
* `entries.bytes` ([Array(UInt64)](/ar/reference/data-types/index)) — مصفوفة بعدد البايتات لكل استعلام إدراج منتظر في قائمة الانتظار.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql title="Query"
SELECT * FROM system.asynchronous_inserts LIMIT 1 \G;
```

