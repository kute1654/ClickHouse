  ## الوصف
</div>

يحتوي على قائمة برموز الإكمال.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `word` ([String](/ar/reference/data-types/index)) — رمز الإكمال.
* `context` ([String](/ar/reference/data-types/index)) — نوع الكيان المرتبط بالرمز (مثل جدول).
* `belongs` ([Nullable(String)](/ar/reference/data-types/index)) — الكيان الذي ينتمي إليه هذا الرمز (مثل اسم قاعدة البيانات المالكة).
