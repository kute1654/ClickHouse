<div id="description">
  ## الوصف
</div>

يتضمن قائمة بمحركات قواعد البيانات التي يدعمها الخادم.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم محرك قاعدة البيانات.
* `description` ([String](/ar/reference/data-types/index)) — وصف عام لوظيفة محرك قاعدة البيانات.
* `syntax` ([String](/ar/reference/data-types/index)) — كيفية تحديد محرك قاعدة البيانات في العبارة ENGINE ضمن استعلام CREATE DATABASE.
* `examples` ([String](/ar/reference/data-types/index)) — أمثلة استخدام.
* `introduced_in` ([String](/ar/reference/data-types/index)) — إصدار ClickHouse الذي طُرح فيه محرك قاعدة البيانات لأول مرة، بصيغة major.minor.
* `related` ([Array(String)](/ar/reference/data-types/index)) — أسماء محركات قواعد البيانات ذات الصلة.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```
