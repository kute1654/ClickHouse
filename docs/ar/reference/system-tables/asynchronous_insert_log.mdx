<div id="description">
  ## الوصف
</div>

يتضمن معلومات حول عمليات الإدراج غير المتزامنة. ويمثل كل إدخال استعلام إدراج جرى تخزينه مؤقتًا ضمن استعلام إدراج غير متزامن.

لبدء التسجيل، اضبط المعلمات في قسم [asynchronous&#95;insert&#95;log](/ar/reference/settings/server-settings/settings#asynchronous_insert_log).

تُضبط فترة تفريغ البيانات في المعلمة `flush_interval_milliseconds` ضمن قسم إعدادات الخادم [asynchronous&#95;insert&#95;log](/ar/reference/settings/server-settings/settings#asynchronous_insert_log). ولإجبار التفريغ، استخدم الاستعلام [SYSTEM FLUSH LOGS](/ar/reference/statements/system#flush-logs).

لا يحذف ClickHouse البيانات من الجدول تلقائيًا. راجع [المقدمة](/ar/reference/system-tables/overview#system-tables-introduction) لمزيد من التفاصيل.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم مضيف الخادم الذي ينفّذ الاستعلام.
* `event_date` ([Date](/ar/reference/data-types/date)) — التاريخ الذي حدث فيه الإدراج غير المتزامن.
* `event_time` ([DateTime](/ar/reference/data-types/datetime)) — التاريخ والوقت اللذان انتهى عندهما تنفيذ الإدراج غير المتزامن.
* `event_time_microseconds` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — التاريخ والوقت اللذان انتهى عندهما تنفيذ الإدراج غير المتزامن بدقة الميكروثانية.
* `query` ([String](/ar/reference/data-types/string)) — نص الاستعلام.
* `database` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم قاعدة البيانات التي يوجد فيها الجدول.
* `table` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم الجدول.
* `format` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم التنسيق.
* `query_id` ([String](/ar/reference/data-types/string)) — معرّف الاستعلام الأوّلي.
* `bytes` ([UInt64](/ar/reference/data-types/int-uint)) — عدد البايتات المُدرجة.
* `rows` ([UInt64](/ar/reference/data-types/int-uint)) — عدد الصفوف المُدرجة.
* `exception` ([String](/ar/reference/data-types/string)) — رسالة الاستثناء.
* `status` ([Enum8(&#39;Ok&#39; = 0, &#39;ParsingError&#39; = 1, &#39;FlushError&#39; = 2)](/ar/reference/data-types/enum)) — حالة الإدراج. القيم: &#39;Ok&#39; = 0 — إدراج ناجح، &#39;ParsingError&#39; = 1 — استثناء أثناء تحليل البيانات، &#39;FlushError&#39; = 2 — استثناء أثناء تفريغ البيانات.
* `data_kind` ([Enum8(&#39;Parsed&#39; = 0, &#39;Preprocessed&#39; = 1)](/ar/reference/data-types/enum)) — حالة البيانات. القيم: &#39;Parsed&#39; و&#39;Preprocessed&#39;.
* `flush_time` ([DateTime](/ar/reference/data-types/datetime)) — التاريخ والوقت اللذان حدث عندهما التفريغ.
* `flush_time_microseconds` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — التاريخ والوقت اللذان حدث عندهما التفريغ بدقة الميكروثانية.
* `flush_query_id` ([String](/ar/reference/data-types/string)) — معرّف استعلام التفريغ.
* `timeout_milliseconds` ([UInt64](/ar/reference/data-types/int-uint)) — المهلة المتكيّفة المحسوبة لهذا الإدخال.

{/*AUTOGENERATED_END*/}
