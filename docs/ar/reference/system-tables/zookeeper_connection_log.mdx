<div id="description">
  ## الوصف
</div>

يعرض الجدول `system.zookeeper_connection_log` سجلّ اتصالات ZooKeeper (بما في ذلك مثيلات ZooKeeper الإضافية). ويعرض كل صف معلومات عن حدث واحد يتعلق بهذه الاتصالات.

<Note>
  لا يتضمن الجدول أحداث انقطاع الاتصال الناتجة عن إيقاف تشغيل الخادم.
</Note>

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم مضيف الخادم الذي تم الاتصال بـ ZooKeeper منه أو قطع الاتصال عنه.
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/ar/reference/data-types/enum)) — نوع الحدث. القيم الممكنة: Connected، Disconnected.
* `event_date` ([Date](/ar/reference/data-types/date)) — تاريخ السجل.
* `event_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت السجل
* `event_time_microseconds` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت السجل بدقة الميكروثانية.
* `name` ([String](/ar/reference/data-types/string)) — اسم مجموعة ZooKeeper.
* `host` ([String](/ar/reference/data-types/string)) — اسم المضيف/عنوان IP لعقدة ZooKeeper التي اتصل بها ClickHouse أو انقطع عنها.
* `port` ([UInt16](/ar/reference/data-types/int-uint)) — منفذ عقدة ZooKeeper التي اتصل بها ClickHouse أو انقطع عنها.
* `index` ([UInt8](/ar/reference/data-types/int-uint)) — فهرس عقدة ZooKeeper التي اتصل بها ClickHouse أو انقطع عنها. هذا الفهرس مأخوذ من إعدادات ZooKeeper.
* `client_id` ([Int64](/ar/reference/data-types/int-uint)) — معرّف الجلسة الخاصة بالاتصال.
* `keeper_api_version` ([UInt8](/ar/reference/data-types/int-uint)) — إصدار واجهة برمجة تطبيقات Keeper.
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/ar/reference/data-types/array)) — إشارات الميزات المُمكّنة. ينطبق هذا فقط على ClickHouse Keeper.
* `availability_zone` ([String](/ar/reference/data-types/string)) — نطاق التوافر
* `reason` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — سبب الاتصال أو انقطاع الاتصال.

{/*AUTOGENERATED_END*/}
