<div id="description">
  ## الوصف
</div>

يتضمن معلومات حول [الأدوار](/ar/concepts/features/security/access-rights#role-management) التي تم تكوينها.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم الدور.
* `id` ([UUID](/ar/reference/data-types/index)) — معرّف الدور.
* `storage` ([String](/ar/reference/data-types/index)) — مسار تخزين الأدوار. يُضبط في المعلَمة `access_control_path`.
