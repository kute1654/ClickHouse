يحتوي على إدخالات سجل تتضمن معلومات عن الملفات التي عالجها محرّك AzureQueue.

يمكن تنفيذ truncate أو drop لهذا الجدول بأمان في أي وقت.

الأعمدة:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم المضيف
* `event_date` ([Date](/ar/reference/data-types/date)) — تاريخ الحدث عند كتابة صف السجل هذا
* `event_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت الحدث عند كتابة صف السجل هذا
* `database` ([String](/ar/reference/data-types/string)) — اسم قاعدة البيانات التي يوجد فيها جدول queue (`S3Queue` أو `AzureQueue`).
* `table` ([String](/ar/reference/data-types/string)) — اسم جدول queue (`S3Queue` أو `AzureQueue`).
* `uuid` ([String](/ar/reference/data-types/string)) — المعرّف الفريد العالمي (UUID) لجدول queue (`S3Queue` أو `AzureQueue`).
* `file_name` ([String](/ar/reference/data-types/string)) — اسم الملف قيد المعالجة.
* `rows_processed` ([UInt64](/ar/reference/data-types/int-uint)) — عدد الصفوف التي تمت معالجتها.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/ar/reference/data-types/enum)) — حالة الملف قيد المعالجة.
* `processing_start_time` ([Nullable(DateTime)](/ar/reference/data-types/nullable)) — وقت بدء معالجة الملف.
* `processing_end_time` ([Nullable(DateTime)](/ar/reference/data-types/nullable)) — وقت انتهاء معالجة الملف.
* `exception` ([String](/ar/reference/data-types/string)) — رسالة الاستثناء في حال حدوثه.
* `commit_id` ([UInt64](/ar/reference/data-types/int-uint)) — معرّف transaction التي تم فيها تثبيت هذا الملف.
* `commit_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت تثبيت الملف في Keeper (سواء كانت حالته failed أو processed).
* `transaction_start_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت بدء transaction المعالجة بالكامل.
* `get_object_time_ms` ([UInt64](/ar/reference/data-types/int-uint)) — الوقت الذي استغرقه العثور على object في object storage.
