<div id="description">
  ## الوصف
</div>

يحتوي على معلومات عن جميع أحداث تسجيل الدخول وتسجيل الخروج الناجحة والفاشلة.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم مضيف الخادم الذي ينفّذ الاستعلام.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/ar/reference/data-types/enum)) — نتيجة تسجيل الدخول/الخروج. القيم الممكنة: LoginFailure — خطأ في تسجيل الدخول. LoginSuccess — تم تسجيل الدخول بنجاح. Logout — تسجيل الخروج من النظام.
* `auth_id` ([UUID](/ar/reference/data-types/uuid)) — معرّف المصادقة، وهو UUID يُنشأ تلقائيًا في كل مرة يسجّل فيها المستخدم الدخول.
* `session_id` ([String](/ar/reference/data-types/string)) — معرّف الجلسة الذي يمرّره العميل عبر واجهة HTTP.
* `event_date` ([Date](/ar/reference/data-types/date)) — تاريخ تسجيل الدخول/الخروج.
* `event_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت تسجيل الدخول/الخروج.
* `event_time_microseconds` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت بدء تسجيل الدخول/الخروج بدقة الميكروثانية.
* `user` ([Nullable(String)](/ar/reference/data-types/nullable)) — اسم المستخدم.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/ar/reference/data-types/nullable)) — نوع المصادقة.
* `profiles` ([Array(LowCardinality(String))](/ar/reference/data-types/array)) — قائمة ملفات التعريف المعيّنة لجميع الأدوار و/أو المستخدمين.
* `roles` ([Array(LowCardinality(String))](/ar/reference/data-types/array)) — قائمة الأدوار التي يُطبَّق عليها ملف التعريف.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/ar/reference/data-types/array)) — الإعدادات التي جرى تغييرها عند تسجيل دخول العميل أو خروجه.
* `client_address` ([IPv6](/ar/reference/data-types/ipv6)) — عنوان IP الذي استُخدم لتسجيل الدخول/الخروج.
* `client_port` ([UInt16](/ar/reference/data-types/int-uint)) — منفذ العميل الذي استُخدم لتسجيل الدخول/الخروج.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/ar/reference/data-types/enum)) — الواجهة التي بدأ منها تسجيل الدخول.
* `client_hostname` ([String](/ar/reference/data-types/string)) — اسم مضيف جهاز العميل الذي يعمل عليه `clickhouse-client` أو عميل TCP آخر.
* `client_name` ([String](/ar/reference/data-types/string)) — اسم `clickhouse-client` أو عميل TCP آخر.
* `client_revision` ([UInt32](/ar/reference/data-types/int-uint)) — رقم مراجعة `clickhouse-client` أو عميل TCP آخر.
* `client_version_major` ([UInt32](/ar/reference/data-types/int-uint)) — الإصدار الرئيسي لـ `clickhouse-client` أو لعميل TCP آخر.
* `client_version_minor` ([UInt32](/ar/reference/data-types/int-uint)) — الإصدار الثانوي لـ `clickhouse-client` أو لعميل TCP آخر.
* `client_version_patch` ([UInt32](/ar/reference/data-types/int-uint)) — مكوّن التصحيح في إصدار `clickhouse-client` أو عميل TCP آخر.
* `failure_reason` ([String](/ar/reference/data-types/string)) — رسالة الاستثناء التي تتضمن سبب فشل تسجيل الدخول/الخروج.
* `certificate_subjects` ([Array(LowCardinality(String))](/ar/reference/data-types/array)) — قائمة الموضوعات (الاسم الشائع وSubject Alternative Names) في شهادة عميل TLS المقدَّمة عبر الاتصال، بالصيغة &#39;CN:...&#39; / &#39;SAN:...&#39;. تكون فارغة إذا لم تُقدَّم أي شهادة.
* `certificate_serial` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — الرقم التسلسلي لشهادة عميل TLS. تكون فارغة إذا لم تُقدَّم أي شهادة.
* `certificate_issuer` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — الجهة المُصدِرة لشهادة عميل TLS. تكون فارغة إذا لم تُقدَّم أي شهادة.
* `certificate_not_before` ([Nullable(DateTime64)](/ar/reference/data-types/nullable)) — الوقت الذي تبدأ منه صلاحية شهادة عميل TLS. تكون NULL إذا لم تُقدَّم أي شهادة.
* `certificate_not_after` ([Nullable(DateTime64)](/ar/reference/data-types/nullable)) — الوقت الذي تنتهي بعده صلاحية شهادة عميل TLS. تكون NULL إذا لم تُقدَّم أي شهادة.

{/*AUTOGENERATED_END*/}
