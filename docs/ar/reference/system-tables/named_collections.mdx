<div id="description">
  ## الوصف
</div>

يتضمن قائمة بجميع المجموعات المسماة التي أُنشئت عبر استعلام SQL أو جرى تحليلها من ملف التهيئة.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم المجموعة.
* `collection` ([Map(String, String)](/ar/reference/data-types/index)) — المكوّنات الداخلية للمجموعة.
* `source` ([String](/ar/reference/data-types/index)) — مصدر المجموعة المُسمّاة.
* `create_query` ([String](/ar/reference/data-types/index)) — استعلام إنشاء المجموعة المُسمّاة.

{/*AUTOGENERATED_END*/}
