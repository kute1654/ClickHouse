<div id="description">
  ## الوصف
</div>

يعرض هذا الجدول تحذيرات بشأن خادم ClickHouse.
تُجمَع التحذيرات من النوع نفسه في تحذير واحد.
على سبيل المثال، إذا تجاوز العدد N من قواعد البيانات المرفقة حدًا T قابلًا للتهيئة، فسيُعرض إدخال واحد يتضمن القيمة الحالية N بدلًا من N إدخالات منفصلة.
إذا انخفضت القيمة الحالية دون الحد، يُزال الإدخال من الجدول.

يمكن ضبط الجدول باستخدام الإعدادات التالية:

* [max&#95;table&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_table_num_to_warn)
* [max&#95;database&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_database_num_to_warn)
* [max&#95;dictionary&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_dictionary_num_to_warn)
* [max&#95;view&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_view_num_to_warn)
* [max&#95;part&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_part_num_to_warn)
* [max&#95;pending&#95;mutations&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_pending_mutations_to_warn)
* [max&#95;pending&#95;mutations&#95;execution&#95;time&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_pending_mutations_execution_time_to_warn)
* [max&#95;named&#95;collection&#95;num&#95;to&#95;warn](/ar/reference/settings/server-settings/settings#max_named_collection_num_to_warn)
* [resource&#95;overload&#95;warnings](/ar/concepts/features/configuration/settings/server-overload#resource-overload-warnings)

<div id="columns">
  ## الأعمدة
</div>

* `message` ([String](/ar/reference/data-types/string)) — رسالة تحذير.
* `message_format_string` ([LowCardinality(String)](/ar/reference/data-types/string)) — سلسلة التنسيق المستخدمة في تنسيق الرسالة.

<div id="example">
  ## مثال
</div>

```sql title="Query"
 SELECT * FROM system.warnings LIMIT 2 \G;
```

```text title="Response"
Row 1:
──────
message:               The number of active parts is more than 10.
message_format_string: The number of active parts is more than {}.

Row 2:
──────
message:               The number of attached databases is more than 2.
message_format_string: The number of attached databases is more than {}.
```
