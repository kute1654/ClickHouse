<div id="description">
  ## الوصف
</div>

يتضمن قائمة بالمناطق الزمنية التي يدعمها خادم ClickHouse. وقد تختلف هذه القائمة من المناطق الزمنية حسب إصدار ClickHouse.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/ar/reference/data-types/index)) — قائمة بالمناطق الزمنية المدعومة.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
