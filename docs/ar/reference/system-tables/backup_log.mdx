<div id="description">
  ## الوصف
</div>

يحتوي على إدخالات السجل التي تتضمن معلومات عن عمليتَي `BACKUP` و`RESTORE`.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم مضيف الخادم الذي ينفّذ الاستعلام.
* `event_date` ([Date](/ar/reference/data-types/date)) — تاريخ الإدخال.
* `event_time` ([DateTime](/ar/reference/data-types/datetime)) — وقت الإدخال.
* `event_time_microseconds` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت الإدخال بدقة الميكروثانية.
* `id` ([String](/ar/reference/data-types/string)) — معرّف عملية النسخ الاحتياطي أو الاستعادة.
* `name` ([String](/ar/reference/data-types/string)) — اسم وحدة تخزين النسخة الاحتياطية (محتويات عبارة FROM أو TO).
* `base_backup_name` ([String](/ar/reference/data-types/string)) — اسم النسخة الاحتياطية الأساسية في حالة النسخة الاحتياطية التزايدية.
* `query_id` ([String](/ar/reference/data-types/string)) — معرّف الاستعلام المرتبط بعملية نسخ احتياطي.
* `status` ([Enum8(&#39;CREATING&#95;BACKUP&#39; = 0, &#39;BACKUP&#95;CREATED&#39; = 1, &#39;BACKUP&#95;FAILED&#39; = 2, &#39;RESTORING&#39; = 3, &#39;RESTORED&#39; = 4, &#39;RESTORE&#95;FAILED&#39; = 5, &#39;BACKUP&#95;CANCELLED&#39; = 6, &#39;RESTORE&#95;CANCELLED&#39; = 7)](/ar/reference/data-types/enum)) — حالة العملية.
* `error` ([String](/ar/reference/data-types/string)) — رسالة خطأ العملية الفاشلة (سلسلة فارغة للعمليات الناجحة).
* `start_time` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت بدء العملية.
* `end_time` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت انتهاء العملية.
* `num_files` ([UInt64](/ar/reference/data-types/int-uint)) — عدد الملفات المخزنة في النسخة الاحتياطية.
* `total_size` ([UInt64](/ar/reference/data-types/int-uint)) — الحجم الإجمالي للملفات المخزنة في النسخة الاحتياطية.
* `num_entries` ([UInt64](/ar/reference/data-types/int-uint)) — عدد العناصر في النسخة الاحتياطية، أي عدد الملفات داخل المجلد إذا كانت النسخة الاحتياطية مخزنة كمجلد، أو عدد الملفات داخل الأرشيف إذا كانت النسخة الاحتياطية مخزنة كأرشيف. ولا يساوي `num_files` إذا كانت النسخة الاحتياطية تزايدية أو إذا كانت تحتوي على ملفات فارغة أو مكررة. وينطبق ما يلي دائمًا: `num_entries ≤ num_files`.
* `uncompressed_size` ([UInt64](/ar/reference/data-types/int-uint)) — الحجم غير المضغوط للنسخة الاحتياطية.
* `compressed_size` ([UInt64](/ar/reference/data-types/int-uint)) — الحجم المضغوط للنسخة الاحتياطية. إذا لم تكن النسخة الاحتياطية مخزنة كأرشيف، فإنه يساوي uncompressed&#95;size.
* `files_read` ([UInt64](/ar/reference/data-types/int-uint)) — عدد الملفات المقروءة أثناء عملية الاستعادة.
* `bytes_read` ([UInt64](/ar/reference/data-types/int-uint)) — الحجم الإجمالي للملفات المقروءة أثناء عملية الاستعادة.

{/*AUTOGENERATED_END*/}
