<div id="description">
  ## الوصف
</div>

تتضمن قائمة بجميع دوال الجداول المتاحة وأوصافها.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم دالة الجدول.
* `description` ([String](/ar/reference/data-types/index)) — وصف موجز لدالة الجدول.
* `allow_readonly` ([UInt8](/ar/reference/data-types/index)) — علامة تشير إلى ما إذا كان يمكن لمستخدم للقراءة فقط استخدام هذه الدالة.

{/*AUTOGENERATED_END*/}
