<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن الإعدادات العامة للخادم، المحددة في `config.xml`.
يعرض الجدول حاليًا الإعدادات من المستوى الأول فقط في `config.xml`، ولا يدعم ملفات التهيئة المتداخلة (مثل [logger](/ar/reference/settings/server-settings/settings#logger)).

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم إعداد الخادم.
* `value` ([String](/ar/reference/data-types/index)) — قيمة إعداد الخادم.
* `default` ([String](/ar/reference/data-types/index)) — القيمة الافتراضية لإعداد الخادم.
* `changed` ([UInt8](/ar/reference/data-types/index)) — يوضح ما إذا كان الإعداد محددًا في config.xml
* `description` ([String](/ar/reference/data-types/index)) — وصف مختصر لإعداد الخادم.
* `type` ([String](/ar/reference/data-types/index)) — نوع قيمة إعداد الخادم.
* `changeable_without_restart` ([Enum8(&#39;No&#39; = 0, &#39;IncreaseOnly&#39; = 1, &#39;DecreaseOnly&#39; = 2, &#39;Yes&#39; = 3)](/ar/reference/data-types/index)) — يوضح ما إذا كان يمكن تغيير الإعداد في وقت التشغيل.
* `is_obsolete` ([UInt8](/ar/reference/data-types/index)) — يوضح ما إذا كان الإعداد قديمًا.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

يوضح المثال التالي كيفية الحصول على معلومات عن إعدادات الخادم التي يتضمن اسمها `thread_pool`.

```sql
SELECT *
FROM system.server_settings
WHERE name LIKE '%thread_pool%'
```

مما إذا كانت الإعدادات في ملفات التهيئة قد حُمِّلت بشكل صحيح وتُستخدم بالفعل.
