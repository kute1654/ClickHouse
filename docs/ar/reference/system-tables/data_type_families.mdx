<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن [أنواع البيانات](/ar/reference/data-types/index).

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم نوع البيانات.
* `case_insensitive` ([UInt8](/ar/reference/data-types/index)) — خاصية توضّح ما إذا كان يمكن استخدام اسم نوع البيانات في استعلام دون حساسية لحالة الأحرف أم لا. على سبيل المثال، كلٌّ من `Date` و`date` صحيح.
* `alias_to` ([String](/ar/reference/data-types/index)) — اسم نوع البيانات الذي يُعدّ `name` اسمًا مستعارًا له.
* `description` ([String](/ar/reference/data-types/index)) — وصف عام لنوع البيانات.
* `syntax` ([String](/ar/reference/data-types/index)) — كيفية كتابة نوع البيانات في استعلام.
* `examples` ([String](/ar/reference/data-types/index)) — أمثلة على الاستخدام.
* `introduced_in` ([String](/ar/reference/data-types/index)) — إصدار ClickHouse الذي طُرح فيه نوع البيانات لأول مرة، بصيغة major.minor.
* `related` ([Array(String)](/ar/reference/data-types/index)) — أسماء أنواع البيانات ذات الصلة.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT name, case_insensitive, alias_to FROM system.data_type_families WHERE alias_to = 'String'
```

```text
│ CHAR       │                1 │ String   │
│ MEDIUMTEXT │                1 │ String   │
└────────────┴──────────────────┴──────────┘
```
