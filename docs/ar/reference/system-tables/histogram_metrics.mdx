<div id="description">
  ## الوصف
</div>

يحتوي هذا الجدول على مقاييس مدرج تكراري يمكن حسابها فورًا وتصديرها بتنسيق Prometheus. وهو محدَّث دائمًا. ويحل محل `system.latency_log` المتقادِم.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ar/reference/data-types/index)) — اسم المقياس.
* `value` ([Float64](/ar/reference/data-types/index)) — قيمة المقياس.
* `description` ([String](/ar/reference/data-types/index)) — وصف المقياس.
* `labels` ([Map(String, String)](/ar/reference/data-types/index)) — وسوم المقياس.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

يمكنك استخدام استعلام مثل هذا لتصدير جميع مقاييس المدرج التكراري بتنسيق Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
