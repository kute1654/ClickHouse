<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن [الموارد](/ar/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage) الموجودة على الخادم المحلي. ويحتوي الجدول على صف لكل مورد.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم هذا المورد.
* `read_disks` ([Array(String)](/ar/reference/data-types/index)) — قائمة بأسماء الأقراص التي تستخدم هذا المورد لعمليات القراءة.
* `write_disks` ([Array(String)](/ar/reference/data-types/index)) — قائمة بأسماء الأقراص التي تستخدم هذا المورد لعمليات الكتابة.
* `unit` ([String](/ar/reference/data-types/index)) — وحدة المورد المستخدمة في قياس التكلفة.
* `create_query` ([String](/ar/reference/data-types/index)) — استعلام CREATE للمورد.

{/*AUTOGENERATED_END*/}

مثال:

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
Row 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

Row 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```
