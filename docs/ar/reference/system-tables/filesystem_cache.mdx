<div id="description">
  ## الوصف
</div>

يحتوي على معلومات عن جميع العناصر الموجودة في ذاكرة التخزين المؤقت لنظام الملفات الخاصة بالكائنات البعيدة.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `cache_name` ([String](/ar/reference/data-types/index)) — اسم كائن ذاكرة التخزين المؤقت. يمكن استخدامه في الأمرين `SYSTEM DESCRIBE FILESYSTEM CACHE <name>` و`SYSTEM DROP FILESYSTEM CACHE <name>`
* `cache_base_path` ([String](/ar/reference/data-types/index)) — المسار إلى الدليل الأساسي الذي تُخزَّن فيه جميع ملفات ذاكرة التخزين المؤقت (لذاكرة التخزين المؤقت المعرَّفة بواسطة `cache_name`).
* `cache_path` ([String](/ar/reference/data-types/index)) — المسار إلى ملف معيّن في ذاكرة التخزين المؤقت، ويقابل file segment في ملف مصدر
* `key` ([String](/ar/reference/data-types/index)) — مفتاح ذاكرة التخزين المؤقت الخاص بـ file segment
* `file_segment_range_begin` ([UInt64](/ar/reference/data-types/index)) — الإزاحة المقابلة لبداية نطاق file segment
* `file_segment_range_end` ([UInt64](/ar/reference/data-types/index)) — الإزاحة المقابلة لنهاية نطاق file segment (شاملة)
* `size` ([UInt64](/ar/reference/data-types/index)) — حجم file segment
* `state` ([String](/ar/reference/data-types/index)) — حالة file segment ‏(DOWNLOADED, DOWNLOADING, PARTIALLY&#95;DOWNLOADED, ...)
* `finished_download_time` ([DateTime](/ar/reference/data-types/index)) — الوقت الذي اكتمل فيه تنزيل file segment.
* `cache_hits` ([UInt64](/ar/reference/data-types/index)) — عدد مرات العثور على file segment المقابل في ذاكرة التخزين المؤقت
* `references` ([UInt64](/ar/reference/data-types/index)) — عدد المراجع إلى file segment المقابل. تعني القيمة 1 أنه لا أحد يستخدمه حاليًا (والمرجع الوحيد الموجود هو في تخزين ذاكرة التخزين المؤقت نفسه)
* `downloaded_size` ([UInt64](/ar/reference/data-types/index)) — الحجم الذي تم تنزيله من file segment
* `kind` ([String](/ar/reference/data-types/index)) — نوع file segment (يُستخدم للتمييز بين file segments المضافة كجزء من &#39;البيانات المؤقتة في ذاكرة التخزين المؤقت&#39;)
* `unbound` ([UInt8](/ar/reference/data-types/index)) — علامة تنفيذ داخلية
* `user_id` ([String](/ar/reference/data-types/index)) — معرّف المستخدم الذي أنشأ file segment
* `segment_type` ([String](/ar/reference/data-types/index)) — نوع المقطع. يُستخدم للفصل بين data files‏ (`.json`, `.txt` وما إلى ذلك) وdata files‏ (`.bin`, mark files).
* `file_size` ([Nullable(UInt64)](/ar/reference/data-types/index)) — حجم الملف الذي ينتمي إليه file segment الحالي
