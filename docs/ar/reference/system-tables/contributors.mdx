<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن المساهمين. يكون الترتيب عشوائيًا عند تنفيذ الاستعلام.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم المساهم (المؤلف) في سجل Git.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.contributors LIMIT 10
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
│ Max Vetrov       │
│ LiuYangkuan      │
│ svladykin        │
│ zamulla          │
│ Šimon Podlipský  │
│ BayoNet          │
│ Ilya Khomutov    │
│ Amy Krishnevsky  │
│ Loud_Scream      │
└──────────────────┘
```

للعثور بنفسك على بياناتك في الجدول، استخدم الاستعلام التالي:

```sql
SELECT * FROM system.contributors WHERE name = 'Olga Khvostikova'
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
└──────────────────┘
```
