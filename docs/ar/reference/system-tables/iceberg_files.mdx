<div id="description">
  ## الوصف
</div>

يحتوي جدول النظام هذا على البيانات الوصفية الخاصة بكل ملف في جداول Iceberg الموجودة في ClickHouse، بحيث يقابل كل ملف بيانات أو حذف مشار إليه في اللقطة الحالية لكل جدول صفٌّ واحد. وسيكون هذا الجدول فارغًا إذا لم تكن لديك أي جداول Iceberg في ClickHouse.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات.
* `table` ([String](/ar/reference/data-types/index)) — اسم الجدول.
* `snapshot_id` ([Int64](/ar/reference/data-types/index)) — معرّف اللقطة التي أُضيف الملف عندها.
* `content` ([Enum8(&#39;DATA&#39; = 0, &#39;POSITION&#95;DELETE&#39; = 1, &#39;EQUALITY&#95;DELETE&#39; = 2)](/ar/reference/data-types/index)) — نوع محتوى الملف.
* `file_path` ([String](/ar/reference/data-types/index)) — مسار التخزين النهائي للملف.
* `file_format` ([String](/ar/reference/data-types/index)) — تنسيق الملف، مثل &#39;PARQUET&#39;.
* `record_count` ([Int64](/ar/reference/data-types/index)) — عدد السجلات في الملف.
* `file_size_in_bytes` ([Int64](/ar/reference/data-types/index)) — حجم الملف بالبايت.
* `partition` ([String](/ar/reference/data-types/index)) — التمثيل النصي لـ tuple الخاص بالتقسيم.
* `schema_id` ([Int32](/ar/reference/data-types/index)) — معرّف الـschema المحدَّد لإدخال الـmanifest هذا.
* `sequence_number` ([Int64](/ar/reference/data-types/index)) — رقم التسلسل المحدَّد لإدخال الـmanifest (ويكون دائمًا 0 للتنسيق v1).
* `sort_order_id` ([Nullable(Int32)](/ar/reference/data-types/index)) — معرّف ترتيب الفرز للملف، إذا كان محددًا.
* `null_value_counts` ([Map(Int32, Int64)](/ar/reference/data-types/index)) — عدد قيم NULL لكل عمود (`column id` -&gt; العدد).
* `column_sizes` ([Map(Int32, Int64)](/ar/reference/data-types/index)) — الحجم على القرص لكل عمود بالبايت (`column id` -&gt; البايت).
* `value_counts` ([Map(Int32, Int64)](/ar/reference/data-types/index)) — العدد الإجمالي للقيم لكل عمود (`column id` -&gt; العدد).
* `equality_ids` ([Array(Int32)](/ar/reference/data-types/index)) — معرّفات الحقول الخاصة بالمساواة لملفات حذف المساواة (وتكون فارغة للملفات التي لا تستخدم حذف المساواة).

{/*AUTOGENERATED_END*/}
