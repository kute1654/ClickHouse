<div id="description">
  ## الوصف
</div>

يتضمن معلومات حول [استعلامات DDL الموزعة (عبارة ON CLUSTER)](/ar/reference/statements/distributed-ddl) التي نُفِّذت على مستوى العنقود.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/ar/reference/data-types/index)) — معرّف الاستعلام.
* `entry_version` ([Nullable(UInt8)](/ar/reference/data-types/index)) — إصدار السجل.
* `initiator_host` ([Nullable(String)](/ar/reference/data-types/index)) — المضيف الذي بدأ عملية DDL.
* `initiator_port` ([Nullable(UInt16)](/ar/reference/data-types/index)) — المنفذ الذي يستخدمه المضيف البادئ.
* `cluster` ([String](/ar/reference/data-types/index)) — اسم العنقود، ويكون فارغًا إذا لم يتم تحديده.
* `query` ([String](/ar/reference/data-types/index)) — الاستعلام الذي نُفِّذ.
* `settings` ([Map(String, String)](/ar/reference/data-types/index)) — الإعدادات المستخدمة في عملية DDL.
* `query_create_time` ([DateTime](/ar/reference/data-types/index)) — وقت إنشاء الاستعلام.
* `host` ([Nullable(String)](/ar/reference/data-types/index)) — اسم المضيف.
* `port` ([Nullable(UInt16)](/ar/reference/data-types/index)) — منفذ المضيف.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/ar/reference/data-types/index)) — حالة الاستعلام.
* `exception_code` ([Nullable(UInt16)](/ar/reference/data-types/index)) — رمز الاستثناء.
* `exception_text` ([Nullable(String)](/ar/reference/data-types/index)) — رسالة الاستثناء.
* `query_finish_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — وقت انتهاء الاستعلام.
* `query_duration_ms` ([Nullable(UInt64)](/ar/reference/data-types/index)) — مدة تنفيذ الاستعلام (بالملي ثانية).

{/*AUTOGENERATED_END*/}
