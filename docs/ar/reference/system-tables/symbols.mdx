<div id="description">
  ## الوصف
</div>

يحتوي على معلومات لاستبطان الملف التنفيذي `clickhouse`. ويتطلب الوصول إليه امتياز استبطان.
لا يفيد هذا الجدول إلا خبراء C++ ومهندسي ClickHouse.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/ar/reference/data-types/index)) — اسم الرمز في الملف التنفيذي. يكون بصيغة مُشوَّهة. يمكنك تطبيق ‎demangle(symbol)‎ للحصول على اسم قابل للقراءة.
* `address_begin` ([UInt64](/ar/reference/data-types/index)) — عنوان بداية الرمز في الملف التنفيذي.
* `address_end` ([UInt64](/ar/reference/data-types/index)) — عنوان نهاية الرمز في الملف التنفيذي.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
