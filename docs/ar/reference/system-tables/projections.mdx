<div id="description">
  ## الوصف
</div>

يحتوي على معلومات حول الإسقاطات الموجودة في جميع الجداول.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات.
* `table` ([String](/ar/reference/data-types/index)) — اسم الجدول.
* `name` ([String](/ar/reference/data-types/index)) — اسم الإسقاط.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](/ar/reference/data-types/index)) — نوع الإسقاط.
* `sorting_key` ([Array(String)](/ar/reference/data-types/index)) — مفتاح ترتيب الإسقاط.
* `query` ([String](/ar/reference/data-types/index)) — استعلام الإسقاط.
* `settings` ([Map(String, String)](/ar/reference/data-types/index)) — إعدادات الإسقاط.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       landing
name:        improved_sorting_key
type:        Normal
sorting_key: ['user_id','date']
query:       SELECT * ORDER BY user_id, date
settings:     {}

Row 2:
──────
database:    default
table:       landing
name:        agg_no_key
type:        Aggregate
sorting_key: []
query:       SELECT count()
settings:     {}
```
