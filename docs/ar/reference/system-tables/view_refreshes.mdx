<div id="description">
  ## الوصف
</div>

معلومات حول [العروض المادية القابلة للتحديث](/ar/reference/statements/create/view#refreshable-materialized-view). تحتوي على جميع العروض المادية القابلة للتحديث، سواء أكانت هناك عملية تحديث جارية أم لا.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات التي يوجد فيها الجدول.
* `view` ([String](/ar/reference/data-types/index)) — اسم الجدول.
* `uuid` ([UUID](/ar/reference/data-types/index)) — معرّف UUID للجدول (قاعدة بيانات Atomic).
* `status` ([String](/ar/reference/data-types/index)) — الحالة الحالية لعملية التحديث.
* `last_success_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — الوقت الذي بدأت فيه أحدث عملية تحديث ناجحة. تكون القيمة NULL إذا لم تحدث أي عمليات تحديث ناجحة منذ بدء تشغيل الخادم أو إنشاء الجدول.
* `last_success_duration_ms` ([Nullable(UInt64)](/ar/reference/data-types/index)) — المدة التي استغرقتها أحدث عملية تحديث.
* `last_refresh_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — الوقت الذي انتهت فيه أحدث محاولة تحديث (إن كان معروفًا) أو بدأت فيه (إن لم يكن معروفًا أو كانت لا تزال قيد التشغيل). تكون القيمة NULL إذا لم تحدث أي محاولات تحديث منذ بدء تشغيل الخادم أو إنشاء الجدول.
* `last_refresh_replica` ([String](/ar/reference/data-types/index)) — إذا كان التنسيق مفعّلًا، فهذا هو اسم النسخة المتماثلة التي أجرت محاولة التحديث الحالية (إذا كانت قيد التشغيل) أو السابقة (إذا لم تكن قيد التشغيل).
* `next_refresh_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — الوقت المجدول لبدء التحديث التالي. تكون القيمة NULL إذا لم يكن وقت التحديث التالي معروفًا حاليًا، على سبيل المثال عند انتظار التبعيات (الحالة `WaitingForDependencies` أو `MissingDependencies`).
* `exception` ([String](/ar/reference/data-types/index)) — رسالة الخطأ من المحاولة السابقة إذا كانت قد فشلت.
* `retry` ([UInt64](/ar/reference/data-types/index)) — عدد المحاولات الفاشلة حتى الآن للتحديث الحالي. غير متاح إذا كانت الحالة `RunningOnAnotherReplica`.
* `progress` ([Nullable(Float64)](/ar/reference/data-types/index)) — مدى تقدّم عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة، بين 0 و1. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica` أو إذا لم تكن عملية التحديث قيد التشغيل.
* `read_rows` ([Nullable(UInt64)](/ar/reference/data-types/index)) — عدد الصفوف التي قرأتها عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica`.
* `read_bytes` ([Nullable(UInt64)](/ar/reference/data-types/index)) — عدد البايتات التي قرأتها عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica`
* `total_rows` ([Nullable(UInt64)](/ar/reference/data-types/index)) — العدد الإجمالي التقديري للصفوف التي يجب أن تقرأها عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica`
* `written_rows` ([Nullable(UInt64)](/ar/reference/data-types/index)) — عدد الصفوف التي كتبتها عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica`
* `written_bytes` ([Nullable(UInt64)](/ar/reference/data-types/index)) — عدد البايتات التي كتبتها عملية التحديث الجارية حاليًا أو أحدث عملية تحديث مكتملة على النسخة المتماثلة المحددة. تكون القيمة NULL إذا كانت الحالة `RunningOnAnotherReplica`

{/*AUTOGENERATED_END*/}
