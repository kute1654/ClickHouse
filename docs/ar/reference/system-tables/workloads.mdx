<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن [أحمال العمل](/ar/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage) الموجودة على الخادم المحلي. يحتوي الجدول على صف لكل حمل عمل.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم حمل العمل.
* `parent` ([String](/ar/reference/data-types/index)) — اسم حمل العمل الأب.
* `create_query` ([String](/ar/reference/data-types/index)) — استعلام CREATE الخاص بحمل العمل.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT *
FROM system.workloads
FORMAT Vertical
```

```text
Row 1:
──────
name:         production
parent:       all
create_query: CREATE WORKLOAD production IN `all` SETTINGS weight = 9

Row 2:
──────
name:         development
parent:       all
create_query: CREATE WORKLOAD development IN `all`

Row 3:
──────
name:         all
parent:
create_query: CREATE WORKLOAD `all`
```
