يحتوي على معلومات عن جميع المعاملات المنفَّذة على الخادم الحالي.

من الآمن تنفيذ TRUNCATE أو DROP لهذا الجدول في أي وقت.

الأعمدة:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ar/reference/data-types/lowcardinality)) — اسم المضيف الذي نُفِّذت عليه المعاملة.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/ar/reference/data-types/enum)) — نوع المعاملة. القيم الممكنة: Begin وCommit وRollback وAddPart وLockPart وUnlockPart.
* `event_date` ([Date](/ar/reference/data-types/date)) — تاريخ الإدخال.
* `event_time` ([DateTime64(6)](/ar/reference/data-types/datetime64)) — وقت الإدخال
* `thread_id` ([UInt64](/ar/reference/data-types/int-uint)) — معرّف خيط التنفيذ.
* `query_id` ([String](/ar/reference/data-types/string)) — معرّف الاستعلام الذي نُفِّذ ضمن نطاق المعاملة.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/ar/reference/data-types/tuple)) — معرّف المعاملة.
* `tid_hash` ([UInt64](/ar/reference/data-types/int-uint)) — تجزئة المعرّف.
* `csn` ([UInt64](/ar/reference/data-types/int-uint)) — رقم تسلسل Commit.
* `database` ([String](/ar/reference/data-types/string)) — اسم قاعدة البيانات التي نُفِّذت عليها المعاملة.
* `table` ([String](/ar/reference/data-types/string)) — اسم الجدول الذي نُفِّذت عليه المعاملة.
* `uuid` ([UUID](/ar/reference/data-types/uuid)) — معرّف UUID للجدول الذي نُفِّذت عليه المعاملة.
* `part` ([String](/ar/reference/data-types/string)) — اسم الجزء الذي شارك في المعاملة.

{/*AUTOGENERATED_END*/}
