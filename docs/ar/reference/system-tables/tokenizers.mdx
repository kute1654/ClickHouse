<div id="description">
  ## الوصف
</div>

يعرض جميع مُقسِّمات الرموز المتاحة.
يمكن استخدام هذه الأدوات في الدوال [tokens](/ar/reference/functions/regular-functions/splitting-merging-functions#tokens)، و[hasAllTokens](/ar/reference/functions/regular-functions/string-search-functions#hasAllTokens)، و[hasAnyTokens](/ar/reference/functions/regular-functions/string-search-functions#hasAnyTokens)، وفي [الفهرس النصي](/ar/reference/engines/table-engines/mergetree-family/textindexes).

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم مُقسِّم الرموز

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.tokenizers;
```

```text
┌─name────────────┐
│ ngrams          │
│ splitByNonAlpha │
│ sparseGrams     │

│ ngrambf_v1      │
│ array           │
│ splitByString   │

└─────────────────┘
```
