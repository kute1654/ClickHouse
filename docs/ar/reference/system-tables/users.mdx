<div id="description">
  ## الوصف
</div>

يتضمن قائمة بـ[حسابات المستخدمين](/ar/concepts/features/security/access-rights#user-account-management) المُعرَّفة على الخادم.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم المستخدم.
* `id` ([UUID](/ar/reference/data-types/index)) — معرّف المستخدم.
* `storage` ([String](/ar/reference/data-types/index)) — المسار إلى وحدة تخزين المستخدمين. يُضبط في المعلمة `access_control_path`.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/ar/reference/data-types/index)) — يعرض أنواع المصادقة. توجد عدة طرق للتحقق من هوية المستخدم: من دون كلمة مرور، أو باستخدام كلمة مرور بنص صريح، أو كلمة مرور مُرمّزة بـ SHA256، أو كلمة مرور مُرمّزة بـ SHA-1 مزدوج، أو كلمة مرور مُرمّزة بـ bcrypt.
* `auth_params` ([Array(String)](/ar/reference/data-types/index)) — معلمات المصادقة بتنسيق JSON وفقًا لـ `auth_type`.
* `valid_until` ([Array(DateTime)](/ar/reference/data-types/index)) — تاريخ ووقت انتهاء صلاحية بيانات اعتماد المستخدم.
* `host_ip` ([Array(String)](/ar/reference/data-types/index)) — عناوين IP للمضيفين المسموح لهم بالاتصال بخادم ClickHouse.
* `host_names` ([Array(String)](/ar/reference/data-types/index)) — أسماء المضيفين المسموح لهم بالاتصال بخادم ClickHouse.
* `host_names_regexp` ([Array(String)](/ar/reference/data-types/index)) — تعبير نمطي لأسماء المضيفين المسموح لهم بالاتصال بخادم ClickHouse.
* `host_names_like` ([Array(String)](/ar/reference/data-types/index)) — أسماء المضيفين المسموح لهم بالاتصال بخادم ClickHouse، والمحددة باستخدام المسند LIKE.
* `default_roles_all` ([UInt8](/ar/reference/data-types/index)) — يوضّح أن جميع الأدوار الممنوحة مضبوطة لهذا المستخدم افتراضيًا.
* `default_roles_list` ([Array(String)](/ar/reference/data-types/index)) — قائمة الأدوار الممنوحة افتراضيًا.
* `default_roles_except` ([Array(String)](/ar/reference/data-types/index)) — جميع الأدوار الممنوحة المضبوطة كأدوار افتراضية، باستثناء المدرجة في القائمة.
* `grantees_any` ([UInt8](/ar/reference/data-types/index)) — العلامة التي تشير إلى ما إذا كان المستخدم الذي لديه أي خيارات منح يمكنه منحها لأي شخص.
* `grantees_list` ([Array(String)](/ar/reference/data-types/index)) — قائمة المستخدمين أو الأدوار المسموح لهذا المستخدم بمنح الخيارات لهم.
* `grantees_except` ([Array(String)](/ar/reference/data-types/index)) — قائمة المستخدمين أو الأدوار الممنوع على هذا المستخدم منح الخيارات لهم.
* `default_database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات الافتراضية لهذا المستخدم.

{/*AUTOGENERATED_END*/}
