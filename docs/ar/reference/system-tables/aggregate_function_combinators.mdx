<div id="description">
  ## الوصف
</div>

يتضمن قائمة بجميع مُعدِّل الدوال التجميعية المتاحة، والتي يمكن تطبيقها على دوال التجميع لتغيير طريقة عملها.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم مُعدِّل الدوال التجميعية.
* `is_internal` ([UInt8](/ar/reference/data-types/index)) — ما إذا كان هذا الـ مُعدِّل الدوال التجميعية مخصّصًا للاستخدام الداخلي فقط.
* `description` ([String](/ar/reference/data-types/index)) — وصف عام لوظيفة الـ مُعدِّل الدوال التجميعية.
* `syntax` ([String](/ar/reference/data-types/index)) — كيفية تطبيق الـ مُعدِّل الدوال التجميعية على اسم دالة تجميع.
* `examples` ([String](/ar/reference/data-types/index)) — أمثلة استخدام.
* `introduced_in` ([String](/ar/reference/data-types/index)) — إصدار ClickHouse الذي ظهر فيه الـ مُعدِّل الدوال التجميعية لأول مرة، بصيغة major.minor.
* `related` ([Array(String)](/ar/reference/data-types/index)) — أسماء الـ مُعدِّل الدوال التجميعية ذات الصلة.

{/*AUTOGENERATED_END*/}
