<div id="description">
  ## الوصف
</div>

يتضمن معلومات عن المعلمات [graphite&#95;rollup](/ar/reference/settings/server-settings/settings#graphite) المستخدمة في الجداول ذات محركات [*GraphiteMergeTree](/ar/reference/engines/table-engines/mergetree-family/graphitemergetree).

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/ar/reference/data-types/index)) — اسم المعلمة graphite&#95;rollup.
* `rule_type` ([String](/ar/reference/data-types/index)) — نوع القاعدة. القيم الممكنة: RuleTypeAll = 0 - الافتراضي، مع تعبير نمطي، ومتوافق مع الصيغة القديمة؛ RuleTypePlain = 1 - مقاييس عادية، مع تعبير نمطي، ومتوافق مع الصيغة القديمة؛ RuleTypeTagged = 2 - مقاييس موسومة، مع تعبير نمطي، ومتوافق مع الصيغة القديمة؛ RuleTypeTagList = 3 - مقاييس موسومة، مع تعبير نمطي (تُحوَّل إلى RuleTypeTagged من سلسلة مثل &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](/ar/reference/data-types/index)) — نمط لاسم المقياس.
* `function` ([String](/ar/reference/data-types/index)) — اسم دالة التجميع.
* `age` ([UInt64](/ar/reference/data-types/index)) — الحد الأدنى لعمر البيانات بالثواني.
* `precision` ([UInt64](/ar/reference/data-types/index)) — مدى دقة تحديد عمر البيانات بالثواني.
* `priority` ([UInt16](/ar/reference/data-types/index)) — أولوية النمط.
* `is_default` ([UInt8](/ar/reference/data-types/index)) — ما إذا كان النمط هو الافتراضي.
* `Tables.database` ([Array(String)](/ar/reference/data-types/index)) — مصفوفة بأسماء قواعد البيانات الخاصة بالجداول التي تستخدم المعلمة `config_name`.
* `Tables.table` ([Array(String)](/ar/reference/data-types/index)) — مصفوفة بأسماء الجداول التي تستخدم المعلمة `config_name`.

{/*AUTOGENERATED_END*/}
