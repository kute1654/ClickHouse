<div id="description">
  ## الوصف
</div>

تحتوي على الحالة المخزنة في الذاكرة للبيانات الوصفية لـ AzureQueue والصفوف الجاري معالجتها لكل ملف.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/ar/reference/data-types/index)) — المسار في ZooKeeper إلى البيانات الوصفية
* `file_path` ([String](/ar/reference/data-types/index)) — مسار الملف الجاري معالجته
* `file_name` ([String](/ar/reference/data-types/index)) — اسم الملف الجاري معالجته
* `rows_processed` ([UInt64](/ar/reference/data-types/index)) — عدد الصفوف التي تجري معالجتها حاليًا
* `status` ([String](/ar/reference/data-types/index)) — حالة المعالجة: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — الوقت الذي بدأت فيه معالجة الملف
* `processing_end_time` ([Nullable(DateTime)](/ar/reference/data-types/index)) — الوقت الذي انتهت فيه معالجة الملف
* `exception` ([String](/ar/reference/data-types/index)) — الاستثناء الذي وقع أثناء المعالجة

{/*AUTOGENERATED_END*/}
