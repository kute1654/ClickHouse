<div id="description">
  ## الوصف
</div>

تحتوي على معلومات عن ملفات البيانات المخزنة على الأقراص البعيدة (مثل S3 وAzure Blob Storage)، بما في ذلك الربط بين مسارات البيانات الوصفية المحلية ومسارات كائنات blob البعيدة.

يمثل كل صف كائن blob بعيدًا واحدًا مرتبطًا بملف من ملفات البيانات.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `disk_name` ([String](/ar/reference/data-types/index)) — اسم القرص.
* `path` ([String](/ar/reference/data-types/index)) — مسار القرص.
* `cache_base_path` ([String](/ar/reference/data-types/index)) — الدليل الأساسي لملفات التخزين المؤقت.
* `local_path` ([String](/ar/reference/data-types/index)) — مسار ملف ClickHouse، ويُستخدم أيضًا كمسار للبيانات الوصفية.
* `remote_path` ([String](/ar/reference/data-types/index)) — مسار الـ blob في تخزين الكائنات المرتبط به ملف ClickHouse.
* `size` ([UInt64](/ar/reference/data-types/index)) — حجم الملف (مضغوط).
* `common_prefix_for_blobs` ([String](/ar/reference/data-types/index)) — البادئة المشتركة لكائنات الـ blob في تخزين الكائنات.
* `cache_paths` ([Array(String)](/ar/reference/data-types/index)) — ملفات التخزين المؤقت الخاصة بالـ blob المقابل.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
Row 1:
──────

path:                   /var/lib/clickhouse/disks/s3/

local_path:             store/123/1234abcd-1234-1234-1234-1234abcd1234/all_0_0_0/data.bin
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
