<div id="description">
  ## الوصف
</div>

يتضمن معلومات حول قواعد البيانات المتاحة للمستخدم الحالي.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات.
* `engine` ([String](/ar/reference/data-types/index)) — محرك قاعدة البيانات.
* `data_path` ([String](/ar/reference/data-types/index)) — مسار البيانات.
* `metadata_path` ([String](/ar/reference/data-types/index)) — مسار البيانات الوصفية.
* `uuid` ([UUID](/ar/reference/data-types/index)) — UUID لقاعدة البيانات.
* `engine_full` ([String](/ar/reference/data-types/index)) — معلمات محرك قاعدة البيانات.
* `comment` ([String](/ar/reference/data-types/index)) — تعليق قاعدة البيانات.
* `is_external` ([UInt8](/ar/reference/data-types/index)) — قاعدة البيانات خارجية (أي PostgreSQL/DataLakeCatalog).

{/*AUTOGENERATED_END*/}

يُستخدم العمود `name` في هذا الجدول النظامي لتنفيذ الاستعلام `SHOW DATABASES`.

<div id="example">
  ## مثال
</div>

أنشئ قاعدة بيانات.

```sql title="Query"
CREATE DATABASE test;
```

اعرض جميع قواعد البيانات المتاحة للمستخدم.

```sql title="Query"
SELECT * FROM system.databases;
```
