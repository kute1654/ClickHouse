<div id="description">
  ## الوصف
</div>

يتضمن هذا القسم قائمةً بجميع الإعدادات الخاصة بمحرك ReplicatedMergeTree، مع قيمها الحالية والافتراضية وأوصافها. ويمكنك تغيير أيٍّ منها في قسم SETTINGS ضمن استعلام CREATE.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم الإعداد.
* `value` ([String](/ar/reference/data-types/index)) — قيمة الإعداد.
* `default` ([String](/ar/reference/data-types/index)) — القيمة الافتراضية للإعداد.
* `changed` ([UInt8](/ar/reference/data-types/index)) — 1 إذا كان الإعداد قد حُدِّد صراحةً في config أو جرى تغييره صراحةً.
* `description` ([String](/ar/reference/data-types/index)) — وصف الإعداد.
* `min` ([Nullable(String)](/ar/reference/data-types/index)) — الحد الأدنى لقيمة الإعداد، إذا كان مضبوطًا عبر القيود. وإذا لم تكن هناك قيمة دنيا للإعداد، فستكون NULL.
* `max` ([Nullable(String)](/ar/reference/data-types/index)) — الحد الأقصى لقيمة الإعداد، إذا كان مضبوطًا عبر القيود. وإذا لم تكن هناك قيمة قصوى للإعداد، فستكون NULL.
* `disallowed_values` ([Array(String)](/ar/reference/data-types/index)) — قائمة القيم غير المسموح بها
* `readonly` ([UInt8](/ar/reference/data-types/index)) — يوضح ما إذا كان بإمكان المستخدم الحالي تغيير الإعداد: 0 — يمكن للمستخدم الحالي تغيير الإعداد، 1 — لا يمكن للمستخدم الحالي تغيير الإعداد.
* `type` ([String](/ar/reference/data-types/index)) — نوع الإعداد (قيمة نصية خاصة بالتنفيذ).
* `is_obsolete` ([UInt8](/ar/reference/data-types/index)) — يوضح ما إذا كان الإعداد متقادمًا.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/ar/reference/data-types/index)) —
  مستوى الدعم لهذه الميزة. تُنظَّم ميزات ClickHouse ضمن مستويات، وتختلف بحسب حالتها الحالية
  في دورة التطوير والتوقعات التي يمكن أن تكون لديك عند استخدامها:

- PRODUCTION: الميزة مستقرة وآمنة للاستخدام، ولا توجد مشكلات في تفاعلها مع ميزات PRODUCTION الأخرى.
- BETA: الميزة مستقرة وآمنة. نتيجة استخدامها مع الميزات الأخرى غير معروفة، ولا يمكن ضمان صحتها. نرحب بالاختبارات والتقارير.
- EXPERIMENTAL: الميزة قيد التطوير. وهي مخصصة فقط للمطورين ومتحمسي ClickHouse. قد تعمل أو لا تعمل، وقد تُزال في أي وقت.
- OBSOLETE: لم تعد مدعومة. إما أنها أُزيلت بالفعل أو ستُزال في الإصدارات المستقبلية.
