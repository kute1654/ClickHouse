<div id="description">
  ## الوصف
</div>

يحتوي على خصائص ملفات تعريف الإعدادات التي تم تكوينها.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ar/reference/data-types/index)) — اسم ملف تعريف الإعدادات.
* `id` ([UUID](/ar/reference/data-types/index)) — معرّف ملف تعريف الإعدادات.
* `storage` ([String](/ar/reference/data-types/index)) — path تخزين ملفات تعريف الإعدادات. يُضبط في parameter ‏`access_control_path`.
* `num_elements` ([UInt64](/ar/reference/data-types/index)) — عدد العناصر لهذا الملف التعريفي في الجدول `system.settings_profile_elements`.
* `apply_to_all` ([UInt8](/ar/reference/data-types/index)) — يوضّح أن ملف تعريف الإعدادات مُطبَّق على جميع roles و/أو المستخدمين.
* `apply_to_list` ([Array(String)](/ar/reference/data-types/index)) — قائمة roles و/أو المستخدمين الذين يُطبَّق عليهم ملف تعريف الإعدادات.
* `apply_to_except` ([Array(String)](/ar/reference/data-types/index)) — يُطبَّق ملف تعريف الإعدادات على جميع roles و/أو المستخدمين باستثناء المُدرجين في القائمة.

{/*AUTOGENERATED_END*/}
