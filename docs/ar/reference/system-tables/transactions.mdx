يتضمن قائمة بالمعاملات وحالتها.

<div id="columns">
  ## الأعمدة
</div>

{/*بداية_الإنشاء_التلقائي*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/ar/reference/data-types/tuple)) — معرّف المعاملة.
* `tid_hash` ([UInt64](/ar/reference/data-types/int-uint)) — قيمة hash للمعرّف.
* `elapsed` ([Float64](/ar/reference/data-types/float)) — الوقت المستغرَق في معالجة المعاملة.
* `is_readonly` ([UInt8](/ar/reference/data-types/int-uint)) — العلامة التي تشير إلى ما إذا كانت المعاملة قد نفّذت أي عملية كتابة.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/ar/reference/data-types/enum)) — حالة المعاملة. القيم الممكنة: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.
