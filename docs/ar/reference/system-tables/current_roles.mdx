<div id="description">
  ## الوصف
</div>

يحتوي هذا الجدول على الأدوار النشطة للمستخدم الحالي. يغيّر `SET ROLE` محتويات هذا الجدول.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/ar/reference/data-types/index)) — اسم الدور.
* `with_admin_option` ([UInt8](/ar/reference/data-types/index)) — تكون القيمة 1 إذا كان الدور يتمتع بامتياز ADMIN OPTION.
* `is_default` ([UInt8](/ar/reference/data-types/index)) — تكون القيمة 1 إذا كان الدور مُعيَّنًا كقيمة افتراضية.

{/*AUTOGENERATED_END*/}
