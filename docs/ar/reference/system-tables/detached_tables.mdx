<div id="description">
  ## الوصف
</div>

يحتوي على معلومات عن كل جدول منفصل.

<div id="columns">
  ## الأعمدة
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ar/reference/data-types/index)) — اسم قاعدة البيانات التي ينتمي إليها الجدول.
* `table` ([String](/ar/reference/data-types/index)) — اسم الجدول.
* `uuid` ([UUID](/ar/reference/data-types/index)) — معرّف UUID الخاص بالجدول (قاعدة بيانات Atomic).
* `metadata_path` ([String](/ar/reference/data-types/index)) — مسار البيانات الوصفية للجدول في نظام الملفات.
* `is_permanently` ([UInt8](/ar/reference/data-types/index)) — فُصل الجدول نهائيًا.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## مثال
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
