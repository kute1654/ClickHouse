<div id="description">
  ## الوصف
</div>

في هذه الصيغة، يُخرج ClickHouse كل صف على هيئة كائن JSON منفصل، بحيث يكون كل كائن في سطر جديد.

تُعرَف هذه الصيغة أيضًا باسم `JSONEachRow` و`JSONLines` و`NDJSON` ‏(JSON المحدَّد بأسطر جديدة) أو `JSONL`. وهذه الأسماء هي أسماء بديلة للصيغة نفسها، ويمكن استخدامها بالتبادل لكلٍّ من الإدخال والإخراج.

```sql

```

  ### قراءة البيانات
</div>

اقرأ البيانات بتنسيق `JSONLines`:

```sql
SELECT *
FROM football
FORMAT JSONLines
```
