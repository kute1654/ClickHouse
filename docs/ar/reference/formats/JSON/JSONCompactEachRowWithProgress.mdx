<div id="description">
  ## الوصف
</div>

يجمع هذا التنسيق بين المخرجات المدمجة صفًا بصف في JSONCompactEachRow ومعلومات التقدّم المتدفقة.
ويُخرج البيانات على شكل كائنات JSON منفصلة للبيانات الوصفية، والصفوف الفردية، وتحديثات التقدّم،
والإجماليات، والاستثناءات. وتُمثَّل القيم بأنواعها الأصلية.

الميزات الرئيسية:

* يُخرج البيانات الوصفية أولًا، متضمنةً أسماء الأعمدة وأنواعها
* يكون كل صف كائن JSON منفصلًا، مع مفتاح &quot;row&quot; يحتوي على مصفوفة من القيم
* يتضمن تحديثات للتقدّم أثناء تنفيذ الاستعلام (على شكل كائنات `{"progress":...}`)
* يدعم الإجماليات والقيم القصوى
* تحتفظ القيم بأنواعها الأصلية (الأرقام كأرقام، والسلاسل النصية كسلاسل نصية)

<div id="example-usage">
  ## مثال للاستخدام
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactEachRowWithProgress
```
