<div id="description">
  ## الوصف
</div>

في هذا التنسيق، يُخرج ClickHouse كل صف على شكل كائن JSON مستقل في سطر جديد.

أدرج البيانات:

```sql

```

<div id="reading-data">
  ### قراءة البيانات
</div>

اقرأ البيانات بتنسيق `JSONEachRow`:

```sql
SELECT *
FROM football
FORMAT JSONEachRow
```
