---
title: إعدادات Pretty Format الشائعة
---

{/* ملاحظة: يُستخدم هذا الملف كمقتطف في كل الملفات التي تستورده */}

الإعدادات التالية مشتركة بين جميع تنسيقات `Pretty`:

| الإعداد                                                                                                                                              | الوصف                                                                                                                                                                                                     | الافتراضي |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| [`output_format_pretty_max_rows`](/ar/reference/settings/formats#output_format_pretty_max_rows)                                                         | الحد الأقصى لعدد الصفوف في تنسيقات Pretty.                                                                                                                                                                | `10000`   |
| [`output_format_pretty_max_column_pad_width`](/ar/reference/settings/formats#output_format_pretty_max_column_pad_width)                                 | الحد الأقصى لعرض الحشو لجميع القيم في عمود ضمن تنسيقات Pretty.                                                                                                                                            | `250`     |
| [`output_format_pretty_max_value_width`](/ar/reference/settings/formats#output_format_pretty_max_value_width)                                           | الحد الأقصى لعرض القيمة المعروضة في تنسيقات Pretty. وإذا تجاوزته القيمة، فسيتم اقتطاعها.                                                                                                                  | `10000`   |
| [`output_format_pretty_color`](/ar/reference/settings/formats#output_format_pretty_color)                                                               | استخدم تسلسلات ANSI الهروبية لتلوين المخرجات في تنسيقات Pretty.                                                                                                                                           | `true`    |
| [`output_format_pretty_grid_charset`](/ar/reference/settings/formats#output_format_pretty_grid_charset)                                                 | مجموعة المحارف المستخدمة لطباعة حدود الشبكة. مجموعات المحارف المتاحة: ASCII وUTF-8.                                                                                                                       | `UTF-8`   |
| [`output_format_pretty_row_numbers`](/ar/reference/settings/formats#output_format_pretty_row_numbers)                                                   | أضف أرقام الصفوف قبل كل صف في تنسيق الإخراج Pretty.                                                                                                                                                       | `true`    |
| [`output_format_pretty_display_footer_column_names`](/ar/reference/settings/formats#output_format_pretty_display_footer_column_names)                   | اعرض أسماء الأعمدة في التذييل إذا كان الجدول يحتوي على عدد كبير من الصفوف.                                                                                                                                | `true`    |
| [`output_format_pretty_display_footer_column_names_min_rows`](/ar/reference/settings/formats#output_format_pretty_display_footer_column_names_min_rows) | يحدد الحد الأدنى لعدد الصفوف الذي عنده سيُعرض التذييل إذا كان [`output_format_pretty_display_footer_column_names`](/ar/reference/settings/formats#output_format_pretty_display_footer_column_names) ممكّنًا. | `50`      |
