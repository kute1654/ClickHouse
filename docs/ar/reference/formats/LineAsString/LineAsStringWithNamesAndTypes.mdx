<div id="description">
  ## الوصف
</div>

تنسيق `LineAsStringWithNames` مشابه لتنسيق [`LineAsString`](/ar/reference/formats/LineAsString/LineAsString)،
لكنه يعرض صفَّي ترويسة: أحدهما لأسماء الأعمدة، والآخر للأنواع.

<div id="example-usage">
  ## مثال للاستخدام
</div>

```sql title="Query"
CREATE TABLE example (
    name String,
    value Int32
)
ENGINE = Memory;

INSERT INTO example VALUES ('John', 30), ('Jane', 25), ('Peter', 35);

SELECT * FROM example FORMAT LineAsStringWithNamesAndTypes;
```

```response title="Response"
name    value
String    Int32
John    30
Jane    25
Peter    35
```
