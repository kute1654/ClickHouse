<div id="description">
  ## الوصف
</div>

يمكن استخدام تنسيق `Form` لقراءة سجل واحد بتنسيق `application/x-www-form-urlencoded`،
حيث تُنسَّق البيانات بالشكل `key1=value1&key2=value2`.

<div id="example-usage">
  ## مثال على الاستخدام
</div>

إذا كان هناك ملف `data.tmp` موضوع في المسار `user_files` ويحتوي على بعض البيانات المرمّزة بترميز URL:

```text title="data.tmp"
t_page=116&c.e=ls7xfkpm&c.tti.m=raf&rt.start=navigation&rt.bmr=390%2C11%2C10
```

```sql title="Query"
SELECT * FROM file(data.tmp, Form) FORMAT vertical;
```

```response title="Response"
Row 1:
──────
t_page:   116
c.e:      ls7xfkpm
c.tti.m:  raf
rt.start: navigation
rt.bmr:   390,11,10
```
