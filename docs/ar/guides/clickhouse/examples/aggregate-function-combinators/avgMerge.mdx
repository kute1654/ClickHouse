<div id="description">
  ## الوصف
</div>

يمكن تطبيق المُركِّب [`Merge`](/ar/reference/functions/aggregate-functions/combinators#-state)

  ## مثال للاستخدام
</div>

يرتبط المُركِّب `Merge` ارتباطًا وثيقًا بالمُركِّب `State`. راجع
[&quot;مثال لاستخدام avgState&quot;](/ar/guides/clickhouse/examples/aggregate-function-combinators/avgState#example-usage)
