> يشرح هذا الدليل كيفية إعداد [AnythingLLM](https://anythingllm.com/) مع خادم ClickHouse MCP باستخدام Docker
> وربطه بمجموعات بيانات ClickHouse التجريبية.

<Steps>
  <Step>
    ## تثبيت Docker

    ستحتاج إلى Docker لتشغيل LibreChat وخادم MCP. للحصول على Docker:

    1. انتقل إلى [docker.com](https://www.docker.com/products/docker-desktop)
    2. نزّل Docker Desktop لنظام التشغيل لديك
    3. ثبّت Docker باتباع التعليمات الخاصة بنظام التشغيل لديك
    4. افتح Docker Desktop وتأكد من أنه يعمل

    <br />

    لمزيد من المعلومات، راجع [وثائق Docker](https://docs.docker.com/get-docker/).
  </Step>

  <Step>
    ## سحب صورة Docker الخاصة بـ AnythingLLM

    شغّل الأمر التالي لسحب صورة Docker الخاصة بـ AnythingLLM إلى جهازك:

    ```bash
    docker pull anythingllm/anythingllm
    ```
  </Step>

  <Step>
    ## إعداد موقع التخزين

    أنشئ دليلًا للتخزين وهيّئ ملف البيئة:

    ```bash
    export STORAGE_LOCATION=$PWD/anythingllm && \
    mkdir -p $STORAGE_LOCATION && \
    touch "$STORAGE_LOCATION/.env" 
    ```
  </Step>

  <Step>
    ## تهيئة ملف إعدادات خادم MCP

    أنشئ دليل `plugins`:

    ```bash
    mkdir -p "$STORAGE_LOCATION/plugins"
    ```

    أنشئ ملفًا باسم `anythingllm_mcp_servers.json` داخل دليل `plugins` وأضف المحتويات التالية:

    ```json
    {
      "mcpServers": {
        "mcp-clickhouse": {
          "command": "uv",
          "args": [
            "run",
            "--with",
            "mcp-clickhouse",
            "--python",
            "3.10",
            "mcp-clickhouse"
          ],
          "env": {
            "CLICKHOUSE_HOST": "sql-clickhouse.clickhouse.com",
            "CLICKHOUSE_USER": "demo",
            "CLICKHOUSE_PASSWORD": ""
          }
        }
      }
    }
    ```

    إذا أردت استكشاف بياناتك الخاصة، فيمكنك فعل ذلك
    باستخدام [المضيف واسم المستخدم وكلمة المرور](/ar/get-started/setup/cloud#connect-with-your-app)
    الخاصة بخدمة ClickHouse Cloud لديك.
  </Step>

  <Step>
    ## تشغيل حاوية Docker الخاصة بـ AnythingLLM

    شغّل الأمر التالي لبدء حاوية Docker الخاصة بـ AnythingLLM:

    ```bash
    docker run -p 3001:3001 \
    --cap-add SYS_ADMIN \
    -v ${STORAGE_LOCATION}:/app/server/storage \
    -v ${STORAGE_LOCATION}/.env:/app/server/.env \
    -e STORAGE_DIR="/app/server/storage" \
    mintplexlabs/anythingllm
    ```

    بعد بدء التشغيل، انتقل في المتصفح إلى `http://localhost:3001`.
    حدّد النموذج الذي تريد استخدامه وأدخل مفتاح API الخاص بك.
  </Step>

  <Step>
    ## انتظر حتى تبدأ خوادم MCP

    انقر على أيقونة الأداة في أسفل يسار واجهة المستخدم:

    <Image img="/images/use-cases/AI_ML/MCP/alm_tool-icon.webp" alt="أيقونة الأداة" size="md" />

    انقر على `Agent Skills` وابحث ضمن قسم `MCP servers`.
    انتظر حتى ترى أن `Mcp ClickHouse` أصبح على `On`

    <Image img="/images/use-cases/AI_ML/MCP/allm_mcp-servers.webp" alt="خوادم MCP جاهزة" size="md" />
  </Step>

  <Step>
    ## أجرِ محادثة مع خادم ClickHouse MCP باستخدام AnythingLLM

    أصبحنا الآن جاهزين لبدء محادثة.
    ولجعل خوادم MCP متاحة في المحادثة، ستحتاج إلى إضافة السابقة `@agent` إلى الرسالة الأولى في المحادثة.

    <Image img="/images/use-cases/AI_ML/MCP/allm_conversation.webp" alt="محادثة" size="md" />
  </Step>
</Steps>
