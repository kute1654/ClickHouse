뷰를 구현하는 데 사용됩니다(자세한 내용은 `CREATE VIEW query`를 참조하십시오). 데이터는 저장하지 않고, 지정된 `SELECT` 쿼리만 저장합니다. 테이블을 읽을 때는 이 쿼리를 실행하고 쿼리에서 불필요한 모든 컬럼을 제거합니다.
