임시 [Merge](/ko/reference/engines/table-engines/special/merge) 테이블을 생성합니다.
테이블 스키마(schema)는 기반 테이블의 컬럼 합집합과 공통 타입을 바탕으로 결정됩니다.
[Merge](/ko/reference/engines/table-engines/special/merge) 테이블 엔진에서와 동일한 가상 컬럼을 사용할 수 있습니다.

<div id="syntax">
  ## 구문
</div>

```sql
merge(['db_name',] 'tables_regexp')
```

<div id="arguments">
  ## 인수
</div>

| 인수              | 설명                                                                                                                                                                                                   |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `db_name`       | 가능한 값(선택 사항, 기본값은 `currentDatabase()`):<br />    - 데이터베이스 이름,<br />    - 데이터베이스 이름을 문자열로 반환하는 상수 표현식(예: `currentDatabase()`),<br />    - `REGEXP(expression)`, 여기서 `expression`은 DB 이름과 일치하는 정규식입니다. |
| `tables_regexp` | 지정된 DB 또는 여러 DB의 테이블 이름과 일치하는 정규식입니다.                                                                                                                                                                |
