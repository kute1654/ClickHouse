`timeSeriesTags(db_name.time_series_table)` - 테이블 엔진이 [TimeSeries](/ko/reference/engines/table-engines/integrations/time-series) 엔진인
테이블 `db_name.time_series_table`에서 사용되는 [Tags 테이블](/ko/reference/engines/table-engines/integrations/time-series#tags-table)을 반환합니다:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS tags_table
```

이 함수는 *tags* 테이블이 내부 테이블이어도 작동합니다:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

다음 쿼리는 동일한 의미입니다:

```sql
SELECT * FROM timeSeriesTags(db_name.time_series_table);
SELECT * FROM timeSeriesTags('db_name.time_series_table');
SELECT * FROM timeSeriesTags('db_name', 'time_series_table');
```
