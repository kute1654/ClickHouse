지정된 구조의 임시 테이블을 [Null](/ko/reference/engines/table-engines/special/null) table engine으로 생성합니다. `Null` 엔진의 속성에 따라 테이블 데이터는 무시되며, 테이블 자체는 쿼리 실행 직후 즉시 삭제됩니다. 이 함수는 테스트 작성과 데모를 쉽게 하기 위해 사용됩니다.

<div id="syntax">
  ## 구문
</div>

```sql
null('structure')
```

<div id="argument">
  ## 인수
</div>

* `structure` — 컬럼 이름과 컬럼 타입의 목록입니다. [String](/ko/reference/data-types/string).

<div id="returned_value">
  ## 반환 값
</div>

지정된 구조의 임시 `Null` 엔진 테이블입니다.

<div id="example">
  ## 예시
</div>

`null` 함수를 사용한 쿼리:

```sql
INSERT INTO function null('x UInt64') SELECT * FROM numbers_mt(1000000000);
```

다음 세 개의 쿼리로 대체할 수 있습니다:

```sql
CREATE TABLE t (x UInt64) ENGINE = Null;
INSERT INTO t SELECT * FROM numbers_mt(1000000000);
DROP TABLE IF EXISTS t;
```
