<div id="timeSeriesGroupArray">
  ## timeSeriesGroupArray
</div>

지원 시작 버전: v25.8.0

타임스탬프 기준으로 시계열 데이터를 오름차순으로 정렬합니다.

<Note>
  이 함수는 실험적 기능이므로 `allow_experimental_ts_to_grid_aggregate_function=true`를 설정하여 활성화하십시오.
</Note>

**구문**

```sql
timeSeriesGroupArray(timestamp, value)
```

**인수**

* `timestamp` — 샘플의 타임스탬프입니다. [`DateTime`](/ko/reference/data-types/datetime) 또는 [`UInt32`](/ko/reference/data-types/int-uint) 또는 [`UInt64`](/ko/reference/data-types/int-uint)
* `value` — 해당 타임스탬프에 대응하는 시계열 값입니다. [`(U)Int*`](/ko/reference/data-types/int-uint) 또는 [`Float*`](/ko/reference/data-types/float) 또는 [`Decimal`](/ko/reference/data-types/decimal)

**반환 값**

타임스탬프 기준 오름차순으로 정렬된 튜플 배열 `(timestamp, value)`을 반환합니다. 동일한 타임스탬프에 여러 값이 있으면 함수는 그중 가장 큰 값을 선택합니다. [`Array(Tuple(T1, T2))`](/ko/reference/data-types/array)

**예시**

**개별 값을 사용한 기본 사용법**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamp, value)
FROM
(
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesGroupArray(timestamp, value)───────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]  │
└──────────────────────────────────────────────────────┘
```

**동일한 크기의 배열로 여러 개의 타임스탬프와 값을 전달하기**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamps, values);
```

```response title=Response
┌─timeSeriesGroupArray(timestamps, values)──────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]   │
└───────────────────────────────────────────────────────┘
```
