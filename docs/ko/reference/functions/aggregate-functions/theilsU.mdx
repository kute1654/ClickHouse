<div id="theilsU">
  ## theilsU
</div>

도입 버전: v22.1.0

`theilsU` 함수는 [Theil&#39;s U 불확실성 계수](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient)를 계산합니다. 이 값은 테이블(table)의 두 컬럼(column) 간 연관성을 측정합니다.
값의 범위는 0.0(연관성 없음)부터 1.0(완전 일치)까지입니다.

**구문**

```sql
theilsU(column1, column2)
```

**인수**

* `column1` — 비교할 첫 번째 컬럼입니다. [`Any`](/ko/reference/data-types/index)
* `column2` — 비교할 두 번째 컬럼입니다. [`Any`](/ko/reference/data-types/index)

**반환 값**

0과 1 사이의 값을 반환합니다. [`Float64`](/ko/reference/data-types/float)

**예시**

**사용 예시**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
