<div id="timeSeriesChangesToGrid">
  ## timeSeriesChangesToGrid
</div>

도입 버전: v25.6.0

시계열 데이터를 타임스탬프와 값의 쌍으로 받아, 시작 타임스탬프, 종료 타임스탬프, 간격으로 정의된 규칙적인 시간 그리드에서 이 데이터의 [PromQL과 유사한 변경 횟수](https://prometheus.io/docs/prometheus/latest/querying/functions/#changes)를 계산하는 집계 함수입니다. 그리드의 각 지점에서 `changes` 계산에 사용할 샘플은 지정된 시간 윈도우 내에서 고려됩니다.

<Note>
  이 함수는 실험적 기능이므로 `allow_experimental_ts_to_grid_aggregate_function=true`를 설정하여 활성화하십시오.
</Note>

**구문**

```sql
timeSeriesChangesToGrid(start_timestamp, end_timestamp, grid_step, staleness)(timestamp, value)
```

**매개변수**

* `start_timestamp` — 그리드의 시작 지점을 지정합니다. - `end_timestamp` — 그리드의 끝 지점을 지정합니다. - `grid_step` — 초 단위의 그리드 간격을 지정합니다. - `staleness` — 고려하는 샘플에 대해 허용되는 최대 &quot;staleness&quot;를 초 단위로 지정합니다.

**인수**

* `timestamp` — 샘플의 타임스탬프입니다. 개별 값 또는 배열일 수 있습니다. - `value` — 해당 타임스탬프에 대응하는 시계열 값입니다. 개별 값 또는 배열일 수 있습니다.

**반환 값**

지정된 그리드의 `changes` 값을 `Array(Nullable(Float64))` 형식으로 반환합니다. 반환된 배열에는 각 시간 그리드 지점마다 하나의 값이 포함됩니다. 특정 그리드 지점에서 `changes` 값을 계산할 수 있는 샘플이 윈도우 내에 없으면 해당 값은 NULL입니다.

**예시**

**그리드 [90, 105, 120, 135, 150, 165, 180, 195, 210, 225]에서 `changes` 값을 계산합니다**

```sql title=Query
WITH
    -- 참고: 130과 190 사이의 간격은 window 매개변수에 따라 ts = 180의 값이 어떻게 채워지는지 보여주기 위한 것입니다
    [110, 120, 130, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- 위 타임스탬프에 대응하는 값의 배열
    90 AS start_ts,       -- 타임스탬프 그리드의 시작
    90 + 135 AS end_ts,   -- 타임스탬프 그리드의 끝
    15 AS step_seconds,   -- 타임스탬프 그리드의 간격
    45 AS window_seconds  -- "staleness" 윈도우
SELECT timeSeriesChangesToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- 이 서브쿼리는 타임스탬프와 값의 배열을 `timestamp`, `value` 행으로 변환합니다
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesChangesToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,0,1,1,1,NULL,0,1,2]                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
```

**배열 인수를 사용한 동일한 쿼리**

```sql title=Query
WITH
    [110, 120, 130, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 5, 5, 8, 12, 13]::Array(Float32) AS values,
    90 AS start_ts,
    90 + 135 AS end_ts,
    15 AS step_seconds,
    45 AS window_seconds
SELECT timeSeriesChangesToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values);
```

```response title=Response
┌─timeSeriesChangesToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,0,1,1,1,NULL,0,1,2]                                                            │
└───────────────────────────────────────────────────────────────────────────────────────────┘
```
