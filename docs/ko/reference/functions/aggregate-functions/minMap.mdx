<div id="minMappedArrays">
  ## minMappedArrays
</div>

도입 버전: v20.5.0

`key` 배열에 지정된 키에 따라 `value` 배열의 최솟값을 계산합니다.

<Note>
  * 키 Tuple과 값 배열을 전달하는 것은 키 배열과 값 배열을 전달하는 것과 동일합니다.
  * 집계되는 각 행에서 `key`와 `value`의 요소 수는 반드시 같아야 합니다.
</Note>

**구문**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**인수**

* `key` — 키 배열입니다. [`Array(T)`](/ko/reference/data-types/array)
* `value` — 값 배열입니다. [`Array(T)`](/ko/reference/data-types/array)

**반환 값**

두 개의 배열로 이루어진 Tuple을 반환합니다. 첫 번째는 정렬된 순서의 키 배열이고, 두 번째는 해당 키에 대해 계산된 값 배열입니다. [`Tuple(Array(T), Array(T))`](/ko/reference/data-types/tuple)

**예시**

**사용 예시**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
