<div id="maxMappedArrays">
  ## maxMappedArrays
</div>

도입 버전: v20.5.0

`key` 배열에 지정된 키를 기준으로 `value` 배열의 최댓값을 계산합니다.

<Note>
  * 키 튜플과 값 배열을 전달하는 것은 키 배열과 값 배열을 전달하는 것과 동일합니다.
  * 집계되는 각 행에서 `key`와 `value`의 요소 수는 같아야 합니다.
</Note>

**구문**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**인수**

* `key` — 키 배열입니다. [`Array(T)`](/ko/reference/data-types/array)
* `value` — 값 배열입니다. [`Array(T)`](/ko/reference/data-types/array)

**반환 값**

정렬된 순서의 키와 해당 키에 대응하는 계산된 값을 담은 두 개의 배열로 구성된 튜플을 반환합니다. [`Tuple(Array(T), Array(T))`](/ko/reference/data-types/tuple)

**예시**

**사용 예시**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)───┐
│ (['x','y','z'],[2,3,1]) │
└─────────────────────────┘
```
