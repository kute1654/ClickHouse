<div id="anyLast">
  ## anyLast
</div>

도입 버전: v1.1.0

컬럼에서 마지막으로 처리된 값을 선택합니다.

<Warning>
  쿼리는 임의의 순서로 실행될 수 있으므로 이 함수의 결과는 결정적이지 않습니다.
  임의의 값이지만 결정적인 결과가 필요하면 [min](/ko/reference/functions/aggregate-functions/min) 또는 [max](/ko/reference/functions/aggregate-functions/max) 함수를 사용하십시오.
</Warning>

기본적으로 이 함수는 NULL을 반환하지 않으며, 즉 입력 컬럼의 NULL 값을 무시합니다.
하지만 이 함수를 `RESPECT NULLS` 수정자와 함께 사용하면 NULL 여부와 관계없이 마지막으로 읽은 값을 반환합니다.

**구문**

```sql
anyLast(column) [RESPECT NULLS]
```

**별칭**: `last_value`

**인수**

* `column` — 컬럼 이름입니다. [`Any`](/ko/reference/data-types/index)

**반환 값**

마지막으로 나타난 값을 반환합니다. [`Any`](/ko/reference/data-types/index)

**예시**

**사용 예시**

```sql title=Query
CREATE TABLE tab(city Nullable(String)) ENGINE=Memory;
INSERT INTO tab (city) VALUES ('Amsterdam'), (NULL), ('New York'), ('Tokyo'), ('Valencia'), (NULL);
SELECT anyLast(city), anyLastRespectNulls(city) FROM tab;
```

```response title=Response
┌─anyLast(city)─┬─anyLastRespectNulls(city)─┐
│ Valencia      │ ᴺᵁᴸᴸ                      │
└───────────────┴───────────────────────────┘
```
