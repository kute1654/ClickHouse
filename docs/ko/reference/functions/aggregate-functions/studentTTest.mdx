<div id="studentTTest">
  ## studentTTest
</div>

도입 버전: v21.1.0

두 모집단의 표본에 Student t-검정을 적용합니다.

두 표본의 값은 모두 `sample_data` 컬럼에 저장됩니다. `sample_index`가 0이면 해당 행의 값은 첫 번째 모집단의 표본에 속합니다. 그렇지 않으면 두 번째 모집단의 표본에 속합니다.
귀무가설은 두 모집단의 평균이 같다는 것입니다. 분산이 동일한 정규분포를 가정합니다.

**관련 항목**

* [Student t-검정](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [welchTTest 함수](/ko/reference/functions/aggregate-functions/welchTTest)

**구문**

```sql
studentTTest([confidence_level])(sample_data, sample_index)
```

**매개변수**

* `confidence_level` — 선택 사항입니다. 신뢰구간을 계산하기 위한 신뢰 수준입니다. [`Float`](/ko/reference/data-types/float)

**인수**

* `sample_data` — 샘플 데이터입니다. [`Integer`](/ko/reference/data-types/int-uint) 또는 [`Float`](/ko/reference/data-types/float) 또는 [`Decimal`](/ko/reference/data-types/decimal)
* `sample_index` — 샘플 인덱스입니다. [`Integer`](/ko/reference/data-types/int-uint)

**반환 값**

2개 또는 4개의 요소로 이루어진 튜플을 반환합니다(선택 사항인 `confidence_level`이 지정된 경우): 계산된 t-통계량, 계산된 p-값, [계산된 신뢰구간 하한], [계산된 신뢰구간 상한]. [`Tuple(Float64, Float64)`](/ko/reference/data-types/tuple) 또는 [`Tuple(Float64, Float64, Float64, Float64)`](/ko/reference/data-types/tuple)

**예시**

**기본 사용법**

```sql title=Query
SELECT studentTTest(sample_data, sample_index) FROM student_ttest;
```

```response title=Response
┌─studentTTest(sample_data, sample_index)───┐
│ (-0.21739130434783777,0.8385421208415731) │
└───────────────────────────────────────────┘
```
