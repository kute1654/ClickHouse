<div id="anyHeavy">
  ## anyHeavy
</div>

도입 버전: v1.1.0

[heavy hitters](https://doi.org/10.1145/762471.762473) 알고리즘을 사용해 자주 나타나는 값을 선택합니다.
쿼리의 각 실행 스레드에서 절반을 초과해 나타나는 값이 있으면 해당 값을 반환합니다.
일반적으로 결과는 결정적이지 않습니다.

**구문**

```sql
anyHeavy(column)
```

**인수**

* `column` — 컬럼 이름입니다. [`String`](/ko/reference/data-types/string)

**반환 값**

빈번하게 나타나는 값을 반환합니다. 결과는 결정적이지 않습니다. [`Any`](/ko/reference/data-types/index)

**예시**

**사용 예시**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
