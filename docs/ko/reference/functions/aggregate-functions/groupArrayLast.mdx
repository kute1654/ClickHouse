<div id="groupArrayLast">
  ## groupArrayLast
</div>

도입 버전: v23.1.0

마지막 인수 값으로 이루어진 배열을 생성합니다.
예를 들어 `groupArrayLast(1)(x)`는 `[anyLast(x)]`와 같습니다.
경우에 따라서는 실행 순서에 계속 의존할 수 있습니다.
이는 SELECT가 ORDER BY를 사용하는 서브쿼리를 기반으로 하며, 해당 서브쿼리의 결과가 충분히 작은 경우에 적용됩니다.

**구문**

```sql
groupArrayLast(max_size)(x)
```

**매개변수**

* `max_size` — 결과 배열의 최대 크기입니다. [`UInt64`](/ko/reference/data-types/int-uint)

**인수**

* `max_size` — 결과 배열의 최대 크기입니다. [`UInt64`](/ko/reference/data-types/int-uint)
* `x` — 인수입니다(컬럼 이름 또는 표현식). [`Any`](/ko/reference/data-types/index)

**반환 값**

마지막 인수 값들로 구성된 배열을 반환합니다. [`Array(T)`](/ko/reference/data-types/array)

**예시**

**사용 예시**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**groupArray와 비교**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
