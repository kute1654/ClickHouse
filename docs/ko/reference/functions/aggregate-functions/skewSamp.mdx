<div id="skewSamp">
  ## skewSamp
</div>

도입 버전: v20.1.0

시퀀스의 [표본 왜도](https://en.wikipedia.org/wiki/Skewness)를 계산합니다.

입력된 값이 해당 확률 변수의 표본인 경우, 이는 왜도의 비편향 추정량입니다.

**구문**

```sql
skewSamp(expr)
```

**인수**

* `expr` — 숫자를 반환하는 표현식입니다. [`표현식`](/ko/reference/data-types/special-data-types/expression)

**반환 값**

주어진 분포의 왜도를 반환합니다. `n <= 1`인 경우(`n`은 표본의 크기), 함수는 `nan`을 반환합니다. [`Float64`](/ko/reference/data-types/float)

**예시**

**대칭 분포**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**우측으로 치우친 분포**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
