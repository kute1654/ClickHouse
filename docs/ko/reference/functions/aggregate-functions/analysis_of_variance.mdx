<div id="analysisOfVariance">
  ## analysisOfVariance
</div>

도입 버전: v22.10.0

일원분산분석(ANOVA 검정)을 위한 통계적 검정을 제공합니다. 정규 분포를 따르는 여러 그룹의 관측값을 대상으로, 모든 그룹의 평균이 같은지 여부를 판별하는 검정입니다.

<Note>
  그룹 번호는 0부터 시작하며, 검정을 수행하려면 최소 2개의 그룹이 있어야 합니다.
  관측값이 2개 이상인 그룹이 최소 1개는 있어야 합니다.
</Note>

**구문**

```sql
analysisOfVariance(val, group_no)
```

**별칭**: `anova`

**인수**

* `val` — 값. [`(U)Int*`](/ko/reference/data-types/int-uint) 또는 [`Float*`](/ko/reference/data-types/float) 또는 [`Decimal`](/ko/reference/data-types/decimal)
* `group_no` — `val`이 속한 그룹 번호. [`(U)Int*`](/ko/reference/data-types/int-uint) 또는 [`Float*`](/ko/reference/data-types/float) 또는 [`Decimal`](/ko/reference/data-types/decimal)

**반환 값**

F-통계량과 p-값으로 구성된 튜플을 반환합니다. [`Tuple(Float64, Float64)`](/ko/reference/data-types/tuple)

**예시**

**기본 사용법**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

```response title=Response
┌─analysisOfVariance(number, modulo(number, 2))─┐
│ (0,1)                                         │
└───────────────────────────────────────────────┘
```
