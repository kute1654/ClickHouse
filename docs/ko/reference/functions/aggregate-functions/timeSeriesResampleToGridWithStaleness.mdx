<div id="timeSeriesResampleToGridWithStaleness">
  ## timeSeriesResampleToGridWithStaleness
</div>

도입 버전: v25.6.0

타임스탬프와 값의 쌍으로 이루어진 시계열(time series) 데이터를 받아, 시작 타임스탬프, 종료 타임스탬프, 간격으로 정의된 규칙적인 시간 그리드로 리샘플링하는 집계 함수입니다. 그리드의 각 지점에서는 가장 최근의 샘플(지정된 시간 윈도우 내)을 선택합니다.

별칭: `timeSeriesLastToGrid`.

<Warning>
  이 함수는 실험적 기능이므로, `allow_experimental_ts_to_grid_aggregate_function=true`를 설정하여 활성화하십시오.
</Warning>

**구문**

```sql
timeSeriesResampleToGridWithStaleness(start_timestamp, end_timestamp, grid_step, staleness_window)(timestamp, value)
```

**별칭**: `timeSeriesLastToGrid`

**매개변수**

* `start_timestamp` — 그리드의 시작 시점을 지정합니다. [`UInt32`](/ko/reference/data-types/int-uint) 또는 [`DateTime`](/ko/reference/data-types/datetime)
* `end_timestamp` — 그리드의 종료 시점을 지정합니다. [`UInt32`](/ko/reference/data-types/int-uint) 또는 [`DateTime`](/ko/reference/data-types/datetime)
* `grid_step` — 초 단위 그리드 간격을 지정합니다. [`UInt32`](/ko/reference/data-types/int-uint)
* `staleness_window` — 가장 최근 샘플에 대해 허용되는 최대 staleness를 초 단위로 지정합니다. [`UInt32`](/ko/reference/data-types/int-uint)

**인수**

* `timestamp` — 샘플의 타임스탬프입니다. 개별 값 또는 배열일 수 있습니다. [`UInt32`](/ko/reference/data-types/int-uint) 또는 [`DateTime`](/ko/reference/data-types/datetime) 또는 [`Array(UInt32)`](/ko/reference/data-types/array) 또는 [`Array(DateTime)`](/ko/reference/data-types/array)
* `value` — 타임스탬프에 해당하는 시계열 값입니다. 개별 값 또는 배열일 수 있습니다. [`Float*`](/ko/reference/data-types/float) 또는 [`Array(Float*)`](/ko/reference/data-types/array)

**반환 값**

지정된 그리드로 리샘플링된 시계열 값을 반환합니다. 반환되는 배열에는 각 시간 그리드 지점마다 하나의 값이 포함됩니다. 특정 그리드 지점에 대한 샘플이 없으면 해당 값은 NULL입니다. [`Array(Nullable(Float64))`](/ko/reference/data-types/array)

**예시**

**개별 타임스탬프-값 쌍을 사용하는 기본 사용법**

```sql title=Query
WITH
    -- 참고: 140과 190 사이의 간격은 staleness 윈도우 매개변수에 따라 ts = 150, 165, 180의 값이 어떻게 채워지는지 보여줍니다
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- 위 타임스탬프에 대응하는 값 배열
    90 AS start_ts,       -- 타임스탬프 그리드 시작
    90 + 120 AS end_ts,   -- 타임스탬프 그리드 끝
    15 AS step_seconds,   -- 타임스탬프 그리드 간격
    30 AS window_seconds  -- "staleness" 윈도우
SELECT timeSeriesResampleToGridWithStaleness(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- 이 서브쿼리는 타임스탬프와 값의 배열을 `timestamp`, `value` 행으로 변환합니다
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesResampleToGridWithStaleness(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,1,3,4,4,NULL,5,8]                                                                           │
└────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

**배열 인수 사용하기**

```sql title=Query
WITH
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values,
    90 AS start_ts,
    90 + 120 AS end_ts,
    15 AS step_seconds,
    30 AS window_seconds
SELECT timeSeriesResampleToGridWithStaleness(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values);
```

```response title=Response
┌─timeSeriesResampleToGridWithStaleness(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values)─┐
│ [NULL,NULL,1,3,4,4,NULL,5,8]                                                                             │
└──────────────────────────────────────────────────────────────────────────────────────────────────────┘
```
