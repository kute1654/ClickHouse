<div id="cramersVBiasCorrected">
  ## cramersVBiasCorrected
</div>

도입 버전: v22.1.0

Cramer&#39;s V는 테이블의 두 컬럼 간 연관성을 나타내는 척도입니다.
[`cramersV` 함수](/ko/reference/functions/aggregate-functions/cramersV)의 결과는 0(변수 간 연관성이 없음을 의미)에서 1 사이이며, 각 값이 다른 값에 의해 완전히 결정될 때만 1에 도달할 수 있습니다.
이 함수는 편향이 크게 나타날 수 있으므로, 이 버전의 Cramer&#39;s V는 [편향 보정](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V#Bias_correction)을 사용합니다.

**구문**

```sql
cramersVBiasCorrected(column1, column2)
```

**인수**

* `column1` — 비교할 첫 번째 컬럼입니다. [`Any`](/ko/reference/data-types/index)
* `column2` — 비교할 두 번째 컬럼입니다. [`Any`](/ko/reference/data-types/index)

**반환 값**

0(컬럼 값 간 연관성이 없음을 의미)부터 1(완전한 연관성)에 이르는 값을 반환합니다. [`Float64`](/ko/reference/data-types/float)

**예시**

**일반 cramersV와 비교**

```sql title=Query
SELECT
    cramersV(a, b),
    cramersVBiasCorrected(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 4 AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─────cramersV(a, b)─┬─cramersVBiasCorrected(a, b)─┐
│ 0.5798088336225178 │          0.5305112825189074 │
└────────────────────┴─────────────────────────────┘
```
