다음 작업을 수행할 수 있습니다:

<div id="modify">
  ## 수정
</div>

```sql
ALTER TABLE [db].name [ON CLUSTER cluster] MODIFY SAMPLE BY new_expression
```

이 명령은 테이블의 [샘플링 키](/ko/reference/engines/table-engines/mergetree-family/mergetree)를 `new_expression`(표현식 또는 표현식의 튜플)로 변경합니다. 기본 키에는 새 샘플링 키가 포함되어야 합니다.

<div id="remove">
  ## 삭제
</div>

```sql
ALTER TABLE [db].name [ON CLUSTER cluster] REMOVE SAMPLE BY
```

이 명령은 테이블의 [샘플링 키](/ko/reference/engines/table-engines/mergetree-family/mergetree)를 제거합니다.

`MODIFY` 및 `REMOVE` 명령은 메타데이터만 변경하거나 파일만 제거하므로 가볍습니다.

<Note>
  이 기능은 [MergeTree](/ko/reference/engines/table-engines/mergetree-family/mergetree) 계열 테이블([복제된](/ko/reference/engines/table-engines/mergetree-family/replication) 테이블 포함)에서만 작동합니다.
</Note>
