사용자는 SQL 문으로 ClickHouse를 사용합니다. ClickHouse는 [SELECT](/ko/reference/statements/select/index), [CREATE](/ko/reference/statements/create/index)와 같은 일반적인 SQL 문을 지원하며, [KILL](/ko/reference/statements/kill), [OPTIMIZE](/ko/reference/statements/optimize)와 같은 특화된 SQL 문도 제공합니다.
