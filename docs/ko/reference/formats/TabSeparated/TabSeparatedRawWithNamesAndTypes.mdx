<div id="description">
  ## 설명
</div>

[`TabSeparatedWithNamesAndTypes`](/ko/reference/formats/TabSeparated/TabSeparatedWithNamesAndTypes) 포맷과 다른 점은
행이 이스케이프 처리 없이 기록된다는 것입니다.

<Note>
  이 포맷으로 파싱할 때는 각 필드에 탭이나 줄바꿈 문자를 포함할 수 없습니다.
</Note>
