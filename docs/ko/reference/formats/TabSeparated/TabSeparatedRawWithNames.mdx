<div id="description">
  ## 설명
</div>

[`TabSeparatedWithNames`](/ko/reference/formats/TabSeparated/TabSeparatedWithNames) 포맷과는 달리,
행이 이스케이프 처리 없이 기록됩니다.

<Note>
  이 포맷으로 파싱할 때는 각 필드에 탭 또는 줄바꿈 문자를 사용할 수 없습니다.
</Note>

```sql

FROM football
FORMAT TabSeparatedRawWithNames
```
