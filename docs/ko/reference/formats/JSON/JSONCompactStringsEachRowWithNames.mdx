<div id="description">
  ## 설명
</div>

[`JSONCompactEachRow`](/ko/reference/formats/JSON/JSONCompactEachRow) 포맷과의 차이점은 [TabSeparatedWithNames](/ko/reference/formats/TabSeparated/TabSeparatedWithNames) 포맷과 마찬가지로 컬럼 이름이 포함된 헤더 행도 출력한다는 점입니다.

<div id="example-usage">
  ## 사용 예시
</div>

<div id="inserting-data">

</div>

```sql

```

<div id="format-settings">
  ## 포맷 설정
</div>

<Note>
  설정 [`input_format_with_names_use_header`](/ko/reference/settings/formats#input_format_with_names_use_header)가 `1`로 설정되면
  입력 데이터의 컬럼이 이름을 기준으로 테이블 컬럼에 매핑되며, 설정 [`input_format_skip_unknown_fields`](/ko/reference/settings/formats#input_format_skip_unknown_fields)가 `1`로 설정된 경우 알 수 없는 이름의 컬럼은 건너뜁니다.
  그렇지 않으면 첫 번째 행을 건너뜁니다.
</Note>
