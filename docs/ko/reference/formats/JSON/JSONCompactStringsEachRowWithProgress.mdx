<div id="description">
  ## 설명
</div>

[`JSONCompactEachRowWithProgress`](/ko/reference/formats/JSON/JSONCompactEachRowWithProgress)와 비슷하지만, 모든 값이 문자열로 변환됩니다.
모든 데이터 타입을 일관된 문자열 형태로 표현해야 할 때 유용합니다.

주요 기능:

* `JSONCompactEachRowWithProgress`와 동일한 구조
* 모든 값이 문자열로 표현됩니다(숫자, 배열 등도 모두 따옴표로 묶인 문자열임)
* 진행률 업데이트, 합계, 예외 처리를 포함합니다
* 문자열 기반 데이터를 선호하거나 필요로 하는 클라이언트에 유용합니다

<div id="example-usage">
  ## 사용 예시
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactStringsEachRowWithProgress
```
