<div id="description">
  ## 설명
</div>

이 포맷에서는 모든 데이터가 하나의 JSON 배열로 표현됩니다.

<Note>
  `JSONCompactColumns` 출력 형식은 모든 데이터를 메모리에 버퍼링한 뒤 하나의 블록으로 출력하므로 메모리 사용량이 높아질 수 있습니다.
</Note>

</div>
