<div id="description">
  ## 설명
</div>

이 포맷에서는 ClickHouse가 각 행을 줄바꿈으로 구분된 개별 JSON 객체로 출력합니다.

이 포맷은 `JSONEachRow`, `JSONLines`, `NDJSON`(Newline Delimited JSON), 또는 `JSONL`라고도 합니다. 이 이름들은 모두 동일한 포맷의 별칭이며 입력과 출력 모두에서 서로 바꿔서 사용할 수 있습니다.

</div>

FORMAT JSONLines
