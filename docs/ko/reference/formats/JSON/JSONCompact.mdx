<div id="description">
  ## 설명
</div>

[JSON](/ko/reference/formats/JSON/JSON)과 다른 점은 데이터 행이 객체가 아니라 배열로 출력된다는 점뿐입니다.

  ## 사용 예시
</div>

<div id="inserting-data">

            "name": "home_team_goals",

```sql

```

<div id="reading-data">
  ### 데이터 읽기
</div>

`JSONCompact` 포맷으로 데이터를 읽습니다:

```sql
SELECT *
FROM football
FORMAT JSONCompact
```
