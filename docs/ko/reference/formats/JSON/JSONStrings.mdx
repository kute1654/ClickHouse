<div id="description">
  ## 설명
</div>

[JSON](/ko/reference/formats/JSON/JSON) 포맷과의 차이점은 데이터 필드가 타입이 지정된 JSON 값이 아니라 문자열로 출력된다는 점뿐입니다. 이 포맷은 출력 전용입니다.

<div id="example-usage">
  ## 사용 예시
</div>

<div id="reading-data">
  ### 데이터 읽기
</div>

다음과 같이 `JSONStrings` 포맷을 사용해 데이터를 읽습니다:

```sql
SELECT *
FROM football
FORMAT JSONStrings
```
