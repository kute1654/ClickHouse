| ✗  | ✔  | `PrettyJSONLines`, `PrettyNDJSON` |

<div id="description">
  ## 설명
</div>

JSON에 줄바꿈 구분 기호와 공백 4칸 들여쓰기를 적용해 보기 좋게 포맷했다는 점만 제외하면 [JSONEachRow](/ko/reference/formats/JSON/JSONEachRow)와 동일합니다.

<div id="example-usage">
  ## 예시 사용법
</div>

<div id="reading-data">
  ### 데이터 읽기
</div>

`PrettyJSONEachRow` 포맷을 사용해 데이터를 읽습니다:

```sql
SELECT *
FROM football
FORMAT PrettyJSONEachRow
```
