| [`input_format_json_empty_as_default`](/ko/reference/settings/formats#input_format_json_empty_as_default)                                                         | JSON 입력에서 비어 있는 필드를 기본값으로 처리합니다.                                                            | `false` | 복잡한 기본값 표현식을 사용하려면 [input&#95;format&#95;defaults&#95;for&#95;omitted&#95;fields](/ko/reference/settings/formats#input_format_defaults_for_omitted_fields)도 함께 활성화해야 합니다. |
| [`output_format_json_quote_64bit_integers`](/ko/reference/settings/formats#output_format_json_quote_64bit_integers)                                               | JSON 출력 형식에서 64비트 정수에 따옴표를 붙일지 제어합니다.                                                       | `true`  |                                                                                                                                                                        |

| [`output_format_json_named_tuples_as_objects`](/ko/reference/settings/formats#output_format_json_named_tuples_as_objects)                                         | named tuple 컬럼을 JSON 객체로 직렬화합니다.                                                            | `true`  |                                                                                                                                                                        |
