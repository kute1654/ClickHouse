<div id="description">
  ## 설명
</div>

[Prometheus 텍스트 기반 노출 포맷](https://prometheus.io/docs/instrumenting/exposition_formats/#text-based-format)으로 메트릭을 노출합니다.

이 포맷에서는 다음 규칙에 따라 출력 테이블이 올바르게 구성되어야 합니다.

* `name`([String](/ko/reference/data-types/string)) 및 `value`(숫자) 컬럼이 필수입니다.
* 행에는 선택적으로 `help`([String](/ko/reference/data-types/string)) 및 `timestamp`(숫자)가 포함될 수 있습니다.
* `type`([String](/ko/reference/data-types/string)) 컬럼은 `counter`, `gauge`, `histogram`, `summary`, `untyped` 중 하나이거나 비어 있어야 합니다.
* 각 메트릭 값에는 `labels`([Map(String, String)](/ko/reference/data-types/map))가 추가로 포함될 수도 있습니다.
* 연속된 여러 행이 서로 다른 레이블을 사용해 동일한 메트릭을 나타낼 수 있습니다. 테이블은 메트릭 이름으로 정렬되어야 합니다(예: `ORDER BY name`).

`histogram` 및 `summary` 레이블에는 특별한 요구 사항이 있습니다. 자세한 내용은 [Prometheus 문서](https://prometheus.io/docs/instrumenting/exposition_formats/#histograms-and-summaries)를 참조하십시오.
`{'count':''}` 및 `{'sum':''}` 레이블이 있는 행에는 특별 규칙이 적용되며, 각각 `<metric_name>_count` 및 `<metric_name>_sum`으로 변환됩니다.

│ http_request_duration_seconds       │ histogram │                                           │ {'le':'0.5'}                   │   129389 │             0 │

다음과 같이 포맷됩니다:

```text
# HELP http_request_duration_seconds 요청 지속 시간의 히스토그램.
# TYPE http_request_duration_seconds histogram
http_request_duration_seconds_bucket{le="0.05"} 24054
http_request_duration_seconds_bucket{le="0.1"} 33444
http_request_duration_seconds_bucket{le="0.5"} 129389
http_request_duration_seconds_bucket{le="1"} 133988
http_request_duration_seconds_bucket{le="+Inf"} 144320
http_request_duration_seconds_sum 53423
http_request_duration_seconds_count 144320

# HELP http_requests_total HTTP 요청의 총 수
# TYPE http_requests_total counter
http_requests_total{code="200",method="post"} 1027 1395066363000
http_requests_total{code="400",method="post"} 3 1395066363000

metric_without_timestamp_and_labels 12.47

# HELP rpc_duration_seconds RPC 지속 시간(초)의 요약.
# TYPE rpc_duration_seconds summary
rpc_duration_seconds{quantile="0.01"} 3102
rpc_duration_seconds{quantile="0.05"} 3272
rpc_duration_seconds{quantile="0.5"} 4773
rpc_duration_seconds{quantile="0.9"} 9001
rpc_duration_seconds{quantile="0.99"} 76656
rpc_duration_seconds_sum 17560473
rpc_duration_seconds_count 2693

something_weird{problem="division by zero"} +Inf -3982045
```
