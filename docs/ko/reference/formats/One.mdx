<div id="description">
  ## 설명
</div>

`One` 포맷은 파일에서 데이터를 전혀 읽지 않고, [`UInt8`](/ko/reference/data-types/int-uint) 타입의 컬럼 하나만 있는 1개의 행만 반환하는 특수한 입력 형식입니다. 이 컬럼의 이름은 `dummy`이고 값은 `0`입니다(`system.one` 테이블과 유사).
실제 데이터를 읽지 않고 모든 파일을 나열할 때 가상 컬럼 `_file/_path`와 함께 사용할 수 있습니다.

<div id="example-usage">
  ## 사용 예시
</div>

예시:

```sql title="Query"
SELECT _file FROM file('path/to/files/data*', One);
```

```text title="Response"
┌─_file────┐
│ data.csv │
└──────────┘
┌─_file──────┐
│ data.jsonl │
└────────────┘
┌─_file────┐
│ data.tsv │
└──────────┘
┌─_file────────┐
│ data.parquet │
└──────────────┘
```
