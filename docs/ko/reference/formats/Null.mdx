<div id="description">
  ## 설명
</div>

`Null` 포맷에서는 아무것도 출력되지 않습니다.
처음에는 이상하게 느껴질 수 있지만, 아무것도 출력되지 않아도 쿼리는 계속 처리되며,
command-line client를 사용할 경우 데이터는 여전히 클라이언트로 전송된다는 점에 유의해야 합니다.

<Tip>
  `Null` 포맷은 성능 테스트에 유용하게 사용할 수 있습니다.
</Tip>
