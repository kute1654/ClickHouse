<div id="description">
  ## 설명
</div>

`XML` 포맷은 출력에만 적합하며 파싱에는 적합하지 않습니다.

컬럼 이름이 허용 가능한 포맷이 아니면 요소 이름으로는 단순히 &#39;field&#39;가 사용됩니다. 일반적으로 XML 구조는 JSON 구조를 따릅니다.
JSON과 마찬가지로 잘못된 UTF-8 시퀀스는 대체 문자 `�`로 바뀌므로, 출력 텍스트는 유효한 UTF-8 시퀀스로만 구성됩니다.

문자열 값에서는 문자 `<` 및 `&`가 `<` 및 `&`로 이스케이프됩니다.

배열은 `<array><elem>Hello</elem><elem>World</elem>...</array>`로 출력되며, 튜플은 `<tuple><elem>Hello</elem><elem>World</elem>...</tuple>`로 출력됩니다.
