<div id="description">
  ## 설명
</div>

일반 함수와 집계 함수에 관한 정보입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 함수의 이름입니다.
* `is_aggregate` ([UInt8](/ko/reference/data-types/index)) — 함수가 집계 함수인지 여부입니다.
* `case_insensitive` ([UInt8](/ko/reference/data-types/index)) — 함수 이름을 대소문자를 구분하지 않고 사용할 수 있는지 여부입니다.
* `alias_to` ([String](/ko/reference/data-types/index)) — 함수 이름이 별칭인 경우 원래 함수 이름입니다.
* `create_query` ([String](/ko/reference/data-types/index)) — 더 이상 사용되지 않습니다.
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/ko/reference/data-types/index)) — 더 이상 사용되지 않습니다.
* `description` ([String](/ko/reference/data-types/index)) — 함수가 수행하는 작업에 대한 상위 수준 설명입니다.
* `syntax` ([String](/ko/reference/data-types/index)) — 함수의 시그니처입니다.
* `arguments` ([String](/ko/reference/data-types/index)) — 함수 인수입니다.
* `parameters` ([String](/ko/reference/data-types/index)) — 함수 매개변수입니다(집계 함수에만 해당).
* `returned_value` ([String](/ko/reference/data-types/index)) — 함수가 반환하는 값입니다.
* `examples` ([String](/ko/reference/data-types/index)) — 사용 예시입니다.
* `introduced_in` ([String](/ko/reference/data-types/index)) — 함수가 처음 도입된 ClickHouse 버전입니다.
* `categories` ([String](/ko/reference/data-types/index)) — 함수의 범주입니다.
* `deterministic` ([Nullable(UInt8)](/ko/reference/data-types/index)) — 동일한 인수에 대해 함수가 항상 같은 결과를 반환하는지 여부입니다. 알 수 없는 경우(예: 집계 함수 또는 사용자 정의 함수)에는 NULL입니다.
* `higher_order` ([Nullable(UInt8)](/ko/reference/data-types/index)) — 함수가 고차 함수인지 여부입니다. 즉, 최소 하나의 람다 표현식을 인수로 받는 함수인지 여부입니다(예: arrayMap, arrayFilter, mapApply). 알 수 없는 경우에는 NULL입니다.

{/*AUTOGENERATED_END*/}
