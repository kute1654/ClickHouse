<div id="description">
  ## 설명
</div>

현재 사용자의 QUOTA 사용량: 사용된 양과 남은 양입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ko/reference/data-types/index)) — QUOTA 이름입니다.
* `quota_key` ([String](/ko/reference/data-types/index)) — 키 값입니다.
* `start_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 리소스 활용을 계산하기 위한 시작 시간입니다.
* `end_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 리소스 활용을 계산하기 위한 종료 시간입니다.
* `duration` ([Nullable(UInt32)](/ko/reference/data-types/index)) — 리소스 활용을 계산하기 위한 시간 인터벌의 길이(초)입니다.
* `queries` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재까지 실행된 쿼리 수입니다.
* `max_queries` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 모든 유형의 쿼리 수의 최대값입니다.
* `query_selects` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재까지 실행된 SELECT 쿼리 수입니다.
* `max_query_selects` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 SELECT 쿼리 수의 최대값입니다.
* `query_inserts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재까지 실행된 INSERT 쿼리 수입니다.
* `max_query_inserts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 INSERT 쿼리 수의 최대값입니다.
* `errors` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재까지 오류가 발생한 쿼리 수입니다.
* `max_errors` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 오류 발생 쿼리 수의 최대값입니다.
* `result_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 모든 쿼리의 결과 집합(result set)에 포함된 행의 총개수입니다.
* `max_result_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 모든 쿼리의 결과 집합(result set) 내 행의 총최대 개수입니다.
* `result_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 모든 쿼리의 결과 집합(result set)에 포함된 바이트의 총개수입니다.
* `max_result_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 모든 쿼리의 결과 집합(result set) 내 바이트의 총최대 개수입니다.
* `read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 모든 쿼리 실행 중 읽은 행의 총개수입니다.
* `max_read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 행의 최대 개수입니다.
* `read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 모든 쿼리 실행 중 읽은 바이트의 총개수입니다.
* `max_read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 바이트의 최대 개수입니다.
* `execution_time` ([Nullable(Float64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 쿼리 실행에 소요된 총시간(나노초)입니다.
* `max_execution_time` ([Nullable(Float64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내 모든 쿼리 실행에 허용되는 최대 시간(나노초)입니다.
* `written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 모든 쿼리 실행 중 기록된 바이트의 총개수입니다.
* `max_written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 모든 쿼리 실행 중 기록할 수 있는 바이트의 최대 개수입니다.
* `failed_sequential_authentications` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 연속된 authentication 실패 횟수입니다.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 연속된 authentication 실패 횟수의 최대값입니다.
* `queries_per_normalized_hash` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 시간 주기 내 단일 정규화된 쿼리의 최대 실행 횟수입니다.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 시간 주기 내에 허용되는 단일 정규화된 쿼리의 최대 실행 횟수입니다.
