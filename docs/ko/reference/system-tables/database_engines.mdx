<div id="description">
  ## 설명
</div>

서버에서 지원하는 데이터베이스 엔진 목록입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 데이터베이스 엔진의 이름입니다.
* `description` ([String](/ko/reference/data-types/index)) — 데이터베이스 엔진의 역할을 개괄적으로 설명한 내용입니다.
* `syntax` ([String](/ko/reference/data-types/index)) — CREATE DATABASE 쿼리의 ENGINE 절에서 데이터베이스 엔진을 지정하는 방식입니다.
* `examples` ([String](/ko/reference/data-types/index)) — 사용 예시입니다.
* `introduced_in` ([String](/ko/reference/data-types/index)) — 데이터베이스 엔진이 처음 도입된 ClickHouse 버전이며, 형식은 major.minor입니다.
* `related` ([Array(String)](/ko/reference/data-types/index)) — 관련 데이터베이스 엔진의 이름 목록입니다.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```

```text title="Response"

│ Atomic   │
```
