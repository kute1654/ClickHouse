<div id="description">
  ## 설명
</div>

Kafka 컨슈머 관련 정보를 제공합니다.
[Kafka 테이블 엔진](/ko/reference/engines/table-engines/integrations/kafka)에 적용됩니다(ClickHouse 네이티브 통합)

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — Kafka Engine를 사용하는 테이블이 속한 데이터베이스입니다.
* `table` ([String](/ko/reference/data-types/index)) — Kafka Engine를 사용하는 테이블의 이름입니다.
* `consumer_id` ([String](/ko/reference/data-types/index)) — Kafka 컨슈머 식별자입니다. 하나의 테이블에는 여러 컨슈머가 있을 수 있습니다. `kafka_num_consumers` 매개변수로 지정됩니다.
* `assignments.topic` ([Array(String)](/ko/reference/data-types/index)) — Kafka 토픽입니다.
* `assignments.partition_id` ([Array(Int32)](/ko/reference/data-types/index)) — Kafka 파티션 ID입니다. 하나의 파티션에는 하나의 컨슈머만 할당될 수 있습니다.
* `assignments.current_offset` ([Array(Int64)](/ko/reference/data-types/index)) — 현재 offset입니다.
* `assignments.intent_size` ([Array(Nullable(Int64))](/ko/reference/data-types/index)) — 새 StorageKafka에서 push되었지만 아직 커밋되지 않은 메시지 수입니다.
* `exceptions.time` ([Array(DateTime)](/ko/reference/data-types/index)) — 가장 최근 예외 10개가 생성된 시점의 타임스탬프입니다.
* `exceptions.text` ([Array(String)](/ko/reference/data-types/index)) — 가장 최근 예외 10개의 텍스트입니다.
* `last_poll_time` ([DateTime](/ko/reference/data-types/index)) — 가장 최근 poll의 타임스탬프입니다.
* `num_messages_read` ([UInt64](/ko/reference/data-types/index)) — 컨슈머가 읽은 메시지 수입니다.
* `last_commit_time` ([DateTime](/ko/reference/data-types/index)) — 가장 최근 poll의 타임스탬프입니다.
* `num_commits` ([UInt64](/ko/reference/data-types/index)) — 해당 컨슈머의 총 커밋 수입니다.
* `last_rebalance_time` ([DateTime](/ko/reference/data-types/index)) — 가장 최근 Kafka 리밸런싱의 타임스탬프입니다.
* `num_rebalance_revocations` ([UInt64](/ko/reference/data-types/index)) — 컨슈머의 파티션 할당이 취소된 횟수입니다.
* `num_rebalance_assignments` ([UInt64](/ko/reference/data-types/index)) — 컨슈머가 Kafka cluster에 할당된 횟수입니다.
* `is_currently_used` ([UInt8](/ko/reference/data-types/index)) — 컨슈머가 현재 사용 중인지 나타내는 플래그입니다.
* `last_used` ([DateTime64(6)](/ko/reference/data-types/index)) — 이 컨슈머가 마지막으로 사용된 시각입니다.
* `rdkafka_stat` ([String](/ko/reference/data-types/index)) — 라이브러리 내부 통계 정보입니다. 비활성화하려면 statistics&#95;interval&#95;ms를 0으로 설정하십시오. 기본값은 3000(3초마다 1회)입니다.
* `dependencies` ([Array(Array(String))](/ko/reference/data-types/index)) — 전이적 데이터베이스 종속성입니다.
* `missing_dependencies` ([Array(Array(String))](/ko/reference/data-types/index)) — 누락된 전이적 데이터베이스 종속성입니다.

{/*AUTOGENERATED_END*/}
