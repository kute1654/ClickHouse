<div id="description">
  ## 설명
</div>

현재 시점에 활성화된 모든 역할이 포함됩니다. 여기에는 현재 사용자의 현재 역할과 현재 역할에 부여된 역할이 포함됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/ko/reference/data-types/index)) — 역할 이름입니다.
* `with_admin_option` ([UInt8](/ko/reference/data-types/index)) — 역할에 ADMIN OPTION 권한이 있으면 1입니다.
* `is_current` ([UInt8](/ko/reference/data-types/index)) — `enabled_role`이 현재 사용자의 현재 역할인지 나타내는 플래그입니다.
* `is_default` ([UInt8](/ko/reference/data-types/index)) — `enabled_role`이 기본 역할인지 나타내는 플래그입니다.

{/*AUTOGENERATED_END*/}
