<div id="description">
  ## 설명
</div>

모든 로그인 및 로그아웃 성공/실패 이벤트에 대한 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 쿼리를 실행하는 서버의 호스트명입니다.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/ko/reference/data-types/enum)) — 로그인/로그아웃 결과입니다. 가능한 값: LoginFailure — 로그인 오류. LoginSuccess — 로그인 성공. Logout — 시스템에서 로그아웃.
* `auth_id` ([UUID](/ko/reference/data-types/uuid)) — 인증 ID로, 사용자가 로그인할 때마다 자동으로 생성되는 UUID입니다.
* `session_id` ([String](/ko/reference/data-types/string)) — 클라이언트가 HTTP 인터페이스를 통해 전달하는 세션 ID입니다.
* `event_date` ([Date](/ko/reference/data-types/date)) — 로그인/로그아웃 날짜입니다.
* `event_time` ([DateTime](/ko/reference/data-types/datetime)) — 로그인/로그아웃 시각입니다.
* `event_time_microseconds` ([DateTime64(6)](/ko/reference/data-types/datetime64)) — 마이크로초 정밀도의 로그인/로그아웃 시작 시각입니다.
* `user` ([Nullable(String)](/ko/reference/data-types/nullable)) — 사용자 이름입니다.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/ko/reference/data-types/nullable)) — 인증 유형입니다.
* `profiles` ([Array(LowCardinality(String))](/ko/reference/data-types/array)) — 모든 역할 및/또는 사용자에 설정된 프로필 목록입니다.
* `roles` ([Array(LowCardinality(String))](/ko/reference/data-types/array)) — 프로필이 적용되는 역할 목록입니다.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/ko/reference/data-types/array)) — 클라이언트가 로그인/로그아웃할 때 변경된 설정입니다.
* `client_address` ([IPv6](/ko/reference/data-types/ipv6)) — 로그인/로그아웃에 사용된 IP 주소입니다.
* `client_port` ([UInt16](/ko/reference/data-types/int-uint)) — 로그인/로그아웃에 사용된 클라이언트 포트입니다.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/ko/reference/data-types/enum)) — 로그인이 시작된 인터페이스입니다.
* `client_hostname` ([String](/ko/reference/data-types/string)) — `clickhouse-client` 또는 다른 TCP 클라이언트가 실행되는 클라이언트 머신의 호스트명입니다.
* `client_name` ([String](/ko/reference/data-types/string)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 이름입니다.
* `client_revision` ([UInt32](/ko/reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 리비전입니다.
* `client_version_major` ([UInt32](/ko/reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 메이저 버전입니다.
* `client_version_minor` ([UInt32](/ko/reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트의 마이너 버전입니다.
* `client_version_patch` ([UInt32](/ko/reference/data-types/int-uint)) — `clickhouse-client` 또는 다른 TCP 클라이언트 버전의 패치 구성 요소입니다.
* `failure_reason` ([String](/ko/reference/data-types/string)) — 로그인/로그아웃 실패 원인이 포함된 예외 메시지입니다.
* `certificate_subjects` ([Array(LowCardinality(String))](/ko/reference/data-types/array)) — 연결 시 제시된 TLS 클라이언트 인증서의 주체 목록(Common Name 및 Subject Alternative Name)으로, &#39;CN:...&#39; / &#39;SAN:...&#39; 형식입니다. 인증서가 제시되지 않은 경우 비어 있습니다.
* `certificate_serial` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — TLS 클라이언트 인증서의 일련번호입니다. 인증서가 제시되지 않은 경우 비어 있습니다.
* `certificate_issuer` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — TLS 클라이언트 인증서의 발급자입니다. 인증서가 제시되지 않은 경우 비어 있습니다.
* `certificate_not_before` ([Nullable(DateTime64)](/ko/reference/data-types/nullable)) — TLS 클라이언트 인증서가 유효해지는 시각입니다. 인증서가 제시되지 않은 경우 NULL입니다.
* `certificate_not_after` ([Nullable(DateTime64)](/ko/reference/data-types/nullable)) — TLS 클라이언트 인증서가 만료되는 시각입니다. 인증서가 제시되지 않은 경우 NULL입니다.

{/*AUTOGENERATED_END*/}
