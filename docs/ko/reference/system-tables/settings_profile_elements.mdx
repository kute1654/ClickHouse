<div id="description">
  ## 설명
</div>

설정 프로필에 포함된 내용은 다음과 같습니다:

* 제약 조건
* 설정이 적용되는 역할 및 사용자
* 상위 설정 프로필

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/ko/reference/data-types/index)) — 설정 프로필 이름입니다.
* `user_name` ([Nullable(String)](/ko/reference/data-types/index)) — 사용자 이름입니다.
* `role_name` ([Nullable(String)](/ko/reference/data-types/index)) — 역할 이름입니다.
* `index` ([UInt64](/ko/reference/data-types/index)) — 설정 프로필 요소의 순번입니다.
* `setting_name` ([Nullable(String)](/ko/reference/data-types/index)) — 설정 이름입니다.
* `value` ([Nullable(String)](/ko/reference/data-types/index)) — 설정 값입니다.
* `min` ([Nullable(String)](/ko/reference/data-types/index)) — 설정의 최솟값입니다. 설정되지 않은 경우 NULL입니다.
* `max` ([Nullable(String)](/ko/reference/data-types/index)) — 설정의 최댓값입니다. 설정되지 않은 경우 NULL입니다.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/ko/reference/data-types/index)) — 설정 변경 가능 여부를 나타내는 속성입니다.
* `inherit_profile` ([Nullable(String)](/ko/reference/data-types/index)) — 이 설정 프로필의 부모 프로필입니다. 설정되지 않은 경우 NULL입니다. 설정 프로필은 부모 프로필로부터 모든 설정 값과 제약 조건(min, max, readonly)을 상속합니다.

{/*AUTOGENERATED_END*/}
