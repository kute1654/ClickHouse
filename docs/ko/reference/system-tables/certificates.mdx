<div id="description">
  ## 설명
</div>

사용 가능한 인증서와 해당 인증서의 출처에 관한 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/ko/reference/data-types/index)) — 인증서의 버전입니다. 값은 v1은 0, v2는 1, v3는 2입니다.
* `serial_number` ([Nullable(String)](/ko/reference/data-types/index)) — 발급자가 인증서에 할당한 일련번호입니다.
* `signature_algo` ([Nullable(String)](/ko/reference/data-types/index)) — 서명 알고리즘입니다. 발급자가 이 인증서에 서명할 때 사용하는 알고리즘입니다.
* `issuer` ([Nullable(String)](/ko/reference/data-types/index)) — 발급자입니다. 이 인증서를 발급한 인증 기관(Certificate Authority)의 고유 식별자입니다.
* `not_before` ([Nullable(String)](/ko/reference/data-types/index)) — 이 인증서의 유효 기간 시작 시각입니다.
* `not_after` ([Nullable(String)](/ko/reference/data-types/index)) — 이 인증서의 유효 기간 종료 시각입니다.
* `subject` ([Nullable(String)](/ko/reference/data-types/index)) — 주체입니다. 공개 키의 소유자를 식별합니다.
* `pkey_algo` ([Nullable(String)](/ko/reference/data-types/index)) — 공개 키 알고리즘입니다. 공개 키와 함께 사용할 수 있는 알고리즘을 정의합니다.
* `path` ([String](/ko/reference/data-types/index)) — 이 인증서가 들어 있는 파일 또는 디렉터리의 경로입니다.
* `default` ([UInt8](/ko/reference/data-types/index)) — 인증서가 기본 인증서 위치에 있음을 나타냅니다.
* `protocol` ([String](/ko/reference/data-types/index)) — 인증서가 프로토콜별 TLS 구성에서 가져온 경우 해당 프로토콜 이름이며, 그렇지 않으면 빈 값입니다.
