<div id="description">
  ## 설명
</div>

큐에 있는 처리 대기 중인 비동기 삽입에 대한 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ko/reference/data-types/index)) — 쿼리 텍스트.
* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름.
* `format` ([String](/ko/reference/data-types/index)) — 포맷 이름.
* `first_update` ([DateTime64(6)](/ko/reference/data-types/index)) — 마이크로초 해상도의 첫 삽입 시각.
* `total_bytes` ([UInt64](/ko/reference/data-types/index)) — 큐에서 대기 중인 총 바이트 수.
* `entries.query_id` ([Array(String)](/ko/reference/data-types/index)) — 큐에서 대기 중인 삽입의 쿼리 id 배열.
* `entries.bytes` ([Array(UInt64)](/ko/reference/data-types/index)) — 큐에서 대기 중인 각 삽입 쿼리의 바이트 수 배열.

{/*AUTOGENERATED_END*/}

  ## 예시
</div>

SELECT * FROM system.asynchronous_inserts LIMIT 1 \G;
```

```
