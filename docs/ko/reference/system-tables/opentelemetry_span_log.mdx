<div id="description">
  ## 설명
</div>

실행된 쿼리의 [트레이스 스팬](https://opentracing.io/docs/overview/spans/) 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 이 스팬이 수집된 호스트명입니다.
* `trace_id` ([UUID](/ko/reference/data-types/uuid)) — 실행된 쿼리의 trace ID입니다.
* `span_id` ([UInt64](/ko/reference/data-types/int-uint)) — 트레이스 스팬의 ID입니다.
* `parent_span_id` ([UInt64](/ko/reference/data-types/int-uint)) — 부모 트레이스 스팬의 ID입니다.
* `operation_name` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 작업 이름입니다.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/ko/reference/data-types/enum)) — 스팬의 SpanKind입니다. INTERNAL — 스팬이 애플리케이션 내부 작업을 나타냄을 의미합니다. SERVER — 스팬이 동기식 RPC 또는 기타 원격 요청에 대한 서버 측 처리를 나타냄을 의미합니다. CLIENT — 스팬이 일부 원격 서비스에 대한 요청을 나타냄을 의미합니다. PRODUCER — 스팬이 비동기 요청의 시작자를 나타냄을 의미합니다. 이 부모 스팬은 해당 자식 CONSUMER 스팬보다 먼저 종료되는 경우가 많으며, 자식 스팬이 시작되기 전일 수도 있습니다. CONSUMER - 스팬이 비동기 PRODUCER 요청의 자식을 나타냄을 의미합니다.
* `start_time_us` ([UInt64](/ko/reference/data-types/int-uint)) — 트레이스 스팬의 시작 시간(마이크로초)입니다.
* `finish_time_us` ([UInt64](/ko/reference/data-types/int-uint)) — 트레이스 스팬의 종료 시간(마이크로초)입니다.
* `finish_date` ([Date](/ko/reference/data-types/date)) — 트레이스 스팬의 종료 날짜입니다.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/ko/reference/data-types/enum)) — 스팬의 상태 코드입니다.
* `status_message` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 오류 메시지입니다.
* `attribute` ([Map(LowCardinality(String), String)](/ko/reference/data-types/map)) — 트레이스 스팬에 따라 달라지는 속성입니다. OpenTelemetry 표준의 권장 사항에 따라 채워집니다.

**별칭:**

* `attribute.names` — `mapKeys(attribute)`의 별칭입니다.
* `attribute.values` — `mapValues(attribute)`의 별칭입니다.

{/*AUTOGENERATED_END*/}
