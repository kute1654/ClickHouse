<div id="description">
  ## м„ӨлӘ…
</div>

`config.xml`м—җ м§Җм •лҗң serverмқҳ м „м—ӯ м„Өм • м •ліҙлҘј нҸ¬н•Ён•©лӢҲлӢӨ.
нҳ„мһ¬ мқҙ н…Ңмқҙлё”м—җлҠ” `config.xml`мқҳ мІ« лІҲм§ё кі„мёөм—җ мһҲлҠ” м„Өм •л§Ң н‘ңмӢңлҗҳл©°, мӨ‘мІ©лҗң config(мҳҲ: [logger](/ko/reference/settings/server-settings/settings#logger))лҠ” м§Җмӣҗлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

<div id="columns">
  ## м»¬лҹј
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) вҖ” м„ңлІ„ м„Өм • мқҙлҰ„мһ…лӢҲлӢӨ.
* `value` ([String](/ko/reference/data-types/index)) вҖ” м„ңлІ„ м„Өм • к°’мһ…лӢҲлӢӨ.
* `default` ([String](/ko/reference/data-types/index)) вҖ” м„ңлІ„ м„Өм • кё°ліёк°’мһ…лӢҲлӢӨ.
* `changed` ([UInt8](/ko/reference/data-types/index)) вҖ” м„Өм •мқҙ config.xmlм—җ м§Җм •лҗҳм–ҙ мһҲлҠ”м§Җ н‘ңмӢңн•©лӢҲлӢӨ.
* `description` ([String](/ko/reference/data-types/index)) вҖ” м„ңлІ„ м„Өм •м—җ лҢҖн•ң м§§мқҖ м„ӨлӘ…мһ…лӢҲлӢӨ.
* `type` ([String](/ko/reference/data-types/index)) вҖ” м„ңлІ„ м„Өм • к°’мқҳ нғҖмһ…мһ…лӢҲлӢӨ.
* `changeable_without_restart` ([Enum8(&#39;No&#39; = 0, &#39;IncreaseOnly&#39; = 1, &#39;DecreaseOnly&#39; = 2, &#39;Yes&#39; = 3)](/ko/reference/data-types/index)) вҖ” м„Өм •мқ„ лҹ°нғҖмһ„м—җ ліҖкІҪн•  мҲҳ мһҲлҠ”м§Җ н‘ңмӢңн•©лӢҲлӢӨ.
* `is_obsolete` ([UInt8](/ko/reference/data-types/index)) вҖ” м„Өм •мқҙ лҚ” мқҙмғҒ мӮ¬мҡ©лҗҳм§Җ м•ҠлҠ” н•ӯлӘ©мқём§Җ н‘ңмӢңн•©лӢҲлӢӨ.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## мҳҲмӢң
</div>

лӢӨмқҢ мҳҲмӢңлҠ” мқҙлҰ„м—җ `thread_pool`мқҙ нҸ¬н•Ёлҗң м„ңлІ„ м„Өм • м •ліҙлҘј к°Җм ёмҳӨлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

```sql
SELECT *
FROM system.server_settings
WHERE name LIKE '%thread_pool%'
```
