<div id="description">
  ## 설명
</div>

[*GraphiteMergeTree](/ko/reference/engines/table-engines/mergetree-family/graphitemergetree) 엔진을 사용하는 테이블에서 사용하는 [graphite&#95;rollup](/ko/reference/settings/server-settings/settings#graphite) 매개변수에 대한 정보를 담고 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/ko/reference/data-types/index)) — graphite&#95;rollup 매개변수 이름입니다.
* `rule_type` ([String](/ko/reference/data-types/index)) — 규칙 유형입니다. 가능한 값: RuleTypeAll = 0 - 기본값으로, 정규식을 사용하며 기존 scheme과 호환됩니다. RuleTypePlain = 1 - 일반 메트릭으로, 정규식을 사용하며 기존 scheme과 호환됩니다. RuleTypeTagged = 2 - 태그된 메트릭으로, 정규식을 사용하며 기존 scheme과 호환됩니다. RuleTypeTagList = 3 - 태그된 메트릭으로, 정규식을 사용합니다(예: &#39;retention=10min ; env=(staging|prod)&#39;와 같은 문자열에서 RuleTypeTagged로 변환됨).
* `regexp` ([String](/ko/reference/data-types/index)) — 메트릭 이름에 대한 패턴입니다.
* `function` ([String](/ko/reference/data-types/index)) — 집계 함수 이름입니다.
* `age` ([UInt64](/ko/reference/data-types/index)) — 초 단위 데이터의 최소 경과 시간입니다.
* `precision` ([UInt64](/ko/reference/data-types/index)) — 초 단위 데이터 경과 시간을 얼마나 세밀하게 정의할지 나타냅니다.
* `priority` ([UInt16](/ko/reference/data-types/index)) — 패턴 우선순위입니다.
* `is_default` ([UInt8](/ko/reference/data-types/index)) — 해당 패턴이 기본값인지 여부입니다.
* `Tables.database` ([Array(String)](/ko/reference/data-types/index)) — `config_name` 매개변수를 사용하는 데이터베이스 테이블 이름 배열입니다.
* `Tables.table` ([Array(String)](/ko/reference/data-types/index)) — `config_name` 매개변수를 사용하는 테이블 이름 배열입니다.

{/*AUTOGENERATED_END*/}
