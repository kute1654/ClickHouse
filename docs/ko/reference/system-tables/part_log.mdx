<div id="description">
  ## 설명
</div>

`system.part_log` 테이블은 [part&#95;log](/ko/reference/settings/server-settings/settings#part_log) 서버 설정이 지정된 경우에만 생성됩니다.

이 테이블에는 데이터 추가나 머지와 같이 [MergeTree](/ko/reference/engines/table-engines/mergetree-family/mergetree) 엔진 계열 테이블의 [데이터 파트](/ko/reference/engines/table-engines/mergetree-family/custom-partitioning-key)에서 발생한 이벤트 정보가 포함됩니다.

`system.part_log` 테이블에는 다음 컬럼이 포함됩니다:

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 쿼리를 실행하는 server의 호스트명입니다.
* `query_id` ([String](/ko/reference/data-types/string)) — 이 데이터 파트를 생성한 INSERT 쿼리의 식별자입니다.
* `event_type` ([Enum8(&#39;NewPart&#39; = 1, &#39;MergeParts&#39; = 2, &#39;DownloadPart&#39; = 3, &#39;RemovePart&#39; = 4, &#39;MutatePart&#39; = 5, &#39;MovePart&#39; = 6, &#39;MergePartsStart&#39; = 7, &#39;MutatePartStart&#39; = 8)](/ko/reference/data-types/enum)) — 데이터 파트에서 발생한 이벤트의 유형입니다. 다음 값 중 하나를 가질 수 있습니다: NewPart — 새 데이터 파트가 삽입됨, MergePartsStart — 데이터 파트 머지가 시작됨, MergeParts — 데이터 파트 머지가 완료됨, DownloadPart — 데이터 파트를 다운로드함, RemovePart — [DETACH PARTITION](/ko/reference/statements/alter/partition#detach-partitionpart)을 사용해 데이터 파트를 제거하거나 분리함, MutatePartStart — 데이터 파트 변경이 시작됨, MutatePart — 데이터 파트 변경이 완료됨, MovePart — 데이터 파트를 한 디스크에서 다른 디스크로 이동함.
* `merge_reason` ([Enum8(&#39;NotAMerge&#39; = 1, &#39;RegularMerge&#39; = 2, &#39;TTLDeleteMerge&#39; = 3, &#39;TTLRecompressMerge&#39; = 4, &#39;TTLDropMerge&#39; = 5)](/ko/reference/data-types/enum)) — 유형이 MERGE&#95;PARTS인 이벤트의 발생 이유입니다. 다음 값 중 하나를 가질 수 있습니다: NotAMerge — 현재 이벤트의 유형이 MERGE&#95;PARTS가 아닙니다, RegularMerge — 일반적인 머지입니다, TTLDeleteMerge, TTLDropMerge — 만료된 데이터를 정리합니다. TTLRecompressMerge — 데이터 파트를 재압축하는 경우입니다.
* `merge_algorithm` ([Enum8(&#39;Undecided&#39; = 0, &#39;Vertical&#39; = 1, &#39;Horizontal&#39; = 2)](/ko/reference/data-types/enum)) — MERGE&#95;PARTS 유형 이벤트의 머지 알고리즘입니다. 다음 값 중 하나를 가질 수 있습니다: Undecided, Horizontal, Vertical
* `event_date` ([Date](/ko/reference/data-types/date)) — 이벤트 날짜.
* `event_time` ([DateTime](/ko/reference/data-types/datetime)) — 이벤트 발생 시간입니다.
* `event_time_microseconds` ([DateTime64(6)](/ko/reference/data-types/datetime64)) — 마이크로초 단위 정밀도의 이벤트 시간입니다.
* `duration_ms` ([UInt64](/ko/reference/data-types/int-uint)) — 이 작업의 소요 시간입니다.
* `database` ([String](/ko/reference/data-types/string)) — 데이터 파트가 속한 데이터베이스의 이름입니다.
* `table` ([String](/ko/reference/data-types/string)) — data part가 속한 테이블 이름입니다.
* `table_uuid` ([UUID](/ko/reference/data-types/uuid)) — 데이터 파트(data part)가 속한 테이블의 UUID입니다.
* `part_name` ([String](/ko/reference/data-types/string)) — 데이터 파트 이름입니다.
* `partition_id` ([String](/ko/reference/data-types/string)) — 데이터 파트가 삽입된 파티션의 ID입니다. 파티셔닝 기준이 `tuple()`이면 이 컬럼의 값은 `all`이 됩니다.
* `partition` ([String](/ko/reference/data-types/string)) — 파티션 이름입니다.
* `part_type` ([String](/ko/reference/data-types/string)) — 파트 유형입니다. 가능한 값: Wide 및 Compact.
* `part_storage_type` ([String](/ko/reference/data-types/string)) — DataPartStorage의 유형입니다. 가능한 값: Packed - 모든 파일이 하나의 blob에 저장됩니다. Full - 각 파일이 개별 blob에 저장됩니다.
* `disk_name` ([String](/ko/reference/data-types/string)) — data part가 저장된 디스크 이름입니다.
* `path_on_disk` ([String](/ko/reference/data-types/string)) — data part 파일이 저장된 폴더의 절대 경로입니다.
* `rows` ([UInt64](/ko/reference/data-types/int-uint)) — 데이터 파트에 있는 행 수입니다.
* `size_in_bytes` ([UInt64](/ko/reference/data-types/int-uint)) — 디스크에 저장된 데이터 파트의 크기(바이트 단위)입니다.
* `merged_from` ([배열(String)](/ko/reference/data-types/array)) — 현재 파트를 구성한 원본 파트 이름들의 배열입니다.
* `bytes_uncompressed` ([UInt64](/ko/reference/data-types/int-uint)) — 결과 part의 바이트 단위 압축되지 않은 크기입니다.
* `read_rows` ([UInt64](/ko/reference/data-types/int-uint)) — 머지 중에 읽은 행 수입니다.
* `read_bytes` ([UInt64](/ko/reference/data-types/int-uint)) — 머지 중에 읽은 바이트 수입니다.
* `peak_memory_usage` ([UInt64](/ko/reference/data-types/int-uint)) — 머지 중 사용된 RAM의 최대 사용량
* `deduplication_block_ids` ([Array(String)](/ko/reference/data-types/array)) — 이 part를 삽입할 때 중복 제거에 사용되는 block ID의 배열입니다.
* `error` ([UInt16](/ko/reference/data-types/int-uint)) — 발생한 예외에 대한 오류 코드입니다.
* `exception` ([String](/ko/reference/data-types/string)) — 발생한 오류에 대한 텍스트 메시지입니다.
* `mutation_ids` ([Array(String)](/ko/reference/data-types/array)) — 유형이 MUTATE&#95;PART&#95;START 및 MUTATE&#95;PART인 이벤트에서 원본 파트(merged&#95;from)에 적용된 mutation ID의 배열입니다.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/ko/reference/data-types/map)) — 이 작업 중에 수집된 모든 프로파일 이벤트입니다.
* `projections_duration_ms` ([Map(LowCardinality(String), UInt64)](/ko/reference/data-types/map)) — 프로젝션별 머지/재구축에 걸린 시간(밀리초)입니다.

**별칭:**

* `ProfileEvents.Names` — `mapKeys(ProfileEvents)`를 가리키는 별칭입니다.
* `ProfileEvents.Values` — `mapValues(ProfileEvents)`를 가리키는 별칭입니다.
* `name` — `part_name`을 가리키는 별칭입니다.

{/*AUTOGENERATED_END*/}
