<div id="description">
  ## 설명
</div>

시스템에서 발생한 이벤트 개수에 관한 정보를 담고 있습니다. 예를 들어, 이 테이블에서는 ClickHouse 서버가 시작된 이후 처리된 `SELECT` 쿼리가 몇 건인지 확인할 수 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `event` ([String](/ko/reference/data-types/index)) — 이벤트 이름.
* `value` ([UInt64](/ko/reference/data-types/index)) — 발생한 이벤트의 수.
* `description` ([String](/ko/reference/data-types/index)) — 이벤트 설명.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql
SELECT * FROM system.events LIMIT 5
```

```text
┌─event─────────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│ Query                                 │    12 │ 해석 및 실행 대상 쿼리의 수입니다. 파싱에 실패하거나 AST 크기 제한, QUOTA 제한, 동시 실행 쿼리 수 제한으로 인해 거부된 쿼리는 포함되지 않습니다. ClickHouse 자체에서 시작된 내부 쿼리가 포함될 수 있습니다. 서브쿼리는 카운트되지 않습니다.                  │
│ SelectQuery                           │     8 │ Query와 동일하나 SELECT 쿼리에만 해당됩니다.                                                                                                                                                                                                                │
│ FileOpen                              │    73 │ 열린 파일의 수입니다.                                                                                                                                                                                                                                    │
│ ReadBufferFromFileDescriptorRead      │   155 │ 파일 디스크립터에서의 읽기(read/pread) 횟수입니다. 소켓은 포함되지 않습니다.                                                                                                                                                                             │
│ ReadBufferFromFileDescriptorReadBytes │  9931 │ 파일 디스크립터에서 읽은 바이트 수입니다. 파일이 압축된 경우 압축 데이터 크기가 표시됩니다.                                                                                                                                                              │
└───────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```
