<div id="description">
  ## 설명
</div>

압축 및 암호화 코덱에 관한 정보를 제공합니다.

이 테이블을 사용하면 사용 가능한 압축 및 암호화 코덱 정보를 확인할 수 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 코덱 이름입니다.
* `method_byte` ([UInt8](/ko/reference/data-types/index)) — 압축된 파일에서 코덱을 나타내는 바이트입니다.
* `is_compression` ([UInt8](/ko/reference/data-types/index)) — 이 코덱이 무언가를 압축하면 참입니다. 그렇지 않으면 압축에 도움이 되는 변환일 수 있습니다.
* `is_generic_compression` ([UInt8](/ko/reference/data-types/index)) — 이 코덱이 lz4, zstd와 같은 범용 압축 알고리즘이면 참입니다.
* `is_encryption` ([UInt8](/ko/reference/data-types/index)) — 이 코덱이 암호화를 수행하면 참입니다.
* `is_timeseries_codec` ([UInt8](/ko/reference/data-types/index)) — 이 코덱이 부동소수점 시계열용 코덱이면 참입니다.
* `is_experimental` ([UInt8](/ko/reference/data-types/index)) — 이 코덱이 Experimental 상태이면 참입니다.
* `description` ([String](/ko/reference/data-types/index)) — 코덱에 대한 개괄적인 설명입니다.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```

```text title="Response"

name:                   LZ4

```
