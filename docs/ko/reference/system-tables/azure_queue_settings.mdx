<div id="description">
  ## 설명
</div>

[AzureQueue](/ko/reference/engines/table-engines/integrations/azure-queue) 테이블 설정에 대한 정보를 제공합니다.
서버 버전 `24.10`부터 사용할 수 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — AzureQueue engine을 사용하는 테이블이 속한 데이터베이스입니다.
* `table` ([String](/ko/reference/data-types/index)) — AzureQueue engine을 사용하는 테이블 이름입니다.
* `name` ([String](/ko/reference/data-types/index)) — 설정 이름입니다.
* `value` ([String](/ko/reference/data-types/index)) — 설정 값입니다.
* `type` ([String](/ko/reference/data-types/index)) — 설정 유형입니다(구현에 따라 달라지는 문자열 값).
* `changed` ([UInt8](/ko/reference/data-types/index)) — 설정이 config에 명시적으로 정의되었거나 명시적으로 변경된 경우 1입니다.
* `description` ([String](/ko/reference/data-types/index)) — 설정 설명입니다.
* `alterable` ([UInt8](/ko/reference/data-types/index)) — 현재 사용자가 ALTER TABLE MODIFY SETTING을 통해 이 설정을 변경할 수 있는지를 나타냅니다: 0 — 현재 사용자는 이 설정을 변경할 수 없음, 1 — 현재 사용자는 이 설정을 변경할 수 있음.

{/*AUTOGENERATED_END*/}
