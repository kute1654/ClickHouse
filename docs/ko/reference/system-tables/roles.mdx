<div id="description">
  ## 설명
</div>

설정된 [역할](/ko/concepts/features/security/access-rights#role-management) 정보가 포함됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 역할 이름입니다.
* `id` ([UUID](/ko/reference/data-types/index)) — 역할 ID입니다.
* `storage` ([String](/ko/reference/data-types/index)) — 역할 스토리지의 경로입니다. `access_control_path` 매개변수에서 설정합니다.

{/*AUTOGENERATED_END*/}
