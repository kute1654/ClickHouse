<div id="description">
  ## 설명
</div>

클러스터에서 실행된 [분산 DDL 쿼리(ON CLUSTER 절)](/ko/reference/statements/distributed-ddl)에 대한 정보를 담고 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/ko/reference/data-types/index)) — 쿼리 ID입니다.
* `entry_version` ([Nullable(UInt8)](/ko/reference/data-types/index)) — 항목의 버전입니다.
* `initiator_host` ([Nullable(String)](/ko/reference/data-types/index)) — DDL 작업을 시작한 호스트입니다.
* `initiator_port` ([Nullable(UInt16)](/ko/reference/data-types/index)) — initiator가 사용한 포트입니다.
* `cluster` ([String](/ko/reference/data-types/index)) — 클러스터 이름이며, 확인할 수 없으면 비어 있습니다.
* `query` ([String](/ko/reference/data-types/index)) — 실행된 쿼리입니다.
* `settings` ([Map(String, String)](/ko/reference/data-types/index)) — DDL 작업에 사용된 설정입니다.
* `query_create_time` ([DateTime](/ko/reference/data-types/index)) — 쿼리가 생성된 시간입니다.
* `host` ([Nullable(String)](/ko/reference/data-types/index)) — 호스트명입니다.
* `port` ([Nullable(UInt16)](/ko/reference/data-types/index)) — 호스트 포트입니다.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/ko/reference/data-types/index)) — 쿼리 상태입니다.
* `exception_code` ([Nullable(UInt16)](/ko/reference/data-types/index)) — 예외 코드입니다.
* `exception_text` ([Nullable(String)](/ko/reference/data-types/index)) — 예외 메시지입니다.
* `query_finish_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 쿼리 종료 시간입니다.
* `query_duration_ms` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 쿼리 실행 시간(밀리초)입니다.

{/*AUTOGENERATED_END*/}
