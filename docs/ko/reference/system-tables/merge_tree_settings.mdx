<div id="description">
  ## 설명
</div>

`MergeTree` 테이블의 설정 정보를 제공합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 설정 이름입니다.
* `value` ([String](/ko/reference/data-types/index)) — 설정 값입니다.
* `default` ([String](/ko/reference/data-types/index)) — 설정 기본값입니다.
* `changed` ([UInt8](/ko/reference/data-types/index)) — 설정이 config에 명시적으로 정의되었거나 명시적으로 변경된 경우 1입니다.
* `description` ([String](/ko/reference/data-types/index)) — 설정 설명입니다.
* `min` ([Nullable(String)](/ko/reference/data-types/index)) — constraints를 통해 설정된 경우의 최솟값입니다. 최솟값이 없으면 NULL을 포함합니다.
* `max` ([Nullable(String)](/ko/reference/data-types/index)) — constraints를 통해 설정된 경우의 최댓값입니다. 최댓값이 없으면 NULL을 포함합니다.
* `disallowed_values` ([Array(String)](/ko/reference/data-types/index)) — 허용되지 않는 값 목록입니다.
* `readonly` ([UInt8](/ko/reference/data-types/index)) — 현재 사용자가 이 설정을 변경할 수 있는지 나타냅니다. 0 — 현재 사용자가 설정을 변경할 수 있음, 1 — 현재 사용자가 설정을 변경할 수 없음.
* `type` ([String](/ko/reference/data-types/index)) — 설정 유형(구현별 문자열 값)입니다.
* `is_obsolete` ([UInt8](/ko/reference/data-types/index)) — 설정이 더 이상 사용되지 않는 상태인지 나타냅니다.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/ko/reference/data-types/index)) —
  이 기능의 지원 수준입니다. ClickHouse 기능은 현재
  개발 상태와 사용 시 기대할 수 있는 수준에 따라 여러 등급으로 구분됩니다:

- PRODUCTION: 이 기능은 안정적이며 안전하게 사용할 수 있고, 다른 PRODUCTION 기능과 함께 사용해도 상호작용 문제가 없습니다.
- BETA: 이 기능은 안정적이고 안전합니다. 다른 기능과 함께 사용할 때의 결과는 알려져 있지 않으며 정확성은 보장되지 않습니다. 테스트와 보고를 환영합니다.
- EXPERIMENTAL: 이 기능은 개발 중입니다. 개발자와 ClickHouse 사용에 익숙한 사용자만을 대상으로 합니다. 기능이 작동할 수도, 작동하지 않을 수도 있으며 언제든지 제거될 수 있습니다.
- OBSOLETE: 더 이상 지원되지 않습니다. 이미 제거되었거나 향후 릴리스에서 제거될 예정입니다.

{/*AUTOGENERATED_END*/}
