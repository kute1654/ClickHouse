<div id="description">
  ## 설명
</div>

이 테이블에는 [MergeTree](/ko/reference/engines/table-engines/mergetree-family/mergetree) 테이블에서 현재 진행 중인 [데이터 파트 이동 작업](/ko/reference/statements/alter/partition#move-partitionpart) 정보가 포함되어 있습니다. 각 데이터 파트 이동 작업은 하나의 행으로 나타납니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 이동 중인 데이터 파트가 포함된 테이블 이름입니다.
* `elapsed` ([Float64](/ko/reference/data-types/index)) — 데이터 파트 이동 시작 후 경과한 시간(초)입니다.
* `target_disk_name` ([String](/ko/reference/data-types/index)) — 데이터 파트가 이동 중인 대상 디스크 이름입니다.
* `target_disk_path` ([String](/ko/reference/data-types/index)) — 파일 시스템에서 해당 디스크의 마운트 지점 경로입니다.
* `part_name` ([String](/ko/reference/data-types/index)) — 이동 중인 데이터 파트 이름입니다.
* `part_size` ([UInt64](/ko/reference/data-types/index)) — 데이터 파트 크기입니다.
* `thread_id` ([UInt64](/ko/reference/data-types/index)) — 이동 작업을 수행하는 스레드의 식별자입니다.

{/*AUTOGENERATED_END*/}
