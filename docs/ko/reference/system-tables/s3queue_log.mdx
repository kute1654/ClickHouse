S3Queue 엔진이 처리한 파일 정보를 담은 로그 항목을 포함합니다.

이 테이블은 언제든지 TRUNCATE하거나 삭제해도 안전합니다.

컬럼:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 호스트명
* `event_date` ([Date](/ko/reference/data-types/date)) — 이 로그 행이 기록된 이벤트 날짜
* `event_time` ([DateTime](/ko/reference/data-types/datetime)) — 이 로그 행이 기록된 이벤트 시간
* `database` ([String](/ko/reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)이 속한 데이터베이스의 이름입니다.
* `table` ([String](/ko/reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)의 이름입니다.
* `uuid` ([String](/ko/reference/data-types/string)) — 큐 테이블(`S3Queue` 또는 `AzureQueue`)의 UUID입니다.
* `file_name` ([String](/ko/reference/data-types/string)) — 처리 중인 파일의 이름입니다.
* `rows_processed` ([UInt64](/ko/reference/data-types/int-uint)) — 처리된 행 수입니다.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/ko/reference/data-types/enum)) — 처리 중인 파일의 상태입니다.
* `processing_start_time` ([Nullable(DateTime)](/ko/reference/data-types/nullable)) — 파일 처리가 시작된 시간입니다.
* `processing_end_time` ([Nullable(DateTime)](/ko/reference/data-types/nullable)) — 파일 처리가 종료된 시간입니다.
* `exception` ([String](/ko/reference/data-types/string)) — 예외가 발생한 경우의 메시지입니다.
* `commit_id` ([UInt64](/ko/reference/data-types/int-uint)) — 이 파일이 커밋된 transaction의 ID입니다.
* `commit_time` ([DateTime](/ko/reference/data-types/datetime)) — 파일이 Keeper에 커밋된 시간입니다(실패 또는 처리 완료 상태로).
* `transaction_start_time` ([DateTime](/ko/reference/data-types/datetime)) — 전체 처리 transaction이 시작된 시간입니다.
* `get_object_time_ms` ([UInt64](/ko/reference/data-types/int-uint)) — 객체 스토리지에서 객체를 찾는 데 걸린 시간입니다.

{/*AUTOGENERATED_END*/}
