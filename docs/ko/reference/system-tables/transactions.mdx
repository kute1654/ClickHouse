<div id="description">
  ## 설명
</div>

트랜잭션 목록과 해당 상태를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/ko/reference/data-types/tuple)) — 트랜잭션의 식별자입니다.
* `tid_hash` ([UInt64](/ko/reference/data-types/int-uint)) — 식별자의 해시입니다.
* `elapsed` ([Float64](/ko/reference/data-types/float)) — 트랜잭션 처리에 경과된 시간입니다.
* `is_readonly` ([UInt8](/ko/reference/data-types/int-uint)) — 트랜잭션이 쓰기 작업을 수행했는지 여부를 나타내는 플래그입니다.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/ko/reference/data-types/enum)) — 트랜잭션의 상태입니다. 가능한 값은 RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK입니다.

{/*AUTOGENERATED_END*/}
