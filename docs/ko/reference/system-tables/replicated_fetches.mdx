<div id="description">
  ## 설명
</div>

현재 실행 중인 백그라운드 페치 정보를 제공합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `elapsed` ([Float64](/ko/reference/data-types/index)) — 현재 실행 중인 백그라운드 페치의 표시가 시작된 후 경과한 시간(초)입니다.
* `progress` ([Float64](/ko/reference/data-types/index)) — 완료된 작업의 비율(0~1)입니다.
* `result_part_name` ([String](/ko/reference/data-types/index)) — 현재 실행 중인 백그라운드 페치를 표시한 결과로 생성될 파트의 이름입니다.
* `result_part_path` ([String](/ko/reference/data-types/index)) — 현재 실행 중인 백그라운드 페치를 표시한 결과로 생성될 파트의 절대 경로입니다.
* `partition_id` ([String](/ko/reference/data-types/index)) — 파티션 ID입니다.
* `total_size_bytes_compressed` ([UInt64](/ko/reference/data-types/index)) — 결과 파트에 있는 압축된 데이터의 전체 크기(바이트)입니다.
* `bytes_read_compressed` ([UInt64](/ko/reference/data-types/index)) — 결과 파트에서 읽은 압축된 바이트 수입니다.
* `source_replica_path` ([String](/ko/reference/data-types/index)) — 소스 레플리카의 절대 경로입니다.
* `source_replica_hostname` ([String](/ko/reference/data-types/index)) — 소스 레플리카의 호스트명입니다.
* `source_replica_port` ([UInt16](/ko/reference/data-types/index)) — 소스 레플리카의 포트 번호입니다.
* `interserver_scheme` ([String](/ko/reference/data-types/index)) — interserver 스킴의 이름입니다.
* `URI` ([String](/ko/reference/data-types/index)) — URI(Uniform Resource Identifier)입니다.
* `to_detached` ([UInt8](/ko/reference/data-types/index)) — 현재 실행 중인 백그라운드 페치가 TO DETACHED 표현식을 사용해 수행되는지 여부를 나타내는 플래그입니다.
* `thread_id` ([UInt64](/ko/reference/data-types/index)) — 스레드 식별자입니다.

{/*AUTOGENERATED_END*/}
