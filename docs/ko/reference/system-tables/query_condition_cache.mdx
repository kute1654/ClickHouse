<div id="description">
  ## 설명
</div>

[쿼리 조건 캐시](/ko/concepts/features/performance/caches/query-condition-cache)의 콘텐츠를 표시합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `key_hash` ([UInt128](/ko/reference/data-types/index)) — (table&#95;uuid, part&#95;name, condition&#95;hash)의 해시값입니다.
* `entry_size` ([UInt64](/ko/reference/data-types/index)) — 항목 크기(바이트)입니다.
* `matching_marks` ([String](/ko/reference/data-types/index)) — 일치하는 마크입니다.

{/*AUTOGENERATED_END*/}
