<div id="description">
  ## 설명
</div>

[사용자 정의 함수(UDFs)](/ko/reference/functions/regular-functions/udf)의 로딩 상태, 오류 정보, 구성 메타데이터를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — UDF 이름입니다.
* `load_status` ([Enum8(&#39;Success&#39; = 0, &#39;Failed&#39; = 1)](/ko/reference/data-types/index)) — 로딩 상태입니다. 가능한 값:
  * **Success** — UDF가 로드되어 사용할 준비가 되었습니다
  * **Failed** — UDF 로드에 실패했습니다(자세한 내용은 필드 &#39;loading&#95;error&#95;message&#39;를 참조하십시오).
* `loading_error_message` ([String](/ko/reference/data-types/index)) — 로드에 실패했을 때의 자세한 오류 메시지입니다. 성공적으로 로드된 경우 비어 있습니다.
* `last_successful_update_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 마지막으로 성공한 업데이트의 타임스탬프입니다. 한 번도 성공하지 않았으면 NULL입니다.
* `loading_duration_ms` ([UInt64](/ko/reference/data-types/index)) — UDF를 로드하는 데 걸린 시간(밀리초)입니다.
* `type` ([Enum8(&#39;executable&#39; = 0, &#39;executable&#95;pool&#39; = 1)](/ko/reference/data-types/index)) — UDF 유형입니다: &#39;executable&#39;(단일 프로세스) 또는 &#39;executable&#95;pool&#39;(프로세스 풀).
* `command` ([String](/ko/reference/data-types/index)) — 이 UDF에 대해 실행할 스크립트 또는 명령입니다.
* `format` ([String](/ko/reference/data-types/index)) — I/O용 데이터 포맷입니다(예: &#39;TabSeparated&#39;, &#39;JSONEachRow&#39;).
* `return_type` ([String](/ko/reference/data-types/index)) — 함수 반환 유형입니다(예: &#39;String&#39;, &#39;UInt64&#39;).
* `return_name` ([String](/ko/reference/data-types/index)) — 선택적 반환값 식별자입니다. 구성되지 않은 경우 비어 있습니다.
* `argument_types` ([Array(String)](/ko/reference/data-types/index)) — 인수 타입 배열입니다(예: [&#39;String&#39;, &#39;UInt64&#39;]).
* `argument_names` ([Array(String)](/ko/reference/data-types/index)) — 인수 이름 배열입니다. 이름이 없는 인수에는 빈 문자열이 사용됩니다.
* `max_command_execution_time` ([UInt64](/ko/reference/data-types/index)) — 데이터 블록을 처리할 수 있는 최대 시간(초)입니다. &#39;executable&#95;pool&#39; 유형에서만 사용됩니다.
* `command_termination_timeout` ([UInt64](/ko/reference/data-types/index)) — 명령 프로세스에 SIGTERM을 보내기 전까지의 시간(초)입니다.
* `command_read_timeout` ([UInt64](/ko/reference/data-types/index)) — 명령의 stdout에서 읽을 때 적용되는 시간 제한(밀리초)입니다.
* `command_write_timeout` ([UInt64](/ko/reference/data-types/index)) — 명령의 stdin에 쓸 때 적용되는 시간 제한(밀리초)입니다.
* `pool_size` ([UInt64](/ko/reference/data-types/index)) — 명령 프로세스 인스턴스 수입니다. &#39;executable&#95;pool&#39; 유형에서만 사용됩니다.
* `send_chunk_header` ([UInt8](/ko/reference/data-types/index)) — 각 데이터 청크 앞에 행 수를 보낼지 여부입니다(불리언).
* `execute_direct` ([UInt8](/ko/reference/data-types/index)) — 명령을 직접(1) 실행할지, /bin/bash를 통해(0) 실행할지 여부입니다.
* `lifetime` ([UInt64](/ko/reference/data-types/index)) — 다시 로드 간격(초)입니다. 0은 다시 로드가 비활성화되었음을 의미합니다.
* `deterministic` ([UInt8](/ko/reference/data-types/index)) — 함수가 동일한 인수에 대해 항상 동일한 결과를 반환하는지 여부입니다(불리언).

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

모든 UDF와 해당 로딩 상태를 확인하세요:

```sql
SELECT
    name,
    load_status,
    type,
    command,
    return_type,
    argument_types
FROM system.user_defined_functions
FORMAT Vertical;
```

```response
Row 1:
──────
name:           my_sum_udf
load_status:    Success
type:           executable
command:        /var/lib/clickhouse/user_scripts/sum.py
return_type:    UInt64
argument_types: ['UInt64','UInt64']
```

실패한 UDF 찾기:

```sql
SELECT
    name,
    loading_error_message
FROM system.user_defined_functions
WHERE load_status = 'Failed';
```
