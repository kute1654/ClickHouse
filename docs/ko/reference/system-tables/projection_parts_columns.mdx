<div id="description">
  ## 설명
</div>

이 테이블에는 MergeTree 엔진 계열 테이블의 프로젝션 파트에 있는 컬럼 정보가 포함되어 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `partition` ([String](/ko/reference/data-types/index)) — 파티션 이름입니다.
* `name` ([String](/ko/reference/data-types/index)) — 데이터 파트의 이름입니다.
* `part_type` ([String](/ko/reference/data-types/index)) — 데이터 파트의 저장 포맷입니다.
* `parent_name` ([String](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트의 이름입니다.
* `parent_uuid` ([UUID](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트의 UUID입니다.
* `parent_part_type` ([String](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트의 저장 포맷입니다.
* `active` ([UInt8](/ko/reference/data-types/index)) — 데이터 파트가 활성 상태인지 나타내는 플래그입니다.
* `marks` ([UInt64](/ko/reference/data-types/index)) — 마크 수입니다.
* `rows` ([UInt64](/ko/reference/data-types/index)) — 행 수입니다.
* `bytes_on_disk` ([UInt64](/ko/reference/data-types/index)) — 모든 데이터 파트 파일의 총 크기(바이트)입니다.
* `data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 데이터 파트에 있는 압축된 데이터의 총 크기입니다. 모든 보조 파일(예: 마크 파일)은 포함되지 않습니다.
* `data_uncompressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 데이터 파트에 있는 비압축 데이터의 총 크기입니다. 모든 보조 파일(예: 마크 파일)은 포함되지 않습니다.
* `marks_bytes` ([UInt64](/ko/reference/data-types/index)) — 마크 파일의 크기입니다.
* `parent_marks` ([UInt64](/ko/reference/data-types/index)) — 소스(상위) 파트의 마크 수입니다.
* `parent_rows` ([UInt64](/ko/reference/data-types/index)) — 소스(상위) 파트의 행 수입니다.
* `parent_bytes_on_disk` ([UInt64](/ko/reference/data-types/index)) — 모든 소스(상위) 데이터 파트 파일의 총 크기(바이트)입니다.
* `parent_data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트에 있는 압축된 데이터의 총 크기입니다.
* `parent_data_uncompressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트에 있는 비압축 데이터의 총 크기입니다.
* `parent_marks_bytes` ([UInt64](/ko/reference/data-types/index)) — 소스(상위) 데이터 파트에 있는 마크 파일의 크기입니다.
* `modification_time` ([DateTime](/ko/reference/data-types/index)) — 데이터 파트가 있는 디렉터리가 수정된 시간입니다. 일반적으로 이는 데이터 파트가 생성된 시각에 해당합니다.
* `remove_time` ([DateTime](/ko/reference/data-types/index)) — 데이터 파트가 비활성 상태가 된 시간입니다.
* `refcount` ([UInt32](/ko/reference/data-types/index)) — 데이터 파트가 사용되는 위치 수입니다. 값이 2보다 크면 해당 데이터 파트가 쿼리 또는 머지 작업에서 사용되고 있음을 나타냅니다.
* `min_date` ([Date](/ko/reference/data-types/index)) — 파티션 키에 포함된 경우 Date 컬럼의 최솟값입니다.
* `max_date` ([Date](/ko/reference/data-types/index)) — 파티션 키에 포함된 경우 Date 컬럼의 최댓값입니다.
* `min_time` ([DateTime](/ko/reference/data-types/index)) — 파티션 키에 포함된 경우 DateTime 컬럼의 최솟값입니다.
* `max_time` ([DateTime](/ko/reference/data-types/index)) — 파티션 키에 포함된 경우 DateTime 컬럼의 최댓값입니다.
* `partition_id` ([String](/ko/reference/data-types/index)) — 파티션 ID입니다.
* `min_block_number` ([Int64](/ko/reference/data-types/index)) — 머지 후 현재 파트를 구성하는 데이터 파트의 최소 번호입니다.
* `max_block_number` ([Int64](/ko/reference/data-types/index)) — 머지 후 현재 파트를 구성하는 데이터 파트의 최대 번호입니다.
* `level` ([UInt32](/ko/reference/data-types/index)) — 머지 트리의 깊이입니다. 0은 현재 파트가 다른 파트를 머지해서 생성된 것이 아니라 삽입으로 생성되었음을 의미합니다.
* `data_version` ([UInt64](/ko/reference/data-types/index)) — 데이터 파트에 적용해야 할 뮤테이션을 판단하는 데 사용되는 번호입니다 (`data_version`보다 버전이 높은 뮤테이션).
* `primary_key_bytes_in_memory` ([UInt64](/ko/reference/data-types/index)) — 기본 키 값에 사용되는 메모리 양(바이트)입니다.
* `primary_key_bytes_in_memory_allocated` ([UInt64](/ko/reference/data-types/index)) — 기본 키 값용으로 예약된 메모리 양(바이트)입니다.
* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `engine` ([String](/ko/reference/data-types/index)) — 매개변수를 제외한 테이블 엔진 이름입니다.
* `disk_name` ([String](/ko/reference/data-types/index)) — 데이터 파트를 저장하는 디스크 이름입니다.
* `path` ([String](/ko/reference/data-types/index)) — 데이터 파트 파일이 있는 폴더의 절대 경로입니다.
* `column` ([String](/ko/reference/data-types/index)) — 컬럼 이름입니다.
* `type` ([String](/ko/reference/data-types/index)) — 컬럼 타입입니다.
* `column_position` ([UInt64](/ko/reference/data-types/index)) — 테이블에서 1부터 시작하는 컬럼의 순서 위치입니다.
* `default_kind` ([String](/ko/reference/data-types/index)) — 기본값의 표현식 유형(DEFAULT, MATERIALIZED, ALIAS)이며, 정의되지 않은 경우 빈 문자열입니다.
* `default_expression` ([String](/ko/reference/data-types/index)) — 기본값의 표현식이며, 정의되지 않은 경우 빈 문자열입니다.
* `column_bytes_on_disk` ([UInt64](/ko/reference/data-types/index)) — 컬럼의 총 크기(바이트)입니다.
* `column_data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 컬럼에 있는 압축된 데이터의 총 크기(바이트)입니다.
* `column_data_uncompressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 컬럼에 있는 압축 해제된 데이터의 총 크기(바이트)입니다.
* `column_marks_bytes` ([UInt64](/ko/reference/data-types/index)) — 컬럼의 마크 크기(바이트)입니다.
* `column_modification_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 컬럼이 마지막으로 수정된 시각입니다.
