<div id="description">
  ## 설명
</div>

모든 사용자의 QUOTA 사용량입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ko/reference/data-types/index)) — QUOTA 이름.
* `quota_key` ([String](/ko/reference/data-types/index)) — 키 값.
* `is_current` ([UInt8](/ko/reference/data-types/index)) — 현재 사용자의 QUOTA 사용량.
* `start_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 리소스 사용량 계산 시작 시각.
* `end_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 리소스 사용량 계산 종료 시각.
* `duration` ([Nullable(UInt32)](/ko/reference/data-types/index)) — 리소스 사용량 계산을 위한 시간 인터벌의 길이(초).
* `queries` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 실행된 쿼리 수.
* `max_queries` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용된 모든 유형의 쿼리의 최대 개수.
* `query_selects` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 실행된 SELECT 쿼리 수.
* `max_query_selects` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용된 SELECT 쿼리의 최대 개수.
* `query_inserts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 실행된 INSERT 쿼리 수.
* `max_query_inserts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용된 INSERT 쿼리의 최대 개수.
* `errors` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 오류가 발생한 쿼리 수.
* `max_errors` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 허용되는 오류 발생 쿼리의 최대 개수.
* `result_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 모든 쿼리의 결과 집합에 포함된 행의 총수.
* `max_result_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리의 결과 집합에 허용되는 행의 최대 총수.
* `result_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 모든 쿼리의 결과 집합에 포함된 바이트의 총수.
* `max_result_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리의 결과 집합에 허용되는 바이트의 최대 총수.
* `read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 모든 쿼리 실행 중 읽은 행의 총수.
* `max_read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리 실행 중 읽을 수 있는 행의 최대 개수.
* `read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 모든 쿼리 실행 중 읽은 바이트의 총수.
* `max_read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리 실행 중 읽을 수 있는 바이트의 최대 개수.
* `execution_time` ([Nullable(Float64)](/ko/reference/data-types/index)) — 현재 주기 내 쿼리 실행에 소요된 총 시간(나노초)
* `max_execution_time` ([Nullable(Float64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리 실행에 허용되는 최대 총 시간(나노초)
* `written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 모든 쿼리 실행 중 기록된 바이트의 총수.
* `max_written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 모든 쿼리 실행 중 기록할 수 있는 바이트의 최대 개수.
* `failed_sequential_authentications` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 연속된 인증 실패 횟수.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 허용되는 연속 인증 실패의 최대 횟수.
* `queries_per_normalized_hash` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 현재 주기 내 단일 정규화된 쿼리의 최대 실행 횟수.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내 단일 정규화된 쿼리에 허용되는 최대 실행 횟수.
