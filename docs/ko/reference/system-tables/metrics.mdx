<div id="description">
  ## 설명
</div>

즉시 계산할 수 있거나 현재값을 갖는 메트릭이 포함됩니다. 예를 들어, 동시에 처리되는 쿼리 수나 현재 레플리카 지연 등이 있습니다. 이 테이블은 항상 최신 상태입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ko/reference/data-types/index)) — 메트릭 이름.
* `value` ([Int64](/ko/reference/data-types/index)) — 메트릭 값.
* `description` ([String](/ko/reference/data-types/index)) — 메트릭 설명.

{/*AUTOGENERATED_END*/}

지원되는 모든 메트릭은 소스 파일 [src/Common/CurrentMetrics.cpp](https://github.com/ClickHouse/ClickHouse/blob/master/src/Common/CurrentMetrics.cpp)에서 확인할 수 있습니다.

<div id="example">
  ## 예시
</div>

```sql
SELECT * FROM system.metrics LIMIT 10
```

```text
┌─metric───────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────┐
│ Query                                │     1 │ 실행 중인 쿼리 수                                                      │
│ Merge                                │     0 │ 실행 중인 백그라운드 머지 수                                           │
│ PartMutation                         │     0 │ 뮤테이션 수 (ALTER DELETE/UPDATE)                                      │
│ ReplicatedFetch                      │     0 │ 레플리카에서 가져오는 중인 데이터 파트 수                              │
│ ReplicatedSend                       │     0 │ 레플리카로 전송 중인 데이터 파트 수                                    │
│ ReplicatedChecks                     │     0 │ 일관성 검사 중인 데이터 파트 수                                        │
│ BackgroundMergesAndMutationsPoolTask │     0 │ 연결된 백그라운드 풀에서 활성 상태인 머지 및 뮤테이션 수              │
│ BackgroundFetchesPoolTask            │     0 │ 연결된 백그라운드 풀에서 활성 상태인 fetch 수                          │
│ BackgroundCommonPoolTask             │     0 │ 연결된 백그라운드 풀에서 활성 상태인 작업 수                           │
│ BackgroundMovePoolTask               │     0 │ 이동 작업을 위한 BackgroundProcessingPool의 활성 작업 수               │
└──────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────┘
```

<div id="primaryindexcachebytes">

</div>
