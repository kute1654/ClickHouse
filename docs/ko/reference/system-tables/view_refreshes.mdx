<div id="description">
  ## 설명
</div>

[갱신 가능 구체화 뷰](/ko/reference/statements/create/view#refreshable-materialized-view)에 대한 정보입니다. 현재 갱신 진행 여부와 관계없이 모든 갱신 가능 구체화 뷰를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 테이블이 속한 데이터베이스의 이름입니다.
* `view` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `uuid` ([UUID](/ko/reference/data-types/index)) — 테이블 UUID입니다(Atomic 데이터베이스).
* `status` ([String](/ko/reference/data-types/index)) — 현재 갱신 상태입니다.
* `last_success_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 가장 최근에 성공한 갱신이 시작된 시각입니다. 서버 시작 이후 또는 테이블 생성 이후 성공한 갱신이 없었다면 NULL입니다.
* `last_success_duration_ms` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 가장 최근 갱신에 걸린 시간입니다.
* `last_refresh_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 가장 최근 갱신 시도가 종료된 시각(알 수 있는 경우) 또는 시작된 시각(알 수 없거나 아직 실행 중인 경우)입니다. 서버 시작 이후 또는 테이블 생성 이후 갱신 시도가 없었다면 NULL입니다.
* `last_refresh_replica` ([String](/ko/reference/data-types/index)) — coordination이 활성화된 경우, 현재(실행 중인 경우) 또는 이전(실행 중이 아닌 경우) 갱신 시도를 수행한 레플리카 이름입니다.
* `next_refresh_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 다음 갱신이 시작되도록 예약된 시각입니다. 현재 다음 갱신 시각을 알 수 없는 경우(예: 의존성을 기다리는 중으로, status가 `WaitingForDependencies` 또는 `MissingDependencies`인 경우) NULL입니다.
* `exception` ([String](/ko/reference/data-types/index)) — 이전 시도가 실패한 경우 해당 시도의 오류 메시지입니다.
* `retry` ([UInt64](/ko/reference/data-types/index)) — 현재 갱신에서 지금까지 실패한 시도 횟수입니다. status가 `RunningOnAnotherReplica`이면 사용할 수 없습니다.
* `progress` ([Nullable(Float64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신의 진행률로, 0과 1 사이의 값입니다. status가 `RunningOnAnotherReplica`이거나 갱신이 실행 중이 아니면 NULL입니다.
* `read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신이 읽은 행 수입니다. status가 `RunningOnAnotherReplica`이면 NULL입니다.
* `read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신이 읽은 바이트 수입니다. status가 `RunningOnAnotherReplica`이면 NULL입니다.
* `total_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신이 읽어야 하는 것으로 추정되는 전체 행 수입니다. status가 `RunningOnAnotherReplica`이면 NULL입니다.
* `written_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신이 기록한 행 수입니다. status가 `RunningOnAnotherReplica`이면 NULL입니다.
* `written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 레플리카에서 현재 실행 중이거나 가장 최근에 완료된 갱신이 기록한 바이트 수입니다. status가 `RunningOnAnotherReplica`이면 NULL입니다.

{/*AUTOGENERATED_END*/}
