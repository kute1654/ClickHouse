<div id="description">
  ## 설명
</div>

`system.errors` 테이블의 오류 값 이력을 포함합니다. 이 데이터는 주기적으로 디스크에 플러시됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 쿼리를 실행하는 서버의 호스트명입니다.
* `event_date` ([Date](/ko/reference/data-types/date)) — 이벤트 날짜입니다.
* `event_time` ([DateTime](/ko/reference/data-types/datetime)) — 이벤트 시간입니다.
* `code` ([Int32](/ko/reference/data-types/int-uint)) — 오류 코드입니다.
* `error` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 오류 이름입니다.
* `value` ([UInt64](/ko/reference/data-types/int-uint)) — 일정 시간 인터벌 동안 발생한 오류 수입니다.
* `remote` ([UInt8](/ko/reference/data-types/int-uint)) — 원격 예외입니다(즉, 분산 쿼리 중 하나에서 수신된 예외).
* `last_error_time` ([DateTime](/ko/reference/data-types/datetime)) — 마지막 오류가 발생한 시간입니다.
* `last_error_message` ([String](/ko/reference/data-types/string)) — 마지막 오류 메시지입니다.
* `last_error_query_id` ([String](/ko/reference/data-types/string)) — 마지막 오류를 일으킨 쿼리의 ID입니다(사용 가능한 경우).
* `last_error_trace` ([Array(UInt64)](/ko/reference/data-types/array)) — 호출된 메서드가 저장된 물리 주소 목록을 나타내는 스택 트레이스입니다.

{/*AUTOGENERATED_END*/}
