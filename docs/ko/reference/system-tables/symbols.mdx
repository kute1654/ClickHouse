<div id="description">
  ## 설명
</div>

`clickhouse` 바이너리의 내부 검사 정보를 포함합니다. 액세스하려면 내부 검사 권한이 필요합니다.
이 테이블(table)은 C++ 전문가와 ClickHouse 엔지니어에게만 유용합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/ko/reference/data-types/index)) — 바이너리의 심볼 이름입니다. 맹글링된 상태이며, 사람이 읽을 수 있는 이름을 얻으려면 `demangle(symbol)`을 적용할 수 있습니다.
* `address_begin` ([UInt64](/ko/reference/data-types/index)) — 바이너리 내 심볼의 시작 주소입니다.
* `address_end` ([UInt64](/ko/reference/data-types/index)) — 바이너리 내 심볼의 종료 주소입니다.

{/*AUTOGENERATED_END*/}

  ## 예시
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
