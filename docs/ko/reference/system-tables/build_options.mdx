<div id="description">
  ## 설명
</div>

ClickHouse 서버의 빌드 옵션 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 빌드 옵션의 이름입니다.
* `value` ([String](/ko/reference/data-types/index)) — 빌드 옵션의 값입니다.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
