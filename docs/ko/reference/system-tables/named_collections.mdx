  ## 설명

SQL 쿼리를 통해 생성되었거나 설정 파일에서 파싱된 모든 이름이 지정된 컬렉션의 목록입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 컬렉션의 이름입니다.
* `collection` ([Map(String, String)](/ko/reference/data-types/index)) — 컬렉션의 내부 정보입니다.
* `source` ([String](/ko/reference/data-types/index)) — 명명된 컬렉션의 소스입니다.
* `create_query` ([String](/ko/reference/data-types/index)) — 명명된 컬렉션을 생성하는 쿼리입니다.
