<div id="description">
  ## 설명
</div>

ZooKeeper가 구성되어 있지 않으면 이 테이블은 존재하지 않습니다. `system.zookeeper_connection` 테이블은 현재 ZooKeeper에 대한 연결(보조 ZooKeeper 포함) 상태를 보여줍니다. 각 행에는 하나의 연결에 대한 정보가 표시됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/string)) — ZooKeeper 클러스터의 이름입니다.
* `host` ([String](/ko/reference/data-types/string)) — ClickHouse가 연결된 ZooKeeper 노드의 호스트명/IP입니다.
* `port` ([UIn16](/ko/reference/data-types/int-uint)) — ClickHouse가 연결된 ZooKeeper 노드의 포트입니다.
* `index` ([Nullable(UInt8)](/ko/reference/data-types/int-uint)) — ClickHouse가 연결된 ZooKeeper 노드의 인덱스입니다. 이 인덱스는 ZooKeeper 설정의 값을 사용합니다. 연결되어 있지 않으면 이 컬럼은 NULL입니다.
* `connected_time` ([DateTime](/ko/reference/data-types/datetime)) — 연결이 설정된 시점입니다.
* `session_uptime_elapsed_seconds` ([UInt64](/ko/reference/data-types/int-uint)) — 연결이 설정된 후 경과한 시간(초)입니다.
* `is_expired` ([UInt8](/ko/reference/data-types/int-uint)) — 현재 연결의 만료 여부를 나타냅니다.
* `keeper_api_version` ([UInt8](/ko/reference/data-types/int-uint)) — Keeper API 버전입니다.
* `client_id` ([Int64](/ko/reference/data-types/int-uint)) — 연결의 세션 ID입니다.
* `xid` ([Int64](/ko/reference/data-types/int-uint)) — 현재 세션의 XID입니다.
* `enabled_feature_flags` ([Array(Enum16)](/ko/reference/data-types/array)) — 활성화된 기능 플래그입니다. ClickHouse Keeper에만 적용됩니다. 가능한 값은 `FILTERED_LIST`, `MULTI_READ`, `CHECK_NOT_EXISTS`, `CREATE_IF_NOT_EXISTS`, `REMOVE_RECURSIVE`입니다.
* `availability_zone` ([String](/ko/reference/data-types/string)) — 가용 영역입니다.
* `session_timeout_ms` ([UInt64](/ko/reference/data-types/int-uint)) — 세션 timeout(밀리초)입니다.
* `last_zxid_seen` ([Int64](/ko/reference/data-types/int-uint)) — 현재 세션에서 마지막으로 확인된 zxid입니다.

{/*AUTOGENERATED_END*/}
