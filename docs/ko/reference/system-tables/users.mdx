<div id="description">
  ## 설명
</div>

서버에 구성된 [사용자 계정](/ko/concepts/features/security/access-rights#user-account-management) 목록이 포함됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 사용자 이름입니다.
* `id` ([UUID](/ko/reference/data-types/index)) — 사용자 ID입니다.
* `storage` ([String](/ko/reference/data-types/index)) — 사용자 저장소의 경로입니다. `access_control_path` 매개변수에서 구성됩니다.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/ko/reference/data-types/index)) — 인증 유형을 표시합니다. 사용자 식별 방법은 여러 가지이며, 비밀번호 없음, 평문 비밀번호, SHA256으로 인코딩된 비밀번호, 이중 SHA-1로 인코딩된 비밀번호 또는 bcrypt로 인코딩된 비밀번호를 사용할 수 있습니다.
* `auth_params` ([Array(String)](/ko/reference/data-types/index)) — `auth_type`에 따라 달라지는 JSON 포맷의 인증 매개변수입니다.
* `valid_until` ([Array(DateTime)](/ko/reference/data-types/index)) — 사용자 자격 증명의 만료 날짜 및 시간입니다.
* `host_ip` ([Array(String)](/ko/reference/data-types/index)) — ClickHouse 서버에 연결할 수 있도록 허용된 호스트의 IP 주소입니다.
* `host_names` ([Array(String)](/ko/reference/data-types/index)) — ClickHouse 서버에 연결할 수 있도록 허용된 호스트 이름입니다.
* `host_names_regexp` ([Array(String)](/ko/reference/data-types/index)) — ClickHouse 서버에 연결할 수 있도록 허용된 호스트 이름에 대한 정규식입니다.
* `host_names_like` ([Array(String)](/ko/reference/data-types/index)) — LIKE 프레디케이트를 사용해 설정된, ClickHouse 서버에 연결할 수 있도록 허용된 호스트 이름입니다.
* `default_roles_all` ([UInt8](/ko/reference/data-types/index)) — 사용자에게 부여된 모든 역할이 기본값으로 설정되는지 표시합니다.
* `default_roles_list` ([Array(String)](/ko/reference/data-types/index)) — 기본값으로 제공되는 부여된 역할 목록입니다.
* `default_roles_except` ([Array(String)](/ko/reference/data-types/index)) — 나열된 역할을 제외한 모든 부여된 역할이 기본값으로 설정됩니다.
* `grantees_any` ([UInt8](/ko/reference/data-types/index)) — grant option을 가진 사용자가 이를 누구에게나 부여할 수 있는지 나타내는 플래그입니다.
* `grantees_list` ([Array(String)](/ko/reference/data-types/index)) — 이 사용자가 grant option을 부여할 수 있는 사용자 또는 역할 목록입니다.
* `grantees_except` ([Array(String)](/ko/reference/data-types/index)) — 이 사용자가 grant option을 부여할 수 없는 사용자 또는 역할 목록입니다.
* `default_database` ([String](/ko/reference/data-types/index)) — 이 사용자의 기본 데이터베이스 이름입니다.
