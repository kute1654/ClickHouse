<div id="description">
  ## 설명
</div>

세그먼트로 전송하기 위해 큐에 대기 중인 로컬 파일에 관한 정보를 포함합니다. 이러한 로컬 파일에는 비동기 모드에서 분산 테이블에 새 데이터를 삽입할 때 생성되는 새 파트가 들어 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `data_path` ([String](/ko/reference/data-types/index)) — 로컬 파일이 있는 폴더의 경로입니다.
* `is_blocked` ([UInt8](/ko/reference/data-types/index)) — 로컬 파일을 서버로 전송할 수 없는지 여부를 나타내는 값입니다.
* `error_count` ([UInt64](/ko/reference/data-types/index)) — 오류 개수입니다.
* `data_files` ([UInt64](/ko/reference/data-types/index)) — 폴더 내 로컬 파일 개수입니다.
* `data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 로컬 파일 내 압축된 데이터 크기(바이트)입니다.
* `broken_data_files` ([UInt64](/ko/reference/data-types/index)) — 손상된 것으로 표시된 파일 개수입니다(오류로 인해).
* `broken_data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 손상된 파일 내 압축된 데이터 크기(바이트)입니다.
* `last_exception` ([String](/ko/reference/data-types/index)) — 마지막으로 발생한 오류에 대한 텍스트 메시지입니다(있는 경우).
* `last_exception_time` ([DateTime](/ko/reference/data-types/index)) — 마지막 예외가 발생한 시각입니다.

{/*AUTOGENERATED_END*/}
