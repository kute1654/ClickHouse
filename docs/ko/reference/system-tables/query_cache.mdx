<div id="description">
  ## 설명
</div>

[쿼리 캐시](/ko/concepts/features/performance/caches/query-cache)의 내용을 보여줍니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ko/reference/data-types/index)) — 쿼리 문자열입니다.
* `query_id` ([String](/ko/reference/data-types/index)) — 쿼리 ID입니다.
* `is_subquery` ([UInt8](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리가 서브쿼리 결과인지 여부입니다.
* `result_size` ([UInt64](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리의 크기입니다.
* `tag` ([LowCardinality(String)](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리의 태그입니다.
* `stale` ([UInt8](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리가 오래된 상태인지 여부입니다.
* `shared` ([UInt8](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리가 여러 사용자 간에 공유되는지 여부입니다.
* `compressed` ([UInt8](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리가 압축되었는지 여부입니다.
* `expires_at` ([DateTime](/ko/reference/data-types/index)) — 쿼리 캐시 엔트리가 오래된 상태가 되는 시점입니다.
* `key_hash` ([UInt64](/ko/reference/data-types/index)) — 쿼리 문자열의 해시로, 쿼리 캐시 엔트리를 찾는 키로 사용됩니다.

{/*AUTOGENERATED_END*/}
