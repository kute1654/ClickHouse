<div id="description">
  ## 설명
</div>

구성된 설정 프로필의 속성이 포함되어 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 설정 프로필 이름입니다.
* `id` ([UUID](/ko/reference/data-types/index)) — 설정 프로필 ID입니다.
* `storage` ([String](/ko/reference/data-types/index)) — 설정 프로필 저장소의 경로입니다. `access_control_path` 매개변수에서 구성합니다.
* `num_elements` ([UInt64](/ko/reference/data-types/index)) — `system.settings_profile_elements` 테이블(table)에서 이 프로필의 요소 수입니다.
* `apply_to_all` ([UInt8](/ko/reference/data-types/index)) — 이 설정 프로필이 모든 역할 및/또는 사용자에게 적용됨을 나타냅니다.
* `apply_to_list` ([Array(String)](/ko/reference/data-types/index)) — 이 설정 프로필이 적용되는 역할 및/또는 사용자의 목록입니다.
* `apply_to_except` ([Array(String)](/ko/reference/data-types/index)) — 나열된 역할 및/또는 사용자를 제외한 모든 역할 및/또는 사용자에게 이 설정 프로필이 적용됩니다.

{/*AUTOGENERATED_END*/}
