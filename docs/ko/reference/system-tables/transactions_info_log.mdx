현재 서버에서 실행된 모든 트랜잭션의 정보를 포함합니다.

이 테이블은 언제든지 TRUNCATE하거나 삭제해도 안전합니다.

컬럼:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ko/reference/data-types/lowcardinality)) — 트랜잭션이 실행된 호스트명입니다.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/ko/reference/data-types/enum)) — 트랜잭션의 유형입니다. 가능한 값: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/ko/reference/data-types/date)) — 항목의 날짜입니다.
* `event_time` ([DateTime64(6)](/ko/reference/data-types/datetime64)) — 항목의 시간입니다.
* `thread_id` ([UInt64](/ko/reference/data-types/int-uint)) — 스레드 식별자입니다.
* `query_id` ([String](/ko/reference/data-types/string)) — 트랜잭션 범위에서 실행된 쿼리의 ID입니다.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/ko/reference/data-types/tuple)) — 트랜잭션 식별자입니다.
* `tid_hash` ([UInt64](/ko/reference/data-types/int-uint)) — 식별자의 해시값입니다.
* `csn` ([UInt64](/ko/reference/data-types/int-uint)) — Commit Sequence Number입니다.
* `database` ([String](/ko/reference/data-types/string)) — 트랜잭션이 실행된 데이터베이스의 이름입니다.
* `table` ([String](/ko/reference/data-types/string)) — 트랜잭션이 실행된 테이블의 이름입니다.
* `uuid` ([UUID](/ko/reference/data-types/uuid)) — 트랜잭션이 실행된 테이블의 UUID입니다.
* `part` ([String](/ko/reference/data-types/string)) — 트랜잭션에 참여한 part의 이름입니다.
