<div id="description">
  ## 설명
</div>

로컬 서버에 있는 [리소스](/ko/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage)에 관한 정보를 담고 있습니다. 이 테이블에는 각 리소스에 대해 1개의 행이 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 리소스의 이름입니다.
* `read_disks` ([Array(String)](/ko/reference/data-types/index)) — 읽기 작업에 이 리소스를 사용하는 디스크 이름의 목록입니다.
* `write_disks` ([Array(String)](/ko/reference/data-types/index)) — 쓰기 작업에 이 리소스를 사용하는 디스크 이름의 목록입니다.
* `unit` ([String](/ko/reference/data-types/index)) — 비용 측정에 사용되는 리소스 단위입니다.
* `create_query` ([String](/ko/reference/data-types/index)) — 리소스의 CREATE 쿼리입니다.

{/*AUTOGENERATED_END*/}

예시:

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
Row 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

Row 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```
