<div id="description">
  ## 설명
</div>

[서버 구성](/ko/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure)에 정의된 디스크 정보가 포함됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 서버 구성에 있는 디스크 이름입니다.
* `path` ([String](/ko/reference/data-types/index)) — 파일 시스템의 마운트 지점 경로입니다.
* `free_space` ([UInt64](/ko/reference/data-types/index)) — 디스크의 남은 공간으로, 바이트 단위입니다.
* `total_space` ([UInt64](/ko/reference/data-types/index)) — 디스크의 전체 용량으로, 바이트 단위입니다.
* `unreserved_space` ([UInt64](/ko/reference/data-types/index)) — 예약에 사용되지 않은 여유 공간입니다(`free&#95;space`에서 현재 실행 중인 머지, 삽입 및 기타 디스크 쓰기 작업이 차지한 예약 크기를 뺀 값).
* `keep_free_space` ([UInt64](/ko/reference/data-types/index)) — 디스크에서 항상 비워 두어야 하는 공간의 양으로, 바이트 단위입니다. 디스크 구성의 `keep&#95;free&#95;space&#95;bytes` 매개변수에서 정의됩니다.
* `type` ([String](/ko/reference/data-types/index)) — 이 디스크가 데이터를 저장하는 위치를 나타내는 디스크 유형입니다. RAM, 로컬 드라이브 또는 원격 스토리지일 수 있습니다.
* `object_storage_type` ([String](/ko/reference/data-types/index)) — 디스크 유형이 `object&#95;storage`인 경우의 객체 스토리지 유형입니다.
* `metadata_type` ([String](/ko/reference/data-types/index)) — 디스크 유형이 `object&#95;storage`인 경우의 메타데이터 저장소 유형입니다.
* `is_encrypted` ([UInt8](/ko/reference/data-types/index)) — 이 디스크가 저장된 데이터를 암호화하는지 여부를 나타내는 플래그입니다.
* `is_read_only` ([UInt8](/ko/reference/data-types/index)) — 이 디스크에서는 읽기 작업만 수행할 수 있음을 나타내는 플래그입니다.
* `is_write_once` ([UInt8](/ko/reference/data-types/index)) — 디스크가 write-once인지 여부를 나타내는 플래그입니다. 즉, 이 디스크로의 BACKUP은 지원하지만, 이 디스크의 MergeTree 테이블에 대한 INSERT는 지원하지 않습니다.
* `is_remote` ([UInt8](/ko/reference/data-types/index)) — 이 디스크에 대한 작업이 네트워크 통신을 수반하는지 여부를 나타내는 플래그입니다.
* `is_broken` ([UInt8](/ko/reference/data-types/index)) — 디스크가 손상되었는지 여부를 나타내는 플래그입니다. 손상된 디스크는 사용 가능한 공간이 0이며 사용할 수 없습니다.
* `cache_path` ([String](/ko/reference/data-types/index)) — 디스크가 캐싱을 지원하는 경우 로컬 드라이브에 있는 캐시 디렉터리의 경로입니다.

{/*AUTOGENERATED_END*/}
