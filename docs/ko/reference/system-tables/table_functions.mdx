<div id="description">
  ## 설명
</div>

사용 가능한 모든 테이블 함수와 각 함수에 대한 설명 목록입니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 테이블 함수의 이름입니다.
* `description` ([String](/ko/reference/data-types/index)) — 테이블 함수에 대한 간단한 설명입니다.
* `allow_readonly` ([UInt8](/ko/reference/data-types/index)) — readonly 사용자가 이 함수를 사용할 수 있는지 여부를 나타내는 플래그입니다.

{/*AUTOGENERATED_END*/}
