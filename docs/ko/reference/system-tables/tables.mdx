<div id="description">
  ## 설명
</div>

서버가 인식하고 있는 각 테이블의 메타데이터를 포함합니다.

[분리된](/ko/reference/statements/detach) 테이블은 `system.tables`에 표시되지 않습니다.

[임시 테이블](/ko/reference/statements/create/table#temporary-tables)은 생성된 세션에서만 `system.tables`에 표시됩니다. 이러한 테이블은 `database` 필드가 비어 있고 `is_temporary` 플래그가 켜진 상태로 표시됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 테이블이 속한 데이터베이스 이름입니다.
* `name` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `uuid` ([UUID](/ko/reference/data-types/index)) — 테이블 UUID입니다(Atomic 데이터베이스).
* `engine` ([String](/ko/reference/data-types/index)) — 테이블 엔진 이름입니다(매개변수 제외).
* `is_temporary` ([UInt8](/ko/reference/data-types/index)) — 테이블이 임시 테이블인지 나타내는 플래그입니다.
* `data_paths` ([Array(String)](/ko/reference/data-types/index)) — 파일 시스템에 있는 테이블 데이터의 경로입니다.
* `metadata_path` ([String](/ko/reference/data-types/index)) — 파일 시스템에 있는 테이블 메타데이터의 경로입니다.
* `metadata_modification_time` ([DateTime](/ko/reference/data-types/index)) — 테이블 메타데이터가 마지막으로 수정된 시간입니다.
* `metadata_version` ([Int32](/ko/reference/data-types/index)) — ReplicatedMergeTree 테이블의 메타데이터 버전이며, ReplicatedMergeTree 테이블이 아니면 0입니다.
* `dependencies_database` ([Array(String)](/ko/reference/data-types/index)) — 데이터베이스 의존성입니다.
* `dependencies_table` ([Array(String)](/ko/reference/data-types/index)) — 테이블 의존성입니다(현재 테이블에 대한 materialized view).
* `create_table_query` ([String](/ko/reference/data-types/index)) — 테이블 생성에 사용된 쿼리입니다.
* `engine_full` ([String](/ko/reference/data-types/index)) — 테이블 엔진의 매개변수입니다.
* `as_select` ([String](/ko/reference/data-types/index)) — 뷰에 대한 SELECT 쿼리입니다.
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](/ko/reference/data-types/index)) — parameterized view의 매개변수입니다.
* `partition_key` ([String](/ko/reference/data-types/index)) — 테이블에 지정된 파티션 키 표현식입니다.
* `sorting_key` ([String](/ko/reference/data-types/index)) — 테이블에 지정된 정렬 키 표현식입니다.
* `primary_key` ([String](/ko/reference/data-types/index)) — 테이블에 지정된 프라이머리 키 표현식입니다.
* `sampling_key` ([String](/ko/reference/data-types/index)) — 테이블에 지정된 샘플링 키 표현식입니다.
* `unique_key` ([String](/ko/reference/data-types/index)) — 테이블에 지정된 고유 키 표현식입니다(UNIQUE KEY 절).
* `storage_policy` ([String](/ko/reference/data-types/index)) — 스토리지 정책입니다. MergeTree 및 분산 엔진을 사용하는 테이블에 적용됩니다.
* `total_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 전체 행 수입니다. 테이블의 정확한 행 수를 빠르게 확인할 수 있으면 해당 값을, 그렇지 않으면 NULL을 반환합니다(기반 Buffer 테이블 포함).
* `total_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 전체 바이트 수입니다. 스토리지에 있는 테이블의 정확한 바이트 수를 빠르게 확인할 수 있으면 해당 값을, 그렇지 않으면 NULL을 반환합니다(기반 스토리지는 포함하지 않음). 테이블이 데이터를 디스크에 저장하는 경우 디스크에서 사용 중인 공간(즉, 압축된 크기)을 반환합니다. 테이블이 데이터를 메모리에 저장하는 경우 메모리에서 사용 중인 바이트 수의 근사값을 반환합니다.
* `total_bytes_uncompressed` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 압축되지 않은 전체 바이트 수입니다. 스토리지에 있는 테이블의 파트 checksums에서 정확한 바이트 수를 빠르게 확인할 수 있으면 해당 값을, 그렇지 않으면 NULL을 반환합니다(기반 스토리지가 있는 경우에도 이를 고려하지 않음).
* `parts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 이 테이블의 전체 파트 수입니다.
* `active_parts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 이 테이블의 활성 파트 수입니다.
* `total_marks` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 이 테이블의 모든 파트에 있는 마크의 총수입니다.
* `active_on_fly_data_mutations` ([UInt64](/ko/reference/data-types/index)) — 즉시 적용 가능한 활성 데이터 뮤테이션(UPDATE 및 DELETE)의 총수입니다.
* `active_on_fly_alter_mutations` ([UInt64](/ko/reference/data-types/index)) — 즉시 적용 가능한 활성 ALTER 뮤테이션(MODIFY COLUMN)의 총수입니다.
* `active_on_fly_metadata_mutations` ([UInt64](/ko/reference/data-types/index)) — 즉시 적용 가능한 활성 메타데이터 뮤테이션(RENAME)의 총수입니다.
* `columns_descriptions_cache_size` ([UInt64](/ko/reference/data-types/index)) — *MergeTree 테이블용 컬럼 설명 캐시 크기입니다.
* `lifetime_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 서버 시작 이후 INSERT된 총 행 수입니다(Buffer 테이블에만 해당).
* `lifetime_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 서버 시작 이후 INSERT된 총 바이트 수입니다(Buffer 테이블에만 해당).
* `comment` ([String](/ko/reference/data-types/index)) — 테이블 주석입니다.
* `has_own_data` ([UInt8](/ko/reference/data-types/index)) — 테이블 자체가 디스크에 데이터를 저장하는지, 아니면 다른 소스에만 접근하는지를 나타내는 플래그입니다.
* `loading_dependencies_database` ([Array(String)](/ko/reference/data-types/index)) — 데이터베이스 로딩 의존성(현재 객체보다 먼저 로드되어야 하는 객체 목록)입니다.
* `loading_dependencies_table` ([Array(String)](/ko/reference/data-types/index)) — 테이블 로딩 의존성(현재 객체보다 먼저 로드되어야 하는 객체 목록)입니다.
* `loading_dependent_database` ([Array(String)](/ko/reference/data-types/index)) — 로딩이 종속된 데이터베이스입니다.
* `loading_dependent_table` ([Array(String)](/ko/reference/data-types/index)) — 로딩이 종속된 테이블입니다.
* `target_database` ([String](/ko/reference/data-types/index)) — materialized view의 경우, 뷰가 데이터를 기록하는 대상 테이블의 데이터베이스입니다(`TO` 대상 또는 암시적 `.inner.*` 테이블). 다른 엔진에서는 비어 있습니다.
* `target_table` ([String](/ko/reference/data-types/index)) — materialized view의 경우, 뷰가 데이터를 기록하는 대상 테이블의 이름입니다(`TO` 대상 또는 암시적 `.inner.*` 테이블). 다른 엔진에서는 비어 있습니다.
* `definer` ([String](/ko/reference/data-types/index)) — 테이블에 사용되는 SQL security definer의 이름입니다.

{/*AUTOGENERATED_END*/}
