<div id="description">
  ## 설명
</div>

drop table이 실행되었으나 아직 데이터 정리가 수행되지 않은 테이블에 대한 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `index` ([UInt32](/ko/reference/data-types/index)) — marked&#95;dropped&#95;tables 큐의 인덱스입니다.
* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `uuid` ([UUID](/ko/reference/data-types/index)) — 테이블 UUID입니다.
* `engine` ([String](/ko/reference/data-types/index)) — 테이블 엔진 이름입니다.
* `metadata_dropped_path` ([String](/ko/reference/data-types/index)) — metadata&#95;dropped 디렉터리에 있는 테이블 메타데이터 파일의 경로입니다.
* `table_dropped_time` ([DateTime](/ko/reference/data-types/index)) — 테이블 데이터에 대한 다음 삭제 시도가 예약된 시간입니다. 일반적으로 테이블이 삭제된 시각에 `database_atomic_delay_before_drop_table_sec`를 더한 값입니다.

{/*AUTOGENERATED_END*/}
