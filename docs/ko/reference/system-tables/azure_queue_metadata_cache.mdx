<div id="description">
  ## 설명
</div>

AzureQueue 메타데이터의 메모리에 저장된 상태와 파일별 현재 처리 중인 행을 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/ko/reference/data-types/index)) — 메타데이터의 ZooKeeper 경로
* `file_path` ([String](/ko/reference/data-types/index)) — 처리 중인 파일의 경로
* `file_name` ([String](/ko/reference/data-types/index)) — 처리 중인 파일의 이름
* `rows_processed` ([UInt64](/ko/reference/data-types/index)) — 현재까지 처리된 행 수
* `status` ([String](/ko/reference/data-types/index)) — 처리 상태: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 파일 처리가 시작된 시각
* `processing_end_time` ([Nullable(DateTime)](/ko/reference/data-types/index)) — 파일 처리가 종료된 시각
* `exception` ([String](/ko/reference/data-types/index)) — 처리 중 발생한 예외

{/*AUTOGENERATED_END*/}
