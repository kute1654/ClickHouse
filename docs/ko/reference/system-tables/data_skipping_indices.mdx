<div id="description">
  ## 설명
</div>

모든 테이블에 있는 데이터 스키핑 인덱스에 대한 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `name` ([String](/ko/reference/data-types/index)) — 인덱스 이름입니다.
* `type` ([String](/ko/reference/data-types/index)) — 인덱스 유형입니다.
* `type_full` ([String](/ko/reference/data-types/index)) — CREATE 문에서의 인덱스 유형 표현식입니다.
* `expr` ([String](/ko/reference/data-types/index)) — 인덱스 계산용 표현식입니다.
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/ko/reference/data-types/index)) — 인덱스가 암시적으로 생성되었는지 여부입니다(`add_minmax_index_for_numeric_columns` 또는 이와 유사한 방식을 통해 생성된 경우).
* `granularity` ([UInt64](/ko/reference/data-types/index)) — 블록 내 그래뉼 수입니다.
* `data_compressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 압축된 데이터의 크기(바이트)입니다.
* `data_uncompressed_bytes` ([UInt64](/ko/reference/data-types/index)) — 압축 해제된 데이터의 크기(바이트)입니다.
* `marks_bytes` ([UInt64](/ko/reference/data-types/index)) — 마크의 크기(바이트)입니다.

{/*AUTOGENERATED_END*/}
