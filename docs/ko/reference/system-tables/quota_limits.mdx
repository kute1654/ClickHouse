<div id="description">
  ## 설명
</div>

각 QUOTA의 모든 인터벌에 대한 최대값 정보가 포함됩니다. 하나의 QUOTA에 0개 이상의 행이 해당할 수 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ko/reference/data-types/index)) — Quota 이름입니다.
* `duration` ([UInt32](/ko/reference/data-types/index)) — 리소스 활용을 계산하는 시간 인터벌의 길이(초)입니다.
* `is_randomized_interval` ([UInt8](/ko/reference/data-types/index)) — Boolean 값입니다. 인터벌이 무작위화되었는지 여부를 나타냅니다. 무작위화되지 않은 경우 인터벌은 항상 같은 시각에 시작됩니다. 예를 들어 1분 인터벌은 항상 분 단위의 정각에 시작됩니다(즉, 11:20:00에는 시작할 수 있지만 11:20:01에는 시작하지 않습니다). 1일 인터벌은 항상 UTC 자정에 시작됩니다. 인터벌이 무작위화된 경우 첫 번째 인터벌은 임의의 시각에 시작하고, 이후 인터벌은 순차적으로 시작됩니다. 값: 0 — 인터벌이 무작위화되지 않음, 1 — 인터벌이 무작위화됨.
* `max_queries` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 모든 타입의 쿼리 최대 개수입니다.
* `max_query_selects` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 SELECT 쿼리의 최대 개수입니다.
* `max_query_inserts` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 실행이 허용되는 INSERT 쿼리의 최대 개수입니다.
* `max_errors` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 오류 쿼리의 최대 개수입니다.
* `max_result_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 모든 쿼리의 결과 집합(result set)에 포함될 수 있는 행의 총 최대 개수입니다.
* `max_result_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 모든 쿼리의 결과 집합(result set)에 포함될 수 있는 바이트의 총 최대 개수입니다.
* `max_read_rows` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 행의 최대 개수입니다.
* `max_read_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 모든 쿼리 실행 중 읽을 수 있는 바이트의 최대 개수입니다.
* `max_execution_time` ([Nullable(Float64)](/ko/reference/data-types/index)) — 지정된 주기 내에 모든 쿼리가 실행되는 데 허용되는 최대 시간(나노초)입니다.
* `max_written_bytes` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 모든 쿼리 실행 중 쓸 수 있는 바이트의 최대 개수입니다.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 연속 인증 실패의 최대 개수입니다.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ko/reference/data-types/index)) — 지정된 주기 내에 허용되는 단일 정규화된 쿼리의 최대 실행 횟수입니다.
