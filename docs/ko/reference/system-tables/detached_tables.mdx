<div id="description">
  ## 설명
</div>

각 분리된 테이블의 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 테이블이 속한 데이터베이스 이름입니다.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름입니다.
* `uuid` ([UUID](/ko/reference/data-types/index)) — 테이블 UUID입니다(Atomic 데이터베이스).
* `metadata_path` ([String](/ko/reference/data-types/index)) — 파일 시스템에서 테이블 메타데이터의 경로입니다.
* `is_permanently` ([UInt8](/ko/reference/data-types/index)) — 테이블이 영구적으로 분리되었습니다.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
