<div id="description">
  ## 설명
</div>

모든 테이블에 존재하는 제약 조건에 대한 정보를 포함합니다.

[임시 테이블(temporary tables)](/ko/reference/statements/create/table#temporary-tables)에 정의된 제약 조건은 생성된 세션에서만 `system.constraints`에 표시됩니다. 이 제약 조건은 `database` 필드가 비어 있는 상태로 표시됩니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ko/reference/data-types/index)) — 데이터베이스 이름.
* `table` ([String](/ko/reference/data-types/index)) — 테이블 이름.
* `name` ([String](/ko/reference/data-types/index)) — 제약 조건 이름.
* `type` ([Enum8(&#39;CHECK&#39; = 0, &#39;ASSUME&#39; = 1)](/ko/reference/data-types/index)) — 제약 조건 유형.
* `expression` ([String](/ko/reference/data-types/index)) — 제약 조건 표현식.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql
SELECT * FROM system.constraints LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       hits
name:        check_hits
type:        CHECK
expression:  CounterID > 0

Row 2:
──────
database:    default
table:       hits
name:        assume_positive
type:        ASSUME
expression:  WatchID > 0
```
