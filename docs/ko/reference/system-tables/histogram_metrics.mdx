<div id="description">
  ## 설명
</div>

이 테이블(table)에는 즉시 계산되어 Prometheus 포맷으로 내보낼 수 있는 히스토그램 메트릭이 포함되어 있습니다. 항상 최신 상태입니다. 더 이상 사용이 권장되지 않는 `system.latency_log`를 대체합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ko/reference/data-types/index)) — 메트릭 이름.
* `value` ([Float64](/ko/reference/data-types/index)) — 메트릭 값.
* `description` ([String](/ko/reference/data-types/index)) — 메트릭 설명.
* `labels` ([Map(String, String)](/ko/reference/data-types/index)) — 메트릭 레이블.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

다음과 같은 쿼리를 사용하면 모든 히스토그램 메트릭을 Prometheus 포맷으로 내보낼 수 있습니다.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
