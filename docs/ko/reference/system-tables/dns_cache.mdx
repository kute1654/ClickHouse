<div id="description">
  ## 설명
</div>

캐시된 DNS 레코드 정보를 포함합니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/ko/reference/data-types/index)) — 호스트명.
* `ip_address` ([String](/ko/reference/data-types/index)) — IP 주소.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/ko/reference/data-types/index)) — IP 주소 유형.
* `cached_at` ([DateTime](/ko/reference/data-types/index)) — 레코드가 캐시된 시각.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| 호스트명      | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
