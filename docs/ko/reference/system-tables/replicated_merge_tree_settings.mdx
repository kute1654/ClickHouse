<div id="description">
  ## 설명
</div>

모든 ReplicatedMergeTree 엔진별 설정의 현재값, 기본값 및 설명을 포함한 목록입니다. 이 값들은 CREATE 쿼리의 SETTINGS 섹션에서 변경할 수 있습니다.

<div id="columns">
  ## 컬럼
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 설정 이름입니다.
* `value` ([String](/ko/reference/data-types/index)) — 설정 값입니다.
* `default` ([String](/ko/reference/data-types/index)) — 설정 기본값입니다.
* `changed` ([UInt8](/ko/reference/data-types/index)) — config에서 설정이 명시적으로 정의되었거나 명시적으로 변경된 경우 1입니다.
* `description` ([String](/ko/reference/data-types/index)) — 설정 설명입니다.
* `min` ([Nullable(String)](/ko/reference/data-types/index)) — constraints에 의해 설정된 최소값입니다. 설정에 최소값이 없으면 NULL을 포함합니다.
* `max` ([Nullable(String)](/ko/reference/data-types/index)) — constraints에 의해 설정된 최대값입니다. 설정에 최대값이 없으면 NULL을 포함합니다.
* `disallowed_values` ([Array(String)](/ko/reference/data-types/index)) — 허용되지 않는 값 목록입니다.
* `readonly` ([UInt8](/ko/reference/data-types/index)) — 현재 사용자가 설정을 변경할 수 있는지 여부를 나타냅니다: 0 — 현재 사용자가 설정을 변경할 수 있습니다. 1 — 현재 사용자가 설정을 변경할 수 없습니다.
* `type` ([String](/ko/reference/data-types/index)) — 설정 유형(구현별 문자열 값)입니다.
* `is_obsolete` ([UInt8](/ko/reference/data-types/index)) — 설정이 더 이상 사용되지 않는 상태인지 여부를 나타냅니다.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/ko/reference/data-types/index)) —
  이 기능의 지원 수준입니다. ClickHouse 기능은 현재 개발 상태와
  사용 시 기대할 수 있는 수준에 따라 여러 tier로 구분됩니다:

- PRODUCTION: 이 기능은 안정적이고 안전하게 사용할 수 있으며, 다른 PRODUCTION 기능과 함께 사용해도 문제가 없습니다.
- BETA: 이 기능은 안정적이고 안전합니다. 다른 기능과 함께 사용할 때의 결과는 알려져 있지 않으며 정확성은 보장되지 않습니다. 테스트와 보고를 환영합니다.
- EXPERIMENTAL: 이 기능은 개발 중입니다. 개발자와 ClickHouse에 관심이 많은 사용자를 위한 기능입니다. 이 기능은 작동할 수도 있고 작동하지 않을 수도 있으며 언제든지 제거될 수 있습니다.
- OBSOLETE: 더 이상 지원되지 않습니다. 이미 제거되었거나 향후 릴리스에서 제거될 예정입니다.
