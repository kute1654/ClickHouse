<div id="description">
  ## 설명
</div>

서버에서 지원하는 테이블 엔진과 각 엔진의 기능 지원 정보에 대한 설명입니다.

이 표에는 다음 컬럼이 포함되어 있습니다(컬럼 유형은 괄호 안에 표시됩니다):

{/*AUTOGENERATED_START*/}

* `name` ([String](/ko/reference/data-types/index)) — 테이블 엔진의 이름입니다.
* `supports_settings` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 SETTINGS 절 지원 여부를 나타내는 플래그입니다.
* `supports_skipping_indices` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 스킵 인덱스 지원 여부를 나타내는 플래그입니다.
* `supports_projections` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 프로젝션 지원 여부를 나타내는 플래그입니다.
* `supports_sort_order` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 PARTITION&#95;BY, PRIMARY&#95;KEY, ORDER&#95;BY, SAMPLE&#95;BY 절 지원 여부를 나타내는 플래그입니다.
* `supports_ttl` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 TTL 지원 여부를 나타내는 플래그입니다.
* `supports_replication` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 데이터 복제 지원 여부를 나타내는 플래그입니다.
* `supports_deduplication` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 데이터 중복 제거 지원 여부를 나타내는 플래그입니다.
* `supports_parallel_insert` ([UInt8](/ko/reference/data-types/index)) — 테이블 엔진의 병렬 삽입 지원 여부를 나타내는 플래그입니다 (`max&#95;insert&#95;threads` 설정 참고).
* `description` ([String](/ko/reference/data-types/index)) — 테이블 엔진이 수행하는 작업에 대한 설명입니다. 전용 문서 페이지가 있는 엔진의 경우 해당 페이지의 전체 Markdown 본문이 들어가며, 나머지 엔진의 경우 간결한 요약이 들어갑니다.
* `syntax` ([String](/ko/reference/data-types/index)) — 테이블 생성 시 테이블 엔진을 사용하는 방법입니다. 대부분의 엔진에서는 CREATE TABLE 쿼리의 ENGINE 절을 의미하지만, 일부 엔진(예: 여러 종류의 뷰 또는 Loop 엔진)은 CREATE VIEW나 테이블 함수로부터 SELECT하는 방식처럼 다른 형태로 사용됩니다.
* `examples` ([String](/ko/reference/data-types/index)) — 사용 예시입니다.
* `introduced_in` ([String](/ko/reference/data-types/index)) — 테이블 엔진이 처음 도입된 ClickHouse 버전이며, 형식은 major.minor입니다.
* `related` ([Array(String)](/ko/reference/data-types/index)) — 관련 테이블 엔진의 이름입니다.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 예시
</div>

```sql title="Query"
SELECT
    name,
    supports_settings,
    supports_skipping_indices,
    supports_sort_order,
    supports_ttl,
    supports_replication,
    supports_deduplication,
    supports_parallel_insert
FROM system.table_engines
WHERE name IN ('Kafka', 'MergeTree', 'ReplicatedCollapsingMergeTree')
```
