<Note>
  MySQL 및 MongoDB CDC ClickPipes는 일반 제공(GA) 전까지 무료로 사용할 수 있습니다. 고객에게는 GA 출시 전에 사전 안내하여 ClickPipes 사용량을 검토하고 최적화할 수 있도록 합니다.
</Note>

[ClickPipes](/ko/integrations/clickpipes/home) 청구는 **컴퓨트 사용량**과 **수집된 데이터**를 기준으로 책정됩니다. 각 범주의 요금 모델에 대한 자세한 내용은 해당 청구 페이지를 참조하십시오:

| 페이지                                                                                                                                     | 설명                                |
| --------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------- |
| [ClickPipes for Postgres CDC](/ko/products/cloud/reference/billing/clickpipes/clickpipes-for-cdc)                                          | PostgreSQL CDC ClickPipes의 요금입니다. |
| [ClickPipes for streaming and object storage](/ko/products/cloud/reference/billing/clickpipes/clickpipes-for-streaming-and-object-storage) | 스트리밍 및 객체 스토리지 ClickPipes의 요금입니다. |
