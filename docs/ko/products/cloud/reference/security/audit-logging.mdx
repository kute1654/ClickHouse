<div id="console-audit-log-events">
  ## 콘솔 감사 로그 이벤트
</div>

조직에서 수집되는 다양한 유형의 이벤트는 **조직**, **서비스**, **사용자**의 3개 범주로 나뉩니다. 감사 로그와 API 통합 내보내기 또는 추가 방법에 관한 자세한 내용은 위의 Guides 섹션에 있는 [콘솔 감사 로그](/ko/products/cloud/guides/security/audit-logging/console-audit-log) 문서를 참조하십시오.

다음 이벤트가 감사 로그에 기록됩니다.

<div id="organization">
  ### 조직
</div>

* 조직이 생성됨
* 조직이 삭제됨
* 조직 이름이 변경됨

<div id="service">
  ### 서비스
</div>

* 서비스 생성됨
* 서비스 삭제됨
* 서비스 중지됨
* 서비스 시작됨
* 서비스 이름 변경됨
* 서비스 IP 액세스 목록 변경됨
* 서비스 비밀번호 재설정됨

<div id="user">
  ### 사용자
</div>

* 사용자 역할 변경
* 사용자가 조직에서 제거됨
* 사용자가 조직에 초대됨
* 사용자가 조직에 가입함
* 사용자 초대 삭제
* 사용자가 조직을 떠남
