HyperDX는 [**ClickStack**](/ko/clickstack/overview)의 사용자 인터페이스(UI)입니다. ClickStack은 ClickHouse와 OpenTelemetry (OTel)를 기반으로 구축된 프로덕션급 관측성 플랫폼으로, 로그, 트레이스, 메트릭, 세션 데이터를 단일 고성능 솔루션으로 통합합니다. 복잡한 시스템의 모니터링과 디버깅을 위해 설계된 ClickStack은 개발자와 SRE가 도구를 오가거나 타임스탬프 또는 상관관계 ID를 사용해 데이터를 수동으로 짜 맞추지 않고도 문제를 처음부터 끝까지 추적할 수 있게 합니다.

HyperDX는 관측성 데이터를 탐색하고 시각화하도록 특별히 설계된 프런트엔드로, Lucene 스타일 쿼리와 SQL 쿼리, 대화형 대시보드, 알림, 트레이스 탐색 등을 지원하며, 모두 백엔드인 ClickHouse에 최적화되어 있습니다.

ClickHouse Cloud의 HyperDX를 사용하면 더욱 턴키에 가까운 ClickStack 환경을 이용할 수 있습니다. 관리할 인프라도 없고, 별도로 구성할 인증도 없습니다.
HyperDX는 한 번의 클릭으로 실행하여 데이터에 연결할 수 있으며, ClickHouse Cloud 인증 시스템에 완전히 통합되어 관측성 인사이트에 원활하고 안전하게 액세스할 수 있습니다.

<div id="main-concepts">
  ## 배포
</div>

ClickHouse Cloud의 HyperDX는 현재 비공개 프리뷰 상태이며, 조직 수준에서 활성화해야 합니다. 활성화되면 서비스가 무엇이든 선택 시 왼쪽 기본 탐색 메뉴에서 HyperDX를 사용할 수 있습니다.

<Image img="/images/use-cases/observability/hyperdx_cloud.webp" alt="ClickHouse Cloud HyperDX" size="lg" />

ClickHouse Cloud에서 HyperDX를 시작하려면 전용 [시작하기 가이드](/ko/clickstack/deployment/managed)를 참조하는 것이 좋습니다.

ClickStack에 대한 자세한 내용은 [전체 문서](/ko/clickstack/overview)를 참조하십시오.
