ClickHouse BYOC는 보안을 강화하고 서비스에 직접 연결할 수 있도록 다양한 프라이빗 네트워킹 옵션을 지원합니다. 이 가이드는 자체 AWS 또는 GCP 계정에서 실행되는 ClickHouse Cloud 배포를 내부 애플리케이션이나 분석 도구와 같은 다른 네트워크 또는 서비스에 안전하게 연결하기 위한 권장 방식을 안내합니다. VPC Peering, AWS PrivateLink, GCP Private Service Connect 등의 옵션을 다루며, 각 옵션의 주요 단계와 고려 사항을 설명합니다.

ClickHouse BYOC 배포에 프라이빗 네트워크 연결이 필요한 경우, 각 가이드의 단계를 따르거나 더 고급 시나리오에 대한 도움이 필요하면 ClickHouse Support에 문의하십시오.
