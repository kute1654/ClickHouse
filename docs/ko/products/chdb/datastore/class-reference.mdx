이 참고 문서에서는 DataStore API의 핵심 클래스를 다룹니다.

<div id="datastore">
  ## DataStore
</div>

데이터를 다루기 위한 대표적인 DataFrame 유사 클래스입니다.

```python
from chdb.datastore import DataStore
```

<div id="datastore-constructor">
  ### 생성자
</div>

```python
DataStore(data=None, columns=None, index=None, dtype=None, copy=None)
```

**매개변수:**

| 매개변수      | 유형                            | 설명        |
| --------- | ----------------------------- | --------- |
| `data`    | dict/list/DataFrame/DataStore | 입력 데이터    |
| `columns` | list                          | 컬럼 이름     |
| `index`   | Index                         | 행 인덱스     |
| `dtype`   | dict                          | 컬럼 데이터 타입 |
| `copy`    | bool                          | 데이터 복사 여부 |

**예시:**

```python
# 딕셔너리에서 생성
ds = DataStore({'a': [1, 2, 3], 'b': ['x', 'y', 'z']})

# pandas DataFrame에서 생성
import pandas as pd
ds = DataStore(pd.DataFrame({'a': [1, 2, 3]}))

# 빈 DataStore
ds = DataStore()
```

<div id="columnexpr-arithmetic">
  ### 산술 연산
</div>

```python
ds['total'] = ds['price'] * ds['quantity']
```

<div id="columnexpr-comparison">
  ### 비교 연산
</div>

```python
ds[ds['age'] > 25]           # 초과
ds[ds['age'] >= 25]          # 이상
ds[ds['age'] < 25]           # 미만
ds[ds['age'] <= 25]          # 이하
ds[ds['name'] == 'Alice']    # 같음
ds[ds['name'] != 'Bob']      # 같지 않음
```

<div id="columnexpr-logical">
  ### 논리 연산
</div>

```python
ds[(ds['age'] > 25) & (ds['city'] == 'NYC')]    # AND
ds[(ds['age'] > 25) | (ds['city'] == 'NYC')]    # OR
ds[~(ds['status'] == 'inactive')]               # NOT
```

<div id="f-class">
  ### F (함수)
</div>

ClickHouse 함수 네임스페이스입니다.

```python
from chdb.datastore import F, Field

# 집계
F.sum(Field('amount'))
F.avg(Field('price'))
F.count(Field('id'))
F.quantile(Field('value'), 0.95)

# 조건부
F.sum_if(Field('amount'), Field('status') == 'completed')
F.count_if(Field('active'))

# 윈도우
F.row_number().over(order_by='date')
F.lag('price', 1).over(partition_by='product', order_by='date')
```

자세한 내용은 [집계](/ko/products/chdb/datastore/aggregation#f-namespace)를 확인하십시오.

<div id="field-class">
  ### 필드
</div>

컬럼을 이름으로 참조합니다.

```python
from chdb.datastore import Field

# 필드 참조 생성
amount = Field('amount')
price = Field('price')

# 표현식에서 사용
F.sum(Field('amount'))
F.avg(Field('price'))
```

<div id="casewhen-class">
  ### CaseWhen
</div>

CASE WHEN 표현식을 생성하는 빌더입니다.

```python
# case-when 표현식 생성
result = (ds
    .when(ds['score'] >= 90, 'A')
    .when(ds['score'] >= 80, 'B')
    .when(ds['score'] >= 70, 'C')
    .otherwise('F')
)

# 컬럼에 할당
ds['grade'] = result
```

<div id="window-class">
  ### 윈도우
</div>

윈도우 함수의 윈도우 사양입니다.

```python
from chdb.datastore import F

# 윈도우 생성
window = F.window(
    partition_by='category',
    order_by='date',
    rows_between=(-7, 0)
)

# 집계에 사용
ds['rolling_avg'] = F.avg('price').over(window)
```
