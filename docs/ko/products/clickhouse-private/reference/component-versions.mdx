<div id="notes">
  ## 참고 사항
</div>

* **operator Helm 차트 버전**(`1.18587.1`)은 operator **이미지 태그**(`main-1.18587.1`)와 다릅니다. `helm install`로 설치할 때는 Helm 차트 버전을 사용하고, 컨테이너 이미지를 복사할 때는 이미지 태그를 사용하세요.
* **FIPS 빌드**는 server와 Keeper의 표준 이미지 태그에 모두 `-fips`를 추가합니다.
* 이 문서가 빌드될 때 버전 태그는 자동으로 치환됩니다. 원시 템플릿 변수(예: `25.12.1.1497`)가 보인다면, 빌드 단계가 아직 실행되지 않은 것입니다.
