관리형 ClickStack과 오픈 소스 ClickStack은 관측성 데이터를 ClickHouse 인스턴스로 수집하는 여러 방법을 제공합니다. 로그, 메트릭, 트레이스 또는 세션 데이터를 수집할 때 OpenTelemetry (OTel) collector를 단일 수집 지점으로 사용할 수 있으며, 특정 사용 사례에 맞는 플랫폼별 통합을 활용할 수도 있습니다.
