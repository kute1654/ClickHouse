ClickHouse는 연결에 사용할 수 있는 다양한 네이티브 클라이언트와 인터페이스를 제공합니다.

자세한 내용은 아래 페이지를 참조하십시오:

| 섹션                                                         | 요약                                                        |
| ---------------------------------------------------------- | --------------------------------------------------------- |
| [clickhousectl](/ko/concepts/features/interfaces/cli)         | 로컬 및 Cloud용 ClickHouse CLI입니다.                            |
| [ClickHouse Client](/ko/concepts/features/interfaces/client)  | command-line 옵션과 설정 파일을 지원하는 네이티브 command-line client입니다. |
| [드라이버 및 인터페이스](/ko/concepts/features/interfaces/overview)     | 다양한 네트워크 인터페이스, 라이브러리 및 시각적 인터페이스를 다룹니다.                  |
| [SQL 콘솔](/ko/integrations/connectors/sql-clients/sql-console) | ClickHouse Cloud에서 데이터를 빠르고 쉽게 다룰 수 있는 방법입니다.             |
