<div id="december-18-2023">
  ## 2023년 12월 18일
</div>

이번 릴리스에는 GCP의 새 리전(us-east1), 보안 엔드포인트 연결을 셀프서비스로 구성하는 기능, DBT 1.7을 포함한 추가 통합 지원, 그리고 다수의 버그 수정과 보안 개선 사항이 포함됩니다.

<div id="general-changes-1">
  ### 일반 변경 사항
</div>

* 이제 ClickHouse Cloud를 GCP us-east1(사우스캐롤라이나) 리전에서도 사용할 수 있습니다
* 이제 OpenAPI를 통해 AWS Private Link와 GCP Private Service Connect를 설정할 수 있게 되었습니다

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* Developer 역할이 있으면 SQL 콘솔에 원활하게 로그인할 수 있도록 했습니다
* 온보딩 중 유휴 상태 전환 제어를 설정하는 절차를 간소화했습니다

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* DBT 커넥터: DBT v1.7까지 지원 추가
* Metabase: Metabase v0.48 지원 추가
* PowerBI 커넥터: PowerBI Cloud에서 실행 지원 추가
* ClickPipes 내부 사용자의 권한을 구성할 수 있도록 변경
* Kafka Connector
  * 중복 제거 로직과 널 허용 타입 수집 개선
  * 텍스트 기반 포맷(CSV, TSV) 지원 추가
* Apache Beam: Boolean 및 LowCardinality 타입 지원 추가
* Nodejs 클라이언트: Parquet 포맷 지원 추가

<div id="security-announcements">
  ### 보안 공지
</div>

* 3개의 보안 취약점을 수정했습니다. 자세한 내용은 [보안 changelog](/ko/resources/changelogs/security-changelog)를 참조하십시오:
  * CVE 2023-47118 (CVSS 7.0) - 기본적으로 9000/tcp 포트에서 실행되는 네이티브 인터페이스에 영향을 미치는 힙 버퍼 오버플로 취약점
  * CVE-2023-48704 (CVSS 7.0) - 기본적으로 9000/tcp 포트에서 실행되는 네이티브 인터페이스에 영향을 미치는 힙 버퍼 오버플로 취약점
  * CVE 2023-48298 (CVSS 5.9) - FPC 압축 코덱의 정수 언더플로우 취약점

<div id="november-22-2023">
  ## 2023년 11월 22일
</div>

이번 릴리스에서는 핵심 데이터베이스 버전을 업그레이드하고, 로그인 및 인증 흐름을 개선했으며, Kafka Connect Sink에 프록시 지원을 추가했습니다.

<div id="clickhouse-version-upgrade-2">
  ### ClickHouse 버전 업그레이드
</div>

* Parquet 파일 읽기 성능이 크게 향상되었습니다. 자세한 내용은 [23.8 release blog](https://clickhouse.com/blog/clickhouse-release-23-08)를 참조하십시오.
* JSON에 대한 유형 추론 지원이 추가되었습니다. 자세한 내용은 [23.9 release blog](https://clickhouse.com/blog/clickhouse-release-23-09)를 참조하십시오.
* `ArrayFold`와 같은 강력한 분석가용 함수가 도입되었습니다. 자세한 내용은 [23.10 release blog](https://clickhouse.com/blog/clickhouse-release-23-10)를 참조하십시오.
* **사용자에게 영향을 주는 하위 호환되지 않는 변경 사항**: JSON 포맷에서 문자열로부터 숫자를 추론하지 않도록 `input_format_json_try_infer_numbers_from_strings` 설정이 기본적으로 비활성화되었습니다. 이는 샘플 데이터에 숫자처럼 보이는 문자열이 포함된 경우 발생할 수 있는 구문 분석 오류를 방지하기 위한 것입니다.
* 수십 개의 새 기능, 성능 개선, 버그 수정이 포함되었습니다. 자세한 내용은 [핵심 데이터베이스 변경 로그](/ko/resources/changelogs/oss/2026)를 참조하십시오.

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* 로그인 및 인증 흐름이 개선되었습니다.
* 대규모 스키마를 보다 잘 지원하도록 AI 기반 쿼리 제안이 개선되었습니다.

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* Kafka Connect Sink: 프록시 지원, `topic-tablename` 매핑, 그리고 Keeper *exactly-once* 전달 속성 설정 기능이 추가되었습니다.
* Node.js 클라이언트: Parquet 포맷 지원이 추가되었습니다.
* Metabase: `datetimeDiff` 함수 지원이 추가되었습니다.
* Python 클라이언트: 컬럼 이름의 특수 문자 지원이 추가되었습니다. 시간대 매개변수 바인딩 문제가 수정되었습니다.

<div id="november-2-2023">
  ## 2023년 11월 2일
</div>

이번 릴리스에서는 아시아 지역 개발 서비스에 대한 Region 지원을 확대하고, 고객 관리형 암호화 키에 키 순환 기능을 도입했으며, 청구 콘솔의 세금 설정 세분화 수준을 개선하고, 지원되는 언어 클라이언트 전반에 걸쳐 다수의 버그 수정을 적용했습니다.

<div id="general-updates-1">
  ### 일반 업데이트
</div>

* 이제 AWS의 `ap-south-1`(뭄바이) 및 `ap-southeast-1`(싱가포르) 리전에서 개발 서비스를 이용할 수 있습니다
* 고객 관리형 암호화 키(CMEK)에 대한 키 순환 지원이 추가되었습니다

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* 신용카드를 추가할 때 세부적인 세금 설정을 구성할 수 있는 기능이 추가되었습니다

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* MySQL
  * MySQL을 통한 Tableau Online 및 QuickSight 지원 개선
* Kafka Connector
  * 텍스트 기반 포맷(CSV, TSV) 지원을 위해 새로운 StringConverter 도입
  * Bytes 및 Decimal 데이터 타입 지원 추가
  * 이제 Retryable Exceptions가 항상 재시도되도록 조정(`errors.tolerance=all`인 경우 포함)
* Node.js client
  * 스트리밍되는 대용량 데이터셋에서 결과가 손상되던 문제 수정
* Python 클라이언트
  * 대용량 삽입 시 timeout 문제 수정
  * NumPy/Pandas Date32 문제 수정
    ​​- Golang 클라이언트
  * JSON 컬럼에 빈 맵을 삽입할 때의 문제, 압축 버퍼 정리, 쿼리 이스케이프 처리, IPv4 및 IPv6에서 zero/nil일 때 발생하는 panic 수정
  * 취소된 삽입에 대한 watchdog 추가
* DBT
  * 테스트와 함께 분산 테이블 지원 개선

<div id="september-28-2023">
  ## 2023년 09월 28일
</div>

이번 릴리스에서는 ClickPipes for Kafka, Confluent Cloud, Amazon MSK 및 Kafka Connect ClickHouse Sink의 정식 출시와, IAM 역할을 사용해 Amazon S3 액세스를 안전하게 보호하는 셀프서비스 워크플로, 그리고 AI 기반 쿼리 제안(비공개 프리뷰)을 제공합니다.

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* [IAM 역할을 통해 Amazon S3에 안전하게 액세스](/ko/products/cloud/guides/data-sources/accessing-s3-data-securely)할 수 있도록 셀프 서비스 워크플로를 추가했습니다
* AI 기반 쿼리 제안을 비공개 프리뷰로 도입했습니다(사용해 보려면 [ClickHouse Cloud 지원팀에 문의하십시오](https://console.clickhouse.cloud/support).)

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* Kafka, Confluent Cloud, Amazon MSK용 ClickPipes(턴키 데이터 수집 서비스)의 정식 출시를 발표했습니다([릴리스 블로그](https://clickhouse.com/blog/clickpipes-is-generally-available) 참조).
* Kafka Connect ClickHouse 싱크가 정식 출시되었습니다.
  * `clickhouse.settings` 속성을 사용한 사용자 지정 ClickHouse 설정 지원을 확장했습니다.
  * 동적 필드를 반영하도록 중복 제거 동작을 개선했습니다.
  * ClickHouse에서 테이블 변경 사항을 다시 가져오도록 `tableRefreshInterval` 지원을 추가했습니다.
* [PowerBI](/ko/integrations/connectors/data-visualization/powerbi-and-clickhouse)와 ClickHouse 데이터 타입 간의 SSL 연결 문제와 타입 매핑 문제를 수정했습니다.

* Node.js client: default&#95;format 설정 지원 추가
* Golang client: bool 유형 처리 수정, string 제한 제거

<div id="aug-24-2023">
  ## 2023년 08월 24일
</div>

이번 릴리스에서는 ClickHouse 데이터베이스에 MySQL 인터페이스 지원이 추가되고, 새로운 공식 PowerBI 커넥터가 도입되며, Cloud Console에 새로운 &quot;실행 중인 쿼리&quot; 뷰가 추가되고, ClickHouse 버전이 23.7로 업데이트됩니다.

<div id="general-updates-1">
  ### 일반 업데이트
</div>

* [MySQL wire protocol](/ko/concepts/features/interfaces/mysql) 지원이 추가되어, 다양한 사용 사례와 함께 기존 BI 도구 다수와의 호환성이 향상되었습니다. 조직에서 이 기능을 활성화하려면 지원팀에 문의하십시오.
* 새로운 공식 PowerBI 커넥터가 추가되었습니다

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* SQL 콘솔에 &quot;실행 중인 쿼리&quot; 보기 지원 추가

<div id="clickhouse-237-version-upgrade">
  ### ClickHouse 23.7 버전 업그레이드
</div>

* Azure 테이블 함수 지원을 추가하고, Geo 데이터 타입을 프로덕션 환경에서 사용할 수 있는 수준으로 높였으며, JOIN 성능을 개선했습니다. 자세한 내용은 23.5 릴리스 [블로그](https://clickhouse.com/blog/clickhouse-release-23-05)를 참조하세요.
* MongoDB 통합 지원을 6.0 버전까지 확장했습니다. 자세한 내용은 23.6 릴리스 [블로그](https://clickhouse.com/blog/clickhouse-release-23-06)를 참조하세요.
* Parquet 포맷에 대한 쓰기 성능을 6배 향상시키고, PRQL 쿼리 언어 지원을 추가했으며, SQL 호환성을 개선했습니다. 자세한 내용은 23.7 릴리스 [발표 자료](https://presentations.clickhouse.com/release_23.7/)를 참조하세요.
* 수십 개의 새 기능, 성능 개선, 버그 수정이 포함되었습니다. 23.5, 23.6, 23.7의 자세한 내용은 [변경 로그](/ko/resources/changelogs/oss/2026)를 참조하세요.

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* Kafka Connector: Avro Date 및 Time 타입 지원 추가
* JavaScript client: 웹 기반 환경용 안정 버전 출시
* Grafana: 필터 로직과 데이터베이스 이름 처리 방식 개선, 초 미만 정밀도의 TimeInteval 지원 추가
* Golang 클라이언트: 여러 Batch 및 비동기 데이터 로딩 문제 수정
* Metabase: v0.47 지원, 연결 가장 기능 추가, 데이터 타입 매핑 수정

<div id="july-27-2023">
  ## 2023년 7월 27일
</div>

이번 릴리스에서는 ClickPipes for Kafka의 비공개 프리뷰, 새로운 데이터 로딩 방식, 그리고 Cloud Console을 사용해 URL에서 파일을 로드할 수 있는 기능을 제공합니다.

<div id="integrations-changes-7">
  ### 통합 변경 사항
</div>

* Kafka용 [ClickPipes](https://clickhouse.com/cloud/clickpipes)의 비공개 프리뷰를 시작했습니다. ClickPipes는 Kafka 및 Confluent Cloud에서 대규모 데이터를 몇 번의 클릭만으로 손쉽게 수집할 수 있게 해주는 클라우드 네이티브 integration 엔진입니다. 대기자 명단은 [여기](https://clickhouse.com/cloud/clickpipes#joinwaitlist)에서 등록하십시오.
* JavaScript client: 웹 기반 환경(브라우저, Cloudflare workers) 지원이 추가되었습니다. 커뮤니티에서 사용자 지정 환경용 커넥터를 만들 수 있도록 코드를 리팩터링했습니다.
* Kafka Connector: Timestamp 및 Time Kafka 타입을 위한 인라인 스키마 지원이 추가되었습니다.
* Python client: 삽입 압축 및 LowCardinality 읽기 문제를 수정했습니다.

<div id="console-changes-7">
  ### Console 변경 사항
</div>

* 테이블 생성 구성 옵션이 더 다양해진 새로운 데이터 로드 환경을 추가했습니다
* Cloud Console을 사용해 URL에서 파일을 로드할 수 있는 기능을 도입했습니다
* 다른 조직에 가입하고 미응답 상태인 모든 초대를 확인할 수 있도록 추가 옵션을 제공해 초대 흐름을 개선했습니다

<div id="july-14-2023">

<div id="general-updates-1">
