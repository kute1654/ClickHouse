| Herramienta/Utilidad                                                                        | Descripción                                                                                                              |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| [clickhouse-local](/es/concepts/features/tools-and-utilities/clickhouse-local)                 | Permite ejecutar consultas SQL sobre datos sin iniciar el servidor de ClickHouse, de forma similar a como lo hace `awk`. |
| [clickhouse-benchmark](/es/concepts/features/tools-and-utilities/clickhouse-benchmark)         | Carga el servidor con consultas y configuraciones personalizadas.                                                        |
| [clickhouse-format](/es/concepts/features/tools-and-utilities/clickhouse-format)               | Permite dar formato a las consultas de entrada.                                                                          |
| [ClickHouse obfuscator](/es/concepts/features/tools-and-utilities/clickhouse-obfuscator)       | Ofusca los datos.                                                                                                        |
| [ClickHouse compressor](/es/concepts/features/tools-and-utilities/clickhouse-compressor)       | Comprime y descomprime datos.                                                                                            |
| [clickhouse-disks](/es/concepts/features/tools-and-utilities/clickhouse-disks)                 | Proporciona operaciones similares a las de un sistema de archivos sobre archivos en distintos discos de ClickHouse.      |
| [clickhouse-odbc-bridge](/es/concepts/features/tools-and-utilities/odbc-bridge)                | Un servidor proxy para el controlador ODBC.                                                                              |
| [clickhouse&#95;backupview](/es/concepts/features/tools-and-utilities/backupview)              | Un módulo de Python para analizar copias de seguridad de ClickHouse.                                                     |
| [clickhouse-keeper-client](/es/concepts/features/tools-and-utilities/clickhouse-keeper-client) | Una aplicación cliente para interactuar con ClickHouse Keeper.                                                           |
| [Keeper HTTP API](/es/concepts/features/tools-and-utilities/clickhouse-keeper-http-api)        | API HTTP y dashboard integrado para ClickHouse Keeper.                                                                   |
