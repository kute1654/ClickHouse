Esta sección ofrece varios conjuntos de datos y ejemplos para ayudarle a empezar a usar ClickStack. Estos ejemplos muestran distintas formas de trabajar con datos de observabilidad en ClickStack, desde el desarrollo local hasta entornos de producción.

| Conjunto de datos                                                              | Descripción                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Datos de ejemplo](/es/clickstack/example-datasets/sample-data)                   | Cargue un conjunto de datos de ejemplo que contiene logs, trazas y métricas de nuestro entorno de demostración                                                                                                                              |
| [Datos locales](/es/clickstack/example-datasets/local-data)                       | Recopile métricas y logs del sistema local y envíelos a ClickStack para analizarlos                                                                                                                                                         |
| [Datos remotos de demostración](/es/clickstack/example-datasets/remote-demo-data) | Conéctese a nuestro clúster remoto de demostración y explore un problema                                                                                                                                                                    |
| [Session Replay Demo](/es/clickstack/example-datasets/session-replay)             | Instrumente una aplicación web de demostración para session replay y vea sus interacciones en ClickStack                                                                                                                                    |
| [Chrome Extension](/es/clickstack/example-datasets/chrome-extension)              | Inyecte el SDK del navegador en cualquier sitio web con la HyperDX Chrome extension, sin necesidad de modificar el código de la aplicación                                                                                                  |
| [Datos sintéticos con otelgen](/es/clickstack/example-datasets/otelgen)           | Use `otelgen` para enviar logs, trazas y métricas sintéticos a un ClickStack OpenTelemetry collector en ejecución                                                                                                                           |
| [Datos sintéticos con telemetrygen](/es/clickstack/example-datasets/telemetrygen) | Use `telemetrygen` para enviar logs, trazas y métricas sintéticos variados, definidos mediante indicadores entre servicios, niveles de severidad, estados de span y tipos de métricas, a un ClickStack OpenTelemetry collector en ejecución |
| [HackerNews Analyzer](/es/clickstack/example-datasets/instrument-application)     | Instrumente HackerNews Analyzer, una aplicación Node.js, con OpenTelemetry y envíe sus logs, métricas y trazas a Managed ClickStack                                                                                                         |
