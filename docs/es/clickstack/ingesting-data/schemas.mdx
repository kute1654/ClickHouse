El OTel collector de ClickStack usa el [exporter de ClickHouse](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/exporter/clickhouseexporter/README.md) para crear tablas en ClickHouse e insertar datos.

Las siguientes tablas se crean para cada tipo de dato en la base de datos `default`. Puede cambiar esta base de datos de destino modificando la variable de entorno `HYPERDX_OTEL_EXPORTER_CLICKHOUSE_DATABASE` en la imagen que ejecuta el OTel collector.
