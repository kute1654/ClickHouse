ClickStack puede desplegarse de varias formas, según sus preferencias y requisitos operativos:

* [Managed ClickStack](/es/clickstack/getting-started/managed) - La forma **más fácil y simple** de empezar: desplegar **Managed ClickStack** en **ClickHouse Cloud**.
* [ClickStack Open-Source](/es/clickstack/getting-started/oss) - Para quienes deseen ejecutar por su cuenta una arquitectura completamente open source.
