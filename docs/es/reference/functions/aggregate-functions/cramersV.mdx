<div id="cramersV">
  ## cramersV
</div>

Introducido en: v22.1.0

[V de Cramer](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V) (a veces tambiГ©n denominada phi de Cramer) es una medida de asociaciГіn entre dos columnas de una tabla.
El resultado de la funciГіn `cramersV` varГӯa entre 0 (que corresponde a la ausencia de asociaciГіn entre las variables) y 1, y solo puede alcanzar 1 cuando cada valor estГЎ completamente determinado por el otro.
Puede interpretarse como la asociaciГіn entre dos variables expresada como un porcentaje de su variaciГіn mГЎxima posible.

<Note>
  Para una versiГіn con correcciГіn de sesgo de la V de Cramer, consulte: [cramersVBiasCorrected](/es/reference/functions/aggregate-functions/cramersVBiasCorrected)
</Note>

**Sintaxis**

```sql

```

**Argumentos**

* `column1` вҖ” Primera columna que se compararГЎ. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)
* `column2` вҖ” Segunda columna que se compararГЎ. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve un valor entre 0 (que corresponde a la ausencia de asociaciГіn entre los valores de las columnas) y 1 (asociaciГіn completa). [`Float64`](/es/reference/data-types/float)
