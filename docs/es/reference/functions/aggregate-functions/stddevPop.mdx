<div id="stddevPop">
  ## stddevPop
</div>

Introducido en: v1.1.0

Devuelve la desviación estándar poblacional de una secuencia de datos numéricos.
El resultado es igual a la raíz cuadrada de [`varPop`](/es/reference/functions/aggregate-functions/varPop).

<Note>
  Esta función utiliza un algoritmo numéricamente inestable. Si necesitas [estabilidad numérica](https://en.wikipedia.org/wiki/Numerical_stability) en los cálculos, usa la función [`stddevPopStable`](/es/reference/functions/aggregate-functions/stddevPopStable). Funciona más lentamente, pero proporciona un menor error de cálculo.
</Note>

**Sintaxis**

```sql
stddevPop(x)
```

**Alias**: `STD`, `STDDEV_POP`

**Argumentos**

* `x` — Conjunto de valores cuya desviación estándar poblacional se desea calcular. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la raíz cuadrada de la varianza poblacional de `x`. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Cálculo de la desviación estándar poblacional**

```sql title=Query
CREATE TABLE test_data (population UInt8) ENGINE = Log;
INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT stddevPop(population) AS stddev FROM test_data;
```

```response title=Response
┌────────────stddev─┐
│ 3.794733192202055 │
└───────────────────┘
```
