<div id="anyLast">
  ## anyLast
</div>

Introducido en: v1.1.0

Selecciona el último valor encontrado en una columna.

<Warning>
  Dado que una consulta puede ejecutarse en un orden arbitrario, el resultado de esta función no es determinista.
  Si necesita un resultado arbitrario pero determinista, use las funciones [min](/es/reference/functions/aggregate-functions/min) o [max](/es/reference/functions/aggregate-functions/max).
</Warning>

De forma predeterminada, la función nunca devuelve NULL; es decir, ignora los valores NULL de la columna de entrada.
Sin embargo, si la función se usa con el modificador `RESPECT NULLS`, devuelve el último valor leído, independientemente de si es NULL o no.

**Sintaxis**

```sql
anyLast(column) [RESPECT NULLS]
```

**Alias**: `last_value`

**Argumentos**

* `column` — Nombre de la columna. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve el último valor encontrado. [`Any`](/es/reference/data-types/index)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
CREATE TABLE tab(city Nullable(String)) ENGINE=Memory;
INSERT INTO tab (city) VALUES ('Amsterdam'), (NULL), ('New York'), ('Tokyo'), ('Valencia'), (NULL);
SELECT anyLast(city), anyLastRespectNulls(city) FROM tab;
```

```response title=Response
┌─anyLast(city)─┬─anyLastRespectNulls(city)─┐
│ Valencia      │ ᴺᵁᴸᴸ                      │
└───────────────┴───────────────────────────┘
```
