<div id="anyHeavy">
  ## anyHeavy
</div>

Introducido en: v1.1.0

Selecciona un valor frecuente mediante el algoritmo de [heavy hitters](https://doi.org/10.1145/762471.762473).
Si hay un valor que aparece en más de la mitad de los casos en cada uno de los hilos de ejecución de la consulta, se devuelve ese valor.
Normalmente, el resultado es no determinista.

**Sintaxis**

```sql
anyHeavy(column)
```

**Argumentos**

* `column` — El nombre de la columna. [`String`](/es/reference/data-types/string)

**Valor devuelto**

Devuelve un valor frecuente. El resultado es no determinista. [`Any`](/es/reference/data-types/index)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
