<div id="argMin">
  ## argMin
</div>

Introducido en: v1.1.0

Calcula el valor de `arg` para el valor mГӯnimo de `val`. Si hay varias filas con el mismo `val` y este es el mГЎximo, no es determinista cuГЎl de los `arg` asociados se devuelve.
Tanto `arg` como `min` se comportan como [funciones de agregado](/es/reference/functions/aggregate-functions/index); ambos [omiten `Null`](/es/reference/functions/aggregate-functions/index#null-processing) durante el procesamiento y devuelven valores distintos de `Null` si hay valores distintos de `Null` disponibles.

**VГ©ase tambiГ©n**

* [Tuple](/es/reference/data-types/tuple)

**Sintaxis**

```sql

```

**Alias**: `min_by`

**Argumentos**

* `arg` вҖ” Argumento para el que se busca el valor mГЎximo. [`const String`](/es/reference/data-types/string)
* `val` вҖ” El valor mГӯnimo. [`(U)Int8/16/32/64`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Date`](/es/reference/data-types/date) o [`DateTime`](/es/reference/data-types/datetime) o [`Tuple`](/es/reference/data-types/tuple)

**Valor devuelto**

Devuelve el valor de `arg` que corresponde al valor mГӯnimo de `val`. El tipo coincide con el de `arg`.
