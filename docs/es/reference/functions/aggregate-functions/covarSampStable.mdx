## covarSampStable {#covarSampStable}

Introducido en: v1.1.0

Calcula la covarianza muestral:

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{n - 1}
$$

<br/>

Es similar a [`covarSamp`](/reference/functions/aggregate-functions/covarSamp) pero utiliza un algoritmo numéricamente estable.
Como resultado, `covarSampStable` es más lento que `covarSamp` pero ofrece un menor error computacional.
    

**Sintaxis**

```sql
covarSampStable(x, y)
```

**Argumentos**

- `x` — Primera variable. [`(U)Int*`](/reference/data-types/int-uint) o [`Float*`](/reference/data-types/float) o [`Decimal`](/reference/data-types/decimal)
- `y` — Segunda variable. [`(U)Int*`](/reference/data-types/int-uint) o [`Float*`](/reference/data-types/float) o [`Decimal`](/reference/data-types/decimal)

**Valor devuelto**

Devuelve la covarianza muestral entre `x` e `y`. Para `n <= 1`, se devuelve `inf`. [`Float64`](/reference/data-types/float)

**Ejemplos**

**Cálculo básico de covarianza muestral con algoritmo estable**

```sql title=Query
DROP TABLE IF EXISTS series;
CREATE TABLE series(i UInt32, x_value Float64, y_value Float64) ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6,-4.4),(2, -9.6,3),(3, -1.3,-4),(4, 5.3,9.7),(5, 4.4,0.037),(6, -8.6,-7.8),(7, 5.1,9.3),(8, 7.9,-3.6),(9, -8.2,0.62),(10, -3,7.3);

SELECT covarSampStable(x_value, y_value)
FROM
(
    SELECT
        x_value,
        y_value
    FROM series
);
```

```response title=Response
┌─covarSampStable(x_value, y_value)─┐
│                 7.206275555555556 │
└───────────────────────────────────┘
```

**Un único valor devuelve inf**

```sql title=Query
SELECT covarSampStable(x_value, y_value)
FROM
(
    SELECT
        x_value,
        y_value
    FROM series LIMIT 1
);
```

```response title=Response
┌─covarSampStable(x_value, y_value)─┐
│                               inf │
└───────────────────────────────────┘
```
