<div id="theilsU">
  ## theilsU
</div>

Introducido en: v22.1.0

La función `theilsU` calcula el [coeficiente de incertidumbre U de Theil](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient), un valor que mide la asociación entre dos columnas de una tabla.
Sus valores oscilan entre 0.0 (sin asociación) y 1.0 (acuerdo perfecto).

**Sintaxis**

```sql
theilsU(column1, column2)
```

**Argumentos**

* `column1` — Primera columna que se comparará. [`Any`](/es/reference/data-types/index)
* `column2` — Segunda columna que se comparará. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve un valor entre 0 y 1. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
