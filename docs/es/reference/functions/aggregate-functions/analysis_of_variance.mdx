<div id="analysisOfVariance">
  ## analysisOfVariance
</div>

Introducido en: v22.10.0

Proporciona una prueba estadística para el análisis de varianza de un factor (prueba ANOVA). Se trata de una prueba aplicada a varios grupos de observaciones con distribución normal para determinar si todos los grupos tienen la misma media o no.

<Note>
  Los grupos se enumeran a partir de 0 y debe haber al menos dos grupos para realizar la prueba.
  Debe haber al menos un grupo con más de una observación.
</Note>

**Sintaxis**

```sql
analysisOfVariance(val, group_no)
```

**Alias**: `anova`

**Argumentos**

* `val` — Valor. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)
* `group_no` — Número de grupo al que pertenece `val`. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve una tupla con el estadístico F y el valor p. [`Tuple(Float64, Float64)`](/es/reference/data-types/tuple)

**Ejemplos**

**Uso básico**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

