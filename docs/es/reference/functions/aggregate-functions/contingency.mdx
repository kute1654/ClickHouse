<div id="contingency">
  ## contingency
</div>

Introducido en: v22.1.0

La función `contingency` calcula el [coeficiente de contingencia](https://en.wikipedia.org/wiki/Contingency_table#Cram%C3%A9r's_V_and_the_contingency_coefficient_C), un valor que mide la asociación entre dos columnas de una tabla.
El cálculo es similar al de la función [`cramersV`](/es/reference/functions/aggregate-functions/cramersV), pero utiliza un denominador distinto en la raíz cuadrada.

**Sintaxis**

```sql
contingency(column1, column2)
```

**Argumentos**

* `column1` — Primera columna a comparar. [`Any`](/es/reference/data-types/index)
* `column2` — Segunda columna a comparar. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve un valor entre 0 y 1. Cuanto mayor sea el resultado, más estrecha será la asociación entre las dos columnas. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Comparación con cramersV**

```sql title=Query
SELECT
    cramersV(a, b),
    contingency(a, b)
FROM
(
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌─────cramersV(a, b)─┬──contingency(a, b)─┐
│ 0.5798088336225178 │ 0.708607540104077  │
└────────────────────┴────────────────────┘
```
