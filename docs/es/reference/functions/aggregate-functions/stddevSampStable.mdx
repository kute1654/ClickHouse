<div id="stddevSampStable">
  ## stddevSampStable
</div>

Introducido en: v1.1.0

El resultado es igual a la raíz cuadrada de [varSamp](/es/reference/functions/aggregate-functions/varSamp). A diferencia de [stddevSamp](/es/reference/functions/aggregate-functions/stddevSamp), esta función utiliza un algoritmo numéricamente estable. Es más lenta, pero proporciona un menor error computacional.

**Sintaxis**

```sql
stddevSampStable(x)
```

**Argumentos**

* `x` — Valores para los que se calcula la raíz cuadrada de la varianza muestral. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la raíz cuadrada de la varianza muestral de `x`. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Uso básico**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSampStable(population)
FROM test_data;
```

```response title=Response
┌─stddevSampStable(population)─┐
│                            4 │
└──────────────────────────────┘
```
