<div id="maxMappedArrays">
  ## maxMappedArrays
</div>

Introducido en: v20.5.0

Calcula el valor máximo del array `value` según las claves especificadas en el array `key`.

<Note>
  * Pasar una tupla de claves y arrays de valores es idéntico a pasar un array de claves y un array de valores.
  * El número de elementos de `key` y `value` debe ser el mismo en cada fila que se totalice.
</Note>

**Sintaxis**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**Argumentos**

* `key` — Array de claves. [`Array(T)`](/es/reference/data-types/array)
* `value` — Array de valores. [`Array(T)`](/es/reference/data-types/array)

**Valor devuelto**

Devuelve una tupla de dos arrays: las claves ordenadas y los valores calculados para las claves correspondientes. [`Tuple(Array(T), Array(T))`](/es/reference/data-types/tuple)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)───┐
│ (['x','y','z'],[2,3,1]) │
└─────────────────────────┘
```
