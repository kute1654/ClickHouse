<div id="boundingRatio">
  ## boundingRatio
</div>

Introducido en: v20.1.0

Calcula la pendiente entre el punto más a la izquierda y el punto más a la derecha de un grupo de valores.

**Sintaxis**

```sql
boundingRatio(x, y)
```

**Argumentos**

* `x` — Valores de la coordenada X. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)
* `y` — Valores de la coordenada Y. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la pendiente de la recta entre los puntos situados más a la izquierda y más a la derecha; en caso contrario, devuelve `NaN` si los datos están vacíos. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Datos de ejemplo**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**Ejemplo de uso**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
