<div id="groupBitAnd">
  ## groupBitAnd
</div>

Introducido en: v1.1.0

Realiza la operación AND bit a bit sobre una serie de números.

**Sintaxis**

```sql
groupBitAnd(expr)
```

**Alias**: `BIT_AND`

**Argumentos**

* `expr` — Expresión de tipo `(U)Int*`. [`(U)Int*`](/es/reference/data-types/int-uint)

**Valor devuelto**

Devuelve un valor de tipo `(U)Int*`. [`(U)Int*`](/es/reference/data-types/int-uint)

**Ejemplos**

**Ejemplo de AND a nivel de bits**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Datos de prueba:
-- binario     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Resultado:
-- binario     decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
