<div id="studentTTestOneSample">
  ## studentTTestOneSample
</div>

Introducido en: v25.10.0

Aplica la prueba t de Student para una muestra para determinar si la media de una muestra difiere de una media poblacional conocida.

Se asume normalidad. La hipótesis nula es que la media muestral es igual a la media poblacional.

El parámetro opcional `confidence_level` permite calcular el intervalo de confianza.

**Notas:**

* Se requieren al menos 2 observaciones; de lo contrario, el resultado es `(nan, nan)` (y los intervalos, si se solicitan, son `nan`).
* Una entrada constante o casi constante también devolverá `nan` debido a un error estándar nulo (o prácticamente nulo).

**Véase también**

* [Prueba t de Student](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [función studentTTest](/es/reference/functions/aggregate-functions/studentTTest)

**Sintaxis**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**Parámetros**

* `confidence_level` — Opcional. Nivel de confianza para los intervalos de confianza. `Float` en (0, 1). [`Float*`](/es/reference/data-types/float)

**Argumentos**

* `sample_data` — Datos de la muestra. [`Integer`](/es/reference/data-types/int-uint) o [`Float`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)
* `population_mean` — Media poblacional conocida con la que contrastar (normalmente, una constante). [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve una tupla con dos o cuatro elementos (si se especifica `confidence_level`): el estadístico t calculado, el valor p calculado (de dos colas), [límite inferior calculado del intervalo de confianza], [límite superior calculado del intervalo de confianza]. Los intervalos de confianza corresponden a la media muestral en el nivel de confianza indicado. [`Tuple(Float64, Float64)`](/es/reference/data-types/tuple) o [`Tuple(Float64, Float64, Float64, Float64)`](/es/reference/data-types/tuple)

**Ejemplos**

**Sin intervalo de confianza**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
