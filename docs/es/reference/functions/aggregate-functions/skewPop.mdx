<div id="skewPop">
  ## skewPop
</div>

Introducido en: v20.1.0

Calcula la [asimetría](https://en.wikipedia.org/wiki/Skewness) de una serie.

**Sintaxis**

```sql
skewPop(expr)
```

**Argumentos**

* `expr` — Una expresión que devuelve un número. [`Expression`](/es/reference/data-types/special-data-types/expression)

**Valor devuelto**

Devuelve la asimetría de la distribución indicada. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Distribución simétrica**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**Distribución asimétrica a la derecha**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
