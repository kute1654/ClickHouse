<div id="groupArrayLast">
  ## groupArrayLast
</div>

Introducido en: v23.1.0

Crea un array con los últimos valores del argumento.
Por ejemplo, `groupArrayLast(1)(x)` equivale a `[anyLast(x)]`.
En algunos casos, todavía se puede confiar en el orden de ejecución.
Esto se aplica cuando SELECT proviene de una subconsulta que usa ORDER BY, si el resultado de la subconsulta es lo suficientemente pequeño.

**Sintaxis**

```sql
groupArrayLast(max_size)(x)
```

**Parámetros**

* `max_size` — Tamaño máximo del array resultante. [`UInt64`](/es/reference/data-types/int-uint)

**Argumentos**

* `max_size` — Tamaño máximo del array resultante. [`UInt64`](/es/reference/data-types/int-uint)
* `x` — Argumento (nombre de columna o expresión). [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve un array con los últimos valores del argumento. [`Array(T)`](/es/reference/data-types/array)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**Comparación con groupArray**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
