<div id="sum">
  ## sum
</div>

Introducida en: v1.1.0

Calcula la suma de valores numéricos.

**Sintaxis**

```sql
sum(num)
```

**Argumentos**

* `num` — Columna de valores numéricos. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la suma de los valores. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Ejemplos**

**Cálculo de la suma de los salarios de los empleados**

```sql title=Query
CREATE TABLE employees
(
    id UInt32,
    name String,
    salary UInt32
)
ENGINE = Memory;

INSERT INTO employees VALUES
    (87432, 'John Smith', 45680),
    (59018, 'Jane Smith', 72350),
    (20376, 'Ivan Ivanovich', 58900),
    (71245, 'Anastasia Ivanovna', 89210);

SELECT sum(salary) FROM employees;
```

```response title=Response
┌─sum(salary)─┐
│      266140 │
└─────────────┘
```
