<div id="varPopStable">
  ## varPopStable
</div>

Introducido en: v1.1.0

Devuelve la varianza poblacional.
A diferencia de [`varPop`](/es/reference/functions/aggregate-functions/varPop), esta función utiliza un algoritmo [numéricamente estable](https://en.wikipedia.org/wiki/Numerical_stability).
Funciona más lentamente, pero ofrece un menor error de cálculo.

**Sintaxis**

```sql
varPopStable(x)
```

**Argumentos**

* `x` — Conjunto de valores del que se calculará la varianza poblacional. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la varianza poblacional de `x`. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Cálculo de la varianza poblacional estable**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

```response title=Response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
