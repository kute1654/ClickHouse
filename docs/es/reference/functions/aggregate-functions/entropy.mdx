<div id="entropy">
  ## entropy
</div>

Introducida en: v20.1.0

Calcula la [entropía de Shannon](https://en.wikipedia.org/wiki/Entropy_\(information_theory\)) de una columna de valores.

**Sintaxis**

```sql
entropy(val)
```

**Argumentos**

* `val` — Columna de valores de cualquier tipo. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve la entropía de Shannon. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Cálculo básico de la entropía**

```sql title=Query
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy
```

```response title=Response
┌─entropy(vals)─┬─entropy(strings)─┐
│             1 │             1.75 │
└───────────────┴──────────────────┘
```
