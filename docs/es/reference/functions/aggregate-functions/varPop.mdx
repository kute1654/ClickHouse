## varPop {#varPop}

Introducido en: v1.1.0

Calcula la varianza poblacional.

La varianza poblacional se calcula con la siguiente fórmula:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

Donde:
- $x$ es cada valor de la población
- $\bar{x}$ es la media poblacional
- $n$ es el tamaño de la población

<Note>
Esta función utiliza un algoritmo numéricamente inestable. Si necesita [estabilidad numérica](https://en.wikipedia.org/wiki/Numerical_stability) en los cálculos, use la función [`varPopStable`](/reference/functions/aggregate-functions/varPopStable). Es más lenta, pero ofrece un menor error computacional.
</Note>
    

**Sintaxis**

```sql
varPop(x)
```

**Alias**: `VAR_POP`

**Argumentos**

- `x` — Conjunto de valores sobre el que se calcula la varianza poblacional. [`(U)Int*`](/reference/data-types/int-uint) o [`Float*`](/reference/data-types/float) o [`Decimal*`](/reference/data-types/decimal)

**Valor devuelto**

Devuelve la varianza poblacional de `x`. [`Float64`](/reference/data-types/float)

**Ejemplos**

**Cálculo de la varianza poblacional**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
