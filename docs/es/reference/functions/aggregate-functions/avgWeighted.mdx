<div id="avgWeighted">
  ## avgWeighted
</div>

Introducido en: v20.1.0

Calcula la [media aritmética ponderada](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean).

**Sintaxis**

```sql
avgWeighted(x, weight)
```

**Argumentos**

* `x` — Valores. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float)
* `weight` — Pesos de los valores. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float)

**Valor devuelto**

Devuelve `NaN` si todos los pesos son iguales a 0 o si el parámetro de pesos proporcionado está vacío; de lo contrario, devuelve la media ponderada. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Pesos mixtos enteros y de punto flotante**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Si todos los pesos son cero, devuelve NaN**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**Una tabla vacía devuelve NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
