<div id="minMappedArrays">
  ## minMappedArrays
</div>

Introducido en: v20.5.0

Calcula el valor mínimo del array `value` según las claves especificadas en el array `key`.

<Note>
  * Pasar una tupla de claves y arrays de valores es equivalente a pasar un array de claves y un array de valores.
  * El número de elementos de `key` y `value` debe ser el mismo en cada fila totalizada.
</Note>

**Sintaxis**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**Argumentos**

* `key` — Array con las claves. [`Array(T)`](/es/reference/data-types/array)
* `value` — Array con los valores. [`Array(T)`](/es/reference/data-types/array)

**Valor devuelto**

Devuelve una tupla de dos arrays: las claves ordenadas y los valores calculados para las claves correspondientes. [`Tuple(Array(T), Array(T))`](/es/reference/data-types/tuple)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
