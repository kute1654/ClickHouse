<div id="groupUniqArray">
  ## groupUniqArray
</div>

Introducido en: v1.1.0

Crea un array a partir de distintos valores del argumento.
El consumo de memoria de esta función es el mismo que el de la función [`uniqExact`](/es/reference/functions/aggregate-functions/uniqExact).

**Sintaxis**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**Parámetros**

* `max_size` — Limita el tamaño del array resultante a `max_size` elementos. `groupUniqArray(1)(x)` equivale a `[any(x)]`. [`UInt64`](/es/reference/data-types/int-uint)

**Argumentos**

* `x` — Expresión. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve un array con valores únicos. [`Array`](/es/reference/data-types/array)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**Con el parámetro max&#95;size**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
