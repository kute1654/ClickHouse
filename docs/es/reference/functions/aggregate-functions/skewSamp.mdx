<div id="skewSamp">
  ## skewSamp
</div>

Introducido en: v20.1.0

Calcula la [asimetría muestral](https://en.wikipedia.org/wiki/Skewness) de una secuencia.

Representa una estimación insesgada de la asimetría de una variable aleatoria si los valores proporcionados constituyen una muestra de esta.

**Sintaxis**

```sql
skewSamp(expr)
```

**Argumentos**

* `expr` — Una expresión que devuelve un número. [`Expresión`](/es/reference/data-types/special-data-types/expression)

**Valor devuelto**

Devuelve la asimetría de la distribución dada. Si `n <= 1` (`n` es el tamaño de la muestra), la función devuelve `nan`. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Distribución simétrica**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**Distribución asimétrica a la derecha**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
