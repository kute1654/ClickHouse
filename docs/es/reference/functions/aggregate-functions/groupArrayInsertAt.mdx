<div id="groupArrayInsertAt">
  ## groupArrayInsertAt
</div>

Introducido en: v1.1.0

Inserta un valor en el array en la posiciГіn especificada.

Si en una consulta se insertan varios valores en la misma posiciГіn, la funciГіn se comporta del siguiente modo:

* Si una consulta se ejecuta en un solo hilo, se usa el primero de los valores insertados.
* Si una consulta se ejecuta en varios hilos, el valor resultante es uno indeterminado entre los valores insertados.

**Sintaxis**

```sql
groupArrayInsertAt(default_x, size)([x, pos])
```

**ParГЎmetros**

* `default_x` вҖ” Opcional. Valor predeterminado para sustituir las posiciones vacГӯas. [`Any`](/es/reference/data-types/index)
* `size` вҖ” Opcional. Longitud del array resultante. Al usar este parГЎmetro, se debe especificar el valor predeterminado `default_x`. [`UInt32`](/es/reference/data-types/int-uint)

**Argumentos**

* `x` вҖ” Valor que se va a insertar. [`Any`](/es/reference/data-types/index)
* `pos` вҖ” PosiciГіn en la que se insertarГЎ el elemento `x` especificado. La numeraciГіn de los Гӯndices del array comienza en cero. [`UInt32`](/es/reference/data-types/int-uint)

**Valor devuelto**

Devuelve un array con los valores insertados. [`Array`](/es/reference/data-types/array)
