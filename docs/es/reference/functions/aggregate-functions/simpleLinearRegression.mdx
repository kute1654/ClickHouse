<div id="simpleLinearRegression">
  ## simpleLinearRegression
</div>

Introducido en: v20.1.0

Realiza una regresión lineal simple (unidimensional).

**Sintaxis**

```sql
simpleLinearRegression(x, y)
```

**Argumentos**

* `x` — Columna con los valores de la variable explicativa. [`Float64`](/es/reference/data-types/float)
* `y` — Columna con los valores de la variable dependiente. [`Float64`](/es/reference/data-types/float)

**Valor devuelto**

Devuelve las constantes `(k, b)` de la recta resultante `y = k*x + b`. [`Tuple(Float64, Float64)`](/es/reference/data-types/tuple)

**Ejemplos**

**Ajuste lineal perfecto**

```sql title=Query
SELECT arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [0, 1, 2, 3]);
```

```response title=Response
┌─arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [0, 1, 2, 3])─┐
│ (1,0)                                                             │
└───────────────────────────────────────────────────────────────────┘
```

**Ajuste lineal con desplazamiento**

```sql title=Query
SELECT arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [3, 4, 5, 6]);
```

```response title=Response
┌─arrayReduce('simpleLinearRegression', [0, 1, 2, 3], [3, 4, 5, 6])─┐
│ (1,3)                                                             │
└───────────────────────────────────────────────────────────────────┘
```
