<div id="stddevPopStable">
  ## stddevPopStable
</div>

Introducido en: v1.1.0

El resultado es igual a la raíz cuadrada de [varPop](/es/reference/functions/aggregate-functions/varPop). A diferencia de [stddevPop](/es/reference/functions/aggregate-functions/stddevPop), esta función utiliza un algoritmo numéricamente estable. Es más lenta, pero ofrece un menor error computacional.

**Sintaxis**

```sql
stddevPopStable(x)
```

**Argumentos**

* `x` — Conjunto de valores del que se calculará la desviación estándar. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal)

**Valor devuelto**

Devuelve la raíz cuadrada de la varianza de `x`. [`Float64`](/es/reference/data-types/float)

**Ejemplos**

**Uso básico**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population Float64,
)
ENGINE = Log;

INSERT INTO test_data SELECT randUniform(5.5, 10) FROM numbers(1000000);

SELECT
    stddevPopStable(population) AS stddev
FROM test_data;
```

```response title=Response
┌─────────────stddev─┐
│ 1.2999977786592576 │
└────────────────────┘
```
