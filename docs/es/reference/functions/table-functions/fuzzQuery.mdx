Modifica la cadena de consulta dada con variaciones aleatorias.

<div id="syntax">
  ## Sintaxis
</div>

```sql
fuzzQuery(query[, max_query_length[, random_seed]])
```

<div id="arguments">
  ## Argumentos
</div>

| Argumento          | DescripciГіn                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------------- |
| `query`            | (String) - La consulta de entrada sobre la que se realizarГЎ el fuzzing.                     |
| `max_query_length` | (UInt64) - La longitud mГЎxima que puede alcanzar la consulta durante el proceso de fuzzing. |
| `random_seed`      | (UInt64) - Una semilla aleatoria para obtener resultados estables.                          |

<div id="returned_value">
  ## Valor devuelto
</div>

Un objeto de tabla con una sola columna que contiene cadenas de consulta alteradas.

<div id="usage-example">
  ## Ejemplo de uso
</div>

```sql
SELECT * FROM fuzzQuery('SELECT materialize(\'a\' AS key) GROUP BY key') LIMIT 2;
```
