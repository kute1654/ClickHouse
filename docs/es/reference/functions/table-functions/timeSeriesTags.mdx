`timeSeriesTags(db_name.time_series_table)` - Devuelve la tabla de [etiquetas](/es/reference/engines/table-engines/integrations/time-series#tags-table)
que utiliza la tabla `db_name.time_series_table` cuyo motor de tabla es [TimeSeries](/es/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS tags_table
```

La función también sirve si la tabla *tags* es interna:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

Las siguientes consultas son equivalentes:

```sql
SELECT * FROM timeSeriesTags(db_name.time_series_table);
SELECT * FROM timeSeriesTags('db_name.time_series_table');
SELECT * FROM timeSeriesTags('db_name', 'time_series_table');
```
