<div id="types-of-functions">
  ## Tipos de funciones JSON
</div>

Hay dos conjuntos de funciones para parsear JSON:

* [`simpleJSON*` (`visitParam*`)](#simplejson-visitparam-functions), diseñadas para parsear un subconjunto limitado de JSON con gran rapidez.
* [`JSONExtract*`](#jsonextract-functions), diseñadas para parsear JSON normal.

<div id="simplejson-visitparam-functions">
  ### funciones simpleJSON (visitParam)
</div>

ClickHouse tiene funciones especiales para trabajar con JSON simplificado. Todas estas funciones JSON se basan en suposiciones estrictas sobre cómo puede ser el JSON. Intentan hacer lo mínimo indispensable para realizar la tarea lo más rápido posible.

Se hacen las siguientes suposiciones:

1. El nombre del campo (argumento de la función) debe ser una constante.
2. El nombre del campo está codificado de forma canónica en JSON. Por ejemplo: `simpleJSONHas('{"abc":"def"}', 'abc') = 1`, pero `simpleJSONHas('{"\\u0061\\u0062\\u0063":"def"}', 'abc') = 0`
3. Los campos se buscan en cualquier nivel de anidamiento, sin distinción. Si hay varios campos coincidentes, se usa la primera aparición.
4. El JSON no tiene caracteres de espacio fuera de los literales de cadena.

<div id="jsonextract-functions">
  ### Funciones JSONExtract
</div>

Estas funciones se basan en [simdjson](https://github.com/lemire/simdjson) y están diseñadas para casos de análisis de JSON más complejos.

<div id="case-insensitive-jsonextract-functions">
  ### Funciones JSONExtract sin distinción entre mayúsculas y minúsculas
</div>

Estas funciones realizan coincidencia de claves ASCII sin distinción entre mayúsculas y minúsculas al extraer valores de objetos JSON.
Funcionan de forma idéntica a sus equivalentes sensibles a mayúsculas y minúsculas, salvo que las claves del objeto se comparan sin tener en cuenta las mayúsculas y minúsculas.
Cuando varias claves coinciden con distintas combinaciones de mayúsculas y minúsculas, se devuelve la primera coincidencia.

<Note>
  Estas funciones pueden tener un rendimiento inferior al de sus equivalentes sensibles a mayúsculas y minúsculas, así que use las funciones JSONExtract habituales siempre que sea posible.
</Note>

<div id="JSONAllPaths">
  ## JSONAllPaths
</div>

Introducido en: v24.8.0

Devuelve la lista de todas las rutas almacenadas en cada fila de una columna JSON.

**Sintaxis**

```sql
JSONAllPaths(json)
```

**Argumentos**

* `json` — columna JSON. [`JSON`](/es/reference/data-types/newjson)

**Valor devuelto**

Devuelve un array con todas las rutas de la columna JSON. [`Array(String)`](/es/reference/data-types/array)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
CREATE TABLE test (json JSON(max_dynamic_paths=1)) ENGINE = Memory;
INSERT INTO test FORMAT JSONEachRow {"json" : {"a" : 42}}, {"json" : {"b" : "Hello"}}, {"json" : {"a" : [1, 2, 3], "c" : "2020-01-01"}}
SELECT json, JSONAllPaths(json) FROM test;
```

```response title=Response
┌─json─────────────────────────────────┬─JSONAllPaths(json)─┐
│ {"a":"42"}                           │ ['a']              │
│ {"b":"Hello"}                        │ ['b']              │
│ {"a":["1","2","3"],"c":"2020-01-01"} │ ['a','c']          │
└──────────────────────────────────────┴────────────────────┘
```

<div id="JSONExtract">
  ## JSONExtract
</div>

Introducido en: v19.14.0

Analiza JSON y extrae un valor del tipo de dato de ClickHouse especificado.

**Sintaxis**

```sql
JSONExtract(json[, indices_or_keys, ...], return_type)
```

**Argumentos**

* `json` — Cadena JSON para analizar. [`String`](/es/reference/data-types/string)
* `indices_or_keys` — Una lista de cero o más argumentos, cada uno de los cuales puede ser una cadena o un entero. [`String`](/es/reference/data-types/string) o [`(U)Int*`](/es/reference/data-types/int-uint)
* `return_type` — Tipo de dato de ClickHouse que se va a devolver. [`String`](/es/reference/data-types/string)

**Valor devuelto**

Devuelve un valor del tipo de dato de ClickHouse especificado, si es posible; de lo contrario, devuelve el valor predeterminado de ese tipo.

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT JSONExtract('{"a": "hello", "b": [-100, 200.0, 300]}', 'Tuple(String, Array(Float64))') AS res;
```

```response title=Response
┌─res──────────────────────────────┐
│ ('hello',[-100,200,300])         │
└──────────────────────────────────┘
```
