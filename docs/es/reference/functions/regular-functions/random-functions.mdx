devuelvan valores aleatorios diferentes.

<div id="fuzzBits">
  ## fuzzBits
</div>

Introducido en: v20.5.0

Invierte los bits de la cadena de entrada `s`, con una probabilidad `p` para cada bit.

**Sintaxis**

```sql
fuzzBits(s, p)
```

**Argumentos**

* `s` — `String` o `FixedString` sobre el que se realiza el fuzzing de bits en [`String`](/es/reference/data-types/string) o [`FixedString`](/es/reference/data-types/fixedstring)
* `p` — Probabilidad de invertir cada bit, como un número entre `0.0` y `1.0` [`Float*`](/es/reference/data-types/float)

**Valor devuelto**

Devuelve una cadena con fuzzing del mismo tipo que `s`: [`String`](/es/reference/data-types/string) o [`FixedString`](/es/reference/data-types/fixedstring)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT fuzzBits(materialize('abacaba'), 0.1)
FROM numbers(3)
```

```response title=Response
┌─fuzzBits(materialize('abacaba'), 0.1)─┐
│ abaaaja                               │
│ a*cjab+                               │
│ aeca2A                                │
└───────────────────────────────────────┘
```

<div id="rand">
  ## rand
</div>

Introducido en: v1.1.0

Devuelve un número `UInt32` aleatorio con distribución uniforme.

Utiliza un generador congruencial lineal con un estado inicial obtenido del sistema, lo que significa que, aunque parece aleatorio, no es verdaderamente aleatorio y puede resultar predecible si se conoce el estado inicial.
En los casos en que la aleatoriedad real sea crucial, considere usar métodos alternativos, como llamadas a nivel del sistema o la integración con bibliotecas externas.

**Sintaxis**

```sql
rand([x])
```

**Alias**: `rand32`

**Argumentos**

* `x` — Opcional y se ignora. El único propósito del argumento es evitar la [eliminación de subexpresiones comunes](/es/reference/functions/regular-functions/overview#common-subexpression-elimination) cuando la misma llamada a la función se usa varias veces en una consulta. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve un número aleatorio de tipo `UInt32`. [`UInt32`](/es/reference/data-types/int-uint)

**Ejemplos**

**Ejemplo de uso**

```sql title=Query
SELECT rand();
```

```response title=Response
1569354847
```

<div id="randConstant">
  ## randConstant
</div>

Introducida en: v1.1.0

Genera un único valor aleatorio que permanece constante en todas las filas de la ejecución actual de la consulta.

Esta función:

* Devuelve el mismo valor aleatorio para cada fila dentro de una misma consulta
* Produce valores diferentes en ejecuciones de consultas distintas

Resulta útil para aplicar semillas aleatorias o identificadores consistentes en todas las filas de un conjunto de datos

**Sintaxis**

```sql
randConstant([x])
```

**Argumentos**

* `x` — Opcional e ignorado. El único propósito del argumento es evitar la [eliminación de subexpresiones comunes](/es/reference/functions/regular-functions/overview#common-subexpression-elimination) cuando la misma llamada a la función se usa varias veces en una consulta. [`Any`](/es/reference/data-types/index)

**Valor devuelto**

Devuelve una columna de tipo `UInt32` que contiene el mismo valor aleatorio en cada fila. [`UInt32`](/es/reference/data-types/int-uint)

**Ejemplos**

**Uso básico**

```sql title=Query
SELECT randConstant() AS random_value;
```

```response title=Response
| random_value |
|--------------|
| 1234567890   |
```

**Uso con un parámetro**

```sql title=Query
SELECT randConstant(10) AS random_value;
```

```response title=Response
| random_value |
|--------------|
| 9876543210   |
```
