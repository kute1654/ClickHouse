<div id="file">
  ## file
</div>

Lee un archivo como una cadena y carga los datos en la columna especificada. El contenido del archivo no se interpreta.

Véase también la función de tabla [file](/es/reference/functions/table-functions/file).

**Sintaxis**

```sql
file(path[, default])
```

**Argumentos**

* `path` — La ruta del archivo relativa a [user&#95;files&#95;path](/es/reference/settings/server-settings/settings#user_files_path). Admite comodines `*`, `**`, `?`, `{abc,def}` y `{N..M}`, donde `N` y `M` son números, y `'abc'` y `'def'` son cadenas.
* `default` — El valor que se devuelve si el archivo no existe o no se puede acceder a él. Tipos de datos admitidos: [String](/es/reference/data-types/string) y [NULL](/es/reference/settings/formats#input_format_null_as_default).

**Ejemplo**

Insertar en una tabla, como cadenas, datos de los archivos a.txt y b.txt:

```sql
INSERT INTO table SELECT file('a.txt'), file('b.txt');
```
