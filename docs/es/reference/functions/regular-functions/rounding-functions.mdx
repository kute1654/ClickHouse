<div id="ceil">
  ## ceil
</div>

Introducido en: v1.1.0

Como [`floor`](#floor), pero devuelve el menor nГәmero redondeado que sea mayor o igual que `x`.
Si el redondeo provoca un desbordamiento (por ejemplo, `ceiling(255, -1)`), el resultado no estГЎ definido.

**Sintaxis**

```sql

```

**Alias**: `ceiling`

**Argumentos**

* `x` вҖ” El valor que se desea redondear. [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`(U)Int*`](/es/reference/data-types/int-uint)
* `N` вҖ” Opcional. El nГәmero de posiciones decimales al que se redondea. El valor predeterminado es cero, lo que significa que se redondea a un entero. Puede ser negativo. [`(U)Int*`](/es/reference/data-types/int-uint)

**Valor devuelto**

Devuelve un nГәmero redondeado del mismo tipo que `x`. [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`(U)Int*`](/es/reference/data-types/int-uint)

**Ejemplos**

<div id="round">
  ## round
</div>

Introducido en: v1.1.0

Redondea un valor a un nГәmero determinado de posiciones decimales `N`.

* Si `N > 0`, la funciГіn redondea a la derecha del punto decimal.
* Si `N < 0`, la funciГіn redondea a la izquierda del punto decimal.
* Si `N = 0`, la funciГіn redondea al entero mГЎs prГіximo.

La funciГіn devuelve el nГәmero mГЎs cercano del orden especificado.
Si el valor de entrada estГЎ a igual distancia de dos nГәmeros adyacentes, la funciГіn usa redondeo bancario para entradas `Float*` y redondea alejГЎndose de cero para los demГЎs tipos numГ©ricos (`Decimal*`).

Si el redondeo provoca un desbordamiento (por ejemplo, `round(255, -1)`), el resultado no estГЎ definido.

**Sintaxis**

```sql
round(x[, N])
```

**Argumentos**

* `x` вҖ” Un nГәmero que se va a redondear. [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`(U)Int*`](/es/reference/data-types/int-uint)
* `N` вҖ” Opcional. El nГәmero de posiciones decimales al que se redondea. El valor predeterminado es `0`. [`(U)Int*`](/es/reference/data-types/int-uint)

**Valor devuelto**

Devuelve un nГәmero redondeado del mismo tipo que `x`. [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`(U)Int*`](/es/reference/data-types/int-uint)

**Ejemplos**

**Valores de entrada de coma flotante**

<div id="roundBankers">
  ## roundBankers
</div>

Introducido en: v20.1.0

Redondea un nГәmero a la posiciГіn decimal `N` especificada.
Si el nГәmero a redondear estГЎ exactamente a mitad de camino entre dos nГәmeros, la funciГіn utiliza un mГ©todo de redondeo llamado redondeo bancario, que es el mГ©todo de redondeo predeterminado para los nГәmeros de coma flotante definidos en IEEE 754.

* Si `N > 0`, la funciГіn redondea a la derecha del punto decimal
* Si `N < 0`, la funciГіn redondea a la izquierda del punto decimal
* Si `N = 0`, la funciГіn redondea al entero mГЎs cercano

<Info>
  **Notas**

  * Cuando el nГәmero a redondear estГЎ exactamente a mitad de camino entre dos nГәmeros, se redondea al dГӯgito par mГЎs cercano en la posiciГіn decimal especificada.
    Por ejemplo: `3.5` se redondea hacia arriba a `4`, `2.5` se redondea hacia abajo a `2`.
  * La funciГіn `round` realiza el mismo tipo de redondeo para nГәmeros de coma flotante.
  * La funciГіn `roundBankers` tambiГ©n redondea enteros de la misma forma; por ejemplo, `roundBankers(45, -1) = 40`.
  * En los demГЎs casos, la funciГіn redondea los nГәmeros al entero mГЎs cercano.
</Info>

<Tip>
  **Use el redondeo bancario para sumar o restar nГәmeros**

  Al usar el redondeo bancario, puede reducir el efecto que tiene el redondeo de nГәmeros en los resultados de sumarlos o restarlos.

  Por ejemplo, sume los nГәmeros `1.5, 2.5, 3.5, 4.5` con distintos tipos de redondeo:

  * Sin redondeo: `1.5 + 2.5 + 3.5 + 4.5 = 12`.
  * Redondeo bancario: `2 + 2 + 4 + 4 = 12`.
  * Redondeo al entero mГЎs cercano: `2 + 3 + 4 + 5 = 14`.
</Tip>

**Sintaxis**

```sql
roundBankers(x[, N])
```

**Argumentos**

* `x` вҖ” Un nГәmero para redondear. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Decimal*`](/es/reference/data-types/decimal) o [`Float*`](/es/reference/data-types/float)
* `[, N]` вҖ” Opcional. El nГәmero de posiciones decimales al que se redondea. El valor predeterminado es `0`. [`(U)Int*`](/es/reference/data-types/int-uint)

**Valor devuelto**

Devuelve un valor redondeado mediante el mГ©todo de redondeo bancario. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Decimal*`](/es/reference/data-types/decimal) o [`Float*`](/es/reference/data-types/float)

**Ejemplos**

**Uso bГЎsico**

```sql title=Query
SELECT number / 2 AS x, roundBankers(x, 0) AS b FROM system.numbers LIMIT 10
```

