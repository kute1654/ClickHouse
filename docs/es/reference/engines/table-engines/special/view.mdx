Se utiliza para implementar vistas (para más información, consulte la `consulta CREATE VIEW`). No almacena datos, sino únicamente la consulta `SELECT` especificada. Al leer desde una tabla, ejecuta esta consulta (y elimina de ella todas las columnas innecesarias).
