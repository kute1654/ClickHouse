Hay tres categorías principales de motores de tabla:

* [Familia de motores MergeTree](/es/reference/engines/table-engines/mergetree-family/index) para uso principal en producción.
* [Familia de motores Log](/es/reference/engines/table-engines/log-family/index) para datos temporales pequeños.
* [Motores de tabla para integraciones](/es/reference/engines/table-engines/integrations/index).

Los demás motores tienen propósitos específicos y aún no están agrupados en familias, por lo que se incluyen en esta categoría &quot;especial&quot;.

Está pensado para usarse en el lado derecho del operador `IN`.                                                                                                                                                                      |
| [Motor de tabla Join](/es/reference/engines/table-engines/special/join)                                        | Estructura de datos preparada opcional para su uso en operaciones JOIN.                                                                                                                                                                                                           |

| [Motor de tabla Memory](/es/reference/engines/table-engines/special/memory)                                    | El motor Memory almacena datos en RAM, sin comprimir. En otras palabras, leer de esta tabla es completamente gratuito.                                                                         |

Durante la operación de lectura, los datos se leen simultáneamente del búfer y de la otra tabla.                                                                               |
| [Datos externos para el procesamiento de consultas](/es/reference/engines/table-engines/special/external-data) | ClickHouse permite enviar a un servidor los datos necesarios para procesar una consulta, junto con una consulta `SELECT`. Estos datos se colocan en una tabla temporal y pueden usarse en la consulta (por ejemplo, en operadores `IN`).                                          |
| [Motor de tabla GenerateRandom](/es/reference/engines/table-engines/special/generate)                          | El motor de tabla GenerateRandom produce datos aleatorios para un esquema de tabla determinado.                                                                                                                                                                                   |
| [Motor de tabla KeeperMap](/es/reference/engines/table-engines/special/keepermap)                              | Este motor permite usar un clúster de Keeper/ZooKeeper como un almacén coherente de clave-valor con escrituras linearizables y lecturas secuencialmente consistentes.                                                                                                             |
| [Motor de tabla FileLog](/es/reference/engines/table-engines/special/filelog)                                  | Este motor permite procesar archivos de registro de aplicaciones como un flujo de registros.                                                                                                                                                                                      |
