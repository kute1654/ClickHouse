```sql
EXISTS [TEMPORARY] [TABLE|DICTIONARY|DATABASE] [db.]name [INTO OUTFILE filename] [FORMAT format]
```

Devuelve una sola columna de tipo `UInt8`, que contiene el valor `0` si la tabla o la base de datos no existen, o `1` si la tabla existe en la base de datos especificada.
