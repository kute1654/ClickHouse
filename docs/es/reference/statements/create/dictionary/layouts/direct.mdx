<div id="direct">
  ## direct
</div>

El diccionario no se almacena en memoria y consulta directamente la fuente al procesar una solicitud.

La clave del diccionario es de tipo [UInt64](/es/reference/data-types/int-uint).

Se admiten todos los tipos de [fuentes](/es/reference/statements/create/dictionary/sources/overview#dictionary-sources), excepto los archivos locales.

Ejemplo de configuración:

<Tabs>
  <Tab title="DDL">
    ```sql
    LAYOUT(DIRECT())
    ```
  </Tab>

  <Tab title="Archivo de configuración">
    ```xml
    <layout>
      <direct />
    </layout>
    ```
  </Tab>
</Tabs>

<br />

  ## complex_key_direct

Este tipo de almacenamiento se utiliza con [claves](/es/reference/statements/create/dictionary/attributes#composite-key) compuestas. Similar a `direct`.
