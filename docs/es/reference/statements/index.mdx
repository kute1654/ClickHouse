Los usuarios interactúan con ClickHouse mediante sentencias SQL. ClickHouse admite sentencias SQL comunes como [SELECT](/es/reference/statements/select/index) y [CREATE](/es/reference/statements/create/index), pero también ofrece sentencias especializadas como [KILL](/es/reference/statements/kill) y [OPTIMIZE](/es/reference/statements/optimize).
