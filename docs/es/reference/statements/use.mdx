```sql
USE [DATABASE] db
```

Permite establecer la base de datos actual de la sesión.

La base de datos actual se utiliza para buscar tablas si la base de datos no se especifica explícitamente en la consulta con un punto antes del nombre de la tabla.

Esta consulta no se puede realizar al usar el protocolo HTTP, ya que no existe el concepto de sesión.
