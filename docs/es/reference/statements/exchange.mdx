Intercambia de forma atГіmica los nombres de dos tablas o diccionarios.
Esta tarea tambiГ©n puede realizarse con una consulta [`RENAME`](/es/reference/statements/rename) usando un nombre temporal, pero en ese caso la operaciГіn no es atГіmica.

<Note>
  La consulta `EXCHANGE` solo es compatible con los motores de base de datos [`Atomic`](/es/reference/engines/database-engines/atomic) y [`Shared`](/es/products/cloud/features/infrastructure/shared-catalog#shared-database-engine).
</Note>

**Sintaxis**

```sql
EXCHANGE TABLES|DICTIONARIES [db0.]name_A AND [db1.]name_B [ON CLUSTER cluster]
```

<div id="exchange-tables">
  ## EXCHANGE TABLES
</div>

Intercambia los nombres de dos tablas.

**Sintaxis**

```sql
EXCHANGE TABLES [db0.]table_A AND [db1.]table_B [ON CLUSTER cluster]
```

<div id="exchange-multiple-tables">
  ### EXCHANGE DE VARIAS TABLAS
</div>

Puede intercambiar varios pares de tablas en una sola consulta separГЎndolos con comas.

<Note>
  Al intercambiar varios pares de tablas, los intercambios se realizan **de forma secuencial, no atГіmica**. Si se produce un error durante la operaciГіn, es posible que algunos pares de tablas se hayan intercambiado y otros no.
</Note>

**Ejemplo**

```sql title="Query"
-- Crear tablas
CREATE TABLE a (a UInt8) ENGINE=Memory;
CREATE TABLE b (b UInt8) ENGINE=Memory;
CREATE TABLE c (c UInt8) ENGINE=Memory;
CREATE TABLE d (d UInt8) ENGINE=Memory;

-- Intercambiar dos pares de tablas en una sola consulta
EXCHANGE TABLES a AND b, c AND d;

SHOW TABLE a;
SHOW TABLE b;
SHOW TABLE c;
SHOW TABLE d;
```
