<div id="description">
  ## Descripción
</div>

Esta tabla contiene estadísticas agregadas de las operaciones de ZooKeeper (p. ej., número de operaciones, latencia media, errores), agrupadas por `(session_id, parent_path, operation, component, is_subrequest)` y volcadas periódicamente a disco.

A diferencia de [system.zookeeper&#95;log](/es/reference/system-tables/zookeeper_log), que registra cada solicitud y respuesta individual, esta tabla agrupa las operaciones, por lo que es mucho más ligera y, en consecuencia, más adecuada para workloads de producción.

Las operaciones que forman parte de un lote `Multi` o `MultiRead` se registran por separado mediante la columna `is_subrequest`. Las subsolicitudes tienen latencia cero porque la latencia total se atribuye a la operación `Multi`/`MultiRead` que las engloba.

* `operation` ([Enum16(&#39;Close&#39; = -11, &#39;Error&#39; = -1, &#39;Watch&#39; = 0, &#39;Create&#39; = 1, &#39;Remove&#39; = 2, &#39;Exists&#39; = 3, &#39;Get&#39; = 4, &#39;Set&#39; = 5, &#39;GetACL&#39; = 6, &#39;SetACL&#39; = 7, &#39;SimpleList&#39; = 8, &#39;Sync&#39; = 9, &#39;Heartbeat&#39; = 11, &#39;List&#39; = 12, &#39;Check&#39; = 13, &#39;Multi&#39; = 14, &#39;Create2&#39; = 15, &#39;Reconfig&#39; = 16, &#39;CheckWatch&#39; = 17, &#39;RemoveWatch&#39; = 18, &#39;MultiRead&#39; = 22, &#39;Auth&#39; = 100, &#39;SetWatch&#39; = 101, &#39;SetWatch2&#39; = 105, &#39;AddWatch&#39; = 106, &#39;FilteredList&#39; = 500, &#39;CheckNotExists&#39; = 501, &#39;CreateIfNotExists&#39; = 502, &#39;RemoveRecursive&#39; = 503, &#39;CheckStat&#39; = 504, &#39;TryRemove&#39; = 505, &#39;FilteredListWithStatsAndData&#39; = 506, &#39;ListRecursive&#39; = 507, &#39;SessionID&#39; = 997)](/es/reference/data-types/enum)) — Tipo de operación de ZooKeeper.
* `is_subrequest` ([UInt8](/es/reference/data-types/int-uint)) — Indica si esta operación fue una subsolicitud dentro de una operación Multi o MultiRead.
* `count` ([UInt32](/es/reference/data-types/int-uint)) — Número de operaciones en el grupo (session&#95;id, parent&#95;path, operation, component, is&#95;subrequest).
* `errors` ([Map(Enum8(&#39;ZNOWATCHER&#39; = -121, &#39;ZNOTREADONLY&#39; = -119, &#39;ZSESSIONMOVED&#39; = -118, &#39;ZNOTHING&#39; = -117, &#39;ZCLOSING&#39; = -116, &#39;ZAUTHFAILED&#39; = -115, &#39;ZINVALIDACL&#39; = -114, &#39;ZINVALIDCALLBACK&#39; = -113, &#39;ZSESSIONEXPIRED&#39; = -112, &#39;ZNOTEMPTY&#39; = -111, &#39;ZNODEEXISTS&#39; = -110, &#39;ZNOCHILDRENFOREPHEMERALS&#39; = -108, &#39;ZBADVERSION&#39; = -103, &#39;ZNOAUTH&#39; = -102, &#39;ZNONODE&#39; = -101, &#39;ZAPIERROR&#39; = -100, &#39;ZOUTOFMEMORY&#39; = -10, &#39;ZINVALIDSTATE&#39; = -9, &#39;ZBADARGUMENTS&#39; = -8, &#39;ZOPERATIONTIMEOUT&#39; = -7, &#39;ZUNIMPLEMENTED&#39; = -6, &#39;ZMARSHALLINGERROR&#39; = -5, &#39;ZCONNECTIONLOSS&#39; = -4, &#39;ZDATAINCONSISTENCY&#39; = -3, &#39;ZRUNTIMEINCONSISTENCY&#39; = -2, &#39;ZSYSTEMERROR&#39; = -1, &#39;ZOK&#39; = 0), UInt32)](/es/reference/data-types/map)) — Errores en el grupo (session&#95;id, parent&#95;path, operation, component, is&#95;subrequest).
* `average_latency` ([Float64](/es/reference/data-types/float)) — Latencia media de todas las operaciones del grupo (session&#95;id, parent&#95;path, operation, component, is&#95;subrequest), en microsegundos. Las subsolicitudes tienen latencia cero porque la latencia se atribuye a la operación Multi o MultiRead que las contiene.
* `component` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Componente que provocó el evento.
