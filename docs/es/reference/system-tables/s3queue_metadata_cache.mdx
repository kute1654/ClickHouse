<div id="description">
  ## Descripción
</div>

Contiene el estado en memoria de los metadatos de S3Queue y las filas actualmente en proceso de cada archivo.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/es/reference/data-types/index)) — Ruta en ZooKeeper a los metadatos
* `file_path` ([String](/es/reference/data-types/index)) — Ruta del archivo que se está procesando
* `file_name` ([String](/es/reference/data-types/index)) — Nombre del archivo que se está procesando
* `rows_processed` ([UInt64](/es/reference/data-types/index)) — Número de filas procesadas actualmente
* `status` ([String](/es/reference/data-types/index)) — Estado del procesamiento: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](/es/reference/data-types/index)) — Hora a la que comenzó el procesamiento del archivo
* `processing_end_time` ([Nullable(DateTime)](/es/reference/data-types/index)) — Hora a la que finalizó el procesamiento del archivo
* `exception` ([String](/es/reference/data-types/index)) — Excepción ocurrida durante el procesamiento

{/*AUTOGENERATED_END*/}
