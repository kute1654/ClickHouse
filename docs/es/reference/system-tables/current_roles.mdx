<div id="description">
  ## Descripción
</div>

Contiene los roles activos del usuario actual. `SET ROLE` modifica el contenido de esta tabla.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/es/reference/data-types/index)) — Nombre del rol.
* `with_admin_option` ([UInt8](/es/reference/data-types/index)) — 1 si el rol tiene el privilegio ADMIN OPTION.
* `is_default` ([UInt8](/es/reference/data-types/index)) — 1 si el rol está configurado como predeterminado.

{/*AUTOGENERATED_END*/}
