<div id="description">
  ## Descripción
</div>

Contiene información sobre cada tabla en estado desvinculado.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — El nombre de la base de datos en la que se encuentra la tabla.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `uuid` ([UUID](/es/reference/data-types/index)) — UUID de la tabla (base de datos Atomic).
* `metadata_path` ([String](/es/reference/data-types/index)) — Ruta de los metadatos de la tabla en el sistema de archivos.
* `is_permanently` ([UInt8](/es/reference/data-types/index)) — La tabla quedó desvinculada de forma permanente.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

