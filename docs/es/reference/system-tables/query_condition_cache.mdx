<div id="description">
  ## Descripción
</div>

Muestra el contenido de la [caché de condiciones de consulta](/es/concepts/features/performance/caches/query-condition-cache).

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `key_hash` ([UInt128](/es/reference/data-types/index)) — Hash de (table&#95;uuid, part&#95;name, condition&#95;hash).
* `entry_size` ([UInt64](/es/reference/data-types/index)) — Tamaño de la entrada en bytes.
* `matching_marks` ([String](/es/reference/data-types/index)) — Marcas correspondientes.

{/*AUTOGENERATED_END*/}
