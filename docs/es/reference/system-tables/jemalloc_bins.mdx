<div id="description">
  ## Descripción
</div>

Contiene información sobre las asignaciones de memoria realizadas mediante el asignador jemalloc en distintas clases de tamaño (bins), agregadas de todas las arenas.
Es posible que estas estadísticas no sean totalmente precisas debido a la caché local por hilo en jemalloc.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `index` ([UInt16](/es/reference/data-types/index)) — Índice del bin ordenado por tamaño.
* `large` ([UInt8](/es/reference/data-types/index)) — Verdadero para las asignaciones grandes y Falso para las pequeñas.
* `size` ([UInt64](/es/reference/data-types/index)) — Tamaño de las asignaciones en este bin.
* `allocations` ([Int64](/es/reference/data-types/index)) — Número de asignaciones de memoria.
* `deallocations` ([Int64](/es/reference/data-types/index)) — Número de liberaciones de memoria.
* `nregs` ([Int64](/es/reference/data-types/index)) — Número de regiones por slab.
* `curslabs` ([Int64](/es/reference/data-types/index)) — Número actual de slabs.
* `curregs` ([Int64](/es/reference/data-types/index)) — Número actual de regiones para esta clase de tamaño.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

Encuentre los tamaños de las asignaciones de memoria que más contribuyeron al uso total de memoria actual.

```sql
SELECT
    *,
    allocations - deallocations AS active_allocations,
    size * active_allocations AS allocated_bytes
FROM system.jemalloc_bins
WHERE allocated_bytes > 0
ORDER BY allocated_bytes DESC
LIMIT 10
```
