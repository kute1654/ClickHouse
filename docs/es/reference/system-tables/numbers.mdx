<div id="description">
  ## Descripción
</div>

Esta tabla contiene una única columna UInt64 llamada `number`, que contiene casi todos los números naturales a partir de cero.

Puede usar esta tabla para realizar pruebas o si necesita hacer una búsqueda por fuerza bruta.

La lectura de esta tabla no está paralelizada.
