<div id="description">
  ## Descripción
</div>

Contiene una lista de todos los combinadores de funciones de agregación disponibles, que se pueden aplicar a las funciones de agregación para cambiar su funcionamiento.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — El nombre del combinador.
* `is_internal` ([UInt8](/es/reference/data-types/index)) — Indica si este combinador es solo para uso interno.
* `description` ([String](/es/reference/data-types/index)) — Una descripción general de lo que hace el combinador.
* `syntax` ([String](/es/reference/data-types/index)) — Cómo se aplica el combinador al nombre de una función de agregación.
* `examples` ([String](/es/reference/data-types/index)) — Ejemplos de uso.
* `introduced_in` ([String](/es/reference/data-types/index)) — La versión de ClickHouse en la que se introdujo por primera vez el combinador, en el formato major.minor.
* `related` ([Array(String)](/es/reference/data-types/index)) — Los nombres de los combinadores relacionados.

{/*AUTOGENERATED_END*/}
