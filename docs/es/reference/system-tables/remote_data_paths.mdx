<div id="description">
  ## Descripción
</div>

Contiene información sobre los archivos de datos almacenados en discos remotos (p. ej., S3, Azure Blob Storage), incluida la correspondencia entre las rutas de metadatos locales y las rutas de blobs remotos.

Cada fila representa un objeto blob remoto asociado a un archivo de datos.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `disk_name` ([String](/es/reference/data-types/index)) — Nombre del disco.
* `path` ([String](/es/reference/data-types/index)) — Ruta del disco.
* `cache_base_path` ([String](/es/reference/data-types/index)) — Directorio base de los archivos de caché.
* `local_path` ([String](/es/reference/data-types/index)) — Ruta del archivo de ClickHouse, que también se usa como ruta de metadatos.
* `remote_path` ([String](/es/reference/data-types/index)) — Ruta del blob en el almacenamiento de objetos, a la que está asociado el archivo de ClickHouse.
* `size` ([UInt64](/es/reference/data-types/index)) — Tamaño del archivo (comprimido).
* `common_prefix_for_blobs` ([String](/es/reference/data-types/index)) — Prefijo común de los blobs en el almacenamiento de objetos.
* `cache_paths` ([Array(String)](/es/reference/data-types/index)) — Archivos de caché del blob correspondiente.

{/*AUTOGENERATED_END*/}

</div>

