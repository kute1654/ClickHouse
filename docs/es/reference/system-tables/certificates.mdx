<div id="description">
  ## Descripción
</div>

Contiene información sobre los certificados disponibles y sus fuentes.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/es/reference/data-types/index)) — Versión del certificado. Los valores son 0 para v1, 1 para v2 y 2 para v3.
* `serial_number` ([Nullable(String)](/es/reference/data-types/index)) — Número de serie del certificado asignado por el emisor.
* `signature_algo` ([Nullable(String)](/es/reference/data-types/index)) — Algoritmo de firma: algoritmo utilizado por el emisor para firmar este certificado.
* `issuer` ([Nullable(String)](/es/reference/data-types/index)) — Emisor: identificador único de la autoridad de certificación que emite este certificado.
* `not_before` ([Nullable(String)](/es/reference/data-types/index)) — Inicio de la ventana de tiempo en la que este certificado es válido.
* `not_after` ([Nullable(String)](/es/reference/data-types/index)) — Fin de la ventana de tiempo en la que este certificado es válido.
* `subject` ([Nullable(String)](/es/reference/data-types/index)) — Sujeto: identifica al propietario de la clave pública.
* `pkey_algo` ([Nullable(String)](/es/reference/data-types/index)) — Algoritmo de clave pública: define el algoritmo con el que se puede usar la clave pública.
* `path` ([String](/es/reference/data-types/index)) — Ruta al archivo o directorio que contiene este certificado.
* `default` ([UInt8](/es/reference/data-types/index)) — El certificado se encuentra en la ubicación predeterminada de certificados.
* `protocol` ([String](/es/reference/data-types/index)) — Nombre del protocolo si el certificado proviene de la configuración TLS por protocolo; de lo contrario, está vacío.

{/*AUTOGENERATED_END*/}
