<div id="description">
  ## Descripción
</div>

Esta tabla del sistema puede utilizarse para obtener una visión general del uso de memoria y de los ProfileEvents de los usuarios.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `user` ([String](/es/reference/data-types/index)) — Nombre de usuario.
* `memory_usage` ([Int64](/es/reference/data-types/index)) — Suma de la RAM utilizada por todos los procesos del usuario. Es posible que no incluya algunos tipos de memoria dedicada. Consulte la configuración max&#95;memory&#95;usage.
* `peak_memory_usage` ([Int64](/es/reference/data-types/index)) — Pico de uso de memoria del usuario. Puede restablecerse cuando no se ejecutan consultas para ese usuario.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/es/reference/data-types/index)) — Resumen de los ProfileEvents que miden distintas métricas del usuario. Su descripción puede encontrarse en la tabla system.events

{/*AUTOGENERATED_END*/}

</div>

SELECT * FROM system.user_processes LIMIT 10 FORMAT Vertical;
