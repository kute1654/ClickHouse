<div id="description">
  ## Descripción
</div>

La tabla `system.part_log` se crea solo si se ha especificado la configuración del servidor [part&#95;log](/es/reference/settings/server-settings/settings#part_log).

Esta tabla contiene información sobre eventos ocurridos con las [partes de datos](/es/reference/engines/table-engines/mergetree-family/custom-partitioning-key) en las tablas de la familia [MergeTree](/es/reference/engines/table-engines/mergetree-family/mergetree), como la adición o la fusión de datos.

La tabla `system.part_log` contiene las siguientes columnas:

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Nombre de host del servidor que ejecuta la consulta.
* `query_id` ([String](/es/reference/data-types/string)) — Identificador de la consulta INSERT que creó esta parte de datos.
* `event_type` ([Enum8(&#39;NewPart&#39; = 1, &#39;MergeParts&#39; = 2, &#39;DownloadPart&#39; = 3, &#39;RemovePart&#39; = 4, &#39;MutatePart&#39; = 5, &#39;MovePart&#39; = 6, &#39;MergePartsStart&#39; = 7, &#39;MutatePartStart&#39; = 8)](/es/reference/data-types/enum)) — Tipo del evento que se produjo con la parte de datos. Puede tener uno de los siguientes valores: NewPart — Inserción de una nueva parte de datos, MergePartsStart — Ha comenzado la fusión de partes de datos, MergeParts — La fusión de partes de datos ha finalizado, DownloadPart — Descarga de una parte de datos, RemovePart — Eliminación o desacoplamiento de una parte de datos mediante [DETACH PARTITION](/es/reference/statements/alter/partition#detach-partitionpart).MutatePartStart — Ha comenzado la mutación de una parte de datos, MutatePart — La mutación de una parte de datos ha finalizado, MovePart — Movimiento de la parte de datos de un disco a otro.
* `merge_reason` ([Enum8(&#39;NotAMerge&#39; = 1, &#39;RegularMerge&#39; = 2, &#39;TTLDeleteMerge&#39; = 3, &#39;TTLRecompressMerge&#39; = 4, &#39;TTLDropMerge&#39; = 5)](/es/reference/data-types/enum)) — El motivo del evento de tipo MERGE&#95;PARTS. Puede tener uno de los siguientes valores: NotAMerge — El evento actual tiene un tipo distinto de MERGE&#95;PARTS, RegularMerge — Una fusión normal, TTLDeleteMerge, TTLDropMerge — Limpieza de datos caducados. TTLRecompressMerge — Recompresión de la parte de datos con el.
* `merge_algorithm` ([Enum8(&#39;Undecided&#39; = 0, &#39;Vertical&#39; = 1, &#39;Horizontal&#39; = 2)](/es/reference/data-types/enum)) — Algoritmo de fusión para el evento de tipo MERGE&#95;PARTS. Puede tomar uno de los siguientes valores: Undecided, Horizontal, Vertical
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha del evento.
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — Hora del evento.
* `event_time_microseconds` ([DateTime64(6)](/es/reference/data-types/datetime64)) — Hora del evento con precisión de microsegundos.
* `duration_ms` ([UInt64](/es/reference/data-types/int-uint)) — Duración de esta operación.
* `database` ([String](/es/reference/data-types/string)) — Nombre de la base de datos a la que pertenece la parte de datos.
* `table` ([String](/es/reference/data-types/string)) — Nombre de la tabla a la que pertenece la parte de datos.
* `table_uuid` ([UUID](/es/reference/data-types/uuid)) — UUID de la tabla a la que pertenece la parte de datos.
* `part_name` ([String](/es/reference/data-types/string)) — Nombre de la parte de datos.
* `partition_id` ([String](/es/reference/data-types/string)) — ID de la partición en la que se insertó la parte de datos. La columna toma el valor `all` si la partición se define mediante `tuple()`.
* `partition` ([String](/es/reference/data-types/string)) — El nombre de la partición.
* `part_type` ([String](/es/reference/data-types/string)) — El tipo de la parte. Posibles valores: Wide y Compact.
* `part_storage_type` ([String](/es/reference/data-types/string)) — El tipo de DataPartStorage. Valores posibles: Packed - todos los archivos se almacenan en un solo blob, Full - un blob por archivo.
* `disk_name` ([String](/es/reference/data-types/string)) — El nombre del disco en el que reside la parte de datos.
* `path_on_disk` ([String](/es/reference/data-types/string)) — Ruta absoluta de la carpeta que contiene los archivos de la parte de datos.
* `rows` ([UInt64](/es/reference/data-types/int-uint)) — El número de filas en la parte de datos.
* `size_in_bytes` ([UInt64](/es/reference/data-types/int-uint)) — Tamaño de la parte de datos en disco, en bytes.
* `merged_from` ([Array(String)](/es/reference/data-types/array)) — Un array con los nombres de las partes de origen de las que se compone la parte actual.
* `bytes_uncompressed` ([UInt64](/es/reference/data-types/int-uint)) — Tamaño sin comprimir de la parte resultante, en bytes.
* `read_rows` ([UInt64](/es/reference/data-types/int-uint)) — El número de filas leídas durante la fusión.
* `read_bytes` ([UInt64](/es/reference/data-types/int-uint)) — El número de bytes leídos durante la fusión.
* `peak_memory_usage` ([UInt64](/es/reference/data-types/int-uint)) — La cantidad máxima de RAM utilizada durante la fusión
* `deduplication_block_ids` ([Array(String)](/es/reference/data-types/array)) — Un array de identificadores de bloque utilizado para la deduplicación al insertar esta parte.
* `error` ([UInt16](/es/reference/data-types/int-uint)) — El código de error de la excepción ocurrida.
* `exception` ([String](/es/reference/data-types/string)) — Mensaje de texto del error que se produjo.
* `mutation_ids` ([Array(String)](/es/reference/data-types/array)) — Un array de ID de mutación aplicados a la parte de origen (merged&#95;from) para el evento de tipo MUTATE&#95;PART&#95;START y MUTATE&#95;PART.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/es/reference/data-types/map)) — Todos los eventos de perfil registrados durante esta operación.
* `projections_duration_ms` ([Map(LowCardinality(String), UInt64)](/es/reference/data-types/map)) — Duración de la fusión/reconstrucción por proyección, en milisegundos.

**Alias:**

* `ProfileEvents.Names` — alias de `mapKeys(ProfileEvents)`.
* `ProfileEvents.Values` — alias de `mapValues(ProfileEvents)`.
* `name` — alias de `part_name`.

{/*AUTOGENERATED_END*/}

La tabla `system.part_log` se crea tras la primera inserción de datos en la tabla `MergeTree`.
