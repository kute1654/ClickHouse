<div id="description">
  ## Descripción
</div>

Contiene información sobre las [cargas de trabajo](/es/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage) que residen en el servidor local. La tabla contiene una fila por cada carga de trabajo.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — El nombre de la carga de trabajo.
* `parent` ([String](/es/reference/data-types/index)) — El nombre de la carga de trabajo principal.
* `create_query` ([String](/es/reference/data-types/index)) — La consulta CREATE de la carga de trabajo.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT *
FROM system.workloads
FORMAT Vertical
```

```text
Row 1:
──────
name:         production
parent:       all
create_query: CREATE WORKLOAD production IN `all` SETTINGS weight = 9

Row 2:
──────
name:         development
parent:       all
create_query: CREATE WORKLOAD development IN `all`

Row 3:
──────
name:         all
parent:
create_query: CREATE WORKLOAD `all`
```
