<div id="description">
  ## Descripción
</div>

La tabla &#39;system.zookeeper&#95;connection&#95;log&#39; muestra el historial de las conexiones a ZooKeeper (incluidos los ZooKeepers auxiliares). Cada fila muestra información sobre un evento relacionado con una conexión.

<Note>
  La tabla no contiene eventos de desconexión provocados por el apagado del servidor.
</Note>

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Nombre de host del servidor que se conectó o se desconectó de ZooKeeper.
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/es/reference/data-types/enum)) — Tipo de evento. Posibles valores: Connected, Disconnected.
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha del registro.
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — Hora del registro
* `event_time_microseconds` ([DateTime64(6)](/es/reference/data-types/datetime64)) — Hora del registro con precisión de microsegundos.
* `name` ([String](/es/reference/data-types/string)) — Nombre del cluster de ZooKeeper.
* `host` ([String](/es/reference/data-types/string)) — El hostname/IP del nodo de ZooKeeper al que ClickHouse se conectó o del que se desconectó.
* `port` ([UInt16](/es/reference/data-types/int-uint)) — El puerto del nodo de ZooKeeper al que ClickHouse se conectó o del que se desconectó.
* `index` ([UInt8](/es/reference/data-types/int-uint)) — El índice del nodo de ZooKeeper al que ClickHouse se conectó o del que se desconectó. El índice proviene de la configuración de ZooKeeper.
* `client_id` ([Int64](/es/reference/data-types/int-uint)) — ID de sesión de la conexión.
* `keeper_api_version` ([UInt8](/es/reference/data-types/int-uint)) — Versión de la API de Keeper.
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/es/reference/data-types/array)) — Feature flags habilitadas. Solo se aplica a ClickHouse Keeper.
* `availability_zone` ([String](/es/reference/data-types/string)) — Zona de disponibilidad
* `reason` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Motivo de la conexión o desconexión.

{/*AUTOGENERATED_END*/}
