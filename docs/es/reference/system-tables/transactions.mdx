<div id="description">
  ## Descripción
</div>

Contiene una lista de transacciones y sus estados.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/es/reference/data-types/tuple)) — El identificador de la transacción.
* `tid_hash` ([UInt64](/es/reference/data-types/int-uint)) — El hash del identificador.
* `elapsed` ([Float64](/es/reference/data-types/float)) — El tiempo transcurrido de la transacción que se está procesando.
* `is_readonly` ([UInt8](/es/reference/data-types/int-uint)) — El indicador que muestra si la transacción ha ejecutado alguna operación de escritura.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/es/reference/data-types/enum)) — El estado de la transacción. Valores posibles: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/}
