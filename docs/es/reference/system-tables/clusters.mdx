<div id="description">
  ## Descripción
</div>

Contiene información sobre los clústeres disponibles en el archivo de configuración y los servidores que los componen.

<div id="Columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `cluster` ([String](/es/reference/data-types/index)) — El nombre del clúster.
* `shard_num` ([UInt32](/es/reference/data-types/index)) — El número del segmento en el clúster, a partir de 1.
* `shard_name` ([String](/es/reference/data-types/index)) — El nombre del segmento en el clúster.
* `shard_weight` ([UInt32](/es/reference/data-types/index)) — El peso relativo del segmento al escribir datos.
* `internal_replication` ([UInt8](/es/reference/data-types/index)) — Indicador que señala si este host forma parte de un conjunto que puede replicar los datos por sí mismo.
* `replica_num` ([UInt32](/es/reference/data-types/index)) — El número de la réplica en el segmento, a partir de 1.
* `host_name` ([String](/es/reference/data-types/index)) — El nombre del host, tal como se especifica en la configuración.
* `host_address` ([String](/es/reference/data-types/index)) — La dirección IP del host obtenida mediante DNS.
* `port` ([UInt16](/es/reference/data-types/index)) — El puerto que se usa para conectarse al servidor.
* `is_local` ([UInt8](/es/reference/data-types/index)) — Indicador que señala si el host es local.
* `user` ([String](/es/reference/data-types/index)) — El nombre del usuario para conectarse al servidor.
* `default_database` ([String](/es/reference/data-types/index)) — El nombre de la base de datos predeterminada.
* `errors_count` ([UInt32](/es/reference/data-types/index)) — El número de veces que este host no pudo alcanzar la réplica.
* `slowdowns_count` ([UInt32](/es/reference/data-types/index)) — El número de ralentizaciones que provocaron un cambio de réplica al establecer una conexión con solicitudes hedged.
* `estimated_recovery_time` ([UInt32](/es/reference/data-types/index)) — Segundos restantes hasta que el recuento de errores de la réplica se restablezca a cero y se considere que ha vuelto a la normalidad.
* `database_shard_name` ([String](/es/reference/data-types/index)) — El nombre del segmento de la base de datos `Replicated` (para clústeres que pertenecen a una base de datos `Replicated`).
* `database_replica_name` ([String](/es/reference/data-types/index)) — El nombre de la réplica de la base de datos `Replicated` (para clústeres que pertenecen a una base de datos `Replicated`).
* `is_shared_catalog_cluster` ([UInt8](/es/reference/data-types/index)) — Bool que indica si el clúster pertenece al Shared Catalog.
* `is_active` ([Nullable(UInt8)](/es/reference/data-types/index)) — El estado de la réplica de la base de datos `Replicated` (para clústeres que pertenecen a una base de datos `Replicated`): 1 significa &#39;la réplica está en línea&#39;, 0 significa &#39;la réplica está fuera de línea&#39;, NULL significa &#39;desconocido&#39;.
* `unsynced_after_recovery` ([Nullable(UInt8)](/es/reference/data-types/index)) — Indica si una réplica de la base de datos `Replicated` tiene un retraso de replicación superior a max&#95;replication&#95;lag&#95;to&#95;enqueue después de crear o recuperar la réplica.
* `replication_lag` ([Nullable(UInt32)](/es/reference/data-types/index)) — El retraso de replicación de la réplica de la base de datos `Replicated` (para clústeres que pertenecen a una base de datos `Replicated`).
* `recovery_time` ([Nullable(UInt64)](/es/reference/data-types/index)) — El tiempo de recuperación de la réplica de la base de datos `Replicated` (para clústeres que pertenecen a una base de datos `Replicated`), en milisegundos.

{/*AUTOGENERATED_END*/}
