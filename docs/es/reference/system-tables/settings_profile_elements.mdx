<div id="description">
  ## Descripción
</div>

Describe el contenido del perfil de configuración:

* Restricciones.
* Roles y usuarios a los que se aplica la configuración.
* Perfiles de configuración superiores.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre del perfil de SETTING.
* `user_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre de usuario.
* `role_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre del rol.
* `index` ([UInt64](/es/reference/data-types/index)) — Número secuencial del elemento del perfil de SETTING.
* `setting_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre del SETTING.
* `value` ([Nullable(String)](/es/reference/data-types/index)) — Valor del SETTING.
* `min` ([Nullable(String)](/es/reference/data-types/index)) — Valor mínimo del SETTING. NULL si no está establecido.
* `max` ([Nullable(String)](/es/reference/data-types/index)) — Valor máximo del SETTING. NULL si no está establecido.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/es/reference/data-types/index)) — Propiedad que indica si un SETTING puede modificarse o no.
* `inherit_profile` ([Nullable(String)](/es/reference/data-types/index)) — Perfil padre de este perfil de SETTING. NULL si no está establecido. El perfil de SETTING heredará todos los valores y restricciones de los ajustes (min, max, readonly) de sus perfiles padre.

{/*AUTOGENERATED_END*/}
