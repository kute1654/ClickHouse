<div id="description">
  ## Descripción
</div>

Contiene información sobre las partes que actualmente se están moviendo entre segmentos y su progreso.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — El nombre de la base de datos donde se realiza el movimiento.
* `table` ([String](/es/reference/data-types/index)) — El nombre de la tabla donde se realiza el movimiento.
* `task_name` ([String](/es/reference/data-types/index)) — El nombre de la tarea de movimiento.
* `task_uuid` ([UUID](/es/reference/data-types/index)) — El identificador de la tarea de movimiento.
* `create_time` ([DateTime](/es/reference/data-types/index)) — La hora a la que se creó la tarea.
* `part_name` ([String](/es/reference/data-types/index)) — El nombre de la parte que está en proceso de movimiento.
* `part_uuid` ([UUID](/es/reference/data-types/index)) — El UUID de la parte que está en proceso de movimiento.
* `to_shard` ([String](/es/reference/data-types/index)) — El nombre del segmento de destino.
* `dst_part_name` ([String](/es/reference/data-types/index)) — El nombre de la parte resultante.
* `update_time` ([DateTime](/es/reference/data-types/index)) — La última vez que se actualizó.
* `state` ([String](/es/reference/data-types/index)) — El estado actual del movimiento.
* `rollback` ([UInt8](/es/reference/data-types/index)) — El indicador que indica si la operación se revirtió.
* `num_tries` ([UInt32](/es/reference/data-types/index)) — El número de intentos para completar la operación.
* `last_exception` ([String](/es/reference/data-types/index)) — El nombre de la última excepción, si la hay.

{/*AUTOGENERATED_END*/}
