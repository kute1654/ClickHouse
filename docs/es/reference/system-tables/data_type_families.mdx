<div id="description">
  ## Descripción
</div>

Contiene información sobre los [tipos de datos](/es/reference/data-types/index).

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del tipo de dato.
* `case_insensitive` ([UInt8](/es/reference/data-types/index)) — Propiedad que indica si un nombre de tipo de dato puede usarse en una consulta sin distinguir entre mayúsculas y minúsculas. Por ejemplo, `Date` y `date` son válidos.
* `alias_to` ([String](/es/reference/data-types/index)) — Nombre del tipo de dato del que `name` es un alias.
* `description` ([String](/es/reference/data-types/index)) — Descripción general del tipo de dato.
* `syntax` ([String](/es/reference/data-types/index)) — Cómo se escribe el tipo de dato en una consulta.
* `examples` ([String](/es/reference/data-types/index)) — Ejemplos de uso.
* `introduced_in` ([String](/es/reference/data-types/index)) — La versión de ClickHouse en la que se introdujo por primera vez el tipo de dato, con el formato major.minor.
* `related` ([Array(String)](/es/reference/data-types/index)) — Nombres de tipos de dato relacionados.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT name, case_insensitive, alias_to FROM system.data_type_families WHERE alias_to = 'String'
```

```text
└────────────┴──────────────────┴──────────┘
```
