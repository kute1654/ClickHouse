<div id="description">
  ## Descripción
</div>

Uso de la QUOTA del usuario actual: cuánto se ha usado y cuánto queda.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/es/reference/data-types/index)) — Nombre de la QUOTA.
* `quota_key` ([String](/es/reference/data-types/index)) — Valor de la clave.
* `start_time` ([Nullable(DateTime)](/es/reference/data-types/index)) — Hora de inicio para calcular el consumo de recursos.
* `end_time` ([Nullable(DateTime)](/es/reference/data-types/index)) — Hora de finalización para calcular el consumo de recursos.
* `duration` ([Nullable(UInt32)](/es/reference/data-types/index)) — Duración del intervalo de tiempo para calcular el consumo de recursos, en segundos.
* `queries` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número actual de consultas ejecutadas.
* `max_queries` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de consultas de cualquier tipo que pueden ejecutarse.
* `query_selects` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número actual de consultas SELECT ejecutadas.
* `max_query_selects` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de consultas SELECT que pueden ejecutarse.
* `query_inserts` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número actual de consultas INSERT ejecutadas.
* `max_query_inserts` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de consultas INSERT que pueden ejecutarse.
* `errors` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número actual de consultas que dieron error.
* `max_errors` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de consultas que dieron error dentro del período de tiempo especificado.
* `result_rows` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total actual de filas en el conjunto de resultados de todas las consultas dentro del período de tiempo actual.
* `max_result_rows` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total máximo permitido de filas en el conjunto de resultados de todas las consultas dentro del período de tiempo especificado.
* `result_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total actual de bytes en el conjunto de resultados de todas las consultas dentro del período de tiempo actual.
* `max_result_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total máximo permitido de bytes en el conjunto de resultados de todas las consultas dentro del período de tiempo especificado.
* `read_rows` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total actual de filas leídas durante la ejecución de todas las consultas dentro del período de tiempo actual.
* `max_read_rows` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de filas que se pueden leer durante la ejecución de todas las consultas dentro del período de tiempo especificado.
* `read_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total actual de bytes leídos durante la ejecución de todas las consultas dentro del período de tiempo actual.
* `max_read_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de bytes que se pueden leer durante la ejecución de todas las consultas dentro del período de tiempo especificado.
* `execution_time` ([Nullable(Float64)](/es/reference/data-types/index)) — Tiempo total actual (en nanosegundos) empleado en ejecutar consultas dentro del período de tiempo actual
* `max_execution_time` ([Nullable(Float64)](/es/reference/data-types/index)) — Tiempo máximo permitido (en nanosegundos) para la ejecución de todas las consultas dentro del período de tiempo especificado
* `written_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número total actual de bytes escritos durante la ejecución de todas las consultas dentro del período de tiempo actual.
* `max_written_bytes` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de bytes que se pueden escribir durante la ejecución de todas las consultas dentro del período de tiempo especificado.
* `failed_sequential_authentications` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número actual de fallos consecutivos de autenticación dentro del período de tiempo actual.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de fallos consecutivos de autenticación dentro del período de tiempo especificado.
* `queries_per_normalized_hash` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo actual de ejecuciones de una misma consulta normalizada dentro del período de tiempo actual.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/es/reference/data-types/index)) — Número máximo permitido de ejecuciones de una misma consulta normalizada dentro del período de tiempo especificado.
