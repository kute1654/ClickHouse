<div id="description">
  ## Descripción
</div>

Contiene información sobre todas las políticas de enmascaramiento definidas en el sistema.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/string)) — Nombre de una política de enmascaramiento. El formato del nombre completo es `short_name ON database.table`.
* `short_name` ([String](/es/reference/data-types/string)) — Nombre abreviado de una política de enmascaramiento. Por ejemplo, si el nombre completo es `mask_email ON mydb.mytable`, el nombre abreviado es `mask_email`.
* `database` ([String](/es/reference/data-types/string)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/string)) — Nombre de la tabla.
* `id` ([UUID](/es/reference/data-types/uuid)) — ID de la política de enmascaramiento.
* `storage` ([String](/es/reference/data-types/string)) — Nombre del directorio donde se almacena la política de enmascaramiento.
* `update_assignments` ([Nullable(String)](/es/reference/data-types/nullable)) — Asignaciones de UPDATE que definen cómo deben enmascararse los datos. Por ejemplo: `email = '***masked***', phone = '***-***-****'`.
* `where_condition` ([Nullable(String)](/es/reference/data-types/nullable)) — Condición WHERE opcional que especifica cuándo debe aplicarse el enmascaramiento.
* `priority` ([Int64](/es/reference/data-types/int-uint)) — Prioridad para aplicar varias políticas de enmascaramiento. Las políticas con mayor prioridad se aplican primero. El valor predeterminado es 0.
* `apply_to_all` ([UInt8](/es/reference/data-types/int-uint)) — Indica si la política de enmascaramiento se aplica a todos los roles y/o usuarios. 1 si es así, 0 en caso contrario.
* `apply_to_list` ([Array(String)](/es/reference/data-types/array)) — Lista de roles y/o usuarios a los que se aplica la política de enmascaramiento.
* `apply_to_except` ([Array(String)](/es/reference/data-types/array)) — La política de enmascaramiento se aplica a todos los roles y/o usuarios excepto a los enumerados. Solo se rellena cuando `apply_to_all` es 1.

{/*AUTOGENERATED_END*/}
