<div id="description">
  ## Descripción
</div>

Esta tabla del sistema contiene el historial de instantáneas de las tablas Iceberg que existen en ClickHouse. Estará vacía si no hay ninguna tabla Iceberg en ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `made_current_at` ([Nullable(DateTime64(3))](/es/reference/data-types/index)) — Fecha y hora en que esta instantánea pasó a ser la instantánea actual.
* `snapshot_id` ([UInt64](/es/reference/data-types/index)) — ID de la instantánea que se utiliza para identificar una instantánea.
* `parent_id` ([UInt64](/es/reference/data-types/index)) — ID principal de esta instantánea.
* `is_current_ancestor` ([UInt8](/es/reference/data-types/index)) — Indicador de si esta instantánea es un ancestro de la instantánea actual.

{/*AUTOGENERATED_END*/}
