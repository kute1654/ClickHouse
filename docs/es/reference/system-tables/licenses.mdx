<div id="description">
  ## Descripción
</div>

Contiene las licencias de bibliotecas de terceros ubicadas en el directorio [contrib](https://github.com/ClickHouse/ClickHouse/tree/master/contrib) del código fuente de ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `library_name` ([String](/es/reference/data-types/index)) — Nombre de la biblioteca.
* `license_type` ([String](/es/reference/data-types/index)) — Tipo de licencia — p. ej., Apache, MIT.
* `license_path` ([String](/es/reference/data-types/index)) — Ruta del archivo con el texto de la licencia.
* `license_text` ([String](/es/reference/data-types/index)) — Texto de la licencia.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT library_name, license_type, license_path FROM system.licenses LIMIT 15
```

│ brotli             │ MIT          │ /contrib/brotli/LICENSE             │

└────────────────────┴──────────────┴─────────────────────────────────────┘
