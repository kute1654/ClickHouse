<div id="description">
  ## Descripción
</div>

Esta tabla contiene una única fila con una única columna `dummy` de tipo UInt8 que contiene el valor 0.

Esta tabla se utiliza si una consulta `SELECT` no especifica la cláusula `FROM`.

Es similar a la tabla `DUAL` presente en otros DBMS.
