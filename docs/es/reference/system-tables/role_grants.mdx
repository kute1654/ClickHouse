<div id="description">
  ## Descripción
</div>

Contiene los otorgamientos de roles para usuarios y roles. Para agregar registros a esta tabla, use `GRANT role TO user`.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `user_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre de usuario.
* `role_name` ([Nullable(String)](/es/reference/data-types/index)) — Nombre del rol.
* `granted_role_name` ([String](/es/reference/data-types/index)) — Nombre del rol otorgado al rol `role_name`. Para otorgar un rol a otro, use `GRANT role1 TO role2`.
* `granted_role_id` ([UUID](/es/reference/data-types/index)) — Identificador del rol.
* `granted_role_is_default` ([UInt8](/es/reference/data-types/index)) — Indicador que muestra si `granted_role` es un rol predeterminado. Valores posibles: • 1 — `granted_role` es un rol predeterminado, • 0 — `granted_role` no es un rol predeterminado.
* `with_admin_option` ([UInt8](/es/reference/data-types/index)) — Indicador que muestra si `granted_role` es un rol con el privilegio `ADMIN OPTION`. Valores posibles: • 1 — El rol tiene el privilegio `ADMIN OPTION`. • 0 — El rol no tiene el privilegio `ADMIN OPTION`.

{/*AUTOGENERATED_END*/}
