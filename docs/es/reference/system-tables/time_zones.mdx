<div id="description">
  ## Descripción
</div>

Contiene una lista de las zonas horarias compatibles con el servidor ClickHouse. Esta lista de zonas horarias puede variar según la versión de ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/es/reference/data-types/index)) — Lista de zonas horarias admitidas.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```
