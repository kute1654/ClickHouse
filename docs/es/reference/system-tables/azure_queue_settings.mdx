<div id="description">
  ## Descripción
</div>

Contiene información sobre los ajustes de las tablas [AzureQueue](/es/reference/engines/table-engines/integrations/azure-queue).
Está disponible a partir de la versión `24.10` del servidor.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Base de datos de la tabla con el engine AzureQueue.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla con el engine AzureQueue.
* `name` ([String](/es/reference/data-types/index)) — Nombre del ajuste.
* `value` ([String](/es/reference/data-types/index)) — Valor del ajuste.
* `type` ([String](/es/reference/data-types/index)) — Tipo de ajuste (valor de cadena específico de la implementación).
* `changed` ([UInt8](/es/reference/data-types/index)) — 1 si el ajuste se definió explícitamente en la configuración o se cambió explícitamente.
* `description` ([String](/es/reference/data-types/index)) — Descripción del ajuste.
* `alterable` ([UInt8](/es/reference/data-types/index)) — Indica si el usuario actual puede cambiar el ajuste mediante ALTER TABLE MODIFY SETTING: 0 — El usuario actual no puede cambiar el ajuste, 1 — El usuario actual puede cambiar el ajuste.

{/*AUTOGENERATED_END*/}
