<div id="description">
  ## Descripción
</div>

Contiene información sobre los parámetros [graphite&#95;rollup](/es/reference/settings/server-settings/settings#graphite) que se usan en tablas con motores [*GraphiteMergeTree](/es/reference/engines/table-engines/mergetree-family/graphitemergetree).

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/es/reference/data-types/index)) — Nombre del parámetro graphite&#95;rollup.
* `rule_type` ([String](/es/reference/data-types/index)) — El tipo de regla. Valores posibles: RuleTypeAll = 0 - predeterminado, con regex, compatible con el esquema anterior; RuleTypePlain = 1 - métricas simples, con regex, compatible con el esquema anterior; RuleTypeTagged = 2 - métricas etiquetadas, con regex, compatible con el esquema anterior; RuleTypeTagList = 3 - métricas etiquetadas, con regex (convertido a RuleTypeTagged a partir de una cadena como &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](/es/reference/data-types/index)) — Un patrón para el nombre de la métrica.
* `function` ([String](/es/reference/data-types/index)) — El nombre de la función de agregación.
* `age` ([UInt64](/es/reference/data-types/index)) — La antigüedad mínima de los datos, en segundos.
* `precision` ([UInt64](/es/reference/data-types/index)) — Con qué precisión se define la antigüedad de los datos, en segundos.
* `priority` ([UInt16](/es/reference/data-types/index)) — Prioridad del patrón.
* `is_default` ([UInt8](/es/reference/data-types/index)) — Indica si el patrón es el predeterminado.
* `Tables.database` ([Array(String)](/es/reference/data-types/index)) — Array de nombres de tablas de bases de datos que usan el parámetro `config_name`.
* `Tables.table` ([Array(String)](/es/reference/data-types/index)) — Array de nombres de tablas que usan el parámetro `config_name`.
