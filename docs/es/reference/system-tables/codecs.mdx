<div id="description">
  ## Descripción
</div>

Contiene información sobre los códecs de compresión y cifrado.

Puede consultar esta tabla para obtener información sobre los códecs de compresión y cifrado disponibles

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del códec.
* `method_byte` ([UInt8](/es/reference/data-types/index)) — Byte que indica el códec en el archivo comprimido.
* `is_compression` ([UInt8](/es/reference/data-types/index)) — Verdadero si este códec comprime algo. De lo contrario, puede ser simplemente una transformación que ayuda a la compresión.
* `is_generic_compression` ([UInt8](/es/reference/data-types/index)) — El códec es un algoritmo de compresión genérico, como lz4 o zstd.
* `is_encryption` ([UInt8](/es/reference/data-types/index)) — El códec cifra.
* `is_timeseries_codec` ([UInt8](/es/reference/data-types/index)) — El códec es para series temporales de coma flotante.
* `is_experimental` ([UInt8](/es/reference/data-types/index)) — El códec es experimental.
* `description` ([String](/es/reference/data-types/index)) — Descripción general del códec.

{/*AUTOGENERATED_END*/}
