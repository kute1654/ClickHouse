<div id="description">
  ## Descripción
</div>

Esta tabla no existe si ZooKeeper no está configurado. La tabla &#39;system.zookeeper&#95;connection&#39; muestra las conexiones actuales a ZooKeeper (incluidos los ZooKeeper auxiliares). Cada fila muestra información sobre una conexión.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/string)) — Nombre del clúster de ZooKeeper.
* `host` ([String](/es/reference/data-types/string)) — El hostname/IP del nodo de ZooKeeper al que se conectó ClickHouse.
* `port` ([UIn16](/es/reference/data-types/int-uint)) — El puerto del nodo de ZooKeeper al que se conectó ClickHouse.
* `index` ([Nullable(UInt8)](/es/reference/data-types/int-uint)) — El índice del nodo de ZooKeeper al que se conectó ClickHouse. El índice proviene de la configuración de ZooKeeper. Si no hay conexión, esta columna es NULL.
* `connected_time` ([DateTime](/es/reference/data-types/datetime)) — Cuándo se estableció la conexión.
* `session_uptime_elapsed_seconds` ([UInt64](/es/reference/data-types/int-uint)) — Segundos transcurridos desde que se estableció la conexión.
* `is_expired` ([UInt8](/es/reference/data-types/int-uint)) — Indica si la conexión actual ha expirado.
* `keeper_api_version` ([UInt8](/es/reference/data-types/int-uint)) — Versión de la API de Keeper.
* `client_id` ([Int64](/es/reference/data-types/int-uint)) — ID de sesión de la conexión.
* `xid` ([Int64](/es/reference/data-types/int-uint)) — XID de la sesión actual.
* `enabled_feature_flags` ([Array(Enum16)](/es/reference/data-types/array)) — Feature flags habilitadas. Solo se aplica a ClickHouse Keeper. Los posibles valores son `FILTERED_LIST`, `MULTI_READ`, `CHECK_NOT_EXISTS`, `CREATE_IF_NOT_EXISTS`, `REMOVE_RECURSIVE`.
* `availability_zone` ([String](/es/reference/data-types/string)) — Zona de disponibilidad.
* `session_timeout_ms` ([UInt64](/es/reference/data-types/int-uint)) — Tiempo de espera de la sesión (en milisegundos).
* `last_zxid_seen` ([Int64](/es/reference/data-types/int-uint)) — Último zxid visto por la sesión actual.

{/*AUTOGENERATED_END*/}
