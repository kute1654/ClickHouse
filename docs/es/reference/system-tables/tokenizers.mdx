<div id="description">
  ## Descripción
</div>

Muestra todos los tokenizadores disponibles.
Se pueden usar en las funciones [tokens](/es/reference/functions/regular-functions/splitting-merging-functions#tokens), [hasAllTokens](/es/reference/functions/regular-functions/string-search-functions#hasAllTokens), [hasAnyTokens](/es/reference/functions/regular-functions/string-search-functions#hasAnyTokens) y en el [índice de texto](/es/reference/engines/table-engines/mergetree-family/textindexes).

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del tokenizador

{/*AUTOGENERATED_END*/}

  ## Ejemplo

```sql
SELECT * FROM system.tokenizers;
```
