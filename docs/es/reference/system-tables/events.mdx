<div id="description">
  ## Descripción
</div>

Contiene información sobre la cantidad de eventos que se han producido en el sistema. Por ejemplo, en la tabla se puede ver cuántas consultas `SELECT` se han procesado desde que se inició el servidor de ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `event` ([String](/es/reference/data-types/index)) — Nombre del evento.
* `value` ([UInt64](/es/reference/data-types/index)) — Cantidad de eventos ocurridos.
* `description` ([String](/es/reference/data-types/index)) — Descripción del evento.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.events LIMIT 5
```

2. Uso de otro método de compresión.
