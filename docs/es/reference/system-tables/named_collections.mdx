<div id="description">
  ## Descripción
</div>

Contiene una lista de todas las colecciones con nombre que se crearon mediante una consulta SQL o se analizaron a partir del archivo de configuración.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre de la colección.
* `collection` ([Map(String, String)](/es/reference/data-types/index)) — Parámetros internos de la colección.
* `source` ([String](/es/reference/data-types/index)) — Fuente de la colección con nombre.
* `create_query` ([String](/es/reference/data-types/index)) — Consulta CREATE de la colección con nombre.

{/*AUTOGENERATED_END*/}
