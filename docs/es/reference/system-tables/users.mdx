<div id="description">
  ## Descripción
</div>

Contiene una lista de [cuentas de usuario](/es/concepts/features/security/access-rights#user-account-management) configuradas en el servidor.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre de usuario.
* `id` ([UUID](/es/reference/data-types/index)) — ID de usuario.
* `storage` ([String](/es/reference/data-types/index)) — Ruta al almacenamiento de usuarios. Se configura en el parámetro access&#95;control&#95;path.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/es/reference/data-types/index)) — Muestra los tipos de autenticación. Existen varias formas de identificar al usuario: sin contraseña, con contraseña en texto sin formato, con contraseña codificada con SHA256, con contraseña codificada con doble SHA-1 o con contraseña codificada con bcrypt.
* `auth_params` ([Array(String)](/es/reference/data-types/index)) — Parámetros de autenticación en formato JSON según el auth&#95;type.
* `valid_until` ([Array(DateTime)](/es/reference/data-types/index)) — Fecha y hora de vencimiento de las credenciales del usuario.
* `host_ip` ([Array(String)](/es/reference/data-types/index)) — Direcciones IP de los hosts autorizados para conectarse al servidor de ClickHouse.
* `host_names` ([Array(String)](/es/reference/data-types/index)) — Nombres de hosts autorizados para conectarse al servidor de ClickHouse.
* `host_names_regexp` ([Array(String)](/es/reference/data-types/index)) — Expresión regular para los nombres de hosts autorizados para conectarse al servidor de ClickHouse.
* `host_names_like` ([Array(String)](/es/reference/data-types/index)) — Nombres de hosts autorizados para conectarse al servidor de ClickHouse, establecidos mediante el predicado LIKE.
* `default_roles_all` ([UInt8](/es/reference/data-types/index)) — Muestra que todos los roles concedidos se establecen de forma predeterminada para el usuario.
* `default_roles_list` ([Array(String)](/es/reference/data-types/index)) — Lista de roles concedidos establecidos de forma predeterminada.
* `default_roles_except` ([Array(String)](/es/reference/data-types/index)) — Todos los roles concedidos se establecen de forma predeterminada, excepto los enumerados.
* `grantees_any` ([UInt8](/es/reference/data-types/index)) — El indicador que señala si un usuario con cualquier opción de concesión puede concederla a cualquiera.
* `grantees_list` ([Array(String)](/es/reference/data-types/index)) — La lista de usuarios o roles a los que este usuario puede conceder opciones de concesión.
* `grantees_except` ([Array(String)](/es/reference/data-types/index)) — La lista de usuarios o roles a los que este usuario tiene prohibido conceder opciones de concesión.
* `default_database` ([String](/es/reference/data-types/index)) — El nombre de la base de datos predeterminada de este usuario.
