<div id="description">
  ## Descripción
</div>

Contiene información sobre las bases de datos disponibles para el usuario actual.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `engine` ([String](/es/reference/data-types/index)) — Motor de la base de datos.
* `data_path` ([String](/es/reference/data-types/index)) — Ruta de los datos.
* `metadata_path` ([String](/es/reference/data-types/index)) — Ruta de los metadatos.
* `uuid` ([UUID](/es/reference/data-types/index)) — UUID de la base de datos.
* `engine_full` ([String](/es/reference/data-types/index)) — Parámetros del motor de la base de datos.
* `comment` ([String](/es/reference/data-types/index)) — Comentario de la base de datos.
* `is_external` ([UInt8](/es/reference/data-types/index)) — La base de datos es externa (es decir, PostgreSQL/DataLakeCatalog).

{/*AUTOGENERATED_END*/}

La columna `name` de esta tabla del sistema se utiliza para implementar la consulta `SHOW DATABASES`.

<div id="example">
  ## Ejemplo
</div>

Cree una base de datos.

```sql title="Query"
CREATE DATABASE test;
```

Compruebe todas las bases de datos a las que tiene acceso el usuario.

```sql title="Query"
SELECT * FROM system.databases;
```
