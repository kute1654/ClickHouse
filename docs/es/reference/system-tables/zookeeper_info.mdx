<div id="description">
  ## Descripción
</div>

Esta tabla muestra información de introspección combinada sobre ZooKeeper, y los nodos se toman de la configuración.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_cluster_name` ([String](/es/reference/data-types/string)) — nombre del clúster de ZooKeeper.
* `host` ([String](/es/reference/data-types/string)) — El nombre de host o la IP del nodo de ZooKeeper al que se conectó ClickHouse.
* `port` ([UInt16](/es/reference/data-types/int-uint)) — El puerto del nodo de ZooKeeper al que se conectó ClickHouse.
* `index` ([Nullable(UInt8)](/es/reference/data-types/int-uint)) — El índice del nodo de ZooKeeper al que se conectó ClickHouse. El índice procede de la configuración de ZooKeeper. Si no está conectado, esta columna es NULL.
* `is_connected` ([Nullable(UInt8)](/es/reference/data-types/int-uint)) — Indica si ZooKeeper está conectado o no.
* `is_readonly` ([UInt8](/es/reference/data-types/int-uint)) — Indica si es de solo lectura.
* `version` ([String](/es/reference/data-types/string)) — La versión de ZooKeeper.
* `avg_latency` ([UInt64](/es/reference/data-types/int-uint)) — La latencia media.
* `max_latency` ([UInt64](/es/reference/data-types/int-uint)) — La latencia máxima.
* `min_latency` ([UInt64](/es/reference/data-types/int-uint)) — La latencia mínima.
* `packets_received` ([UInt64](/es/reference/data-types/int-uint)) — Número de paquetes recibidos.
* `packets_sent` ([UInt64](/es/reference/data-types/int-uint)) — El número de paquetes enviados.
* `outstanding_requests` ([UInt64](/es/reference/data-types/int-uint)) — El número de solicitudes pendientes.
* `server_state` ([String](/es/reference/data-types/string)) — Estado del servidor.
* `is_leader` ([UInt8](/es/reference/data-types/int-uint)) — Indica si es el líder de ZooKeeper.
* `znode_count` ([UInt64](/es/reference/data-types/int-uint)) — El número de znodes.
* `watch_count` ([UInt64](/es/reference/data-types/int-uint)) — El número de watches.
* `ephemerals_count` ([UInt64](/es/reference/data-types/int-uint)) — El recuento de valores efímeros.
* `approximate_data_size` ([UInt64](/es/reference/data-types/int-uint)) — El tamaño aproximado de los datos.
* `followers` ([UInt64](/es/reference/data-types/int-uint)) — Los seguidores del líder. Este campo solo lo expone el líder.
* `synced_followers` ([UInt64](/es/reference/data-types/int-uint)) — Los seguidores sincronizados del líder. Este campo solo está expuesto por el líder.
* `pending_syncs` ([UInt64](/es/reference/data-types/int-uint)) — Las sincronizaciones pendientes del líder. Este campo solo está expuesto por el líder.
* `open_file_descriptor_count` ([UInt64](/es/reference/data-types/int-uint)) — El número de descriptores de archivo abiertos. Solo está disponible en plataformas Unix.
* `max_file_descriptor_count` ([UInt64](/es/reference/data-types/int-uint)) — El número máximo de descriptores de archivo. Solo está disponible en plataformas Unix.
* `connections` ([UInt64](/es/reference/data-types/int-uint)) — Las conexiones a ZooKeeper.
* `outstanding` ([UInt64](/es/reference/data-types/int-uint)) — Solicitudes pendientes de ZooKeeper.
* `zxid` ([UInt64](/es/reference/data-types/int-uint)) — El zxid de ZooKeeper.
* `node_count` ([UInt64](/es/reference/data-types/int-uint)) — El número de nodos de ZooKeeper.
* `snapshot_dir_size` ([UInt64](/es/reference/data-types/int-uint)) — El tamaño del directorio de snapshots de ZooKeeper.
* `log_dir_size` ([UInt64](/es/reference/data-types/int-uint)) — El tamaño del directorio de logs de ZooKeeper.
* `first_log_idx` ([UInt64](/es/reference/data-types/int-uint)) — El primer índice del registro de ZooKeeper.
* `first_log_term` ([UInt64](/es/reference/data-types/int-uint)) — El primer término del log de ZooKeeper.
* `last_log_idx` ([UInt64](/es/reference/data-types/int-uint)) — El último índice del log de ZooKeeper.
* `last_log_term` ([UInt64](/es/reference/data-types/int-uint)) — El último término del log de ZooKeeper.
* `last_committed_idx` ([UInt64](/es/reference/data-types/int-uint)) — El último índice confirmado en ZooKeeper.
* `leader_committed_log_idx` ([UInt64](/es/reference/data-types/int-uint)) — El índice de log confirmado por el líder de ZooKeeper.
* `target_committed_log_idx` ([UInt64](/es/reference/data-types/int-uint)) — El índice del log confirmado de destino en ZooKeeper.
* `last_snapshot_idx` ([UInt64](/es/reference/data-types/int-uint)) — El índice de la última instantánea en ZooKeeper.
