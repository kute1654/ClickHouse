<div id="description">
  ## Descripción
</div>

Esta tabla contiene métricas de histograma que se pueden calcular de forma instantánea y exportar en formato Prometheus. Siempre está actualizada. Reemplaza a `system.latency_log`, que está obsoleto.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/es/reference/data-types/index)) — Nombre de la métrica.
* `value` ([Float64](/es/reference/data-types/index)) — Valor de la métrica.
* `description` ([String](/es/reference/data-types/index)) — Descripción de la métrica.
* `labels` ([Map(String, String)](/es/reference/data-types/index)) — Etiquetas de la métrica.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

Puede usar una consulta como esta para exportar todas las métricas de histograma en formato Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
