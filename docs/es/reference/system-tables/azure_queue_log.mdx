Contiene registros con información sobre los archivos procesados por el motor AzureQueue.

Se puede truncar o eliminar esta tabla de forma segura en cualquier momento.

Columnas:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Nombre de host
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha del evento en el que se escribió esta fila de log
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — Hora del evento en el que se escribió esta fila de log
* `database` ([String](/es/reference/data-types/string)) — El nombre de la base de datos donde reside la tabla de cola (`S3Queue` o `AzureQueue`).
* `table` ([String](/es/reference/data-types/string)) — El nombre de la tabla de cola (`S3Queue` o `AzureQueue`).
* `uuid` ([String](/es/reference/data-types/string)) — El UUID de la tabla de cola (`S3Queue` o `AzureQueue`).
* `file_name` ([String](/es/reference/data-types/string)) — Nombre del archivo que se está procesando.
* `rows_processed` ([UInt64](/es/reference/data-types/int-uint)) — Número de filas procesadas.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/es/reference/data-types/enum)) — Estado del archivo en procesamiento.
* `processing_start_time` ([Nullable(DateTime)](/es/reference/data-types/nullable)) — Hora de inicio del procesamiento del archivo.
* `processing_end_time` ([Nullable(DateTime)](/es/reference/data-types/nullable)) — Hora de finalización del procesamiento del archivo.
* `exception` ([String](/es/reference/data-types/string)) — Mensaje de la excepción, si se produjo.
* `commit_id` ([UInt64](/es/reference/data-types/int-uint)) — ID de la transacción en la que se confirmó este archivo.
* `commit_time` ([DateTime](/es/reference/data-types/datetime)) — Hora de confirmación del archivo en Keeper (ya sea como fallido o procesado).
* `transaction_start_time` ([DateTime](/es/reference/data-types/datetime)) — Hora en que comenzó toda la transacción de procesamiento.
* `get_object_time_ms` ([UInt64](/es/reference/data-types/int-uint)) — Tiempo que tomó encontrar el objeto en el almacenamiento de objetos.

{/*AUTOGENERATED_END*/}
