<div id="description">
  ## Descripción
</div>

Contiene información sobre la configuración de las tablas [S3Queue](/es/reference/engines/table-engines/integrations/s3queue). Disponible a partir de la versión `24.10` del servidor.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Base de datos de la tabla con el motor S3Queue.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla con el motor S3Queue.
* `name` ([String](/es/reference/data-types/index)) — Nombre de la configuración.
* `value` ([String](/es/reference/data-types/index)) — Valor de la configuración.
* `type` ([String](/es/reference/data-types/index)) — Tipo de configuración (valor de cadena específico de la implementación).
* `changed` ([UInt8](/es/reference/data-types/index)) — 1 si la configuración se definió explícitamente en el archivo de configuración o se modificó explícitamente.
* `description` ([String](/es/reference/data-types/index)) — Descripción de la configuración.
* `alterable` ([UInt8](/es/reference/data-types/index)) — Indica si el usuario actual puede cambiar la configuración mediante ALTER TABLE MODIFY SETTING: 0 — El usuario actual no puede cambiar la configuración, 1 — El usuario actual puede cambiar la configuración.

{/*AUTOGENERATED_END*/}
