<div id="description">
  ## Descripción
</div>

Contiene las propiedades de los perfiles de configuración definidos.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del perfil de configuración.
* `id` ([UUID](/es/reference/data-types/index)) — ID del perfil de configuración.
* `storage` ([String](/es/reference/data-types/index)) — Ruta al almacenamiento de los perfiles de configuración. Se configura en el parámetro `access_control_path`.
* `num_elements` ([UInt64](/es/reference/data-types/index)) — Número de elementos de este perfil en la tabla `system.settings_profile_elements`.
* `apply_to_all` ([UInt8](/es/reference/data-types/index)) — Indica que el perfil de configuración está establecido para todos los roles y/o usuarios.
* `apply_to_list` ([Array(String)](/es/reference/data-types/index)) — Lista de roles y/o usuarios a los que se aplica el perfil de configuración.
* `apply_to_except` ([Array(String)](/es/reference/data-types/index)) — El perfil de configuración se aplica a todos los roles y/o usuarios, excepto a los que figuran en la lista.

{/*AUTOGENERATED_END*/}
