<div id="description">
  ## Descripción
</div>

Contiene información sobre las proyecciones existentes de todas las tablas.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `name` ([String](/es/reference/data-types/index)) — Nombre de la proyección.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](/es/reference/data-types/index)) — Tipo de proyección.
* `sorting_key` ([Array(String)](/es/reference/data-types/index)) — Clave de ordenación de la proyección.
* `query` ([String](/es/reference/data-types/index)) — Consulta de la proyección.
* `settings` ([Map(String, String)](/es/reference/data-types/index)) — Configuración de la proyección.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```

Row 1:

Row 2:
