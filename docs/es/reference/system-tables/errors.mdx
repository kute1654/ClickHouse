<div id="description">
  ## Descripción
</div>

Contiene los códigos de error y el número de veces que se han activado.

Para mostrar todos los códigos de error posibles, incluidos los que no se hayan activado, establezca [system&#95;events&#95;show&#95;zero&#95;values](/es/reference/settings/session-settings#system_events_show_zero_values) en 1.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del error (errorCodeToName).
* `code` ([Int32](/es/reference/data-types/index)) — Número de código del error.
* `value` ([UInt64](/es/reference/data-types/index)) — Número de veces que se produjo este error.
* `last_error_time` ([DateTime](/es/reference/data-types/index)) — Hora en la que se produjo el último error.
* `last_error_message` ([String](/es/reference/data-types/index)) — Mensaje del último error.
* `last_error_format_string` ([String](/es/reference/data-types/index)) — Cadena de formato del último error.
* `last_error_trace` ([Array(UInt64)](/es/reference/data-types/index)) — Una traza de pila que representa una lista de direcciones físicas donde se almacenan los métodos invocados.
* `remote` ([UInt8](/es/reference/data-types/index)) — Excepción remota (es decir, recibida durante una de las consultas distribuidas).
* `query_id` ([String](/es/reference/data-types/index)) — ID de una consulta que provocó un error (si está disponible).

{/*AUTOGENERATED_END*/}

<Note>
  Los contadores de algunos errores pueden aumentar durante la ejecución correcta de consultas. No se recomienda usar esta tabla para tareas de monitorización del servidor, a menos que esté seguro de que el error correspondiente no pueda ser un falso positivo.
</Note>

<div id="example">
  ## Ejemplo
</div>

```sql title="Query"
SELECT name, code, value
FROM system.errors
WHERE value > 0
ORDER BY code ASC
LIMIT 1

┌─name─────────────┬─code─┬─value─┐
│ CANNOT_OPEN_FILE │   76 │     1 │
└──────────────────┴──────┴───────┘
```

```sql title="Response"
WITH arrayMap(x -> demangle(addressToSymbol(x)), last_error_trace) AS all
SELECT name, arrayStringConcat(all, '\n') AS res
FROM system.errors
LIMIT 1
SETTINGS allow_introspection_functions=1\G
```
