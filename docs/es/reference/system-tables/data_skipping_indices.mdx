<div id="description">
  ## Descripción
</div>

Contiene información sobre los índices de omisión de datos presentes en todas las tablas.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `name` ([String](/es/reference/data-types/index)) — Nombre del índice.
* `type` ([String](/es/reference/data-types/index)) — Tipo de índice.
* `type_full` ([String](/es/reference/data-types/index)) — Expresión del tipo de índice de la sentencia CREATE.
* `expr` ([String](/es/reference/data-types/index)) — Expresión para calcular el índice.
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/es/reference/data-types/index)) — Indica si el índice se creó implícitamente (mediante add&#95;minmax&#95;index&#95;for&#95;numeric&#95;columns o similar)
* `granularity` ([UInt64](/es/reference/data-types/index)) — El número de gránulos del bloque.
* `data_compressed_bytes` ([UInt64](/es/reference/data-types/index)) — El tamaño de los datos comprimidos, en bytes.
* `data_uncompressed_bytes` ([UInt64](/es/reference/data-types/index)) — El tamaño de los datos descomprimidos, en bytes.
* `marks_bytes` ([UInt64](/es/reference/data-types/index)) — El tamaño de las marcas, en bytes.

{/*AUTOGENERATED_END*/}
