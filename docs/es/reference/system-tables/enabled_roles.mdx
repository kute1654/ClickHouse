<div id="description">
  ## Descripción
</div>

Contiene todos los roles activos en este momento, incluido el rol actual del usuario actual y los roles concedidos al rol actual.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/es/reference/data-types/index)) — Nombre del rol.
* `with_admin_option` ([UInt8](/es/reference/data-types/index)) — 1 si el rol tiene el privilegio ADMIN OPTION.
* `is_current` ([UInt8](/es/reference/data-types/index)) — Indicador de si `enabled_role` es un rol actual del usuario actual.
* `is_default` ([UInt8](/es/reference/data-types/index)) — Indicador de si `enabled_role` es un rol predeterminado.

{/*AUTOGENERATED_END*/}
