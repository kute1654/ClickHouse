<div id="description">
  ## Descripción
</div>

Contiene información sobre las tablas en las que se ha ejecutado `drop table`, pero cuya limpieza de datos aún no se ha realizado.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `index` ([UInt32](/es/reference/data-types/index)) — Índice en la cola marked&#95;dropped&#95;tables.
* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `uuid` ([UUID](/es/reference/data-types/index)) — UUID de la tabla.
* `engine` ([String](/es/reference/data-types/index)) — Nombre del motor de la tabla.
* `metadata_dropped_path` ([String](/es/reference/data-types/index)) — Ruta del archivo de metadatos de la tabla en el directorio metadata&#95;dropped.
* `table_dropped_time` ([DateTime](/es/reference/data-types/index)) — Momento en que está programado el siguiente intento de eliminar los datos de la tabla. Normalmente, es la hora en que se eliminó la tabla más `database_atomic_delay_before_drop_table_sec`.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

El siguiente ejemplo muestra cómo obtener información sobre `dropped_tables`.

```sql
SELECT *
FROM system.dropped_tables\G
```

```text
Row 1:
──────
index:                 0
database:              default
table:                 test
uuid:                  03141bb2-e97a-4d7c-a172-95cc066bb3bd

metadata_dropped_path: /data/ClickHouse/build/programs/data/metadata_dropped/default.test.03141bb2-e97a-4d7c-a172-95cc066bb3bd.sql
table_dropped_time:    2023-03-16 23:43:31
```
