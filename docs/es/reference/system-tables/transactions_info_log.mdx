Contiene información sobre todas las transacciones ejecutadas en el servidor actual.

Es seguro truncar o eliminar esta tabla en cualquier momento.

Columnas:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — El nombre de host donde se ejecutó la transacción.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/es/reference/data-types/enum)) — El tipo de transacción. Valores posibles: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha del evento.
* `event_time` ([DateTime64(6)](/es/reference/data-types/datetime64)) — Hora del evento
* `thread_id` ([UInt64](/es/reference/data-types/int-uint)) — El identificador de un hilo.
* `query_id` ([String](/es/reference/data-types/string)) — El ID de una consulta ejecutada en el ámbito de una transacción.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/es/reference/data-types/tuple)) — El identificador de una transacción.
* `tid_hash` ([UInt64](/es/reference/data-types/int-uint)) — El hash del identificador.
* `csn` ([UInt64](/es/reference/data-types/int-uint)) — El número de secuencia de commit
* `database` ([String](/es/reference/data-types/string)) — El nombre de la base de datos en la que se ejecutó la transacción.
* `table` ([String](/es/reference/data-types/string)) — El nombre de la tabla en la que se ejecutó la transacción.
* `uuid` ([UUID](/es/reference/data-types/uuid)) — El UUID de la tabla en la que se ejecutó la transacción.
* `part` ([String](/es/reference/data-types/string)) — El nombre de la parte que participó en la transacción.

{/*AUTOGENERATED_END*/}
