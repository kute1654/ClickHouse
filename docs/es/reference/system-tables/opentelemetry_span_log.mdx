<div id="description">
  ## Descripción
</div>

Contiene información sobre los [spans de traza](https://opentracing.io/docs/overview/spans/) de las consultas ejecutadas.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — El hostname donde se registró este span.
* `trace_id` ([UUID](/es/reference/data-types/uuid)) — ID de la traza de la consulta ejecutada.
* `span_id` ([UInt64](/es/reference/data-types/int-uint)) — ID del span de traza.
* `parent_span_id` ([UInt64](/es/reference/data-types/int-uint)) — ID del span de traza principal.
* `operation_name` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — El nombre de la operación.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/es/reference/data-types/enum)) — El SpanKind del span. INTERNAL — Indica que el span representa una operación interna dentro de una aplicación. SERVER — Indica que el span cubre el procesamiento, del lado del servidor, de una RPC síncrona u otra solicitud remota. CLIENT — Indica que el span describe una solicitud a algún servicio remoto. PRODUCER — Indica que el span describe a los iniciadores de una solicitud asíncrona. Este span principal a menudo terminará antes que el span hijo CONSUMER correspondiente, posiblemente incluso antes de que comience el span hijo. CONSUMER - Indica que el span describe un hijo de una solicitud PRODUCER asíncrona.
* `start_time_us` ([UInt64](/es/reference/data-types/int-uint)) — La hora de inicio del span de traza (en microsegundos).
* `finish_time_us` ([UInt64](/es/reference/data-types/int-uint)) — La hora de finalización del span de traza (en microsegundos).
* `finish_date` ([Date](/es/reference/data-types/date)) — La fecha de finalización del span de traza.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/es/reference/data-types/enum)) — El código de estado del span.
* `status_message` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Mensaje de error.
* `attribute` ([Map(LowCardinality(String), String)](/es/reference/data-types/map)) — Atributos en función del span de traza. Se completan de acuerdo con las recomendaciones del estándar OpenTelemetry.

**Alias:**

* `attribute.names` — Alias de `mapKeys(attribute)`.
* `attribute.values` — Alias de `mapValues(attribute)`.

{/*AUTOGENERATED_END*/}
