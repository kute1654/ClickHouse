<div id="description">
  ## Descripción
</div>

Contiene información sobre los registros DNS en caché.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/es/reference/data-types/index)) — Nombre de host.
* `ip_address` ([String](/es/reference/data-types/index)) — Dirección IP.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/es/reference/data-types/index)) — Familia de direcciones IP.
* `cached_at` ([DateTime](/es/reference/data-types/index)) — Marca temporal en la que se almacenó en caché el registro.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
