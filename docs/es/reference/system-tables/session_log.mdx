<div id="description">
  ## Descripción
</div>

Contiene información sobre todos los eventos de inicio y cierre de sesión, tanto exitosos como fallidos.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Hostname del servidor que ejecuta la consulta.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/es/reference/data-types/enum)) — Resultado del inicio/cierre de sesión. Posibles valores: LoginFailure — Error de inicio de sesión. LoginSuccess — Inicio de sesión correcto. Logout — Cierre de sesión del sistema.
* `auth_id` ([UUID](/es/reference/data-types/uuid)) — ID de autenticación, un UUID que se genera automáticamente cada vez que un usuario inicia sesión.
* `session_id` ([String](/es/reference/data-types/string)) — ID de sesión que el cliente envía a través de la interfaz HTTP.
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha de inicio/cierre de sesión.
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — Hora de inicio/cierre de sesión.
* `event_time_microseconds` ([DateTime64(6)](/es/reference/data-types/datetime64)) — Hora de inicio del proceso de inicio/cierre de sesión con precisión de microsegundos.
* `user` ([Nullable(String)](/es/reference/data-types/nullable)) — Nombre de usuario.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/es/reference/data-types/nullable)) — Tipo de autenticación.
* `profiles` ([Array(LowCardinality(String))](/es/reference/data-types/array)) — Lista de perfiles establecidos para todos los roles y/o usuarios.
* `roles` ([Array(LowCardinality(String))](/es/reference/data-types/array)) — Lista de roles a los que se aplica el perfil.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/es/reference/data-types/array)) — Configuración que se modificó cuando el cliente inició/cerró sesión.
* `client_address` ([IPv6](/es/reference/data-types/ipv6)) — Dirección IP utilizada para iniciar/cerrar sesión.
* `client_port` ([UInt16](/es/reference/data-types/int-uint)) — Puerto del cliente utilizado para iniciar/cerrar sesión.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/es/reference/data-types/enum)) — Interfaz desde la que se inició la sesión.
* `client_hostname` ([String](/es/reference/data-types/string)) — Hostname de la máquina cliente en la que se ejecuta `clickhouse-client` u otro cliente TCP.
* `client_name` ([String](/es/reference/data-types/string)) — Nombre de `clickhouse-client` u otro cliente TCP.
* `client_revision` ([UInt32](/es/reference/data-types/int-uint)) — Revisión de `clickhouse-client` u otro cliente TCP.
* `client_version_major` ([UInt32](/es/reference/data-types/int-uint)) — Versión principal de `clickhouse-client` u otro cliente TCP.
* `client_version_minor` ([UInt32](/es/reference/data-types/int-uint)) — Versión secundaria de `clickhouse-client` u otro cliente TCP.
* `client_version_patch` ([UInt32](/es/reference/data-types/int-uint)) — Componente de parche de la versión de `clickhouse-client` u otro cliente TCP.
* `failure_reason` ([String](/es/reference/data-types/string)) — Mensaje de excepción que contiene el motivo del fallo de inicio/cierre de sesión.
* `certificate_subjects` ([Array(LowCardinality(String))](/es/reference/data-types/array)) — Lista de sujetos (Common Name y Subject Alternative Names) del certificado TLS del cliente presentado en la conexión, con el formato &#39;CN:...&#39; / &#39;SAN:...&#39;. Vacío si no se presentó ningún certificado.
* `certificate_serial` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Número de serie del certificado TLS del cliente. Vacío si no se presentó ningún certificado.
* `certificate_issuer` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Emisor del certificado TLS del cliente. Vacío si no se presentó ningún certificado.
* `certificate_not_before` ([Nullable(DateTime64)](/es/reference/data-types/nullable)) — Hora a partir de la cual el certificado TLS del cliente es válido. NULL si no se presentó ningún certificado.
* `certificate_not_after` ([Nullable(DateTime64)](/es/reference/data-types/nullable)) — Hora a partir de la cual el certificado TLS del cliente caduca. NULL si no se presentó ningún certificado.

{/*AUTOGENERATED_END*/}
