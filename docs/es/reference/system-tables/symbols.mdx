<div id="description">
  ## Descripción
</div>

Contiene información para la introspección del binario `clickhouse`. Para acceder a ella, se requiere el privilegio de introspección.
Esta tabla solo es útil para expertos en C++ y los ingenieros de ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/es/reference/data-types/index)) — Nombre del símbolo en el binario. Está con name mangling. Puede usar demangle(symbol) para obtener un nombre legible.
* `address_begin` ([UInt64](/es/reference/data-types/index)) — Dirección de inicio del símbolo en el binario.
* `address_end` ([UInt64](/es/reference/data-types/index)) — Dirección final del símbolo en el binario.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
