<div id="description">
  ## Descripción
</div>

Contiene información sobre las opciones de compilación del servidor ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre de la opción de compilación.
* `value` ([String](/es/reference/data-types/index)) — Valor de la opción de compilación.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
