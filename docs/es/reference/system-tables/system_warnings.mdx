<div id="description">
  ## Descripción
</div>

Esta tabla muestra advertencias sobre el servidor de ClickHouse.
Las advertencias del mismo tipo se combinan en una sola.
Por ejemplo, si el número N de bases de datos adjuntas supera un umbral configurable T, se muestra una única entrada con el valor actual N en lugar de N entradas independientes.
Si el valor actual desciende por debajo del umbral, la entrada se elimina de la tabla.

La tabla puede configurarse con estos ajustes:

* [max&#95;table&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_table_num_to_warn)
* [max&#95;database&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_database_num_to_warn)
* [max&#95;dictionary&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_dictionary_num_to_warn)
* [max&#95;view&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_view_num_to_warn)
* [max&#95;part&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_part_num_to_warn)
* [max&#95;pending&#95;mutations&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_pending_mutations_to_warn)
* [max&#95;pending&#95;mutations&#95;execution&#95;time&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_pending_mutations_execution_time_to_warn)
* [max&#95;named&#95;collection&#95;num&#95;to&#95;warn](/es/reference/settings/server-settings/settings#max_named_collection_num_to_warn)
* [resource&#95;overload&#95;warnings](/es/concepts/features/configuration/settings/server-overload#resource-overload-warnings)

<div id="columns">
  ## Columnas
</div>

* `message` ([String](/es/reference/data-types/string)) — Mensaje de advertencia.
* `message_format_string` ([LowCardinality(String)](/es/reference/data-types/string)) — La cadena de formato usada para formatear el mensaje.

<div id="example">
  ## Ejemplo
</div>

```sql title="Query"
 SELECT * FROM system.warnings LIMIT 2 \G;
```

```text title="Response"
Row 1:
──────

──────

message_format_string: The number of attached databases is more than {}.
```
