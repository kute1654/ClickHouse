<div id="description">
  ## Descripción
</div>

Contiene la lista de motores de base de datos admitidos por el servidor.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — El nombre del motor de base de datos.
* `description` ([String](/es/reference/data-types/index)) — Una descripción general de la función del motor de base de datos.
* `syntax` ([String](/es/reference/data-types/index)) — Cómo se especifica el motor de base de datos en la cláusula ENGINE de una consulta CREATE DATABASE.
* `examples` ([String](/es/reference/data-types/index)) — Ejemplos de uso.
* `introduced_in` ([String](/es/reference/data-types/index)) — La versión de ClickHouse en la que se introdujo por primera vez el motor de base de datos, en el formato major.minor.
* `related` ([Array(String)](/es/reference/data-types/index)) — Los nombres de los motores de base de datos relacionados.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```
