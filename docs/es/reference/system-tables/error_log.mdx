<div id="description">
  ## Descripción
</div>

Contiene el historial de los valores de error de la tabla `system.errors`, que se vuelcan periódicamente a disco.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Nombre de host del servidor que ejecuta la consulta.
* `event_date` ([Date](/es/reference/data-types/date)) — Fecha del evento.
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — Hora del evento.
* `code` ([Int32](/es/reference/data-types/int-uint)) — Código de error.
* `error` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — Nombre del error.
* `value` ([UInt64](/es/reference/data-types/int-uint)) — Número de errores ocurridos en el intervalo de tiempo.
* `remote` ([UInt8](/es/reference/data-types/int-uint)) — Excepción remota (es decir, recibida durante una de las consultas distribuidas).
* `last_error_time` ([DateTime](/es/reference/data-types/datetime)) — Hora en la que ocurrió el último error.
* `last_error_message` ([String](/es/reference/data-types/string)) — Mensaje del último error.
* `last_error_query_id` ([String](/es/reference/data-types/string)) — ID de la consulta que causó el último error (si está disponible).
* `last_error_trace` ([Array(UInt64)](/es/reference/data-types/array)) — Una traza de pila que representa una lista de direcciones físicas donde se almacenan los métodos invocados.

{/*AUTOGENERATED_END*/}
