<div id="description">
  ## Descripción
</div>

Contiene una lista de todos los ajustes específicos del motor ReplicatedMergeTree, sus valores actuales y predeterminados, junto con sus descripciones. Puede cambiar cualquiera de ellas en la sección SETTINGS de la consulta CREATE.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del ajuste.
* `value` ([String](/es/reference/data-types/index)) — Valor del ajuste.
* `default` ([String](/es/reference/data-types/index)) — Valor predeterminado del ajuste.
* `changed` ([UInt8](/es/reference/data-types/index)) — 1 si el ajuste se definió explícitamente en la configuración o se modificó explícitamente.
* `description` ([String](/es/reference/data-types/index)) — Descripción del ajuste.
* `min` ([Nullable(String)](/es/reference/data-types/index)) — Valor mínimo del ajuste, si se ha establecido alguno mediante restricciones. Si el ajuste no tiene valor mínimo, contiene NULL.
* `max` ([Nullable(String)](/es/reference/data-types/index)) — Valor máximo del ajuste, si se ha establecido alguno mediante restricciones. Si el ajuste no tiene valor máximo, contiene NULL.
* `disallowed_values` ([Array(String)](/es/reference/data-types/index)) — Lista de valores no permitidos
* `readonly` ([UInt8](/es/reference/data-types/index)) — Indica si el usuario actual puede cambiar el ajuste: 0 — El usuario actual puede cambiar el ajuste, 1 — El usuario actual no puede cambiar el ajuste.
* `type` ([String](/es/reference/data-types/index)) — Tipo de ajuste (valor de cadena específico de la implementación).
* `is_obsolete` ([UInt8](/es/reference/data-types/index)) — Indica si un ajuste es obsoleto.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/es/reference/data-types/index)) —
  Nivel de soporte de esta funcionalidad. Las funcionalidades de ClickHouse se organizan en niveles, que varían según el estado actual de su
  desarrollo y las expectativas que cabe tener al usarlas:

- PRODUCTION: La funcionalidad es estable, su uso es seguro y no presenta problemas al interactuar con otras funcionalidades PRODUCTION.
- BETA: La funcionalidad es estable y segura. Se desconoce el resultado de usarla junto con otras funcionalidades y no se garantiza su corrección. Se agradecen las pruebas y los informes.
- EXPERIMENTAL: La funcionalidad está en desarrollo. Está pensada únicamente para desarrolladores y entusiastas de ClickHouse. Puede funcionar o no, y podría eliminarse en cualquier momento.
- OBSOLETE: Ya no se admite. O bien ya se ha eliminado o se eliminará en futuras releases.

{/*AUTOGENERATED_END*/}
