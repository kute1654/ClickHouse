<div id="description">
  ## Descripción
</div>

Contiene información sobre las columnas de todas las tablas.

Puede usar esta tabla para obtener información similar a la de la consulta [DESCRIBE TABLE](/es/reference/statements/describe-table), pero para varias tablas a la vez.

Las columnas de las [tablas temporales](/es/reference/statements/create/table#temporary-tables) solo son visibles en `system.columns` en las sesiones en las que se han creado. Se muestran con el campo `database` vacío.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `name` ([String](/es/reference/data-types/index)) — Nombre de la columna.
* `type` ([String](/es/reference/data-types/index)) — Tipo de la columna.
* `position` ([UInt64](/es/reference/data-types/index)) — Posición ordinal de una columna en una tabla, a partir de 1.
* `default_kind` ([String](/es/reference/data-types/index)) — Tipo de expresión (DEFAULT, MATERIALIZED, ALIAS) para el valor predeterminado, o una cadena vacía si no está definido.
* `default_expression` ([String](/es/reference/data-types/index)) — Expresión del valor predeterminado, o una cadena vacía si no está definida.
* `data_compressed_bytes` ([UInt64](/es/reference/data-types/index)) — Tamaño de los datos comprimidos, en bytes.
* `data_uncompressed_bytes` ([UInt64](/es/reference/data-types/index)) — Tamaño de los datos descomprimidos, en bytes.
* `marks_bytes` ([UInt64](/es/reference/data-types/index)) — Tamaño de las marcas, en bytes.
* `comment` ([String](/es/reference/data-types/index)) — Comentario de la columna, o una cadena vacía si no está definido.
* `is_in_partition_key` ([UInt8](/es/reference/data-types/index)) — Indicador de si la columna está en la expresión de partición.
* `is_in_sorting_key` ([UInt8](/es/reference/data-types/index)) — Indicador de si la columna está en la expresión de la clave de ordenación.
* `is_in_primary_key` ([UInt8](/es/reference/data-types/index)) — Indicador de si la columna está en la expresión de la clave primaria.
* `is_in_sampling_key` ([UInt8](/es/reference/data-types/index)) — Indicador de si la columna está en la expresión de la clave de muestreo.
* `compression_codec` ([String](/es/reference/data-types/index)) — Nombre del códec de compresión.
* `character_octet_length` ([Nullable(UInt64)](/es/reference/data-types/index)) — Longitud máxima en bytes para datos binarios, datos de caracteres o datos de texto e imágenes. En ClickHouse, solo tiene sentido para el tipo de dato FixedString. En caso contrario, se devuelve el valor NULL.
* `numeric_precision` ([Nullable(UInt64)](/es/reference/data-types/index)) — Precisión de datos numéricos aproximados, datos numéricos exactos, datos enteros o datos monetarios. En ClickHouse, es el ancho en bits para los tipos enteros y la precisión decimal para los tipos Decimal. En caso contrario, se devuelve el valor NULL.
* `numeric_precision_radix` ([Nullable(UInt64)](/es/reference/data-types/index)) — La base del sistema numérico para la precisión de datos numéricos aproximados, datos numéricos exactos, datos enteros o datos monetarios. En ClickHouse, es 2 para los tipos enteros y 10 para los tipos Decimal. En caso contrario, se devuelve el valor NULL.
* `numeric_scale` ([Nullable(UInt64)](/es/reference/data-types/index)) — La escala de los datos numéricos aproximados, los datos numéricos exactos, los datos enteros o los datos monetarios. En ClickHouse, solo tiene sentido para los tipos Decimal. En caso contrario, se devuelve el valor NULL.
* `datetime_precision` ([Nullable(UInt64)](/es/reference/data-types/index)) — Precisión decimal del tipo de dato DateTime64. Para otros tipos de datos, se devuelve el valor NULL.
* `serialization_hint` ([Nullable(String)](/es/reference/data-types/index)) — Sugerencia para que la columna elija la serialización en los inserts según las estadísticas.
* `statistics` ([String](/es/reference/data-types/index)) — Tipos de estadísticas creados en esta columna.

{/*AUTOGENERATED_END*/}
