<div id="description">
  ## Descripción
</div>

Contiene información sobre los colaboradores. El orden es aleatorio en el momento de ejecutar la consulta.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del colaborador (autor) obtenido de `git log`.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Ejemplo
</div>

```sql
SELECT * FROM system.contributors LIMIT 10
```

Para encontrarte en la tabla, usa una consulta:

```sql
SELECT * FROM system.contributors WHERE name = 'Olga Khvostikova'
```
