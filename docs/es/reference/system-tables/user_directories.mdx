<div id="description">
  ## Descripción
</div>

Contiene información sobre los directorios de usuario configurados: directorios del sistema de archivos desde los que el servidor de ClickHouse puede leer los datos proporcionados por el usuario.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — El nombre del directorio.
* `type` ([String](/es/reference/data-types/index)) — El tipo de almacenamiento de acceso, por ejemplo, `users.xml`, `replicated` o `memory`.
* `params` ([String](/es/reference/data-types/index)) — JSON con los parámetros del almacenamiento de acceso.
* `precedence` ([UInt64](/es/reference/data-types/index)) — El orden en que este directorio se declara en la configuración. ClickHouse usa ese mismo orden cuando intenta encontrar un usuario o un rol.

{/*AUTOGENERATED_END*/}
