<div id="description">
  ## Descripción
</div>

Contiene información sobre las trazas de pila de los errores fatales. La tabla no existe de forma predeterminada en la base de datos; solo se crea cuando se producen errores fatales.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/es/reference/data-types/lowcardinality)) — El nombre de host donde ocurrió el fallo.
* `event_date` ([Date](/es/reference/data-types/date)) — La fecha del fallo.
* `event_time` ([DateTime](/es/reference/data-types/datetime)) — La hora del fallo.
* `timestamp_ns` ([UInt64](/es/reference/data-types/int-uint)) — Marca de tiempo del evento con nanosegundos.
* `signal` ([Int32](/es/reference/data-types/int-uint)) — Número de señal.
* `signal_code` ([Int32](/es/reference/data-types/int-uint)) — Código de señal. Proporciona contexto adicional sobre la causa de la señal.
* `thread_id` ([UInt64](/es/reference/data-types/int-uint)) — ID del hilo.
* `query_id` ([String](/es/reference/data-types/string)) — ID de consulta.
* `query` ([String](/es/reference/data-types/string)) — Texto de la consulta que se estaba ejecutando cuando ocurrió el fallo.
* `trace` ([Array(UInt64)](/es/reference/data-types/array)) — Traza de pila en el momento del fallo. Cada elemento es una dirección de memoria virtual dentro del proceso del servidor ClickHouse.
* `trace_full` ([Array(String)](/es/reference/data-types/array)) — Traza de pila en el momento del fallo. Cada elemento contiene un método llamado dentro del proceso del servidor ClickHouse.
* `fault_address` ([Nullable(UInt64)](/es/reference/data-types/nullable)) — Dirección de memoria que causó el fallo.
* `fault_access_type` ([String](/es/reference/data-types/string)) — Tipo de acceso a memoria que causó el fallo (p. ej., &#39;read&#39;, &#39;write&#39;).
* `signal_description` ([String](/es/reference/data-types/string)) — Descripción legible basada en signal&#95;code (p. ej., &#39;Address not mapped to object&#39;).
* `current_exception_trace_full` ([Array(String)](/es/reference/data-types/array)) — Traza de pila de la excepción actual con marcos simbolizados (solo para fallos de std::terminate).
* `version` ([String](/es/reference/data-types/string)) — Versión del servidor ClickHouse.
* `revision` ([UInt32](/es/reference/data-types/int-uint)) — Revisión del servidor ClickHouse.
* `build_id` ([String](/es/reference/data-types/string)) — BuildID generado por el compilador.
* `git_hash` ([String](/es/reference/data-types/string)) — Hash del commit de Git del código fuente de ClickHouse.
* `architecture` ([String](/es/reference/data-types/string)) — Arquitectura de CPU (p. ej., x86&#95;64, aarch64).

{/*AUTOGENERATED_END*/}
