<div id="description">
  ## Descripción
</div>

Esta tabla del sistema contiene metadatos por archivo de las tablas Iceberg existentes en ClickHouse, con una fila por cada archivo de datos o de borrado al que hace referencia la instantánea actual de cada tabla. Estará vacía si no hay ninguna tabla Iceberg en ClickHouse.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/es/reference/data-types/index)) — Nombre de la base de datos.
* `table` ([String](/es/reference/data-types/index)) — Nombre de la tabla.
* `snapshot_id` ([Int64](/es/reference/data-types/index)) — ID de la instantánea en la que se añadió el archivo.
* `content` ([Enum8(&#39;DATA&#39; = 0, &#39;POSITION&#95;DELETE&#39; = 1, &#39;EQUALITY&#95;DELETE&#39; = 2)](/es/reference/data-types/index)) — Tipo de contenido del archivo.
* `file_path` ([String](/es/reference/data-types/index)) — Ruta de almacenamiento resuelta del archivo.
* `file_format` ([String](/es/reference/data-types/index)) — Formato del archivo, p. ej., &#39;PARQUET&#39;.
* `record_count` ([Int64](/es/reference/data-types/index)) — Número de registros del archivo.
* `file_size_in_bytes` ([Int64](/es/reference/data-types/index)) — Tamaño del archivo en bytes.
* `partition` ([String](/es/reference/data-types/index)) — Representación textual de la tupla de partición.
* `schema_id` ([Int32](/es/reference/data-types/index)) — ID del esquema resuelto para esta entrada del manifiesto.
* `sequence_number` ([Int64](/es/reference/data-types/index)) — Número de secuencia resuelto de la entrada del manifiesto (siempre 0 para el formato v1).
* `sort_order_id` ([Nullable(Int32)](/es/reference/data-types/index)) — ID de ordenación del archivo, si se especifica.
* `null_value_counts` ([Map(Int32, Int64)](/es/reference/data-types/index)) — Recuento de valores NULL por columna (id de columna -&gt; recuento).
* `column_sizes` ([Map(Int32, Int64)](/es/reference/data-types/index)) — Tamaño en disco por columna en bytes (id de columna -&gt; bytes).
* `value_counts` ([Map(Int32, Int64)](/es/reference/data-types/index)) — Recuento total de valores por columna (id de columna -&gt; recuento).
* `equality_ids` ([Array(Int32)](/es/reference/data-types/index)) — ID de campos de igualdad para archivos de eliminación por igualdad (vacío para archivos que no son de este tipo).
