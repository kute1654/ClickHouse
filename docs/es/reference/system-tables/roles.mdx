<div id="description">
  ## Descripción
</div>

Contiene información sobre los [roles](/es/concepts/features/security/access-rights#role-management) configurados.

<div id="columns">
  ## Columnas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/es/reference/data-types/index)) — Nombre del rol.
* `id` ([UUID](/es/reference/data-types/index)) — ID del rol.
* `storage` ([String](/es/reference/data-types/index)) — Ruta del almacenamiento de roles. Se configura en el parámetro `access_control_path`.

{/*AUTOGENERATED_END*/}
