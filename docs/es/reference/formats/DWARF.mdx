<div id="example-usage">
  ## Ejemplo de uso
</div>

El formato `DWARF` puede utilizarse para encontrar las unidades de compilaciГіn que tienen el mayor nГәmero de definiciones de funciones (incluidas las instanciaciones de plantillas y las funciones de los archivos de cabecera incluidos):

FROM file('programs/clickhouse', DWARF)
