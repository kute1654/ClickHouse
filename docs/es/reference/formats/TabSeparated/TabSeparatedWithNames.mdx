<div id="description">
  ## Descripción
</div>

Se diferencia del formato [`TabSeparated`](/es/reference/formats/TabSeparated/TabSeparated) en que los nombres de las columnas se escriben en la primera fila.

Durante el análisis, se espera que la primera fila contenga los nombres de las columnas. Puede usar estos nombres para determinar su posición y comprobar que sean correctos.

<Note>
  Si la configuración [`input_format_with_names_use_header`](/es/reference/settings/formats#input_format_with_names_use_header) está establecida en `1`,
  las columnas de los datos de entrada se asignarán a las columnas de la tabla según sus nombres; las columnas con nombres desconocidos se omitirán si la configuración [`input_format_skip_unknown_fields`](/es/reference/settings/formats#input_format_skip_unknown_fields) está establecida en `1`.
  De lo contrario, se omitirá la primera fila.
</Note>

2022-05-07      2021    Bristol Rovers  Scunthorpe United       7       0
2022-05-07      2021    Exeter City     Port Vale       0       1

2022-05-07      2021    Hartlepool United       Colchester United       0       2

Inserta los datos:

```sql
INSERT INTO football FROM INFILE 'football.tsv' FORMAT TabSeparatedWithNames;
```

```sql
SELECT *
FROM football
FORMAT TabSeparatedWithNames
```
