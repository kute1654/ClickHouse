<div id="description">
  ## Descripción
</div>

Se diferencia del formato [`PrettyCompact`](/es/reference/formats/Pretty/PrettyCompact) en que no utiliza [secuencias de escape ANSI](http://en.wikipedia.org/wiki/ANSI_escape_code).
Esto es necesario para mostrar el formato en un navegador, así como para usar la utilidad de línea de comandos &#39;watch&#39;.
