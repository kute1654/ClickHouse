<div id="description">
  ## Descripción
</div>

Se diferencia del formato [`Pretty`](/es/reference/formats/Pretty/Pretty) en que la tabla se muestra con una cuadrícula entre las filas.
Por eso, el resultado es más compacto.

<Note>
  Este formato se usa de forma predeterminada en el cliente de línea de comandos en modo interactivo.
</Note>
