<div id="description">
  ## Descripción
</div>

Se diferencia del formato [`PrettySpace`](/es/reference/formats/Pretty/PrettySpace) en que no usa [secuencias de escape ANSI](http://en.wikipedia.org/wiki/ANSI_escape_code).
Esto es necesario para poder mostrar este formato en un navegador, así como para usar el comando de línea de comandos &#39;watch&#39;.
