<div id="description">
  ## Descripción
</div>

En este formato, ClickHouse genera cada fila como un objeto JSON independiente, uno por línea.

<div id="example-usage">
  ## Ejemplo de uso
</div>

<div id="inserting-data">

</div>

Inserte los datos:

```sql

```

<div id="reading-data">
  ### Leer datos
</div>

Lea los datos con el formato `JSONEachRow`:

```sql
SELECT *
FROM football
FORMAT JSONEachRow
```
