<div id="description">
  ## Descripción
</div>

Se diferencia de [JSONEachRow](/es/reference/formats/JSON/JSONEachRow) solo en que el JSON está formateado de forma legible, con delimitadores de nueva línea y sangrías de 4 espacios.

<div id="example-usage">
  ## Ejemplo de uso
</div>

<div id="reading-data">
  ### Lectura de datos
</div>

```sql
SELECT *
FROM football
FORMAT PrettyJSONEachRow
```
