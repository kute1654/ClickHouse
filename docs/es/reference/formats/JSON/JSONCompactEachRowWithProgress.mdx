<div id="description">
  ## Descripción
</div>

Este formato combina la salida compacta fila por fila de JSONCompactEachRow con información de progreso en streaming.
Genera los datos como objetos JSON independientes para los metadatos, las filas individuales, las actualizaciones de progreso,
los totales y las excepciones. Los valores se representan en sus tipos nativos.

Características principales:

* Genera primero los metadatos con los nombres y tipos de las columnas
* Cada fila es un objeto JSON independiente con una clave &quot;row&quot; que contiene un array de valores
* Incluye actualizaciones de progreso durante la ejecución de la consulta (como objetos `{"progress":...}`)
* Admite totales y extremos
* Los valores conservan sus tipos nativos (los números como números y las cadenas como cadenas)

<div id="example-usage">
  ## Ejemplo de uso
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactEachRowWithProgress
```
