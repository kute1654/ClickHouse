<div id="description">
  ## Descripción
</div>

<Tip>
  La salida de los formatos JSONColumns* proporciona el nombre del campo de ClickHouse y, a continuación, el contenido de cada fila de la tabla para ese campo;
  visualmente, los datos se giran 90 grados hacia la izquierda.
</Tip>

En este formato, todos los datos se representan como un único objeto JSON.

<Note>
  El formato `JSONColumns` almacena todos los datos en memoria intermedia y luego los genera como un único bloque, por lo que puede provocar un alto consumo de memoria.
</Note>

```sql

FROM football
```

<div id="format-settings">
  ## Configuración del formato
</div>

Durante la importación, se omitirán las columnas con nombres desconocidos si la configuración [`input_format_skip_unknown_fields`](/es/reference/settings/formats#input_format_skip_unknown_fields) está establecida en `1`.
Las columnas que no estén presentes en el bloque se completarán con valores predeterminados (aquí puede usar la configuración [`input_format_defaults_for_omitted_fields`](/es/reference/settings/formats#input_format_defaults_for_omitted_fields))
