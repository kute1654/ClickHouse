<div id="description">
  ## Descripción
</div>

Similar a [`JSONCompactEachRowWithProgress`](/es/reference/formats/JSON/JSONCompactEachRowWithProgress), pero todos los valores se convierten en cadenas.
Esto resulta útil cuando necesitas una representación en cadena uniforme para todos los tipos de datos.

Características principales:

* Tiene la misma estructura que `JSONCompactEachRowWithProgress`
* Todos los valores se representan como cadenas (los números, arrays, etc., aparecen entre comillas)
* Incluye actualizaciones de progreso, totales y manejo de excepciones
* Útil para clientes que prefieren o requieren datos en formato de cadena

<div id="example-usage">
  ## Ejemplo de uso
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactStringsEachRowWithProgress
```
