<div id="description">
  ## Descripción
</div>

El formato `JSONCompactStrings` se diferencia de [JSONStrings](/es/reference/formats/JSON/JSONStrings) solo en que las filas de datos se devuelven como arrays y no como objetos.

<div id="example-usage">
  ## Ejemplo de uso
</div>

<div id="reading-data">
  ### Leer datos
</div>

Lea los datos con el formato `JSONCompactStrings`:

```sql
SELECT *
FROM football
FORMAT JSONCompactStrings
```
