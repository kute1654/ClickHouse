<div id="description">
  ## Descripción
</div>

Se diferencia de [`JSONEachRow`](/es/reference/formats/JSON/JSONEachRow) únicamente en que los campos de datos se muestran como cadenas, no como valores JSON tipados.

<div id="example-usage">
  ## Ejemplo de uso
</div>

FROM football
