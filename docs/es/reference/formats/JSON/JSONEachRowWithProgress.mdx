<div id="description">
  ## Descripción
</div>

Se diferencia de [`JSONEachRow`](/es/reference/formats/JSON/JSONEachRow)/[`JSONStringsEachRow`](/es/reference/formats/JSON/JSONStringsEachRow) en que ClickHouse también devuelve información de progreso en forma de valores JSON.

<div id="example-usage">
  ## Ejemplo de uso
</div>

```json
{"row":{"num":42,"str":"hello","arr":[0,1]}}
{"row":{"num":43,"str":"hello","arr":[0,1,2]}}
{"row":{"num":44,"str":"hello","arr":[0,1,2,3]}}
{"progress":{"read_rows":"3","read_bytes":"24","written_rows":"0","written_bytes":"0","total_rows_to_read":"3"}}
```
