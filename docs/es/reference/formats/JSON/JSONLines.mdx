<div id="description">
  ## Descripción
</div>

En este formato, ClickHouse genera cada fila como un objeto JSON independiente, delimitado por saltos de línea.

Este formato también se conoce como `JSONEachRow`, `JSONLines`, `NDJSON` (JSON delimitado por saltos de línea) o `JSONL`. Estos nombres son alias del mismo formato y pueden usarse indistintamente tanto para la entrada como para la salida.

```sql
SELECT *
FROM football
FORMAT JSONLines
```
