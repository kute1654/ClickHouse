<div id="description">
  ## Descripción
</div>

El formato `BSONEachRow` interpreta los datos como una secuencia de documentos Binary JSON (BSON) sin ningún separador entre ellos.
Cada fila se representa como un único documento y cada columna como un único campo de documento BSON, con el nombre de la columna como clave.

<div id="data-types-matching">
  ## Correspondencia de tipos de datos
</div>

Para la salida, se utiliza la siguiente correspondencia entre los tipos de ClickHouse y los tipos BSON:

| Tipo de ClickHouse                                                                      | Tipo BSON                                                                                                                                      |
| --------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| [Bool](/es/reference/data-types/boolean)                                                   | `\x08` boolean                                                                                                                                 |
| [Int8/UInt8](/es/reference/data-types/int-uint)/[Enum8](/es/reference/data-types/enum)        | `\x10` int32                                                                                                                                   |
| [Int16/UInt16](/es/reference/data-types/int-uint)/[Enum16](/es/reference/data-types/enum)     | `\x10` int32                                                                                                                                   |
| [Int32](/es/reference/data-types/int-uint)                                                 | `\x10` int32                                                                                                                                   |
| [UInt32](/es/reference/data-types/int-uint)                                                | `\x12` int64                                                                                                                                   |
| [Int64/UInt64](/es/reference/data-types/int-uint)                                          | `\x12` int64                                                                                                                                   |
| [Float32/Float64](/es/reference/data-types/float)                                          | `\x01` double                                                                                                                                  |
| [Date](/es/reference/data-types/date)/[Date32](/es/reference/data-types/date32)               | `\x10` int32                                                                                                                                   |
| [DateTime](/es/reference/data-types/datetime)                                              | `\x12` int64                                                                                                                                   |
| [DateTime64](/es/reference/data-types/datetime64)                                          | `\x09` datetime                                                                                                                                |
| [Decimal32](/es/reference/data-types/decimal)                                              | `\x10` int32                                                                                                                                   |
| [Decimal64](/es/reference/data-types/decimal)                                              | `\x12` int64                                                                                                                                   |
| [Decimal128](/es/reference/data-types/decimal)                                             | `\x05` binary, `\x00` binary subtype, tamaño = 16                                                                                              |
| [Decimal256](/es/reference/data-types/decimal)                                             | `\x05` binary, `\x00` binary subtype, tamaño = 32                                                                                              |
| [Int128/UInt128](/es/reference/data-types/int-uint)                                        | `\x05` binary, `\x00` binary subtype, tamaño = 16                                                                                              |
| [Int256/UInt256](/es/reference/data-types/int-uint)                                        | `\x05` binary, `\x00` binary subtype, tamaño = 32                                                                                              |
| [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring) | `\x05` binary, `\x00` binary subtype o \x02 string si la configuración output&#95;format&#95;bson&#95;string&#95;as&#95;string está habilitada |
| [UUID](/es/reference/data-types/uuid)                                                      | `\x05` binary, `\x04` uuid subtype, tamaño = 16                                                                                                |
| [Array](/es/reference/data-types/array)                                                    | `\x04` array                                                                                                                                   |
| [Tuple](/es/reference/data-types/tuple)                                                    | `\x04` array                                                                                                                                   |
| [Named Tuple](/es/reference/data-types/tuple)                                              | `\x03` document                                                                                                                                |
| [Map](/es/reference/data-types/map)                                                        | `\x03` document                                                                                                                                |
| [IPv4](/es/reference/data-types/ipv4)                                                      | `\x10` int32                                                                                                                                   |
| [IPv6](/es/reference/data-types/ipv6)                                                      | `\x05` binary, `\x00` binary subtype                                                                                                           |

Para la entrada, se utiliza la siguiente correspondencia entre los tipos BSON y los tipos de ClickHouse:

| Tipo BSON                                      | Tipo de ClickHouse                                                                                                                                                      |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `\x01` doble                                   | [Float32/Float64](/es/reference/data-types/float)                                                                                                                          |
| `\x02` string                                  | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)                                                                                 |
| `\x03` documento                               | [Map](/es/reference/data-types/map)/[Tupla con nombre](/es/reference/data-types/tuple)                                                                                        |
| `\x04` array                                   | [Array](/es/reference/data-types/array)/[Tuple](/es/reference/data-types/tuple)                                                                                               |
| `\x05` binario, subtipo binario `\x00`         | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)/[IPv6](/es/reference/data-types/ipv6)                                              |
| `\x05` binario, subtipo binario antiguo `\x02` | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)                                                                                 |
| `\x05` binario, subtipo uuid antiguo `\x03`    | [UUID](/es/reference/data-types/uuid)                                                                                                                                      |
| `\x05` binario, subtipo uuid `\x04`            | [UUID](/es/reference/data-types/uuid)                                                                                                                                      |
| `\x07` ObjectId                                | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)                                                                                 |
| `\x08` boolean                                 | [Bool](/es/reference/data-types/boolean)                                                                                                                                   |
| `\x09` datetime                                | [DateTime64](/es/reference/data-types/datetime64)                                                                                                                          |
| `\x0A` valor NULL                              | [NULL](/es/reference/data-types/nullable)                                                                                                                                  |
| `\x0D` código JavaScript                       | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)                                                                                 |
| `\x0E` símbolo                                 | [String](/es/reference/data-types/string)/[FixedString](/es/reference/data-types/fixedstring)                                                                                 |
| `\x10` int32                                   | [Int32/UInt32](/es/reference/data-types/int-uint)/[Decimal32](/es/reference/data-types/decimal)/[IPv4](/es/reference/data-types/ipv4)/[Enum8/Enum16](/es/reference/data-types/enum) |
| `\x12` int64                                   | [Int64/UInt64](/es/reference/data-types/int-uint)/[Decimal64](/es/reference/data-types/decimal)/[DateTime64](/es/reference/data-types/datetime64)                                |

No se admiten otros tipos BSON. Además, realiza conversiones entre distintos tipos enteros.
Por ejemplo, es posible insertar un valor BSON `int32` en ClickHouse como [`UInt8`](/es/reference/data-types/int-uint).

Los enteros grandes y decimales, como `Int128`/`UInt128`/`Int256`/`UInt256`/`Decimal128`/`Decimal256`, se pueden analizar a partir de un valor BSON Binary con el subtipo binario `\x00`.
En este caso, el formato validará que el tamaño de los datos binarios coincida con el tamaño del valor esperado.

<Note>
  Este formato no funciona correctamente en plataformas Big-Endian.
</Note>


<div id="reading-data">
  ### Lectura de datos
</div>

Lea datos con el formato `BSONEachRow`:

```sql
SELECT *
FROM football INTO OUTFILE 'docs_data/bson/football.bson'
FORMAT BSONEachRow
```

<Tip>
  BSON es un formato binario que no se puede ver en la terminal de forma legible para humanos. Use `INTO OUTFILE` para generar archivos BSON.
</Tip>
