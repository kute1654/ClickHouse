<div id="description">
  ## DescripciГіn
</div>

El formato `One` es un formato de entrada especial que no lee ningГәn dato del archivo y devuelve una sola fila con una columna de tipo [`UInt8`](/es/reference/data-types/int-uint), de nombre `dummy` y valor `0` (como la tabla `system.one`).
Puede usarse con las columnas virtuales `_file/_path` para listar todos los archivos sin leer los datos reales.

<div id="example-usage">
  ## Ejemplo de uso
</div>

```sql title="Query"
SELECT _file FROM file('path/to/files/data*', One);
```
