<div id="description">
  ## Descripción
</div>

El formato `LineAsStringWithNames` es similar al formato [`LineAsString`](/es/reference/formats/LineAsString/LineAsString), pero imprime la fila de encabezado con los nombres de las columnas.

<div id="example-usage">
  ## Ejemplo de uso
</div>

```sql title="Query"
CREATE TABLE example (
    name String,
    value Int32
)
ENGINE = Memory;

INSERT INTO example VALUES ('John', 30), ('Jane', 25), ('Peter', 35);

SELECT * FROM example FORMAT LineAsStringWithNames;
```

```response title="Response"
name    value
John    30
Jane    25
Peter    35
```
