<div id="description">
  ## Descripción
</div>

El formato `XML` es adecuado solo para la salida, no para el análisis.

Si el nombre de la columna no tiene un formato aceptable, simplemente se usa &#39;field&#39; como nombre del elemento. En general, la estructura de XML sigue la de JSON.
Al igual que en JSON, las secuencias UTF-8 no válidas se sustituyen por el carácter de reemplazo `�`, de modo que el texto de salida constará de secuencias UTF-8 válidas.

En los valores de cadena, los caracteres `<` y `&` se escapan como `<` y `&`.

Los arrays se muestran como `<array><elem>Hello</elem><elem>World</elem>...</array>`,y las tuplas como `<tuple><elem>Hello</elem><elem>World</elem>...</tuple>`.
