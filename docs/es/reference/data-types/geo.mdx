ClickHouse admite tipos de datos para representar objetos geogrГЎficos: ubicaciones, terrenos, etc.

**VГ©ase tambiГ©n**

* [RepresentaciГіn de entidades geogrГЎficas simples](https://en.wikipedia.org/wiki/GeoJSON).

<div id="point">
  ## Point
</div>

`Point` se representa mediante sus coordenadas X e Y, almacenadas como un [Tuple](/es/reference/data-types/tuple)([Float64](/es/reference/data-types/float), [Float64](/es/reference/data-types/float)).

**Ejemplo**

```sql title="Query"
CREATE TABLE geo_point (p Point) ENGINE = Memory();
INSERT INTO geo_point VALUES((10, 10));
SELECT p, toTypeName(p) FROM geo_point;
```

<div id="ring">
  ## Ring

`Ring` es un polГӯgono simple sin agujeros almacenado como un array de puntos: [Array](/es/reference/data-types/array)([Point](#point)).

```sql title="Query"
CREATE TABLE geo_ring (r Ring) ENGINE = Memory();
INSERT INTO geo_ring VALUES([(0, 0), (10, 0), (10, 10), (0, 10)]);
SELECT r, toTypeName(r) FROM geo_ring;
```

<div id="multilinestring">

<div id="polygon">
  ## Polygon
</div>

`Polygon` es un polГӯgono con huecos almacenado como un array de anillos: [Array](/es/reference/data-types/array)([Ring](#ring)). El primer elemento del array exterior es el contorno exterior del polГӯgono y todos los elementos siguientes son huecos.

**Ejemplo**

Este es un polГӯgono con un hueco:

```sql title="Query"
CREATE TABLE geo_polygon (pg Polygon) ENGINE = Memory();
INSERT INTO geo_polygon VALUES([[(20, 20), (50, 20), (50, 50), (20, 50)], [(30, 30), (50, 50), (50, 30)]]);
SELECT pg, toTypeName(pg) FROM geo_polygon;
```

<div id="multipolygon">
  ## MultiPolygon
</div>

`MultiPolygon` consta de varios polГӯgonos y se almacena como un array de polГӯgonos: [Array](/es/reference/data-types/array)([Polygon](#polygon)).

```sql title="Query"
INSERT INTO geo_multipolygon VALUES([[[(0, 0), (10, 0), (10, 10), (0, 10)]], [[(20, 20), (50, 20), (50, 50), (20, 50)],[(30, 30), (50, 50), (50, 30)]]]);
SELECT mpg, toTypeName(mpg) FROM geo_multipolygon;
```

`Geometry` es un tipo comГәn para todos los tipos anteriores. Equivale a un `Variant` de esos tipos.

**Ejemplo**

```sql title="Query"

INSERT INTO geo VALUES ((1, 2));
SELECT * FROM geo;
```
