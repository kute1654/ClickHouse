Patrones habituales de pandas y sus equivalentes en DataStore. ¡La mayor parte del código funciona sin cambios!

<div id="loading">
  ## Carga de datos
</div>

<div id="read-csv">
  ### Leer archivos CSV
</div>

```python
# Pandas
import pandas as pd
df = pd.read_csv("data.csv")

# DataStore - ¡igual!
from chdb import datastore as pd
df = pd.read_csv("data.csv")
```

<div id="read-multiple-files">
  ### Leer varios archivos
</div>

```python
# Pandas
import glob
dfs = [pd.read_csv(f) for f in glob.glob("data/*.csv")]
df = pd.concat(dfs)

# DataStore - más eficiente con glob pattern
df = pd.read_csv("data/*.csv")
```

<div id="filtering">
  ## Filtrado
</div>

<div id="single-condition">
  ### Una sola condición
</div>

```python
# Pandas y DataStore: idénticos
df[df['age'] > 25]
df[df['city'] == 'NYC']
df[df['name'].str.contains('John')]
```

<div id="multiple-conditions">
  ### Varias condiciones
</div>

```python
# AND
df[(df['age'] > 25) & (df['city'] == 'NYC')]

# OR
df[(df['age'] < 18) | (df['age'] > 65)]

# NOT
df[~(df['status'] == 'inactive')]
```

<div id="using-query">
  ### Uso de query()
</div>

```python
# Pandas y DataStore - idénticos
df.query('age > 25 and city == "NYC"')
df.query('salary > 50000')
```

<div id="isin">

</div>

```python
# Pandas y DataStore - idéntico
```

<div id="between">

</div>

```python
# Pandas y DataStore - idénticos
```

<div id="groupby">
  ## GroupBy y agregación
</div>

<div id="simple-groupby">
  ### GroupBy básico
</div>

```python
# Pandas y DataStore - idéntico
df.groupby('city')['salary'].mean()
df.groupby('city')['salary'].sum()
df.groupby('city').size()  # conteo
```

<div id="multiple-aggregations">
  ### Múltiples agregaciones
</div>

```python
# Pandas y DataStore - idéntico
df.groupby('city')['salary'].agg(['sum', 'mean', 'count'])

df.groupby('city').agg({
    'salary': ['sum', 'mean'],
    'age': ['min', 'max']
})
```

<div id="named-aggregations">
  ### Agregaciones nombradas
</div>

```python
# Pandas y DataStore - idénticos
df.groupby('city').agg(
    total_salary=('salary', 'sum'),
    avg_salary=('salary', 'mean'),
    employee_count=('id', 'count')
)
```

<div id="multiple-groupby-keys">
  ### Varias claves de GroupBy
</div>

```python
# Pandas y DataStore - idéntico
```

<div id="apply-for-custom-logic">
  ### apply() para lógica personalizada
</div>

```python
# Funciona, pero activa la ejecución de pandas
df['category'] = df['amount'].apply(lambda x: 'high' if x > 1000 else 'low')

# Alternativa con DataStore (permanece diferida)
df['category'] = (
    df.when(df['amount'] > 1000, 'high')
      .otherwise('low')
)
```
