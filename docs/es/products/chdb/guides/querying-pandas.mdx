[Pandas](https://pandas.pydata.org/) es una biblioteca popular para la manipulación y el análisis de datos en Python.
En la versión 2 de chDB, hemos mejorado el rendimiento de las consultas sobre DataFrames de Pandas y hemos incorporado la función de tabla `Python`.
En esta guía, aprenderemos a consultar Pandas con la función de tabla `Python`.

<div id="setup">
  ## Configuración
</div>

Primero, creemos un entorno virtual:

```bash
python -m venv .venv
source .venv/bin/activate
```

Y ahora instalaremos chDB.
Asegúrate de tener la versión 2.0.2 o una superior:

```bash
pip install "chdb>=2.0.2"
```

Y ahora vamos a instalar Pandas y un par de librerías más:

```bash
pip install pandas requests ipython
```

Vamos a usar `ipython` para ejecutar los comandos del resto de la guía; puedes iniciarlo ejecutando:

```bash
ipython
```

También puedes usar el código en un script de Python o en tu notebook favorito.

<div id="querying-pandas-dataframes">
  ## Consultar DataFrames de Pandas
</div>

A continuación, veamos cómo consultar estos DataFrames con chDB.
Importaremos la biblioteca:

```python
import chdb
```

Podemos consultar DataFrames de Pandas mediante la función de tabla `Python`:

```sql
SELECT *
FROM Python(<name-of-variable>)
```

Así, si quisiéramos listar las columnas de `matches_df`, podríamos escribir lo siguiente:

```python
chdb.query("""
DESCRIBE Python(matches_df)
SETTINGS describe_compact_output=1
""", "DataFrame")
```

<div id="joining-pandas-dataframes">
  ## Unir DataFrames de Pandas
</div>

También podemos unir DataFrames en una consulta.
Por ejemplo, para obtener una visión general del partido, podríamos escribir la siguiente consulta:

```python
chdb.query("""
SELECT home_team_home_team_name, away_team_away_team_name, home_score, away_score,
       countIf(type_name = 'Pass' AND possession_team_id=home_team_home_team_id) AS home_passes,
       countIf(type_name = 'Pass' AND possession_team_id=away_team_away_team_id) AS away_passes,
       countIf(type_name = 'Shot' AND possession_team_id=home_team_home_team_id) AS home_shots,
       countIf(type_name = 'Shot' AND possession_team_id=away_team_away_team_id) AS away_shots
FROM Python(matches_df) AS matches
JOIN Python(events_df) AS events ON events.match_id = matches.match_id
GROUP BY ALL
LIMIT 5
""", "DataFrame").iloc[0]
```


<div id="populating-a-table-from-a-dataframe">
  ## Rellenar una tabla a partir de un DataFrame
</div>

También podemos crear y rellenar tablas de ClickHouse a partir de DataFrames.
Si queremos crear una tabla en chDB, debemos usar la API de Stateful Session.

Importemos el módulo `session`:

```python
from chdb import session as chs
```

Inicializa una sesión:

```python
sess = chs.Session()
```

A continuación, crearemos una base de datos:

```python
sess.query("CREATE DATABASE statsbomb")
```

A continuación, crea la tabla `events` a partir de `events_df`:

```python
sess.query("""
CREATE TABLE statsbomb.events ORDER BY id AS
SELECT * 
FROM Python(events_df)
""")
```

A continuación, podemos ejecutar la consulta que devuelve el jugador que más pases recibe:

```python
sess.query("""
SELECT pass_recipient_name, count()
FROM statsbomb.events
WHERE type_name = 'Pass' AND pass_recipient_name <> ''
GROUP BY ALL
ORDER BY count() DESC
LIMIT 10
""", "DataFrame")
```
