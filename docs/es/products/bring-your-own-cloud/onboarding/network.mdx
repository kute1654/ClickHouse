ClickHouse BYOC admite varias opciones de red privada para mejorar la seguridad y permitir la conectividad directa de sus servicios. Esta guía le presenta los enfoques recomendados para conectar de forma segura los despliegues de ClickHouse Cloud en su propia cuenta de AWS o GCP con otras redes o servicios, como sus aplicaciones internas o herramientas de análisis. Se describen opciones como VPC Peering, AWS PrivateLink y GCP Private Service Connect, así como los pasos principales y las consideraciones de cada una.

Si necesita una conexión de red privada para su despliegue de ClickHouse BYOC, siga los pasos de las guías o póngase en contacto con el equipo de soporte de ClickHouse para recibir ayuda con escenarios más avanzados.
