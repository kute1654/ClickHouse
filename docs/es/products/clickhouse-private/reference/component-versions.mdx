<div id="notes">
  ## Notas
</div>

* La **versión del gráfico de Helm del operador** (`1.18587.1`) difiere de la **etiqueta de imagen** del operador (`main-1.18587.1`). Usa la versión del gráfico de Helm al instalar con `helm install` y la etiqueta de imagen al copiar imágenes de contenedor.
* Las **compilaciones FIPS** añaden `-fips` a las etiquetas de imagen estándar tanto del server como de Keeper.
* Las etiquetas de versión se sustituyen automáticamente al compilar esta documentación. Si ves variables de plantilla sin sustituir (por ejemplo, `25.12.1.1497`), el paso de compilación no se ha ejecutado.
