ClickHouse cuenta con soporte experimental para LoongArch64

<div id="build-clickhouse">
  ## Compilar ClickHouse
</div>

La versión de llvm necesaria para la compilación debe ser mayor o igual que 21.1.0.

```bash
cd ClickHouse
mkdir build-loongarch64
cmake . -Bbuild-loongarch64 -DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-loongarch64.cmake
ninja -C build-loongarch64
```

El binario resultante solo se ejecutará en Linux con la arquitectura de CPU LoongArch64.
