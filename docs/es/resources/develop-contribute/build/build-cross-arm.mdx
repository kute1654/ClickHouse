No se requieren pasos especiales para compilar ClickHouse para Aarch64 en una máquina Aarch64.

Para realizar una compilación cruzada de ClickHouse para AArch64 en una máquina Linux x86, pasa la siguiente opción a `cmake`: `-DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-aarch64.cmake`
