<a href="https://www.zingdata.com/" target="_blank">Zing Data</a> es una plataforma de exploración y visualización de datos. Zing Data se conecta a ClickHouse mediante el controlador de JS proporcionado por ClickHouse.

<div id="how-to-connect">
  ## Cómo conectarse
</div>

1. Reúne tus datos de conexión.

<ConnectionDetails />

2. Descarga o visita Zing Data

   * Para usar ClickHouse con Zing Data en un dispositivo móvil, descarga la aplicación de Zing Data en [Google Play Store](https://play.google.com/store/apps/details?id=com.getzingdata.android) o en la [Apple App Store](https://apps.apple.com/us/app/zing-data-collaborative-bi/id1563294091).

   * Para usar ClickHouse con Zing Data en la web, visita la [consola web de Zing](https://console.getzingdata.com/) y crea una cuenta.

3. Añade un origen de datos

   * Para interactuar con tus datos de ClickHouse en Zing Data, debes definir un ***origen de datos***. En el menú de la aplicación móvil de Zing Data, selecciona **Sources** y luego haz clic en **Add a Datasource**.

   * Para añadir un origen de datos en la web, haz clic en **Data Sources** en el menú superior, luego en **New Datasource** y selecciona **ClickHouse** en el menú desplegable

   <Image size="md" img="/images/integrations/data-visualization/zing_01.webp" alt="Interfaz de Zing Data que muestra el botón New Datasource y la opción ClickHouse en el menú desplegable" border />

   <br />

4. Completa los datos de conexión y haz clic en **Check Connection**.

   <Image size="md" img="/images/integrations/data-visualization/zing_02.webp" alt="Formulario de configuración de conexión de ClickHouse en Zing Data con campos para server, port, database, username y password" border />

   <br />

5. Si la conexión se realiza correctamente, Zing te llevará a la selección de tablas. Selecciona las tablas necesarias y haz clic en **Save**. Si Zing no puede conectarse a tu origen de datos, verás un mensaje pidiéndote que compruebes tus credenciales y lo intentes de nuevo. Si sigues teniendo problemas incluso después de comprobar tus credenciales e intentarlo de nuevo, <a id="contact_link" href="mailto:hello@getzingdata.com">contacta aquí con el soporte de Zing.</a>

   <Image size="md" img="/images/integrations/data-visualization/zing_03.webp" alt="Interfaz de selección de tablas de Zing Data que muestra las tablas de ClickHouse disponibles con casillas de verificación" border />

   <br />

6. Una vez añadido el origen de datos de ClickHouse, estará disponible para todos los miembros de tu organización de Zing, en la pestaña **Data Sources** / **Sources**.

<div id="creating-charts-and-dashboards-in-zing-data">
  ## Creación de gráficos y dashboards en Zing Data
</div>

1. Después de agregar tu origen de datos de ClickHouse, haz clic en **Zing App** en la web, o en el origen de datos desde el móvil para empezar a crear gráficos.

2. Haz clic en una tabla de la lista para crear un gráfico.

   <Image size="sm" img="/images/integrations/data-visualization/zing_04.webp" alt="Interfaz de Zing Data que muestra la lista de tablas con las tablas disponibles de ClickHouse" border />

   <br />

3. Usa el generador visual de consultas para seleccionar los campos, agregaciones, etc., y haz clic en **Run Question**.

   <Image size="md" img="/images/integrations/data-visualization/zing_05.webp" alt="Interfaz del generador visual de consultas de Zing Data con selección de campos y opciones de agregación" border />

   <br />

4. Si conoces SQL, también puedes escribir SQL personalizado para ejecutar consultas y crear un gráfico.

   <Image size="md" img="/images/integrations/data-visualization/zing_06.webp" alt="Modo de SQL Editor en Zing Data que muestra la interfaz para escribir consultas SQL" border />

   <Image size="md" img="/images/integrations/data-visualization/zing_07.webp" alt="Resultados de consultas SQL en Zing Data con los datos mostrados en formato tabular" border />

5. Un gráfico de ejemplo tendría este aspecto. La consulta se puede guardar desde el menú de tres puntos. Puedes comentar en el gráfico, etiquetar a miembros de tu equipo, crear alertas en tiempo real, cambiar el tipo de gráfico, etc.

   <Image size="md" img="/images/integrations/data-visualization/zing_08.webp" alt="Visualización de gráfico de ejemplo en Zing Data que muestra datos de ClickHouse con menú de opciones" border />

   <br />

6. Los dashboards se pueden crear con el icono &quot;+&quot; en **Dashboards**, en la pantalla de inicio. Las consultas existentes se pueden arrastrar para mostrarlas en el dashboard.

   <Image size="md" img="/images/integrations/data-visualization/zing_09.webp" alt="Vista de dashboard de Zing Data que muestra varias visualizaciones organizadas en un diseño de dashboard" border />

   <br />
