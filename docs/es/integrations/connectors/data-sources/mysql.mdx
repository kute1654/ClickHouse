<div id="connecting-clickhouse-to-mysql-using-the-mysql-table-engine">
  ## Conectar ClickHouse con MySQL mediante el motor de tabla MySQL
</div>

El motor de tabla `MySQL` le permite conectar ClickHouse con MySQL. Las sentencias **SELECT** e **INSERT** pueden ejecutarse tanto en ClickHouse como en la tabla de MySQL. Este artГӯculo muestra las formas bГЎsicas de usar el motor de tabla `MySQL`.

<div id="1-configure-mysql">
  ### 1. Configurar MySQL
</div>

1. Crea una base de datos en MySQL:

```sql
  CREATE DATABASE db1;
```

2. Cree una tabla:

```sql
  CREATE TABLE db1.table1 (
    id INT,
    column1 VARCHAR(255)
  );
```

3. Inserte algunas filas de ejemplo:

```sql
  INSERT INTO db1.table1
    (id, column1)
  VALUES
    (1, 'abc'),
    (2, 'def'),
    (3, 'ghi');
```

4. Cree un usuario para conectarse desde ClickHouse:

```sql
  CREATE USER 'mysql_clickhouse'@'%' IDENTIFIED BY 'Password123!';
```

5. Otorgue los privilegios necesarios. (Con fines de demostraciГіn, se otorgan privilegios de administrador al usuario `mysql_clickhouse`.)

```sql

```

<Note>
  Si utiliza esta funciГіn en ClickHouse Cloud, puede que necesite permitir que las direcciones IP de ClickHouse Cloud accedan a su instancia de MySQL.
  Consulte la [API de Cloud Endpoints de ClickHouse](/es/products/cloud/guides/sql-console/query-endpoints) para obtener detalles sobre el trГЎfico de salida.
</Note>

<div id="2-define-a-table-in-clickhouse">
  ### 2. Definir una tabla en ClickHouse
</div>

1. Ahora vamos a crear una tabla en ClickHouse que use el motor de tabla `MySQL`:

```sql
  CREATE TABLE mysql_table1 (
    id UInt64,
    column1 String
  )
  ENGINE = MySQL('mysql-host.domain.com','db1','table1','mysql_clickhouse','Password123!')
```

Los parГЎmetros mГӯnimos son:

| parГЎmetro | descripciГіn                               | ejemplo               |
| --------- | ----------------------------------------- | --------------------- |
| host      | nombre de host o IP                       | mysql-host.domain.com |
| database  | nombre de la base de datos de MySQL       | db1                   |
| table     | nombre de la tabla de MySQL               | table1                |
| user      | nombre de usuario para conectarse a MySQL | mysql&#95;clickhouse  |
| password  | contraseГұa para conectarse a MySQL        | Password123!          |

<Note>
  Consulta la pГЎgina de documentaciГіn de [motor de tabla MySQL](/es/reference/engines/table-engines/integrations/mysql) para ver la lista completa de parГЎmetros.
