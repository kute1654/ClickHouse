Esta sección incluye las siguientes guías avanzadas:

| Guía                                                                                                                                                    | Descripción                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Lenguajes de consulta alternativos](/es/concepts/features/configuration/settings/alternative-query-languages)                                             | Una guía sobre los dialectos alternativos compatibles y cómo usarlos. Incluye ejemplos de consultas en cada dialecto.                                                                                                                                                                                                                                                             |
| [Vistas materializadas en cascada](/es/concepts/features/materialized-views/cascading-materialized-views)                                                  | Una guía sobre cómo crear vistas materializadas y encadenarlas en cascada, combinando varias tablas de origen en una sola tabla de destino. Incluye un ejemplo del uso de vistas materializadas en cascada para agregar datos por mes y año para un grupo de nombres de dominio.                                                                                                  |
| [Depuración de problemas de memoria](/es/concepts/features/performance/troubleshoot/debugging-memory-issues)                                               | Una guía sobre cómo depurar problemas de memoria en ClickHouse.                                                                                                                                                                                                                                                                                                                   |
| [Deduplicación de inserciones en reintentos](/es/concepts/features/operations/insert/deduplicating-inserts-on-retries)                                     | Una guía sobre cómo gestionar situaciones en las que tengas que reintentar inserciones fallidas.                                                                                                                                                                                                                                                                                  |
| [Estrategias de deduplicación](/es/concepts/features/operations/insert/deduplication)                                                                      | Una guía que profundiza en la deduplicación de datos, una técnica para eliminar filas duplicadas de tu base de datos. Explica las diferencias con la deduplicación basada en claves primarias en sistemas OLTP, el enfoque de ClickHouse para la deduplicación y cómo gestionar escenarios de datos duplicados en tus consultas de ClickHouse.                                    |
| [Rellenar huecos en datos de series temporales](/es/guides/use-cases/real-time-analytics/time-series/time-series-filling-gaps)                             | Una guía que ofrece información sobre las capacidades de ClickHouse para trabajar con datos de series temporales, incluidas técnicas para rellenar huecos en los datos y crear una representación más completa y continua de la información de series temporales.                                                                                                                 |
| [Gestionar datos con TTL (tiempo de vida)](/es/concepts/features/operations/delete/ttl)                                                                    | Una guía que explica cómo usar la cláusula `WITH FILL` para rellenar huecos en datos de series temporales. Cubre cómo rellenar huecos con valores 0, cómo especificar un punto de inicio para hacerlo, cómo rellenar huecos hasta un punto final concreto y cómo interpolar valores para cálculos acumulativos.                                                                   |
| [Procedimientos almacenados y parámetros de consulta](/es/guides/clickhouse/data-modelling/stored-procedures-and-prepared-statements)                      | Una guía que explica que ClickHouse no admite procedimientos almacenados tradicionales y ofrece alternativas recomendadas, como funciones definidas por el usuario (UDF), vistas parametrizadas, vistas materializadas y orquestación externa. También cubre los parámetros de consulta para consultas parametrizadas seguras (similares a las sentencias preparadas).            |
| [Comprender la ejecución de consultas con el analizador](/es/guides/clickhouse/performance-and-monitoring/understanding-query-execution-with-the-analyzer) | Una guía que aclara cómo funciona la ejecución de consultas en ClickHouse mediante la introducción de la herramienta analizador. Explica cómo el analizador descompone una consulta en una serie de pasos, lo que te permite visualizar y depurar todo el proceso de ejecución para lograr un rendimiento óptimo.                                                                 |
| [Uso de JOIN en ClickHouse](/es/concepts/features/operations/select/joining-tables)                                                                        | Una guía que simplifica la unión de tablas en ClickHouse. Cubre distintos tipos de join (`INNER`, `LEFT`, `RIGHT`, etc.), explora buenas prácticas para joins eficientes (como colocar las tablas más pequeñas a la derecha) y ofrece información sobre los algoritmos internos de join de ClickHouse para ayudarte a optimizar tus consultas para relaciones de datos complejas. |
