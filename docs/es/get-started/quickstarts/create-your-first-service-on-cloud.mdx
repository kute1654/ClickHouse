<div id="prerequisites">
  ## Requisitos previos
</div>

No hay requisitos previos para esta guía de inicio rápido.

<div id="what-youll-build">
  ## Lo que crearás
</div>

En esta guía de inicio rápido crearás tu primer servicio de ClickHouse Cloud, te familiarizarás con la consola de Cloud y la consola SQL integrada, y cargarás un conjunto de datos de ejemplo para disponer de datos reales que consultar.
Al final, tendrás un servicio de ClickHouse Cloud en funcionamiento, con datos cargados y listo para usarlo como base para el resto de la serie de guías de inicio rápido.

<Steps titleSize="h3">
  <Step>
    ### Regístrate en ClickHouse Cloud

    Ve a [console.clickhouse.cloud](https://console.clickhouse.cloud/). Verás la página de inicio de sesión. Haz clic en **Register** junto a &quot;¿Todavía no tienes una cuenta?&quot;, en la parte inferior.

    <Frame caption="Haz clic en Register para crear una cuenta nueva">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="Página de inicio de sesión de ClickHouse Cloud con enlace Register" />
    </Frame>

    En la página de registro, regístrate con Google, Microsoft o introduce tu correo electrónico de trabajo. No se requiere tarjeta de crédito; la prueba gratuita te ofrece 300 $ en créditos y 30 días para explorar ClickHouse Cloud.

    <Frame caption="Regístrate con Google, Microsoft o tu correo electrónico de trabajo">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="Página de registro de ClickHouse Cloud" />
    </Frame>

    Si te registras con correo electrónico, se te pedirá que definas una password. Debe cumplir los siguientes requisitos:

    * Al menos 12 caracteres
    * Al menos 3 de los siguientes: letras minúsculas, letras mayúsculas, números y caracteres especiales (p. ej., `!@#$%^&*`)

    <Frame caption="Introduce tu password y acepta los Terms of Service">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="Registro por correo electrónico en ClickHouse Cloud con campo de password" />
    </Frame>

    Acepta los Terms of Service y la Política de privacidad y, a continuación, haz clic en **Sign up**. Revisa tu correo electrónico, busca el enlace de confirmación y haz clic en **Verify email** para activar tu cuenta.

    Una vez verificada, inicia sesión con tu nuevo correo electrónico y password.
  </Step>

  <Step>
    ### Crear un nuevo servicio

    Después de iniciar sesión por primera vez, se te llevará directamente a la página de configuración del servicio. Configura lo siguiente:

    | Configuración           | Recomendación                                                                                                 |
    | ----------------------- | ------------------------------------------------------------------------------------------------------------- |
    | **Nombre del servicio** | Asigna a tu servicio un nombre descriptivo (por ejemplo, `My first service`)                                  |
    | **Proveedor de nube**   | Elige AWS, GCP o Azure según prefieras                                                                        |
    | **Región**              | Selecciona la región más cercana para obtener la menor latencia posible                                       |
    | **Memoria y escalado**  | Selecciona **Mini 12GB** (1 réplica) para esta guía de inicio rápido; es la opción más rentable para aprender |

    Haz clic en **Create service** para aprovisionar tu servicio.

    <Frame caption="Configura tu servicio en la nube">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="Página de configuración del servicio de ClickHouse Cloud" />
    </Frame>

    Llegarás a la página principal del servicio, donde el estado aparecerá inicialmente como **Provisioning**. Esto suele tardar menos de un minuto. Una vez completado, el estado cambiará a **Running**.

    <Frame caption="Tu servicio está en ejecución">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="Página principal del servicio de ClickHouse Cloud que muestra el estado Running" />
    </Frame>
  </Step>

  <Step>
    ### Cargar un conjunto de datos de ejemplo

    Haz clic en **Data sources** en la barra lateral izquierda. Verás varias opciones para incorporar datos a tu servicio.

    <Frame caption="Página de Data sources con opciones para cargar datos">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="Página de Data sources de ClickHouse Cloud" />
    </Frame>

    Haz clic en **Add sample data** para ver los conjuntos de datos de ejemplo disponibles. Verás tres opciones: Cell Towers, UK Property Price Paid y New York Taxi Data. Haz clic en **Get started** en el conjunto de datos **UK Property Price Paid**.

    <Frame caption="Selecciona el conjunto de datos UK Property Price Paid">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="Selección de conjuntos de datos de ejemplo de ClickHouse Cloud" />
    </Frame>

    Verás una vista previa de los datos junto con el esquema de la tabla. Haz clic en **Import dataset** para cargar los datos en tu servicio.

    <Frame caption="Previsualiza los datos y haz clic en Import dataset">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="Vista previa de la importación del conjunto de datos de ClickHouse Cloud con datos de propiedades del Reino Unido" />
    </Frame>

    La importación tardará uno o dos minutos. Cuando termine, podrás consultar los datos.
  </Step>

  <Step>
    ### Consultar los datos

    Haz clic en **SQL Console** en la barra lateral izquierda para abrir el editor de consultas integrado. Aquí podrás escribir y ejecutar consultas SQL directamente en tu navegador, sin necesidad de herramientas externas.

    Prueba algunas consultas para explorar los datos de precios inmobiliarios del Reino Unido.

    Encuentra las propiedades más caras vendidas hasta la fecha:

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    Encuentre el precio de venta promedio por año:

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    Estas consultas escanean millones de filas en segundos: así funcionan el almacenamiento columnar y la compresión de ClickHouse.
  </Step>
</Steps>
