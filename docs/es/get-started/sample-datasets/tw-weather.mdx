Este conjunto de datos contiene observaciones meteorolГіgicas histГіricas de los Гәltimos 128 aГұos. Cada fila corresponde a una mediciГіn en una fecha y hora concretas y en una estaciГіn meteorolГіgica.

> Las fuentes de estos conjuntos de datos meteorolГіgicos incluyen las estaciones meteorolГіgicas establecidas por la Central Weather Administration (el cГіdigo de estaciГіn empieza por C0, C1 y 4) y las estaciones meteorolГіgicas agrГӯcolas pertenecientes al Council of Agriculture (cГіdigos de estaciГіn distintos de los mencionados anteriormente):

* StationId

  * SeaPres, la presiГіn al nivel del mar

  * RH, la humedad relativa

<div id="create-table-schema">
  ## Crear el esquema de la tabla
</div>

Cree la tabla MergeTree en ClickHouse (desde el cliente de ClickHouse).

```bash
CREATE TABLE tw_weather_data (
    StationId String null,
    MeasuredDate DateTime64,
    StnPres Float64 null,
    SeaPres Float64 null,
    Tx Float64 null,
    Td Float64 null,
    RH Float64 null,
    WS Float64 null,
    WD Float64 null,
    WSGust Float64 null,
    WDGust Float64 null,
    Precp Float64 null,
    PrecpHour Float64 null,
    SunShine Float64 null,
    GloblRad Float64 null,
    TxSoil0cm Float64 null,
    TxSoil5cm Float64 null,
    TxSoil10cm Float64 null,
    TxSoil20cm Float64 null,
    TxSoil50cm Float64 null,
    TxSoil100cm Float64 null,
    TxSoil30cm Float64 null,
    TxSoil200cm Float64 null,
    TxSoil300cm Float64 null,
    TxSoil500cm Float64 null,
    VaporPressure Float64 null,
    UVI Float64 null,
    "Cloud Amount" Float64 null,
    EvapA Float64 null,
    Visb Float64 null
)
ENGINE = MergeTree
ORDER BY (MeasuredDate);
```
