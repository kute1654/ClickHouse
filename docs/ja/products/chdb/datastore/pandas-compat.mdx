DataStore は、完全な API 互換性を実現するために **209 個の pandas DataFrame メソッド**を実装しています。既存の pandas コードも、ほとんど変更せずにそのまま動作します。

<div id="approach">
  ## 互換性へのアプローチ
</div>

```python
# 典型的な移行 - インポートを変更するだけ
- import pandas as pd
+ from chdb import datastore as pd

# コードはそのまま動作する
df = pd.read_csv("data.csv")
result = df[df['age'] > 25].groupby('city')['salary'].mean()
```

**主な原則:**

* pandas の 209 個の DataFrame メソッドをすべて実装
* SQL 最適化のための遅延評価
* 型の自動ラップ (DataFrame → DataStore、Series → ColumnExpr)
* イミュータブルな操作 (`inplace=True` はなし)

  ## 属性とプロパティ
</div>

| プロパティ     | 説明               | 実行をトリガーするか |
| --------- | ---------------- | ---------- |
| `shape`   |  (行数、カラム数) のタプル  | はい         |
| `columns` | カラム名 (Index)     | はい         |
| `dtypes`  | カラムのデータ型         | はい         |
| `values`  | NumPy配列          | はい         |
| `index`   | 行インデックス          | はい         |
| `size`    | 要素数              | はい         |
| `ndim`    | 次元数              | いいえ        |
| `empty`   | DataFrame が空かどうか | はい         |
| `T`       | 転置               | はい         |
| `axes`    | 軸のリスト            | はい         |

**例:**

```python
from chdb import datastore as pd

ds = pd.read_csv("data.csv")

print(ds.shape)      # (1000, 5)
print(ds.columns)    # Index(['name', 'age', 'city', 'salary', 'dept'])
print(ds.dtypes)     # name: object, age: int64, ...
print(ds.empty)      # False
```

  ## 索引参照と選択
</div>

| 方法                     | 説明            | 例                                    |
| ---------------------- | ------------- | ------------------------------------ |
| `df['col']`            | カラムを選択        | `ds['age']`                          |
| `df[['col1', 'col2']]` | 複数のカラムを選択     | `ds[['name', 'age']]`                |
| `df[condition]`        | ブール索引     | `ds[ds['age'] > 25]`                 |
| `df.loc[...]`          | ラベルベースのアクセス   | `ds.loc[0:10, 'name']`               |
| `df.iloc[...]`         | 整数位置ベースのアクセス  | `ds.iloc[0:10, 0:3]`                 |
| `df.at[...]`           | ラベルで単一の値にアクセス | `ds.at[0, 'name']`                   |
| `df.iat[...]`          | 位置で単一の値にアクセス  | `ds.iat[0, 0]`                       |
| `df.head(n)`           | 先頭の n 行       | `ds.head(10)`                        |
| `df.tail(n)`           | 末尾の n 行       | `ds.tail(10)`                        |
| `df.sample(n)`         | ランダムサンプル      | `ds.sample(100)`                     |
| `df.select_dtypes()`   | Dtype で選択     | `ds.select_dtypes(include='number')` |
| `df.query()`           | クエリ式          | `ds.query('age > 25')`               |
| `df.where()`           | 条件に応じて置換      | `ds.where(ds['age'] > 0, 0)`         |
| `df.mask()`            | `where` の逆    | `ds.mask(ds['age'] < 0, 0)`          |
| `df.isin()`            | 値が含まれるか判定     | `ds['city'].isin(['NYC', 'LA'])`     |
| `df.get()`             | 安全にカラムへアクセス   | `ds.get('col', default=None)`        |
| `df.xs()`              | 断面を取得         | `ds.xs('key')`                       |
| `df.pop()`             | カラムを削除        | `ds.pop('col')`                      |

<div id="differences">
  ## Pandasとの主な違い
</div>

<div id="return-types">
  ### 1. 戻り値の型
</div>

```python
# Pandas は Series を返す
pdf['col']  # → pd.Series

# DataStore は（遅延評価の）ColumnExpr を返す
ds['col']   # → ColumnExpr
```

<div id="lazy-execution">
  ### 2. 遅延実行
</div>

```python
# DataStore の操作は遅延評価されます
result = ds.filter(ds['age'] > 25)  # この時点ではまだ実行されません
df = result.to_df()  # ここで実行されます
```

<div id="no-inplace-parameter">
  ### 3. inplace パラメータはない
</div>

```python
# Pandas
df.drop(columns=['col'], inplace=True)

# DataStore（常に新しいオブジェクトを返す）
ds = ds.drop(columns=['col'])
```

<div id="comparing-results">
  ### 4. 結果を比較する
</div>

```python
# 比較には to_pandas() を使用します
pd.testing.assert_frame_equal(
    ds.to_pandas(),
    expected_df
)
```
