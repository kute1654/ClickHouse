一般的な pandas のパターンと、それに対応する DataStore での書き方を紹介します。ほとんどのコードはそのまま動作します！

<div id="loading">
  ## データの読み込み
</div>

<div id="read-csv">
  ### CSVを読み込む
</div>

```python
# Pandas
import pandas as pd
df = pd.read_csv("data.csv")

# DataStore - 同じです！
from chdb import datastore as pd
df = pd.read_csv("data.csv")
```

<div id="read-multiple-files">
  ### 複数のファイルを読み込む
</div>

```python
# Pandas
import glob
dfs = [pd.read_csv(f) for f in glob.glob("data/*.csv")]
df = pd.concat(dfs)

# DataStore - glob パターンを使用したより効率的な方法
df = pd.read_csv("data/*.csv")
```

<div id="filtering">
  ## フィルタリング
</div>

<div id="single-condition">
  ### 1つの条件
</div>

```python
# Pandas と DataStore - 同一
df[df['age'] > 25]
df[df['city'] == 'NYC']
df[df['name'].str.contains('John')]
```

<div id="multiple-conditions">
  ### 複数条件
</div>

```python
# AND
df[(df['age'] > 25) & (df['city'] == 'NYC')]

# OR
df[(df['age'] < 18) | (df['age'] > 65)]

# NOT
df[~(df['status'] == 'inactive')]
```

<div id="using-query">
  ### query() を使用する
</div>

```python
# Pandas と DataStore - 同一
df.query('age > 25 and city == "NYC"')
df.query('salary > 50000')
```

<div id="isin">
  ### isin()
</div>

```python
# Pandas と DataStore - 同一
df[df['city'].isin(['NYC', 'LA', 'SF'])]
```

<div id="between">
  ### between()
</div>

```python
# Pandas と DataStore - 同一
df[df['age'].between(18, 65)]
```

<div id="groupby">
  ## GroupBy と集計
</div>

<div id="simple-groupby">
  ### 基本的なGroupBy
</div>

```python
# Pandas と DataStore - 同一
df.groupby('city')['salary'].mean()
df.groupby('city')['salary'].sum()
df.groupby('city').size()  # カウント
```

<div id="multiple-aggregations">
  ### 複数の集計
</div>

```python
# Pandas と DataStore - 同一
df.groupby('city')['salary'].agg(['sum', 'mean', 'count'])

df.groupby('city').agg({
    'salary': ['sum', 'mean'],
    'age': ['min', 'max']
})
```

<div id="named-aggregations">
  ### 名前付き集計
</div>

```python
# Pandas と DataStore - 同一
df.groupby('city').agg(
    total_salary=('salary', 'sum'),
    avg_salary=('salary', 'mean'),
    employee_count=('id', 'count')
)
```

<div id="multiple-groupby-keys">
  ### 複数の GroupBy キー
</div>

```python
# Pandas と DataStore - 同一
```

<div id="new-columns">

</div>

<div id="simple-assignment">
  ### 基本的な代入
</div>


<div id="using-assign">
  ### assign() を使用する
</div>


<div id="conditional-where-mask">
  ### 条件処理 (where/mask)
</div>

```python
# Pandas と DataStore は同一

```

<div id="apply-for-custom-logic">
  ### カスタムロジック用の apply()
</div>

```python
# 動作しますが、pandas の実行がトリガーされます
df['category'] = df['amount'].apply(lambda x: 'high' if x > 1000 else 'low')

# DataStore の代替手段（遅延実行のまま）
df['category'] = (
    df.when(df['amount'] > 1000, 'high')
      .otherwise('low')
)
```
