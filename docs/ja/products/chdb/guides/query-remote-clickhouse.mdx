このガイドでは、chDB からリモート ClickHouse サーバーにクエリを実行する方法を学びます。

<div id="setup">
  ## セットアップ
</div>

まず、仮想環境を作成しましょう：

```bash
python -m venv .venv
source .venv/bin/activate
```

それでは、chDB をインストールしましょう。
バージョン 2.0.2 以上であることを確認してください。

```bash
pip install "chdb>=2.0.2"
```

それでは、pandas と ipython をインストールします。

```bash
pip install pandas ipython
```

このガイドの以降のコマンド実行には `ipython` を使用します。次を実行して起動できます。

```bash
ipython
```

このコードは、Pythonスクリプトやお好みのノートブックでも使用できます。

<div id="an-intro-to-clickpy">
  ## ClickPy の紹介
</div>

これからクエリを実行するリモートの ClickHouseサーバーは、[ClickPy](https://clickpy.clickhouse.com) です。
ClickPy は PyPI パッケージのすべてのダウンロードを記録しており、UI からパッケージの統計情報を確認できます。
基盤となるデータベースには、`play` ユーザーを使ってクエリを実行できます。

ClickPy の詳細については、[GitHub リポジトリ](https://github.com/ClickHouse/clickpy) を参照してください。

<div id="querying-the-clickpy-clickhouse-service">
  ## ClickPy の ClickHouse service をクエリする
</div>

まず、chDB をインポートしましょう:

```python
import chdb
```

`remoteSecure` 関数を使って、ClickPy にクエリを実行します。
この関数には、少なくともホスト名、テーブル名、username を指定する必要があります。

次のクエリを記述すると、[`openai` パッケージ](https://clickpy.clickhouse.com/dashboard/openai) の日ごとのダウンロード数を Pandas DataFrame として取得できます。

```python
query = """
SELECT
    toStartOfDay(date)::Date32 AS x,
    sum(count) AS y
FROM remoteSecure(
  'clickpy-clickhouse.clickhouse.com', 
  'pypi.pypi_downloads_per_day', 
  'play'
)
WHERE project = 'openai'
GROUP BY x
ORDER BY x ASC
"""

openai_df = chdb.query(query, "DataFrame")
openai_df.sort_values(by=["x"], ascending=False).head(n=10)
```

では、[`scikit-learn`](https://clickpy.clickhouse.com/dashboard/scikit-learn) についても同様に、ダウンロード数を返してみましょう。

```python
query = """
```

<div id="merging-pandas-dataframes">
  ## Pandas DataFrame の結合
</div>

これで 2 つの DataFrame がそろったので、日付 (`x` カラム) をキーとして、次のように結合できます。

```python
df = openai_df.merge(
  sklearn_df, 
  on="x", 
  suffixes=("_openai", "_sklearn")
)
df.head(n=5)
```

```text
            x  y_openai  y_sklearn
0  2018-02-26        83      33971
1  2018-02-27        31      25211
2  2018-02-28         8      26023
3  2018-03-01         8      20912
4  2018-03-02         5      23842
```

次に、Open AI のダウンロード数と `scikit-learn` のダウンロード数の比率は、次のように計算できます。

```python
df['ratio'] = df['y_openai'] / df['y_sklearn']
df.head(n=5)
```

```text
            x  y_openai  y_sklearn     ratio
0  2018-02-26        83      33971  0.002443
1  2018-02-27        31      25211  0.001230
2  2018-02-28         8      26023  0.000307
3  2018-03-01         8      20912  0.000383
4  2018-03-02         5      23842  0.000210
```

<div id="querying-pandas-dataframes">
  ## Pandas DataFrames をクエリする
</div>

次に、比率が最も高い日と最も低い日を見つけるとします。
chDB に戻って、それらの値を算出できます。

```python
chdb.query("""
SELECT max(ratio) AS bestRatio,
       argMax(x, ratio) AS bestDate,
       min(ratio) AS worstRatio,
       argMin(x, ratio) AS worstDate
FROM Python(df)
""", "DataFrame")
```

```text
   bestRatio    bestDate  worstRatio   worstDate
0   0.693855  2024-09-19    0.000003  2020-02-09
```

Pandas DataFrames のクエリについてさらに詳しくは、[Pandas DataFrames 開発者ガイド](/ja/products/chdb/guides/querying-pandas)を参照してください。
