データフォーマットに関して、chDB は ClickHouse と 100% の機能互換性があります。

入力フォーマットは、`File`、`URL`、`S3` などのファイルベースのテーブルに対する `INSERT` や `SELECT` に渡されるデータをパースするために使用されます。
出力フォーマットは、`SELECT` の結果を整形したり、ファイルベースのテーブルに `INSERT` したりするために使用されます。
chDB は、ClickHouse がサポートするデータフォーマットに加えて、以下もサポートしています。

* 出力フォーマットとしての `ArrowTable`。型は Python の `pyarrow.Table` です
* 入出力フォーマットとしての `DataFrame`。型は Python の `pandas.DataFrame` です。例については、[`test_joindf.py`](https://github.com/chdb-io/chdb-core/blob/main/tests/test_joindf.py) を参照してください
* 出力としての `Debug` (`CSV` のエイリアス) 。ClickHouse のデバッグ用詳細出力が有効になります
