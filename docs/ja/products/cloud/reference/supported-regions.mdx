<div id="private-regions">
  ## プライベートリージョン
</div>

<EnterprisePlanFeatureBadge feature="プライベートリージョン機能" />

Enterprise tier のサービスでは、プライベートリージョンを提供しています。プライベートリージョンをご希望の場合は、[お問い合わせください](https://clickhouse.com/company/contact)。

プライベートリージョンに関する主な注意点:

* サービスは自動スケーリングされませんが、手動での垂直スケーリングおよび水平スケーリングには対応しています。
* サービスをアイドル状態にすることはできません。
* プライベートリージョンではステータスページを利用できません。

HIPAA コンプライアンスには、追加要件が適用される場合があります (BAA の締結を含む) 。なお、HIPAA は現在 Enterprise tier のサービスでのみ利用可能です

<div id="hipaa-compliant-regions">
  ## HIPAA 準拠リージョン
</div>

<EnterprisePlanFeatureBadge feature="HIPAA" support="true" />

HIPAA 準拠リージョンでサービスをセットアップするには、Business Associate Agreement (BAA) に署名し、営業またはサポートを通じてオンボーディングを依頼する必要があります。HIPAA 準拠に対応しているリージョンは次のとおりです。

* AWS af-south-1 (南アフリカ)  **プライベートリージョン**
* AWS ca-central-1 (カナダ)  **プライベートリージョン**
* AWS eu-central-1 (フランクフルト) 
* AWS eu-north-1 (ストックホルム)  **プライベートリージョン**
* AWS eu-west-1 (アイルランド) 
* AWS eu-west-2 (ロンドン) 
* AWS sa-east-1 (南アメリカ)  **プライベートリージョン**
* AWS us-east-1 (北バージニア) 
* AWS us-east-2 (オハイオ) 
* AWS us-west-2 (オレゴン) 
* GCP europe-west4 (オランダ) 
* GCP us-central1 (アイオワ) 
* GCP us-east1 (サウスカロライナ)
