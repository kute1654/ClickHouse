<Note>
  MySQL および MongoDB CDC ClickPipes は、一般提供 (GA) までは無料で利用できます。GA 開始前には、お客様が ClickPipes の利用状況を確認して最適化できるよう、事前にお知らせします。
</Note>

[ClickPipes](/ja/integrations/clickpipes/home) の課金は、**コンピュート使用量**と**取り込みデータ**に基づきます。各カテゴリの料金モデルの詳細については、各専用の課金ページを参照してください。
