<div id="console-audit-log-events">
  ## Console 監査ログ のイベント
</div>

組織で取得される各種イベントは、**Organization**、**Service**、**User** の 3 つのカテゴリに分類されています。監査ログの詳細や、API インテグレーションをエクスポートまたは追加する方法については、上記の Guides セクションにある [console audit log](/ja/products/cloud/guides/security/audit-logging/console-audit-log) のドキュメントを参照してください。

以下のイベントが監査ログに記録されます。

<div id="organization">
  ### 組織
</div>

* 組織が作成されました
* 組織が削除されました
* 組織名が変更されました

<div id="service">
  ### サービス
</div>

* サービスが作成された
* サービスが削除された
* サービスが停止された
* サービスが起動された
* サービス名が変更された
* サービスの IP Access List が変更された
* サービスのパスワードがリセットされた

<div id="user">
  ### ユーザー
</div>

* ユーザーのロールが変更された
* ユーザーが組織から削除された
* ユーザーが組織に招待された
* ユーザーが組織に参加した
* ユーザーへの招待が削除された
* ユーザーが組織を脱退した
