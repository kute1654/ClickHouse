<div id="notes">
  ## 注記
</div>

* **operator の Helm チャートのバージョン** (`1.18587.1`) は、operator の **イメージタグ** (`main-1.18587.1`) と異なります。`helm install` でインストールする場合は Helm チャートのバージョンを使用し、コンテナーイメージをコピーする場合はイメージタグを使用してください。
* **FIPS ビルド** では、server と Keeper の両方で、標準のイメージタグの末尾に `-fips` が追加されます。
* これらのドキュメントのビルド時に、バージョンタグは自動的に置換されます。未展開のテンプレート変数 (例: `25.12.1.1497`) が表示されている場合は、ビルド手順がまだ実行されていません。
