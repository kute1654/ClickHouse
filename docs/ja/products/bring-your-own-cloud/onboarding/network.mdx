ClickHouse BYOC では、セキュリティを強化し、サービスへの直接接続を可能にするために、さまざまなプライベートネットワーキングのオプションをサポートしています。このガイドでは、お客様の AWS または GCP アカウント内の ClickHouse Cloud デプロイメントを、社内アプリケーションや分析ツールなどの他のネットワークやサービスに安全に接続するための推奨アプローチを説明します。VPC Peering、AWS PrivateLink、GCP Private Service Connect などのオプションを取り上げ、それぞれの主な手順と考慮事項を概説します。

ClickHouse BYOC デプロイメントへのプライベートネットワーク接続が必要な場合は、各ガイドの手順に従ってください。より高度なシナリオについては、ClickHouse Support にお問い合わせください。
