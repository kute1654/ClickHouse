ClickHouse は、ClickHouse Cloud のユーザーおよびお客様向けにサポートサービスを提供しています。私たちの目標は、ClickHouse 製品そのものを体現するサポートチームを提供することです。すなわち、比類のない性能、使いやすさ、そして卓越したスピードと高品質な対応です。詳細については、[ClickHouse Support Program](https://clickhouse.com/support/program/) ページをご覧ください。

[Cloud Console にログイン](https://console.clickhouse.cloud/support)し、メニューから **Help -&gt; Support** を選択すると、新しいサポートケースを作成したり、送信済みケースのステータスを確認したりできます。

また、[ステータスページ](https://status.clickhouse.com) を購読すると、当社プラットフォームに影響するインシデントについて迅速に通知を受け取れます。

<Note>
  サポートインシデントに関するサービスレベル契約が適用されるのは、サブスクリプション契約のお客様のみである点にご留意ください。現在 ClickHouse Cloud ユーザーでない場合でも、ご質問には可能な限り回答いたしますが、代わりに以下のコミュニティリソースのご利用をお勧めします。

  * [ClickHouse コミュニティ Slack チャンネル](https://clickhouse.com/slack)
  * [その他のコミュニティオプション](https://github.com/ClickHouse/ClickHouse/blob/master/README.md#useful-links)
</Note>
