ClickHouse Cloud は、人気の高いオープンソース OLAP データベースである ClickHouse を開発したチームが提供するクラウドサービスです。
[無料トライアルを開始](https://console.clickhouse.cloud/signUp)して、ClickHouse Cloud をお試しください。

<div id="clickhouse-cloud-benefits">
  ## ClickHouse Cloud のメリット
</div>

ClickHouse Cloud を利用する主なメリットは次のとおりです。

* **価値実現までの時間を短縮**: クラスターのサイジングやスケーリングを行わなくても、すぐに構築を始められます。
* **シームレスなスケーリング**: 自動スケーリングが変動するワークロードに対応するため、ピーク時に備えて過剰にプロビジョニングする必要がありません。
* **サーバーレス運用**: サイジング、スケーリング、セキュリティ、信頼性の確保、アップグレードは当社にお任せください。
* **透明性の高い料金体系**: リソース予約やスケーリング制御を活用しながら、使った分だけお支払いいただけます。
* **総保有コスト**: 優れた価格性能比と低い運用管理負荷を実現します。
* **幅広いエコシステム**: お使いのデータコネクタ、可視化ツール、SQL、各種言語クライアントをそのまま利用できます。

<div id="oss-vs-clickhouse-cloud">
  ## ClickHouse Cloud ではどのバージョンの ClickHouse を使用していますか？
</div>

ClickHouse Cloud では、修正、新機能、パフォーマンス向上を含む新しいバージョンへ、お客様のサービスを定期的にアップグレードしています。コアデータベースのバージョンを open source で公開した後、Cloud の staging 環境で追加の検証を行います。通常、本番環境への rollout までに 6〜8 週間かかります。rollout は、cloud service provider、service の種類、および region ごとに段階的に実施されます。

特定の release channel を購読することで、ClickHouse Cloud サービスのアップグレード schedule を指定できます。たとえば、通常の release schedule より早く更新を受け取るための [&quot;Fast&quot; Release Channel](/ja/products/cloud/features/admin-features/upgrades#fast-release-channel-early-upgrades) や、アップグレードを延期するための [&quot;Slow&quot; Release Channel](/ja/products/cloud/features/admin-features/upgrades#slow-release-channel-deferred-upgrades)、さらに細かい schedule を設定できるその他のオプションを提供しています。

後方互換性の保証を含む ClickHouse Cloud のアップグレード process の概要については、[Upgrades](/ja/products/cloud/features/admin-features/upgrades) の reference ドキュメントを参照してください。
