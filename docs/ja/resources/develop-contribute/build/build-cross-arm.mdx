Aarch64 マシン上で Aarch64 向けに ClickHouse をビルドする場合、特別な手順は必要ありません。

x86 Linux マシン上で AArch64 向けに ClickHouse をクロスコンパイルするには、`cmake` に次のフラグを指定します: `-DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-aarch64.cmake`
