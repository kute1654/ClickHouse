<div id="what-youll-build">
  ## 作成するもの
</div>

このクイックスタートでは、最初の ClickHouse Cloud サービスを作成し、Cloud コンソールと組み込みの SQL コンソールの基本を確認したうえで、実際にクエリできるデータを用意するためにサンプルデータセットを読み込みます。
完了するころには、データが読み込まれた稼働中の ClickHouse Cloud サービスが用意され、このクイックスタートシリーズの残りを進めるための土台として使えるようになります。

<Steps titleSize="h3">
  <Step>
    ### ClickHouse Cloud に登録する

    [console.clickhouse.cloud](https://console.clickhouse.cloud/) にアクセスします。サインインページが表示されたら、下部の「Don&#39;t have an account yet?」の横にある **Register** をクリックします。

    <Frame caption="新しいアカウントを作成するには Register をクリック">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="Register リンクが表示された ClickHouse Cloud のサインインページ" />
    </Frame>

    登録ページで、Google または Microsoft で登録するか、業務用メールアドレスを入力します。クレジットカードは不要です。無料トライアルでは、300 ドル分のクレジットと、ClickHouse Cloud を試用できる 30 日間が提供されます。

    <Frame caption="Google、Microsoft、または業務用メールアドレスで登録">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="ClickHouse Cloud の登録ページ" />
    </Frame>

    メールアドレスで登録する場合は、パスワードの設定を求められます。パスワードは次の要件を満たす必要があります。

    * 12 文字以上
    * 次のうち少なくとも 3 種類: 英小文字、英大文字、数字、特殊文字 (例: `!@#$%^&*`)

    <Frame caption="パスワードを入力し、Terms of Service に同意">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="パスワード フィールドがある ClickHouse Cloud のメール登録画面" />
    </Frame>

    Terms of Service と Privacy Policy に同意し、**Sign up** をクリックします。確認リンクが記載されたメールを確認し、**Verify email** をクリックしてアカウントを有効化します。

    確認が完了したら、新しいメールアドレスとパスワードでサインインします。
  </Step>

  <Step>
    ### 新しいサービスを作成する

    初回サインイン後、すぐにサービス設定ページが表示されます。次の項目を設定してください。

    | 設定             | 推奨値                                                                 |
    | -------------- | ------------------------------------------------------------------- |
    | **サービス名**      | わかりやすい名前を付けます (例: `My first service`)                               |
    | **クラウドプロバイダー** | 好みに応じて AWS、GCP、または Azure を選択します                                     |
    | **リージョン**      | レイテンシを最小限に抑えるため、最寄りのリージョンを選択します                                     |
    | **メモリとスケーリング** | このクイックスタートでは **Mini 12GB** (1 レプリカ) を選択してください。学習用途では最も費用対効果の高い選択肢です |

    **Create service** をクリックして、サービスをプロビジョニングします。

    <Frame caption="クラウドサービスを設定する">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="ClickHouse Cloud service の設定ページ" />
    </Frame>

    サービスのホームページが表示され、最初はステータスが **Provisioning** と表示されます。通常、1 分もかかりません。完了すると、ステータスは **Running** に変わります。

    <Frame caption="サービスは稼働中です">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="Running status が表示された ClickHouse Cloud service のホームページ" />
    </Frame>
  </Step>

  <Step>
    ### サンプルデータセットを読み込む

    左側のサイドバーで **Data sources** をクリックします。サービスにデータを取り込むためのオプションがいくつか表示されます。

    <Frame caption="データ読み込みオプションが表示された Data sources ページ">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="ClickHouse Cloud の Data sources ページ" />
    </Frame>

    利用可能なサンプルデータセットを確認するには、**Add sample data** をクリックします。3 つのオプション (Cell Towers、UK Property Price Paid、New York Taxi Data) が表示されます。**UK Property Price Paid** データセットの **Get started** をクリックします。

    <Frame caption="UK Property Price Paid データセットを選択">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="ClickHouse Cloud のサンプルデータセット選択画面" />
    </Frame>

    テーブルのスキーマとともにデータのプレビューが表示されます。**Import dataset** をクリックして、サービスにデータを読み込みます。

    <Frame caption="データをプレビューして Import dataset をクリック">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="UK の不動産データを含む ClickHouse Cloud のデータセットインポートプレビュー" />
    </Frame>

    インポートには 1 ～ 2 分ほどかかります。完了すると、データをクエリできるようになります。
  </Step>

  <Step>
    ### データをクエリする

    左側のサイドバーで **SQL Console** をクリックし、組み込みのクエリエディタを開きます。ここではブラウザ上で直接 SQL クエリを記述して実行できます。外部ツールは必要ありません。

    いくつかクエリを試して、英国の不動産価格データを見てみましょう。

    これまでに売却された中で最も高額な物件を見つけてみましょう:

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    年ごとの平均販売価格を求めます。

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    これらのクエリが数秒で数百万行をスキャンできるのは、ClickHouseの列指向ストレージと圧縮の効果です。
  </Step>
</Steps>
