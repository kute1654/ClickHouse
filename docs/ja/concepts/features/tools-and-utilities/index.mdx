| ツール/ユーティリティ                                                                                 | 説明                                                         |
| ------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| [clickhouse-local](/ja/concepts/features/tools-and-utilities/clickhouse-local)                 | `awk` と同様に、ClickHouse server を起動せずにデータに対して SQL クエリを実行できます。 |
| [clickhouse-benchmark](/ja/concepts/features/tools-and-utilities/clickhouse-benchmark)         | カスタムクエリと設定を使ってサーバーに負荷をかけます。                                |
| [clickhouse-format](/ja/concepts/features/tools-and-utilities/clickhouse-format)               | 入力クエリをフォーマットできます。                                          |
| [ClickHouse obfuscator](/ja/concepts/features/tools-and-utilities/clickhouse-obfuscator)       | データを難読化します。                                                |
| [ClickHouse compressor](/ja/concepts/features/tools-and-utilities/clickhouse-compressor)       | データを圧縮・展開します。                                              |
| [clickhouse-disks](/ja/concepts/features/tools-and-utilities/clickhouse-disks)                 | 異なる ClickHouse ディスク上のファイルに対して、ファイルシステムのような操作を提供します。        |
| [clickhouse-odbc-bridge](/ja/concepts/features/tools-and-utilities/odbc-bridge)                | ODBC ドライバ用のプロキシサーバーです。                                     |
| [clickhouse&#95;backupview](/ja/concepts/features/tools-and-utilities/backupview)              | ClickHouse バックアップを分析するための Python モジュールです。                  |
| [clickhouse-keeper-client](/ja/concepts/features/tools-and-utilities/clickhouse-keeper-client) | ClickHouse Keeper とやり取りするためのクライアントアプリケーションです。              |
| [Keeper HTTP API](/ja/concepts/features/tools-and-utilities/clickhouse-keeper-http-api)        | ClickHouse Keeper 用の HTTP API と組み込みダッシュボードです。              |
