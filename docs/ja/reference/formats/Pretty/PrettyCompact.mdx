<div id="description">
  ## 説明
</div>

[`Pretty`](/ja/reference/formats/Pretty/Pretty)フォーマットとは異なり、テーブルは行間に罫線が引かれたグリッド付きで表示されます。
そのため、結果はよりコンパクトになります。

<Note>
  このフォーマットは、対話型モードのコマンドラインクライアントでデフォルトで使用されます。
</Note>
