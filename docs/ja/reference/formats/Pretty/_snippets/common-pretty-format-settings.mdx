---
title: Prettyフォーマットの共通設定
---

{/* 注: このファイルは、このファイルをインポートするすべてのファイルでスニペットとして使用されます */}

以下の設定は、すべての`Pretty`フォーマットに共通です。

| 設定                                                                                                                                                   | 説明                                                                                                                                                               | デフォルト   |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| [`output_format_pretty_max_rows`](/ja/reference/settings/formats#output_format_pretty_max_rows)                                                         | Prettyフォーマットの行数上限です。                                                                                                                                             | `10000` |
| [`output_format_pretty_max_column_pad_width`](/ja/reference/settings/formats#output_format_pretty_max_column_pad_width)                                 | Prettyフォーマットで、カラム内のすべての値に適用するパディングの最大幅です。                                                                                                                        | `250`   |
| [`output_format_pretty_max_value_width`](/ja/reference/settings/formats#output_format_pretty_max_value_width)                                           | Prettyフォーマットで表示する値の最大幅です。これを超える場合は切り詰められます。                                                                                                                      | `10000` |
| [`output_format_pretty_color`](/ja/reference/settings/formats#output_format_pretty_color)                                                               | ANSIエスケープシーケンスを使用して、Prettyフォーマットに色を付けます。                                                                                                                         | `true`  |
| [`output_format_pretty_grid_charset`](/ja/reference/settings/formats#output_format_pretty_grid_charset)                                                 | 罫線の表示に使用する文字セットです。使用可能な文字セット: ASCII、UTF-8。                                                                                                                       | `UTF-8` |
| [`output_format_pretty_row_numbers`](/ja/reference/settings/formats#output_format_pretty_row_numbers)                                                   | Pretty出力フォーマットで、各行の前に行番号を追加します。                                                                                                                                  | `true`  |
| [`output_format_pretty_display_footer_column_names`](/ja/reference/settings/formats#output_format_pretty_display_footer_column_names)                   | テーブルに多数の行が含まれる場合、フッターにカラム名を表示します。                                                                                                                                | `true`  |
| [`output_format_pretty_display_footer_column_names_min_rows`](/ja/reference/settings/formats#output_format_pretty_display_footer_column_names_min_rows) | [`output_format_pretty_display_footer_column_names`](/ja/reference/settings/formats#output_format_pretty_display_footer_column_names) が有効な場合に、フッターを表示する最小の行数を設定します。 | `50`    |
