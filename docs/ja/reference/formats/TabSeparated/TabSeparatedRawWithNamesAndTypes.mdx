<div id="description">
  ## 説明
</div>

[`TabSeparatedWithNamesAndTypes`](/ja/reference/formats/TabSeparated/TabSeparatedWithNamesAndTypes) フォーマットとは異なり、
行はエスケープせずに書き込まれます。

<Note>
  このフォーマットのパース時には、各フィールド内でタブまたは改行を使用できません。
</Note>

</div>

```sql

FROM football
FORMAT TabSeparatedRawWithNamesAndTypes
```
