<div id="description">
  ## 説明
</div>

[`TabSeparated`](/ja/reference/formats/TabSeparated/TabSeparated)フォーマットとは異なり、このフォーマットでは行がエスケープされずに書き込まれます。

<Note>
  このフォーマットをパースする際、各フィールド内にタブや改行を含めることはできません。
</Note>

`TabSeparatedRaw`フォーマットと`RawBlob`フォーマットの比較については、[Raw Formats Comparison](/ja/reference/formats/RawBLOB#raw-formats-comparison)を参照してください。

```sql
SELECT *
FROM football
FORMAT TabSeparatedRaw
```
