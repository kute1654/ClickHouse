<div id="description">
  ## 説明
</div>

最初の行にカラム名が記述される点が、[`TabSeparated`](/ja/reference/formats/TabSeparated/TabSeparated)フォーマットとは異なります。

パース時には、最初の行にカラム名が含まれていることを想定しています。カラム名を使ってカラムの位置を特定し、正しいかどうかを確認できます。

<Note>
  設定 [`input_format_with_names_use_header`](/ja/reference/settings/formats#input_format_with_names_use_header) が `1` の場合、
  入力データのカラムは名前に基づいてテーブルのカラムにマッピングされ、設定 [`input_format_skip_unknown_fields`](/ja/reference/settings/formats#input_format_skip_unknown_fields) が `1` であれば、不明な名前のカラムはスキップされます。
  それ以外の場合、最初の行はスキップされます。
</Note>

データを挿入します:

```sql

```

<div id="reading-data">
  ### データの読み取り
</div>

`TabSeparatedWithNames` フォーマットを使用してデータを読み取ります。

```sql
SELECT *
FROM football
FORMAT TabSeparatedWithNames
```
