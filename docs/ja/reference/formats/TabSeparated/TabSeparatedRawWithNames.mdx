<div id="description">
  ## 説明
</div>

[`TabSeparatedWithNames`](/ja/reference/formats/TabSeparated/TabSeparatedWithNames) フォーマットとの違いは、
このフォーマットでは行がエスケープされずに書き込まれる点です。

<Note>
  このフォーマットでパースする場合、各フィールド内にタブや改行を含めることはできません。
</Note>

</div>

<div id="reading-data">
  ### データの読み取り
</div>

データを読み取るには、`TabSeparatedRawWithNames` フォーマットを使用します。

```sql
SELECT *
FROM football
FORMAT TabSeparatedRawWithNames
```
