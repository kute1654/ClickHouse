<div id="description">
  ## 説明
</div>

`Null` フォーマットでは、何も出力されません。
一見すると奇妙に思えるかもしれませんが、何も出力されなくてもクエリ自体は処理されます。さらに、コマンドラインクライアントを使用している場合は、データはクライアントに送信されます。

<Tip>
  `Null` フォーマットは、パフォーマンステストに役立つことがあります。
</Tip>
