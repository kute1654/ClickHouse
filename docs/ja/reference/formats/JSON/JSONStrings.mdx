<div id="description">
  ## 説明
</div>

[JSON](/ja/reference/formats/JSON/JSON)フォーマットとの違いは、データフィールドが型付きの JSON 値ではなく、文字列として出力される点のみです。これは出力専用のフォーマットです。

<div id="example-usage">
  ## 使用例
</div>

<div id="reading-data">
  ### データの読み込み
</div>

`JSONStrings` フォーマットを使用してデータを読み込みます。

```sql
SELECT *
FROM football
FORMAT JSONStrings
```
