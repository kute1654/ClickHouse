<div id="description">
  ## 概要
</div>

このフォーマットでは、ClickHouse は各行を独立した改行区切りのJSONオブジェクトとして出力します。

このフォーマットは、`JSONEachRow`、`JSONLines`、`NDJSON` (改行区切りJSON)、または `JSONL` とも呼ばれます。これらはすべて同じフォーマットの別名であり、入力と出力の両方で相互に置き換えて使用できます。

</div>

<div id="reading-data">
  ### データの読み込み
</div>

`JSONLines`フォーマットを使用してデータを読み込みます。

```sql
SELECT *
FROM football
FORMAT JSONLines
```
