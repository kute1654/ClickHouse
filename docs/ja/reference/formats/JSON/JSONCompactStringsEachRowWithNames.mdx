<div id="description">
  ## 説明
</div>

[`JSONCompactEachRow`](/ja/reference/formats/JSON/JSONCompactEachRow) フォーマットとの違いは、[TabSeparatedWithNames](/ja/reference/formats/TabSeparated/TabSeparatedWithNames) フォーマットと同様に、カラム名を含むヘッダー行も出力する点です。

<div id="example-usage">
  ## 使用例
</div>

<div id="inserting-data">
  ### データの挿入
</div>

次のデータが含まれる `football.json` という名前の JSON ファイルを使用します。

["2022-05-07", "2021", "Bradford City", "Carlisle United", "2", "0"]

データを挿入します:

```sql

```

FROM football
FORMAT JSONCompactStringsEachRowWithNames

<div id="format-settings">
  ## フォーマット設定
</div>

<Note>
  設定 [`input_format_with_names_use_header`](/ja/reference/settings/formats#input_format_with_names_use_header) が `1` の場合、
  入力データのカラムは名前に基づいてテーブルのカラムに対応付けられ、設定 [`input_format_skip_unknown_fields`](/ja/reference/settings/formats#input_format_skip_unknown_fields) も `1` の場合は、不明な名前のカラムはスキップされます。
  それ以外の場合は、最初の行がスキップされます。
</Note>
