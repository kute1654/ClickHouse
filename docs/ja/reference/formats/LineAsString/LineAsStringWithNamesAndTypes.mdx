<div id="description">
  ## 説明
</div>

`LineAsStringWithNames` フォーマットは [`LineAsString`](/ja/reference/formats/LineAsString/LineAsString) フォーマットに似ていますが、
2 つのヘッダー行を出力します。1 つはカラム名、もう 1 つは型情報です。

<div id="example-usage">
  ## 使用例
</div>

```sql title="Query"
CREATE TABLE example (
    name String,
    value Int32
)
ENGINE = Memory;

INSERT INTO example VALUES ('John', 30), ('Jane', 25), ('Peter', 35);

SELECT * FROM example FORMAT LineAsStringWithNamesAndTypes;
```

```response title="Response"
name    value
String    Int32
John    30
Jane    25
Peter    35
```
