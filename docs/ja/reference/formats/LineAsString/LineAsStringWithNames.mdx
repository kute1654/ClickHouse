<div id="description">
  ## 説明
</div>

`LineAsStringWithNames` フォーマットは [`LineAsString`](/ja/reference/formats/LineAsString/LineAsString) フォーマットに似ていますが、カラム名を含むヘッダー行を出力します。

<div id="example-usage">
  ## 使用例
</div>

```sql title="Query"
CREATE TABLE example (
    name String,
    value Int32
)
ENGINE = Memory;

INSERT INTO example VALUES ('John', 30), ('Jane', 25), ('Peter', 35);

SELECT * FROM example FORMAT LineAsStringWithNames;
```

```response title="Response"
name    value
John    30
Jane    25
Peter    35
```
