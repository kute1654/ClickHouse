ビューの実装に使用されます (詳細は `CREATE VIEW query` を参照してください) 。データは保存せず、指定された `SELECT` クエリのみを保持します。テーブルから読み取る際には、このクエリを実行し、クエリから不要なカラムをすべて削除します。
