<div id="IPv4CIDRToRange">
  ## IPv4CIDRToRange
</div>

導入バージョン: v20.1.0

IPv4 アドレスとその Classless Inter-Domain Routing (CIDR) プレフィックス長を受け取り、そのサブネットのアドレス範囲を、2 つの IPv4 値からなるタプルとして返します。つまり、そのサブネットの先頭アドレスと末尾アドレスです。
IPv6 版については [`IPv6CIDRToRange`](#IPv4CIDRToRange) を参照してください。

**構文**

```sql
IPv4CIDRToRange(ipv4, cidr)
```

**引数**

* `ipv4` — IPv4 アドレス。[`IPv4`](/ja/reference/data-types/ipv4) または [`String`](/ja/reference/data-types/string)
* `cidr` — CIDR 値。[`UInt8`](/ja/reference/data-types/int-uint)

**戻り値**

サブネットの範囲を表す 2 つの IPv4 アドレスからなるタプルを返します。[`Tuple(IPv4, IPv4)`](/ja/reference/data-types/tuple)

**例**

**使用例**

```sql title=Query
SELECT IPv4CIDRToRange(toIPv4('192.168.5.2'), 16);
```

```response title=Response
┌─IPv4CIDRToRange(toIPv4('192.168.5.2'), 16)─┐
│ ('192.168.0.0','192.168.255.255')          │
└────────────────────────────────────────────┘
```

MAC アドレスを表す数値を返します。形式が無効な場合は `0` を返します。[`String`](/ja/reference/data-types/string)

**例**

**使用例**

```sql title=Query

```


**例**

**基本例**

```sql title=Query
SELECT addr, cutIPv6(IPv6StringToNum(addr), 0, 0) FROM (SELECT ['notaddress', '127.0.0.1', '1111::ffff'] AS addr) ARRAY JOIN addr;
```

```response title=Response
┌─addr───────┬─cutIPv6(IPv6StringToNum(addr), 0, 0)─┐
```

**例**

**使用例**

```sql title=Query

    IPv6StringToNum('2001:0DB8:AC10:FE01:FEED:BABE:CAFE:F00D') AS ipv6,
    IPv4ToIPv6(IPv4StringToNum('192.168.0.1')) AS ipv4

    cutIPv6(ipv4, 0, 2)
```

<div id="isIPAddressInRange">
  ## isIPAddressInRange
</div>

導入バージョン: v21.4.0

IP アドレスが、[Classless Inter-Domain Routing (CIDR)](https://en.wikipedia.org/wiki/Classless_Inter-Domain_Routing) 表記のネットワークに含まれているかどうかを判定します。

この関数は、文字列として表された IPv4 と IPv6 の両方のアドレス (およびネットワーク) を受け取ります。アドレスの IP バージョンと CIDR が一致しない場合は、`0` を返します。

**構文**

```sql
isIPAddressInRange(address, prefix)
```

**引数**

* `address` — IPv4 または IPv6 のアドレス。[`String`](/ja/reference/data-types/string)
* `prefix` — CIDR 形式の IPv4 または IPv6 ネットワークプレフィックス。[`String`](/ja/reference/data-types/string)

**戻り値**

アドレスの IP バージョンと CIDR が一致する場合は `1`、それ以外の場合は `0` を返します。[`UInt8`](/ja/reference/data-types/int-uint)

**例**

**範囲内の IPv4 アドレス**

```sql title=Query
SELECT isIPAddressInRange('127.0.0.1', '127.0.0.0/8')
```

```response title=Response
1
```

**範囲外のIPv4アドレス**

```sql title=Query
SELECT isIPAddressInRange('127.0.0.1', 'ffff::/16')
```

```response title=Response
0
```

**範囲外のIPv6アドレス**

```sql title=Query
SELECT isIPAddressInRange('::ffff:192.168.0.1', '::ffff:192.168.0.4/128')
```

```response title=Response
0
```
