幾何オブジェクトを扱うための関数です。たとえば、[球面上の点間の距離を計算する](/ja/reference/functions/regular-functions/geo/coordinates)、[geohash を計算する](/ja/reference/functions/regular-functions/geo/geohash)、[h3 index を扱う](/ja/reference/functions/regular-functions/geo/h3)、および [Mapbox Vector Tiles をエンコードする](/ja/reference/functions/regular-functions/geo/mvt)関数があります。
