指定された入力フォーマットに従って、引数内のデータを解析します。structure 引数が指定されていない場合は、データから抽出されます。

<div id="syntax">
  ## 構文
</div>

```sql
format(format_name, [structure], data)
```

<div id="arguments">
  ## 引数
</div>

* `format_name` — データの[フォーマット](/ja/reference/formats/index)。
* `structure` - テーブルの構造。省略可能。形式は &#39;column1&#95;name column1&#95;type, column2&#95;name column2&#95;type, ...&#39; です。
* `data` — 指定したフォーマットのデータを含む文字列を返す文字列リテラルまたは定数式

<div id="returned_value">
  ## 戻り値
</div>

指定されたフォーマットと、指定または抽出された構造に従って `data` 引数を解析して得られるデータを含むテーブル。

<div id="examples">
  ## 例
</div>

`structure` 引数を指定しない場合：

```sql title="Query"
SELECT * FROM format(JSONEachRow,
$$
{"a": "Hello", "b": 111}
{"a": "World", "b": 123}
{"a": "Hello", "b": 112}
{"a": "World", "b": 124}
$$)
```

```response title="Response"
┌───b─┬─a─────┐
│ 111 │ Hello │
│ 123 │ World │
│ 112 │ Hello │
│ 124 │ World │
└─────┴───────┘
```

```sql title="Query"
DESC format(JSONEachRow,
$$
{"a": "Hello", "b": 111}
{"a": "World", "b": 123}
{"a": "Hello", "b": 112}
{"a": "World", "b": 124}
$$)
```

```response title="Response"
┌─name─┬─type──────────────┬─default_type─┬─default_expression─┬─comment─┬─codec_expression─┬─ttl_expression─┐
│ b    │ Nullable(Float64) │              │                    │         │                  │                │
│ a    │ Nullable(String)  │              │                    │         │                  │                │
└──────┴───────────────────┴──────────────┴────────────────────┴─────────┴──────────────────┴────────────────┘
```

`structure` 引数を指定する場合:

```sql title="Query"
SELECT * FROM format(JSONEachRow, 'a String, b UInt32',
$$
{"a": "Hello", "b": 111}
{"a": "World", "b": 123}
{"a": "Hello", "b": 112}
{"a": "World", "b": 124}
$$)
```

```response title="Response"
┌─a─────┬───b─┐
│ Hello │ 111 │
│ World │ 123 │
│ Hello │ 112 │
│ World │ 124 │
└───────┴─────┘
```
