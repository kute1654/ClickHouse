<div id="minMappedArrays">
  ## minMappedArrays
</div>

導入バージョン: v20.5.0

`key` 配列で指定されたキーに対応する `value` 配列の最小値を計算します。

<Note>
  * キーと値配列のタプルを渡すことは、キーの配列と値の配列を渡すのと同じです。
  * 集計対象となる各行では、`key` と `value` の要素数が同じである必要があります。
</Note>

**構文**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**引数**

* `key` — キーの配列。[`Array(T)`](/ja/reference/data-types/array)
* `value` — 値の配列。[`Array(T)`](/ja/reference/data-types/array)

**戻り値**

2 つの配列からなるタプルを返します。キーはソート順に並び、値はそれに対応して計算されます。[`Tuple(Array(T), Array(T))`](/ja/reference/data-types/tuple)

**例**

**使用例**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
