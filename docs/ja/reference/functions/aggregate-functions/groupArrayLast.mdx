<div id="groupArrayLast">
  ## groupArrayLast
</div>

導入バージョン: v23.1.0

最後の引数値からなる配列を作成します。
たとえば、`groupArrayLast(1)(x)` は `[anyLast(x)]` と同等です。
場合によっては、実行順序に依存できます。
これは、SELECT の取得元が ORDER BY を使用するサブクエリで、そのサブクエリの結果が十分に小さい場合に当てはまります。

**構文**

```sql
groupArrayLast(max_size)(x)
```

**パラメータ**

* `max_size` — 返される配列の最大サイズ。[`UInt64`](/ja/reference/data-types/int-uint)

**引数**

* `max_size` — 返される配列の最大サイズ。[`UInt64`](/ja/reference/data-types/int-uint)
* `x` — 引数 (カラム名または式) 。[`Any`](/ja/reference/data-types/index)

**戻り値**

最後の引数値からなる配列を返します。[`Array(T)`](/ja/reference/data-types/array)

**例**

**使用例**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**groupArrayとの比較**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
