<div id="exponentialMovingAverage">
  ## exponentialMovingAverage
</div>

導入バージョン: v21.11.0

指定した時点における値の指数移動平均を計算します。

各 `value` は、対応する `timeunit` に対応します。
半減期 `x` は、指数重みが半分に減衰するまでの時間差です。
この関数は加重平均を返します。時点が古いほど、対応する値に与えられる重みは小さくなります。

**構文**

```sql
exponentialMovingAverage(x)(value, timeunit)
```

**パラメータ**

* `x` — 半減期。[`(U)Int*`](/ja/reference/data-types/int-uint) または [`Float*`](/ja/reference/data-types/float) または [`Decimal`](/ja/reference/data-types/decimal)

**引数**

* `value` — 値。[`(U)Int*`](/ja/reference/data-types/int-uint) または [`Float*`](/ja/reference/data-types/float) または [`Decimal`](/ja/reference/data-types/decimal)
* `timeunit` — 時間単位。timeunit はタイムスタンプ (秒) ではなく、時間間隔の索引です。`intDiv` を使用して計算できます。[`(U)Int*`](/ja/reference/data-types/int-uint) または [`Float*`](/ja/reference/data-types/float) または [`Decimal`](/ja/reference/data-types/decimal)

**戻り値**

最新時点での、過去 `x` 時間にわたる値の指数平滑移動平均を返します。[`Float64`](/ja/reference/data-types/float)

**例**

**基本的な指数移動平均**

```sql title=Query
-- 気温データを含む入力テーブル
SELECT exponentialMovingAverage(5)(temperature, timestamp)
FROM VALUES('temperature Int32, timestamp Int32',
    (95, 1), (95, 2), (95, 3), (96, 4), (96, 5), (96, 6), (96, 7),
    (97, 8), (97, 9), (97, 10), (97, 11), (98, 12), (98, 13), (98, 14),
    (98, 15), (99, 16), (99, 17), (99, 18), (100, 19), (100, 20))
```


**`bar` 関数の例**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 1, 50) AS bar
FROM
(
    SELECT
        (number = 0) OR (number >= 25) AS value,
        number AS time,
        exponentialMovingAverage(10)(value, time) OVER (Rows BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
)
```

**時間ベースの計算におけるウィンドウ関数の使用**

```sql title=Query
CREATE TABLE data
ENGINE = Memory AS
SELECT
    10 AS value,
    toDateTime('2020-01-01') + (3600 * number) AS time
FROM numbers_mt(10);

-- intDivを使用してtimeunitを計算する
SELECT
    value,
    time,
    exponentialMovingAverage(1)(value, intDiv(toUInt32(time), 3600)) OVER (ORDER BY time ASC) AS res,
    intDiv(toUInt32(time), 3600) AS timeunit
FROM data
ORDER BY time ASC
```
