<div id="contingency">
  ## contingency
</div>

導入バージョン: v22.1.0

`contingency` 関数は、[連関係数](https://en.wikipedia.org/wiki/Contingency_table#Cram%C3%A9r's_V_and_the_contingency_coefficient_C) を計算します。これは、テーブル内の 2 つのカラム間の関連性を表す値です。
計算方法は [`cramersV`](/ja/reference/functions/aggregate-functions/cramersV) 関数と似ていますが、平方根内の分母が異なります。

**構文**

```sql
contingency(column1, column2)
```

**引数**

* `column1` — 比較対象の1つ目のカラム。[`Any`](/ja/reference/data-types/index)
* `column2` — 比較対象の2つ目のカラム。[`Any`](/ja/reference/data-types/index)

**戻り値**

0 から 1 の値を返します。結果が大きいほど、2 つのカラムの関連性が高いことを示します。[`Float64`](/ja/reference/data-types/float)

**例**

**cramersV との比較**

```sql title=Query
SELECT
    cramersV(a, b),
    contingency(a, b)
FROM
(
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌─────cramersV(a, b)─┬──contingency(a, b)─┐
│ 0.5798088336225178 │ 0.708607540104077  │
└────────────────────┴────────────────────┘
```
