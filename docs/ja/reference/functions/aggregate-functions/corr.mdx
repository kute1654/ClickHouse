## corr {#corr}

導入バージョン: v1.1.0

[ピアソン相関係数](https://en.wikipedia.org/wiki/Pearson_correlation_coefficient)を計算します：

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{\sqrt{\Sigma{(x - \bar{x})^2} * \Sigma{(y - \bar{y})^2}}}
$$

<br/>

<Note>
この関数は数値的に不安定なアルゴリズムを使用しています。計算に[数値安定性](https://en.wikipedia.org/wiki/Numerical_stability)が必要な場合は、[`corrStable`](/reference/functions/aggregate-functions/corrStable)関数を使用してください。処理速度は遅くなりますが、より正確な結果が得られます。
</Note>
    

**構文**

```sql
corr(x, y)
```

**引数**

- `x` — 第1変数。[`(U)Int*`](/reference/data-types/int-uint) または [`Float*`](/reference/data-types/float)
- `y` — 第2変数。[`(U)Int*`](/reference/data-types/int-uint) または [`Float*`](/reference/data-types/float)

**戻り値**

ピアソン相関係数を返します。[`Float64`](/reference/data-types/float)

**例**

**基本的な相関計算**

```sql title=Query
DROP TABLE IF EXISTS series;
CREATE TABLE series
(
    i UInt32,
    x_value Float64,
    y_value Float64
)
ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6, -4.4),(2, -9.6, 3),(3, -1.3, -4),(4, 5.3, 9.7),(5, 4.4, 0.037),(6, -8.6, -7.8),(7, 5.1, 9.3),(8, 7.9, -3.6),(9, -8.2, 0.62),(10, -3, 7.3);

SELECT corr(x_value, y_value)
FROM series
```

```response title=Response
┌─corr(x_value, y_value)─┐
│     0.1730265755453256 │
└────────────────────────┘
```
