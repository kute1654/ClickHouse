<div id="groupBitOr">
  ## groupBitOr
</div>

導入バージョン: v1.1.0

一連の数値に対してビット単位の OR を適用します。

**構文**

```sql
groupBitOr(expr)
```

**別名**: `BIT_OR`

**引数**

* `expr` — `(U)Int*` 型の式。[`(U)Int*`](/ja/reference/data-types/int-uint)

**戻り値**

`(U)Int*` 型の値を返します。[`(U)Int*`](/ja/reference/data-types/int-uint)

**例**

**ビット演算 OR の例**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- テストデータ:
-- 2進数      10進数
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- 結果:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
