<div id="skewSamp">
  ## skewSamp
</div>

導入バージョン: v20.1.0

数列の[標本歪度](https://en.wikipedia.org/wiki/Skewness)を計算します。

渡された値がその標本を構成している場合、これは確率変数の歪度の不偏推定量を表します。

**構文**

```sql
skewSamp(expr)
```

**引数**

* `expr` — 数値を返す式。[`Expression`](/ja/reference/data-types/special-data-types/expression)

**戻り値**

指定した分布の歪度を返します。`n <= 1` の場合 (`n` はサンプルサイズ) 、この関数は `nan` を返します。[`Float64`](/ja/reference/data-types/float)

**例**

**対称分布**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**右に歪んだ分布**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
