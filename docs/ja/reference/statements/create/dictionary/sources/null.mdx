ダミーの (空の) Dictionary を作成するために使用できる特別なソースです。
ダミーの Dictionary は、テスト用途や、データノードとクエリノードを分離して分散テーブルを使用する構成で役立つことがあります。

```sql
CREATE DICTIONARY null_dict (
    id              UInt64,
    val             UInt8,
    default_val     UInt8 DEFAULT 123,
    nullable_val    Nullable(UInt8)
)
PRIMARY KEY id
SOURCE(NULL())
LAYOUT(FLAT())
LIFETIME(0);
```
