```sql
EXISTS [TEMPORARY] [TABLE|DICTIONARY|DATABASE] [db.]name [INTO OUTFILE filename] [FORMAT format]
```

単一の `UInt8` 型カラムを返します。このカラムには、テーブルまたはデータベースが存在しない場合は `0`、テーブルが指定したデータベースに存在する場合は `1` という単一の値が含まれます。
