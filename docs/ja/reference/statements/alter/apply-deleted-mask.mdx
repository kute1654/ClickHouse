```sql
ALTER TABLE [db].name [ON CLUSTER cluster] APPLY DELETED MASK [IN PARTITION partition_id]
```

このコマンドは、[論理削除](/ja/reference/statements/delete)で作成されたマスクを適用し、削除済みとしてマークされた行をディスクから強制的に削除します。このコマンドはヘビーウェイトなミューテーションであり、意味的にはクエリ `ALTER TABLE [db].name DELETE WHERE _row_exists = 0` と同等です。

<Note>
  このコマンドは、[`MergeTree`](/ja/reference/engines/table-engines/mergetree-family/mergetree) ファミリーのテーブル ([replicated](/ja/reference/engines/table-engines/mergetree-family/replication) テーブルを含む) でのみ使用できます。
</Note>
