ユーザーは、SQL ステートメントを使用して ClickHouse を操作します。ClickHouse は、[SELECT](/ja/reference/statements/select/index) や [CREATE](/ja/reference/statements/create/index) のような一般的な SQL ステートメントをサポートしているほか、[KILL](/ja/reference/statements/kill) や [OPTIMIZE](/ja/reference/statements/optimize) のような専用のステートメントも提供しています。
