このステートメントを使用すると、アクセスエンティティをあるアクセスストレージから別のアクセスストレージに移動できます。

構文:

```sql
MOVE {USER, ROLE, QUOTA, SETTINGS PROFILE, ROW POLICY} name1 [, name2, ...] TO access_storage_type
```

現在、ClickHouse には 5 種類のアクセスストレージがあります。

* `local_directory`
* `memory`
* `replicated`
* `users_xml` (ro)
* `ldap` (ro)

例:

```sql
MOVE USER test TO local_directory
```

```sql
MOVE ROLE test TO memory
```
