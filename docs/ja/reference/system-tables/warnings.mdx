<div id="description">
  ## 説明
</div>

clickhouse-client がサーバーに接続した直後に表示される、サーバー設定に関する警告が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `message` ([String](/ja/reference/data-types/index)) — ClickHouse server が出力した警告メッセージ。
* `message_format_string` ([LowCardinality(String)](/ja/reference/data-types/index)) — メッセージの整形に使用された書式文字列。

{/*AUTOGENERATED_END*/}
