<div id="description">
  ## 説明
</div>

AzureQueue メタデータのインメモリ状態と、ファイルごとに現在処理中の行が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/ja/reference/data-types/index)) — zookeeper 内のメタデータのパス
* `file_path` ([String](/ja/reference/data-types/index)) — 処理中のファイルのファイルパス
* `file_name` ([String](/ja/reference/data-types/index)) — 処理中のファイルのファイル名
* `rows_processed` ([UInt64](/ja/reference/data-types/index)) — 現在処理済みの行数
* `status` ([String](/ja/reference/data-types/index)) — 処理のステータス: Processed、Processing、Failed
* `processing_start_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — ファイルの処理が開始された時刻
* `processing_end_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — ファイルの処理が終了した時刻
* `exception` ([String](/ja/reference/data-types/index)) — 処理中に発生した例外

{/*AUTOGENERATED_END*/}
