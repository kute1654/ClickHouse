<div id="description">
  ## 説明
</div>

`config.xml` で指定されるサーバーのグローバル設定に関する情報です。
現在、このテーブルには `config.xml` の第1階層の設定のみが表示されており、ネストされた設定 (例: [logger](/ja/reference/settings/server-settings/settings#logger)) には対応していません。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — サーバー設定名。
* `value` ([String](/ja/reference/data-types/index)) — サーバー設定の値。
* `default` ([String](/ja/reference/data-types/index)) — サーバー設定のデフォルト値。
* `changed` ([UInt8](/ja/reference/data-types/index)) — 設定が config.xml で指定されているかどうかを示します。
* `description` ([String](/ja/reference/data-types/index)) — サーバー設定の簡単な説明。
* `type` ([String](/ja/reference/data-types/index)) — サーバー設定の値の型。
* `changeable_without_restart` ([Enum8(&#39;No&#39; = 0, &#39;IncreaseOnly&#39; = 1, &#39;DecreaseOnly&#39; = 2, &#39;Yes&#39; = 3)](/ja/reference/data-types/index)) — 設定を実行時に変更できるかどうかを示します。
* `is_obsolete` ([UInt8](/ja/reference/data-types/index)) — 設定が廃止されたものかどうかを示します。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

次の例では、名前に `thread_pool` を含むサーバー設定の情報を取得する方法を示します。

```sql
SELECT *
FROM system.server_settings
WHERE name LIKE '%thread_pool%'
```
