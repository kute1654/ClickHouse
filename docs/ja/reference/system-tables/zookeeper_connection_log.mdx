<div id="description">
  ## 説明
</div>

`system.zookeeper_connection_log` テーブルには、ZooKeeper 接続 (補助的な ZooKeeper を含む) の履歴が記録されます。各行には、接続に関する 1 件のイベント情報が含まれます。

<Note>
  このテーブルには、サーバーのシャットダウンによって発生した切断イベントは記録されません。
</Note>

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ja/reference/data-types/lowcardinality)) — ZooKeeper への接続先または切断元のサーバーのホスト名。
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/ja/reference/data-types/enum)) — イベントの種類。設定可能な値: Connected、Disconnected。
* `event_date` ([Date](/ja/reference/data-types/date)) — エントリの日付。
* `event_time` ([DateTime](/ja/reference/data-types/datetime)) — エントリの時刻
* `event_time_microseconds` ([DateTime64(6)](/ja/reference/data-types/datetime64)) — マイクロ秒精度のエントリの時刻。
* `name` ([String](/ja/reference/data-types/string)) — ZooKeeper クラスターの名前。
* `host` ([String](/ja/reference/data-types/string)) — ClickHouse が接続または切断した ZooKeeper ノードのホスト名/IP。
* `port` ([UInt16](/ja/reference/data-types/int-uint)) — ClickHouse が接続または切断した ZooKeeper ノードのポート。
* `index` ([UInt8](/ja/reference/data-types/int-uint)) — ClickHouse が接続または切断した ZooKeeper ノードの索引。索引は ZooKeeper の設定に基づきます。
* `client_id` ([Int64](/ja/reference/data-types/int-uint)) — connection の session ID。
* `keeper_api_version` ([UInt8](/ja/reference/data-types/int-uint)) — Keeper API のバージョン。
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/ja/reference/data-types/array)) — 有効になっている feature flags。ClickHouse Keeper にのみ適用されます。
* `availability_zone` ([String](/ja/reference/data-types/string)) — Availability zone
* `reason` ([LowCardinality(String)](/ja/reference/data-types/lowcardinality)) — 接続または切断の理由。

{/*AUTOGENERATED_END*/}

SELECT * FROM system.zookeeper_connection_log;
