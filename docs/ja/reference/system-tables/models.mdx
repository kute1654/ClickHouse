<div id="description">
  ## 説明
</div>

LibraryBridge のメモリに読み込まれている CatBoost モデルの一覧と、その読み込み時刻が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `model_path` ([String](/ja/reference/data-types/index)) — 学習済みモデルへのパス。
* `type` ([String](/ja/reference/data-types/index)) — モデルのタイプ。現在は catboost のみです。
* `loading_start_time` ([DateTime](/ja/reference/data-types/index)) — モデルの読み込みが開始された時刻。
* `loading_duration` ([Float32](/ja/reference/data-types/index)) — モデルの読み込みに要した時間。

{/*AUTOGENERATED_END*/}
