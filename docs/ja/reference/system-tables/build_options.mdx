<div id="description">
  ## 説明
</div>

ClickHouse server のビルドオプションに関する情報が含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — ビルドオプション名。
* `value` ([String](/ja/reference/data-types/index)) — ビルドオプションの値。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
