<div id="description">
  ## 説明
</div>

現在のユーザーの クォータ使用状況: 使用量と残量。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ja/reference/data-types/index)) — クォータ名。
* `quota_key` ([String](/ja/reference/data-types/index)) — キー値。
* `start_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — リソース消費を計算する開始時刻。
* `end_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — リソース消費を計算する終了時刻。
* `duration` ([Nullable(UInt32)](/ja/reference/data-types/index)) — リソース消費の計算対象となる時間間隔の長さ (秒) 。
* `queries` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されたクエリ数。
* `max_queries` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行が許可される全種類のクエリの最大数。
* `query_selects` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されたSELECTクエリ数。
* `max_query_selects` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行が許可されるSELECTクエリの最大数。
* `query_inserts` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されたINSERTクエリ数。
* `max_query_inserts` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行が許可されるINSERTクエリの最大数。
* `errors` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在のエラーとなったクエリ数。
* `max_errors` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、エラーとなったクエリの最大数。
* `result_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、すべてのクエリの結果セット内の現在の合計行数。
* `max_result_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、すべてのクエリの結果セット内の合計行数の最大値。
* `result_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、すべてのクエリの結果セット内の現在の合計バイト数。
* `max_result_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、すべてのクエリの結果セット内の合計バイト数の最大値。
* `read_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、すべてのクエリの実行中に読み取られた現在の合計行数。
* `max_read_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、すべてのクエリの実行中に読み取る行数の最大値。
* `read_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、すべてのクエリの実行中に読み取られた現在の合計バイト数。
* `max_read_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、すべてのクエリの実行中に読み取るバイト数の最大値。
* `execution_time` ([Nullable(Float64)](/ja/reference/data-types/index)) — 現在の期間内にクエリの実行に費やされた現在の合計時間 (ナノ秒)
* `max_execution_time` ([Nullable(Float64)](/ja/reference/data-types/index)) — 指定された期間内にすべてのクエリの実行に許可される最大時間 (ナノ秒)
* `written_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、すべてのクエリの実行中に書き込まれた現在の合計バイト数。
* `max_written_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、すべてのクエリの実行中に書き込むバイト数の最大値。
* `failed_sequential_authentications` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、連続した認証失敗回数の現在値。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、連続した認証失敗回数の最大値。
* `queries_per_normalized_hash` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、単一の正規化クエリの現在の最大実行回数。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定された期間内に許可される、単一の正規化クエリの最大実行回数。
