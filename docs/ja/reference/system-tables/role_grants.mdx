<div id="description">
  ## 説明
</div>

ユーザーおよびロールのロール付与に関する情報が含まれます。このテーブルにエントリを追加するには、`GRANT role TO user` を使用します。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `user_name` ([Nullable(String)](/ja/reference/data-types/index)) — ユーザー名。
* `role_name` ([Nullable(String)](/ja/reference/data-types/index)) — ロール名。
* `granted_role_name` ([String](/ja/reference/data-types/index)) — `role_name` ロールに付与されているロールの名前。あるロールを別のロールに付与するには、`GRANT role1 TO role2` を使用します。
* `granted_role_id` ([UUID](/ja/reference/data-types/index)) — ロールの ID。
* `granted_role_is_default` ([UInt8](/ja/reference/data-types/index)) — `granted_role` がデフォルトロールかどうかを示すフラグ。設定可能な値: • 1 — `granted_role` はデフォルトロールです。 • 0 — `granted_role` はデフォルトロールではありません。
* `with_admin_option` ([UInt8](/ja/reference/data-types/index)) — `granted_role` が `ADMIN OPTION` 権限を持つロールかどうかを示すフラグ。設定可能な値: • 1 — このロールは `ADMIN OPTION` 権限を持ちます。 • 0 — このロールは `ADMIN OPTION` 権限を持ちません。

{/*AUTOGENERATED_END*/}
