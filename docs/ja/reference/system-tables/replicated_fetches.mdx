<div id="description">
  ## 説明
</div>

現在実行中のバックグラウンドフェッチに関する情報が含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — データベース名。
* `table` ([String](/ja/reference/data-types/index)) — テーブル名。
* `elapsed` ([Float64](/ja/reference/data-types/index)) — 現在実行中のバックグラウンドフェッチの表示開始からの経過時間 (秒) 。
* `progress` ([Float64](/ja/reference/data-types/index)) — 完了した作業の割合 (0 ～ 1) 。
* `result_part_name` ([String](/ja/reference/data-types/index)) — 現在実行中のバックグラウンドフェッチの表示結果として生成される part の名前。
* `result_part_path` ([String](/ja/reference/data-types/index)) — 現在実行中のバックグラウンドフェッチの表示結果として生成される part への絶対パス。
* `partition_id` ([String](/ja/reference/data-types/index)) — パーティションの ID。
* `total_size_bytes_compressed` ([UInt64](/ja/reference/data-types/index)) — 結果の part 内の圧縮データの合計サイズ (バイト) 。
* `bytes_read_compressed` ([UInt64](/ja/reference/data-types/index)) — 結果の part から読み取られた圧縮済みデータのバイト数。
* `source_replica_path` ([String](/ja/reference/data-types/index)) — ソースレプリカへの絶対パス。
* `source_replica_hostname` ([String](/ja/reference/data-types/index)) — ソースレプリカのホスト名。
* `source_replica_port` ([UInt16](/ja/reference/data-types/index)) — ソースレプリカのポート番号。
* `interserver_scheme` ([String](/ja/reference/data-types/index)) — interserver スキームの名前。
* `URI` ([String](/ja/reference/data-types/index)) — 統一リソース識別子。
* `to_detached` ([UInt8](/ja/reference/data-types/index)) — 現在実行中のバックグラウンドフェッチが TO DETACHED 式を使用して実行されているかどうかを示すフラグです。
* `thread_id` ([UInt64](/ja/reference/data-types/index)) — スレッド ID。

{/*AUTOGENERATED_END*/}
