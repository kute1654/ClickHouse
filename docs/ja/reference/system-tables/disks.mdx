<div id="description">
  ## 説明
</div>

[サーバー設定](/ja/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure)で定義されたディスクに関する情報を含みます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — サーバー設定内のディスク名。
* `path` ([String](/ja/reference/data-types/index)) — ファイルシステム内のマウントポイントへのパス。
* `free_space` ([UInt64](/ja/reference/data-types/index)) — ディスク上の空き容量 (バイト単位) 。
* `total_space` ([UInt64](/ja/reference/data-types/index)) — ディスクの総容量 (バイト単位) 。
* `unreserved_space` ([UInt64](/ja/reference/data-types/index)) — 予約で使用されていない空き容量 (free&#95;space から、現在実行中のマージ、insert、その他のディスク書き込み操作で確保されている予約サイズを差し引いた値) 。
* `keep_free_space` ([UInt64](/ja/reference/data-types/index)) — ディスク上で空きのまま維持すべき容量 (バイト単位) 。ディスク設定の keep&#95;free&#95;space&#95;bytes パラメータで定義されます。
* `type` ([String](/ja/reference/data-types/index)) — このディスクがデータを保存する場所を示すディスクの種類。RAM、ローカルドライブ、またはリモートストレージです。
* `object_storage_type` ([String](/ja/reference/data-types/index)) — ディスクの種類が object&#95;storage の場合のオブジェクトストレージの種類。
* `metadata_type` ([String](/ja/reference/data-types/index)) — ディスクの種類が object&#95;storage の場合のメタデータストレージの種類。
* `is_encrypted` ([UInt8](/ja/reference/data-types/index)) — このディスクが基盤となるデータを暗号化するかどうかを示すフラグ。
* `is_read_only` ([UInt8](/ja/reference/data-types/index)) — このディスクでは読み取り操作のみ実行できることを示すフラグ。
* `is_write_once` ([UInt8](/ja/reference/data-types/index)) — ディスクが write-once かどうかを示すフラグ。つまり、このディスクへの BACKUP はサポートされますが、このディスク上の MergeTree テーブルへの INSERT はサポートされません。
* `is_remote` ([UInt8](/ja/reference/data-types/index)) — このディスクに対するどの操作でネットワーク通信が発生するかを示すフラグ。
* `is_broken` ([UInt8](/ja/reference/data-types/index)) — ディスクが破損しているかどうかを示すフラグ。破損したディスクの容量は 0 となり、使用できません。
* `cache_path` ([String](/ja/reference/data-types/index)) — ディスクがキャッシュをサポートしている場合の、ローカルドライブ上のキャッシュディレクトリへのパス。

{/*AUTOGENERATED_END*/}
