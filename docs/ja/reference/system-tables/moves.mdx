<div id="description">
  ## 説明
</div>

このテーブルには、[MergeTree](/ja/reference/engines/table-engines/mergetree-family/mergetree) テーブルにおける進行中の [データパーツの移動](/ja/reference/statements/alter/partition#move-partitionpart) に関する情報が含まれています。各データパーツの移動は 1 行で表されます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — データベース名。
* `table` ([String](/ja/reference/data-types/index)) — 移動中のデータパーツを含むテーブル名。
* `elapsed` ([Float64](/ja/reference/data-types/index)) — データパーツの移動開始からの経過時間 (秒) 。
* `target_disk_name` ([String](/ja/reference/data-types/index)) — データパーツの移動先ディスク名。
* `target_disk_path` ([String](/ja/reference/data-types/index)) — ファイルシステム内のディスクのマウントポイントへのパス。
* `part_name` ([String](/ja/reference/data-types/index)) — 移動対象のデータパーツ名。
* `part_size` ([UInt64](/ja/reference/data-types/index)) — データパーツのサイズ。
* `thread_id` ([UInt64](/ja/reference/data-types/index)) — 移動処理を実行しているスレッドの識別子。

{/*AUTOGENERATED_END*/}
