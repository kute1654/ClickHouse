<div id="description">
  ## 説明
</div>

[リフレッシュ可能なマテリアライズドビュー](/ja/reference/statements/create/view#refreshable-materialized-view)に関する情報です。リフレッシュが進行中かどうかにかかわらず、すべてのリフレッシュ可能なマテリアライズドビューが含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — このテーブルが属するデータベースの名前。
* `view` ([String](/ja/reference/data-types/index)) — テーブルの名前。
* `uuid` ([UUID](/ja/reference/data-types/index)) — テーブルの uuid (Atomic データベース) 。
* `status` ([String](/ja/reference/data-types/index)) — 現在のリフレッシュの状態。
* `last_success_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — 直近で成功したリフレッシュが開始された時刻。サーバーの起動またはテーブル作成以降に成功したリフレッシュがない場合は NULL。
* `last_success_duration_ms` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 直近のリフレッシュにかかった時間。
* `last_refresh_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — 直近のリフレッシュ試行が終了した時刻 (判明している場合) 、または開始した時刻 (不明な場合、またはまだ実行中の場合) 。サーバーの起動またはテーブル作成以降にリフレッシュの試行がない場合は NULL。
* `last_refresh_replica` ([String](/ja/reference/data-types/index)) — 協調が有効な場合、現在の (実行中であれば) または前回の (実行中でなければ) リフレッシュ試行を行ったレプリカの名前。
* `next_refresh_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — 次のリフレッシュが開始される予定の時刻。次回のリフレッシュ時刻が現在不明な場合 (たとえば依存関係を待機中で、status が `WaitingForDependencies` または `MissingDependencies` の場合) は NULL。
* `exception` ([String](/ja/reference/data-types/index)) — 前回の試行が失敗した場合のエラーメッセージ。
* `retry` ([UInt64](/ja/reference/data-types/index)) — 現在のリフレッシュで、これまでに失敗した試行回数。status が `RunningOnAnotherReplica` の場合は利用できません。
* `progress` ([Nullable(Float64)](/ja/reference/data-types/index)) — 指定されたレプリカにおける、現在実行中または直近で完了したリフレッシュの進行状況。値は 0 から 1 です。status が `RunningOnAnotherReplica` の場合、またはリフレッシュが実行中でない場合は NULL。
* `read_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定されたレプリカにおいて、現在実行中または直近で完了したリフレッシュが読み取った行数。status が `RunningOnAnotherReplica` の場合は NULL。
* `read_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定されたレプリカにおいて、現在実行中または直近で完了したリフレッシュが読み取ったバイト数。status が `RunningOnAnotherReplica` の場合は NULL
* `total_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定されたレプリカにおいて、現在実行中または直近で完了したリフレッシュで読み取る必要があると見積もられる合計行数。status が `RunningOnAnotherReplica` の場合は NULL
* `written_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定されたレプリカにおいて、現在実行中または直近で完了したリフレッシュが書き込んだ行数。status が `RunningOnAnotherReplica` の場合は NULL
* `written_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定されたレプリカにおいて、現在実行中または直近で完了したリフレッシュが書き込んだバイト数。status が `RunningOnAnotherReplica` の場合は NULL

{/*AUTOGENERATED_END*/}

    last_refresh_time,
    next_refresh_time
