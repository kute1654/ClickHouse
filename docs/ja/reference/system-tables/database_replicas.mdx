<div id="description">
  ## 説明
</div>

各 Replicated データベース のレプリカに関する情報が含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — Replicated データベースの名前。
* `is_readonly` ([UInt8](/ja/reference/data-types/index)) — データベースのレプリカが読み取り専用モードかどうか。
* `max_log_ptr` ([Int32](/ja/reference/data-types/index)) — 一般アクティビティログ内の最大エントリ番号。
* `replica_name` ([String](/ja/reference/data-types/index)) — ClickHouse Keeper 内のレプリカ名。
* `replica_path` ([String](/ja/reference/data-types/index)) — ClickHouse Keeper 内のレプリカデータへのパス。
* `zookeeper_path` ([String](/ja/reference/data-types/index)) — ClickHouse Keeper 内のデータベースデータへのパス。
* `shard_name` ([String](/ja/reference/data-types/index)) — クラスター内の分片の名前。
* `log_ptr` ([Int32](/ja/reference/data-types/index)) — レプリカが実行キューにコピーした一般アクティビティログ内の最大エントリ番号に 1 を加えた値。
* `total_replicas` ([UInt32](/ja/reference/data-types/index)) — このデータベースで認識されているレプリカの総数。
* `zookeeper_exception` ([String](/ja/reference/data-types/index)) — 最後に発生した例外メッセージ。ClickHouse Keeper から情報を取得中にエラーが発生した場合に返されます。
* `is_session_expired` ([UInt8](/ja/reference/data-types/index)) — ClickHouse Keeper とのセッションの有効期限が切れていることを示します。基本的には `is_readonly` と同じです。

{/*AUTOGENERATED_END*/}
