<div id="description">
  ## 説明
</div>

各デタッチされたテーブルに関する情報を含みます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — テーブルが属するデータベースの名前。
* `table` ([String](/ja/reference/data-types/index)) — テーブル名。
* `uuid` ([UUID](/ja/reference/data-types/index)) — テーブルの uuid (Atomic データベース) 。
* `metadata_path` ([String](/ja/reference/data-types/index)) — ファイルシステム内のテーブルのメタデータへのパス。
* `is_permanently` ([UInt8](/ja/reference/data-types/index)) — テーブルが永続的にデタッチされたかどうか。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
Row 1:
──────
database:                   base
table:                      t1
uuid:                       81b1c20a-b7c6-4116-a2ce-7583fb6b6736
metadata_path:              /var/lib/clickhouse/store/461/461cf698-fd0b-406d-8c01-5d8fd5748a91/t1.sql
is_permanently:             1
```
