<div id="description">
  ## 説明
</div>

致命的なエラーのスタックトレースに関する情報を含みます。このテーブルはデータベースにはデフォルトでは存在せず、致命的なエラーが発生したときにのみ作成されます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ja/reference/data-types/lowcardinality)) — クラッシュが発生したホスト名。
* `event_date` ([Date](/ja/reference/data-types/date)) — クラッシュが発生した日付。
* `event_time` ([DateTime](/ja/reference/data-types/datetime)) — クラッシュが発生した時刻。
* `timestamp_ns` ([UInt64](/ja/reference/data-types/int-uint)) — ナノ秒精度のイベントタイムスタンプ。
* `signal` ([Int32](/ja/reference/data-types/int-uint)) — シグナル番号。
* `signal_code` ([Int32](/ja/reference/data-types/int-uint)) — シグナルコード。シグナルの原因に関する追加情報を提供します。
* `thread_id` ([UInt64](/ja/reference/data-types/int-uint)) — スレッド ID。
* `query_id` ([String](/ja/reference/data-types/string)) — クエリ ID。
* `query` ([String](/ja/reference/data-types/string)) — クラッシュ発生時に実行されていたクエリテキスト。
* `trace` ([Array(UInt64)](/ja/reference/data-types/array)) — クラッシュ発生時点のスタックトレース。各要素は ClickHouse server process 内の仮想メモリアドレスです。
* `trace_full` ([Array(String)](/ja/reference/data-types/array)) — クラッシュ発生時点のスタックトレース。各要素には ClickHouse server process 内で呼び出されたメソッドが含まれます。
* `fault_address` ([Nullable(UInt64)](/ja/reference/data-types/nullable)) — 障害の原因となったメモリアドレス。
* `fault_access_type` ([String](/ja/reference/data-types/string)) — 障害の原因となったメモリアクセスの種類 (例: &#39;read&#39;、&#39;write&#39;) 。
* `signal_description` ([String](/ja/reference/data-types/string)) — signal&#95;code に基づくわかりやすい説明 (例: &#39;Address not mapped to object&#39;) 。
* `current_exception_trace_full` ([Array(String)](/ja/reference/data-types/array)) — シンボル化されたフレームを含む現在の例外スタックトレース (`std::terminate` によるクラッシュのみ) 。
* `version` ([String](/ja/reference/data-types/string)) — ClickHouse server のバージョン。
* `revision` ([UInt32](/ja/reference/data-types/int-uint)) — ClickHouse server のリビジョン。
* `build_id` ([String](/ja/reference/data-types/string)) — コンパイラによって生成された BuildID。
* `git_hash` ([String](/ja/reference/data-types/string)) — ClickHouse のソースコードの Git コミットハッシュ。
* `architecture` ([String](/ja/reference/data-types/string)) — CPU アーキテクチャ (例: x86&#95;64、aarch64) 。

{/*AUTOGENERATED_END*/}
