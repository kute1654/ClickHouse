<div id="description">
  ## 説明
</div>

jemalloc の統計情報を、1 行 1 カラムで返します。SYSTEM JEMALLOC STATS command に相当します。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `stats` ([String](/ja/reference/data-types/index)) — JEMalloc の `malloc_stats_print` による統計出力。

{/*AUTOGENERATED_END*/}
