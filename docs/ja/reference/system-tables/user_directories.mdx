<div id="description">
  ## 説明
</div>

設定済みのユーザーディレクトリ、つまり ClickHouse server がユーザー提供データの読み取りを許可されているファイルシステム上のディレクトリに関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — ディレクトリ名。
* `type` ([String](/ja/reference/data-types/index)) — アクセスストレージの種類 (例: users.xml、replicated、memory など) 。
* `params` ([String](/ja/reference/data-types/index)) — アクセスストレージのパラメータを含む JSON。
* `precedence` ([UInt64](/ja/reference/data-types/index)) — このディレクトリが設定内で宣言される順序です。ClickHouse がユーザーまたはロールを探す際にも、同じ順序が使われます。

{/*AUTOGENERATED_END*/}
