<div id="description">
  ## 説明
</div>

キャッシュされた DNS レコードに関する情報が含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/ja/reference/data-types/index)) — ホスト名。
* `ip_address` ([String](/ja/reference/data-types/index)) — IPアドレス。
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/ja/reference/data-types/index)) — IPアドレスファミリー。
* `cached_at` ([DateTime](/ja/reference/data-types/index)) — レコードがキャッシュされた時刻のタイムスタンプ。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
