<div id="description">
  ## 説明
</div>

[ユーザー定義関数 (UDF) ](/ja/reference/functions/regular-functions/udf) の読み込みステータス、エラー情報、および設定メタデータが含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — UDF 名。
* `load_status` ([Enum8(&#39;Success&#39; = 0, &#39;Failed&#39; = 1)](/ja/reference/data-types/index)) — 読み込みステータス。設定可能な値:
  * **Success** — UDF は読み込み済みで、すぐに使用できます
  * **Failed** — UDF の読み込みに失敗しました (詳細はフィールド &#39;loading&#95;error&#95;message&#39; を参照) 。
* `loading_error_message` ([String](/ja/reference/data-types/index)) — 読み込み失敗時の詳細なエラーメッセージ。正常に読み込まれた場合は空です。
* `last_successful_update_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — 最後に正常に更新された時刻のタイムスタンプ。一度も成功していない場合は NULL です。
* `loading_duration_ms` ([UInt64](/ja/reference/data-types/index)) — UDF の読み込みにかかった時間 (ミリ秒) 。
* `type` ([Enum8(&#39;executable&#39; = 0, &#39;executable&#95;pool&#39; = 1)](/ja/reference/data-types/index)) — UDF のタイプ: &#39;executable&#39; (単一プロセス) または &#39;executable&#95;pool&#39; (プロセスプール) 。
* `command` ([String](/ja/reference/data-types/index)) — この UDF で実行するスクリプトまたはコマンド。
* `format` ([String](/ja/reference/data-types/index)) — I/O 用のデータフォーマット (例: &#39;TabSeparated&#39;、&#39;JSONEachRow&#39;) 。
* `return_type` ([String](/ja/reference/data-types/index)) — 関数の戻り値の型 (例: &#39;String&#39;、&#39;UInt64&#39;) 。
* `return_name` ([String](/ja/reference/data-types/index)) — 任意の戻り値識別子。設定されていない場合は空です。
* `argument_types` ([Array(String)](/ja/reference/data-types/index)) — 引数の型の Array (例: [&#39;String&#39;, &#39;UInt64&#39;]) 。
* `argument_names` ([Array(String)](/ja/reference/data-types/index)) — 引数名の Array。名前のない引数には空文字列が入ります。
* `max_command_execution_time` ([UInt64](/ja/reference/data-types/index)) — データブロックの処理に許可される最大秒数。&#39;executable&#95;pool&#39; 型でのみ使用されます。
* `command_termination_timeout` ([UInt64](/ja/reference/data-types/index)) — コマンドプロセスに SIGTERM を送信するまでの秒数。
* `command_read_timeout` ([UInt64](/ja/reference/data-types/index)) — コマンドの stdout から読み取る際のタイムアウト (ミリ秒) 。
* `command_write_timeout` ([UInt64](/ja/reference/data-types/index)) — コマンドの stdin に書き込む際のタイムアウト (ミリ秒) 。
* `pool_size` ([UInt64](/ja/reference/data-types/index)) — コマンドプロセスのインスタンス数。&#39;executable&#95;pool&#39; 型でのみ使用されます。
* `send_chunk_header` ([UInt8](/ja/reference/data-types/index)) — 各 data chunk の前に行数を送信するかどうか (ブール値) 。
* `execute_direct` ([UInt8](/ja/reference/data-types/index)) — コマンドを直接実行するか (1) 、/bin/bash 経由で実行するか (0) 。
* `lifetime` ([UInt64](/ja/reference/data-types/index)) — 再読み込み間隔 (秒) 。0 は再読み込みが無効であることを意味します。
* `deterministic` ([UInt8](/ja/reference/data-types/index)) — 同じ引数に対して関数が常に同じ結果を返すかどうか (ブール値) 。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

すべてのUDFとその読み込み状況を表示します。

```sql
SELECT
    name,
    load_status,
    type,
    command,
    return_type,
    argument_types
FROM system.user_defined_functions
FORMAT Vertical;
```

```response
Row 1:
──────
name:           my_sum_udf
load_status:    Success
type:           executable
command:        /var/lib/clickhouse/user_scripts/sum.py
return_type:    UInt64
argument_types: ['UInt64','UInt64']
```

失敗したUDFを見つける:

```sql
SELECT
    name,
    loading_error_message
FROM system.user_defined_functions
WHERE load_status = 'Failed';
```
