リモートファイルシステム上のオブジェクトに対するファイルシステムキャッシュで発生した、すべてのイベントの履歴を保持します。

このテーブルはいつでも安全に TRUNCATE または DROP できます。

カラム:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ja/reference/data-types/lowcardinality)) — ホスト名
* `event_date` ([Date](/ja/reference/data-types/date)) — イベント日付
* `event_time` ([DateTime](/ja/reference/data-types/datetime)) — イベント時刻
* `query_id` ([String](/ja/reference/data-types/string)) — クエリ ID
* `source_file_path` ([String](/ja/reference/data-types/string)) — ファイルシステム上の file segment のパス
* `file_segment_range` ([Tuple(UInt64, UInt64)](/ja/reference/data-types/tuple)) — file segment の範囲
* `total_requested_range` ([Tuple(UInt64, UInt64)](/ja/reference/data-types/tuple)) — 読み取り範囲全体
* `key` ([String](/ja/reference/data-types/string)) — file segment のキー
* `offset` ([UInt64](/ja/reference/data-types/int-uint)) — file segment のオフセット
* `size` ([UInt64](/ja/reference/data-types/int-uint)) — 読み取りサイズ
* `read_type` ([String](/ja/reference/data-types/string)) — 読み取りタイプ: READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/ja/reference/data-types/int-uint)) — cache からの読み取りが試行されたかどうか
* `read_buffer_id` ([String](/ja/reference/data-types/string)) — 内部実装の読み取りバッファ ID
* `user_id` ([String](/ja/reference/data-types/string)) — file segment を作成したユーザーの ID

{/*AUTOGENERATED_END*/}
