<div id="description">
  ## 説明
</div>

ReplicatedMergeTree エンジン固有のすべての設定について、現在値、デフォルト値、および説明の一覧を示します。これらはすべて、CREATE クエリの SETTINGS セクションで変更できます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — 設定名。
* `value` ([String](/ja/reference/data-types/index)) — 設定値。
* `default` ([String](/ja/reference/data-types/index)) — 設定のデフォルト値。
* `changed` ([UInt8](/ja/reference/data-types/index)) — 設定が config で明示的に定義されているか、明示的に変更されている場合は 1。
* `description` ([String](/ja/reference/data-types/index)) — 設定の説明。
* `min` ([Nullable(String)](/ja/reference/data-types/index)) — 制約によって最小値が設定されている場合、その値。設定に最小値がない場合は NULL になります。
* `max` ([Nullable(String)](/ja/reference/data-types/index)) — 制約によって最大値が設定されている場合、その値。設定に最大値がない場合は NULL になります。
* `disallowed_values` ([Array(String)](/ja/reference/data-types/index)) — 許可されていない値の一覧
* `readonly` ([UInt8](/ja/reference/data-types/index)) — 現在のユーザーがこの設定を変更できるかどうかを示します: 0 — 現在のユーザーはこの設定を変更できます、1 — 現在のユーザーはこの設定を変更できません。
* `type` ([String](/ja/reference/data-types/index)) — 設定の型 (実装固有の文字列値) 。
* `is_obsolete` ([UInt8](/ja/reference/data-types/index)) — 設定が廃止されたものかどうかを示します。
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/ja/reference/data-types/index)) —
  この機能のサポートレベル。ClickHouse の機能はティアごとに整理されており、現在の
  開発状況や、使用時に期待できる内容に応じて分類されています:

- PRODUCTION: この機能は安定しており、安全に使用でき、他の PRODUCTION 機能と組み合わせても問題はありません。
- BETA: この機能は安定しており、安全に使用できます。他の機能と組み合わせた場合の結果は不明で、正確性は保証されません。テストと報告を歓迎します。
- EXPERIMENTAL: この機能は開発中です。開発者と ClickHouse 愛好家のみを対象としています。動作する場合もしない場合もあり、いつでも削除される可能性があります。
- OBSOLETE: すでにサポートされていません。すでに削除されているか、今後のリリースで削除される予定です。

{/*AUTOGENERATED_END*/}
