<div id="description">
  ## 説明
</div>

`clickhouse` バイナリのイントロスペクション情報が含まれています。アクセスするには、introspection 権限が必要です。
このテーブルが役立つのは、C++ の専門家と ClickHouse のエンジニアに限られます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/ja/reference/data-types/index)) — バイナリ内のシンボル名です。マングルされているため、読みやすい名前を取得するには `demangle(symbol)` を適用できます。
* `address_begin` ([UInt64](/ja/reference/data-types/index)) — バイナリ内のシンボルの開始アドレスです。
* `address_end` ([UInt64](/ja/reference/data-types/index)) — バイナリ内のシンボルの終了アドレスです。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
