<div id="description">
  ## 説明
</div>

通常の関数および集約関数に関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — 関数名。
* `is_aggregate` ([UInt8](/ja/reference/data-types/index)) — その関数が集約関数かどうか。
* `case_insensitive` ([UInt8](/ja/reference/data-types/index)) — 関数名を大文字と小文字を区別せずに使用できるかどうか。
* `alias_to` ([String](/ja/reference/data-types/index)) — 関数名がエイリアスである場合の元の関数名。
* `create_query` ([String](/ja/reference/data-types/index)) — 廃止された。
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/ja/reference/data-types/index)) — 廃止された。
* `description` ([String](/ja/reference/data-types/index)) — 関数の動作の概要。
* `syntax` ([String](/ja/reference/data-types/index)) — 関数のシグネチャ。
* `arguments` ([String](/ja/reference/data-types/index)) — 関数の引数。
* `parameters` ([String](/ja/reference/data-types/index)) — 関数のパラメーター (集約関数の場合のみ) 。
* `returned_value` ([String](/ja/reference/data-types/index)) — 関数が返す値。
* `examples` ([String](/ja/reference/data-types/index)) — 使用例。
* `introduced_in` ([String](/ja/reference/data-types/index)) — 関数が最初に導入された ClickHouse のバージョン。
* `categories` ([String](/ja/reference/data-types/index)) — 関数のカテゴリ。
* `deterministic` ([Nullable(UInt8)](/ja/reference/data-types/index)) — 同じ引数に対して常に同じ結果を返すかどうか。未知の場合 (たとえば集約関数やユーザー定義関数) は NULL。
* `higher_order` ([Nullable(UInt8)](/ja/reference/data-types/index)) — その関数が高階関数かどうか。つまり、少なくとも 1 つのラムダ式を引数として受け取るかどうか (例: arrayMap, arrayFilter, mapApply) 。未知の場合は NULL。

{/*AUTOGENERATED_END*/}
