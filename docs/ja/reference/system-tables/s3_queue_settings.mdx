<div id="description">
  ## 説明
</div>

[S3Queue](/ja/reference/engines/table-engines/integrations/s3queue) テーブルの設定に関する情報が含まれています。サーバーバージョン `24.10` 以降で利用できます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — S3Queueエンジンを持つテーブルのdatabase。
* `table` ([String](/ja/reference/data-types/index)) — S3Queueエンジンを持つtableの名前。
* `name` ([String](/ja/reference/data-types/index)) — 設定名。
* `value` ([String](/ja/reference/data-types/index)) — 設定の値。
* `type` ([String](/ja/reference/data-types/index)) — 設定の型 (実装固有の文字列値) 。
* `changed` ([UInt8](/ja/reference/data-types/index)) — 設定がconfigで明示的に定義されている、または明示的に変更されている場合は 1。
* `description` ([String](/ja/reference/data-types/index)) — 設定の説明。
* `alterable` ([UInt8](/ja/reference/data-types/index)) — 現在のユーザーが ALTER TABLE MODIFY SETTING で設定を変更できるかどうかを示します: 0 — 現在のユーザーは設定を変更できません、1 — 現在のユーザーは設定を変更できます。

{/*AUTOGENERATED_END*/}
