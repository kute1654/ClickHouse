<div id="description">
  ## 説明
</div>

クラスターで実行された[分散 DDL クエリ (ON CLUSTER 句) ](/ja/reference/statements/distributed-ddl)に関する情報が含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/ja/reference/data-types/index)) — クエリ ID。
* `entry_version` ([Nullable(UInt8)](/ja/reference/data-types/index)) — エントリのバージョン。
* `initiator_host` ([Nullable(String)](/ja/reference/data-types/index)) — DDL 操作を開始したホスト。
* `initiator_port` ([Nullable(UInt16)](/ja/reference/data-types/index)) — イニシエーターが使用したポート。
* `cluster` ([String](/ja/reference/data-types/index)) — クラスター名。判別できない場合は空です。
* `query` ([String](/ja/reference/data-types/index)) — 実行されたクエリ。
* `settings` ([Map(String, String)](/ja/reference/data-types/index)) — DDL 操作で使用された設定。
* `query_create_time` ([DateTime](/ja/reference/data-types/index)) — クエリの作成時刻。
* `host` ([Nullable(String)](/ja/reference/data-types/index)) — ホスト名。
* `port` ([Nullable(UInt16)](/ja/reference/data-types/index)) — ホストのポート番号。
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/ja/reference/data-types/index)) — クエリのステータス。
* `exception_code` ([Nullable(UInt16)](/ja/reference/data-types/index)) — 例外コード。
* `exception_text` ([Nullable(String)](/ja/reference/data-types/index)) — 例外メッセージ。
* `query_finish_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — クエリの終了時刻。
* `query_duration_ms` ([Nullable(UInt64)](/ja/reference/data-types/index)) — クエリ実行時間 (ミリ秒単位) 。

{/*AUTOGENERATED_END*/}
