<div id="description">
  ## 説明
</div>

[クエリキャッシュ](/ja/concepts/features/performance/caches/query-cache) の内容を表示します。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ja/reference/data-types/index)) — クエリ文字列。
* `query_id` ([String](/ja/reference/data-types/index)) — クエリの ID。
* `is_subquery` ([UInt8](/ja/reference/data-types/index)) — クエリキャッシュ のエントリがサブクエリの結果かどうか。
* `result_size` ([UInt64](/ja/reference/data-types/index)) — クエリキャッシュ のエントリのサイズ。
* `tag` ([LowCardinality(String)](/ja/reference/data-types/index)) — クエリキャッシュ のエントリのタグ。
* `stale` ([UInt8](/ja/reference/data-types/index)) — クエリキャッシュ のエントリが stale かどうか。
* `shared` ([UInt8](/ja/reference/data-types/index)) — クエリキャッシュ のエントリが複数ユーザー間で共有されているかどうか。
* `compressed` ([UInt8](/ja/reference/data-types/index)) — クエリキャッシュ のエントリが圧縮されているかどうか。
* `expires_at` ([DateTime](/ja/reference/data-types/index)) — クエリキャッシュ のエントリが stale になる時刻。
* `key_hash` ([UInt64](/ja/reference/data-types/index)) — クエリ文字列のハッシュ。クエリキャッシュ のエントリを見つけるためのキーとして使用されます。

{/*AUTOGENERATED_END*/}
