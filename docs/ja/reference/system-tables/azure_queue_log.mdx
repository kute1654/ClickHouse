AzureQueue エンジンで処理されたファイルに関する情報を含むログエントリが格納されています。

このテーブルは、いつでも TRUNCATE または DROP しても安全です。

カラム:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ja/reference/data-types/lowcardinality)) — ホスト名
* `event_date` ([Date](/ja/reference/data-types/date)) — このログ行が書き込まれたイベントの日付
* `event_time` ([DateTime](/ja/reference/data-types/datetime)) — このログ行が書き込まれたイベントの時刻
* `database` ([String](/ja/reference/data-types/string)) — キュー テーブル (`S3Queue` または `AzureQueue`) が存在するデータベース名。
* `table` ([String](/ja/reference/data-types/string)) — キュー テーブル (`S3Queue` または `AzureQueue`) の名前。
* `uuid` ([String](/ja/reference/data-types/string)) — キュー テーブル (`S3Queue` または `AzureQueue`) の UUID。
* `file_name` ([String](/ja/reference/data-types/string)) — 処理対象ファイルの名前。
* `rows_processed` ([UInt64](/ja/reference/data-types/int-uint)) — 処理された行数。
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/ja/reference/data-types/enum)) — 処理対象ファイルのステータス。
* `processing_start_time` ([Nullable(DateTime)](/ja/reference/data-types/nullable)) — ファイル処理の開始時刻。
* `processing_end_time` ([Nullable(DateTime)](/ja/reference/data-types/nullable)) — ファイル処理の終了時刻。
* `exception` ([String](/ja/reference/data-types/string)) — 発生した場合の例外メッセージ。
* `commit_id` ([UInt64](/ja/reference/data-types/int-uint)) — このファイルがコミットされたトランザクションの ID。
* `commit_time` ([DateTime](/ja/reference/data-types/datetime)) — Keeper でファイルが (失敗または処理済みとして) コミットされた時刻。
* `transaction_start_time` ([DateTime](/ja/reference/data-types/datetime)) — 処理トランザクション全体の開始時刻。
* `get_object_time_ms` ([UInt64](/ja/reference/data-types/int-uint)) — オブジェクトストレージ内でオブジェクトを見つけるのに要した時間。

{/*AUTOGENERATED_END*/}
