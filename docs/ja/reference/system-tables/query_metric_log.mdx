<div id="description">
  ## 説明
</div>

個々のクエリについて、`system.events` テーブルのメモリおよびメトリックの値の履歴が保存され、定期的にディスクへフラッシュされます。

クエリが開始されると、`query_metric_log_interval` ミリ秒間隔でデータが収集されます (デフォルトは 1000
です) 。また、クエリの実行時間が `query_metric_log_interval` より長い場合は、クエリ終了時にもデータが収集されます。
