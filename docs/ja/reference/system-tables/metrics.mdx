<div id="description">
  ## 説明
</div>

即座に計算できるメトリクス、または現在の値を持つメトリクスが含まれます。たとえば、同時に処理されているクエリ数や、現在のレプリカ遅延です。このテーブルは常に最新の状態に保たれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ja/reference/data-types/index)) — メトリック名。
* `value` ([Int64](/ja/reference/data-types/index)) — メトリック値。
* `description` ([String](/ja/reference/data-types/index)) — メトリックの説明。

{/*AUTOGENERATED_END*/}

サポートされているすべてのメトリクスは、ソースファイル [src/Common/CurrentMetrics.cpp](https://github.com/ClickHouse/ClickHouse/blob/master/src/Common/CurrentMetrics.cpp) で確認できます。

<div id="example">
  ## 例
</div>

```sql
SELECT * FROM system.metrics LIMIT 10
```

```text
┌─metric───────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────┐
│ Query                                │     1 │ 実行中のクエリ数                                                       │
│ Merge                                │     0 │ 実行中のバックグラウンドマージ数                                       │
│ PartMutation                         │     0 │ ミューテーション数 (ALTER DELETE/UPDATE)                               │
│ ReplicatedFetch                      │     0 │ レプリカからフェッチ中のデータパーツ数                                 │
│ ReplicatedSend                       │     0 │ レプリカへ送信中のデータパーツ数                                       │
│ ReplicatedChecks                     │     0 │ 整合性チェック中のデータパーツ数                                       │
│ BackgroundMergesAndMutationsPoolTask │     0 │ 関連するバックグラウンドプール内のアクティブなマージおよびミューテーション数 │
│ BackgroundFetchesPoolTask            │     0 │ 関連するバックグラウンドプール内のアクティブなフェッチ数               │
│ BackgroundCommonPoolTask             │     0 │ 関連するバックグラウンドプール内のアクティブなタスク数                 │
│ BackgroundMovePoolTask               │     0 │ 移動用 BackgroundProcessingPool 内のアクティブなタスク数              │
└──────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────┘
```
