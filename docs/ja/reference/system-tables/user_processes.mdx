<div id="description">
  ## 説明
</div>

このシステムテーブルを使用すると、ユーザーのメモリ使用量と ProfileEvents の概要を確認できます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `user` ([String](/ja/reference/data-types/index)) — ユーザー名。
* `memory_usage` ([Int64](/ja/reference/data-types/index)) — ユーザーのすべてのプロセスで使用される RAM の合計。一部の専用メモリ種別は含まれない場合があります。`max_memory_usage` 設定を参照してください。
* `peak_memory_usage` ([Int64](/ja/reference/data-types/index)) — ユーザーのメモリ使用量のピーク値。ユーザーに対してクエリが実行されていない場合は、リセットされることがあります。
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/ja/reference/data-types/index)) — ユーザーに関するさまざまなメトリクスを測定する ProfileEvents のサマリーです。各項目の説明は `system.events` テーブルを参照してください。

{/*AUTOGENERATED_END*/}
