<div id="description">
  ## 説明
</div>

ClickHouse server でサポートされているタイムゾーンの一覧です。この一覧は、ClickHouse のバージョンによって異なる場合があります。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/ja/reference/data-types/index)) — 対応しているタイムゾーンの一覧。

{/*AUTOGENERATED_END*/}

<div id="example">
  ## 例
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```

```text
┌─time_zone──────────┐
│ Africa/Abidjan     │
│ Africa/Accra       │
│ Africa/Addis_Ababa │
│ Africa/Algiers     │
│ Africa/Asmara      │
│ Africa/Asmera      │
│ Africa/Bamako      │
│ Africa/Bangui      │
│ Africa/Banjul      │
│ Africa/Bissau      │
└────────────────────┘
```
