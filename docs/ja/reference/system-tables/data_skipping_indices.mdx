<div id="description">
  ## 説明
</div>

すべてのテーブルにある既存のデータスキッピングインデックスに関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — データベース名。
* `table` ([String](/ja/reference/data-types/index)) — テーブル名。
* `name` ([String](/ja/reference/data-types/index)) — 索引名。
* `type` ([String](/ja/reference/data-types/index)) — 索引のタイプ。
* `type_full` ([String](/ja/reference/data-types/index)) — CREATEステートメント内の索引タイプの式。
* `expr` ([String](/ja/reference/data-types/index)) — 索引の計算式。
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/ja/reference/data-types/index)) — 索引が暗黙的に作成されたかどうか (`add_minmax_index_for_numeric_columns` などによる)
* `granularity` ([UInt64](/ja/reference/data-types/index)) — block 内の granule の数。
* `data_compressed_bytes` ([UInt64](/ja/reference/data-types/index)) — 圧縮データのサイズ (バイト単位) 。
* `data_uncompressed_bytes` ([UInt64](/ja/reference/data-types/index)) — 非圧縮データのサイズ (バイト単位) 。
* `marks_bytes` ([UInt64](/ja/reference/data-types/index)) — mark のサイズ (バイト単位) 。

{/*AUTOGENERATED_END*/}
