<div id="description">
  ## 説明
</div>

このシステムテーブルには、ClickHouse に存在する Iceberg テーブルのスナップショット履歴が格納されます。ClickHouse に Iceberg テーブルが存在しない場合、このテーブルは空になります。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — データベース名。
* `table` ([String](/ja/reference/data-types/index)) — テーブル名。
* `made_current_at` ([Nullable(DateTime64(3))](/ja/reference/data-types/index)) — このスナップショットが現在のスナップショットになった日時。
* `snapshot_id` ([UInt64](/ja/reference/data-types/index)) — スナップショットの識別に使用されるスナップショット ID。
* `parent_id` ([UInt64](/ja/reference/data-types/index)) — このスナップショットの親 ID。
* `is_current_ancestor` ([UInt8](/ja/reference/data-types/index)) — このスナップショットが現在のスナップショットの祖先であるかどうかを示すフラグ。

{/*AUTOGENERATED_END*/}
