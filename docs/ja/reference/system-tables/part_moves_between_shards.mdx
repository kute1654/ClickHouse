<div id="description">
  ## 説明
</div>

現在、分片間で移動中のパーツと、その進捗に関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — 移動操作が実行されるデータベース名。
* `table` ([String](/ja/reference/data-types/index)) — 移動操作が実行されるテーブル名。
* `task_name` ([String](/ja/reference/data-types/index)) — 移動タスクの名前。
* `task_uuid` ([UUID](/ja/reference/data-types/index)) — 移動タスクの識別子。
* `create_time` ([DateTime](/ja/reference/data-types/index)) — タスクが作成された時刻。
* `part_name` ([String](/ja/reference/data-types/index)) — 移動中のパーツ名。
* `part_uuid` ([UUID](/ja/reference/data-types/index)) — 移動中のパーツの UUID。
* `to_shard` ([String](/ja/reference/data-types/index)) — 宛先分片の名前。
* `dst_part_name` ([String](/ja/reference/data-types/index)) — 結果のパーツ名。
* `update_time` ([DateTime](/ja/reference/data-types/index)) — 最後に更新が行われた時刻。
* `state` ([String](/ja/reference/data-types/index)) — 移動の現在の状態。
* `rollback` ([UInt8](/ja/reference/data-types/index)) — 操作がロールバックされたかどうかを示すフラグ。
* `num_tries` ([UInt32](/ja/reference/data-types/index)) — 操作を完了するまでの試行回数。
* `last_exception` ([String](/ja/reference/data-types/index)) — 最後の例外名 (存在する場合) 。

{/*AUTOGENERATED_END*/}
