<div id="description">
  ## 説明
</div>

設定プロファイルの内容を示します。

* 制約。
* この設定が適用されるロールとユーザー。
* 親設定プロファイル。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/ja/reference/data-types/index)) — 設定プロファイル名。
* `user_name` ([Nullable(String)](/ja/reference/data-types/index)) — ユーザー名。
* `role_name` ([Nullable(String)](/ja/reference/data-types/index)) — ロール名。
* `index` ([UInt64](/ja/reference/data-types/index)) — 設定プロファイル要素の連番。
* `setting_name` ([Nullable(String)](/ja/reference/data-types/index)) — 設定名。
* `value` ([Nullable(String)](/ja/reference/data-types/index)) — 設定値。
* `min` ([Nullable(String)](/ja/reference/data-types/index)) — 設定の最小値。設定されていない場合は NULL。
* `max` ([Nullable(String)](/ja/reference/data-types/index)) — 設定の最大値。設定されていない場合は NULL。
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/ja/reference/data-types/index)) — 設定を変更できるかどうかを示すプロパティ。
* `inherit_profile` ([Nullable(String)](/ja/reference/data-types/index)) — この設定プロファイルの親となるプロファイル。設定されていない場合は NULL。設定プロファイルは、親プロファイルからすべての設定値と制約 (min、max、readonly) を継承します。

{/*AUTOGENERATED_END*/}
