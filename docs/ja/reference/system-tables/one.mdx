<div id="description">
  ## 説明
</div>

このテーブルには、値 0 を含む `dummy` UInt8 カラムが 1 つだけある 1 行のデータが含まれます。

このテーブルは、`SELECT` クエリで `FROM` 句が指定されていない場合に使用されます。

これは、他の DBMS にある `DUAL` テーブルに似ています。
