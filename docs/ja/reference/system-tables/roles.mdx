<div id="description">
  ## 説明
</div>

設定されている[ロール](/ja/concepts/features/security/access-rights#role-management)に関する情報を含みます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — ロール名。
* `id` ([UUID](/ja/reference/data-types/index)) — ロール ID。
* `storage` ([String](/ja/reference/data-types/index)) — ロールのストレージへのパス。`access_control_path` パラメータで設定します。

{/*AUTOGENERATED_END*/}
