<div id="description">
  ## 説明

すべてのユーザーのクォータ使用状況。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ja/reference/data-types/index)) — クォータ名。
* `quota_key` ([String](/ja/reference/data-types/index)) — キー値。
* `is_current` ([UInt8](/ja/reference/data-types/index)) — 現在のユーザーのクォータ使用状況。
* `start_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — リソース消費量の計算開始時刻。
* `end_time` ([Nullable(DateTime)](/ja/reference/data-types/index)) — リソース消費量の計算終了時刻。
* `duration` ([Nullable(UInt32)](/ja/reference/data-types/index)) — リソース消費量の計算対象となる時間間隔の長さ (秒単位) 。
* `queries` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されているクエリ数。
* `max_queries` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行可能な全種類のクエリの最大数。
* `query_selects` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されているSELECTクエリ数。
* `max_query_selects` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行可能なSELECTクエリの最大数。
* `query_inserts` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在実行されているINSERTクエリ数。
* `max_query_inserts` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 実行可能なINSERTクエリの最大数。
* `errors` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在エラーになっているクエリ数。
* `max_errors` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可されるエラー発生クエリ数の最大値。
* `result_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、全クエリの結果セット内の行の合計数。
* `max_result_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、全クエリの結果セット内の行の合計最大数。
* `result_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、全クエリの結果セット内のバイト総数。
* `max_result_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、全クエリの結果セット内のバイト総数の最大値。
* `read_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、全クエリの実行中に読み取られた行の合計数。
* `max_read_rows` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、全クエリの実行中に読み取る行数の最大値。
* `read_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、全クエリの実行中に読み取られたバイト総数。
* `max_read_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、全クエリの実行中に読み取るバイト数の最大値。
* `execution_time` ([Nullable(Float64)](/ja/reference/data-types/index)) — 現在の期間内にクエリの実行に費やされた時間の合計 (ナノ秒単位)
* `max_execution_time` ([Nullable(Float64)](/ja/reference/data-types/index)) — 指定期間内に全クエリの実行に許可される時間の最大値 (ナノ秒単位)
* `written_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、全クエリの実行中に書き込まれたバイト総数。
* `max_written_bytes` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、全クエリの実行中に書き込むバイト数の最大値。
* `failed_sequential_authentications` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、連続した認証失敗回数。
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、連続した認証失敗回数の最大値。
* `queries_per_normalized_hash` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 現在の期間内における、単一の正規化クエリの最大実行回数。
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ja/reference/data-types/index)) — 指定期間内に許可される、単一の正規化クエリの最大実行回数。
