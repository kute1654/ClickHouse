<div id="description">
  ## 説明
</div>

現時点で有効なすべてのロールが含まれます。これには、現在のユーザーの現在のロールと、現在のロールに付与されているロールが含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/ja/reference/data-types/index)) — ロール名。
* `with_admin_option` ([UInt8](/ja/reference/data-types/index)) — ロールに ADMIN OPTION 権限がある場合は 1。
* `is_current` ([UInt8](/ja/reference/data-types/index)) — `enabled_role` が現在のユーザーの現在のロールかどうかを示すフラグ。
* `is_default` ([UInt8](/ja/reference/data-types/index)) — `enabled_role` がデフォルトロールかどうかを示すフラグ。

{/*AUTOGENERATED_END*/}
