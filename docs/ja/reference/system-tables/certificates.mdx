<div id="description">
  ## 説明
</div>

利用可能な証明書とそのSOURCESに関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/ja/reference/data-types/index)) — 証明書のバージョン。値は、v1 が 0、v2 が 1、v3 が 2 です。
* `serial_number` ([Nullable(String)](/ja/reference/data-types/index)) — 発行者によって割り当てられた証明書のシリアル番号。
* `signature_algo` ([Nullable(String)](/ja/reference/data-types/index)) — 署名アルゴリズム - 発行者がこの証明書に署名するために使用するアルゴリズムです。
* `issuer` ([Nullable(String)](/ja/reference/data-types/index)) — 発行者 - この証明書を発行した認証局を一意に識別する識別子です。
* `not_before` ([Nullable(String)](/ja/reference/data-types/index)) — この証明書の有効期間の開始時刻です。
* `not_after` ([Nullable(String)](/ja/reference/data-types/index)) — この証明書の有効期間の終了時刻です。
* `subject` ([Nullable(String)](/ja/reference/data-types/index)) — サブジェクト - 公開鍵の所有者を識別します。
* `pkey_algo` ([Nullable(String)](/ja/reference/data-types/index)) — 公開鍵アルゴリズム - この公開鍵で使用できるアルゴリズムを定義します。
* `path` ([String](/ja/reference/data-types/index)) — この証明書を含むファイルまたはディレクトリへのパス。
* `default` ([UInt8](/ja/reference/data-types/index)) — 証明書がデフォルトの証明書格納場所にあることを示します。
* `protocol` ([String](/ja/reference/data-types/index)) — 証明書がプロトコルごとの TLS 設定に由来する場合はプロトコル名、それ以外の場合は空です。

{/*AUTOGENERATED_END*/}
