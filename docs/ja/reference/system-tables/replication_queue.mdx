<div id="description">
  ## 説明
</div>

`ReplicatedMergeTree` ファミリーのテーブルについて、ClickHouse Keeper または ZooKeeper に保存されているレプリケーションキュー内のタスクに関する情報が含まれます。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ja/reference/data-types/index)) — データベース名。
* `table` ([String](/ja/reference/data-types/index)) — テーブル名。
* `replica_name` ([String](/ja/reference/data-types/index)) — ClickHouse Keeper 内のレプリカ名。同じテーブルのレプリカには、それぞれ異なる名前が付けられます。
* `position` ([UInt32](/ja/reference/data-types/index)) — キュー内のタスクの位置。
* `node_name` ([String](/ja/reference/data-types/index)) — ClickHouse Keeper 内のノード名。
* `type` ([String](/ja/reference/data-types/index)) — キュー内のタスクの種類。次のいずれかです: • GET&#95;PART — 別のレプリカからパーツを取得します。 • ATTACH&#95;PART — パーツをアタッチします。必要に応じて、自身のレプリカからアタッチすることもあります (detached フォルダー内に見つかった場合) 。ほぼ同じ処理ですが一部最適化が加えられているため、GET&#95;PART の最適化版と考えることができます。 • MERGE&#95;PARTS — パーツをマージします。 • DROP&#95;RANGE — 指定したパーティション内の指定範囲の番号にあるパーツを削除します。 • CLEAR&#95;COLUMN — 注: 非推奨。指定したパーティションから特定のカラムを削除します。 • CLEAR&#95;INDEX — 注: 非推奨。指定したパーティションから特定の索引を削除します。 • REPLACE&#95;RANGE — 特定範囲のパーツを削除し、新しいパーツに置き換えます。 • MUTATE&#95;PART — 1 つ以上のミューテーションをパーツに適用します。 • ALTER&#95;METADATA — グローバルな /metadata および /columns パスに従って ALTER による変更を適用します。
* `create_time` ([DateTime](/ja/reference/data-types/index)) — タスクが実行のために送信された日時。
* `required_quorum` ([UInt32](/ja/reference/data-types/index)) — 完了確認付きでタスクの完了を待機しているレプリカ数。このカラムは GET&#95;PARTS タスクにのみ関連します。
* `source_replica` ([String](/ja/reference/data-types/index)) — ソースレプリカ名。
* `new_part_name` ([String](/ja/reference/data-types/index)) — 新しいパーツ名。
* `parts_to_merge` ([Array(String)](/ja/reference/data-types/index)) — マージまたは更新するパーツの名前。
* `is_detach` ([UInt8](/ja/reference/data-types/index)) — DETACH&#95;PARTS タスクがキュー内にあるかどうかを示すフラグ。
* `is_currently_executing` ([UInt8](/ja/reference/data-types/index)) — 特定のタスクが現在実行中かどうかを示すフラグ。
* `num_tries` ([UInt32](/ja/reference/data-types/index)) — タスクの完了に失敗した試行回数。
* `last_exception` ([String](/ja/reference/data-types/index)) — 最後に発生したエラーに関するテキストメッセージ (ある場合) 。
* `last_exception_time` ([DateTime](/ja/reference/data-types/index)) — 最後のエラーが発生した日時。
* `last_attempt_time` ([DateTime](/ja/reference/data-types/index)) — タスクが最後に試行された日時。
* `num_postponed` ([UInt32](/ja/reference/data-types/index)) — アクションが延期された回数。
* `postpone_reason` ([String](/ja/reference/data-types/index)) — タスクが延期された理由。
* `last_postpone_time` ([DateTime](/ja/reference/data-types/index)) — タスクが最後に延期された日時。
* `merge_type` ([String](/ja/reference/data-types/index)) — 現在のマージの種類。ミューテーションの場合は空です。

{/*AUTOGENERATED_END*/}
