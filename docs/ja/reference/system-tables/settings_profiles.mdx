<div id="description">
  ## 説明
</div>

設定済みの設定プロファイルのプロパティが含まれています。

<div id="columns">
  ## カラム
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ja/reference/data-types/index)) — 設定プロファイル名。
* `id` ([UUID](/ja/reference/data-types/index)) — 設定プロファイルの ID。
* `storage` ([String](/ja/reference/data-types/index)) — 設定プロファイルの保存先へのパス。`access_control_path` パラメーターで設定します。
* `num_elements` ([UInt64](/ja/reference/data-types/index)) — `system.settings_profile_elements` テーブル内のこのプロファイルの要素数。
* `apply_to_all` ([UInt8](/ja/reference/data-types/index)) — この設定プロファイルがすべてのロールおよび/またはユーザーに設定されていることを示します。
* `apply_to_list` ([Array(String)](/ja/reference/data-types/index)) — この設定プロファイルが適用されるロールおよび/またはユーザーの一覧。
* `apply_to_except` ([Array(String)](/ja/reference/data-types/index)) — 一覧に含まれるものを除く、すべてのロールおよび/またはユーザーにこの設定プロファイルが適用されます。

{/*AUTOGENERATED_END*/}
