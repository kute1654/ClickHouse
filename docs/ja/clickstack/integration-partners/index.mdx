ClickStack は、最新のオブザーバビリティに必要な中核機能を提供し、データ収集、インジェスト、保存、可視化を統合した高性能なスタックとして実現します。インテグレーションパートナーは、ClickStack が標準で提供する機能に加えて、専門的な機能を備えた補完的なツールでその機能を拡張します。ClickStack とそのパートナーを組み合わせることで、チームはパフォーマンス、柔軟性、スケーラビリティを損なうことなく、ニーズに合わせた包括的なオブザーバビリティプラットフォームを構築できます。

| セクション                                                   | 説明                                                                                   |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| [Bindplane](/ja/clickstack/integration-partners/bindplane) | Bindplane は、OpenTelemetry Collector の一元管理を実現する OpenTelemetry ネイティブのテレメトリーパイプラインです。   |
| [Odigos](/ja/clickstack/integration-partners/odigos)       | Odigos は eBPF と OpenTelemetry を組み合わせ、バイトコードエージェントよりも大幅に低いオーバーヘッドで、より詳細な分散トレースを実現します。 |
