<div id="clickstack-demo-days">
  ## ClickStack デモデイズ
</div>

これらのフォルダーには、デモデイズの録画が含まれています。デモデイズは社内の開発ミーティングで、ClickStack で現在開発中のものや検証中のものを共有しています。

ここでご覧いただくものの一部は製品に組み込まれます。一部は早い段階でフィードバックを得るために共有されています。また、実験的なものもあり、リリースされない可能性もあります。

気になるものがあれば、リンク先の PR を確認のうえ、ぜひご意見をお聞かせください。ここを、コミュニティが進行中の取り組みを身近に把握し、ClickStack の今後の方向性づくりに関われる場にしたいと考えています。

質問やフィードバックがある場合は、ClickHouse の Slack チャンネル [#o11y-clickstack](https://clickhouse.com/slack) または [公開リポジトリ](https://github.com/hyperdxio/hyperdx/tree/main) からご連絡ください。
