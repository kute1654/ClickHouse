このセクションでは、ClickStackを使い始めるのに役立つ各種サンプルデータセットと例を紹介します。これらの例では、ローカル開発から本番環境まで、ClickStackでオブザーバビリティデータを扱うさまざまな方法を確認できます。

| データセット                                                                        | 説明                                                                                                                                    |
| ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| [Sample Data](/ja/clickstack/example-datasets/sample-data)                       | デモ環境の logs、traces、metrics を含むサンプルデータセットを読み込みます                                                                                        |
| [Local Data](/ja/clickstack/example-datasets/local-data)                         | ローカルシステムのメトリクスと logs を収集し、分析のためにClickStackへ送信します                                                                                      |
| [Remote Demo Data](/ja/clickstack/example-datasets/remote-demo-data)             | リモートのデモクラスターに接続して、問題を調査します                                                                                                            |
| [Session Replay Demo](/ja/clickstack/example-datasets/session-replay)            | セッションリプレイ用にデモWebアプリケーションをインストルメントし、ClickStackで操作内容を確認します                                                                              |
| [Chrome Extension](/ja/clickstack/example-datasets/chrome-extension)             | HyperDX Chrome拡張機能を使って任意のWebサイトにBrowser SDKを挿入できます。application code の変更は不要です                                                          |
| [Synthetic data with otelgen](/ja/clickstack/example-datasets/otelgen)           | `otelgen` を使って、合成 logs、traces、metrics を実行中のClickStack OpenTelemetry collector に送信します                                                  |
| [Synthetic data with telemetrygen](/ja/clickstack/example-datasets/telemetrygen) | `telemetrygen` を使って、サービス、重大度、span ステータス、メトリクスタイプごとにflagで生成した多様な合成 logs、traces、metrics を、実行中のClickStack OpenTelemetry collector に送信します |
| [HackerNews Analyzer](/ja/clickstack/example-datasets/instrument-application)    | Node.jsアプリケーションであるHackerNews AnalyzerをOpenTelemetryでインストルメントし、そのlogs、metrics、tracesをManaged ClickStackに送信します                          |
