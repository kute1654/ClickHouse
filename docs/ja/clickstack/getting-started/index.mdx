ClickStack は、運用方針や要件に応じて、いくつかの方法でデプロイできます。

* [Managed ClickStack](/ja/clickstack/getting-started/managed) - **最も簡単に**始められる方法で、**ClickHouse Cloud** 上に **Managed ClickStack** をデプロイします。
* [Open-Source ClickStack](/ja/clickstack/getting-started/oss) - 完全なオープンソース構成を自分で運用したい方向けです。
