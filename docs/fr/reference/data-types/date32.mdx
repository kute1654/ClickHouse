Une date. Prend en charge la même plage de dates que [DateTime64](/fr/reference/data-types/datetime64). Elle est stockée sous la forme d’un entier signé de 32 bits dans l’ordre natif des octets, la valeur représentant le nombre de jours écoulés depuis `1900-01-01`. **Important !** 0 représente `1970-01-01` et les valeurs négatives représentent les jours antérieurs à `1970-01-01`.

**Exemples**

Création d’une table avec une colonne de type `Date32` et insertion de données dans celle-ci :

```sql
CREATE TABLE dt32
(
    `timestamp` Date32,
    `event_id` UInt8
)
ENGINE = TinyLog;
```

```sql
-- Parse Date
-- - from string,
-- - from 'small' integer interpreted as number of days since 1970-01-01, and
-- - from 'big' integer interpreted as number of seconds since 1970-01-01.
INSERT INTO dt32 VALUES ('2100-01-01', 1), (47482, 2), (4102444800, 3);

SELECT * FROM dt32;
```

```text
┌──timestamp─┬─event_id─┐
│ 2100-01-01 │        1 │
│ 2100-01-01 │        2 │
│ 2100-01-01 │        3 │
└────────────┴──────────┘
```
