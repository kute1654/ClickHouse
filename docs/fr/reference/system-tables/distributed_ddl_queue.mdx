<div id="description">
  ## Description
</div>

Contient des informations sur les [requêtes DDL distribuées (clause ON CLUSTER)](/fr/reference/statements/distributed-ddl) qui ont été exécutées dans un cluster.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/fr/reference/data-types/index)) — ID de la requête.
* `entry_version` ([Nullable(UInt8)](/fr/reference/data-types/index)) — Version de l’entrée.
* `initiator_host` ([Nullable(String)](/fr/reference/data-types/index)) — Hôte ayant lancé l’opération DDL.
* `initiator_port` ([Nullable(UInt16)](/fr/reference/data-types/index)) — Port utilisé par l’initiateur.
* `cluster` ([String](/fr/reference/data-types/index)) — Nom du cluster, vide s’il n’a pas pu être déterminé.
* `query` ([String](/fr/reference/data-types/index)) — Requête exécutée.
* `settings` ([Map(String, String)](/fr/reference/data-types/index)) — Paramètres utilisés pour l’opération DDL.
* `query_create_time` ([DateTime](/fr/reference/data-types/index)) — Heure de création de la requête.
* `host` ([Nullable(String)](/fr/reference/data-types/index)) — Nom d’hôte.
* `port` ([Nullable(UInt16)](/fr/reference/data-types/index)) — Port de l’hôte.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/fr/reference/data-types/index)) — Statut de la requête.
* `exception_code` ([Nullable(UInt16)](/fr/reference/data-types/index)) — Code de l’exception.
* `exception_text` ([Nullable(String)](/fr/reference/data-types/index)) — Message de l’exception.
* `query_finish_time` ([Nullable(DateTime)](/fr/reference/data-types/index)) — Heure de fin de la requête.
* `query_duration_ms` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Durée d’exécution de la requête (en millisecondes).

{/*AUTOGENERATED_END*/}
