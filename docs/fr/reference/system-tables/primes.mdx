<div id="description">
  ## Description
</div>

Cette table contient une unique colonne UInt64 nommée `prime`, qui contient des nombres premiers dans l’ordre croissant, à partir de 2.

Vous pouvez utiliser cette table pour des tests, ou si vous devez effectuer une recherche par force brute sur des nombres premiers.

Les lectures depuis cette table ne sont pas parallélisées.

Cette table est similaire à la fonction de table [`primes`](/fr/reference/functions/table-functions/primes).

Vous pouvez également limiter la sortie à l’aide de prédicats.

```sql
SELECT * FROM system.primes LIMIT 10;
```

Le premier nombre premier supérieur à 1e15.

```sql
SELECT prime FROM system.primes WHERE prime > 1e15 LIMIT 1;
```

```response

  │ 1000000000000037 │ -- 1.00 quadrillion
  └──────────────────┘
```
