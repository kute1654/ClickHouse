<div id="description">
  ## Description
</div>

Contient des informations sur les [dictionnaires](/fr/reference/statements/create/dictionary).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données contenant le dictionnaire créé par la requête DDL. Chaîne vide pour les autres dictionnaires.
* `name` ([String](/fr/reference/data-types/index)) — Nom du dictionnaire.
* `uuid` ([UUID](/fr/reference/data-types/index)) — UUID du dictionnaire.
* `status` ([Enum8(&#39;NOT&#95;LOADED&#39; = 0, &#39;LOADED&#39; = 1, &#39;FAILED&#39; = 2, &#39;LOADING&#39; = 3, &#39;FAILED&#95;AND&#95;RELOADING&#39; = 4, &#39;LOADED&#95;AND&#95;RELOADING&#39; = 5, &#39;NOT&#95;EXIST&#39; = 6)](/fr/reference/data-types/index)) — Statut du dictionnaire. Valeurs possibles :
  * **NOT&#95;LOADED** — Le dictionnaire n’a pas été chargé, car il n’a pas été utilisé
  * **LOADED** — Le dictionnaire a été chargé avec succès
  * **FAILED** — Impossible de charger le dictionnaire à la suite d’une erreur
  * **LOADING** — Le dictionnaire est en cours de chargement
  * **LOADED&#95;AND&#95;RELOADING** — Le dictionnaire est chargé avec succès
  * **and is being reloaded right now (frequent reasons: SYSTEM RELOAD DICTIONARY query** —
  * **timeout** —
  * **dictionary config has changed)** —
  * **FAILED&#95;AND&#95;RELOADING** — Le dictionnaire n’a pas pu être chargé à la suite d’une erreur et est en cours de rechargement.
* `origin` ([String](/fr/reference/data-types/index)) — Chemin vers le fichier de configuration qui décrit le dictionnaire.
* `type` ([String](/fr/reference/data-types/index)) — Type d’allocation du dictionnaire. Stockage des dictionnaires en mémoire.
* `key.names` ([Array(String)](/fr/reference/data-types/index)) — Tableau des noms de clés fournis par le dictionnaire.
* `key.types` ([Array(String)](/fr/reference/data-types/index)) — Tableau correspondant des types de clés fournis par le dictionnaire.
* `attribute.names` ([Array(String)](/fr/reference/data-types/index)) — Tableau des noms d’attributs fournis par le dictionnaire.
* `attribute.types` ([Array(String)](/fr/reference/data-types/index)) — Tableau correspondant des types d’attributs fournis par le dictionnaire.
* `bytes_allocated` ([UInt64](/fr/reference/data-types/index)) — Quantité de RAM allouée au dictionnaire.
* `hierarchical_index_bytes_allocated` ([UInt64](/fr/reference/data-types/index)) — Quantité de RAM allouée à l’index hiérarchique.
* `query_count` ([UInt64](/fr/reference/data-types/index)) — Nombre de requêtes depuis le chargement du dictionnaire ou depuis le dernier redémarrage réussi.
* `hit_rate` ([Float64](/fr/reference/data-types/index)) — Pour les dictionnaires de type cache, pourcentage d’utilisations pour lesquelles la valeur se trouvait dans le cache.
* `found_rate` ([Float64](/fr/reference/data-types/index)) — Pourcentage d’utilisations pour lesquelles la valeur a été trouvée.
* `element_count` ([UInt64](/fr/reference/data-types/index)) — Nombre d’éléments stockés dans le dictionnaire.
* `load_factor` ([Float64](/fr/reference/data-types/index)) — Pourcentage de remplissage du dictionnaire (pour un dictionnaire haché, pourcentage de remplissage de la table de hachage).
* `source` ([String](/fr/reference/data-types/index)) — Texte décrivant la source de données du dictionnaire.
* `lifetime_min` ([UInt64](/fr/reference/data-types/index)) — Durée de vie minimale du dictionnaire en mémoire, au-delà de laquelle ClickHouse essaie de recharger le dictionnaire (si invalidate&#95;query est défini, alors uniquement s’il a changé). Exprimée en secondes.
* `lifetime_max` ([UInt64](/fr/reference/data-types/index)) — Durée de vie maximale du dictionnaire en mémoire, au-delà de laquelle ClickHouse essaie de recharger le dictionnaire (si invalidate&#95;query est défini, alors uniquement s’il a changé). Exprimée en secondes.
* `loading_start_time` ([DateTime](/fr/reference/data-types/index)) — Heure de début du chargement du dictionnaire.
* `last_successful_update_time` ([DateTime](/fr/reference/data-types/index)) — Heure de fin du chargement ou de la mise à jour du dictionnaire. Aide à surveiller certains problèmes liés aux sources de dictionnaire et à en analyser les causes.
* `error_count` ([UInt64](/fr/reference/data-types/index)) — Nombre d’erreurs depuis le dernier chargement réussi. Aide à surveiller certains problèmes liés aux sources de dictionnaire et à en analyser les causes.
* `loading_duration` ([Float32](/fr/reference/data-types/index)) — Durée du chargement d’un dictionnaire.
* `last_exception` ([String](/fr/reference/data-types/index)) — Texte de l’erreur survenue lors de la création ou du rechargement du dictionnaire, si le dictionnaire n’a pas pu être créé.
* `comment` ([String](/fr/reference/data-types/index)) — Texte du commentaire du dictionnaire.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

Configurez le dictionnaire :

```sql
CREATE DICTIONARY dictionary_with_comment
(
    id UInt64,
    value String
)
PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() TABLE 'source_table'))
LAYOUT(FLAT())
LIFETIME(MIN 0 MAX 1000)
COMMENT 'The temporary dictionary';
```

Assurez-vous que le dictionnaire est chargé.

```sql
SELECT * FROM system.dictionaries LIMIT 1 FORMAT Vertical;
```
