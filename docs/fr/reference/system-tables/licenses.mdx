<div id="description">
  ## Description
</div>

Contient les licences des bibliothèques tierces situées dans le répertoire [contrib](https://github.com/ClickHouse/ClickHouse/tree/master/contrib) du code source de ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `library_name` ([String](/fr/reference/data-types/index)) — Nom de la bibliothèque.
* `license_type` ([String](/fr/reference/data-types/index)) — Type de licence — par ex. Apache, MIT.
* `license_path` ([String](/fr/reference/data-types/index)) — Chemin du fichier contenant le texte de la licence.
* `license_text` ([String](/fr/reference/data-types/index)) — Texte de la licence.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT library_name, license_type, license_path FROM system.licenses LIMIT 15
```

```text
┌─library_name───────┬─license_type─┬─license_path────────────────────────┐
│ aws-c-common       │ Apache       │ /contrib/aws-c-common/LICENSE       │
│ base64             │ BSD 2-clause │ /contrib/aklomp-base64/LICENSE      │
│ brotli             │ MIT          │ /contrib/brotli/LICENSE             │
│ [...]              │ [...]        │ [...]                               │
└────────────────────┴──────────────┴─────────────────────────────────────┘
```
