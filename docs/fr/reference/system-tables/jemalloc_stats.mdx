<div id="description">
  ## Description
</div>

Renvoie les statistiques jemalloc sous la forme d’une seule ligne et d’une seule colonne. Équivalent à la commande SYSTEM JEMALLOC STATS.

  ## Colonnes
</div>

* `stats` ([String](/fr/reference/data-types/index)) — sortie des statistiques JEMalloc générée par malloc&#95;stats&#95;print.
