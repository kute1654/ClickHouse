<div id="description">
  ## Description
</div>

Contient des informations sur les colonnes de toutes les tables.

Vous pouvez utiliser cette table pour obtenir des informations similaires à celles de la requête [DESCRIBE TABLE](/fr/reference/statements/describe-table), mais pour plusieurs tables à la fois.

Les colonnes des [tables temporaires](/fr/reference/statements/create/table#temporary-tables) ne sont visibles dans `system.columns` que dans les sessions où elles ont été créées. Elles sont affichées avec le champ `database` vide.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `name` ([String](/fr/reference/data-types/index)) — Nom de la colonne.
* `type` ([String](/fr/reference/data-types/index)) — Type de la colonne.
* `position` ([UInt64](/fr/reference/data-types/index)) — Position ordinale d’une colonne dans une table, à partir de 1.
* `default_kind` ([String](/fr/reference/data-types/index)) — Type d’expression (DEFAULT, MATERIALIZED, ALIAS) de la valeur par défaut, ou chaîne vide s’il n’est pas défini.
* `default_expression` ([String](/fr/reference/data-types/index)) — Expression de la valeur par défaut, ou chaîne vide si elle n’est pas définie.
* `data_compressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données compressées, en octets.
* `data_uncompressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données décompressées, en octets.
* `marks_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des marks, en octets.
* `comment` ([String](/fr/reference/data-types/index)) — Commentaire sur la colonne, ou chaîne vide s’il n’est pas défini.
* `is_in_partition_key` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si la colonne figure dans l’expression de partition.
* `is_in_sorting_key` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si la colonne figure dans l’expression de clé de tri.
* `is_in_primary_key` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si la colonne figure dans l’expression de clé primaire.
* `is_in_sampling_key` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si la colonne figure dans l’expression de clé d’échantillonnage.
* `compression_codec` ([String](/fr/reference/data-types/index)) — Nom du codec de compression.
* `character_octet_length` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Longueur maximale, en octets, des données binaires, des données de caractères ou des données de texte et d’images. Dans ClickHouse, cela n’a de sens que pour le type de données FixedString. Sinon, la valeur NULL est renvoyée.
* `numeric_precision` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Précision des données numériques approximatives, des données numériques exactes, des données entières ou des données monétaires. Dans ClickHouse, il s’agit de la largeur en bits pour les types entiers et de la précision décimale pour les types Decimal. Sinon, la valeur NULL est renvoyée.
* `numeric_precision_radix` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Base du système de numération pour la précision des données numériques approximatives, des données numériques exactes, des données entières ou des données monétaires. Dans ClickHouse, c&#39;est 2 pour les types entiers et 10 pour les types Decimal. Sinon, la valeur NULL est renvoyée.
* `numeric_scale` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Échelle des données numériques approximatives, des données numériques exactes, des données entières ou des données monétaires. Dans ClickHouse, cela n’a de sens que pour les types Decimal. Sinon, la valeur NULL est renvoyée.
* `datetime_precision` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Précision décimale du type de données DateTime64. Pour les autres types de données, la valeur NULL est renvoyée.
* `serialization_hint` ([Nullable(String)](/fr/reference/data-types/index)) — Indication permettant à une colonne de choisir la sérialisation lors des insertions en fonction des statistiques.
* `statistics` ([String](/fr/reference/data-types/index)) — Types de statistiques créés pour cette colonne.

{/*AUTOGENERATED_END*/}
