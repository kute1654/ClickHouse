<div id="description">
  ## Description
</div>

Contient une liste de [comptes utilisateur](/fr/concepts/features/security/access-rights#user-account-management) configurés sur le serveur.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom d’utilisateur.
* `id` ([UUID](/fr/reference/data-types/index)) — ID de l’utilisateur.
* `storage` ([String](/fr/reference/data-types/index)) — Chemin vers l’emplacement de stockage des utilisateurs. Configuré dans le paramètre access&#95;control&#95;path.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/fr/reference/data-types/index)) — Affiche les types d’authentification. Il existe plusieurs façons d’identifier un utilisateur : sans mot de passe, avec un mot de passe en texte brut, avec un mot de passe haché en SHA256, avec un mot de passe haché en double SHA-1 ou avec un mot de passe haché en bcrypt.
* `auth_params` ([Array(String)](/fr/reference/data-types/index)) — Paramètres d’authentification au format JSON selon `auth_type`.
* `valid_until` ([Array(DateTime)](/fr/reference/data-types/index)) — Date et heure d’expiration des informations d’identification de l’utilisateur.
* `host_ip` ([Array(String)](/fr/reference/data-types/index)) — Adresses IP des hôtes autorisés à se connecter au serveur ClickHouse.
* `host_names` ([Array(String)](/fr/reference/data-types/index)) — Noms des hôtes autorisés à se connecter au serveur ClickHouse.
* `host_names_regexp` ([Array(String)](/fr/reference/data-types/index)) — Expression régulière correspondant aux noms d’hôte autorisés à se connecter au serveur ClickHouse.
* `host_names_like` ([Array(String)](/fr/reference/data-types/index)) — Noms des hôtes autorisés à se connecter au serveur ClickHouse, définis à l’aide du prédicat LIKE.
* `default_roles_all` ([UInt8](/fr/reference/data-types/index)) — Indique que tous les rôles accordés sont définis par défaut pour l’utilisateur.
* `default_roles_list` ([Array(String)](/fr/reference/data-types/index)) — Liste des rôles accordés par défaut.
* `default_roles_except` ([Array(String)](/fr/reference/data-types/index)) — Tous les rôles accordés sont définis par défaut, à l’exception de ceux figurant dans la liste.
* `grantees_any` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si un utilisateur disposant d’une option d’octroi peut l’accorder à n’importe qui.
* `grantees_list` ([Array(String)](/fr/reference/data-types/index)) — Liste des utilisateurs ou rôles auxquels cet utilisateur est autorisé à accorder des options.
* `grantees_except` ([Array(String)](/fr/reference/data-types/index)) — Liste des utilisateurs ou rôles auxquels cet utilisateur n’est pas autorisé à accorder des options.
* `default_database` ([String](/fr/reference/data-types/index)) — Nom de la base de données par défaut pour cet utilisateur.
