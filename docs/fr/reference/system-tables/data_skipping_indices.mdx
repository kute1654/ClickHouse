<div id="description">
  ## Description
</div>

Contient des informations sur les index de saut de données présents dans l’ensemble des tables.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `name` ([String](/fr/reference/data-types/index)) — Nom de l’index.
* `type` ([String](/fr/reference/data-types/index)) — Type de l’index.
* `type_full` ([String](/fr/reference/data-types/index)) — Expression du type d’index issue de l’instruction CREATE.
* `expr` ([String](/fr/reference/data-types/index)) — Expression utilisée pour calculer l’index.
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/fr/reference/data-types/index)) — Indique si l’index a été créé implicitement (via add&#95;minmax&#95;index&#95;for&#95;numeric&#95;columns ou un mécanisme similaire)
* `granularity` ([UInt64](/fr/reference/data-types/index)) — Nombre de granules dans le bloc.
* `data_compressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données compressées, en octets.
* `data_uncompressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données décompressées, en octets.
* `marks_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des marks, en octets.

{/*AUTOGENERATED_END*/}
