<div id="description">
  ## Description
</div>

Contient des informations sur toutes les politiques de masquage définies dans le système.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/string)) — Nom d’une politique de masquage. Le format du nom complet est `short_name ON database.table`.
* `short_name` ([String](/fr/reference/data-types/string)) — Nom court d’une politique de masquage. Par exemple, si le nom complet est `mask_email ON mydb.mytable`, le nom court est `mask_email`.
* `database` ([String](/fr/reference/data-types/string)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/string)) — Nom de la table.
* `id` ([UUID](/fr/reference/data-types/uuid)) — ID de la politique de masquage.
* `storage` ([String](/fr/reference/data-types/string)) — Nom du répertoire où la politique de masquage est stockée.
* `update_assignments` ([Nullable(String)](/fr/reference/data-types/nullable)) — Affectations dans UPDATE qui définissent comment les données doivent être masquées. Par exemple : `email = '***masked***', phone = '***-***-****'`.
* `where_condition` ([Nullable(String)](/fr/reference/data-types/nullable)) — Condition WHERE facultative qui indique quand le masquage doit être appliqué.
* `priority` ([Int64](/fr/reference/data-types/int-uint)) — Priorité d’application de plusieurs politiques de masquage. Les politiques ayant la priorité la plus élevée sont appliquées en premier. La valeur par défaut est 0.
* `apply_to_all` ([UInt8](/fr/reference/data-types/int-uint)) — Indique si la politique de masquage s’applique à tous les rôles et/ou utilisateurs. 1 si true, 0 sinon.
* `apply_to_list` ([Array(String)](/fr/reference/data-types/array)) — Liste des rôles et/ou utilisateurs auxquels la politique de masquage s’applique.
* `apply_to_except` ([Array(String)](/fr/reference/data-types/array)) — La politique de masquage s’applique à tous les rôles et/ou utilisateurs, sauf à ceux figurant dans la liste. Renseigné uniquement lorsque `apply_to_all` vaut 1.

{/*AUTOGENERATED_END*/}
