<div id="description">
  ## Description
</div>

Affiche le contenu du [cache des conditions de requête](/fr/concepts/features/performance/caches/query-condition-cache).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `key_hash` ([UInt128](/fr/reference/data-types/index)) — Hachage de (table&#95;uuid, part&#95;name, condition&#95;hash).
* `entry_size` ([UInt64](/fr/reference/data-types/index)) — Taille de l’entrée en octets.
* `matching_marks` ([String](/fr/reference/data-types/index)) — Marks correspondantes.

{/*AUTOGENERATED_END*/}
