<div id="purpose">
  ## Description
</div>

La table `system.delta_lake_metadata_log` enregistre les événements d&#39;accès aux métadonnées et d&#39;analyse des tables Delta Lake lues par ClickHouse. Elle fournit des informations détaillées sur chaque fichier de métadonnées, ce qui est utile pour le débogage, l&#39;audit et pour comprendre l&#39;évolution de la structure des tables Delta.

Cette table consigne chaque fichier de métadonnées lu depuis des tables Delta Lake. Elle aide les utilisateurs à retracer la façon dont ClickHouse interprète les métadonnées des tables Delta et à diagnostiquer les problèmes liés à l&#39;évolution du schéma, à la résolution des instantanés ou à la planification des requêtes.

<Note>
  Cette table est principalement destinée au débogage.
</Note>

<div id="controlling-log-verbosity">
  ### Contrôle de la verbosité des logs
</div>

Vous pouvez définir quels événements de métadonnées sont consignés à l’aide du paramètre [`delta_lake_log_metadata`](/fr/reference/settings/session-settings#delta_lake_log_metadata).

Pour consigner toutes les métadonnées utilisées dans la requête en cours :

```sql
SELECT * FROM my_delta_table SETTINGS delta_lake_log_metadata = 1;

SYSTEM FLUSH LOGS delta_lake_metadata_log;

SELECT *
FROM system.delta_lake_metadata_log
WHERE query_id = '{previous_query_id}';
```
