<div id="description">
  ## Description
</div>

Contient des informations sur les [ressources](/fr/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage) présentes sur le serveur local. La table contient une ligne pour chaque ressource.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Le nom de la ressource.
* `read_disks` ([Array(String)](/fr/reference/data-types/index)) — La liste des noms des disques qui utilisent cette ressource pour les opérations de lecture.
* `write_disks` ([Array(String)](/fr/reference/data-types/index)) — La liste des noms des disques qui utilisent cette ressource pour les opérations d&#39;écriture.
* `unit` ([String](/fr/reference/data-types/index)) — Unité de ressource utilisée pour mesurer les coûts.
* `create_query` ([String](/fr/reference/data-types/index)) — Requête CREATE de la ressource.

{/*AUTOGENERATED_END*/}

Exemple :

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
Row 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

Row 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```
