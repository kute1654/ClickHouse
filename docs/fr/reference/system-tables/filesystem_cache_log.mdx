Contient l&#39;historique de tous les événements survenus dans le cache du système de fichiers pour les objets d&#39;un système de fichiers distant.

Cette table peut être tronquée ou supprimée à tout moment sans risque.

Colonnes :

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Horodatage de l’événement
* `query_id` ([String](/fr/reference/data-types/string)) — Identifiant de la requête
* `source_file_path` ([String](/fr/reference/data-types/string)) — Chemin du segment de fichier dans le système de fichiers
* `file_segment_range` ([Tuple(UInt64, UInt64)](/fr/reference/data-types/tuple)) — Plage du segment de fichier
* `total_requested_range` ([Tuple(UInt64, UInt64)](/fr/reference/data-types/tuple)) — Plage de lecture totale demandée
* `key` ([String](/fr/reference/data-types/string)) — Clé du segment de fichier
* `offset` ([UInt64](/fr/reference/data-types/int-uint)) — Décalage du segment de fichier
* `size` ([UInt64](/fr/reference/data-types/int-uint)) — Taille de lecture
* `read_type` ([String](/fr/reference/data-types/string)) — Type de lecture : READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/fr/reference/data-types/int-uint)) — Indique si une lecture depuis le cache a été tentée
* `read_buffer_id` ([String](/fr/reference/data-types/string)) — Identifiant interne du tampon de lecture
* `user_id` ([String](/fr/reference/data-types/string)) — Identifiant de l’utilisateur ayant créé le segment de fichier

{/*AUTOGENERATED_END*/}
