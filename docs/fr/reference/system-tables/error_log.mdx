<div id="description">
  ## Description
</div>

Contient l’historique des valeurs d’erreur de la table `system.errors`, périodiquement enregistrées sur disque.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui exécute la requête.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’événement.
* `code` ([Int32](/fr/reference/data-types/int-uint)) — Code d’erreur.
* `error` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de l’erreur.
* `value` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre d’erreurs survenues pendant l’intervalle de temps.
* `remote` ([UInt8](/fr/reference/data-types/int-uint)) — Exception distante (c.-à-d. reçue lors de l’une des requêtes distribuées).
* `last_error_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure à laquelle la dernière erreur s’est produite.
* `last_error_message` ([String](/fr/reference/data-types/string)) — Message de la dernière erreur.
* `last_error_query_id` ([String](/fr/reference/data-types/string)) — ID d’une requête ayant provoqué la dernière erreur (si disponible).
* `last_error_trace` ([Array(UInt64)](/fr/reference/data-types/array)) — Trace de pile représentant une liste d’adresses mémoire physiques où sont stockées les méthodes appelées.

{/*AUTOGENERATED_END*/}
