<div id="description">
  ## Description
</div>

Affiche le contenu du [cache de requêtes](/fr/concepts/features/performance/caches/query-cache).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/fr/reference/data-types/index)) — Chaîne de la requête.
* `query_id` ([String](/fr/reference/data-types/index)) — Identifiant de la requête.
* `is_subquery` ([UInt8](/fr/reference/data-types/index)) — Indique si l&#39;entrée du cache de requêtes correspond au résultat d&#39;une sous-requête.
* `result_size` ([UInt64](/fr/reference/data-types/index)) — Taille de l&#39;entrée du cache de requêtes.
* `tag` ([LowCardinality(String)](/fr/reference/data-types/index)) — Tag de l&#39;entrée du cache de requêtes.
* `stale` ([UInt8](/fr/reference/data-types/index)) — Indique si l&#39;entrée du cache de requêtes est obsolète.
* `shared` ([UInt8](/fr/reference/data-types/index)) — Indique si l&#39;entrée du cache de requêtes est partagée entre plusieurs utilisateurs.
* `compressed` ([UInt8](/fr/reference/data-types/index)) — Indique si l&#39;entrée du cache de requêtes est compressée.
* `expires_at` ([DateTime](/fr/reference/data-types/index)) — Moment où l&#39;entrée du cache de requêtes devient obsolète.
* `key_hash` ([UInt64](/fr/reference/data-types/index)) — Hachage de la chaîne de requête, utilisé comme clé pour trouver les entrées du cache de requêtes.

{/*AUTOGENERATED_END*/}
