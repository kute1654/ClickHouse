<div id="description">
  ## Description
</div>

Utilisation du quota par l’utilisateur actuel : quantité utilisée et quantité encore disponible.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/fr/reference/data-types/index)) — Nom du quota.
* `quota_key` ([String](/fr/reference/data-types/index)) — Valeur de la clé.
* `start_time` ([Nullable(DateTime)](/fr/reference/data-types/index)) — Heure de début du calcul de la consommation des ressources.
* `end_time` ([Nullable(DateTime)](/fr/reference/data-types/index)) — Heure de fin du calcul de la consommation des ressources.
* `duration` ([Nullable(UInt32)](/fr/reference/data-types/index)) — Durée de l’intervalle de temps pour le calcul de la consommation des ressources, en secondes.
* `queries` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre actuel de requêtes exécutées.
* `max_queries` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé de requêtes, tous types confondus, pouvant être exécutées.
* `query_selects` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre actuel de requêtes SELECT exécutées.
* `max_query_selects` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé de requêtes SELECT pouvant être exécutées.
* `query_inserts` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre actuel de requêtes INSERT exécutées.
* `max_query_inserts` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé de requêtes INSERT pouvant être exécutées.
* `errors` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre actuel de requêtes ayant généré une erreur.
* `max_errors` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé de requêtes ayant généré une erreur pendant la période spécifiée.
* `result_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total actuel de lignes dans le jeu de résultats de toutes les requêtes pendant la période actuelle.
* `max_result_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total maximal autorisé de lignes dans le jeu de résultats de toutes les requêtes pendant la période spécifiée.
* `result_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total actuel d’octets dans le jeu de résultats de toutes les requêtes pendant la période actuelle.
* `max_result_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total maximal autorisé d’octets dans le jeu de résultats de toutes les requêtes pendant la période spécifiée.
* `read_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total actuel de lignes lues lors de l’exécution de toutes les requêtes pendant la période actuelle.
* `max_read_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé de lignes à lire lors de l’exécution de toutes les requêtes pendant la période spécifiée.
* `read_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total actuel d’octets lus lors de l’exécution de toutes les requêtes pendant la période actuelle.
* `max_read_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé d’octets à lire lors de l’exécution de toutes les requêtes pendant la période spécifiée.
* `execution_time` ([Nullable(Float64)](/fr/reference/data-types/index)) — Temps total actuel (en nanosecondes) consacré à l’exécution des requêtes pendant la période actuelle
* `max_execution_time` ([Nullable(Float64)](/fr/reference/data-types/index)) — Temps maximal autorisé (en nanosecondes) pour l’exécution de toutes les requêtes pendant la période spécifiée
* `written_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre total actuel d’octets écrits lors de l’exécution de toutes les requêtes pendant la période actuelle.
* `max_written_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé d’octets à écrire lors de l’exécution de toutes les requêtes pendant la période spécifiée.
* `failed_sequential_authentications` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre actuel d’échecs d’authentification consécutifs pendant la période actuelle.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé d’échecs d’authentification consécutifs pendant la période spécifiée.
* `queries_per_normalized_hash` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal actuel d’exécutions d’une même requête normalisée pendant la période actuelle.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Nombre maximal autorisé d’exécutions d’une même requête normalisée pendant la période spécifiée.
