<div id="description">
  ## Description
</div>

Contient des informations sur les paramètres [graphite&#95;rollup](/fr/reference/settings/server-settings/settings#graphite), utilisés dans les tables reposant sur les moteurs [*GraphiteMergeTree](/fr/reference/engines/table-engines/mergetree-family/graphitemergetree).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/fr/reference/data-types/index)) — nom du paramètre graphite&#95;rollup.
* `rule_type` ([String](/fr/reference/data-types/index)) — Le type de règle. Valeurs possibles : RuleTypeAll = 0 - par défaut, avec regex, compatible avec l’ancien schéma ; RuleTypePlain = 1 - métriques simples, avec regex, compatible avec l’ancien schéma ; RuleTypeTagged = 2 - métriques avec tags, avec regex, compatible avec l’ancien schéma ; RuleTypeTagList = 3 - métriques avec tags, avec regex (converti en RuleTypeTagged à partir d’une chaîne comme &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](/fr/reference/data-types/index)) — Motif pour le nom de la métrique.
* `function` ([String](/fr/reference/data-types/index)) — Le nom de la fonction d’agrégation.
* `age` ([UInt64](/fr/reference/data-types/index)) — L’âge minimal des données en secondes.
* `precision` ([UInt64](/fr/reference/data-types/index)) — Le niveau de précision utilisé pour définir l’âge des données en secondes.
* `priority` ([UInt16](/fr/reference/data-types/index)) — Priorité du motif.
* `is_default` ([UInt8](/fr/reference/data-types/index)) — Indique si le motif est celui par défaut.
* `Tables.database` ([Array(String)](/fr/reference/data-types/index)) — Tableau des noms des tables de la base de données qui utilisent le paramètre `config_name`.
* `Tables.table` ([Array(String)](/fr/reference/data-types/index)) — Tableau des noms de tables qui utilisent le paramètre `config_name`.
