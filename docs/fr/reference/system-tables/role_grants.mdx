<div id="description">
  ## Description
</div>

Contient les attributions de rôles aux utilisateurs et aux rôles. Pour ajouter des entrées à cette table, utilisez `GRANT role TO user`.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `user_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom d’utilisateur.
* `role_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom du rôle.
* `granted_role_name` ([String](/fr/reference/data-types/index)) — Nom du rôle accordé au rôle `role_name`. Pour accorder un rôle à un autre, utilisez `GRANT role1 TO role2`.
* `granted_role_id` ([UUID](/fr/reference/data-types/index)) — ID du rôle.
* `granted_role_is_default` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si `granted_role` est un rôle par défaut. Valeurs possibles : • 1 — `granted_role` est un rôle par défaut, • 0 — `granted_role` n’est pas un rôle par défaut.
* `with_admin_option` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si `granted_role` dispose du privilège `ADMIN OPTION`. Valeurs possibles : • 1 — Le rôle dispose du privilège `ADMIN OPTION`. • 0 — Le rôle ne dispose pas du privilège `ADMIN OPTION`.

{/*AUTOGENERATED_END*/}
