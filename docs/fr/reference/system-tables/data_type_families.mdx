<div id="description">
  ## Description
</div>

Contient des informations sur les [types de données](/fr/reference/data-types/index).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du type de données.
* `case_insensitive` ([UInt8](/fr/reference/data-types/index)) — Propriété indiquant si un nom de type de données peut être utilisé dans une requête sans tenir compte de la casse. Par exemple, `Date` et `date` sont tous deux valides.
* `alias_to` ([String](/fr/reference/data-types/index)) — Nom du type de données dont `name` est un alias.
* `description` ([String](/fr/reference/data-types/index)) — Description générale du type de données.
* `syntax` ([String](/fr/reference/data-types/index)) — Façon dont le type de données s’écrit dans une requête.
* `examples` ([String](/fr/reference/data-types/index)) — Exemples d’utilisation.
* `introduced_in` ([String](/fr/reference/data-types/index)) — Version de ClickHouse dans laquelle le type de données a été introduit pour la première fois, au format majeur.mineur.
* `related` ([Array(String)](/fr/reference/data-types/index)) — Noms des types de données associés.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT name, case_insensitive, alias_to FROM system.data_type_families WHERE alias_to = 'String'
```
