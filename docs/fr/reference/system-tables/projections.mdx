<div id="description">
  ## Description
</div>

Contient des informations sur les projections existantes de toutes les tables.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `name` ([String](/fr/reference/data-types/index)) — Nom de la projection.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](/fr/reference/data-types/index)) — Type de projection.
* `sorting_key` ([Array(String)](/fr/reference/data-types/index)) — Clé de tri de la projection.
* `query` ([String](/fr/reference/data-types/index)) — Requête de la projection.
* `settings` ([Map(String, String)](/fr/reference/data-types/index)) — Paramètres de la projection.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       landing
name:        improved_sorting_key
type:        Normal
sorting_key: ['user_id','date']
query:       SELECT * ORDER BY user_id, date
settings:     {}

Row 2:
──────
database:    default
table:       landing
name:        agg_no_key
type:        Aggregate
sorting_key: []
query:       SELECT count()
settings:     {}
```
