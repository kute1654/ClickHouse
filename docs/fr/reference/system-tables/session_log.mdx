<div id="description">
  ## Description
</div>

Contient des informations sur tous les événements de connexion et de déconnexion réussis ou ayant échoué.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui exécute la requête.
* `type` ([Enum8(&#39;LoginFailure&#39; = 0, &#39;LoginSuccess&#39; = 1, &#39;Logout&#39; = 2)](/fr/reference/data-types/enum)) — Résultat de connexion/déconnexion. Valeurs possibles : LoginFailure — Erreur de connexion. LoginSuccess — Connexion réussie. Logout — Déconnexion du système.
* `auth_id` ([UUID](/fr/reference/data-types/uuid)) — ID d’authentification, c’est-à-dire un UUID généré automatiquement à chaque connexion de l’utilisateur.
* `session_id` ([String](/fr/reference/data-types/string)) — ID de session transmis par le client via l’interface HTTP.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de connexion/déconnexion.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de connexion/déconnexion.
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de début de la connexion/déconnexion avec une précision à la microseconde.
* `user` ([Nullable(String)](/fr/reference/data-types/nullable)) — Nom d’utilisateur.
* `auth_type` ([Nullable(Enum8(&#39;NO&#95;PASSWORD&#39; = 0, &#39;PLAINTEXT&#95;PASSWORD&#39; = 1, &#39;SHA256&#95;PASSWORD&#39; = 2, &#39;DOUBLE&#95;SHA1&#95;PASSWORD&#39; = 3, &#39;LDAP&#39; = 4, &#39;KERBEROS&#39; = 5, &#39;SSL&#95;CERTIFICATE&#39; = 6, &#39;BCRYPT&#95;PASSWORD&#39; = 7, &#39;SSH&#95;KEY&#39; = 8, &#39;HTTP&#39; = 9, &#39;JWT&#39; = 10, &#39;SCRAM&#95;SHA256&#95;PASSWORD&#39; = 11, &#39;NO&#95;AUTHENTICATION&#39; = 12))](/fr/reference/data-types/nullable)) — Type d’authentification.
* `profiles` ([Array(LowCardinality(String))](/fr/reference/data-types/array)) — Liste des profils définis pour tous les rôles et/ou utilisateurs.
* `roles` ([Array(LowCardinality(String))](/fr/reference/data-types/array)) — Liste des rôles auxquels le profil est appliqué.
* `settings` ([Array(Tuple(LowCardinality(String), String))](/fr/reference/data-types/array)) — Paramètres modifiés lors de la connexion/déconnexion du client.
* `client_address` ([IPv6](/fr/reference/data-types/ipv6)) — Adresse IP utilisée pour la connexion/déconnexion.
* `client_port` ([UInt16](/fr/reference/data-types/int-uint)) — Port client utilisé pour la connexion/déconnexion.
* `interface` ([Enum8(&#39;TCP&#39; = 1, &#39;HTTP&#39; = 2, &#39;gRPC&#39; = 3, &#39;MySQL&#39; = 4, &#39;PostgreSQL&#39; = 5, &#39;Local&#39; = 6, &#39;TCP&#95;Interserver&#39; = 7, &#39;Prometheus&#39; = 8, &#39;Background&#39; = 9)](/fr/reference/data-types/enum)) — Interface depuis laquelle la connexion a été initiée.
* `client_hostname` ([String](/fr/reference/data-types/string)) — Nom d’hôte de la machine cliente sur laquelle `clickhouse-client` ou un autre client TCP est exécuté.
* `client_name` ([String](/fr/reference/data-types/string)) — Nom de `clickhouse-client` ou d’un autre client TCP.
* `client_revision` ([UInt32](/fr/reference/data-types/int-uint)) — Révision de `clickhouse-client` ou d’un autre client TCP.
* `client_version_major` ([UInt32](/fr/reference/data-types/int-uint)) — Version majeure de `clickhouse-client` ou d’un autre client TCP.
* `client_version_minor` ([UInt32](/fr/reference/data-types/int-uint)) — Version mineure de `clickhouse-client` ou d’un autre client TCP.
* `client_version_patch` ([UInt32](/fr/reference/data-types/int-uint)) — Composant de correctif de la version de `clickhouse-client` ou d’un autre client TCP.
* `failure_reason` ([String](/fr/reference/data-types/string)) — Message d’exception indiquant la raison de l’échec de la connexion/déconnexion.
* `certificate_subjects` ([Array(LowCardinality(String))](/fr/reference/data-types/array)) — La liste des sujets (Common Name et Subject Alternative Names) du certificat client TLS présenté lors de la connexion, sous la forme &#39;CN:...&#39; / &#39;SAN:...&#39;. Vide si aucun certificat n’a été présenté.
* `certificate_serial` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Numéro de série du certificat client TLS. Vide si aucun certificat n’a été présenté.
* `certificate_issuer` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Autorité émettrice du certificat client TLS. Vide si aucun certificat n’a été présenté.
* `certificate_not_before` ([Nullable(DateTime64)](/fr/reference/data-types/nullable)) — Date et heure à partir desquelles le certificat client TLS est valide. NULL si aucun certificat n’a été présenté.
* `certificate_not_after` ([Nullable(DateTime64)](/fr/reference/data-types/nullable)) — Date et heure après lesquelles le certificat client TLS expire. NULL si aucun certificat n’a été présenté.

{/*AUTOGENERATED_END*/}
