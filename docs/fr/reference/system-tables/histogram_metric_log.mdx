<div id="description">
  ## Description
</div>

Historique de `system.histogram_metrics`. Instantané pris tous les `collect_interval_milliseconds`, puis écrit sur disque.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui exécute la requête.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’événement.
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’événement avec une précision à la microseconde.
* `metric` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de la métrique.
* `labels` ([Map(LowCardinality(String), LowCardinality(String))](/fr/reference/data-types/map)) — Labels de métrique.
* `histogram` ([Map(Float64, UInt64)](/fr/reference/data-types/map)) — Histogramme cumulatif : associe la borne supérieure d’un intervalle au nombre d’observations ≤ à cette borne ; inclut +inf comme intervalle final.
* `count` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre total d’observations, correspond à histogram[+inf].
* `sum` ([Float64](/fr/reference/data-types/float)) — Somme de toutes les valeurs observées.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT event_time, metric, labels, histogram
FROM system.histogram_metric_log
WHERE metric = 'keeper_response_time_ms'
ORDER BY event_time DESC
LIMIT 1
FORMAT Vertical;
```
