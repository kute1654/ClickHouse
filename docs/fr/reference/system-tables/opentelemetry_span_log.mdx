<div id="description">
  ## Description
</div>

Contient des informations sur les [spans de trace](https://opentracing.io/docs/overview/spans/) relatifs aux requêtes exécutées.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Le nom d’hôte où ce span a été capturé.
* `trace_id` ([UUID](/fr/reference/data-types/uuid)) — ID de la trace de la requête exécutée.
* `span_id` ([UInt64](/fr/reference/data-types/int-uint)) — ID du span de trace.
* `parent_span_id` ([UInt64](/fr/reference/data-types/int-uint)) — ID du span de trace parent.
* `operation_name` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Le nom de l’opération.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/fr/reference/data-types/enum)) — Le SpanKind du span. INTERNAL — Indique que le span représente une opération interne au sein d’une application. SERVER — Indique que le span couvre le traitement côté serveur d’un RPC synchrone ou d’une autre requête distante. CLIENT — Indique que le span décrit une requête vers un service distant. PRODUCER — Indique que le span décrit les initiateurs d’une requête asynchrone. Ce span parent se terminera souvent avant le span enfant CONSUMER correspondant, éventuellement même avant que le span enfant ne commence. CONSUMER - Indique que le span décrit un enfant d’une requête PRODUCER asynchrone.
* `start_time_us` ([UInt64](/fr/reference/data-types/int-uint)) — L’heure de début du span de trace (en microsecondes).
* `finish_time_us` ([UInt64](/fr/reference/data-types/int-uint)) — L’heure de fin du span de trace (en microsecondes).
* `finish_date` ([Date](/fr/reference/data-types/date)) — La date de fin du span de trace.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/fr/reference/data-types/enum)) — Le code de statut du span.
* `status_message` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Message d’erreur.
* `attribute` ([Map(LowCardinality(String), String)](/fr/reference/data-types/map)) — Attributs associés au span de trace. Ils sont renseignés conformément aux recommandations de la norme OpenTelemetry.

**Alias :**

* `attribute.names` — alias de `mapKeys(attribute)`.
* `attribute.values` — alias de `mapValues(attribute)`.
