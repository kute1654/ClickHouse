<div id="description">
  ## Description
</div>

Contient les propriétés des profils de paramètres configurés.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du profil de paramètres.
* `id` ([UUID](/fr/reference/data-types/index)) — ID du profil de paramètres.
* `storage` ([String](/fr/reference/data-types/index)) — Chemin d’accès au stockage des profils de paramètres. Configuré dans le paramètre `access_control_path`.
* `num_elements` ([UInt64](/fr/reference/data-types/index)) — Nombre d’éléments de ce profil dans la table `system.settings_profile_elements`.
* `apply_to_all` ([UInt8](/fr/reference/data-types/index)) — Indique que le profil de paramètres s’applique à tous les rôles et/ou utilisateurs.
* `apply_to_list` ([Array(String)](/fr/reference/data-types/index)) — Liste des rôles et/ou utilisateurs auxquels le profil de paramètres s’applique.
* `apply_to_except` ([Array(String)](/fr/reference/data-types/index)) — Le profil de paramètres s’applique à tous les rôles et/ou utilisateurs, à l’exception de ceux figurant dans la liste.

{/*AUTOGENERATED_END*/}
