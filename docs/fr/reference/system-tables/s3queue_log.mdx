Contient des entrées de journal sur les fichiers traités par le moteur S3Queue.

Cette table peut être tronquée ou supprimée sans risque à tout moment.

Colonnes :

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte
* `event_date` ([Date](/fr/reference/data-types/date)) — Date d’écriture de cette ligne de log
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure d’écriture de cette ligne de log
* `database` ([String](/fr/reference/data-types/string)) — Nom de la base de données où se trouve la table de queue (`S3Queue` ou `AzureQueue`).
* `table` ([String](/fr/reference/data-types/string)) — Nom de la table de queue (`S3Queue` ou `AzureQueue`).
* `uuid` ([String](/fr/reference/data-types/string)) — UUID de la table de queue (`S3Queue` ou `AzureQueue`).
* `file_name` ([String](/fr/reference/data-types/string)) — Nom du fichier en cours de traitement.
* `rows_processed` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre de lignes traitées.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/fr/reference/data-types/enum)) — Statut du fichier en cours de traitement.
* `processing_start_time` ([Nullable(DateTime)](/fr/reference/data-types/nullable)) — Heure de début du traitement du fichier.
* `processing_end_time` ([Nullable(DateTime)](/fr/reference/data-types/nullable)) — Heure de fin du traitement du fichier.
* `exception` ([String](/fr/reference/data-types/string)) — Message d’exception, le cas échéant.
* `commit_id` ([UInt64](/fr/reference/data-types/int-uint)) — ID de la transaction dans laquelle ce fichier a été validé.
* `commit_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de validation du fichier dans Keeper (qu’il soit en échec ou traité).
* `transaction_start_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de début de l’ensemble de la transaction de traitement.
* `get_object_time_ms` ([UInt64](/fr/reference/data-types/int-uint)) — Temps nécessaire pour trouver l’objet dans le stockage objet.
