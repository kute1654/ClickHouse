<div id="description">
  ## Description
</div>

Décrit le contenu du profil de paramètres :

* Contraintes.
* Rôles et utilisateurs auxquels le paramètre s’applique.
* Profils de paramètres parents.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom du profil de paramètres.
* `user_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom d’utilisateur.
* `role_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom du rôle.
* `index` ([UInt64](/fr/reference/data-types/index)) — Numéro séquentiel de l’élément du profil de paramètres.
* `setting_name` ([Nullable(String)](/fr/reference/data-types/index)) — Nom du paramètre.
* `value` ([Nullable(String)](/fr/reference/data-types/index)) — Valeur du paramètre.
* `min` ([Nullable(String)](/fr/reference/data-types/index)) — Valeur minimale du paramètre. NULL s’il n’est pas défini.
* `max` ([Nullable(String)](/fr/reference/data-types/index)) — Valeur maximale du paramètre. NULL si elle n’est pas définie.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/fr/reference/data-types/index)) — Propriété indiquant si un paramètre peut être modifié ou non.
* `inherit_profile` ([Nullable(String)](/fr/reference/data-types/index)) — Profil parent de ce profil de paramètres. NULL s’il n’est pas défini. Le profil de paramètres hérite de toutes les valeurs et contraintes des paramètres (min, max, readonly) de ses profils parents.

{/*AUTOGENERATED_END*/}
