<div id="description">
  ## Description
</div>

Contient l’historique des valeurs de `system.asynchronous_metrics`, enregistrées une fois par intervalle de temps (une seconde par défaut). Activée par défaut.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui exécute la requête.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’événement.
* `metric` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de la métrique.
* `value` ([Float64](/fr/reference/data-types/float)) — Valeur de la métrique.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.asynchronous_metric_log LIMIT 3 \G
```

```text
value:      0
```
