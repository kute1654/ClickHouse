<div id="description">
  ## Description
</div>

Informations sur les [rôles](/fr/concepts/features/security/access-rights#role-management) configurés.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du rôle.
* `id` ([UUID](/fr/reference/data-types/index)) — Identifiant du rôle.
* `storage` ([String](/fr/reference/data-types/index)) — Chemin d’accès au stockage des rôles. Configuré via le paramètre `access_control_path`.
