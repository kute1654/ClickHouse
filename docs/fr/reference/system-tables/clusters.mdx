<div id="description">
  ## Description
</div>

Contient des informations sur les clusters disponibles dans le fichier de configuration et sur les serveurs qui les composent.

<div id="Columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `cluster` ([String](/fr/reference/data-types/index)) — Le nom du cluster.
* `shard_num` ([UInt32](/fr/reference/data-types/index)) — Le numéro du shard dans le cluster, à partir de 1.
* `shard_name` ([String](/fr/reference/data-types/index)) — Le nom du shard dans le cluster.
* `shard_weight` ([UInt32](/fr/reference/data-types/index)) — Le poids relatif du shard lors de l’écriture des données.
* `internal_replication` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si cet hôte fait partie d’un ensemble capable de répliquer les données de manière autonome.
* `replica_num` ([UInt32](/fr/reference/data-types/index)) — Le numéro de la réplique dans le shard, à partir de 1.
* `host_name` ([String](/fr/reference/data-types/index)) — Le nom d’hôte, tel qu’il est spécifié dans la configuration.
* `host_address` ([String](/fr/reference/data-types/index)) — L’adresse IP de l’hôte obtenue via DNS.
* `port` ([UInt16](/fr/reference/data-types/index)) — Le port à utiliser pour se connecter au serveur.
* `is_local` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si l’hôte est local.
* `user` ([String](/fr/reference/data-types/index)) — Le nom de l’utilisateur à utiliser pour se connecter au serveur.
* `default_database` ([String](/fr/reference/data-types/index)) — Le nom de la base de données par défaut.
* `errors_count` ([UInt32](/fr/reference/data-types/index)) — Le nombre de fois où cet hôte n’a pas réussi à joindre la réplique.
* `slowdowns_count` ([UInt32](/fr/reference/data-types/index)) — Le nombre de ralentissements ayant entraîné un changement de réplique lors de l’établissement d’une connexion avec des requêtes hedged.
* `estimated_recovery_time` ([UInt32](/fr/reference/data-types/index)) — Le nombre de secondes restantes avant que le compteur d’erreurs de la réplique soit remis à zéro et qu’elle soit de nouveau considérée comme normale.
* `database_shard_name` ([String](/fr/reference/data-types/index)) — Le nom du shard de la base de données `Replicated` (pour les clusters qui appartiennent à une base de données `Replicated`).
* `database_replica_name` ([String](/fr/reference/data-types/index)) — Le nom de la réplique de la base de données `Replicated` (pour les clusters qui appartiennent à une base de données `Replicated`).
* `is_shared_catalog_cluster` ([UInt8](/fr/reference/data-types/index)) — Bool indiquant si le cluster appartient au catalogue partagé.
* `is_active` ([Nullable(UInt8)](/fr/reference/data-types/index)) — L’état de la réplique de base de données Replicated (pour les clusters qui appartiennent à une base de données Replicated) : 1 signifie &#39;la réplique est en ligne&#39;, 0 signifie &#39;la réplique est hors ligne&#39;, NULL signifie &#39;inconnu&#39;.
* `unsynced_after_recovery` ([Nullable(UInt8)](/fr/reference/data-types/index)) — Indique si une réplique de base de données Replicated présente un retard de réplication supérieur à max&#95;replication&#95;lag&#95;to&#95;enqueue après la création ou la récupération de la réplique.
* `replication_lag` ([Nullable(UInt32)](/fr/reference/data-types/index)) — Le retard de réplication de la réplique de base de données `Replicated` (pour les clusters qui appartiennent à une base de données Replicated).
* `recovery_time` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le temps de récupération de la réplique de base de données `Replicated` (pour les clusters qui appartiennent à une base de données Replicated), en millisecondes.

{/*AUTOGENERATED_END*/}
