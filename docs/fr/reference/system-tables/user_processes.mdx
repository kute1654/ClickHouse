<div id="description">
  ## Description
</div>

Cette table système peut être utilisée pour obtenir un aperçu de l’utilisation de la mémoire et des ProfileEvents des utilisateurs.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `user` ([String](/fr/reference/data-types/index)) — Nom de l’utilisateur.
* `memory_usage` ([Int64](/fr/reference/data-types/index)) — Somme de la RAM utilisée par tous les processus de l’utilisateur. Elle peut ne pas inclure certains types de mémoire dédiée. Voir le paramètre max&#95;memory&#95;usage.
* `peak_memory_usage` ([Int64](/fr/reference/data-types/index)) — Pic d’utilisation de la mémoire de l’utilisateur. Il peut être réinitialisé lorsqu’aucune requête n’est exécutée par l’utilisateur.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/fr/reference/data-types/index)) — Résumé des ProfileEvents qui mesurent différentes métriques pour l’utilisateur. Leur description se trouve dans la table system.events

{/*AUTOGENERATED_END*/}
