<div id="description">
  ## Description
</div>

Contient des informations sur les disques définis dans la [configuration du serveur](/fr/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom d’un disque dans la configuration du serveur.
* `path` ([String](/fr/reference/data-types/index)) — Chemin vers le point de montage dans le système de fichiers.
* `free_space` ([UInt64](/fr/reference/data-types/index)) — Espace libre sur le disque en octets.
* `total_space` ([UInt64](/fr/reference/data-types/index)) — Capacité totale du disque en octets.
* `unreserved_space` ([UInt64](/fr/reference/data-types/index)) — Espace libre non occupé par des réservations (`free_space` moins la taille des réservations utilisées par les fusions, les insertions et les autres opérations d’écriture sur disque actuellement en cours).
* `keep_free_space` ([UInt64](/fr/reference/data-types/index)) — Quantité d’espace disque qui doit rester libre sur le disque, en octets. Définie par le paramètre keep&#95;free&#95;space&#95;bytes de la configuration du disque.
* `type` ([String](/fr/reference/data-types/index)) — Type de disque indiquant où ce disque stocke les données : RAM, disque local ou stockage distant.
* `object_storage_type` ([String](/fr/reference/data-types/index)) — Type de stockage objet si le type de disque est object&#95;storage
* `metadata_type` ([String](/fr/reference/data-types/index)) — Type de stockage des métadonnées si le type de disque est object&#95;storage
* `is_encrypted` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si ce disque chiffre les données sous-jacentes.
* `is_read_only` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant que seules des opérations de lecture peuvent être effectuées avec ce disque.
* `is_write_once` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si le disque est en écriture unique. Cela signifie qu’il prend en charge BACKUP vers ce disque, mais pas INSERT dans une table MergeTree sur ce disque.
* `is_remote` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant quelles opérations effectuées avec ce disque impliquent une interaction réseau.
* `is_broken` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si le disque est défectueux. Les disques défectueux ont un espace de 0 et ne peuvent pas être utilisés.
* `cache_path` ([String](/fr/reference/data-types/index)) — Chemin vers le répertoire de cache sur le disque local lorsque le disque prend en charge la mise en cache.

{/*AUTOGENERATED_END*/}
