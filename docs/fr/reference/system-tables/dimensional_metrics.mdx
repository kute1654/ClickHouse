<div id="description">
  ## Description
</div>

Cette table contient des métriques dimensionnelles qui peuvent être calculées instantanément et exportées au format Prometheus. Elle est toujours à jour.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/fr/reference/data-types/index)) — Nom de la métrique.
* `value` ([Float64](/fr/reference/data-types/index)) — Valeur de la métrique.
* `description` ([String](/fr/reference/data-types/index)) — Description de la métrique.
* `labels` ([Map(String, String)](/fr/reference/data-types/index)) — Labels de métrique.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

Vous pouvez utiliser une requête comme celle-ci pour exporter toutes les métriques dimensionnelles au format Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'gauge' AS type
FROM system.dimensional_metrics
FORMAT Prometheus
```

<div id="metric_descriptions">
  ## Description des métriques
</div>

<div id="merge_failures">
  ### merge_failures
</div>

Nombre total des fusions ayant échoué depuis le démarrage.

<div id="startup_scripts_failure_reason">
  ### startup_scripts_failure_reason
</div>

Indique les échecs des scripts de démarrage par type d’erreur. Vaut 1 lorsqu’un script de démarrage échoue, étiqueté avec le nom de l’erreur.

<div id="merge_tree_parts">
  ### merge_tree_parts
</div>

Nombre de parties de données MergeTree, étiquetées selon l’état de la partie, son type et le fait qu’il s’agisse ou non d’une partie de projection.
