<div id="description">
  ## Description
</div>

Cette table n’existe pas si ZooKeeper n’est pas configuré. La table `system.zookeeper_connection` affiche les connexions actuelles à ZooKeeper (y compris aux ZooKeeper auxiliaires). Chaque ligne contient des informations sur une connexion.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/string)) — Nom du cluster ZooKeeper.
* `host` ([String](/fr/reference/data-types/string)) — Nom d’hôte/adresse IP du nœud ZooKeeper auquel ClickHouse s’est connecté.
* `port` ([UIn16](/fr/reference/data-types/int-uint)) — Port du nœud ZooKeeper auquel ClickHouse s’est connecté.
* `index` ([Nullable(UInt8)](/fr/reference/data-types/int-uint)) — Index du nœud ZooKeeper auquel ClickHouse s’est connecté. L’index provient de la configuration ZooKeeper. En l’absence de connexion, cette colonne vaut NULL.
* `connected_time` ([DateTime](/fr/reference/data-types/datetime)) — Date d’établissement de la connexion
* `session_uptime_elapsed_seconds` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre de secondes écoulées depuis l’établissement de la connexion.
* `is_expired` ([UInt8](/fr/reference/data-types/int-uint)) — Indique si la connexion actuelle a expiré.
* `keeper_api_version` ([UInt8](/fr/reference/data-types/int-uint)) — Version de l’API Keeper.
* `client_id` ([Int64](/fr/reference/data-types/int-uint)) — ID de session de la connexion.
* `xid` ([Int64](/fr/reference/data-types/int-uint)) — XID de la session en cours.
* `enabled_feature_flags` ([Array(Enum16)](/fr/reference/data-types/array)) — Feature flags activés. S’applique uniquement à ClickHouse Keeper. Valeurs possibles : `FILTERED_LIST`, `MULTI_READ`, `CHECK_NOT_EXISTS`, `CREATE_IF_NOT_EXISTS`, `REMOVE_RECURSIVE`.
* `availability_zone` ([String](/fr/reference/data-types/string)) — Zone de disponibilité.
* `session_timeout_ms` ([UInt64](/fr/reference/data-types/int-uint)) — Délai d’expiration de la session (en millisecondes).
* `last_zxid_seen` ([Int64](/fr/reference/data-types/int-uint)) — Dernier zxid vu par la session en cours.

{/*AUTOGENERATED_END*/}
