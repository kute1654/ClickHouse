<div id="description">
  ## Description
</div>

Contient des informations sur chaque réplique de la base de données Replicated.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données Replicated à laquelle elle appartient.
* `is_readonly` ([UInt8](/fr/reference/data-types/index)) — Indique si la réplique de la base de données est en mode lecture seule.
* `max_log_ptr` ([Int32](/fr/reference/data-types/index)) — Numéro d’entrée maximal dans le journal d’activité générale.
* `replica_name` ([String](/fr/reference/data-types/index)) — Nom de la réplique dans ClickHouse Keeper.
* `replica_path` ([String](/fr/reference/data-types/index)) — Chemin vers les données de la réplique dans ClickHouse Keeper.
* `zookeeper_path` ([String](/fr/reference/data-types/index)) — Chemin vers les données de la base de données dans ClickHouse Keeper.
* `shard_name` ([String](/fr/reference/data-types/index)) — Nom du shard dans le cluster.
* `log_ptr` ([Int32](/fr/reference/data-types/index)) — Numéro d’entrée maximal dans le journal d’activité générale que la réplique a copié dans sa file d’exécution, plus un.
* `total_replicas` ([UInt32](/fr/reference/data-types/index)) — Nombre total de répliques connues de cette base de données.
* `zookeeper_exception` ([String](/fr/reference/data-types/index)) — Dernier message d’exception, renvoyé si l’erreur s’est produite lors de la récupération des informations depuis ClickHouse Keeper.
* `is_session_expired` ([UInt8](/fr/reference/data-types/index)) — La session avec ClickHouse Keeper a expiré. C’est essentiellement la même chose que `is_readonly`.

{/*AUTOGENERATED_END*/}
