<div id="description">
  ## Description
</div>

Contient des informations sur les codecs de compression et de chiffrement.

Vous pouvez utiliser cette table pour obtenir des informations sur les codecs de compression et de chiffrement disponibles

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du codec.
* `method_byte` ([UInt8](/fr/reference/data-types/index)) — Octet qui indique le codec dans le fichier compressé.
* `is_compression` ([UInt8](/fr/reference/data-types/index)) — Vrai si ce codec compresse quelque chose. Sinon, il peut simplement s’agir d’une transformation qui facilite la compression.
* `is_generic_compression` ([UInt8](/fr/reference/data-types/index)) — Le codec est un algorithme de compression générique comme lz4 ou zstd.
* `is_encryption` ([UInt8](/fr/reference/data-types/index)) — Le codec sert au chiffrement.
* `is_timeseries_codec` ([UInt8](/fr/reference/data-types/index)) — Le codec est destiné aux séries temporelles à virgule flottante.
* `is_experimental` ([UInt8](/fr/reference/data-types/index)) — Le codec est expérimental.
* `description` ([String](/fr/reference/data-types/index)) — Description générale du codec.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```
