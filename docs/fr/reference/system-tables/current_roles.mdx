<div id="description">
  ## Description
</div>

Contient les rôles actifs de l’utilisateur courant. `SET ROLE` modifie le contenu de cette table.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/fr/reference/data-types/index)) — Nom du rôle.
* `with_admin_option` ([UInt8](/fr/reference/data-types/index)) — 1 si le rôle dispose du privilège ADMIN OPTION.
* `is_default` ([UInt8](/fr/reference/data-types/index)) — 1 si le rôle est défini comme rôle par défaut.

{/*AUTOGENERATED_END*/}
