<div id="description">
  ## Description
</div>

Cette table contient une unique colonne UInt64 nommée `number`, qui contient presque tous les nombres naturels à partir de zéro.

Vous pouvez utiliser cette table pour des tests ou si vous devez effectuer une recherche exhaustive.

Les lectures de cette table ne sont pas parallélisées.
