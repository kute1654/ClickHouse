<div id="description">
  ## Description
</div>

Contient des informations sur les enregistrements DNS mis en cache.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/fr/reference/data-types/index)) — Nom d’hôte.
* `ip_address` ([String](/fr/reference/data-types/index)) — Adresse IP.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/fr/reference/data-types/index)) — Famille d’adresses IP.
* `cached_at` ([DateTime](/fr/reference/data-types/index)) — Horodatage de mise en cache de l’enregistrement.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
