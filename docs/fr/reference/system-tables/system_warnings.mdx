<div id="description">
  ## Description
</div>

Cette table affiche des avertissements concernant le serveur ClickHouse.
Les avertissements du même type sont regroupés en un seul avertissement.
Par exemple, si le nombre N de bases de données attachées dépasse un seuil configurable T, une seule entrée contenant la valeur actuelle N est affichée au lieu de N entrées distinctes.
Si la valeur actuelle repasse sous le seuil, l’entrée est supprimée de la table.

La table peut être configurée avec les paramètres suivants :

* [max&#95;table&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_table_num_to_warn)
* [max&#95;database&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_database_num_to_warn)
* [max&#95;dictionary&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_dictionary_num_to_warn)
* [max&#95;view&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_view_num_to_warn)
* [max&#95;part&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_part_num_to_warn)
* [max&#95;pending&#95;mutations&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_pending_mutations_to_warn)
* [max&#95;pending&#95;mutations&#95;execution&#95;time&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_pending_mutations_execution_time_to_warn)
* [max&#95;named&#95;collection&#95;num&#95;to&#95;warn](/fr/reference/settings/server-settings/settings#max_named_collection_num_to_warn)
* [resource&#95;overload&#95;warnings](/fr/concepts/features/configuration/settings/server-overload#resource-overload-warnings)

<div id="columns">
  ## Colonnes
</div>

* `message` ([String](/fr/reference/data-types/string)) — Message d’avertissement.
* `message_format_string` ([LowCardinality(String)](/fr/reference/data-types/string)) — La chaîne de format utilisée pour mettre en forme le message.

  ## Exemple

```sql title="Query"

```
