<div id="description">
  ## Description
</div>

Contient des informations sur les valeurs maximales pour tous les intervalles de tous les quotas. Un quota peut correspondre à un nombre quelconque de lignes, voire à aucune.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/fr/reference/data-types/index)) — Nom du quota.
* `duration` ([UInt32](/fr/reference/data-types/index)) — Durée de l’intervalle de temps utilisé pour calculer la consommation des ressources, en secondes.
* `is_randomized_interval` ([UInt8](/fr/reference/data-types/index)) — Valeur booléenne. Indique si l’intervalle est aléatoire. Si l’intervalle n’est pas aléatoire, il commence toujours au même moment. Par exemple, un intervalle d’1 minute commence toujours sur une minute entière (c.-à-d. qu’il peut commencer à 11:20:00, mais jamais à 11:20:01) ; un intervalle d’un jour commence toujours à minuit UTC. Si l’intervalle est aléatoire, le tout premier intervalle commence à un moment aléatoire, puis les intervalles suivants s’enchaînent. Valeurs : 0 — l’intervalle n’est pas aléatoire, 1 — l’intervalle est aléatoire.
* `max_queries` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal de requêtes de tous types pouvant être exécutées.
* `max_query_selects` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal de requêtes SELECT pouvant être exécutées.
* `max_query_inserts` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal de requêtes INSERT pouvant être exécutées.
* `max_errors` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal de requêtes ayant entraîné une erreur autorisé pendant la période spécifiée.
* `max_result_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre total maximal de lignes dans le jeu de résultats de toutes les requêtes autorisé pendant la période spécifiée.
* `max_result_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre total maximal d’octets dans le jeu de résultats de toutes les requêtes autorisé pendant la période spécifiée.
* `max_read_rows` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal de lignes pouvant être lues pendant l’exécution de toutes les requêtes autorisé pendant la période spécifiée.
* `max_read_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal d’octets pouvant être lus pendant l’exécution de toutes les requêtes autorisé pendant la période spécifiée.
* `max_execution_time` ([Nullable(Float64)](/fr/reference/data-types/index)) — La durée maximale (en nanosecondes) autorisée pour l’exécution de toutes les requêtes pendant la période spécifiée
* `max_written_bytes` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal d’octets pouvant être écrits pendant l’exécution de toutes les requêtes autorisé pendant la période spécifiée.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal d’échecs d’authentification consécutifs autorisé pendant la période spécifiée.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/fr/reference/data-types/index)) — Le nombre maximal d’exécutions d’une même requête normalisée autorisé pendant la période spécifiée.
