<div id="description">
  ## Description
</div>

Contient des enregistrements de journalisation. Le niveau de journalisation consigné dans cette table peut être limité par le paramètre serveur `text_log.level`.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur exécutant la requête.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’entrée.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’entrée.
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’entrée avec une précision à la microseconde.
* `thread_name` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom du thread ayant effectué la journalisation.
* `thread_id` ([UInt64](/fr/reference/data-types/int-uint)) — ID du thread du système d’exploitation.
* `level` ([Enum8(&#39;Fatal&#39; = 1, &#39;Critical&#39; = 2, &#39;Error&#39; = 3, &#39;Warning&#39; = 4, &#39;Notice&#39; = 5, &#39;Information&#39; = 6, &#39;Debug&#39; = 7, &#39;Trace&#39; = 8, &#39;Test&#39; = 9)](/fr/reference/data-types/enum)) — Niveau de l’entrée. Valeurs possibles : 1 ou &#39;Fatal&#39;, 2 ou &#39;Critical&#39;, 3 ou &#39;Error&#39;, 4 ou &#39;Warning&#39;, 5 ou &#39;Notice&#39;, 6 ou &#39;Information&#39;, 7 ou &#39;Debug&#39;, 8 ou &#39;Trace&#39;.
* `query_id` ([String](/fr/reference/data-types/string)) — ID de la requête.
* `logger_name` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom du logger (par ex., DDLWorker).
* `message` ([String](/fr/reference/data-types/string)) — Le message lui-même.
* `revision` ([UInt32](/fr/reference/data-types/int-uint)) — Révision de ClickHouse.
* `source_file` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Fichier source d’où provient la journalisation.
* `source_line` ([UInt64](/fr/reference/data-types/int-uint)) — Ligne du code source d’où provient la journalisation.
* `message_format_string` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Chaîne de format utilisée pour mettre en forme le message.
* `value1` ([String](/fr/reference/data-types/string)) — Argument 1 utilisé pour mettre en forme le message.
* `value2` ([String](/fr/reference/data-types/string)) — Argument 2 utilisé pour mettre en forme le message.
* `value3` ([String](/fr/reference/data-types/string)) — Argument 3 utilisé pour mettre en forme le message.
* `value4` ([String](/fr/reference/data-types/string)) — Argument 4 utilisé pour mettre en forme le message.

* `value6` ([String](/fr/reference/data-types/string)) — Argument 6 utilisé pour mettre en forme le message.
* `value7` ([String](/fr/reference/data-types/string)) — Argument 7 utilisé pour mettre en forme le message.

* `value10` ([String](/fr/reference/data-types/string)) — Argument 10 utilisé pour mettre en forme le message.
