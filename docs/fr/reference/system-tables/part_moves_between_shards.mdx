<div id="description">
  ## Description
</div>

Contient des informations sur les parts actuellement en cours de déplacement entre les shards, ainsi que sur leur progression.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Le nom de la base de données dans laquelle le déplacement est effectué.
* `table` ([String](/fr/reference/data-types/index)) — Le nom de la table dans laquelle le déplacement est effectué.
* `task_name` ([String](/fr/reference/data-types/index)) — Le nom de la tâche de déplacement.
* `task_uuid` ([UUID](/fr/reference/data-types/index)) — L’identifiant de la tâche de déplacement.
* `create_time` ([DateTime](/fr/reference/data-types/index)) — La date et l’heure de création de la tâche.
* `part_name` ([String](/fr/reference/data-types/index)) — Le nom de la part en cours de déplacement.
* `part_uuid` ([UUID](/fr/reference/data-types/index)) — L’UUID de la part en cours de déplacement.
* `to_shard` ([String](/fr/reference/data-types/index)) — Le nom du shard de destination.
* `dst_part_name` ([String](/fr/reference/data-types/index)) — Le nom de la part résultante.
* `update_time` ([DateTime](/fr/reference/data-types/index)) — La date et l’heure de la dernière mise à jour.
* `state` ([String](/fr/reference/data-types/index)) — L’état actuel du déplacement.
* `rollback` ([UInt8](/fr/reference/data-types/index)) — L’indicateur signalant si l’opération a été annulée.
* `num_tries` ([UInt32](/fr/reference/data-types/index)) — Le nombre de tentatives pour mener l’opération à bien.
* `last_exception` ([String](/fr/reference/data-types/index)) — Le nom de la dernière exception, le cas échéant.

{/*AUTOGENERATED_END*/}
