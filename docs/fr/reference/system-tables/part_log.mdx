<div id="description">
  ## Description
</div>

La table `system.part_log` n’est créée que si le paramètre de serveur [part&#95;log](/fr/reference/settings/server-settings/settings#part_log) est défini.

Cette table contient des informations sur les événements affectant les [parties de données](/fr/reference/engines/table-engines/mergetree-family/custom-partitioning-key) dans les tables de la famille [MergeTree](/fr/reference/engines/table-engines/mergetree-family/mergetree), comme l’ajout ou la fusion de données.

La table `system.part_log` contient les colonnes suivantes :

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur exécutant la requête.
* `query_id` ([String](/fr/reference/data-types/string)) — Identifiant de la requête INSERT qui a créé cette partie de données.
* `event_type` ([Enum8(&#39;NewPart&#39; = 1, &#39;MergeParts&#39; = 2, &#39;DownloadPart&#39; = 3, &#39;RemovePart&#39; = 4, &#39;MutatePart&#39; = 5, &#39;MovePart&#39; = 6, &#39;MergePartsStart&#39; = 7, &#39;MutatePartStart&#39; = 8)](/fr/reference/data-types/enum)) — Type de l’événement survenu sur la partie de données. Peut avoir l’une des valeurs suivantes : NewPart — Insertion d’une nouvelle partie de données, MergePartsStart — La fusion des parties de données a commencé, MergeParts — La fusion des parties de données est terminée, DownloadPart — Téléchargement d’une partie de données, RemovePart — Suppression ou détachement d’une partie de données à l’aide de [DETACH PARTITION](/fr/reference/statements/alter/partition#detach-partitionpart).MutatePartStart — La mutation d’une partie de données a commencé, MutatePart — La mutation d’une partie de données est terminée, MovePart — Déplacement de la partie de données d’un disque à un autre.
* `merge_reason` ([Enum8(&#39;NotAMerge&#39; = 1, &#39;RegularMerge&#39; = 2, &#39;TTLDeleteMerge&#39; = 3, &#39;TTLRecompressMerge&#39; = 4, &#39;TTLDropMerge&#39; = 5)](/fr/reference/data-types/enum)) — Raison de l’événement de type MERGE&#95;PARTS. Peut avoir l’une des valeurs suivantes : NotAMerge — L’événement actuel est d’un type autre que MERGE&#95;PARTS, RegularMerge — Fusion standard, TTLDeleteMerge, TTLDropMerge — Nettoyage des données expirées. TTLRecompressMerge — Recompression de la partie de données avec TTL.
* `merge_algorithm` ([Enum8(&#39;Undecided&#39; = 0, &#39;Vertical&#39; = 1, &#39;Horizontal&#39; = 2)](/fr/reference/data-types/enum)) — Algorithme de fusion pour l’événement de type MERGE&#95;PARTS. Peut avoir l’une des valeurs suivantes : Undecided, Horizontal, Vertical
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’événement.
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’événement avec une précision à la microseconde.
* `duration_ms` ([UInt64](/fr/reference/data-types/int-uint)) — Durée de cette opération.
* `database` ([String](/fr/reference/data-types/string)) — Nom de la base de données dans laquelle se trouve la partie de données.
* `table` ([String](/fr/reference/data-types/string)) — Nom de la table dans laquelle se trouve la partie de données.
* `table_uuid` ([UUID](/fr/reference/data-types/uuid)) — UUID de la table à laquelle appartient la partie de données.
* `part_name` ([String](/fr/reference/data-types/string)) — Nom de la partie de données.
* `partition_id` ([String](/fr/reference/data-types/string)) — ID de la partition dans laquelle la partie de données a été insérée. La colonne prend la valeur `all` si le partitionnement se fait par `tuple()`.
* `partition` ([String](/fr/reference/data-types/string)) — Nom de la partition.
* `part_type` ([String](/fr/reference/data-types/string)) — Type de la partie. Valeurs possibles : Wide et Compact.
* `part_storage_type` ([String](/fr/reference/data-types/string)) — Type de DataPartStorage. Valeurs possibles : Packed - tous les fichiers sont stockés dans un seul blob, Full - un blob par fichier.
* `disk_name` ([String](/fr/reference/data-types/string)) — Nom du disque sur lequel se trouve la partie de données.
* `path_on_disk` ([String](/fr/reference/data-types/string)) — Chemin absolu vers le dossier contenant les fichiers de la partie de données.
* `rows` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre de lignes dans la partie de données.
* `size_in_bytes` ([UInt64](/fr/reference/data-types/int-uint)) — Taille de la partie de données sur le disque, en octets.
* `merged_from` ([Array(String)](/fr/reference/data-types/array)) — Tableau des noms des parties source dont est issue la partie actuelle.
* `bytes_uncompressed` ([UInt64](/fr/reference/data-types/int-uint)) — Taille non compressée de la part résultante, en octets.
* `read_rows` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre de lignes lues pendant la fusion.
* `read_bytes` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre d’octets lus pendant la fusion.
* `peak_memory_usage` ([UInt64](/fr/reference/data-types/int-uint)) — Quantité maximale de RAM utilisée pendant la fusion.
* `deduplication_block_ids` ([Array(String)](/fr/reference/data-types/array)) — Tableau des ID de blocs utilisés pour la déduplication lors de l’insertion de cette part.
* `error` ([UInt16](/fr/reference/data-types/int-uint)) — Code d’erreur de l’exception survenue.
* `exception` ([String](/fr/reference/data-types/string)) — Message texte de l’erreur survenue.
* `mutation_ids` ([Array(String)](/fr/reference/data-types/array)) — Tableau des ID de mutation appliqués à la part source (merged&#95;from) pour les événements de type MUTATE&#95;PART&#95;START et MUTATE&#95;PART.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/fr/reference/data-types/map)) — Tous les profile events capturés pendant cette opération.
* `projections_duration_ms` ([Map(LowCardinality(String), UInt64)](/fr/reference/data-types/map)) — Durée de fusion/reconstruction par projection, en millisecondes.

**Alias :**

* `ProfileEvents.Names` — alias de `mapKeys(ProfileEvents)`.
* `ProfileEvents.Values` — alias de `mapValues(ProfileEvents)`.
* `name` — alias de `part_name`.

{/*AUTOGENERATED_END*/}

La table `system.part_log` est créée lors de la première insertion de données dans la table `MergeTree`.
