<div id="description">
  ## Description
</div>

Contient la liste des moteurs de base de données pris en charge par le serveur.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Le nom du moteur de base de données.
* `description` ([String](/fr/reference/data-types/index)) — Une description générale du rôle du moteur de base de données.
* `syntax` ([String](/fr/reference/data-types/index)) — La façon dont le moteur de base de données est spécifié dans la clause ENGINE d’une requête CREATE DATABASE.
* `examples` ([String](/fr/reference/data-types/index)) — Exemples d’utilisation.
* `introduced_in` ([String](/fr/reference/data-types/index)) — La version de ClickHouse dans laquelle le moteur de base de données a été introduit pour la première fois, sous la forme major.minor.
* `related` ([Array(String)](/fr/reference/data-types/index)) — Les noms des moteurs de base de données connexes.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```
