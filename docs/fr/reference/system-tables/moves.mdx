<div id="description">
  ## Description
</div>

Cette table contient des informations sur les [déplacements de parties de données](/fr/reference/statements/alter/partition#move-partitionpart) en cours dans les tables [MergeTree](/fr/reference/engines/table-engines/mergetree-family/mergetree). Chaque déplacement de partie de données est représenté par une seule ligne.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table contenant la partie de données en cours de déplacement.
* `elapsed` ([Float64](/fr/reference/data-types/index)) — Temps écoulé (en secondes) depuis le début du déplacement de la partie de données.
* `target_disk_name` ([String](/fr/reference/data-types/index)) — Nom du disque vers lequel la partie de données est déplacée.
* `target_disk_path` ([String](/fr/reference/data-types/index)) — Chemin vers le point de montage du disque dans le système de fichiers.
* `part_name` ([String](/fr/reference/data-types/index)) — Nom de la partie de données en cours de déplacement.
* `part_size` ([UInt64](/fr/reference/data-types/index)) — Taille de la partie de données.
* `thread_id` ([UInt64](/fr/reference/data-types/index)) — Identifiant du thread effectuant le déplacement.

{/*AUTOGENERATED_END*/}
