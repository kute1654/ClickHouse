<div id="description">
  ## Description
</div>

Cette table n&#39;existe pas si ce nœud n&#39;est pas configuré pour exécuter un ClickHouse Keeper dans le même processus. Elle contient une ligne par fichier de changelog de Raft (`changelog_<from>_<to>.bin[.zstd]`) suivi par le stockage des logs de Keeper dans le même processus, y compris le fichier actif auquel de nouvelles entrées sont actuellement ajoutées.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `from_log_index` ([UInt64](/fr/reference/data-types/int-uint)) — Premier index de log Raft dans le fichier (inclus).
* `to_log_index` ([UInt64](/fr/reference/data-types/int-uint)) — Dernier index de log Raft couvert par le nom de fichier (inclus). Pour le fichier actif, il s’agit de la cible de rotation et il peut être supérieur à `last_entry_index`.
* `last_entry_index` ([Nullable(UInt64)](/fr/reference/data-types/int-uint)) — Index de log le plus élevé effectivement ajouté à ce fichier. `NULL` si le fichier actif n’a encore reçu aucune entrée ou si le fichier est endommagé.
* `entries` ([UInt64](/fr/reference/data-types/int-uint)) — Nombre d’entrées ajoutées à ce fichier. `ALIAS` de `ifNull(last_entry_index - from_log_index + 1, 0)`.
* `path` ([String](/fr/reference/data-types/string)) — Chemin du fichier sur le disque.
* `disk_name` ([String](/fr/reference/data-types/string)) — Nom du disque contenant le fichier.
* `size_bytes` ([UInt64](/fr/reference/data-types/int-uint)) — Taille du fichier sur le disque.
* `modification_time` ([DateTime](/fr/reference/data-types/datetime)) — Date et heure de la dernière modification du fichier.
* `is_compressed` ([Bool](/fr/reference/data-types/boolean)) — Le contenu du fichier est compressé avec zstd.
* `active` ([Bool](/fr/reference/data-types/boolean)) — De nouvelles entrées sont actuellement ajoutées à ce fichier.
* `is_broken` ([Bool](/fr/reference/data-types/boolean)) — Un enregistrement final corrompu a été détecté au démarrage.

{/*AUTOGENERATED_END*/}

Exemple :

```sql
SELECT from_log_index, to_log_index, entries, path, active FROM system.keeper_changelogs ORDER BY from_log_index;
```
