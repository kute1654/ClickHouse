<div id="description">
  ## Description
</div>

Contient des informations sur les consommateurs Kafka.
S’applique au [moteur de table Kafka](/fr/reference/engines/table-engines/integrations/kafka) (intégration native de ClickHouse)

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Base de données de la table utilisant le moteur Kafka.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table utilisant le moteur Kafka.
* `consumer_id` ([String](/fr/reference/data-types/index)) — Identifiant du consommateur Kafka. Notez qu’une table peut avoir plusieurs consommateurs. Spécifié par le paramètre `kafka_num_consumers`.
* `assignments.topic` ([Array(String)](/fr/reference/data-types/index)) — Kafka topic.
* `assignments.partition_id` ([Array(Int32)](/fr/reference/data-types/index)) — ID de partition Kafka. Notez qu’un seul consommateur peut être affecté à une partition.
* `assignments.current_offset` ([Array(Int64)](/fr/reference/data-types/index)) — Offset actuel.
* `assignments.intent_size` ([Array(Nullable(Int64))](/fr/reference/data-types/index)) — Nombre de messages envoyés, mais pas encore validés, dans le nouveau StorageKafka.
* `exceptions.time` ([Array(DateTime)](/fr/reference/data-types/index)) — Horodatage de génération des 10 exceptions les plus récentes.
* `exceptions.text` ([Array(String)](/fr/reference/data-types/index)) — Texte des 10 exceptions les plus récentes.
* `last_poll_time` ([DateTime](/fr/reference/data-types/index)) — Horodatage du poll le plus récent.
* `num_messages_read` ([UInt64](/fr/reference/data-types/index)) — Nombre de messages lus par le consommateur.
* `last_commit_time` ([DateTime](/fr/reference/data-types/index)) — Horodatage du poll le plus récent.
* `num_commits` ([UInt64](/fr/reference/data-types/index)) — Nombre total de commits du consommateur.
* `last_rebalance_time` ([DateTime](/fr/reference/data-types/index)) — Horodatage du rebalance Kafka le plus récent.
* `num_rebalance_revocations` ([UInt64](/fr/reference/data-types/index)) — Nombre de fois où les partitions du consommateur ont été révoquées.
* `num_rebalance_assignments` ([UInt64](/fr/reference/data-types/index)) — Nombre de fois où le consommateur a été affecté au cluster Kafka.
* `is_currently_used` ([UInt8](/fr/reference/data-types/index)) — Indicateur signalant si le consommateur est actuellement utilisé.
* `last_used` ([DateTime64(6)](/fr/reference/data-types/index)) — Dernière utilisation de ce consommateur.
* `rdkafka_stat` ([String](/fr/reference/data-types/index)) — Statistique interne de la bibliothèque. Définissez statistics&#95;interval&#95;ms sur 0 pour la désactiver ; la valeur par défaut est 3000 (une fois toutes les trois secondes).
* `dependencies` ([Array(Array(String))](/fr/reference/data-types/index)) — Dépendances transitives de la base de données.
* `missing_dependencies` ([Array(Array(String))](/fr/reference/data-types/index)) — Dépendances transitives manquantes de la base de données.

{/*AUTOGENERATED_END*/}
