<div id="description">
  ## Description
</div>

Contient la liste de tous les combinateurs de fonctions d’agrégation disponibles, qui peuvent être appliqués aux fonctions d’agrégation pour en modifier le fonctionnement.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Le nom du combinateur.
* `is_internal` ([UInt8](/fr/reference/data-types/index)) — Indique si ce combinateur est réservé à un usage interne.
* `description` ([String](/fr/reference/data-types/index)) — Une description générale du rôle du combinateur.
* `syntax` ([String](/fr/reference/data-types/index)) — La manière dont le combinateur s’applique au nom d’une fonction d’agrégation.
* `examples` ([String](/fr/reference/data-types/index)) — Exemples d’utilisation.
* `introduced_in` ([String](/fr/reference/data-types/index)) — La version de ClickHouse dans laquelle le combinateur a été introduit pour la première fois, sous la forme major.minor.
* `related` ([Array(String)](/fr/reference/data-types/index)) — Les noms des combinateurs associés.

{/*AUTOGENERATED_END*/}
