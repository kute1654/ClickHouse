<div id="description">
  ## Description
</div>

Contient des informations sur les changements de paramètres dans les versions antérieures de ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `type` ([Enum8(&#39;Session&#39; = 0, &#39;MergeTree&#39; = 1)](/fr/reference/data-types/index)) — Le groupe de paramètres (Session, MergeTree...)
* `version` ([String](/fr/reference/data-types/index)) — La version du serveur ClickHouse.
* `changes` ([Array(Tuple(name String, previous&#95;value String, new&#95;value String, reason String))](/fr/reference/data-types/index)) — La liste des modifications apportées aux paramètres qui ont modifié le comportement de ClickHouse.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT *
FROM system.settings_changes
WHERE version = '23.5'
FORMAT Vertical
```

```text
Row 1:
──────
type:    Core
version: 23.5
changes: [('input_format_parquet_preserve_order','1','0','Allow Parquet reader to reorder rows for better parallelism.'),('parallelize_output_from_storages','0','1','Allow parallelism when executing queries that read from file/url/s3/etc. This may reorder rows.'),('use_with_fill_by_sorting_prefix','0','1','Columns preceding WITH FILL columns in ORDER BY clause form sorting prefix. Rows with different values in sorting prefix are filled independently'),('output_format_parquet_compliant_nested_types','0','1','Change an internal field name in output Parquet file schema.')]
```
