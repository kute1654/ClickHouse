<div id="description">
  ## Description
</div>

Contient des informations sur les répertoires utilisateur configurés : des répertoires du système de fichiers à partir desquels ClickHouse server est autorisé à lire des données fournies par l&#39;utilisateur.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Le nom du répertoire.
* `type` ([String](/fr/reference/data-types/index)) — Le type de stockage des accès, par ex. `users.xml`, `replicated` ou `memory`.
* `params` ([String](/fr/reference/data-types/index)) — Un JSON contenant les paramètres du stockage des accès.
* `precedence` ([UInt64](/fr/reference/data-types/index)) — L&#39;ordre dans lequel ce répertoire est déclaré dans la config. Le même ordre est utilisé lorsque ClickHouse tente de trouver un utilisateur ou un rôle.

{/*AUTOGENERATED_END*/}
