<div id="description">
  ## Description
</div>

Contient des informations sur les fonctions standard et d’agrégation.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Le nom de la fonction.
* `is_aggregate` ([UInt8](/fr/reference/data-types/index)) — Indique si la fonction est une fonction d&#39;agrégation.
* `case_insensitive` ([UInt8](/fr/reference/data-types/index)) — Indique si le nom de la fonction peut être utilisé sans tenir compte de la casse.
* `alias_to` ([String](/fr/reference/data-types/index)) — Le nom d&#39;origine de la fonction, si le nom de la fonction est un alias.
* `create_query` ([String](/fr/reference/data-types/index)) — Obsolète.
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/fr/reference/data-types/index)) — Obsolète.
* `description` ([String](/fr/reference/data-types/index)) — Une description générale de ce que fait la fonction.
* `syntax` ([String](/fr/reference/data-types/index)) — La signature de la fonction.
* `arguments` ([String](/fr/reference/data-types/index)) — Les arguments de la fonction.
* `parameters` ([String](/fr/reference/data-types/index)) — Les paramètres de la fonction (uniquement pour les fonctions d&#39;agrégation).
* `returned_value` ([String](/fr/reference/data-types/index)) — Ce que renvoie la fonction.
* `examples` ([String](/fr/reference/data-types/index)) — Exemple d&#39;utilisation.
* `introduced_in` ([String](/fr/reference/data-types/index)) — Version de ClickHouse dans laquelle la fonction a été introduite pour la première fois.
* `categories` ([String](/fr/reference/data-types/index)) — La catégorie de la fonction.
* `deterministic` ([Nullable(UInt8)](/fr/reference/data-types/index)) — Indique si la fonction renvoie le même résultat pour les mêmes arguments. NULL lorsque cette information est inconnue (par ex. pour les fonctions d&#39;agrégation ou les fonctions définies par l&#39;utilisateur).
* `higher_order` ([Nullable(UInt8)](/fr/reference/data-types/index)) — Indique si la fonction est d&#39;ordre supérieur, c.-à-d. si elle accepte au moins une expression lambda comme argument (par ex. arrayMap, arrayFilter, mapApply). NULL lorsque cette information est inconnue.

{/*AUTOGENERATED_END*/}
