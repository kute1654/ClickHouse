<div id="description">
  ## Description
</div>

Contient des informations sur les insertions asynchrones en attente dans la file d’attente.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/fr/reference/data-types/index)) — Texte de la requête.
* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `format` ([String](/fr/reference/data-types/index)) — Nom du format.
* `first_update` ([DateTime64(6)](/fr/reference/data-types/index)) — Heure de la première insertion, avec une précision à la microseconde.
* `total_bytes` ([UInt64](/fr/reference/data-types/index)) — Nombre total d’octets en attente dans la file d’attente.
* `entries.query_id` ([Array(String)](/fr/reference/data-types/index)) — Tableau des identifiants de requête des insertions en attente dans la file d’attente.
* `entries.bytes` ([Array(UInt64)](/fr/reference/data-types/index)) — Tableau du nombre d’octets de chaque requête d’insertion en attente dans la file d’attente.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql title="Query"
SELECT * FROM system.asynchronous_inserts LIMIT 1 \G;
```

```text title="Response"
Row 1:
──────

total_bytes:      133223

entries.bytes:    [133223]
```
