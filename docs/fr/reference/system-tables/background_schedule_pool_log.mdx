<div id="description">
  ## Description
</div>

La table `system.background_schedule_pool_log` n&#39;est créée que si le paramètre du serveur [background&#95;schedule&#95;pool&#95;log](/fr/reference/settings/server-settings/settings#background_schedule_pool_log) est défini.

Cette table contient l&#39;historique des exécutions de tâches du pool de planification en arrière-plan. Les pools de planification en arrière-plan sont utilisés pour exécuter des tâches périodiques telles que les envois distribués, les vidages de tampon et les opérations du broker de messages.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui exécute la requête.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’événement.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’événement.
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’événement avec une précision à la microseconde.
* `query_id` ([String](/fr/reference/data-types/string)) — Identifiant de la requête associée à la tâche en arrière-plan.
* `database` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de la base de données.
* `table` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de la table.
* `table_uuid` ([UUID](/fr/reference/data-types/uuid)) — UUID de la table à laquelle la tâche en arrière-plan appartient.
* `log_name` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom de la tâche en arrière-plan.
* `duration_ms` ([UInt64](/fr/reference/data-types/int-uint)) — Durée d’exécution de la tâche, en millisecondes.
* `error` ([UInt16](/fr/reference/data-types/int-uint)) — Code d’erreur de l’exception survenue.
* `exception` ([String](/fr/reference/data-types/string)) — Message texte de l’erreur survenue.

{/*AUTOGENERATED_END*/}

La table `system.background_schedule_pool_log` est créée après la première exécution d&#39;une tâche en arrière-plan.
