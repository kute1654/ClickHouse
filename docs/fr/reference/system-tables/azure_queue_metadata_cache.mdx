<div id="description">
  ## Description
</div>

Contient l’état en mémoire des métadonnées d’AzureQueue et les lignes en cours de traitement pour chaque fichier.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/fr/reference/data-types/index)) — Chemin dans ZooKeeper vers les métadonnées
* `file_path` ([String](/fr/reference/data-types/index)) — Chemin du fichier en cours de traitement
* `file_name` ([String](/fr/reference/data-types/index)) — Nom du fichier en cours de traitement
* `rows_processed` ([UInt64](/fr/reference/data-types/index)) — Nombre de lignes actuellement traitées
* `status` ([String](/fr/reference/data-types/index)) — Statut du traitement : Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](/fr/reference/data-types/index)) — Heure de début du traitement du fichier
* `processing_end_time` ([Nullable(DateTime)](/fr/reference/data-types/index)) — Heure de fin du traitement du fichier
* `exception` ([String](/fr/reference/data-types/index)) — Exception survenue lors du traitement

{/*AUTOGENERATED_END*/}
