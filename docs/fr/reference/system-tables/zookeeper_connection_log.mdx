<div id="description">
  ## Description
</div>

La table &#39;system.zookeeper&#95;connection&#95;log&#39; affiche l’historique des connexions à ZooKeeper (y compris aux ZooKeeper auxiliaires). Chaque ligne fournit des informations sur un événement lié aux connexions.

<Note>
  La table ne contient pas d’événements de déconnexion causés par l’arrêt du serveur.
</Note>

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Nom d’hôte du serveur qui s’est connecté à ZooKeeper ou s’en est déconnecté.
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/fr/reference/data-types/enum)) — Type de l’événement. Valeurs possibles : Connected, Disconnected.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’entrée.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure de l’entrée
* `event_time_microseconds` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’entrée avec une précision à la microseconde.
* `name` ([String](/fr/reference/data-types/string)) — Nom du cluster ZooKeeper.
* `host` ([String](/fr/reference/data-types/string)) — Nom d’hôte/IP du nœud ZooKeeper auquel ClickHouse s’est connecté ou duquel il s’est déconnecté.
* `port` ([UInt16](/fr/reference/data-types/int-uint)) — Port du nœud ZooKeeper auquel ClickHouse s’est connecté ou duquel il s’est déconnecté.
* `index` ([UInt8](/fr/reference/data-types/int-uint)) — Index du nœud ZooKeeper auquel ClickHouse s’est connecté ou duquel il s’est déconnecté. L’index provient de la config ZooKeeper.
* `client_id` ([Int64](/fr/reference/data-types/int-uint)) — ID de session de la connexion.
* `keeper_api_version` ([UInt8](/fr/reference/data-types/int-uint)) — Version de l’API Keeper.
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/fr/reference/data-types/array)) — Feature flags activés. S’applique uniquement à ClickHouse Keeper.
* `availability_zone` ([String](/fr/reference/data-types/string)) — Zone de disponibilité
* `reason` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Raison de la connexion ou de la déconnexion.

{/*AUTOGENERATED_END*/}

SELECT * FROM system.zookeeper_connection_log;
