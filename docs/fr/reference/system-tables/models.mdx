<div id="description">
  ## Description
</div>

Contient la liste des modèles CatBoost chargés dans la mémoire de LibraryBridge, ainsi que l’heure de leur chargement.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `model_path` ([String](/fr/reference/data-types/index)) — Chemin d’accès au modèle entraîné.
* `type` ([String](/fr/reference/data-types/index)) — Type de modèle. Actuellement, seul catboost est pris en charge.
* `loading_start_time` ([DateTime](/fr/reference/data-types/index)) — Heure de début du chargement du modèle.
* `loading_duration` ([Float32](/fr/reference/data-types/index)) — Temps nécessaire au chargement du modèle.

{/*AUTOGENERATED_END*/}
