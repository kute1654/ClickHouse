<div id="description">
  ## Description
</div>

Contient la liste de toutes les opérations `BACKUP` ou `RESTORE`, avec leur état actuel et d&#39;autres propriétés. Notez que cette table n&#39;est pas persistante et qu&#39;elle n&#39;affiche que les opérations exécutées depuis le dernier redémarrage du serveur.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `id` ([String](/fr/reference/data-types/index)) — ID de l’opération, qui peut être soit fourni via SETTINGS id=..., soit un UUID généré aléatoirement.
* `name` ([String](/fr/reference/data-types/index)) — Nom de l’opération, une chaîne comme `Disk('backups', 'my_backup')`
* `base_backup_name` ([String](/fr/reference/data-types/index)) — Nom de l’opération de sauvegarde de base, une chaîne comme `Disk('backups', 'my_base_backup')`
* `query_id` ([String](/fr/reference/data-types/index)) — ID de la requête qui a démarré la sauvegarde.
* `status` ([Enum8(&#39;CREATING&#95;BACKUP&#39; = 0, &#39;BACKUP&#95;CREATED&#39; = 1, &#39;BACKUP&#95;FAILED&#39; = 2, &#39;RESTORING&#39; = 3, &#39;RESTORED&#39; = 4, &#39;RESTORE&#95;FAILED&#39; = 5, &#39;BACKUP&#95;CANCELLED&#39; = 6, &#39;RESTORE&#95;CANCELLED&#39; = 7)](/fr/reference/data-types/index)) — Statut de l’opération de sauvegarde ou de restauration.
* `error` ([String](/fr/reference/data-types/index)) — Le message d’erreur, le cas échéant.
* `start_time` ([DateTime64(6)](/fr/reference/data-types/index)) — Heure de début de l’opération.
* `end_time` ([DateTime64(6)](/fr/reference/data-types/index)) — Heure de fin de l’opération.
* `num_files` ([UInt64](/fr/reference/data-types/index)) — Nombre de fichiers stockés dans la sauvegarde.
* `total_size` ([UInt64](/fr/reference/data-types/index)) — Taille totale des fichiers stockés dans la sauvegarde.
* `num_entries` ([UInt64](/fr/reference/data-types/index)) — Nombre d’entrées dans la sauvegarde, c’est-à-dire le nombre de fichiers dans le dossier si la sauvegarde est stockée sous forme de dossier.
* `uncompressed_size` ([UInt64](/fr/reference/data-types/index)) — Taille non compressée de la sauvegarde.
* `compressed_size` ([UInt64](/fr/reference/data-types/index)) — Taille compressée de la sauvegarde.
* `files_read` ([UInt64](/fr/reference/data-types/index)) — Renvoie le nombre de fichiers lus pendant RESTORE depuis cette sauvegarde.
* `bytes_read` ([UInt64](/fr/reference/data-types/index)) — Renvoie la taille totale des fichiers lus pendant RESTORE depuis cette sauvegarde.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/fr/reference/data-types/index)) — Tous les événements de profil capturés pendant cette opération.

{/*AUTOGENERATED_END*/}
