<div id="description">
  ## Description
</div>

Cette table contient une seule ligne avec une seule colonne `dummy` de type UInt8, contenant la valeur 0.

Cette table est utilisée lorsqu&#39;une requête `SELECT` ne spécifie pas de clause `FROM`.

Elle est similaire à la table `DUAL` présente dans d&#39;autres SGBD.
