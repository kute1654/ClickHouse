<div id="description">
  ## Description
</div>

Contient la liste des transactions et de leur état.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/fr/reference/data-types/tuple)) — L’identifiant de la transaction.
* `tid_hash` ([UInt64](/fr/reference/data-types/int-uint)) — Le hash de l’identifiant.
* `elapsed` ([Float64](/fr/reference/data-types/float)) — Le temps nécessaire au traitement de la transaction.
* `is_readonly` ([UInt8](/fr/reference/data-types/int-uint)) — L’indicateur qui indique si la transaction a exécuté au moins une opération d’écriture.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/fr/reference/data-types/enum)) — L’état de la transaction. Valeurs possibles : RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/}
