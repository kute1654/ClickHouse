<div id="description">
  ## Description
</div>

Contient l’historique des valeurs de mémoire et des métriques de la table `system.events` pour chaque requête, écrit périodiquement sur disque.

Lorsqu’une requête démarre, des données sont collectées à intervalles réguliers de `query_metric_log_interval` millisecondes (qui est défini à 1000
par défaut). Les données sont également collectées à la fin de la requête si celle-ci dure plus de `query_metric_log_interval`.
