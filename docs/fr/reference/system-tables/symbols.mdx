<div id="description">
  ## Description
</div>

Contient des informations permettant l’introspection du binaire `clickhouse`. Son accès nécessite le privilège `introspection`.
Cette table n’est utile qu’aux experts C++ et aux ingénieurs ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `symbol` ([String](/fr/reference/data-types/index)) — Nom du symbole dans le binaire. Il est « manglé ». Vous pouvez appliquer demangle(symbol) pour obtenir un nom lisible.
* `address_begin` ([UInt64](/fr/reference/data-types/index)) — Adresse de début du symbole dans le binaire.
* `address_end` ([UInt64](/fr/reference/data-types/index)) — Adresse de fin du symbole dans le binaire.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.symbols WHERE function_id IS NOT NULL LIMIT 5 SETTINGS allow_introspection_functions = 1
```
