<div id="description">
  ## Description
</div>

Contient tous les rôles actifs à l’instant donné, y compris le rôle actuel de l’utilisateur courant et les rôles attribués au rôle actuel.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/fr/reference/data-types/index)) — Nom du rôle.
* `with_admin_option` ([UInt8](/fr/reference/data-types/index)) — 1 si le rôle dispose du privilège ADMIN OPTION.
* `is_current` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si `enabled_role` est un rôle actuel de l’utilisateur courant.
* `is_default` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si `enabled_role` est un rôle par défaut.

{/*AUTOGENERATED_END*/}
