<div id="description">
  ## Description
</div>

Affiche les [watches](https://zookeeper.apache.org/doc/r3.3.3/zookeeperProgrammers.html#ch_zkWatches) actuellement actives enregistrées par ce serveur ClickHouse sur des nœuds ZooKeeper (y compris sur les ZooKeepers auxiliaires). Chaque ligne correspond à une watch.

<div id="columns">
  ## Colonnes
</div>

* `zookeeper_name` ([String](/fr/reference/data-types/string)) — Nom de la connexion ZooKeeper (`default` pour la connexion principale ou le nom auxiliaire).
* `create_time` ([DateTime](/fr/reference/data-types/datetime)) — Heure à laquelle la watch a été créée.
* `create_time_microseconds` ([DateTime64](/fr/reference/data-types/datetime64)) — Heure à laquelle la watch a été créée, avec une précision à la microseconde.
* `path` ([String](/fr/reference/data-types/string)) — Chemin ZooKeeper surveillé.
* `session_id` ([Int64](/fr/reference/data-types/int-uint)) — ID de session de la connexion qui a enregistré la watch.
* `request_xid` ([Int64](/fr/reference/data-types/int-uint)) — XID de la requête qui a créé la watch.
* `op_num` ([Enum](/fr/reference/data-types/enum)) — Type de la requête qui a créé la watch.
* `watch_type` ([Enum8](/fr/reference/data-types/enum)) — Type de watch. Valeurs possibles :
  * `Children` — surveillance des modifications de la liste des nœuds enfants (définie par les opérations `List`).
  * `Exists` — surveillance de la création ou de la suppression d&#39;un nœud.
  * `Data` — surveillance des modifications des données du nœud (définie par les opérations `Get`).

Exemple :

```sql
SELECT * FROM system.zookeeper_watches FORMAT Vertical;
```

```text
Row 1:
──────
zookeeper_name:           default
create_time:              2026-03-16 12:00:00
create_time_microseconds: 2026-03-16 12:00:00.123456
path:                     /clickhouse/task_queue/ddl
session_id:               106662742089334927
request_xid:              10858
op_num:                   List
watch_type:               Children
```
