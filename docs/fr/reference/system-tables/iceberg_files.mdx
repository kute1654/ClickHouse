<div id="description">
  ## Description
</div>

Cette table système contient des métadonnées par fichier pour les tables Iceberg présentes dans ClickHouse, avec une ligne pour chaque fichier de données ou de suppression référencé dans l’instantané actuel de chaque table. Elle sera vide si vous n’avez aucune table Iceberg dans ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `snapshot_id` ([Int64](/fr/reference/data-types/index)) — ID du snapshot auquel le fichier a été ajouté.
* `content` ([Enum8(&#39;DATA&#39; = 0, &#39;POSITION&#95;DELETE&#39; = 1, &#39;EQUALITY&#95;DELETE&#39; = 2)](/fr/reference/data-types/index)) — Type de contenu du fichier.
* `file_path` ([String](/fr/reference/data-types/index)) — Chemin de stockage résolu du fichier.
* `file_format` ([String](/fr/reference/data-types/index)) — Format du fichier, par ex. &#39;PARQUET&#39;.
* `record_count` ([Int64](/fr/reference/data-types/index)) — Nombre d’enregistrements dans le fichier.
* `file_size_in_bytes` ([Int64](/fr/reference/data-types/index)) — Taille du fichier en octets.
* `partition` ([String](/fr/reference/data-types/index)) — Représentation textuelle du tuple de partition.
* `schema_id` ([Int32](/fr/reference/data-types/index)) — ID de schéma résolu pour cette entrée de manifest.
* `sequence_number` ([Int64](/fr/reference/data-types/index)) — Numéro de séquence résolu de l’entrée de manifest (toujours 0 pour le format v1).
* `sort_order_id` ([Nullable(Int32)](/fr/reference/data-types/index)) — ID de l’ordre de tri du fichier, s’il est spécifié.
* `null_value_counts` ([Map(Int32, Int64)](/fr/reference/data-types/index)) — Nombre de valeurs NULL par colonne (ID de colonne -&gt; nombre).
* `column_sizes` ([Map(Int32, Int64)](/fr/reference/data-types/index)) — Taille sur disque par colonne en octets (ID de colonne -&gt; octets).
* `value_counts` ([Map(Int32, Int64)](/fr/reference/data-types/index)) — Nombre total de valeurs par colonne (ID de colonne -&gt; nombre).
* `equality_ids` ([Array(Int32)](/fr/reference/data-types/index)) — ID des champs d’égalité pour les fichiers de suppression par égalité (vide pour les fichiers sans suppression par égalité).
