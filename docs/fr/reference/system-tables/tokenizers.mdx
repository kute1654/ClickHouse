<div id="description">
  ## Description
</div>

Affiche tous les tokenizers disponibles.
Ils s’utilisent dans les fonctions [tokens](/fr/reference/functions/regular-functions/splitting-merging-functions#tokens), [hasAllTokens](/fr/reference/functions/regular-functions/string-search-functions#hasAllTokens), [hasAnyTokens](/fr/reference/functions/regular-functions/string-search-functions#hasAnyTokens) et dans l’[index de texte](/fr/reference/engines/table-engines/mergetree-family/textindexes).

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du tokenizer

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.tokenizers;
```

│ ngrambf_v1      │
│ array           │
│ splitByString   │
│ sparse_grams    │
│ asciiCJK        │
