<div id="description">
  ## Description
</div>

Contient des informations sur les traces de pile des erreurs fatales. Cette table n&#39;existe pas par défaut dans la base de données ; elle n&#39;est créée qu&#39;en cas d&#39;erreurs fatales.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Le nom d’hôte où le plantage s’est produit.
* `event_date` ([Date](/fr/reference/data-types/date)) — La date du plantage.
* `event_time` ([DateTime](/fr/reference/data-types/datetime)) — L’heure du plantage.
* `timestamp_ns` ([UInt64](/fr/reference/data-types/int-uint)) — Horodatage de l’événement en nanosecondes.
* `signal` ([Int32](/fr/reference/data-types/int-uint)) — Numéro du signal.
* `signal_code` ([Int32](/fr/reference/data-types/int-uint)) — Code du signal. Fournit un contexte supplémentaire sur la cause du signal.
* `thread_id` ([UInt64](/fr/reference/data-types/int-uint)) — ID du thread.
* `query_id` ([String](/fr/reference/data-types/string)) — ID de la requête.
* `query` ([String](/fr/reference/data-types/string)) — Texte de la requête en cours d’exécution au moment du plantage.
* `trace` ([Array(UInt64)](/fr/reference/data-types/array)) — Trace de pile au moment du plantage. Chaque élément est une adresse de mémoire virtuelle à l’intérieur du processus du serveur ClickHouse.
* `trace_full` ([Array(String)](/fr/reference/data-types/array)) — Trace de pile au moment du plantage. Chaque élément contient une méthode appelée à l’intérieur du processus du serveur ClickHouse.
* `fault_address` ([Nullable(UInt64)](/fr/reference/data-types/nullable)) — Adresse mémoire à l’origine de la faute.
* `fault_access_type` ([String](/fr/reference/data-types/string)) — Type d’accès mémoire à l’origine de la faute (par ex., &#39;read&#39;, &#39;write&#39;).
* `signal_description` ([String](/fr/reference/data-types/string)) — Description lisible basée sur signal&#95;code (par ex., &#39;Address not mapped to object&#39;).
* `current_exception_trace_full` ([Array(String)](/fr/reference/data-types/array)) — Trace de pile de l’exception en cours avec des frames symbolisées (plantages `std::terminate` uniquement).
* `version` ([String](/fr/reference/data-types/string)) — Version du serveur ClickHouse.
* `revision` ([UInt32](/fr/reference/data-types/int-uint)) — Révision du serveur ClickHouse.
* `build_id` ([String](/fr/reference/data-types/string)) — BuildID généré par le compilateur.
* `git_hash` ([String](/fr/reference/data-types/string)) — Hash du commit Git du code source de ClickHouse.
* `architecture` ([String](/fr/reference/data-types/string)) — Architecture du CPU (par ex., x86&#95;64, aarch64).

{/*AUTOGENERATED_END*/}
