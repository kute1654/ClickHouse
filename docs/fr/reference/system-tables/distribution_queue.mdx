<div id="description">
  ## Description
</div>

Contient des informations sur les fichiers locaux en attente d’envoi vers les shards. Ces fichiers locaux contiennent de nouvelles parts créées lors de l’insertion de nouvelles données dans la table Distributed en mode asynchrone.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `data_path` ([String](/fr/reference/data-types/index)) — Chemin vers le dossier contenant les fichiers locaux.
* `is_blocked` ([UInt8](/fr/reference/data-types/index)) — Indique si l’envoi de fichiers locaux au serveur est bloqué.
* `error_count` ([UInt64](/fr/reference/data-types/index)) — Nombre d’erreurs.
* `data_files` ([UInt64](/fr/reference/data-types/index)) — Nombre de fichiers locaux dans le dossier.
* `data_compressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données compressées dans les fichiers locaux, en octets.
* `broken_data_files` ([UInt64](/fr/reference/data-types/index)) — Nombre de fichiers marqués comme défectueux (à la suite d’une erreur).
* `broken_data_compressed_bytes` ([UInt64](/fr/reference/data-types/index)) — Taille des données compressées dans les fichiers défectueux, en octets.
* `last_exception` ([String](/fr/reference/data-types/index)) — Message texte concernant la dernière erreur survenue, le cas échéant.
* `last_exception_time` ([DateTime](/fr/reference/data-types/index)) — Heure à laquelle la dernière exception s’est produite.

{/*AUTOGENERATED_END*/}
