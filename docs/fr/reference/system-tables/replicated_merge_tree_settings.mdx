<div id="description">
  ## Description
</div>

Contient la liste de tous les paramètres spécifiques au moteur ReplicatedMergeTree, avec leurs valeurs actuelles, leurs valeurs par défaut et leur description. Vous pouvez modifier n’importe lequel d’entre eux dans la section SETTINGS de la requête CREATE.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom du paramètre.
* `value` ([String](/fr/reference/data-types/index)) — Valeur du paramètre.
* `default` ([String](/fr/reference/data-types/index)) — Valeur par défaut du paramètre.
* `changed` ([UInt8](/fr/reference/data-types/index)) — 1 si le paramètre a été explicitement défini dans la configuration ou explicitement modifié.
* `description` ([String](/fr/reference/data-types/index)) — Description du paramètre.
* `min` ([Nullable(String)](/fr/reference/data-types/index)) — Valeur minimale du paramètre, si elle est définie via des contraintes. Si le paramètre n’a pas de valeur minimale, contient NULL.
* `max` ([Nullable(String)](/fr/reference/data-types/index)) — Valeur maximale du paramètre, si elle est définie via des contraintes. Si le paramètre n’a pas de valeur maximale, contient NULL.
* `disallowed_values` ([Array(String)](/fr/reference/data-types/index)) — Liste des valeurs non autorisées
* `readonly` ([UInt8](/fr/reference/data-types/index)) — Indique si l’utilisateur actuel peut modifier le paramètre : 0 — l’utilisateur actuel peut modifier le paramètre, 1 — l’utilisateur actuel ne peut pas modifier le paramètre.
* `type` ([String](/fr/reference/data-types/index)) — Type du paramètre (valeur de chaîne spécifique à l’implémentation).
* `is_obsolete` ([UInt8](/fr/reference/data-types/index)) — Indique si le paramètre est obsolète.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/fr/reference/data-types/index)) —
  Niveau de prise en charge de cette fonctionnalité. Les fonctionnalités de ClickHouse sont organisées en tiers, qui varient selon l’état actuel de leur
  développement et les attentes que l’on peut avoir quant à leur utilisation :

- PRODUCTION : la fonctionnalité est stable, son utilisation est sûre et elle ne présente pas de problèmes d’interaction avec d’autres fonctionnalités PRODUCTION.
- BETA : la fonctionnalité est stable et sûre. Le résultat de son utilisation avec d’autres fonctionnalités est inconnu et son bon fonctionnement n’est pas garanti. Les tests et les signalements sont les bienvenus.
- EXPERIMENTAL : la fonctionnalité est en cours de développement. Elle est destinée uniquement aux développeurs et aux passionnés de ClickHouse. Elle peut fonctionner ou non et être supprimée à tout moment.
- OBSOLETE : n’est plus prise en charge. Soit elle a déjà été supprimée, soit elle le sera dans de futures versions.

{/*AUTOGENERATED_END*/}
