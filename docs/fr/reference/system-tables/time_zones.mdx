<div id="description">
  ## Description
</div>

Contient la liste des fuseaux horaires pris en charge par le serveur ClickHouse. Cette liste peut varier selon la version de ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/fr/reference/data-types/index)) — Liste des fuseaux horaires pris en charge.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```
