<div id="description">
  ## Description
</div>

Cette table fournit des informations d’introspection combinées sur ZooKeeper, et les nœuds sont extraits de la configuration.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_cluster_name` ([String](/fr/reference/data-types/string)) — Nom du cluster ZooKeeper.
* `host` ([String](/fr/reference/data-types/string)) — Le nom d’hôte/l’IP du nœud ZooKeeper auquel ClickHouse s’est connecté.
* `port` ([UInt16](/fr/reference/data-types/int-uint)) — Le port du nœud ZooKeeper auquel ClickHouse s’est connecté.
* `index` ([Nullable(UInt8)](/fr/reference/data-types/int-uint)) — L’index du nœud ZooKeeper auquel ClickHouse s’est connecté. Cet index provient de la config ZooKeeper. En l’absence de connexion, cette colonne vaut NULL.
* `is_connected` ([Nullable(UInt8)](/fr/reference/data-types/int-uint)) — Indique si ZooKeeper est connecté ou non.
* `is_readonly` ([UInt8](/fr/reference/data-types/int-uint)) — Indique si la connexion est en lecture seule.
* `version` ([String](/fr/reference/data-types/string)) — La version de ZooKeeper.
* `avg_latency` ([UInt64](/fr/reference/data-types/int-uint)) — La latence moyenne.
* `max_latency` ([UInt64](/fr/reference/data-types/int-uint)) — La latence maximale.
* `min_latency` ([UInt64](/fr/reference/data-types/int-uint)) — La latence minimale.
* `packets_received` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de paquets reçus.
* `packets_sent` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de paquets envoyés.
* `outstanding_requests` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de requêtes en attente.
* `server_state` ([String](/fr/reference/data-types/string)) — L’état du serveur.
* `is_leader` ([UInt8](/fr/reference/data-types/int-uint)) — Indique si ce ZooKeeper est le leader.
* `znode_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de znodes.
* `watch_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de watches.
* `ephemerals_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de nœuds éphémères.
* `approximate_data_size` ([UInt64](/fr/reference/data-types/int-uint)) — La taille approximative des données.
* `followers` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de followers du leader. Ce champ n’est exposé que par le leader.
* `synced_followers` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de followers synchronisés du leader. Ce champ n’est exposé que par le leader.
* `pending_syncs` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de synchronisations en attente du leader. Ce champ n’est exposé que par le leader.
* `open_file_descriptor_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de descripteurs de fichiers ouverts. Disponible uniquement sur les plateformes Unix.
* `max_file_descriptor_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre maximal de descripteurs de fichiers. Disponible uniquement sur les plateformes Unix.
* `connections` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de connexions à ZooKeeper.
* `outstanding` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre d’opérations ZooKeeper en attente.
* `zxid` ([UInt64](/fr/reference/data-types/int-uint)) — Le zxid de ZooKeeper.
* `node_count` ([UInt64](/fr/reference/data-types/int-uint)) — Le nombre de nœuds ZooKeeper.
* `snapshot_dir_size` ([UInt64](/fr/reference/data-types/int-uint)) — La taille du répertoire des instantanés de ZooKeeper.
* `log_dir_size` ([UInt64](/fr/reference/data-types/int-uint)) — La taille du répertoire des logs de ZooKeeper.
* `first_log_idx` ([UInt64](/fr/reference/data-types/int-uint)) — Le premier index de log de ZooKeeper.
* `first_log_term` ([UInt64](/fr/reference/data-types/int-uint)) — Le premier terme de log de ZooKeeper.
* `last_log_idx` ([UInt64](/fr/reference/data-types/int-uint)) — Le dernier index de log de ZooKeeper.
* `last_log_term` ([UInt64](/fr/reference/data-types/int-uint)) — Le dernier terme de log de ZooKeeper.
* `last_committed_idx` ([UInt64](/fr/reference/data-types/int-uint)) — Le dernier index validé de ZooKeeper.
* `leader_committed_log_idx` ([UInt64](/fr/reference/data-types/int-uint)) — L’index de log validé du leader de ZooKeeper.
* `target_committed_log_idx` ([UInt64](/fr/reference/data-types/int-uint)) — L’index de log validé cible de ZooKeeper.
* `last_snapshot_idx` ([UInt64](/fr/reference/data-types/int-uint)) — Le dernier index d’instantané de ZooKeeper.
