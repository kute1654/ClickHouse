<div id="description">
  ## Description
</div>

Contient des informations sur les bases de données disponibles pour l’utilisateur actuel.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `engine` ([String](/fr/reference/data-types/index)) — Moteur de la base de données.
* `data_path` ([String](/fr/reference/data-types/index)) — Chemin d’accès aux données.
* `metadata_path` ([String](/fr/reference/data-types/index)) — Chemin d’accès aux métadonnées.
* `uuid` ([UUID](/fr/reference/data-types/index)) — UUID de la base de données.
* `engine_full` ([String](/fr/reference/data-types/index)) — Paramètres du moteur de la base de données.
* `comment` ([String](/fr/reference/data-types/index)) — Commentaire de la base de données.
* `is_external` ([UInt8](/fr/reference/data-types/index)) — La base de données est externe (c.-à-d. PostgreSQL/DataLakeCatalog).

{/*AUTOGENERATED_END*/}

La colonne `name` de cette table système sert à implémenter la requête `SHOW DATABASES`.

<div id="example">
  ## Exemple
</div>

Créez une base de données.

```sql title="Query"
CREATE DATABASE test;
```

Vérifiez toutes les bases de données accessibles à l’utilisateur.

```sql title="Query"
SELECT * FROM system.databases;
```
