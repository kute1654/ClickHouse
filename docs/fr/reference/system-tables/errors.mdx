<div id="description">
  ## Description
</div>

Contient les codes d’erreur et le nombre de fois où ils ont été déclenchés.

Pour afficher tous les codes d’erreur possibles, y compris ceux qui n’ont pas été déclenchés, définissez le paramètre [system&#95;events&#95;show&#95;zero&#95;values](/fr/reference/settings/session-settings#system_events_show_zero_values) sur 1.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Nom de l’erreur (errorCodeToName).
* `code` ([Int32](/fr/reference/data-types/index)) — Numéro de code de l’erreur.
* `value` ([UInt64](/fr/reference/data-types/index)) — Nombre de fois où cette erreur s’est produite.
* `last_error_time` ([DateTime](/fr/reference/data-types/index)) — Heure à laquelle la dernière erreur s’est produite.
* `last_error_message` ([String](/fr/reference/data-types/index)) — Message de la dernière erreur.
* `last_error_format_string` ([String](/fr/reference/data-types/index)) — Chaîne de format de la dernière erreur.
* `last_error_trace` ([Array(UInt64)](/fr/reference/data-types/index)) — Une trace de pile représentant une liste d’adresses physiques où les méthodes appelées sont stockées.
* `remote` ([UInt8](/fr/reference/data-types/index)) — Exception distante (c.-à-d. reçue lors de l’une des requêtes distribuées).
* `query_id` ([String](/fr/reference/data-types/index)) — ID d’une requête ayant provoqué une erreur (si disponible).

{/*AUTOGENERATED_END*/}

<Note>
  Les compteurs de certaines erreurs peuvent augmenter même lorsqu’une requête s’exécute correctement. Il n’est pas recommandé d’utiliser cette table pour la surveillance du serveur, sauf si vous êtes certain que l’erreur correspondante ne peut pas être un faux positif.
</Note>

<div id="example">
  ## Exemple
</div>

```sql title="Query"
SELECT name, code, value
FROM system.errors
WHERE value > 0
ORDER BY code ASC
LIMIT 1

┌─name─────────────┬─code─┬─value─┐
│ CANNOT_OPEN_FILE │   76 │     1 │
└──────────────────┴──────┴───────┘
```

```sql title="Response"
WITH arrayMap(x -> demangle(addressToSymbol(x)), last_error_trace) AS all
SELECT name, arrayStringConcat(all, '\n') AS res
FROM system.errors
LIMIT 1
SETTINGS allow_introspection_functions=1\G
```
