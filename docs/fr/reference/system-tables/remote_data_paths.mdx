<div id="description">
  ## Description
</div>

Contient des informations sur les fichiers de données stockés sur des disques distants (par exemple S3, Azure Blob Storage), notamment la correspondance entre les chemins de métadonnées locaux et les chemins des blobs distants.

Chaque ligne représente un objet blob distant associé à un fichier de données.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `disk_name` ([String](/fr/reference/data-types/index)) — Nom du disque.
* `path` ([String](/fr/reference/data-types/index)) — Chemin du disque.
* `cache_base_path` ([String](/fr/reference/data-types/index)) — Répertoire de base des fichiers de cache.
* `local_path` ([String](/fr/reference/data-types/index)) — Chemin du fichier ClickHouse, également utilisé comme chemin des métadonnées.
* `remote_path` ([String](/fr/reference/data-types/index)) — Chemin du blob dans le stockage d’objets, auquel le fichier ClickHouse est associé.
* `size` ([UInt64](/fr/reference/data-types/index)) — Taille du fichier (compressé).
* `common_prefix_for_blobs` ([String](/fr/reference/data-types/index)) — Préfixe commun des blobs dans le stockage d’objets.
* `cache_paths` ([Array(String)](/fr/reference/data-types/index)) — Fichiers de cache du blob correspondant.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
Row 1:
──────
disk_name:              s3
path:                   /var/lib/clickhouse/disks/s3/
cache_base_path:        /var/lib/clickhouse/disks/s3_cache/
local_path:             store/123/1234abcd-1234-1234-1234-1234abcd1234/all_0_0_0/data.bin
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
