<div id="description">
  ## Description
</div>

Contient des informations sur les certificats disponibles et leur source.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/fr/reference/data-types/index)) — Version du certificat. Les valeurs sont 0 pour v1, 1 pour v2 et 2 pour v3.
* `serial_number` ([Nullable(String)](/fr/reference/data-types/index)) — Numéro de série du certificat attribué par l’émetteur.
* `signature_algo` ([Nullable(String)](/fr/reference/data-types/index)) — Algorithme de signature — algorithme utilisé par l’émetteur pour signer ce certificat.
* `issuer` ([Nullable(String)](/fr/reference/data-types/index)) — Émetteur — identifiant unique de l’autorité de certification qui émet ce certificat.
* `not_before` ([Nullable(String)](/fr/reference/data-types/index)) — Début de la période pendant laquelle ce certificat est valide.
* `not_after` ([Nullable(String)](/fr/reference/data-types/index)) — Fin de la période pendant laquelle ce certificat est valide.
* `subject` ([Nullable(String)](/fr/reference/data-types/index)) — Sujet — identifie le propriétaire de la clé publique.
* `pkey_algo` ([Nullable(String)](/fr/reference/data-types/index)) — Algorithme de clé publique — définit l’algorithme avec lequel la clé publique peut être utilisée.
* `path` ([String](/fr/reference/data-types/index)) — Chemin vers le fichier ou le répertoire contenant ce certificat.
* `default` ([UInt8](/fr/reference/data-types/index)) — Le certificat se trouve à l’emplacement par défaut des certificats.
* `protocol` ([String](/fr/reference/data-types/index)) — Nom du protocole si le certificat provient d’une configuration TLS spécifique à un protocole, sinon chaîne vide.

{/*AUTOGENERATED_END*/}
