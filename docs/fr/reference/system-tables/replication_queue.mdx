<div id="description">
  ## Description
</div>

Contient des informations sur les tâches issues des files de réplication stockées dans ClickHouse Keeper ou ZooKeeper, pour les tables de la famille `ReplicatedMergeTree`.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `replica_name` ([String](/fr/reference/data-types/index)) — Nom de la réplique dans ClickHouse Keeper. Les différentes répliques d&#39;une même table portent des noms différents.
* `position` ([UInt32](/fr/reference/data-types/index)) — Position de la tâche dans la file d&#39;attente.
* `node_name` ([String](/fr/reference/data-types/index)) — Nom du nœud dans ClickHouse Keeper.
* `type` ([String](/fr/reference/data-types/index)) — Type de la tâche dans la file d&#39;attente, parmi les suivants : • GET&#95;PART — Récupérer la part depuis une autre réplique, • ATTACH&#95;PART — Attacher la part, éventuellement à partir de notre propre réplique (si elle est trouvée dans le dossier detached). Vous pouvez le voir comme un GET&#95;PART avec quelques optimisations, car ils sont presque identiques, • MERGE&#95;PARTS — Fusionner les parts, • DROP&#95;RANGE — Supprimer les parts de la partition spécifiée dans la plage de numéros indiquée. • CLEAR&#95;COLUMN — REMARQUE : obsolète. Supprimer une colonne spécifique de la partition indiquée. • CLEAR&#95;INDEX — REMARQUE : obsolète. Supprimer un index spécifique de la partition indiquée. • REPLACE&#95;RANGE — Supprimer une certaine plage de parts et la remplacer par de nouvelles. • MUTATE&#95;PART — Appliquer une ou plusieurs mutations à la part. • ALTER&#95;METADATA — Appliquer une modification ALTER selon les paths globaux /metadata et /columns.
* `create_time` ([DateTime](/fr/reference/data-types/index)) — Date et heure auxquelles la tâche a été soumise pour exécution.
* `required_quorum` ([UInt32](/fr/reference/data-types/index)) — Nombre de répliques en attente de l&#39;achèvement de la tâche avec confirmation de sa fin. Cette colonne n&#39;est pertinente que pour la tâche GET&#95;PARTS.
* `source_replica` ([String](/fr/reference/data-types/index)) — Nom de la réplique source.
* `new_part_name` ([String](/fr/reference/data-types/index)) — Nom de la nouvelle part.
* `parts_to_merge` ([Array(String)](/fr/reference/data-types/index)) — Noms des parts à fusionner ou à mettre à jour.
* `is_detach` ([UInt8](/fr/reference/data-types/index)) — Ce flag indique si la tâche DETACH&#95;PARTS se trouve dans la file d&#39;attente.
* `is_currently_executing` ([UInt8](/fr/reference/data-types/index)) — Ce flag indique si une tâche spécifique est en cours d&#39;exécution.
* `num_tries` ([UInt32](/fr/reference/data-types/index)) — Nombre de tentatives échouées pour terminer la tâche.
* `last_exception` ([String](/fr/reference/data-types/index)) — Message texte concernant la dernière erreur survenue (le cas échéant).
* `last_exception_time` ([DateTime](/fr/reference/data-types/index)) — Date et heure de la dernière erreur survenue.
* `last_attempt_time` ([DateTime](/fr/reference/data-types/index)) — Date et heure de la dernière tentative d&#39;exécution de la tâche.
* `num_postponed` ([UInt32](/fr/reference/data-types/index)) — Nombre de fois où l&#39;action a été reportée.
* `postpone_reason` ([String](/fr/reference/data-types/index)) — Raison pour laquelle la tâche a été reportée.
* `last_postpone_time` ([DateTime](/fr/reference/data-types/index)) — Date et heure du dernier report de la tâche.
* `merge_type` ([String](/fr/reference/data-types/index)) — Type de la fusion en cours. Vide s&#39;il s&#39;agit d&#39;une mutation.

{/*AUTOGENERATED_END*/}
