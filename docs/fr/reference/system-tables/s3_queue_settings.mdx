<div id="description">
  ## Description
</div>

Contient des informations sur les paramètres des tables [S3Queue](/fr/reference/engines/table-engines/integrations/s3queue). Disponible à partir de la version `24.10` du serveur.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Base de données de la table utilisant le moteur S3Queue.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table utilisant le moteur S3Queue.
* `name` ([String](/fr/reference/data-types/index)) — Nom du paramètre.
* `value` ([String](/fr/reference/data-types/index)) — Valeur du paramètre.
* `type` ([String](/fr/reference/data-types/index)) — Type du paramètre (valeur de chaîne propre à l’implémentation).
* `changed` ([UInt8](/fr/reference/data-types/index)) — 1 si le paramètre a été explicitement défini dans la config ou explicitement modifié.
* `description` ([String](/fr/reference/data-types/index)) — Description du paramètre.
* `alterable` ([UInt8](/fr/reference/data-types/index)) — Indique si l’utilisateur actuel peut modifier le paramètre via ALTER TABLE MODIFY SETTING : 0 — L’utilisateur actuel ne peut pas modifier le paramètre, 1 — L’utilisateur actuel peut modifier le paramètre.

{/*AUTOGENERATED_END*/}
