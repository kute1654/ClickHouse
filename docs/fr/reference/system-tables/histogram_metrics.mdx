<div id="description">
  ## Description
</div>

Cette table contient des métriques d’histogramme qui peuvent être calculées instantanément et exportées au format Prometheus. Elle est toujours à jour. Elle remplace `system.latency_log`, désormais obsolète.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/fr/reference/data-types/index)) — Nom de la métrique.
* `value` ([Float64](/fr/reference/data-types/index)) — Valeur de la métrique.
* `description` ([String](/fr/reference/data-types/index)) — Description de la métrique.
* `labels` ([Map(String, String)](/fr/reference/data-types/index)) — Labels de métrique.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemple
</div>

Vous pouvez utiliser une requête comme celle-ci pour exporter toutes les métriques sous forme d’histogramme au format Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
