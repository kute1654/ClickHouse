Contient des informations sur toutes les transactions exécutées sur le serveur actuel.

Cette table peut être tronquée ou supprimée à tout moment.

Colonnes :

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/fr/reference/data-types/lowcardinality)) — Le nom d’hôte sur lequel la transaction a été exécutée.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/fr/reference/data-types/enum)) — Le type de la transaction. Valeurs possibles : Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/fr/reference/data-types/date)) — Date de l’entrée.
* `event_time` ([DateTime64(6)](/fr/reference/data-types/datetime64)) — Heure de l’entrée
* `thread_id` ([UInt64](/fr/reference/data-types/int-uint)) — L’identifiant d’un thread.
* `query_id` ([String](/fr/reference/data-types/string)) — L’ID d’une requête exécutée dans le cadre d’une transaction.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/fr/reference/data-types/tuple)) — L’identifiant d’une transaction.
* `tid_hash` ([UInt64](/fr/reference/data-types/int-uint)) — Le hachage de l’identifiant.
* `csn` ([UInt64](/fr/reference/data-types/int-uint)) — Le numéro de séquence de commit
* `database` ([String](/fr/reference/data-types/string)) — Le nom de la base de données sur laquelle la transaction a été exécutée.
* `table` ([String](/fr/reference/data-types/string)) — Le nom de la table sur laquelle la transaction a été exécutée.
* `uuid` ([UUID](/fr/reference/data-types/uuid)) — L’UUID de la table sur laquelle la transaction a été exécutée.
* `part` ([String](/fr/reference/data-types/string)) — Le nom de la part impliquée dans la transaction.

{/*AUTOGENERATED_END*/}
