<div id="groupBitAnd">
  ## groupBitAnd
</div>

Introduit dans : v1.1.0

Applique un ET bit à bit à des séries de nombres.

**Syntaxe**

```sql
groupBitAnd(expr)
```

**Alias** : `BIT_AND`

**Arguments**

* `expr` — Expression de type `(U)Int*`. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Renvoie une valeur de type `(U)Int*`. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Exemples**

**Exemple d’opération ET bit à bit**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
