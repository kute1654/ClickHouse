## varPop {#varPop}

Introduit dans : v1.1.0

Calcule la variance de population.

La variance de population est calculée selon la formule suivante :

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

Où :
- $x$ est chaque valeur de la population
- $\bar{x}$ est la moyenne de la population
- $n$ est la taille de la population

<Note>
Cette fonction utilise un algorithme numériquement instable. Si vous avez besoin de [stabilité numérique](https://en.wikipedia.org/wiki/Numerical_stability) dans vos calculs, utilisez la fonction [`varPopStable`](/reference/functions/aggregate-functions/varPopStable). Elle est plus lente, mais produit une erreur de calcul plus faible.
</Note>
    

**Syntaxe**

```sql
varPop(x)
```

**Alias** : `VAR_POP`

**Arguments**

- `x` — Population de valeurs dont on souhaite calculer la variance. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la variance de population de `x`. [`Float64`](/reference/data-types/float)

**Exemples**

**Calcul de la variance de population**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
