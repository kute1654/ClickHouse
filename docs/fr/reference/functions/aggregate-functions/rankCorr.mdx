<div id="rankCorr">
  ## rankCorr
</div>

Introduit dans : v20.9.0

Calcule un coefficient de corrélation de rang.

Renvoie un coefficient de corrélation de rang pour les rangs de x et y. La valeur du coefficient de corrélation varie de -1 à +1. Si moins de deux arguments sont fournis, la fonction renvoie une exception. Une valeur proche de +1 indique une forte relation linéaire : lorsque l&#39;une des variables aléatoires augmente, l&#39;autre augmente également. Une valeur proche de -1 indique une forte relation linéaire : lorsque l&#39;une des variables aléatoires augmente, l&#39;autre diminue. Une valeur proche de 0, ou égale à 0, indique l&#39;absence de relation entre les deux variables aléatoires.

**Voir aussi**

* [Le coefficient de corrélation de rang de Spearman](https://en.wikipedia.org/wiki/Spearman%27s_rank_correlation_coefficient)

**Syntaxe**

```sql
rankCorr(x, y)
```

**Arguments**

* `x` — Valeur arbitraire. [`Float*`](/fr/reference/data-types/float)
* `y` — Valeur arbitraire. [`Float*`](/fr/reference/data-types/float)

**Valeur renvoyée**

Renvoie le coefficient de corrélation de rang entre les rangs de x et de y. La valeur varie de -1 à +1. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Corrélation parfaite**

```sql title=Query
SELECT rankCorr(number, number) FROM numbers(100);
```

```response title=Response
┌─rankCorr(number, number)─┐
│                        1 │
└──────────────────────────┘
```

**Relation non linéaire**

```sql title=Query
SELECT roundBankers(rankCorr(exp(number), sin(number)), 3) FROM numbers(100);
```

```response title=Response
┌─roundBankers(rankCorr(exp(number), sin(number)), 3)─┐
│                                              -0.037 │
└─────────────────────────────────────────────────────┘
```
