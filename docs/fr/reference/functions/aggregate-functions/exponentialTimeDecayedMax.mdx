<div id="exponentialTimeDecayedMax">
  ## exponentialTimeDecayedMax
</div>

Introduit dans : v21.12.0

Renvoie le maximum entre la moyenne mobile à lissage exponentiel calculée à l’indice temporel `t` et celle calculée à `t-1`.

**Syntaxe**

```sql
exponentialTimeDecayedMax(x)(value, timeunit)
```

**Paramètres**

* `x` — Période de demi-vie. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Arguments**

* `value` — Valeur. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `timeunit` — Unité de temps. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`DateTime64`](/fr/reference/data-types/datetime64)

**Valeur renvoyée**

Renvoie le maximum de la moyenne mobile pondérée à lissage exponentiel en `t` et `t-1`. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Utilisation de la fonction de fenêtre avec représentation visuelle**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 5, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedMax(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
