<div id="entropy">
  ## entropy
</div>

Disponible à partir de : v20.1.0

Calcule l’[entropie de Shannon](https://en.wikipedia.org/wiki/Entropy_\(information_theory\)) d’une colonne de valeurs.

**Syntaxe**

```sql
entropy(val)
```

**Arguments**

* `val` — Colonne de valeurs de n’importe quel type. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie l’entropie de Shannon. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Calcul basique de l’entropie**

```sql title=Query
CREATE TABLE entropy (`vals` UInt32,`strings` String)
ENGINE = Memory;

INSERT INTO entropy VALUES (1, 'A'), (1, 'A'), (1,'A'), (1,'A'), (2,'B'), (2,'B'), (2,'C'), (2,'D');

SELECT entropy(vals), entropy(strings) FROM entropy
```

```response title=Response
┌─entropy(vals)─┬─entropy(strings)─┐
│             1 │             1.75 │
└───────────────┴──────────────────┘
```
