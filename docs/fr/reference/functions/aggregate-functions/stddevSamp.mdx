<div id="stddevSamp">
  ## stddevSamp
</div>

Introduit dans : v1.1.0

Renvoie l’écart-type d’un échantillon d’une séquence de données numériques.
Le résultat est égal à la racine carrée de [`varSamp`](/fr/reference/functions/aggregate-functions/varSamp).

<Note>
  Cette fonction utilise un algorithme numériquement instable.
  Si vous avez besoin de [stabilité numérique](https://en.wikipedia.org/wiki/Numerical_stability) dans vos calculs, utilisez la fonction [`stddevSampStable`](/fr/reference/functions/aggregate-functions/stddevSampStable). Elle est plus lente, mais produit une erreur de calcul plus faible.
</Note>

**Syntaxe**

```sql
stddevSamp(x)
```

**Alias** : `STDDEV_SAMP`, `STDDEV`

**Arguments**

* `x` — Valeurs dont il faut calculer la racine carrée de la variance d’échantillon. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la racine carrée de la variance d’échantillon de `x`. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Calcul de l’écart-type d’échantillon**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSamp(population)
FROM test_data;
```

```response title=Response
┌─stddevSamp(population)─┐
│                      4 │
└────────────────────────┘
```
