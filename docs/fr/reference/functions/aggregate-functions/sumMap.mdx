<div id="sumMappedArrays">
  ## sumMappedArrays
</div>

Introduit dans : v1.1.0

Additionne un ou plusieurs tableaux `value` en fonction des clés indiquées dans le tableau `key`. Renvoyer un tuple de tableaux : les clés dans l’ordre trié, suivies des valeurs additionnées pour les clés correspondantes, sans dépassement.

<Note>
  * Transmettre un tuple de clés et de tableaux `value` revient au même que transmettre un tableau de clés et un tableau de valeurs.
  * Le nombre d’éléments dans `key` et dans tous les tableaux `value` doit être identique pour chaque ligne additionnée.
</Note>

**Syntaxe**

```sql
sumMappedArrays(key, value1 [, value2, ...])
sumMappedArrays(Tuple(key, value1 [, value2, ...]))
```

**Arguments**

* `key` — Tableau de clés. [`Array`](/fr/reference/data-types/array)
* `value1, value2, ...` — Tableaux de valeurs à additionner pour chaque clé. [`Array`](/fr/reference/data-types/array)

**Valeur renvoyée**

Renvoie un tuple de tableaux : le premier tableau contient les clés triées, puis des tableaux contenant les valeurs additionnées pour les clés correspondantes. [`Tuple`](/fr/reference/data-types/tuple)

**Exemples**

**Utilisation de base avec le type Nested**

```sql title=Query
CREATE TABLE sum_map(
    date Date,
    timeslot DateTime,
    statusMap Nested(
        status UInt16,
        requests UInt64
    ),
    statusMapTuple Tuple(Array(Int32), Array(Int32))
) ENGINE = Memory;

INSERT INTO sum_map VALUES
    ('2000-01-01', '2000-01-01 00:00:00', [1, 2, 3], [10, 10, 10], ([1, 2, 3], [10, 10, 10])),
    ('2000-01-01', '2000-01-01 00:00:00', [3, 4, 5], [10, 10, 10], ([3, 4, 5], [10, 10, 10])),
    ('2000-01-01', '2000-01-01 00:01:00', [4, 5, 6], [10, 10, 10], ([4, 5, 6], [10, 10, 10])),
    ('2000-01-01', '2000-01-01 00:01:00', [6, 7, 8], [10, 10, 10], ([6, 7, 8], [10, 10, 10]));

SELECT
    timeslot,
    sumMappedArrays(statusMap.status, statusMap.requests),
    sumMappedArrays(statusMapTuple)
FROM sum_map
GROUP BY timeslot;
```

```response title=Response
┌────────────timeslot─┬─sumMappedArrays(statusMap.status, statusMap.requests)─┬─sumMappedArrays(statusMapTuple)─────────┐
│ 2000-01-01 00:00:00 │ ([1,2,3,4,5],[10,10,20,10,10])                        │ ([1,2,3,4,5],[10,10,20,10,10])          │
│ 2000-01-01 00:01:00 │ ([4,5,6,7,8],[10,10,20,10,10])                        │ ([4,5,6,7,8],[10,10,20,10,10])          │
└─────────────────────┴───────────────────────────────────────────────────────┴─────────────────────────────────────────┘
```

**Exemple de tableaux multivaleurs**

```sql title=Query
CREATE TABLE multi_metrics(
    date Date,
    browser_metrics Nested(
        browser String,
        impressions UInt32,
        clicks UInt32
    )
)
ENGINE = Memory;

INSERT INTO multi_metrics VALUES
    ('2000-01-01', ['Firefox', 'Chrome'], [100, 200], [10, 25]),
    ('2000-01-01', ['Chrome', 'Safari'], [150, 50], [20, 5]),
    ('2000-01-01', ['Firefox', 'Edge'], [80, 40], [8, 4]);

SELECT
    sumMappedArrays(browser_metrics.browser, browser_metrics.impressions, browser_metrics.clicks) AS result
FROM multi_metrics;
```

```response title=Response
┌─result────────────────────────────────────────────────────────────────────────┐
│ (['Chrome', 'Edge', 'Firefox', 'Safari'], [350, 40, 180, 50], [45, 4, 18, 5]) │
└───────────────────────────────────────────────────────────────────────────────┘
-- In this example:
-- The result tuple contains three arrays
-- First array: keys (browser names) in sorted order
-- Second array: total impressions for each browser
-- Third array: total clicks for each browser
```
