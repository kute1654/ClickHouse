<div id="groupArraySample">
  ## groupArraySample
</div>

Introduit dans : v20.3.0

Crée un tableau contenant un échantillon de valeurs d’argument.
La taille du tableau résultant est limitée à `max_size` éléments.
Les valeurs d’argument sont sélectionnées aléatoirement et ajoutées au tableau.

**Syntaxe**

```sql
groupArraySample(max_size[, seed])(x)
```

**Paramètres**

* `max_size` — Taille maximale du tableau résultant. [`UInt64`](/fr/reference/data-types/int-uint)
* `seed` — Facultatif. Graine du générateur de nombres aléatoires. Valeur par défaut : 123456. [`UInt64`](/fr/reference/data-types/int-uint)
* `x` — Argument (nom de colonne ou expression). [`Any`](/fr/reference/data-types/index)

**Arguments**

* `array_column` — Colonne contenant les tableaux à agréger. [`Array`](/fr/reference/data-types/array)

**Valeur retournée**

Tableau de valeurs `x` sélectionnées aléatoirement. [`Array(T)`](/fr/reference/data-types/array)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
CREATE TABLE default.colors (
    id Int32,
    color String
) ENGINE = Memory;

INSERT INTO default.colors VALUES
(1, 'red'),
(2, 'blue'),
(3, 'green'),
(4, 'white'),
(5, 'orange');

SELECT groupArraySample(3)(color) as newcolors FROM default.colors;
```

```response title=Response
┌─newcolors──────────────────┐
│ ['white','blue','green']   │
└────────────────────────────┘
```

**Exemple avec un seed**

```sql title=Query
-- Query with column name and different seed
SELECT groupArraySample(3, 987654321)(color) as newcolors FROM default.colors;
```

```response title=Response
┌─newcolors──────────────────┐
│ ['red','orange','green']   │
└────────────────────────────┘
```

**Utiliser une expression comme argument**

```sql title=Query
-- Query with expression as argument
SELECT groupArraySample(3)(concat('light-', color)) as newcolors FROM default.colors;
```

```response title=Response
┌─newcolors───────────────────────────────────┐
│ ['light-blue','light-orange','light-green'] │
└─────────────────────────────────────────────┘
```
