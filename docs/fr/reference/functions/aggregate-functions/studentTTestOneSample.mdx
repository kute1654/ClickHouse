<div id="studentTTestOneSample">
  ## studentTTestOneSample
</div>

Introduit dans : v25.10.0

Applique le test t de Student à un échantillon pour déterminer si la moyenne d’un échantillon diffère d’une moyenne de population connue.

On suppose une distribution normale. L’hypothèse nulle est que la moyenne de l’échantillon est égale à la moyenne de la population.

Le paramètre facultatif `confidence_level` permet de calculer l’intervalle de confiance.

**Remarques :**

* Au moins 2 observations sont requises ; sinon, le résultat est `(nan, nan)` (et les intervalles, s’ils sont demandés, valent `nan`).
* Une entrée constante ou quasi constante renvoie également `nan` en raison d’une erreur-type nulle (ou pratiquement nulle).

**Voir aussi**

* [Test t de Student](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [fonction studentTTest](/fr/reference/functions/aggregate-functions/studentTTest)

**Syntaxe**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**Paramètres**

* `confidence_level` — Facultatif. Niveau de confiance pour les intervalles de confiance. Float dans l’intervalle (0, 1). [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `sample_data` — Données de l’échantillon. [`Integer`](/fr/reference/data-types/int-uint) ou [`Float`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `population_mean` — Moyenne de la population connue par rapport à laquelle effectuer le test (généralement une constante). [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie un tuple comportant deux ou quatre éléments (si `confidence_level` est spécifié) : statistique t calculée, valeur p calculée (bilatérale), [borne inférieure calculée de l’intervalle de confiance], [borne supérieure calculée de l’intervalle de confiance]. Les intervalles de confiance portent sur la moyenne de l’échantillon pour le niveau de confiance donné. [`Tuple(Float64, Float64)`](/fr/reference/data-types/tuple) ou [`Tuple(Float64, Float64, Float64, Float64)`](/fr/reference/data-types/tuple)

**Exemples**

**Sans intervalle de confiance**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
