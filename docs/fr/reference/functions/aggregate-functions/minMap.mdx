<div id="minMappedArrays">
  ## minMappedArrays
</div>

Introduit dans : v20.5.0

Calcule le minimum du tableau `value` en fonction des clés spécifiées dans le tableau `key`.

<Note>
  * Passer un tuple de clés et de tableaux de valeurs revient au même que passer un tableau de clés et un tableau de valeurs.
  * Le nombre d&#39;éléments dans `key` et `value` doit être identique pour chaque ligne agrégée.
</Note>

**Syntaxe**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**Arguments**

* `key` — Tableau de clés. [`Array(T)`](/fr/reference/data-types/array)
* `value` — Tableau de valeurs. [`Array(T)`](/fr/reference/data-types/array)

**Valeur renvoyée**

Renvoie un tuple de deux tableaux : les clés dans l’ordre de tri et les valeurs calculées pour les clés correspondantes. [`Tuple(Array(T), Array(T))`](/fr/reference/data-types/tuple)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
