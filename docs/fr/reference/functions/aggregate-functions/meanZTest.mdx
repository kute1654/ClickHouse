<div id="meanZTest">
  ## meanZTest
</div>

Introduit dans : v22.2.0

Applique un test z des moyennes à des échantillons provenant de deux populations.

Les valeurs des deux échantillons se trouvent dans la colonne `sample_data`.
Si `sample_index` est égal à 0, la valeur de cette ligne appartient à l’échantillon de la première population.
Sinon, elle appartient à l’échantillon de la seconde population.
L’hypothèse nulle est que les moyennes des populations sont égales.
On suppose une distribution normale.
Les populations peuvent avoir des variances inégales, et ces variances sont connues.

**Syntaxe**

```sql
meanZTest(population_variance_x, population_variance_y, confidence_level)(sample_data, sample_index)
```

**Paramètres**

* `population_variance_x` — Variance de la population x. [`Float*`](/fr/reference/data-types/float)
* `population_variance_y` — Variance de la population y. [`Float*`](/fr/reference/data-types/float)
* `confidence_level` — Niveau de confiance utilisé pour calculer les intervalles de confiance. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `sample_data` — Données de l’échantillon. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `sample_index` — Indice de l’échantillon. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Renvoie un tuple de quatre éléments : la statistique z calculée, la valeur p calculée, la borne inférieure calculée de l’intervalle de confiance et la borne supérieure calculée de l’intervalle de confiance. [`Tuple(Float64, Float64, Float64, Float64)`](/fr/reference/data-types/tuple)

**Exemples**

**Exemple de test z des moyennes**

```sql title=Query
CREATE TABLE mean_ztest (sample_data Float64, sample_index UInt8) ENGINE = Memory;
INSERT INTO mean_ztest VALUES (20.3, 0), (21.9, 0), (22.1, 0), (18.9, 1), (19, 1), (20.3, 1);

SELECT meanZTest(0.7, 0.45, 0.95)(sample_data, sample_index) FROM mean_ztest;
```

```response title=Response
└─────────────────────────────────────────────────────────────────────────────────────┘
```
