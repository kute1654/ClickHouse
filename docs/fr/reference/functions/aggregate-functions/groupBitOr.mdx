<div id="groupBitOr">
  ## groupBitOr
</div>

Introduit dans : v1.1.0

Applique l’opérateur OU bit à bit à une série de nombres.

**Syntaxe**

```sql
groupBitOr(expr)
```

**Alias** : `BIT_OR`

**Arguments**

* `expr` — Expression de type `(U)Int*`. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Renvoie une valeur de type `(U)Int*`. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Exemples**

**Exemple d’opération OR bit à bit**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
