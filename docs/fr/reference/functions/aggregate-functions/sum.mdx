<div id="sum">
  ## sum
</div>

Introduit dans : v1.1.0

Calcule la somme de valeurs numériques.

**Syntaxe**

```sql
sum(num)
```

**Arguments**

* `num` — Colonne de valeurs numériques. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la somme des valeurs. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal)

**Exemples**

**Calcul de la somme des salaires des employés**

```sql title=Query
CREATE TABLE employees
(
    id UInt32,
    name String,
    salary UInt32
)
ENGINE = Memory;

INSERT INTO employees VALUES
    (87432, 'John Smith', 45680),
    (59018, 'Jane Smith', 72350),
    (20376, 'Ivan Ivanovich', 58900),
    (71245, 'Anastasia Ivanovna', 89210);

SELECT sum(salary) FROM employees;
```

```response title=Response
┌─sum(salary)─┐
│      266140 │
└─────────────┘
```
