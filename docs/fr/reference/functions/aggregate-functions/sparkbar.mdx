<div id="sparkbar">
  ## sparkbar
</div>

Introduit dans : v21.11.0

La fonction trace un histogramme de fréquence pour les valeurs `x` et le taux de répétition `y` associé à ces valeurs sur l’intervalle `[min_x, max_x]`.
Le nombre d’occurrences de toutes les valeurs `x` appartenant au même intervalle est moyenné ; les données doivent donc être pré-agrégées.
Les occurrences négatives sont ignorées.

Si aucun intervalle n’est spécifié, la valeur minimale de `x` est utilisée comme début de l’intervalle et la valeur maximale de `x` comme fin de l’intervalle.
Sinon, les valeurs situées en dehors de l’intervalle sont ignorées.

**Syntaxe**

```sql
sparkbar(buckets[, min_x, max_x])(x, y)
```

**Alias** : `sparkBar`

**Paramètres**

* `buckets` — Le nombre de segments. [`(U)Int*`](/fr/reference/data-types/int-uint)
* `min_x` — Facultatif. Le début de l’intervalle. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `max_x` — Facultatif. La fin de l’intervalle. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Arguments**

* `x` — Le champ contenant les valeurs. [`const String`](/fr/reference/data-types/string)
* `y` — Le champ contenant la fréquence des valeurs. [`const String`](/fr/reference/data-types/string)

**Valeur renvoyée**

Renvoie l’histogramme de fréquence. [`String`](/fr/reference/data-types/string)

**Exemples**

**Sans préciser l’intervalle**

```sql title=Query
CREATE TABLE spark_bar_data (`value` Int64, `event_date` Date) ENGINE = MergeTree ORDER BY event_date;

INSERT INTO spark_bar_data VALUES (1,'2020-01-01'), (3,'2020-01-02'), (4,'2020-01-02'), (-3,'2020-01-02'), (5,'2020-01-03'), (2,'2020-01-04'), (3,'2020-01-05'), (7,'2020-01-06'), (6,'2020-01-07'), (8,'2020-01-08'), (2,'2020-01-11');

SELECT sparkbar(9)(event_date, cnt) FROM (SELECT sum(value) AS cnt, event_date FROM spark_bar_data GROUP BY event_date);
```

```response title=Response
┌─sparkbar(9)(event_date, cnt)─┐
│ ▂▅▂▃▆█  ▂                    │
└──────────────────────────────┘
```

**Avec spécification de l’intervalle**

```sql title=Query
SELECT sparkbar(9, toDate('2020-01-01'), toDate('2020-01-10'))(event_date, cnt) FROM (SELECT sum(value) AS cnt, event_date FROM spark_bar_data GROUP BY event_date);
```

```response title=Response
┌─sparkbar(9, toDate('2020-01-01'), toDate('2020-01-10'))(event_date, cnt)─┐
│ ▂▅▂▃▇▆█                                                                  │
└──────────────────────────────────────────────────────────────────────────┘
```
