<div id="groupUniqArray">
  ## groupUniqArray
</div>

Introduit dans : v1.1.0

Crée un tableau à partir de valeurs d’argument distinctes.
La consommation de mémoire de cette fonction est la même que celle de la fonction [`uniqExact`](/fr/reference/functions/aggregate-functions/uniqExact).

**Syntaxe**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**Paramètres**

* `max_size` — Limite la taille du tableau résultant à `max_size` éléments. `groupUniqArray(1)(x)` équivaut à `[any(x)]`. [`UInt64`](/fr/reference/data-types/int-uint)

**Arguments**

* `x` — Expression. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie un tableau de valeurs uniques. [`Array`](/fr/reference/data-types/array)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**Avec le paramètre max&#95;size**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
