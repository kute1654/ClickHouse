<div id="anyLast">
  ## anyLast
</div>

Introduit dans : v1.1.0

Sélectionne la dernière valeur rencontrée dans une colonne.

<Warning>
  Comme une query peut être exécutée dans un ordre arbitraire, le résultat de cette fonction est non déterministe.
  Si vous avez besoin d’un résultat arbitraire mais déterministe, utilisez les fonctions [min](/fr/reference/functions/aggregate-functions/min) ou [max](/fr/reference/functions/aggregate-functions/max).
</Warning>

Par défaut, la fonction ne renvoie jamais NULL, c’est-à-dire qu’elle ignore les valeurs NULL dans la colonne d’entrée.
Cependant, si la fonction est utilisée avec le modificateur `RESPECT NULLS`, elle renvoie la dernière valeur lue, qu’elle soit NULL ou non.

**Syntaxe**

```sql
anyLast(column) [RESPECT NULLS]
```

**Alias** : `last_value`

**Arguments**

* `column` — Le nom de la colonne. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie la dernière valeur rencontrée. [`Any`](/fr/reference/data-types/index)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
CREATE TABLE tab(city Nullable(String)) ENGINE=Memory;
INSERT INTO tab (city) VALUES ('Amsterdam'), (NULL), ('New York'), ('Tokyo'), ('Valencia'), (NULL);
SELECT anyLast(city), anyLastRespectNulls(city) FROM tab;
```

```response title=Response
┌─anyLast(city)─┬─anyLastRespectNulls(city)─┐
│ Valencia      │ ᴺᵁᴸᴸ                      │
└───────────────┴───────────────────────────┘
```
