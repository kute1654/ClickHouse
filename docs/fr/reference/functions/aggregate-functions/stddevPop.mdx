<div id="stddevPop">
  ## stddevPop
</div>

Introduit dans : v1.1.0

Renvoie l’écart-type de la population d’une séquence de données numériques.
Le résultat est égal à la racine carrée de [`varPop`](/fr/reference/functions/aggregate-functions/varPop).

<Note>
  Cette fonction utilise un algorithme numériquement instable. Si vous avez besoin d’une [stabilité numérique](https://en.wikipedia.org/wiki/Numerical_stability) dans vos calculs, utilisez la fonction [`stddevPopStable`](/fr/reference/functions/aggregate-functions/stddevPopStable). Elle est plus lente, mais produit une erreur de calcul plus faible.
</Note>

**Syntaxe**

```sql
stddevPop(x)
```

**Alias** : `STD`, `STDDEV_POP`

**Arguments**

* `x` — Ensemble de valeurs dont il faut calculer l’écart type. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la racine carrée de la variance de la population `x`. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Calcul de l’écart type de la population**

```sql title=Query
CREATE TABLE test_data (population UInt8) ENGINE = Log;
INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT stddevPop(population) AS stddev FROM test_data;
```

```response title=Response
┌────────────stddev─┐
│ 3.794733192202055 │
└───────────────────┘
```
