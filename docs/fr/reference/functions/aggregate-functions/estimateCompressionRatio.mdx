<div id="estimateCompressionRatio">
  ## estimateCompressionRatio
</div>

Introduit dans : v25.4.0

Estime le ratio de compression d&#39;une colonne donnée sans la compresser.

<Note>
  Pour les exemples ci-dessous, le résultat varie selon le codec de compression par défaut du serveur.
  Voir [Codecs de compression des colonnes](/fr/reference/statements/create/table#column_compression_codec).
</Note>

**Syntaxe**

```sql
estimateCompressionRatio([codec, block_size_bytes])(column)
```

**Paramètres**

* `codec` — Chaîne contenant un codec de compression ou plusieurs codecs séparés par des virgules dans une seule chaîne. [`String`](/fr/reference/data-types/string)
* `block_size_bytes` — Taille de bloc des données compressées. Cela revient à définir à la fois [`max_compress_block_size`](/fr/reference/settings/merge-tree-settings#max_compress_block_size) et [`min_compress_block_size`](/fr/reference/settings/merge-tree-settings#min_compress_block_size). La valeur par défaut est de 1 MiB (1048576 octets). La valeur maximale autorisée est de 256 MiB (268435456 octets). [`UInt64`](/fr/reference/data-types/int-uint)

**Arguments**

* `column` — Colonne de n’importe quel type. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie une estimation du ratio de compression pour la colonne donnée. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Utilisation de base avec le codec par défaut**

```sql title=Query
CREATE TABLE compression_estimate_example
(
    `number` UInt64
)
ENGINE = MergeTree()
ORDER BY number
SETTINGS min_bytes_for_wide_part = 0;

INSERT INTO compression_estimate_example
SELECT number FROM system.numbers LIMIT 100_000;

SELECT estimateCompressionRatio(number) AS estimate FROM compression_estimate_example
```

```response title=Response
┌───────────estimate─┐
│ 1.9988506608699999 │
└────────────────────┘
```

**Utiliser un codec spécifique**

```sql title=Query
SELECT estimateCompressionRatio('T64')(number) AS estimate FROM compression_estimate_example
```

```response title=Response
┌──────────estimate─┐
│ 3.762758101688538 │
└───────────────────┘
```

**Utiliser plusieurs codecs**

```sql title=Query
SELECT estimateCompressionRatio('T64, ZSTD')(number) AS estimate FROM compression_estimate_example
```

```response title=Response
┌───────────estimate─┐
│ 143.60078980434392 │
└────────────────────┘
```
