<div id="cramersV">
  ## cramersV
</div>

Introduit dans : v22.1.0

[Le V de Cramer](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V) (parfois aussi appelé phi de Cramer) est une mesure d’association entre deux colonnes d’une table.
Le résultat de la fonction `cramersV` varie de 0 (ce qui correspond à l’absence d’association entre les variables) à 1, et ne peut atteindre 1 que lorsque chaque valeur est entièrement déterminée par l’autre.
Il peut être interprété comme l’association entre deux variables, exprimée en pourcentage de leur variation maximale possible.

<Note>
  Pour une version du V de Cramer corrigée du biais, voir : [cramersVBiasCorrected](/fr/reference/functions/aggregate-functions/cramersVBiasCorrected)
</Note>

**Syntaxe**

```sql
cramersV(column1, column2)
```

**Arguments**

* `column1` — Première colonne à comparer. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `column2` — Deuxième colonne à comparer. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie une valeur comprise entre 0 (correspondant à l&#39;absence d&#39;association entre les valeurs des colonnes) et 1 (association complète). [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Aucune association entre les colonnes**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 3 AS a,
            number % 5 AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─cramersV(a, b)─┐
│              0 │
└────────────────┘
```

**Association élevée entre les colonnes**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            if (number % 12 = 0, (number + 1) % 5, number % 5) AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─────cramersV(a, b)─┐
│ 0.9066801892162646 │
└────────────────────┘
```
