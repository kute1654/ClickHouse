<div id="avg">
  ## avg
</div>

Introduit dans : v1.1.0

Calcule la moyenne arithmétique.

**Syntaxe**

```sql
avg(x)
```

**Arguments**

* `x` — Valeurs d’entrée. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la moyenne arithmétique ; renvoie `NaN` si le paramètre d’entrée `x` est vide. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Utilisation de base**

```sql title=Query
SELECT avg(x) FROM VALUES('x Int8', 0, 1, 2, 3, 4, 5);
```

```response title=Response
┌─avg(x)─┐
│    2.5 │
└────────┘
```

**Une table vide renvoie NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;

SELECT avg(t) FROM test;
```

```response title=Response
┌─avg(x)─┐
│    nan │
└────────┘
```
