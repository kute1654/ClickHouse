<div id="approx_top_k">
  ## approx_top_k
</div>

Introduit dans : v1.1.0

Renvoie un tableau contenant une approximation des valeurs les plus fréquentes et de leur nombre d’occurrences dans la colonne spécifiée.
Le tableau obtenu est trié par fréquence approximative décroissante des valeurs (et non par les valeurs elles-mêmes).

Cette fonction ne garantit pas le résultat.
Dans certaines situations, des erreurs peuvent se produire et elle peut renvoyer des valeurs fréquentes qui ne sont pas les plus fréquentes.

**Syntaxe**

```sql
approx_top_k(N[, reserved])(column)
```

**Alias** : `approx_top_count`

**Paramètres**

* `N` — Le nombre d&#39;éléments à renvoyer. Valeur par défaut : `10`. Valeur maximale : `N = 65536`. [`UInt64`](/fr/reference/data-types/int-uint)
* `reserved` — Facultatif. Définit le nombre de cellules réservées aux valeurs. Si `uniq(column) > reserved`, le résultat sera approximatif. Valeur par défaut : `N * 3`. [`UInt64`](/fr/reference/data-types/int-uint)

**Arguments**

* `column` — Le nom de la colonne pour laquelle rechercher les valeurs les plus fréquentes. [`String`](/fr/reference/data-types/string)

**Valeur renvoyée**

Renvoie un tableau contenant les valeurs approximativement les plus fréquentes et leur nombre d&#39;occurrences, triés par fréquence approximative décroissante. [`Array`](/fr/reference/data-types/array)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
SELECT approx_top_k(2)(k)
FROM VALUES('k Char, w UInt64', ('y', 1), ('y', 1), ('x', 5), ('y', 1), ('z', 10));
```

```response title=Response
┌─approx_top_k(2)(k)────┐
│ [('y',3,0),('x',1,0)] │
└───────────────────────┘
```
