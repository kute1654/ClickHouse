<div id="avgWeighted">
  ## avgWeighted
</div>

Introduite dans : v20.1.0

Calcule la [moyenne arithmétique pondérée](https://en.wikipedia.org/wiki/Weighted_arithmetic_mean).

**Syntaxe**

```sql
avgWeighted(x, weight)
```

**Arguments**

* `x` — Valeurs. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float)
* `weight` — Poids des valeurs. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float)

**Valeur renvoyée**

Renvoie `NaN` si tous les poids sont égaux à 0 ou si le paramètre `weight` fourni est vide ; sinon, renvoie la moyenne pondérée. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Poids mixtes entiers et flottants**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Float64', (4, 1), (1, 0), (10, 2))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│                 8 │
└───────────────────┘
```

**Si tous les poids sont nuls, renvoie NaN**

```sql title=Query
SELECT avgWeighted(x, w)
FROM VALUES('x Int8, w Int8', (0, 0), (1, 0), (10, 0))
```

```response title=Response
┌─avgWeighted(x, w)─┐
│               nan │
└───────────────────┘
```

**Une table vide renvoie NaN**

```sql title=Query
CREATE TABLE test (t UInt8) ENGINE = Memory;
SELECT avgWeighted(t, t) FROM test
```

```response title=Response
┌─avgWeighted(t, t)─┐
│               nan │
└───────────────────┘
```
