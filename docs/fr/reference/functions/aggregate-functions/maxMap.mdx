<div id="maxMappedArrays">
  ## maxMappedArrays
</div>

Introduit dans : v20.5.0

Calcule le maximum du tableau `value` en fonction des clés spécifiées dans le tableau `key`.

<Note>
  * Passer un tuple de clés et de tableaux de valeurs revient au même que passer un tableau de clés et un tableau de valeurs.
  * Le nombre d&#39;éléments de `key` et de `value` doit être identique pour chaque ligne totalisée.
</Note>

**Syntaxe**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**Arguments**

* `key` — Tableau de clés. [`Array(T)`](/fr/reference/data-types/array)
* `value` — Tableau de valeurs. [`Array(T)`](/fr/reference/data-types/array)

**Valeur renvoyée**

Renvoie un tuple de deux tableaux : les clés dans l’ordre trié et les valeurs calculées pour les clés correspondantes. [`Tuple(Array(T), Array(T))`](/fr/reference/data-types/tuple)

**Exemples**

**Exemple d’utilisation**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)───┐
│ (['x','y','z'],[2,3,1]) │
└─────────────────────────┘
```
