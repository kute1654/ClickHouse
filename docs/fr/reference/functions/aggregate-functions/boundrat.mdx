<div id="boundingRatio">
  ## boundingRatio
</div>

Introduit dans : v20.1.0

Calcule la pente entre les points situés le plus à gauche et le plus à droite dans un groupe de valeurs.

**Syntaxe**

```sql
boundingRatio(x, y)
```

**Arguments**

* `x` — Valeurs des coordonnées X. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `y` — Valeurs des coordonnées Y. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la pente de la droite entre les points situés le plus à gauche et le plus à droite, ou `NaN` si les données sont vides. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Exemple de données**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**Exemple d’utilisation**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
