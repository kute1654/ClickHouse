<div id="studentTTest">
  ## studentTTest
</div>

Introduit dans : v21.1.0

Applique le test t de Student à des échantillons provenant de deux populations.

Les valeurs des deux échantillons se trouvent dans la colonne `sample_data`. Si `sample_index` est égal à 0, la valeur de cette ligne appartient à l’échantillon de la première population. Sinon, elle appartient à l’échantillon de la seconde population.
L’hypothèse nulle est que les moyennes des populations sont égales. Une distribution normale avec des variances égales est supposée.

**Voir aussi**

* [test t de Student](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [fonction welchTTest](/fr/reference/functions/aggregate-functions/welchTTest)

**Syntaxe**

```sql
studentTTest([confidence_level])(sample_data, sample_index)
```

**Paramètres**

* `confidence_level` — Facultatif. Niveau de confiance pour calculer les intervalles de confiance. [`Float`](/fr/reference/data-types/float)

**Arguments**

* `sample_data` — Données de l’échantillon. [`Integer`](/fr/reference/data-types/int-uint) or [`Float`](/fr/reference/data-types/float) or [`Decimal`](/fr/reference/data-types/decimal)
* `sample_index` — Indice de l’échantillon. [`Integer`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Renvoie un tuple de deux ou quatre éléments (si le paramètre facultatif `confidence_level` est spécifié) : statistique t calculée, p-value calculée, [borne inférieure calculée de l’intervalle de confiance], [borne supérieure calculée de l’intervalle de confiance]. [`Tuple(Float64, Float64)`](/fr/reference/data-types/tuple) or [`Tuple(Float64, Float64, Float64, Float64)`](/fr/reference/data-types/tuple)

**Exemples**

**Utilisation de base**

```sql title=Query
SELECT studentTTest(sample_data, sample_index) FROM student_ttest;
```
