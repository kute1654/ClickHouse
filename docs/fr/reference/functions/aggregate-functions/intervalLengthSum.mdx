<div id="intervalLengthSum">
  ## intervalLengthSum
</div>

Introduit dans : v21.7.0

Prend plusieurs plages numériques et calcule leur longueur totale après fusion de toutes les parties qui se chevauchent en un seul intervalle.

<Note>
  Les arguments doivent être du même type de données.
  Sinon, une exception sera levée.
</Note>

**Syntaxe**

```sql
intervalLengthSum(start, end)
```

**Arguments**

* `start` — La valeur de début de l’intervalle. [`(U)Int32/64`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`Date`](/fr/reference/data-types/date)
* `end` — La valeur de fin de l’intervalle. [`(U)Int32/64`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`Date`](/fr/reference/data-types/date)

**Valeur renvoyée**

Renvoie la longueur totale de l’union de toutes les plages (segments sur l’axe numérique). Selon le type de l’argument, la valeur renvoyée peut être de type UInt64 ou Float64. [`UInt64`](/fr/reference/data-types/int-uint) ou [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Exemple Float32**

```sql title=Query
CREATE TABLE fl_interval (id String, start Float32, end Float32) ENGINE = Memory;
INSERT INTO fl_interval VALUES ('a', 1.1, 2.9), ('a', 2.5, 3.2), ('a', 4, 5);

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM fl_interval GROUP BY id ORDER BY id;
```

```response title=Response
┌─id─┬─intervalLengthSum(start, end)─┬─toTypeName(intervalLengthSum(start, end))─┐
│ a  │                           3.1 │ Float64                                   │
└────┴───────────────────────────────┴───────────────────────────────────────────┘
```

**Exemple de DateTime**

```sql title=Query
CREATE TABLE dt_interval (id String, start DateTime, end DateTime) ENGINE = Memory;
INSERT INTO dt_interval VALUES ('a', '2020-01-01 01:12:30', '2020-01-01 02:10:10'), ('a', '2020-01-01 02:05:30', '2020-01-01 02:50:31'), ('a', '2020-01-01 03:11:22', '2020-01-01 03:23:31');

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM dt_interval GROUP BY id ORDER BY id;
```

```response title=Response
┌─id─┬─intervalLengthSum(start, end)─┬─toTypeName(intervalLengthSum(start, end))─┐
│ a  │                          6610 │ UInt64                                    │
└────┴───────────────────────────────┴───────────────────────────────────────────┘
```

**Exemple de Date**

```sql title=Query
CREATE TABLE date_interval (id String, start Date, end Date) ENGINE = Memory;
INSERT INTO date_interval VALUES ('a', '2020-01-01', '2020-01-04'), ('a', '2020-01-12', '2020-01-18');

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM date_interval GROUP BY id ORDER BY id;
```

```response title=Response
┌─id─┬─intervalLengthSum(start, end)─┬─toTypeName(intervalLengthSum(start, end))─┐
│ a  │                             9 │ UInt64                                    │
└────┴───────────────────────────────┴───────────────────────────────────────────┘
```
