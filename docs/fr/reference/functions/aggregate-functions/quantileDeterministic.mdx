<div id="quantileDeterministic">
  ## quantileDeterministic
</div>

Introduit dans : v1.1.0

Calcule un [quantile](https://en.wikipedia.org/wiki/Quantile) approximatif d&#39;une séquence de données numériques.

Cette fonction applique un [échantillonnage par réservoir](https://en.wikipedia.org/wiki/Reservoir_sampling) avec un réservoir de taille maximale de 8192 et un algorithme d&#39;échantillonnage déterministe.
Le résultat est déterministe.
Pour obtenir un quantile exact, utilisez la fonction [`quantileExact`](/fr/reference/functions/aggregate-functions/quantileExact#quantileExact).

Lorsque plusieurs fonctions `quantile*` avec différents niveaux sont utilisées dans une requête, les états internes ne sont pas combinés (autrement dit, la requête est moins efficace qu&#39;elle pourrait l&#39;être).
Dans ce cas, utilisez la fonction [`quantiles`](/fr/reference/functions/aggregate-functions/quantiles#quantiles).

**Syntaxe**

```sql
quantileDeterministic(level)(expr, determinator)
```

**Alias** : `medianDeterministic`

**Paramètres**

* `level` — Facultatif. Niveau du quantile. Nombre à virgule flottante constant compris entre 0 et 1. Nous recommandons d&#39;utiliser une valeur de `level` dans l&#39;intervalle `[0.01, 0.99]`. Valeur par défaut : 0.5. Pour `level=0.5`, la fonction calcule la médiane. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression appliquée aux valeurs de la colonne, produisant des types de données numériques, Date ou DateTime. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime)
* `determinator` — Nombre dont le hash est utilisé à la place d&#39;un générateur de nombres aléatoires dans l&#39;algorithme d&#39;échantillonnage par réservoir afin de rendre le résultat de l&#39;échantillonnage déterministe. Comme déterminateur, vous pouvez utiliser n&#39;importe quel nombre positif déterministe, par exemple un identifiant utilisateur ou un identifiant d&#39;événement. Si la même valeur de déterminateur apparaît trop souvent, la fonction ne fonctionne pas correctement. [`(U)Int*`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Renvoie un quantile approximatif du niveau spécifié. [`Float64`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime)

**Exemples**

**Calcul d&#39;un quantile déterministe**

```sql title=Query
CREATE TABLE t (val UInt32) ENGINE = Memory;
INSERT INTO t VALUES (1), (1), (2), (3);

SELECT quantileDeterministic(val, 1) FROM t;
```

```response title=Response
┌─quantileDeterministic(val, 1)─┐
│                           1.5 │
└───────────────────────────────┘
```
