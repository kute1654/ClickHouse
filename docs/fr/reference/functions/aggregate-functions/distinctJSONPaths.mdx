<div id="distinctJSONPaths">
  ## distinctJSONPaths
</div>

Introduit dans : v24.9.0

Calcule une liste des chemins distincts stockés dans une colonne [JSON](/fr/reference/data-types/newjson).

**Syntaxe**

```sql
distinctJSONPaths(json)
```

**Arguments**

* `json` — colonne JSON. [`JSON`](/fr/reference/data-types/newjson)

**Valeur renvoyée**

Renvoie la liste triée des chemins. [`Array(String)`](/fr/reference/data-types/array)

**Exemples**

**Utilisation de base avec un JSON imbriqué**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}');

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)───┐
│ ['a','b','c.d.e','c.d.f'] │
└───────────────────────────┘
```

**Avec des chemins JSON définis**

```sql title=Query
DROP TABLE IF EXISTS test_json;
CREATE TABLE test_json(json JSON) ENGINE = Memory;
INSERT INTO test_json VALUES ('{"a" : 42, "b" : "Hello"}'), ('{"b" : [1, 2, 3], "c" : {"d" : {"e" : "2020-01-01"}}}'), ('{"a" : 43, "c" : {"d" : {"f" : [{"g" : 42}]}}}')

SELECT distinctJSONPaths(json) FROM test_json;
```

```response title=Response
┌─distinctJSONPaths(json)─┐
│ ['a','b','c']           │
└─────────────────────────┘
```
