<div id="exponentialTimeDecayedSum">
  ## exponentialTimeDecayedSum
</div>

Introduit dans : v21.12.0

Renvoie la somme des valeurs d&#39;une moyenne mobile à lissage exponentiel d&#39;une série temporelle à l&#39;indice temporel `t`.

**Syntaxe**

```sql
exponentialTimeDecayedSum(x)(v, t)
```

**Paramètres**

* `x` — Différence de temps nécessaire pour que le poids d&#39;une valeur décroisse jusqu&#39;à 1/e. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)

**Arguments**

* `v` — Valeur. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal)
* `t` — Temps. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal`](/fr/reference/data-types/decimal) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`DateTime64`](/fr/reference/data-types/datetime64)

**Valeur renvoyée**

Renvoie la somme des valeurs de moyenne mobile à lissage exponentiel à l&#39;instant donné. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Utilisation d&#39;une fonction de fenêtre avec représentation visuelle**

```sql title=Query
SELECT
    value,
    time,
    round(exp_smooth, 3),
    bar(exp_smooth, 0, 10, 50) AS bar
FROM
    (
    SELECT
    (number = 0) OR (number >= 25) AS value,
    number AS time,
    exponentialTimeDecayedSum(10)(value, time) OVER (ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS exp_smooth
    FROM numbers(50)
    );
```
