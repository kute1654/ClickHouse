<div id="skewPop">
  ## skewPop
</div>

Introduit dans : v20.1.0

Calcule le [coefficient d’asymétrie](https://en.wikipedia.org/wiki/Skewness) d’une séquence.

**Syntaxe**

```sql
skewPop(expr)
```

**Arguments**

* `expr` — Une expression renvoyant un nombre. [`Expression`](/fr/reference/data-types/special-data-types/expression)

**Valeur renvoyée**

Renvoie le coefficient d’asymétrie de la distribution donnée. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Distribution symétrique**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**Distribution asymétrique à droite**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
