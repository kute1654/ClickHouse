<div id="skewSamp">
  ## skewSamp
</div>

Introduit dans : v20.1.0

Calcule le [coefficient d’asymétrie d’un échantillon](https://en.wikipedia.org/wiki/Skewness) d’une séquence.

Il s’agit d’une estimation non biaisée de l’asymétrie d’une variable aléatoire si les valeurs fournies constituent son échantillon.

**Syntaxe**

```sql
skewSamp(expr)
```

**Arguments**

* `expr` — Une expression renvoyant un nombre. [`Expression`](/fr/reference/data-types/special-data-types/expression)

**Valeur renvoyée**

Renvoie le coefficient d&#39;asymétrie de la distribution donnée. Si `n <= 1` (`n` est la taille de l&#39;échantillon), la fonction renvoie `nan`. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Distribution symétrique**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**Distribution asymétrique à droite**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
