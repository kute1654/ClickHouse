<div id="argAndMax">
  ## argAndMax
</div>

Introduit dans : v1.1.0

Calcule les valeurs `arg` et `val` correspondant à la valeur `val` maximale.
S&#39;il existe plusieurs lignes dont la valeur `val` est égale au maximum, le choix de `arg` et `val` renvoyé parmi les valeurs associées n&#39;est pas déterministe.
Les deux parties, `arg` et `max`, se comportent comme des [fonctions d’agrégation](/fr/reference/functions/aggregate-functions/index) : elles [ignorent `Null`](/fr/reference/functions/aggregate-functions/index#null-processing) pendant le traitement et renvoient des valeurs non `Null` si des valeurs non `Null` sont disponibles.

<Note>
  La seule différence avec `argMax` est que `argAndMax` renvoie à la fois l&#39;argument et la valeur.
</Note>

**Voir aussi**

* [argMax](/fr/reference/functions/aggregate-functions/argMax)
* [Tuple](/fr/reference/data-types/tuple)

**Syntaxe**

```sql
argAndMax(arg, val)
```

**Arguments**

* `arg` — Argument dont on cherche la valeur maximale. [`const String`](/fr/reference/data-types/string)
* `val` — La valeur maximale. [`(U)Int8/16/32/64`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`Tuple`](/fr/reference/data-types/tuple)

**Valeur renvoyée**

Renvoie un tuple contenant la valeur `arg` correspondant à la valeur `val` maximale, ainsi que la valeur `val` maximale. [`Tuple`](/fr/reference/data-types/tuple)

**Exemples**

**Utilisation de base**

```sql title=Query
SELECT argAndMax(user, salary) FROM salary;
```

```response title=Response
┌─argAndMax(user, salary)─┐
│ ('director',5000)       │
└─────────────────────────┘
```

**Exemple plus complet avec gestion de NULL**

```sql title=Query
CREATE TABLE test
(
    a Nullable(String),
    b Nullable(Int64)
)
ENGINE = Memory AS
SELECT *
FROM VALUES(('a', 1), ('b', 2), ('c', 2), (NULL, 3), (NULL, NULL), ('d', NULL));

SELECT argMax(a, b), argAndMax(a, b), max(b) FROM test;
```

```response title=Response
┌─argMax(a, b)─┬─argAndMax(a, b)─┬─max(b)─┐
│ b            │ ('b',2)         │      3 │
└──────────────┴─────────────────┴────────┘
```

**Utilisation de Tuple dans les arguments**

```sql title=Query
SELECT argAndMax(a, (b,a)) FROM test;
```

```response title=Response
┌─argAndMax(a, (b, a))─┐
│ ('c',(2,'c'))        │
└──────────────────────┘
```
