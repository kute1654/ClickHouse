## varSampStable {#varSampStable}

Introduit dans : v1.1.0

Calcule la variance d'échantillon d'un ensemble de données. Contrairement à [`varSamp`](/reference/functions/aggregate-functions/varSamp), cette fonction utilise un algorithme [numériquement stable](https://en.wikipedia.org/wiki/Numerical_stability). Elle est plus lente, mais produit une erreur de calcul plus faible.

La variance d'échantillon est calculée à l'aide de la même formule que [`varSamp`](/reference/functions/aggregate-functions/varSamp) :

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

Où :
- $x$ est chaque point de données individuel dans l'ensemble de données
- $\bar{x}$ est la moyenne arithmétique de l'ensemble de données
- $n$ est le nombre de points de données dans l'ensemble de données
    

**Syntaxe**

```sql
varSampStable(x)
```

**Arguments**

- `x` — La population pour laquelle vous souhaitez calculer la variance d'échantillon. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la variance d'échantillon de l'ensemble de données en entrée. [`Float64`](/reference/data-types/float)

**Exemples**

**Calcul de la variance d'échantillon stable**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSampStable(x),3) AS var_samp_stable FROM test_data;
```

```response title=Response
┌─var_samp_stable─┐
│           0.865 │
└─────────────────┘
```
