<div id="argAndMin">
  ## argAndMin
</div>

Introduit dans : v1.1.0

Calcule les valeurs `arg` et `val` correspondant à la valeur minimale de `val`.
S’il existe plusieurs lignes dont `val` est égal au minimum, la paire `arg`/`val` associée qui est renvoyée n’est pas déterministe.
Les deux composantes, `arg` et `min`, se comportent comme des [fonctions d’agrégation](/fr/reference/functions/aggregate-functions/index) : elles [ignorent `Null`](/fr/reference/functions/aggregate-functions/index#null-processing) lors du traitement et renvoient des valeurs non `Null` si de telles valeurs sont disponibles.

<Note>
  La seule différence avec `argMin` est que `argAndMin` renvoie à la fois l’argument et la valeur.
</Note>

**Voir aussi**

* [argMin](/fr/reference/functions/aggregate-functions/argMin)
* [Tuple](/fr/reference/data-types/tuple)

**Syntaxe**

```sql
argAndMin(arg, val)
```

**Arguments**

* `arg` — Argument pour lequel rechercher la valeur minimale. [`const String`](/fr/reference/data-types/string)
* `val` — La valeur minimale. [`(U)Int8/16/32/64`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`Tuple`](/fr/reference/data-types/tuple)

**Valeur renvoyée**

Renvoie un tuple contenant la valeur de `arg` correspondant à la valeur minimale de `val`, ainsi que cette valeur minimale de `val`. [`Tuple`](/fr/reference/data-types/tuple)

**Exemples**

**Utilisation de base**

```sql title=Query
SELECT argAndMin(user, salary) FROM salary;
```

```response title=Response
┌─argAndMin(user, salary)─┐
│ ('worker',1000)         │
└─────────────────────────┘
```

**Exemple détaillé avec gestion de NULL**

```sql title=Query
CREATE TABLE test
(
    a Nullable(String),
    b Nullable(Int64)
)
ENGINE = Memory AS
SELECT *
FROM VALUES((NULL, 0), ('a', 1), ('b', 2), ('c', 2), (NULL, NULL), ('d', NULL));

SELECT argMin(a,b), argAndMin(a, b), min(b) FROM test;
```

```response title=Response
┌─argMin(a, b)─┬─argAndMin(a, b)─┬─min(b)─┐
│ a            │ ('a',1)         │      0 │
└──────────────┴─────────────────┴────────┘
```

**Utilisation de Tuple dans les arguments**

```sql title=Query
SELECT argAndMin(a, (b, a)), min(tuple(b, a)) FROM test;
```

```response title=Response
┌─argAndMin(a, (b, a))─┬─min((b, a))─┐
│ ('a',(1,'a'))        │ (0,NULL)    │
└──────────────────────┴─────────────┘
```
