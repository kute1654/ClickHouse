<div id="min">
  ## min
</div>

Introduit dans : v1.1.0

Fonction d’agrégation qui calcule la valeur minimale dans un groupe de valeurs.

**Syntaxe**

```sql
min(column)
```

**Arguments**

* `column` — Nom de la colonne ou expression. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie la valeur minimale dans le groupe, avec un type identique à celui de l’entrée. [`Any`](/fr/reference/data-types/index)

**Exemples**

**Exemple simple de min**

```sql title=Query
CREATE TABLE employees (name String, salary UInt32) ENGINE = Memory;
INSERT INTO employees VALUES ('Alice', 3000), ('Bob', 4000), ('Charlie', 3500);

SELECT min(salary) FROM employees;
```

```response title=Response
┌─min(salary)─┐
│        3000 │
└─────────────┘
```

**Min avec GROUP BY**

```sql title=Query
CREATE TABLE sales (department String, revenue UInt32) ENGINE = Memory;
INSERT INTO sales VALUES ('Engineering', 100000), ('Engineering', 120000), ('Marketing', 80000), ('Marketing', 90000);

SELECT department, min(revenue) FROM sales GROUP BY department ORDER BY department;
```

```response title=Response
┌─department──┬─min(revenue)─┐
│ Engineering │       100000 │
│ Marketing   │        80000 │
└─────────────┴──────────────┘
```
