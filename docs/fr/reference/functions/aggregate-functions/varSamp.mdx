## varSamp {#varSamp}

Introduit dans : v1.1.0

Calcule la variance d'échantillon d'un ensemble de données.

La variance d'échantillon est calculée à l'aide de la formule suivante :

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

Où :
- $x$ est chaque point de données individuel de l'ensemble de données
- $\bar{x}$ est la moyenne arithmétique de l'ensemble de données
- $n$ est le nombre de points de données dans l'ensemble de données

La fonction suppose que l'ensemble de données en entrée représente un échantillon d'une population plus large. Si vous souhaitez calculer la variance de la population entière (lorsque vous disposez de l'ensemble de données complet), utilisez [`varPop`](/reference/functions/aggregate-functions/varPop) à la place.

<Note>
Cette fonction utilise un algorithme numériquement instable. Si vous avez besoin de [stabilité numérique](https://en.wikipedia.org/wiki/Numerical_stability) dans vos calculs, utilisez la fonction [`varSampStable`](/reference/functions/aggregate-functions/varSampStable). Elle est plus lente, mais produit une erreur de calcul plus faible.
</Note>
    

**Syntax**

```sql
varSamp(x)
```

**Alias** : `VAR_SAMP`

**Arguments**

- `x` — La population pour laquelle vous souhaitez calculer la variance d'échantillon. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie la variance d'échantillon de l'ensemble de données en entrée `x`. [`Float64`](/reference/data-types/float)

**Exemples**

**Calcul de la variance d'échantillon**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSamp(x),3) AS var_samp FROM test_data;
```

```response title=Response
┌─var_samp─┐
│    0.865 │
└──────────┘
```
