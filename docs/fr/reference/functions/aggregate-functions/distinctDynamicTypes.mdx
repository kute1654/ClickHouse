<div id="distinctDynamicTypes">
  ## distinctDynamicTypes
</div>

Introduit dans : v24.9.0

Calcule la liste des types de données distincts stockés dans une colonne [Dynamic](/fr/reference/data-types/dynamic).

**Syntaxe**

```sql
distinctDynamicTypes(dynamic)
```

**Arguments**

* `dynamic` — colonne Dynamic. [`Dynamic`](/fr/reference/data-types/dynamic)

**Valeur renvoyée**

Renvoie la liste triée des noms des types de données. [`Array(String)`](/fr/reference/data-types/array)

**Exemples**

**Utilisation de base avec des types mixtes**

```sql title=Query
DROP TABLE IF EXISTS test_dynamic;
CREATE TABLE test_dynamic(d Dynamic) ENGINE = Memory;
INSERT INTO test_dynamic VALUES (42), (NULL), ('Hello'), ([1, 2, 3]), ('2020-01-01'), (map(1, 2)), (43), ([4, 5]), (NULL), ('World'), (map(3, 4));

SELECT distinctDynamicTypes(d) FROM test_dynamic;
```

```response title=Response
┌─distinctDynamicTypes(d)──────────────────────────────────────────┐
│ ['Array(Int64)', 'Date', 'Int64', 'Map(UInt8, UInt8)', 'String'] │
└──────────────────────────────────────────────────────────────────┘
```
