<div id="groupArrayIntersect">
  ## groupArrayIntersect
</div>

Introduit dans : v24.2.0

Renvoie l’intersection des tableaux donnés (renvoie tous les éléments présents dans tous les tableaux donnés).

**Syntaxe**

```sql
groupArrayIntersect(x)
```

**Arguments**

* `x` — Argument (nom de colonne ou expression). [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie un tableau contenant les éléments communs à tous les tableaux. [`Array`](/fr/reference/data-types/array)

**Exemples**

**Exemple d’utilisation**

```sql title=Query
-- Create table with Memory engine
CREATE TABLE numbers (
    a Array(Int32)
) ENGINE = Memory;

-- Insert sample data
INSERT INTO numbers VALUES
    ([1,2,4]),
    ([1,5,2,8,-1,0]),
    ([1,5,7,5,8,2]);

SELECT groupArrayIntersect(a) AS intersection FROM numbers;
```

```response title=Response
┌─intersection──────┐
│ [1, 2]            │
└───────────────────┘
```
