<div id="anyHeavy">
  ## anyHeavy
</div>

Introduit dans : v1.1.0

Renvoie une valeur fréquente à l’aide de l’algorithme [heavy hitters](https://doi.org/10.1145/762471.762473).
S’il existe une valeur qui apparaît dans plus de la moitié des cas dans chacun des threads d’exécution de la requête, cette valeur est renvoyée.
Normalement, le résultat est non déterministe.

**Syntaxe**

```sql
anyHeavy(column)
```

**Arguments**

* `column` — Nom de la colonne. [`String`](/fr/reference/data-types/string)

**Valeur renvoyée**

Renvoie une valeur apparaissant fréquemment. Le résultat est non déterministe. [`Any`](/fr/reference/data-types/index)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
