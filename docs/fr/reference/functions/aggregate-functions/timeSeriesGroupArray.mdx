<div id="timeSeriesGroupArray">
  ## timeSeriesGroupArray
</div>

Introduit dans : v25.8.0

Trie les séries temporelles par horodatage dans l’ordre croissant.

<Note>
  Cette fonction est expérimentale ; activez-la en définissant `allow_experimental_ts_to_grid_aggregate_function=true`.
</Note>

**Syntaxe**

```sql
timeSeriesGroupArray(timestamp, value)
```

**Arguments**

* `timestamp` — Horodatage de l’échantillon. [`DateTime`](/fr/reference/data-types/datetime) or [`UInt32`](/fr/reference/data-types/int-uint) or [`UInt64`](/fr/reference/data-types/int-uint)
* `value` — Valeur de la série temporelle correspondant à cet horodatage. [`(U)Int*`](/fr/reference/data-types/int-uint) or [`Float*`](/fr/reference/data-types/float) or [`Decimal`](/fr/reference/data-types/decimal)

**Valeur renvoyée**

Renvoie un tableau de tuples `(timestamp, value)` triés par horodatage dans l’ordre croissant. S’il existe plusieurs valeurs pour le même horodatage, la fonction choisit la plus grande de ces valeurs. [`Array(Tuple(T1, T2))`](/fr/reference/data-types/array)

**Exemples**

**Utilisation de base avec des valeurs individuelles**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamp, value)
FROM
(
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesGroupArray(timestamp, value)───────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]  │
└──────────────────────────────────────────────────────┘
```

**Passage de plusieurs échantillons d’horodatages et de valeurs sous forme de tableaux de même taille**

```sql title=Query
WITH
    [110, 120, 130, 140, 140, 100]::Array(UInt32) AS timestamps,
    [1, 6, 8, 17, 19, 5]::Array(Float32) AS values
SELECT timeSeriesGroupArray(timestamps, values);
```

```response title=Response
┌─timeSeriesGroupArray(timestamps, values)──────────────┐
│ [(100, 5), (110, 1), (120, 6), (130, 8), (140, 19)]   │
└───────────────────────────────────────────────────────┘
```
