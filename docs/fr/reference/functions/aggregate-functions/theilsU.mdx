<div id="theilsU">
  ## theilsU
</div>

Introduit dans : v22.1.0

La fonction `theilsU` calcule le [coefficient d&#39;incertitude U de Theil](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient), qui mesure l&#39;association entre deux colonnes d&#39;une table.
Ses valeurs vont de 0.0 (aucune association) à 1.0 (accord parfait).

**Syntaxe**

```sql
theilsU(column1, column2)
```

**Arguments**

* `column1` — Première colonne à comparer. [`Any`](/fr/reference/data-types/index)
* `column2` — Deuxième colonne à comparer. [`Any`](/fr/reference/data-types/index)

**Valeur renvoyée**

Renvoie une valeur entre 0 et 1. [`Float64`](/fr/reference/data-types/float)

**Exemples**

**Exemple d&#39;utilisation**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
