Modifie la chaîne de requête donnée avec des variations aléatoires.

<div id="syntax">
  ## Syntaxe
</div>

```sql
fuzzQuery(query[, max_query_length[, random_seed]])
```

<div id="arguments">
  ## Arguments
</div>

| Argument           | Description                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------------- |
| `query`            | (String) - La requête source à soumettre au fuzzing.                                        |
| `max_query_length` | (UInt64) - Longueur maximale que la requête peut atteindre pendant le processus de fuzzing. |
| `random_seed`      | (UInt64) - Une graine aléatoire pour produire des résultats stables.                        |

<div id="returned_value">
  ## Valeur de retour
</div>

Une table à une seule colonne contenant des chaînes de requête perturbées.

<div id="usage-example">
  ## Exemple d’utilisation
</div>

```sql
SELECT * FROM fuzzQuery('SELECT materialize(\'a\' AS key) GROUP BY key') LIMIT 2;
```

```response
   ┌─query──────────────────────────────────────────────────────────┐
1. │ SELECT 'a' AS key GROUP BY key                                 │
2. │ EXPLAIN PIPELINE compact = true SELECT 'a' AS key GROUP BY key │
   └────────────────────────────────────────────────────────────────┘
```
