<div id="ytsaurus-table-function">
  # Fonction de table ytsaurus
</div>

<ExperimentalBadge />

La fonction de table permet de lire des données à partir du cluster YTsaurus.

<div id="syntax">
  ## Syntaxe
</div>

```sql
ytsaurus(http_proxy_url, cypress_path, oauth_token, format)
```

<Info>
  Il s&#39;agit d&#39;une fonctionnalité expérimentale susceptible d&#39;évoluer de manière rétro-incompatible dans les prochaines versions.
  Activez l&#39;utilisation de la fonction de table YTsaurus
  avec le paramètre [allow&#95;experimental&#95;ytsaurus&#95;table&#95;function](/fr/reference/settings/session-settings#allow_experimental_ytsaurus_table_engine).
  Saisissez la commande `set allow_experimental_ytsaurus_table_function = 1`.
</Info>

<div id="arguments">
  ## Arguments
</div>

* `http_proxy_url` — URL du proxy HTTP de YTsaurus.
* `cypress_path` — chemin Cypress vers la source de données.
* `oauth_token` — jeton OAuth.
* `format` — [format](/fr/reference/formats/index) de la source de données.

**Valeur retournée**

Une table ayant la structure spécifiée pour lire les données à partir du chemin Cypress spécifié dans le cluster YTsaurus.
