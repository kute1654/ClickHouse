ClickHouse étant open-source, il bénéficie de nombreuses contributions, provenant non seulement des employés de ClickHouse, mais aussi de la communauté. Ces contributions n’avancent pas toujours au même rythme ; certaines fonctionnalités peuvent nécessiter une longue phase de prototypage ou davantage de temps pour recueillir suffisamment de retours de la communauté et être itérées avant d’être considérées comme généralement disponibles (GA).

Comme il peut être difficile de déterminer à quel moment une fonctionnalité devient généralement disponible, nous distinguons deux catégories : **Bêta** et **Expérimentales**.

Les fonctionnalités **Bêta** sont officiellement prises en charge par l’équipe ClickHouse. Les fonctionnalités **Expérimentales** sont des prototypes précoces, portés soit par l’équipe ClickHouse, soit par la communauté, et ne sont pas officiellement prises en charge.

Les sections ci-dessous décrivent explicitement les propriétés des fonctionnalités **Bêta** et **Expérimentales** :

<div id="beta-features">
  ## Fonctionnalités bêta
</div>

* En cours de développement actif en vue d’une disponibilité générale (GA)
* Les principaux problèmes connus peuvent être suivis sur GitHub
* Les fonctionnalités peuvent évoluer à l’avenir
* Peuvent être activées dans ClickHouse Cloud
* L’équipe ClickHouse prend en charge les fonctionnalités bêta

Vous trouverez ci-dessous les fonctionnalités considérées comme bêta dans ClickHouse Cloud et disponibles dans vos services ClickHouse Cloud.

Remarque : assurez-vous d’utiliser une version récente du paramètre ClickHouse [compatibility](/fr/reference/settings/session-settings#compatibility) afin de pouvoir utiliser une fonctionnalité introduite récemment.

<div id="experimental-features">
  ## Fonctionnalités expérimentales
</div>

* Peuvent ne jamais devenir GA
* Peuvent être supprimées
* Peuvent introduire des modifications incompatibles avec les versions précédentes
* Le fonctionnement de la fonctionnalité peut évoluer
* Doivent être activées explicitement
* L’équipe ClickHouse **ne prend pas en charge** les fonctionnalités expérimentales
* Peuvent manquer de fonctionnalités importantes et de documentation
* Ne peuvent pas être activées dans le cloud

Veuillez noter : aucune fonctionnalité expérimentale supplémentaire ne peut être activée dans ClickHouse Cloud en dehors de celles répertoriées ci-dessus comme Beta.
