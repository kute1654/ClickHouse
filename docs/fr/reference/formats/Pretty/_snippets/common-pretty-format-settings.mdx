---
title: Paramètres courants du format Pretty
---

{/* Remarque : ce fichier est utilisé comme extrait dans tous les fichiers qui l’importent */}

Les paramètres suivants sont communs à tous les formats `Pretty` :

| Paramètre                                                                                                                                            | Description                                                                                                                                                                                                                     | Défaut  |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| [`output_format_pretty_max_rows`](/fr/reference/settings/formats#output_format_pretty_max_rows)                                                         | Nombre maximal de lignes pour les formats Pretty.                                                                                                                                                                               | `10000` |
| [`output_format_pretty_max_column_pad_width`](/fr/reference/settings/formats#output_format_pretty_max_column_pad_width)                                 | Largeur maximale de remplissage appliquée à toutes les valeurs d’une colonne dans les formats Pretty.                                                                                                                           | `250`   |
| [`output_format_pretty_max_value_width`](/fr/reference/settings/formats#output_format_pretty_max_value_width)                                           | Largeur maximale de la valeur affichée dans les formats Pretty. Au-delà, elle sera tronquée.                                                                                                                                    | `10000` |
| [`output_format_pretty_color`](/fr/reference/settings/formats#output_format_pretty_color)                                                               | Utilise des séquences d’échappement ANSI pour afficher des couleurs dans les formats Pretty.                                                                                                                                    | `true`  |
| [`output_format_pretty_grid_charset`](/fr/reference/settings/formats#output_format_pretty_grid_charset)                                                 | Jeu de caractères utilisé pour afficher les bordures de la grille. Jeux de caractères disponibles : ASCII, UTF-8.                                                                                                               | `UTF-8` |
| [`output_format_pretty_row_numbers`](/fr/reference/settings/formats#output_format_pretty_row_numbers)                                                   | Ajoute des numéros de ligne avant chaque ligne dans le format de sortie Pretty.                                                                                                                                                 | `true`  |
| [`output_format_pretty_display_footer_column_names`](/fr/reference/settings/formats#output_format_pretty_display_footer_column_names)                   | Affiche les noms de colonnes dans le pied de page si la table contient un grand nombre de lignes.                                                                                                                               | `true`  |
| [`output_format_pretty_display_footer_column_names_min_rows`](/fr/reference/settings/formats#output_format_pretty_display_footer_column_names_min_rows) | Définit le nombre minimal de lignes à partir duquel un pied de page s’affiche si [`output_format_pretty_display_footer_column_names`](/fr/reference/settings/formats#output_format_pretty_display_footer_column_names) est activé. | `50`    |
