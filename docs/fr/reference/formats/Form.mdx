<div id="description">
  ## Description
</div>

Le format `Form` peut être utilisé pour lire un seul enregistrement au format application/x-www-form-urlencoded,
où les données sont présentées sous la forme `key1=value1&key2=value2`.

<div id="example-usage">
  ## Exemple d’utilisation
</div>

Soit un fichier `data.tmp` placé dans le chemin `user_files` et contenant des données encodées dans l’URL :

```text title="data.tmp"
t_page=116&c.e=ls7xfkpm&c.tti.m=raf&rt.start=navigation&rt.bmr=390%2C11%2C10
```

```sql title="Query"
SELECT * FROM file(data.tmp, Form) FORMAT vertical;
```

```response title="Response"
Row 1:
──────
t_page:   116
c.e:      ls7xfkpm
c.tti.m:  raf
rt.start: navigation
rt.bmr:   390,11,10
```
