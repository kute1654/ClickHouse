<div id="description">
  ## Description
</div>

Le format `ProtobufSingle` est identique au format [`Protobuf`](/fr/reference/formats/Protobuf/Protobuf), mais il est destiné à stocker et à analyser un seul message Protobuf, sans délimiteurs de longueur.
