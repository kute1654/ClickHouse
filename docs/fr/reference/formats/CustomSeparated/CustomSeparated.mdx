<div id="description">
  ## Description
</div>

Similaire à [Template](/fr/reference/formats/Template/Template), mais il affiche ou lit tous les noms et types de colonnes, et utilise la règle d’échappement définie par le paramètre [format&#95;custom&#95;escaping&#95;rule](/fr/reference/settings/formats#format_custom_escaping_rule) ainsi que les délimiteurs des paramètres suivants :

* [format&#95;custom&#95;field&#95;delimiter](/fr/reference/settings/formats#format_custom_field_delimiter)
* [format&#95;custom&#95;row&#95;before&#95;delimiter](/fr/reference/settings/formats#format_custom_row_before_delimiter)
* [format&#95;custom&#95;row&#95;after&#95;delimiter](/fr/reference/settings/formats#format_custom_row_after_delimiter)
* [format&#95;custom&#95;row&#95;between&#95;delimiter](/fr/reference/settings/formats#format_custom_row_between_delimiter)
* [format&#95;custom&#95;result&#95;before&#95;delimiter](/fr/reference/settings/formats#format_custom_result_before_delimiter)
* [format&#95;custom&#95;result&#95;after&#95;delimiter](/fr/reference/settings/formats#format_custom_result_after_delimiter)

<Note>
  Il n’utilise ni les paramètres de règles d’échappement ni les délimiteurs des chaînes de format.
</Note>

Il existe également le format [`CustomSeparatedIgnoreSpaces`](/fr/reference/formats/CustomSeparated/CustomSeparatedIgnoreSpaces), similaire à [TemplateIgnoreSpaces](/fr/reference/formats/Template/TemplateIgnoreSpaces).

<div id="example-usage">
  ## Exemple d’utilisation
</div>

<div id="inserting-data">
  ### Insertion de données
</div>

À l’aide du fichier texte suivant, nommé `football.txt` :

```text
row('2022-04-30';2021;'Sutton United';'Bradford City';1;4),row('2022-04-30';2021;'Swindon Town';'Barrow';2;1),row('2022-04-30';2021;'Tranmere Rovers';'Oldham Athletic';2;0),row('2022-05-02';2021;'Salford City';'Mansfield Town';2;2),row('2022-05-02';2021;'Port Vale';'Newport County';1;2),row('2022-05-07';2021;'Barrow';'Northampton Town';1;3),row('2022-05-07';2021;'Bradford City';'Carlisle United';2;0),row('2022-05-07';2021;'Bristol Rovers';'Scunthorpe United';7;0),row('2022-05-07';2021;'Exeter City';'Port Vale';0;1),row('2022-05-07';2021;'Harrogate Town A.F.C.';'Sutton United';0;2),row('2022-05-07';2021;'Hartlepool United';'Colchester United';0;2),row('2022-05-07';2021;'Leyton Orient';'Tranmere Rovers';0;1),row('2022-05-07';2021;'Mansfield Town';'Forest Green Rovers';2;2),row('2022-05-07';2021;'Newport County';'Rochdale';0;2),row('2022-05-07';2021;'Oldham Athletic';'Crawley Town';3;3),row('2022-05-07';2021;'Stevenage Borough';'Salford City';4;2),row('2022-05-07';2021;'Walsall';'Swindon Town';0;3)
```

Configurez les paramètres de délimitation personnalisés :

```sql
SET format_custom_row_before_delimiter = 'row(';
SET format_custom_row_after_delimiter = ')';
SET format_custom_field_delimiter = ';';
SET format_custom_row_between_delimiter = ',';
SET format_custom_escaping_rule = 'Quoted';
```

Insérez les données :

```sql
INSERT INTO football FROM INFILE 'football.txt' FORMAT CustomSeparated;
```

<div id="reading-data">
  ### Lecture des données
</div>

Configurez les paramètres de délimitation personnalisés :

```sql
SET format_custom_row_before_delimiter = 'row(';
SET format_custom_row_after_delimiter = ')';
SET format_custom_field_delimiter = ';';
SET format_custom_row_between_delimiter = ',';
SET format_custom_escaping_rule = 'Quoted';
```

Lisez les données au format `CustomSeparated` :

```sql
SELECT *
FROM football
FORMAT CustomSeparated
```

La sortie sera au format personnalisé configuré :


<div id="format-settings">
  ## Paramètres du format
</div>

Paramètres supplémentaires :

| Paramètre                                                                                                                                                            | Description                                                                                                                                                            | Par défaut |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| [input&#95;format&#95;custom&#95;detect&#95;header](/fr/reference/settings/formats#input_format_custom_detect_header)                                                   | active la détection automatique de l’en-tête avec les noms et les types, s’il y en a un.                                                                               | `true`     |
| [input&#95;format&#95;custom&#95;skip&#95;trailing&#95;empty&#95;lines](/fr/reference/settings/formats#input_format_custom_skip_trailing_empty_lines)                   | ignore les lignes vides finales à la fin du fichier.                                                                                                                   | `false`    |
| [input&#95;format&#95;custom&#95;allow&#95;variable&#95;number&#95;of&#95;columns](/fr/reference/settings/formats#input_format_custom_allow_variable_number_of_columns) | autorise un nombre variable de colonnes au format CustomSeparated, ignore les colonnes supplémentaires et utilise les valeurs par défaut pour les colonnes manquantes. | `false`    |
