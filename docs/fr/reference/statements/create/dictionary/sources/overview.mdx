<div id="dictionary-sources">
  ## Syntaxe
</div>

<CloudDetails />

Un dictionnaire peut être connecté à ClickHouse à partir de nombreuses sources différentes.
La source se configure dans la section `source` du fichier de configuration ou à l’aide de la clause `SOURCE` dans une instruction DDL.

<Tabs>
  <Tab title="DDL">
    ```sql
    CREATE DICTIONARY dict_name (...)
    ...
    SOURCE(SOURCE_TYPE(param1 val1 ... paramN valN)) -- Configuration de la source
    ...
    ```
  </Tab>

  <Tab title="Fichier de configuration">
    ```xml
    <clickhouse>
      <dictionary>
        ...
        <source>
          <source_type>
            <!-- Configuration de la source -->
          </source_type>
        </source>
        ...
      </dictionary>
      ...
    </clickhouse>
    ```
  </Tab>
</Tabs>

des paramètres optionnels sont disponibles :

<Tabs>
  <Tab title="DDL">

  </Tab>
