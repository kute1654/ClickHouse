Cette instruction permet de déplacer une entité d’accès d’un stockage d’accès vers un autre.

Syntaxe :

```sql
MOVE {USER, ROLE, QUOTA, SETTINGS PROFILE, ROW POLICY} name1 [, name2, ...] TO access_storage_type
```

Actuellement, il existe cinq stockages des accès dans ClickHouse :

* `local_directory`
* `memory`
* `replicated`
* `users_xml` (ro)
* `ldap` (ro)

Exemples :

```sql
MOVE USER test TO local_directory
```

```sql
MOVE ROLE test TO memory
```
