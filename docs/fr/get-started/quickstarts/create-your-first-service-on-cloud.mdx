<div id="prerequisites">
  ## Prérequis
</div>

Ce guide de démarrage rapide ne nécessite aucun prérequis

<div id="what-youll-build">
  ## Ce que vous allez créer
</div>

Dans ce guide de démarrage rapide, vous allez créer votre premier service ClickHouse Cloud, vous familiariser avec la console Cloud et la console SQL intégrée, puis charger un jeu de données d’exemple afin de disposer de données réelles sur lesquelles exécuter des requêtes.
À la fin, vous disposerez d’un service ClickHouse Cloud opérationnel, avec des données chargées, prêt à servir de base pour le reste de la série de guides de démarrage rapide.

<Steps titleSize="h3">
  <Step>
    ### Inscrivez-vous à ClickHouse Cloud

    Accédez à [console.clickhouse.cloud](https://console.clickhouse.cloud/). La page de connexion s’affiche. En bas de la page, cliquez sur **Register** à côté de &quot;Don&#39;t have an account yet?&quot;.

    <Frame caption="Cliquez sur Register pour créer un nouveau compte">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="Page de connexion à ClickHouse Cloud avec le lien Register" />
    </Frame>

    Sur la page d’inscription, inscrivez-vous avec Google, Microsoft ou votre adresse e-mail professionnelle. Aucune carte de crédit n’est requise : l’essai gratuit vous offre 300 $ de crédits et 30 jours pour découvrir ClickHouse Cloud.

    <Frame caption="Inscrivez-vous avec Google, Microsoft ou votre adresse e-mail professionnelle">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="Page d’inscription à ClickHouse Cloud" />
    </Frame>

    Si vous vous inscrivez avec une adresse e-mail, vous devrez définir un mot de passe. Il doit respecter les exigences suivantes :

    * Au moins 12 caractères
    * Au moins 3 des éléments suivants : lettres minuscules, lettres majuscules, chiffres, caractères spéciaux (par ex. `!@#$%^&*`)

    <Frame caption="Saisissez votre mot de passe et acceptez les Terms of Service">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="Inscription par e-mail à ClickHouse Cloud avec champ de mot de passe" />
    </Frame>

    Acceptez les Terms of Service et la Privacy Policy, puis cliquez sur **Sign up**. Vérifiez votre messagerie, cliquez sur le lien de confirmation, puis sur **Verify email** pour activer votre compte.

    Une fois l’adresse vérifiée, connectez-vous avec votre nouvelle adresse e-mail et votre mot de passe.
  </Step>

  <Step>
    ### Créer un nouveau service

    Après votre première connexion, vous accéderez directement à la page de configuration du service. Configurez les éléments suivants :

    | Paramètre                      | Recommandation                                                                                                                         |
    | ------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
    | **Nom du service**             | Donnez à votre service un nom explicite (par ex. `Mon premier service`)                                                                |
    | **Fournisseur cloud**          | Choisissez AWS, GCP ou Azure selon vos préférences                                                                                     |
    | **Région**                     | Sélectionnez la région la plus proche de chez vous pour réduire la latence au minimum                                                  |
    | **Mémoire et dimensionnement** | Sélectionnez **Mini 12GB** (1 replica) pour ce guide de démarrage rapide - c’est l’option la plus économique pour découvrir le service |

    Cliquez sur **Créer le service** pour créer votre service.

    <Frame caption="Configurez votre service cloud">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="Page de configuration du service ClickHouse Cloud" />
    </Frame>

    Vous arriverez sur la page d’accueil du service, où le statut affichera d’abord **Provisioning**. Cette étape prend généralement moins d’une minute. Une fois terminée, le statut passera à **Running**.

    <Frame caption="Votre service est en cours d’exécution">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="Page d’accueil du service ClickHouse Cloud affichant le statut Running" />
    </Frame>
  </Step>

  <Step>
    ### Charger un jeu de données d’exemple

    Cliquez sur **Data sources** dans la barre latérale gauche. Vous verrez plusieurs options pour importer des données dans votre service.

    <Frame caption="Page Data sources avec des options de chargement des données">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="Page Data sources de ClickHouse Cloud" />
    </Frame>

    Cliquez sur **Add sample data** pour parcourir les jeux de données d’exemple disponibles. Vous verrez trois options : Cell Towers, UK Property Price Paid et New York Taxi Data. Cliquez sur **Get started** pour le jeu de données **UK Property Price Paid**.

    <Frame caption="Sélection du jeu de données UK Property Price Paid">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="Sélection de jeux de données d’exemple dans ClickHouse Cloud" />
    </Frame>

    Vous verrez un aperçu des données et du schéma de la table. Cliquez sur **Import dataset** pour charger les données dans votre service.

    <Frame caption="Aperçu des données, puis clic sur Import dataset">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="Aperçu de l’importation du jeu de données dans ClickHouse Cloud avec les données immobilières du Royaume-Uni" />
    </Frame>

    L’importation prendra une à deux minutes. Une fois terminée, vous pourrez interroger les données.
  </Step>

  <Step>
    ### Interroger les données

    Cliquez sur **SQL Console** dans la barre latérale de gauche pour ouvrir l’éditeur de requêtes intégré. C’est là que vous rédigerez et exécuterez des requêtes SQL directement dans votre navigateur, sans avoir besoin d’outils externes.

    Essayez quelques requêtes pour explorer les données sur les prix de l’immobilier au Royaume-Uni.

    Trouvez les biens immobiliers les plus chers jamais vendus :

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    Calculez le prix de vente moyen par année :

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    Ces requêtes analysent des millions de lignes en quelques secondes - c’est l’effet du stockage en colonnes et de la compression de ClickHouse.
  </Step>
</Steps>
