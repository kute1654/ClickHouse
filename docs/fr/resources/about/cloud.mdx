ClickHouse Cloud est l’offre cloud conçue par les créateurs de ClickHouse, la célèbre base de données OLAP open source.
Découvrez ClickHouse Cloud en [lançant un essai gratuit](https://console.clickhouse.cloud/signUp).

<div id="clickhouse-cloud-benefits">
  ## Avantages de ClickHouse Cloud
</div>

Voici quelques-uns des avantages de ClickHouse Cloud :

* **Mise en œuvre rapide** : commencez à créer immédiatement, sans avoir à dimensionner ni à faire évoluer votre cluster.
* **Mise à l&#39;échelle fluide** : la mise à l&#39;échelle automatique s&#39;adapte aux workloads variables afin d&#39;éviter le surprovisionnement lors des pics d&#39;utilisation.
* **Opérations serverless** : laissez-nous gérer le dimensionnement, la mise à l&#39;échelle, la sécurité, la fiabilité et les mises à niveau.
* **Tarification transparente** : ne payez que ce que vous utilisez, avec des réservations de ressources et des contrôles de mise à l&#39;échelle.
* **Coût total de possession** : excellent rapport prix/performance et faible charge administrative.
* **Large écosystème** : utilisez vos connecteurs de données, outils de visualisation, SQL et clients pour langages préférés.

<div id="oss-vs-clickhouse-cloud">
  ## Quelle version de ClickHouse utilise ClickHouse Cloud ?
</div>

ClickHouse Cloud met régulièrement votre service à niveau vers des versions plus récentes qui incluent des correctifs, de nouvelles fonctionnalités et des améliorations des performances. Après la publication en open source d&#39;une version majeure du moteur de base de données, nous effectuons une validation supplémentaire dans notre environnement de staging Cloud, ce qui prend généralement 6 à 8 semaines avant le déploiement en production. Le déploiement se fait progressivement, par phases, selon le fournisseur de services Cloud, le type de service et la région.

Vous pouvez définir le calendrier de mise à niveau de votre service ClickHouse Cloud en vous abonnant à un release channel spécifique. Par exemple, nous proposons un [&quot;Fast&quot; Release Channel](/fr/products/cloud/features/admin-features/upgrades#fast-release-channel-early-upgrades) pour recevoir les mises à jour avant le calendrier de release habituel, ainsi qu&#39;un [&quot;Slow&quot; Release Channel](/fr/products/cloud/features/admin-features/upgrades#slow-release-channel-deferred-upgrades) et d&#39;autres options de planification plus granulaires pour différer les mises à niveau.

Pour une vue d&#39;ensemble du processus de mise à niveau de ClickHouse Cloud, y compris les garanties de rétrocompatibilité, consultez la documentation de référence sur les [Upgrades](/fr/products/cloud/features/admin-features/upgrades).
