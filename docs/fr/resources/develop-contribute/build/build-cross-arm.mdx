Aucune étape particulière n’est nécessaire pour compiler ClickHouse pour Aarch64 sur une machine Aarch64.

Pour compiler ClickHouse en cross-compilation pour AArch64 sur une machine Linux x86, passez l’option suivante à `cmake` : `-DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-aarch64.cmake`
