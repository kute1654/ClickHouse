Programme simple de compression et de décompression des données.

<div id="examples">
  ### Exemples
</div>

Compressez les données avec LZ4 :

```bash
$ ./clickhouse-compressor < input_file > output_file
```

Décompresser des données au format LZ4 :

```bash
$ ./clickhouse-compressor --decompress < input_file > output_file
```

Compressez les données avec ZSTD au niveau 5 :

```bash
$ ./clickhouse-compressor --codec 'ZSTD(5)' < input_file > output_file
```

Compressez les données avec Delta de quatre octets et ZSTD de niveau 10.

```bash
$ ./clickhouse-compressor --codec 'Delta(4)' --codec 'ZSTD(10)' < input_file > output_file
```
