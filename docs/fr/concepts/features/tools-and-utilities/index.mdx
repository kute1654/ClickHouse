| Outil/Utilitaire                                                                            | Description                                                                                                    |
| ------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| [clickhouse-local](/fr/concepts/features/tools-and-utilities/clickhouse-local)                 | Permet d’exécuter des requêtes SQL sur des données sans démarrer le serveur ClickHouse, à la manière de `awk`. |
| [clickhouse-benchmark](/fr/concepts/features/tools-and-utilities/clickhouse-benchmark)         | Charge le serveur avec des requêtes et des paramètres personnalisés.                                           |
| [clickhouse-format](/fr/concepts/features/tools-and-utilities/clickhouse-format)               | Permet de mettre en forme les requêtes d’entrée.                                                               |
| [ClickHouse obfuscator](/fr/concepts/features/tools-and-utilities/clickhouse-obfuscator)       | Obfusque les données.                                                                                          |
| [ClickHouse compressor](/fr/concepts/features/tools-and-utilities/clickhouse-compressor)       | Compresse et décompresse les données.                                                                          |
| [clickhouse-disks](/fr/concepts/features/tools-and-utilities/clickhouse-disks)                 | Fournit des opérations de type système de fichiers sur des fichiers situés sur différents disks ClickHouse.    |
| [clickhouse-odbc-bridge](/fr/concepts/features/tools-and-utilities/odbc-bridge)                | Un serveur proxy pour le driver ODBC.                                                                          |
| [clickhouse&#95;backupview](/fr/concepts/features/tools-and-utilities/backupview)              | Un module Python pour analyser les sauvegardes ClickHouse.                                                     |
| [clickhouse-keeper-client](/fr/concepts/features/tools-and-utilities/clickhouse-keeper-client) | Une application cliente pour interagir avec ClickHouse Keeper.                                                 |
| [Keeper HTTP API](/fr/concepts/features/tools-and-utilities/clickhouse-keeper-http-api)        | API HTTP et tableau de bord intégré pour ClickHouse Keeper.                                                    |
