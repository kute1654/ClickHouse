Commencez à utiliser ClickHouse Cloud sur [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps) grâce à une offre publique PAYG (paiement à l’utilisation).

<div id="prerequisites">
  ## Prérequis
</div>

* Un projet Azure pour lequel votre administrateur de la facturation a activé les droits d’achat.
* Pour vous abonner à ClickHouse Cloud sur Azure Marketplace, vous devez être connecté avec un compte disposant de droits d’achat et sélectionner le projet approprié.

1. Accédez à [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps) et recherchez ClickHouse Cloud. Assurez-vous d’être connecté afin de pouvoir acheter une offre sur la marketplace.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-1.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

2. Sur la page de l’offre produit, cliquez sur **Get It Now**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-2.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

3. Vous devrez renseigner un nom, une adresse e-mail et des informations de localisation sur l’écran suivant.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-3.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

4. Sur l’écran suivant, cliquez sur **Subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-4.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

5. Sur l’écran suivant, choisissez l’abonnement, le groupe de ressources et l’emplacement du groupe de ressources. L’emplacement du groupe de ressources n’a pas besoin d’être le même que celui où vous prévoyez de lancer vos services sur ClickHouse Cloud.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-5.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

6. Vous devrez également indiquer un nom pour l’abonnement et choisir la période de facturation parmi les options disponibles. Vous pouvez choisir d’activer ou de désactiver **Recurring billing**. Si vous la définissez sur &quot;off&quot;, votre contrat prendra fin à l’issue de la période de facturation et vos ressources seront mises hors service.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-6.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

7. Cliquez sur **&quot;Review + subscribe&quot;**.

8. Sur l’écran suivant, vérifiez que tout est correct et cliquez sur **Subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-7.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

9. Notez qu’à ce stade, vous serez abonné à l’abonnement Azure de ClickHouse Cloud, mais vous n’aurez pas encore configuré votre compte sur ClickHouse Cloud. Les étapes suivantes sont nécessaires et essentielles pour que ClickHouse Cloud puisse être associé à votre abonnement Azure afin que votre facturation s’effectue correctement via Azure Marketplace.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-8.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

10. Une fois la configuration Azure terminée, le bouton **Configure account now** devrait devenir actif.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-9.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

11. Cliquez sur **Configure account now**.

<br />

Vous recevrez un e-mail comme celui ci-dessous avec les détails relatifs à la configuration de votre compte :

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-10.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

12. Vous serez redirigé vers la page d’inscription ou de connexion de ClickHouse Cloud. Une fois redirigé vers ClickHouse Cloud, vous pouvez soit vous connecter avec un compte existant, soit créer un nouveau compte. Cette étape est très importante afin que nous puissions associer votre organisation ClickHouse Cloud à la facturation Azure Marketplace.

13. Notez que si vous êtes un nouvel utilisateur, vous devrez également fournir quelques informations de base sur votre entreprise. Voir les captures d’écran ci-dessous.

<br />

<Image img="/images/cloud/manage/billing/marketplace/aws-marketplace-payg-8.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

Une fois que vous aurez cliqué sur **Complete sign up**, vous serez redirigé vers votre organisation dans ClickHouse Cloud, où vous pourrez consulter l’écran de facturation pour vérifier que la facturation s’effectue via Azure Marketplace et créer des services.

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-11.webp" size="md" alt="Formulaire d'information d'inscription à ClickHouse Cloud" border />

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-12.webp" size="md" alt="Formulaire d'inscription à ClickHouse Cloud" border />

<br />

14. Si vous rencontrez le moindre problème, n&#39;hésitez pas à contacter [notre équipe d&#39;assistance](https://clickhouse.com/support/program).
