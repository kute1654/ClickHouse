<div id="notes">
  ## Remarques
</div>

* La **version du chart Helm de l’opérateur** (`1.18587.1`) diffère du **tag d’image** de l’opérateur (`main-1.18587.1`). Utilisez la version du chart Helm lors de l’installation avec `helm install`, et le tag d’image lors de la copie des images de conteneur.
* Les **builds FIPS** ajoutent `-fips` aux tags d’image standard, à la fois pour serveur et pour Keeper.
* Les tags de version sont remplacés automatiquement lors de la génération de cette documentation. Si vous voyez des variables de modèle brutes (par exemple `25.12.1.1497`), l’étape de build n’a pas été exécutée.
