chDB-bun fournit des bindings FFI (Foreign Function Interface) expérimentaux pour chDB, vous permettant d’exécuter des requêtes ClickHouse directement dans vos applications Bun avec zéro dépendance externe.

<div id="installation">
  ## Installation
</div>

<div id="install-system-dependencies">
  ### Étape 1 : Installer les dépendances système
</div>

Commencez par installer les dépendances système requises :

<div id="install-libchdb">
  #### Installer libchdb
</div>

```bash
curl -sL https://lib.chdb.io | bash
```

<div id="install-build-tools">
  #### Installer les outils de compilation
</div>

Vous devez avoir `gcc` ou `clang` installé sur votre système :

<div id="install-chdb-bun">
  ### Étape 2 : Installer chDB-bun
</div>

```bash
# Install from the GitHub repository
bun add github:chdb-io/chdb-bun

# Or clone and build locally
git clone https://github.com/chdb-io/chdb-bun.git
cd chdb-bun
bun install
bun run build
```

<div id="usage">
  ## Utilisation
</div>

chDB-bun prend en charge deux modes de requête : les requêtes éphémères pour les opérations ponctuelles et les sessions persistantes pour préserver l’état de la base de données.

<div id="ephemeral-queries">
  ### Requêtes éphémères
</div>

Pour des requêtes simples et ponctuelles qui ne nécessitent pas d’état persistant :

```typescript
import { query } from 'chdb-bun';

// Basic query
const result = query("SELECT version()", "CSV");
console.log(result); // "23.10.1.1"

// Query with different output formats
const jsonResult = query("SELECT 1 as id, 'Hello' as message", "JSON");
console.log(jsonResult);

// Query with calculations
const mathResult = query("SELECT 2 + 2 as sum, pi() as pi_value", "Pretty");
console.log(mathResult);

// Query system information
const systemInfo = query("SELECT * FROM system.functions LIMIT 5", "CSV");
console.log(systemInfo);
```

<div id="persistent-sessions">
  ### Sessions persistantes
</div>

Pour les opérations complexes qui nécessitent de conserver l’état d’une requête à l’autre :

```typescript
import { Session } from 'chdb-bun';

// Create a session with persistent storage
const sess = new Session('./chdb-bun-tmp');

try {
    // Create a database and table
    sess.query(`
        CREATE DATABASE IF NOT EXISTS mydb;
        CREATE TABLE IF NOT EXISTS mydb.users (
            id UInt32,
            name String,
            email String
        ) ENGINE = MergeTree() ORDER BY id
    `, "CSV");

    // Insert data
    sess.query(`
        INSERT INTO mydb.users VALUES 
        (1, 'Alice', 'alice@example.com'),
        (2, 'Bob', 'bob@example.com'),
        (3, 'Charlie', 'charlie@example.com')
    `, "CSV");

    // Query the data
    const users = sess.query("SELECT * FROM mydb.users ORDER BY id", "JSON");
    console.log("Users:", users);

    // Create and use custom functions
    sess.query("CREATE FUNCTION IF NOT EXISTS hello AS () -> 'Hello chDB'", "CSV");
    const greeting = sess.query("SELECT hello() as message", "Pretty");
    console.log(greeting);

    // Aggregate queries
    const stats = sess.query(`
        SELECT 
            COUNT(*) as total_users,
            MAX(id) as max_id,
            MIN(id) as min_id
        FROM mydb.users
    `, "JSON");
    console.log("Statistics:", stats);

} finally {
    // Always cleanup the session to free resources
    sess.cleanup(); // This deletes the database files
}
```
