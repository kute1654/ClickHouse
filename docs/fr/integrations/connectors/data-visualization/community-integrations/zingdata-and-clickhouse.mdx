<a href="https://www.zingdata.com/" target="_blank">Zing Data</a> est une plateforme d&#39;exploration et de visualisation de données. Zing Data se connecte à ClickHouse à l’aide du driver JavaScript fourni par ClickHouse.

<div id="how-to-connect">
  ## Comment se connecter
</div>

1. Récupérez vos informations de connexion.

<ConnectionDetails />

2. Téléchargez Zing Data ou rendez-vous sur le site

   * Pour utiliser ClickHouse avec Zing Data sur mobile, téléchargez l’application Zing Data sur le [Google Play Store](https://play.google.com/store/apps/details?id=com.getzingdata.android) ou l’[Apple App Store](https://apps.apple.com/us/app/zing-data-collaborative-bi/id1563294091).

   * Pour utiliser ClickHouse avec Zing Data sur le web, rendez-vous sur la [console web Zing](https://console.getzingdata.com/) et créez un compte.

3. Ajoutez une source de données

   * Pour interagir avec vos données ClickHouse dans Zing Data, vous devez définir une ***source de données***. Dans le menu de l’application mobile Zing Data, sélectionnez **Sources**, puis cliquez sur **Add a Datasource**.

   * Pour ajouter une source de données sur le web, cliquez sur **Data Sources** dans le menu supérieur, puis sur **New Datasource** et sélectionnez **ClickHouse** dans le menu déroulant

   <Image size="md" img="/images/integrations/data-visualization/zing_01.webp" alt="Interface de Zing Data affichant le bouton New Datasource et l’option ClickHouse dans le menu déroulant" border />

   <br />

4. Renseignez les informations de connexion, puis cliquez sur **Check Connection**.

   <Image size="md" img="/images/integrations/data-visualization/zing_02.webp" alt="Formulaire de configuration de connexion ClickHouse dans Zing Data avec des champs pour le serveur, le port, la base de données, le nom d’utilisateur et le mot de passe" border />

   <br />

5. Si la connexion réussit, Zing vous redirigera vers la sélection des tables. Sélectionnez les tables requises et cliquez sur **Save**. Si Zing ne parvient pas à se connecter à votre source de données, un message vous demandera de vérifier vos identifiants et de réessayer. Si vous rencontrez toujours des problèmes après cela, <a id="contact_link" href="mailto:hello@getzingdata.com">contactez le support Zing ici.</a>

   <Image size="md" img="/images/integrations/data-visualization/zing_03.webp" alt="Interface de sélection des tables de Zing Data affichant les tables ClickHouse disponibles avec des cases à cocher" border />

   <br />

6. Une fois la source de données ClickHouse ajoutée, elle sera disponible pour tous les membres de votre organisation Zing, dans l’onglet **Data Sources** / **Sources**.

<div id="creating-charts-and-dashboards-in-zing-data">
  ## Création de graphiques et de tableaux de bord dans Zing Data
</div>

1. Une fois votre source de données ClickHouse ajoutée, cliquez sur **Zing App** sur le Web, ou sur la source de données sur mobile pour commencer à créer des graphiques.

2. Cliquez sur une table dans la liste des tables pour créer un graphique.

   <Image size="sm" img="/images/integrations/data-visualization/zing_04.webp" alt="Interface de Zing Data affichant la liste des tables avec les tables ClickHouse disponibles" border />

   <br />

3. Utilisez le query builder visuel pour sélectionner les champs, agrégations, etc. souhaités, puis cliquez sur **Run Question**.

   <Image size="md" img="/images/integrations/data-visualization/zing_05.webp" alt="Interface du query builder visuel de Zing Data avec sélection de champs et options d’agrégation" border />

   <br />

4. Si vous maîtrisez SQL, vous pouvez également écrire du SQL personnalisé pour exécuter des requêtes et créer un graphique.

   <Image size="md" img="/images/integrations/data-visualization/zing_06.webp" alt="Mode SQL Editor dans Zing Data affichant l’interface de rédaction de requêtes SQL" border />

   <Image size="md" img="/images/integrations/data-visualization/zing_07.webp" alt="Résultats de requête SQL dans Zing Data avec les données affichées sous forme tabulaire" border />

5. Voici à quoi peut ressembler un graphique. La question peut être enregistrée à l’aide du menu à trois points. Vous pouvez commenter le graphique, identifier des membres de votre équipe, créer des alertes en temps réel, changer le type de graphique, etc.

   <Image size="md" img="/images/integrations/data-visualization/zing_08.webp" alt="Exemple de visualisation de graphique dans Zing Data affichant des données de ClickHouse avec un menu d’options" border />

   <br />

6. Vous pouvez créer des tableaux de bord à l’aide de l’icône « + » sous **Dashboards** sur l’écran d’accueil. Les questions existantes peuvent y être glissées pour être affichées sur le tableau de bord.

   <Image size="md" img="/images/integrations/data-visualization/zing_09.webp" alt="Vue du tableau de bord Zing Data affichant plusieurs visualisations organisées dans une disposition de tableau de bord" border />

   <br />
