<div id="2-create-a-clickhouse-data-source">
  ## 2. Créer une source de données ClickHouse
</div>

Accédez à Admin -&gt; Connections, puis cliquez sur le bouton &quot;Add Connection&quot; en haut à droite.

<Image size="lg" img="/images/integrations/data-visualization/omni_01.webp" alt="Interface d’administration d’Omni montrant le bouton Add Connection dans la section Connections" border />

<br />

Sélectionnez `ClickHouse`. Saisissez vos identifiants dans le formulaire.

<Image size="lg" img="/images/integrations/data-visualization/omni_02.webp" alt="Interface de configuration de la connexion Omni pour ClickHouse, avec les champs du formulaire d’identification" border />

<br />

Vous devriez maintenant pouvoir interroger et visualiser les données ClickHouse dans Omni.
