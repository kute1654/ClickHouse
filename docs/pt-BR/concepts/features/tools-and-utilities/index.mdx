| Ferramenta/Utilitário                                                                       | Descrição                                                                                                   |
| ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| [clickhouse-local](/pt-BR/concepts/features/tools-and-utilities/clickhouse-local)                 | Permite executar consultas SQL em dados sem iniciar o ClickHouse server, de forma semelhante ao `awk`.      |
| [clickhouse-benchmark](/pt-BR/concepts/features/tools-and-utilities/clickhouse-benchmark)         | Carrega o servidor com consultas e configurações personalizadas.                                            |
| [clickhouse-format](/pt-BR/concepts/features/tools-and-utilities/clickhouse-format)               | Permite formatar consultas de entrada.                                                                      |
| [ClickHouse obfuscator](/pt-BR/concepts/features/tools-and-utilities/clickhouse-obfuscator)       | Ofusca dados.                                                                                               |
| [ClickHouse compressor](/pt-BR/concepts/features/tools-and-utilities/clickhouse-compressor)       | Comprime e descomprime dados.                                                                               |
| [clickhouse-disks](/pt-BR/concepts/features/tools-and-utilities/clickhouse-disks)                 | Fornece operações semelhantes às de um sistema de arquivos com arquivos em diferentes discos do ClickHouse. |
| [clickhouse-odbc-bridge](/pt-BR/concepts/features/tools-and-utilities/odbc-bridge)                | Um servidor proxy para ODBC Driver.                                                                         |
| [clickhouse&#95;backupview](/pt-BR/concepts/features/tools-and-utilities/backupview)              | Um módulo Python para analisar backups do ClickHouse.                                                       |
| [clickhouse-keeper-client](/pt-BR/concepts/features/tools-and-utilities/clickhouse-keeper-client) | Um aplicativo cliente para interagir com o ClickHouse Keeper.                                               |
| [Keeper HTTP API](/pt-BR/concepts/features/tools-and-utilities/clickhouse-keeper-http-api)        | API HTTP e dashboard integrado para o ClickHouse Keeper.                                                    |
