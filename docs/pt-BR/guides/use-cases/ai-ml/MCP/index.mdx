[Model Context Protocol](https://www.anthropic.com/news/model-context-protocol) (MCP) é um padrão desenvolvido pela Anthropic que permite que assistentes de IA se integrem de forma transparente a sistemas externos.
Esse protocolo permite que assistentes de IA se conectem a fontes de dados, APIs, bancos de dados e muito mais de maneira segura e padronizada.

O MCP cria uma interface universal entre modelos de IA e vários serviços, eliminando a necessidade de implementar ferramentas personalizadas para cada integração.
Você pode pensar nele como um padrão universal de API projetado especificamente para sistemas de IA.

A principal vantagem do MCP é que as bibliotecas de IA só precisam implementar suporte ao protocolo uma única vez.
Depois disso, todos os serviços compatíveis com MCP ficam imediatamente acessíveis, economizando muito tempo dos mantenedores dessas bibliotecas.

<div id="mcp-architecture">
  ## Como é a arquitetura do MCP?
</div>

O MCP segue uma arquitetura cliente-servidor:

* Clientes (como Claude Desktop, Cursor ou VS Code) estabelecem conexões com servidores MCP. Você pode ver uma lista de clientes no repositório do GitHub [awesome-mcp-clients](https://github.com/punkpeye/awesome-mcp-clients?tab=readme-ov-file#windsurf).
* Os servidores expõem ferramentas e recursos por meio de interfaces padronizadas. Você pode ver uma lista de servidores no repositório do GitHub [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers).
* Os modelos de IA podem então usar essas ferramentas para acessar dados e funcionalidades externas quando necessário

<div id="clickhouse-mcp-server">
  ## O ClickHouse tem um servidor MCP?
</div>

Tem sim!
O [ClickHouse MCP server](https://github.com/ClickHouse/mcp-clickhouse) oferece as seguintes ferramentas:

* `run_select_query` - Execute consultas SQL no seu cluster do ClickHouse.
* `list_databases` - Liste todos os bancos de dados no seu cluster do ClickHouse.
* `list_tables` - Liste todas as tabelas em um banco de dados.

Se você está procurando um servidor MCP remoto no ClickHouse Cloud, consulte a página [&quot;Servidor MCP remoto no Cloud&quot;](/pt-BR/products/cloud/features/ai-ml/remote-mcp)

<Tip>
  **Agent Skills**

  O [repositório ClickHouse Agent Skills](https://github.com/ClickHouse/agent-skills) fornece instruções prontas para uso que ampliam agentes de programação com IA (Claude Code, Cursor, Copilot etc.) com conhecimento especializado do domínio. Este repositório fornece skills para bancos de dados ClickHouse, cobrindo design de schema, otimização de consultas e padrões de ingestão de dados.
</Tip>
