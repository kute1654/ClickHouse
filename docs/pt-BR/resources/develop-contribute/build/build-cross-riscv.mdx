O ClickHouse oferece suporte experimental a RISC-V. Nem todos os recursos podem ser ativados.

<div id="build-clickhouse">
  ## Compile o ClickHouse
</div>

Para fazer compilação cruzada para RISC-V em uma máquina que não seja RISC-V:

```bash
cd ClickHouse
mkdir build-riscv64
cmake . -Bbuild-riscv64 -G Ninja -DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-riscv64.cmake -DGLIBC_COMPATIBILITY=OFF -DENABLE_LDAP=OFF  -DOPENSSL_NO_ASM=ON -DENABLE_JEMALLOC=ON -DENABLE_PARQUET=OFF -DENABLE_GRPC=OFF -DENABLE_HDFS=OFF -DENABLE_MYSQL=OFF
ninja -C build-riscv64
```

O binário resultante será executado apenas no Linux com a arquitetura de CPU RISC-V 64.
