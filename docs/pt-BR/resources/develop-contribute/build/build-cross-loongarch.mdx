O ClickHouse oferece suporte experimental para LoongArch64

<div id="build-clickhouse">
  ## Compilar o ClickHouse
</div>

A versão do llvm necessária para a compilação deve ser igual ou superior a 21.1.0.

```bash
cd ClickHouse
mkdir build-loongarch64
cmake . -Bbuild-loongarch64 -DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-loongarch64.cmake
ninja -C build-loongarch64
```

O binário resultante será executado apenas em sistemas Linux com a arquitetura de CPU LoongArch64.
