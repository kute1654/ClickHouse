<div id="automatic-schema-migration-tools-for-clickhouse">
  ## Ferramenta de migração automática de schema para ClickHouse
</div>

Muitas vezes nos perguntam qual é uma boa ferramenta de migração de schema para ClickHouse e quais são as melhores práticas para gerenciar schemas de banco de dados no ClickHouse que podem mudar ao longo do tempo.

Não existe uma ferramenta padrão de migração de schema para ClickHouse, mas compilamos a lista a seguir (sem nenhuma ordem específica) de ferramentas automáticas de migração de schema com suporte ao ClickHouse que conhecemos:

* [Goose](https://github.com/pressly/goose)
* [Sqitch](https://sqitch.org/docs/manual/sqitchtutorial-clickhouse/)
* [Atlas](https://atlasgo.io/guides/clickhouse?utm_source=clickhouse\&utm_term=knowledge)
* [Bytebase](https://www.bytebase.com/)
* [Flyway](https://documentation.red-gate.com/fd/supported-databases-and-versions-143754067.html)
* [Liquibase](https://www.liquibase.com/)
* Uma [ferramenta simples da comunidade](https://github.com/VVVi/clickhouse-migrations) chamada `clickhouse-migrations`
* Outra [ferramenta da comunidade](https://github.com/golang-migrate/migrate/tree/master/database/clickhouse) escrita em Go
* [Houseplant](https://houseplant.readthedocs.io)
