<div id="2--connect-explo-to-clickhouse">
  ## 2.  Conecte o Explo ao ClickHouse
</div>

1. Crie uma conta no Explo.

2. Clique na aba **Data** do Explo na barra lateral esquerda.

<Image img="/images/integrations/data-visualization/explo_01.webp" size="sm" alt="Aba Data" border />

3. Clique em **Connect Data Source** no canto superior direito.

<Image img="/images/integrations/data-visualization/explo_02.webp" size="sm" alt="Connect Data Source" border />

4. Preencha as informações na página **Getting Started**

<Image img="/images/integrations/data-visualization/explo_03.webp" size="md" alt="Getting Started" border />

5. Selecione **ClickHouse**

<Image img="/images/integrations/data-visualization/explo_04.webp" size="md" alt="ClickHouse" border />

6. Insira suas **credenciais do ClickHouse**.

<Image img="/images/integrations/data-visualization/explo_05.webp" size="md" alt="Credenciais" border />

7. Configure **Security**

<Image img="/images/integrations/data-visualization/explo_06.webp" size="md" alt="Security" border />

8. No ClickHouse, **adicione os IPs do Explo à lista de permissões**.
   `54.211.43.19, 52.55.98.121, 3.214.169.94, and 54.156.141.148`

<div id="3-create-a-dashboard">
  ## 3. Criar um dashboard
</div>

1. Vá para a aba **Dashboard** na barra de navegação à esquerda.

<Image img="/images/integrations/data-visualization/explo_07.webp" size="sm" alt="Dashboard" border />

2. Clique em **Create Dashboard** no canto superior direito e dê um nome ao dashboard. Pronto, o dashboard foi criado!

<Image img="/images/integrations/data-visualization/explo_08.webp" size="sm" alt="Criar dashboard" border />

3. Agora você verá uma tela semelhante a esta:

<Image img="/images/integrations/data-visualization/explo_09.webp" size="md" alt="Dashboard do Explo" border />

<div id="4-run-a-sql-query">
  ## 4. Execute uma consulta SQL
</div>

1. Encontre o nome da sua tabela na barra lateral direita, abaixo do título do schema. Em seguida, insira o seguinte comando no editor do seu conjunto de dados:
   `SELECT * FROM YOUR_TABLE_NAME
   LIMIT 100`

<Image img="/images/integrations/data-visualization/explo_10.webp" size="md" alt="Explo Dashboard" border />

2. Agora clique em Run e vá para a aba Preview para ver seus dados.

<Image img="/images/integrations/data-visualization/explo_11.webp" size="md" alt="Explo Dashboard" border />

<div id="5-build-a-chart">
  ## 5. Criar um gráfico
</div>

1. No lado esquerdo, arraste o ícone de gráfico de barras para a tela.

<Image img="/images/integrations/data-visualization/explo_16.webp" size="sm" alt="Explo Dashboard" border />

2. Selecione o conjunto de dados. Agora você verá uma tela como a seguir:

<Image img="/images/integrations/data-visualization/explo_12.webp" size="sm" alt="Explo Dashboard" border />

3. Preencha **county** no eixo X e **Price** na seção do eixo Y, assim:

<Image img="/images/integrations/data-visualization/explo_13.webp" size="sm" alt="Explo Dashboard" border />

4. Agora, altere a agregação para **AVG**.

<Image img="/images/integrations/data-visualization/explo_14.webp" size="sm" alt="Explo Dashboard" border />

5. Agora temos o preço médio dos imóveis separado por preço!

<Image img="/images/integrations/data-visualization/explo_15.webp" size="md" alt="Explo Dashboard" />
