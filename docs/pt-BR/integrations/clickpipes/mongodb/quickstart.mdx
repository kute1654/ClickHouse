<div id="refreshable-materialized-view">

</div>

VocГӘ pode criar [VIEWs materializadas atualizГЎveis](/pt-BR/concepts/features/materialized-views/refreshable-materialized-view), que permitem agendar a execuГ§ГЈo de consultas para desduplicar linhas e armazenar os resultados em uma tabela de destino desnormalizada.
