<div id="notes">
  ## Observações
</div>

* A **versão do Chart do Helm do operador** (`1.18587.1`) difere da **tag da imagem** do operador (`main-1.18587.1`). Use a versão do Chart do Helm ao instalar via `helm install` e a tag da imagem ao copiar imagens de contêiner.
* As **compilações FIPS** acrescentam `-fips` às tags de imagem padrão tanto para o servidor quanto para o Keeper.
* As tags de versão são substituídas automaticamente quando esta documentação é compilada. Se você vir variáveis de template sem processamento (por exemplo, `25.12.1.1497`), a etapa de compilação não foi executada.
