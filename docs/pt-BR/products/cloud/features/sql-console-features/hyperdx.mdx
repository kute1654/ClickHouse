HyperDX é a interface de usuário do [**ClickStack**](/pt-BR/clickstack/overview) — uma plataforma de observabilidade pronta para produção, construída sobre ClickHouse e OpenTelemetry (OTel), que unifica logs, traces, métricas e dados de sessão em uma única solução de alto desempenho. Projetado para monitoramento e depuração de sistemas complexos, o ClickStack permite que desenvolvedores e SREs rastreiem problemas de ponta a ponta sem alternar entre ferramentas nem correlacionar dados manualmente usando timestamps ou IDs de correlação.

HyperDX é um frontend criado especificamente para explorar e visualizar dados de observabilidade, com suporte a consultas no estilo Lucene e SQL, dashboards interativos, alertas, exploração de traces e muito mais — tudo otimizado para o ClickHouse como backend.

O HyperDX no ClickHouse Cloud permite que você tenha uma experiência com o ClickStack mais pronta para uso — sem infraestrutura para gerenciar, sem autenticação separada para configurar.
O HyperDX pode ser iniciado com um único clique e conectado aos seus dados — totalmente integrado ao sistema de autenticação do ClickHouse Cloud para um acesso contínuo e seguro aos seus insights de observabilidade.

<div id="main-concepts">
  ## Implantação
</div>

O HyperDX no ClickHouse Cloud está atualmente em versão preliminar privada e precisa ser habilitado no nível da organização. Depois de habilitado, o HyperDX ficará disponível no menu principal de navegação à esquerda ao selecionar qualquer serviço.
