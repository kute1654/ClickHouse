<div id="console-audit-log-events">
  ## Eventos do log de auditoria do Console
</div>

Os diferentes tipos de eventos capturados para a organização são agrupados em 3 categorias: **Organização**, **Serviço** e **Usuário**. Para mais informações sobre o audit log e sobre como exportar ou adicionar uma integração de API, consulte a documentação de [log de auditoria do Console](/pt-BR/products/cloud/guides/security/audit-logging/console-audit-log) na seção Guias acima.

Os eventos a seguir são registrados no audit log.

<div id="organization">
  ### Organização
</div>

* Organização criada
* Organização excluída
* Nome da organização alterado

<div id="service">
  ### Serviço
</div>

* Serviço criado
* Serviço removido
* Serviço interrompido
* Serviço iniciado
* Nome do serviço alterado
* Lista de acesso por IP do serviço alterada
* Senha do serviço redefinida

<div id="user">
  ### Usuário
</div>

* Função do usuário alterada
* Usuário removido da organização
* Usuário convidado para a organização
* Usuário ingressou na organização
* Convite de usuário excluído
* Usuário saiu da organização
