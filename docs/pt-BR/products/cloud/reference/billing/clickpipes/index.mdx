<Note>
  O uso de ClickPipes de CDC para MySQL e MongoDB é gratuito antes da Disponibilidade Geral (GA). Os clientes serão avisados com antecedência antes do lançamento em GA para revisar e otimizar o uso do ClickPipes.
</Note>
