Os motores de banco de dados permitem trabalhar com tabelas. Por padrão, ClickHouse usa o motor de banco de dados [Atomic](/pt-BR/reference/engines/database-engines/atomic), que oferece [motores de tabela](/pt-BR/reference/engines/table-engines/index) configuráveis e um [dialeto SQL](/pt-BR/reference/syntax).

Aqui está a lista completa dos motores de banco de dados disponíveis. Siga os links para mais detalhes:

| [Shared](/pt-BR/reference/engines/database-engines/shared)                                  | Página que descreve o engine de banco de dados `Shared`, disponível no ClickHouse Cloud                                                                                                           |
| [Atomic](/pt-BR/reference/engines/database-engines/atomic)                                  | O engine `Atomic` oferece suporte a consultas `DROP TABLE` e `RENAME TABLE` sem bloqueio, além de consultas atômicas `EXCHANGE TABLES`. O engine de banco de dados `Atomic` é usado por padrão.   |

| [MySQL](/pt-BR/reference/engines/database-engines/mysql)                                    | Permite conectar-se a bancos de dados em um servidor MySQL remoto e executar consultas `INSERT` e `SELECT` para trocar dados entre ClickHouse e MySQL.                                            |
| [SQLite](/pt-BR/reference/engines/database-engines/sqlite)                                  | Permite conectar-se a bancos de dados SQLite e executar consultas `INSERT` e `SELECT` para trocar dados entre ClickHouse e SQLite.                                                                |
| [MaterializedPostgreSQL](/pt-BR/reference/engines/database-engines/materialized-postgresql) | Cria um banco de dados ClickHouse com tabelas de um banco de dados PostgreSQL.                                                                                                                    |
| [Backup](/pt-BR/reference/engines/database-engines/backup)                                  | Permite anexar instantaneamente tabelas/bancos de dados a partir de backups em modo somente leitura.                                                                                              |
| [DataLakeCatalog](/pt-BR/reference/engines/database-engines/datalake)                       | O engine de banco de dados DataLakeCatalog permite conectar o ClickHouse a catálogos de dados externos e consultar dados em formatos de tabela abertos                                            |
