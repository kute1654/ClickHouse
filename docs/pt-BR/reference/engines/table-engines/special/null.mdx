Ao gravar dados em uma tabela `Null`, os dados são ignorados.
Ao ler dados de uma tabela `Null`, a resposta é vazia.

O motor de tabela `Null` é útil para transformações de dados em que você não precisa mais dos dados originais depois que eles são transformados.
Para isso, você pode criar uma visão materializada sobre uma tabela `Null`.
Os dados gravados na tabela serão consumidos pela visão, mas os dados brutos originais serão descartados.
