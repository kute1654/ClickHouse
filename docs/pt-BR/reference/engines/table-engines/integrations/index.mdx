O ClickHouse oferece várias formas de integração com sistemas externos, incluindo motores de tabela. Assim como ocorre com todos os outros motores de tabela, a configuração é feita com consultas `CREATE TABLE` ou `ALTER TABLE`. Depois, do ponto de vista do usuário, a integração configurada se parece com uma tabela normal, mas as consultas a ela são encaminhadas para o sistema externo. Essa transparência nas consultas é uma das principais vantagens dessa abordagem em relação a métodos alternativos de integração, como dicionários ou funções de tabela, que exigem o uso de métodos de consulta personalizados a cada utilização.
