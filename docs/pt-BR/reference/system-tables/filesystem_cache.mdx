<div id="description">
  ## Descrição
</div>

Contém informações sobre todas as entradas no cache do sistema de arquivos de objetos remotos.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `cache_name` ([String](/pt-BR/reference/data-types/index)) — Nome do objeto de cache. Pode ser usado nos comandos `SYSTEM DESCRIBE FILESYSTEM CACHE <name>` e `SYSTEM DROP FILESYSTEM CACHE <name>`
* `cache_base_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para o diretório base onde todos os arquivos de cache (de um cache identificado por `cache_name`) são armazenados.
* `cache_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para um arquivo de cache específico, correspondente a um segmento de arquivo em um arquivo de origem
* `key` ([String](/pt-BR/reference/data-types/index)) — Chave de cache do segmento de arquivo
* `file_segment_range_begin` ([UInt64](/pt-BR/reference/data-types/index)) — Deslocamento correspondente ao início do intervalo do segmento de arquivo
* `file_segment_range_end` ([UInt64](/pt-BR/reference/data-types/index)) — Deslocamento correspondente ao fim (inclusive) do intervalo do segmento de arquivo
* `size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho do segmento de arquivo
* `state` ([String](/pt-BR/reference/data-types/index)) — Estado do segmento de arquivo (DOWNLOADED, DOWNLOADING, PARTIALLY&#95;DOWNLOADED, ...)
* `finished_download_time` ([DateTime](/pt-BR/reference/data-types/index)) — Momento em que o download do segmento de arquivo foi concluído.
* `cache_hits` ([UInt64](/pt-BR/reference/data-types/index)) — Número de hits de cache do segmento de arquivo correspondente
* `references` ([UInt64](/pt-BR/reference/data-types/index)) — Número de referências ao segmento de arquivo correspondente. O valor 1 significa que ninguém o está usando no momento (a única referência existente está no próprio armazenamento de cache)
* `downloaded_size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho baixado do segmento de arquivo
* `kind` ([String](/pt-BR/reference/data-types/index)) — Tipo do segmento de arquivo (usado para distinguir segmentos de arquivo adicionados como parte de &#39;Dados temporários no cache&#39;)
* `unbound` ([UInt8](/pt-BR/reference/data-types/index)) — Flag interna de implementação
* `user_id` ([String](/pt-BR/reference/data-types/index)) — ID do usuário que criou o segmento de arquivo
* `segment_type` ([String](/pt-BR/reference/data-types/index)) — Tipo do segmento. Usado para separar arquivos de dados (`.json`, `.txt` etc.) de arquivo de dados (`.bin`, arquivos de marcação).
* `file_size` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Tamanho do arquivo ao qual o segmento de arquivo atual pertence

{/*AUTOGENERATED_END*/}
