<div id="description">
  ## Descrição
</div>

Esta tabela mostra avisos sobre o servidor ClickHouse.
Avisos do mesmo tipo são combinados em um único aviso.
Por exemplo, se o número N de bancos de dados anexados exceder um limite configurável T, uma única entrada contendo o valor atual N é exibida em vez de N entradas separadas.
Se o valor atual cair abaixo do limite, a entrada será removida da tabela.

A tabela pode ser configurada com os seguintes parâmetros:

* [max&#95;table&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_table_num_to_warn)
* [max&#95;database&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_database_num_to_warn)
* [max&#95;dictionary&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_dictionary_num_to_warn)
* [max&#95;view&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_view_num_to_warn)
* [max&#95;part&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_part_num_to_warn)
* [max&#95;pending&#95;mutations&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_pending_mutations_to_warn)
* [max&#95;pending&#95;mutations&#95;execution&#95;time&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_pending_mutations_execution_time_to_warn)
* [max&#95;named&#95;collection&#95;num&#95;to&#95;warn](/pt-BR/reference/settings/server-settings/settings#max_named_collection_num_to_warn)
* [resource&#95;overload&#95;warnings](/pt-BR/concepts/features/configuration/settings/server-overload#resource-overload-warnings)

<div id="columns">
  ## Colunas
</div>

* `message` ([String](/pt-BR/reference/data-types/string)) — Mensagem de aviso.
* `message_format_string` ([LowCardinality(String)](/pt-BR/reference/data-types/string)) — A string de formatação usada para formatar a mensagem.

<div id="example">
  ## Exemplo
</div>

```sql title="Query"
 SELECT * FROM system.warnings LIMIT 2 \G;
```

```text title="Response"

──────

message_format_string: The number of attached databases is more than {}.
```
