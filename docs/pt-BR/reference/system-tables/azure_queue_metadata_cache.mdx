<div id="description">
  ## Descrição
</div>

Contém o estado em memória dos metadados do AzureQueue e as linhas atualmente processadas por arquivo.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_path` ([String](/pt-BR/reference/data-types/index)) — Caminho no ZooKeeper para os metadados
* `file_path` ([String](/pt-BR/reference/data-types/index)) — Caminho do arquivo que está sendo processado
* `file_name` ([String](/pt-BR/reference/data-types/index)) — Nome do arquivo que está sendo processado
* `rows_processed` ([UInt64](/pt-BR/reference/data-types/index)) — Número de linhas processadas até o momento
* `status` ([String](/pt-BR/reference/data-types/index)) — Status do processamento: Processed, Processing, Failed
* `processing_start_time` ([Nullable(DateTime)](/pt-BR/reference/data-types/index)) — Horário em que o processamento do arquivo foi iniciado
* `processing_end_time` ([Nullable(DateTime)](/pt-BR/reference/data-types/index)) — Horário em que o processamento do arquivo foi encerrado
* `exception` ([String](/pt-BR/reference/data-types/index)) — Exceção ocorrida durante o processamento

{/*AUTOGENERATED_END*/}
