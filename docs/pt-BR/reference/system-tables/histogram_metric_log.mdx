<div id="description">
  ## Descrição
</div>

Histórico de `system.histogram_metrics`. Snapshot capturado a cada `collect_interval_milliseconds`, gravado em disco.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname do servidor que executa a consulta.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do evento.
* `event_time_microseconds` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — Hora do evento com resolução de microssegundos.
* `metric` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome da métrica.
* `labels` ([Map(LowCardinality(String), LowCardinality(String))](/pt-BR/reference/data-types/map)) — Rótulos da métrica.
* `histogram` ([Map(Float64, UInt64)](/pt-BR/reference/data-types/map)) — Histograma cumulativo: mapeia o limite superior do bucket para o número de observações ≤ esse limite; inclui +inf como bucket final.
* `count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Número total de observações, igual a histogram[+inf].
* `sum` ([Float64](/pt-BR/reference/data-types/float)) — Soma de todos os valores observados.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT event_time, metric, labels, histogram
FROM system.histogram_metric_log
WHERE metric = 'keeper_response_time_ms'
ORDER BY event_time DESC
LIMIT 1
FORMAT Vertical;
```
