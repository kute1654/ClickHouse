<div id="description">
  ## Descrição
</div>

Contém informações sobre os codecs de compressão e criptografia.

Você pode usar esta tabela para obter informações sobre os codecs de compressão e criptografia disponíveis.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do codec.
* `method_byte` ([UInt8](/pt-BR/reference/data-types/index)) — Byte que indica o codec no arquivo comprimido.
* `is_compression` ([UInt8](/pt-BR/reference/data-types/index)) — true se este codec comprime algo. Caso contrário, pode ser apenas uma transformação que ajuda na compressão.
* `is_generic_compression` ([UInt8](/pt-BR/reference/data-types/index)) — O codec é um algoritmo de compressão genérico, como lz4 e zstd.
* `is_encryption` ([UInt8](/pt-BR/reference/data-types/index)) — O codec criptografa.
* `is_timeseries_codec` ([UInt8](/pt-BR/reference/data-types/index)) — O codec é destinado a séries temporais de ponto flutuante.
* `is_experimental` ([UInt8](/pt-BR/reference/data-types/index)) — O codec é experimental.
* `description` ([String](/pt-BR/reference/data-types/index)) — Uma descrição geral do codec.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```
