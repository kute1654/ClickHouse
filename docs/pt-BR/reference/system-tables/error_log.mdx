<div id="description">
  ## Descrição
</div>

Contém o histórico dos valores de erro da tabela `system.errors`, periodicamente gravados em disco.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname do servidor que executa a consulta.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do evento.
* `code` ([Int32](/pt-BR/reference/data-types/int-uint)) — Código de erro.
* `error` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome do erro.
* `value` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Número de erros ocorridos no intervalo de tempo.
* `remote` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Exceção remota (isto é, recebida durante uma das consultas distribuídas).
* `last_error_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Horário em que o último erro ocorreu.
* `last_error_message` ([String](/pt-BR/reference/data-types/string)) — Mensagem do último erro.
* `last_error_query_id` ([String](/pt-BR/reference/data-types/string)) — ID da consulta que causou o último erro (se disponível).
* `last_error_trace` ([Array(UInt64)](/pt-BR/reference/data-types/array)) — Um stack trace que representa uma lista de endereços físicos em que os métodos chamados estão armazenados.

{/*AUTOGENERATED_END*/}
