<div id="description">
  ## Descrição
</div>

Exibe todos os tokenizadores disponíveis.
Eles podem ser usados nas funções [tokens](/pt-BR/reference/functions/regular-functions/splitting-merging-functions#tokens), [hasAllTokens](/pt-BR/reference/functions/regular-functions/string-search-functions#hasAllTokens), [hasAnyTokens](/pt-BR/reference/functions/regular-functions/string-search-functions#hasAnyTokens) e no [índice de texto](/pt-BR/reference/engines/table-engines/mergetree-family/textindexes).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do tokenizador

{/*AUTOGENERATED_END*/}

SELECT * FROM system.tokenizers;
