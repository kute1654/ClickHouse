<div id="description">
  ## Descrição
</div>

Contém informações sobre as configurações das tabelas [S3Queue](/pt-BR/reference/engines/table-engines/integrations/s3queue). Disponível a partir da versão `24.10` do servidor.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Banco de dados da tabela com engine S3Queue.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela com engine S3Queue.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da configuração.
* `value` ([String](/pt-BR/reference/data-types/index)) — Valor da configuração.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo da configuração (valor textual específico da implementação).
* `changed` ([UInt8](/pt-BR/reference/data-types/index)) — 1 se a configuração foi definida explicitamente na configuração ou alterada explicitamente.
* `description` ([String](/pt-BR/reference/data-types/index)) — Descrição da configuração.
* `alterable` ([UInt8](/pt-BR/reference/data-types/index)) — Mostra se o usuário atual pode alterar a configuração por meio de ALTER TABLE MODIFY SETTING: 0 — O usuário atual não pode alterar a configuração, 1 — O usuário atual pode alterar a configuração.

{/*AUTOGENERATED_END*/}
