<div id="description">
  ## Descrição
</div>

Contém informações sobre os bancos de dados disponíveis para o usuário atual.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `engine` ([String](/pt-BR/reference/data-types/index)) — Mecanismo do banco de dados.
* `data_path` ([String](/pt-BR/reference/data-types/index)) — Caminho dos dados.
* `metadata_path` ([String](/pt-BR/reference/data-types/index)) — Caminho de metadados.
* `uuid` ([UUID](/pt-BR/reference/data-types/index)) — UUID do banco de dados.
* `engine_full` ([String](/pt-BR/reference/data-types/index)) — Parâmetros do mecanismo do banco de dados.
* `comment` ([String](/pt-BR/reference/data-types/index)) — Comentário de banco de dados.
* `is_external` ([UInt8](/pt-BR/reference/data-types/index)) — O banco de dados é externo (isto é, PostgreSQL/DataLakeCatalog).

{/*AUTOGENERATED_END*/}

A coluna `name` desta tabela do sistema é usada para implementar a consulta `SHOW DATABASES`.

<div id="example">
  ## Exemplo
</div>

Crie um banco de dados.

```sql title="Query"
CREATE DATABASE test;
```

Verifique todos os bancos de dados disponíveis para o usuário.

```sql title="Query"
SELECT * FROM system.databases;
```
