<div id="description">
  ## Descrição
</div>

Contém informações sobre as projeções existentes em todas as tabelas.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da projeção.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](/pt-BR/reference/data-types/index)) — Tipo da projeção.
* `sorting_key` ([Array(String)](/pt-BR/reference/data-types/index)) — Chave de ordenação da projeção.
* `query` ([String](/pt-BR/reference/data-types/index)) — Consulta da projeção.
* `settings` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Configurações da projeção.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
```
