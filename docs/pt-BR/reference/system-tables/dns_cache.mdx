<div id="description">
  ## Descrição
</div>

Contém informações sobre registros DNS armazenados em cache.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/pt-BR/reference/data-types/index)) — Hostname.
* `ip_address` ([String](/pt-BR/reference/data-types/index)) — Endereço IP.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/pt-BR/reference/data-types/index)) — Família de endereços IP.
* `cached_at` ([DateTime](/pt-BR/reference/data-types/index)) — Timestamp em que o registro foi armazenado em cache.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
