<div id="description">
  ## Descrição
</div>

Contém informações sobre os clusters disponíveis no arquivo de configuração e os servidores que fazem parte deles.

<div id="Columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `cluster` ([String](/pt-BR/reference/data-types/index)) — O nome do cluster.
* `shard_num` ([UInt32](/pt-BR/reference/data-types/index)) — O número do shard no cluster, começando em 1.
* `shard_name` ([String](/pt-BR/reference/data-types/index)) — O nome do shard no cluster.
* `shard_weight` ([UInt32](/pt-BR/reference/data-types/index)) — O peso relativo do shard ao gravar dados.
* `internal_replication` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se este host faz parte de um conjunto que pode replicar os dados por conta própria.
* `replica_num` ([UInt32](/pt-BR/reference/data-types/index)) — O número da réplica no shard, começando em 1.
* `host_name` ([String](/pt-BR/reference/data-types/index)) — O nome do host, conforme especificado na configuração.
* `host_address` ([String](/pt-BR/reference/data-types/index)) — O endereço IP do host obtido via DNS.
* `port` ([UInt16](/pt-BR/reference/data-types/index)) — A porta a ser usada para se conectar ao servidor.
* `is_local` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se o host é local.
* `user` ([String](/pt-BR/reference/data-types/index)) — O nome do usuário para se conectar ao servidor.
* `default_database` ([String](/pt-BR/reference/data-types/index)) — O nome do banco de dados padrão.
* `errors_count` ([UInt32](/pt-BR/reference/data-types/index)) — O número de vezes que este host não conseguiu alcançar a réplica.
* `slowdowns_count` ([UInt32](/pt-BR/reference/data-types/index)) — O número de lentidões que levaram à troca de réplica ao estabelecer uma conexão com hedged requests.
* `estimated_recovery_time` ([UInt32](/pt-BR/reference/data-types/index)) — Segundos restantes até que a contagem de erros da réplica seja zerada e ela seja considerada novamente normal.
* `database_shard_name` ([String](/pt-BR/reference/data-types/index)) — O nome do shard do banco de dados `Replicated` (para clusters que pertencem a um banco de dados `Replicated`).
* `database_replica_name` ([String](/pt-BR/reference/data-types/index)) — O nome da réplica do banco de dados `Replicated` (para clusters que pertencem a um banco de dados `Replicated`).
* `is_shared_catalog_cluster` ([UInt8](/pt-BR/reference/data-types/index)) — Bool indicando se o cluster pertence ao Shared Catalog.
* `is_active` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — O status da réplica do banco de dados Replicated (para clusters que pertencem a um banco de dados Replicated): 1 significa &#39;a réplica está online&#39;, 0 significa &#39;a réplica está offline&#39;, NULL significa &#39;desconhecido&#39;.
* `unsynced_after_recovery` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Indica se uma réplica do banco de dados Replicated tem defasagem de replicação maior que max&#95;replication&#95;lag&#95;to&#95;enqueue após a criação ou a recuperação da réplica.
* `replication_lag` ([Nullable(UInt32)](/pt-BR/reference/data-types/index)) — A defasagem de replicação da réplica do banco de dados `Replicated` (para clusters que pertencem a um banco de dados Replicated).
* `recovery_time` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — O tempo de recuperação da réplica do banco de dados `Replicated` (para clusters que pertencem a um banco de dados Replicated), em milissegundos.

{/*AUTOGENERATED_END*/}
