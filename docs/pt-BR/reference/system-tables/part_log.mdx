<div id="description">
  ## Descrição
</div>

A tabela `system.part_log` é criada somente se a configuração do servidor [part&#95;log](/pt-BR/reference/settings/server-settings/settings#part_log) estiver definida.

Esta tabela contém informações sobre eventos ocorridos com [partes de dados](/pt-BR/reference/engines/table-engines/mergetree-family/custom-partitioning-key) em tabelas da família [MergeTree](/pt-BR/reference/engines/table-engines/mergetree-family/mergetree), como adição ou mesclagem de dados.

A tabela `system.part_log` contém as seguintes colunas:

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome do host do servidor que executa a consulta.
* `query_id` ([String](/pt-BR/reference/data-types/string)) — Identificador da consulta INSERT que criou esta parte de dados.
* `event_type` ([Enum8(&#39;NewPart&#39; = 1, &#39;MergeParts&#39; = 2, &#39;DownloadPart&#39; = 3, &#39;RemovePart&#39; = 4, &#39;MutatePart&#39; = 5, &#39;MovePart&#39; = 6, &#39;MergePartsStart&#39; = 7, &#39;MutatePartStart&#39; = 8)](/pt-BR/reference/data-types/enum)) — Tipo do evento que ocorreu com a parte de dados. Pode ter um dos seguintes valores: NewPart — Inserção de uma nova parte de dados, MergePartsStart — A mesclagem das partes de dados foi iniciada, MergeParts — A mesclagem das partes de dados foi concluída, DownloadPart — Download de uma parte de dados, RemovePart — Remoção ou desanexação de uma parte de dados usando [DETACH PARTITION](/pt-BR/reference/statements/alter/partition#detach-partitionpart).MutatePartStart — A mutação de uma parte de dados foi iniciada, MutatePart — A mutação de uma parte de dados foi concluída, MovePart — Movimentação da parte de dados de um disco para outro.
* `merge_reason` ([Enum8(&#39;NotAMerge&#39; = 1, &#39;RegularMerge&#39; = 2, &#39;TTLDeleteMerge&#39; = 3, &#39;TTLRecompressMerge&#39; = 4, &#39;TTLDropMerge&#39; = 5)](/pt-BR/reference/data-types/enum)) — O motivo do evento do tipo MERGE&#95;PARTS. Pode ter um dos seguintes valores: NotAMerge — O evento atual tem um tipo diferente de MERGE&#95;PARTS, RegularMerge — Um merge comum, TTLDeleteMerge, TTLDropMerge — Limpeza de dados expirados. TTLRecompressMerge — Recompressão da parte de dados com o.
* `merge_algorithm` ([Enum8(&#39;Undecided&#39; = 0, &#39;Vertical&#39; = 1, &#39;Horizontal&#39; = 2)](/pt-BR/reference/data-types/enum)) — Algoritmo de mesclagem para o evento do tipo MERGE&#95;PARTS. Pode ter um dos seguintes valores: Undecided, Horizontal, Vertical
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Data e hora do evento.
* `event_time_microseconds` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — Horário do evento com precisão de microssegundos.
* `duration_ms` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Duração dessa operação.
* `database` ([String](/pt-BR/reference/data-types/string)) — Nome do banco de dados ao qual a parte de dados pertence.
* `table` ([String](/pt-BR/reference/data-types/string)) — Nome da tabela à qual a parte de dados pertence.
* `table_uuid` ([UUID](/pt-BR/reference/data-types/uuid)) — UUID da tabela à qual a parte de dados pertence.
* `part_name` ([String](/pt-BR/reference/data-types/string)) — Nome da parte de dados.
* `partition_id` ([String](/pt-BR/reference/data-types/string)) — ID da partição em que a parte de dados foi inserida. A coluna assume o valor `all` se o particionamento for por `tuple()`.
* `partition` ([String](/pt-BR/reference/data-types/string)) — O nome da partição.
* `part_type` ([String](/pt-BR/reference/data-types/string)) — O tipo da parte. Valores possíveis: Wide e Compact.
* `part_storage_type` ([String](/pt-BR/reference/data-types/string)) — O tipo de DataPartStorage. Valores possíveis: Packed - todos os arquivos são armazenados em um único blob, Full - um blob para cada arquivo.
* `disk_name` ([String](/pt-BR/reference/data-types/string)) — O nome do disco em que a parte de dados se encontra.
* `path_on_disk` ([String](/pt-BR/reference/data-types/string)) — Caminho absoluto da pasta que contém os arquivos da parte de dados.
* `rows` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de linhas na parte de dados.
* `size_in_bytes` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Tamanho da parte de dados no disco, em bytes.
* `merged_from` ([Array(String)](/pt-BR/reference/data-types/array)) — Um array com os nomes das partes de origem a partir das quais a parte atual foi formada.
* `bytes_uncompressed` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Tamanho descompactado da parte resultante, em bytes.
* `read_rows` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Número de linhas lidas durante a mesclagem.
* `read_bytes` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de bytes lidos durante a mesclagem.
* `peak_memory_usage` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A quantidade máxima de RAM usada durante a operação de merge
* `deduplication_block_ids` ([Array(String)](/pt-BR/reference/data-types/array)) — Um array de IDs de bloco usado na desduplicação ao inserir esta parte.
* `error` ([UInt16](/pt-BR/reference/data-types/int-uint)) — O código de erro da exceção gerada.
* `exception` ([String](/pt-BR/reference/data-types/string)) — Mensagem de texto do erro ocorrido.
* `mutation_ids` ([Array(String)](/pt-BR/reference/data-types/array)) — Um array de IDs de mutação aplicados à parte de origem (merged&#95;from) para o evento do tipo MUTATE&#95;PART&#95;START e MUTATE&#95;PART.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/pt-BR/reference/data-types/map)) — Todos os eventos de perfil coletados durante esta operação.
* `projections_duration_ms` ([Map(LowCardinality(String), UInt64)](/pt-BR/reference/data-types/map)) — Duração, em milissegundos, da mesclagem/reconstrução de cada projeção.

**Aliases:**

* `ProfileEvents.Names` — é um alias de `mapKeys(ProfileEvents)`.
* `ProfileEvents.Values` — é um alias de `mapValues(ProfileEvents)`.
* `name` — é um alias de `part_name`.

{/*AUTOGENERATED_END*/}

A tabela `system.part_log` é criada após a primeira inserção de dados na tabela `MergeTree`.
