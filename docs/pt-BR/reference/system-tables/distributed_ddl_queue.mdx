<div id="description">
  ## Descrição
</div>

Contém informações sobre [consultas de DDL distribuído (cláusula ON CLUSTER)](/pt-BR/reference/statements/distributed-ddl) que foram executadas em um cluster.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/pt-BR/reference/data-types/index)) — ID da consulta.
* `entry_version` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Versão da entrada.
* `initiator_host` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Host que iniciou a operação DDL.
* `initiator_port` ([Nullable(UInt16)](/pt-BR/reference/data-types/index)) — Porta usada pelo iniciador.
* `cluster` ([String](/pt-BR/reference/data-types/index)) — Nome do cluster, vazio se não tiver sido determinado.
* `query` ([String](/pt-BR/reference/data-types/index)) — Consulta executada.
* `settings` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Configurações usadas na operação DDL.
* `query_create_time` ([DateTime](/pt-BR/reference/data-types/index)) — Hora de criação da consulta.
* `host` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome do host.
* `port` ([Nullable(UInt16)](/pt-BR/reference/data-types/index)) — Porta do host.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/pt-BR/reference/data-types/index)) — Status da consulta.
* `exception_code` ([Nullable(UInt16)](/pt-BR/reference/data-types/index)) — Código de exceção.
* `exception_text` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Mensagem de erro.
* `query_finish_time` ([Nullable(DateTime)](/pt-BR/reference/data-types/index)) — Hora de término da consulta.
* `query_duration_ms` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Duração da execução da consulta (em milissegundos).

{/*AUTOGENERATED_END*/}
