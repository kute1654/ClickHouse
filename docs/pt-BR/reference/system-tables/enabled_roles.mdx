<div id="description">
  ## Descrição
</div>

Contém todos os papéis ativos no momento, incluindo o papel atual do usuário atual e os papéis concedidos ao papel atual.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/pt-BR/reference/data-types/index)) — Nome do papel.
* `with_admin_option` ([UInt8](/pt-BR/reference/data-types/index)) — 1 se o papel tiver o privilégio ADMIN OPTION.
* `is_current` ([UInt8](/pt-BR/reference/data-types/index)) — Sinalizador que indica se `enabled_role` é um papel atual do usuário atual.
* `is_default` ([UInt8](/pt-BR/reference/data-types/index)) — Sinalizador que indica se `enabled_role` é um papel padrão.

{/*AUTOGENERATED_END*/}
