<div id="description">
  ## Descrição
</div>

Contém a lista de motores de banco de dados suportados pelo servidor.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — O nome do motor de banco de dados.
* `description` ([String](/pt-BR/reference/data-types/index)) — Uma descrição geral do que o motor de banco de dados faz.
* `syntax` ([String](/pt-BR/reference/data-types/index)) — Como o motor de banco de dados é especificado na cláusula ENGINE de uma consulta CREATE DATABASE.
* `examples` ([String](/pt-BR/reference/data-types/index)) — Exemplos de uso.
* `introduced_in` ([String](/pt-BR/reference/data-types/index)) — A versão do ClickHouse em que o motor de banco de dados foi introduzido pela primeira vez, no formato major.minor.
* `related` ([Array(String)](/pt-BR/reference/data-types/index)) — Os nomes de motores de banco de dados relacionados.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```
