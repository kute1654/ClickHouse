<div id="description">
  ## Descrição
</div>

Contém os metadados de cada tabela conhecida pelo servidor.

As tabelas [Detached](/pt-BR/reference/statements/detach) não são exibidas em `system.tables`.

As [tabelas temporárias](/pt-BR/reference/statements/create/table#temporary-tables) ficam visíveis em `system.tables` apenas na sessão em que foram criadas. Elas são exibidas com o campo `database` vazio e com a flag `is_temporary` ativada.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — O nome do banco de dados ao qual a tabela pertence.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `uuid` ([UUID](/pt-BR/reference/data-types/index)) — UUID da tabela (banco de dados Atomic).
* `engine` ([String](/pt-BR/reference/data-types/index)) — Nome do motor da tabela (sem parâmetros).
* `is_temporary` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a tabela é temporária.
* `data_paths` ([Array(String)](/pt-BR/reference/data-types/index)) — Caminhos para os dados da tabela nos sistemas de arquivos.
* `metadata_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para os metadados da tabela no sistema de arquivos.
* `metadata_modification_time` ([DateTime](/pt-BR/reference/data-types/index)) — Hora da modificação mais recente dos metadados da tabela.
* `metadata_version` ([Int32](/pt-BR/reference/data-types/index)) — Versão dos metadados para tabela ReplicatedMergeTree, 0 para tabela que não é ReplicatedMergeTree.
* `dependencies_database` ([Array(String)](/pt-BR/reference/data-types/index)) — Dependências do banco de dados.
* `dependencies_table` ([Array(String)](/pt-BR/reference/data-types/index)) — Dependências da tabela (visões materializadas da tabela atual).
* `create_table_query` ([String](/pt-BR/reference/data-types/index)) — A consulta usada para criar a tabela.
* `engine_full` ([String](/pt-BR/reference/data-types/index)) — Parâmetros do motor da tabela.
* `as_select` ([String](/pt-BR/reference/data-types/index)) — Consulta SELECT da visão.
* `parameterized_view_parameters` ([Array(Tuple(name String, type String))](/pt-BR/reference/data-types/index)) — Parâmetros da visão parametrizada.
* `partition_key` ([String](/pt-BR/reference/data-types/index)) — A expressão da chave de partição especificada na tabela.
* `sorting_key` ([String](/pt-BR/reference/data-types/index)) — A expressão da chave de ordenação especificada na tabela.
* `primary_key` ([String](/pt-BR/reference/data-types/index)) — A expressão da chave primária especificada na tabela.
* `sampling_key` ([String](/pt-BR/reference/data-types/index)) — A expressão da chave de amostragem especificada na tabela.
* `unique_key` ([String](/pt-BR/reference/data-types/index)) — A expressão da chave única especificada na tabela (cláusula UNIQUE KEY).
* `storage_policy` ([String](/pt-BR/reference/data-types/index)) — A política de armazenamento. Relevante para tabelas que usam os motores MergeTree e Distributed.
* `total_rows` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Número total de linhas, se for possível determinar rapidamente o número exato de linhas da tabela; caso contrário, NULL (incluindo a Buffer table subjacente).
* `total_bytes` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Número total de bytes, se for possível determinar rapidamente o número exato de bytes da tabela no armazenamento; caso contrário, NULL (não inclui nenhum armazenamento subjacente). Se a tabela armazenar dados em disco, retorna o espaço usado em disco (ou seja, comprimido). Se a tabela armazenar dados em memória, retorna o número aproximado de bytes usados na memória.
* `total_bytes_uncompressed` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Número total de bytes não comprimidos, se for possível determinar rapidamente o número exato de bytes a partir dos checksums das partes da tabela no armazenamento; caso contrário, NULL (não leva em conta o armazenamento subjacente, se houver).
* `parts` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — O número total de partes nesta tabela.
* `active_parts` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — O número de partes ativas nesta tabela.
* `total_marks` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — O número total de marcas em todas as partes desta tabela.
* `active_on_fly_data_mutations` ([UInt64](/pt-BR/reference/data-types/index)) — Número total de mutações de dados ativas (UPDATEs e DELETEs) adequadas para aplicação em tempo real.
* `active_on_fly_alter_mutations` ([UInt64](/pt-BR/reference/data-types/index)) — Número total de mutações ALTER ativas (MODIFY COLUMN) adequadas para aplicação em tempo real.
* `active_on_fly_metadata_mutations` ([UInt64](/pt-BR/reference/data-types/index)) — Número total de mutações de metadados ativas (RENAMEs) adequadas para aplicação em tempo real.
* `columns_descriptions_cache_size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho do cache de descrição de colunas para tabelas *MergeTree
* `lifetime_rows` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Número total de linhas INSERTed desde a inicialização do servidor (somente para tabelas Buffer).
* `lifetime_bytes` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Número total de bytes INSERTed desde a inicialização do servidor (somente para tabelas Buffer).
* `comment` ([String](/pt-BR/reference/data-types/index)) — O comentário da tabela.
* `has_own_data` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a própria tabela armazena dados em disco ou apenas acessa outra origem.
* `loading_dependencies_database` ([Array(String)](/pt-BR/reference/data-types/index)) — Dependências de carregamento do banco de dados (lista de objetos que devem ser carregados antes do objeto atual).
* `loading_dependencies_table` ([Array(String)](/pt-BR/reference/data-types/index)) — Dependências de carregamento da tabela (lista de objetos que devem ser carregados antes do objeto atual).
* `loading_dependent_database` ([Array(String)](/pt-BR/reference/data-types/index)) — Banco de dados dependente para carregamento.
* `loading_dependent_table` ([Array(String)](/pt-BR/reference/data-types/index)) — Tabela dependente para carregamento.
* `target_database` ([String](/pt-BR/reference/data-types/index)) — Para uma visão materializada, o banco de dados da tabela de destino na qual a visão grava (o destino `TO` ou a tabela implícita `.inner.*`). Vazio para outros motores.
* `target_table` ([String](/pt-BR/reference/data-types/index)) — Para uma visão materializada, o nome da tabela de destino na qual a visão grava (o destino `TO` ou a tabela implícita `.inner.*`). Vazio para outros motores.
* `definer` ([String](/pt-BR/reference/data-types/index)) — Nome do definidor de SQL security usado para a tabela.

{/*AUTOGENERATED_END*/}
