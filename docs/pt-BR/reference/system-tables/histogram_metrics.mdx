<div id="description">
  ## Descrição
</div>

Esta tabela contém métricas de histograma que podem ser calculadas instantaneamente e exportadas no formato Prometheus. Ela está sempre atualizada. Substitui a `system.latency_log`, que está obsoleta.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/pt-BR/reference/data-types/index)) — Nome da métrica.
* `value` ([Float64](/pt-BR/reference/data-types/index)) — Valor da métrica.
* `description` ([String](/pt-BR/reference/data-types/index)) — Descrição da métrica.
* `labels` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Rótulos da métrica.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

Você pode usar uma consulta como esta para exportar todas as métricas de histograma no formato do Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
