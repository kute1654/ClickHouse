<div id="description">
  ## Descrição
</div>

Contém informações sobre cada tabela em estado desanexado.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — O nome do banco de dados em que a tabela está.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `uuid` ([UUID](/pt-BR/reference/data-types/index)) — UUID da tabela (banco de dados atomic).
* `metadata_path` ([String](/pt-BR/reference/data-types/index)) — Caminho dos metadados da tabela no sistema de arquivos.
* `is_permanently` ([UInt8](/pt-BR/reference/data-types/index)) — A tabela foi desanexada permanentemente.

{/*AUTOGENERATED_END*/}

  ## Exemplo
</div>

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

