<div id="description">
  ## Descrição
</div>

Esta tabela não existe se este nó não estiver configurado para executar um ClickHouse Keeper no próprio processo. Ela contém uma linha por membro do cluster Raft, combinando a topologia estática do cluster (da configuração do Raft) com a posição de log do próprio nó local.

Cada nó preenche exatamente um valor de `last_log_index` — a linha correspondente ao seu próprio `server_id`. As posições de log dos outros nós não são exibidas aqui porque são rastreadas apenas no líder, e essa visão não é simétrica em todo o cluster.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `server_id` ([Int32](/pt-BR/reference/data-types/int-uint)) — ID do servidor Raft deste membro do cluster.
* `host` ([String](/pt-BR/reference/data-types/string)) — Host extraído do endpoint (prefixo antes dos últimos dois-pontos).
* `endpoint` ([String](/pt-BR/reference/data-types/string)) — Endpoint Raft bruto, conforme configurado (`host:port`).
* `is_observer` ([Bool](/pt-BR/reference/data-types/boolean)) — `true` se este membro for um observador sem direito a voto, configurado por meio de `<can_become_leader>false</can_become_leader>`.
* `priority` ([Int32](/pt-BR/reference/data-types/int-uint)) — Prioridade Raft deste membro; valores mais altos têm preferência durante a eleição de líder.
* `is_leader` ([Bool](/pt-BR/reference/data-types/boolean)) — `true` se este membro for o líder Raft atual.
* `is_self` ([Bool](/pt-BR/reference/data-types/boolean)) — `true` se esta linha descrever o nó Keeper local.
* `last_log_index` ([Nullable(UInt64)](/pt-BR/reference/data-types/int-uint)) — Maior índice de log no armazenamento de log Raft local deste nó. Preenchido apenas na linha correspondente ao nó atual (`is_self = true`); `NULL` para as outras linhas.

{/*AUTOGENERATED_END*/}

Exemplo:

```sql
SELECT * FROM system.keeper_cluster ORDER BY server_id;
```
