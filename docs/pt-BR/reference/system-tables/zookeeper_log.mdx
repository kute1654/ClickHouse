<div id="description">
  ## Descrição
</div>

Esta tabela contém informações sobre os parâmetros da solicitação enviada ao servidor ZooKeeper e sobre a resposta recebida dele.

Nas solicitações, apenas as colunas com os parâmetros da solicitação são preenchidas, e as demais colunas recebem valores padrão (`0` ou `NULL`). Quando a resposta chega, os dados da resposta são adicionados às outras colunas.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname do servidor que executa a consulta.
* `type` ([Enum8(&#39;Request&#39; = 1, &#39;Response&#39; = 2, &#39;Finalize&#39; = 3)](/pt-BR/reference/data-types/enum)) — Tipo de evento no cliente do ZooKeeper. Pode assumir um dos seguintes valores: Request — A solicitação foi enviada, Response — A resposta foi recebida, Finalize — A conexão foi perdida; nenhuma resposta foi recebida.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — A data em que ocorreu o evento.
* `event_time` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — A data e a hora em que o evento ocorreu.
* `thread_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O ID da thread que executou esta requisição.
* `query_id` ([String](/pt-BR/reference/data-types/string)) — O ID da consulta em cujo escopo esta solicitação foi executada.
* `address` ([IPv6](/pt-BR/reference/data-types/ipv6)) — endereço IP do servidor ZooKeeper usado para fazer a solicitação.
* `port` ([UInt16](/pt-BR/reference/data-types/int-uint)) — A porta do servidor ZooKeeper usada para fazer a requisição.
* `session_id` ([Int64](/pt-BR/reference/data-types/int-uint)) — O ID da sessão definido pelo servidor ZooKeeper para cada conexão.
* `duration_microseconds` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O tempo que o ZooKeeper levou para executar a requisição.
* `xid` ([Int64](/pt-BR/reference/data-types/int-uint)) — O ID da solicitação na sessão. Geralmente, é um número sequencial da solicitação. Ele é o mesmo tanto para a linha da solicitação quanto para a linha correspondente de resposta/finalização.
* `has_watch` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Indica se o watch foi definido.
* `op_num` ([Enum16(&#39;Close&#39; = -11, &#39;Error&#39; = -1, &#39;Watch&#39; = 0, &#39;Create&#39; = 1, &#39;Remove&#39; = 2, &#39;Exists&#39; = 3, &#39;Get&#39; = 4, &#39;Set&#39; = 5, &#39;GetACL&#39; = 6, &#39;SetACL&#39; = 7, &#39;SimpleList&#39; = 8, &#39;Sync&#39; = 9, &#39;Heartbeat&#39; = 11, &#39;List&#39; = 12, &#39;Check&#39; = 13, &#39;Multi&#39; = 14, &#39;Create2&#39; = 15, &#39;Reconfig&#39; = 16, &#39;CheckWatch&#39; = 17, &#39;RemoveWatch&#39; = 18, &#39;MultiRead&#39; = 22, &#39;Auth&#39; = 100, &#39;SetWatch&#39; = 101, &#39;SetWatch2&#39; = 105, &#39;AddWatch&#39; = 106, &#39;FilteredList&#39; = 500, &#39;CheckNotExists&#39; = 501, &#39;CreateIfNotExists&#39; = 502, &#39;RemoveRecursive&#39; = 503, &#39;CheckStat&#39; = 504, &#39;TryRemove&#39; = 505, &#39;FilteredListWithStatsAndData&#39; = 506, &#39;ListRecursive&#39; = 507, &#39;SessionID&#39; = 997)](/pt-BR/reference/data-types/enum)) — Tipo de solicitação ou resposta.
* `path` ([String](/pt-BR/reference/data-types/string)) — O caminho para o nó do ZooKeeper especificado na solicitação, ou uma string vazia se a solicitação não exigir a especificação de um caminho.
* `data` ([String](/pt-BR/reference/data-types/string)) — Os dados gravados no nó do ZooKeeper (nas requisições SET e CREATE — o que a requisição queria gravar; na resposta à requisição GET — o que foi lido) ou uma string vazia.
* `is_ephemeral` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Indica se o nó do ZooKeeper está sendo criado como efêmero.
* `is_sequential` ([UInt8](/pt-BR/reference/data-types/int-uint)) — O nó do ZooKeeper está sendo criado como um nó sequencial.
* `version` ([Nullable(Int32)](/pt-BR/reference/data-types/nullable)) — A versão do nó do ZooKeeper esperada pela solicitação durante a execução. Há suporte para solicitações CHECK, SET e REMOVE (o valor relevante é -1 se a solicitação não verificar a versão, ou NULL para outras solicitações que não oferecem suporte à verificação de versão).
* `requests_size` ([UInt32](/pt-BR/reference/data-types/int-uint)) — O número de solicitações incluídas na solicitação múltipla (esta é uma solicitação especial que consiste em várias solicitações comuns consecutivas e as executa de forma atômica). Todas as solicitações incluídas na solicitação múltipla terão o mesmo xid.
* `request_idx` ([UInt32](/pt-BR/reference/data-types/int-uint)) — O número da solicitação incluída em uma multissolicitação (para multissolicitação — 0, depois em ordem, a partir de 1).
* `zxid` ([Int64](/pt-BR/reference/data-types/int-uint)) — ID da transação do ZooKeeper. O número de série emitido pelo servidor ZooKeeper em resposta a uma solicitação executada com sucesso (0 se a solicitação não tiver sido executada/retornar um erro/o cliente não souber se a solicitação foi executada).
* `error` ([Nullable(Enum8(&#39;ZNOWATCHER&#39; = -121, &#39;ZNOTREADONLY&#39; = -119, &#39;ZSESSIONMOVED&#39; = -118, &#39;ZNOTHING&#39; = -117, &#39;ZCLOSING&#39; = -116, &#39;ZAUTHFAILED&#39; = -115, &#39;ZINVALIDACL&#39; = -114, &#39;ZINVALIDCALLBACK&#39; = -113, &#39;ZSESSIONEXPIRED&#39; = -112, &#39;ZNOTEMPTY&#39; = -111, &#39;ZNODEEXISTS&#39; = -110, &#39;ZNOCHILDRENFOREPHEMERALS&#39; = -108, &#39;ZBADVERSION&#39; = -103, &#39;ZNOAUTH&#39; = -102, &#39;ZNONODE&#39; = -101, &#39;ZAPIERROR&#39; = -100, &#39;ZOUTOFMEMORY&#39; = -10, &#39;ZINVALIDSTATE&#39; = -9, &#39;ZBADARGUMENTS&#39; = -8, &#39;ZOPERATIONTIMEOUT&#39; = -7, &#39;ZUNIMPLEMENTED&#39; = -6, &#39;ZMARSHALLINGERROR&#39; = -5, &#39;ZCONNECTIONLOSS&#39; = -4, &#39;ZDATAINCONSISTENCY&#39; = -3, &#39;ZRUNTIMEINCONSISTENCY&#39; = -2, &#39;ZSYSTEMERROR&#39; = -1, &#39;ZOK&#39; = 0))](/pt-BR/reference/data-types/nullable)) — Código de erro. Pode ter vários valores; estes são apenas alguns deles: ZOK — A solicitação foi executada com sucesso, ZCONNECTIONLOSS — A conexão foi perdida, ZOPERATIONTIMEOUT — O tempo limite de execução da solicitação expirou, ZSESSIONEXPIRED — A sessão expirou, NULL — A solicitação foi concluída.
* `watch_type` ([Nullable(Enum8(&#39;NOTWATCHING&#39; = -2, &#39;SESSION&#39; = -1, &#39;CREATED&#39; = 1, &#39;DELETED&#39; = 2, &#39;CHANGED&#39; = 3, &#39;CHILD&#39; = 4))](/pt-BR/reference/data-types/nullable)) — O tipo do evento watch (para respostas com op&#95;num = Watch); para as demais respostas: NULL.
* `watch_state` ([Nullable(Enum16(&#39;AUTH&#95;FAILED&#39; = -113, &#39;EXPIRED&#95;SESSION&#39; = -112, &#39;CONNECTING&#39; = 1, &#39;ASSOCIATING&#39; = 2, &#39;CONNECTED&#39; = 3, &#39;READONLY&#39; = 5, &#39;NOTCONNECTED&#39; = 999))](/pt-BR/reference/data-types/nullable)) — O status do evento de watch (para respostas com op&#95;num = Watch); para as demais respostas: NULL.
* `path_created` ([String](/pt-BR/reference/data-types/string)) — O caminho do nó ZooKeeper criado (nas respostas à solicitação CREATE) pode ser diferente do caminho se o nó for criado no modo sequencial.
* `stat_czxid` ([Int64](/pt-BR/reference/data-types/int-uint)) — O zxid da alteração que levou à criação deste nó do ZooKeeper.
* `stat_mzxid` ([Int64](/pt-BR/reference/data-types/int-uint)) — O zxid da última alteração feita neste nó do ZooKeeper.
* `stat_pzxid` ([Int64](/pt-BR/reference/data-types/int-uint)) — O ID da transação correspondente à alteração que modificou pela última vez os filhos deste nó do ZooKeeper.
* `stat_version` ([Int32](/pt-BR/reference/data-types/int-uint)) — O número de alterações nos dados deste nó do ZooKeeper.
* `stat_cversion` ([Int32](/pt-BR/reference/data-types/int-uint)) — O número de alterações nos nós filhos deste nó do ZooKeeper.
* `stat_dataLength` ([Int32](/pt-BR/reference/data-types/int-uint)) — O tamanho do campo de dados deste nó do ZooKeeper.
* `stat_numChildren` ([Int32](/pt-BR/reference/data-types/int-uint)) — O número de nós filhos deste nó do ZooKeeper.
* `children` ([Array(String)](/pt-BR/reference/data-types/array)) — Lista de nós filhos do ZooKeeper (para respostas à solicitação LIST).

{/*AUTOGENERATED_END*/}
