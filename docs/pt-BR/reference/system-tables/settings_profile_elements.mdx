<div id="description">
  ## Descrição
</div>

Descreve o conteúdo do perfil de configurações:

* Restrições.
* Funções e usuários aos quais a configuração se aplica.
* Perfis de configurações pai.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome do perfil de configurações.
* `user_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome do usuário.
* `role_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome da função.
* `index` ([UInt64](/pt-BR/reference/data-types/index)) — Número sequencial do elemento do perfil de configurações.
* `setting_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome da configuração.
* `value` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Valor da configuração.
* `min` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Valor mínimo da configuração. NULL se não estiver definido.
* `max` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Valor máximo da configuração. NULL se não estiver definido.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/pt-BR/reference/data-types/index)) — Propriedade que indica se uma configuração pode ser alterada ou não.
* `inherit_profile` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Perfil pai deste perfil de configurações. NULL se não estiver definido. O perfil de configurações herdará todos os valores e as restrições das configurações (min, max, readonly) de seus perfis pai.

{/*AUTOGENERATED_END*/}
