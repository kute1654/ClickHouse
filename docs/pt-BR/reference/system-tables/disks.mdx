<div id="description">
  ## Descrição
</div>

Contém informações sobre os disks definidos na [configuração do servidor](/pt-BR/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome de um disk na configuração do servidor.
* `path` ([String](/pt-BR/reference/data-types/index)) — Caminho para o ponto de montagem no sistema de arquivos.
* `free_space` ([UInt64](/pt-BR/reference/data-types/index)) — Espaço livre no disk, em bytes.
* `total_space` ([UInt64](/pt-BR/reference/data-types/index)) — Capacidade total do disk, em bytes.
* `unreserved_space` ([UInt64](/pt-BR/reference/data-types/index)) — Espaço livre que não está comprometido com reservas (`free&#95;space` menos o tamanho das reservas usadas por merges, inserções e outras operações de gravação em disk em execução no momento).
* `keep_free_space` ([UInt64](/pt-BR/reference/data-types/index)) — Quantidade de espaço em disk que deve permanecer livre, em bytes. Definida no parâmetro keep&#95;free&#95;space&#95;bytes da configuração do disk.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo de disk, que informa onde este disk armazena os dados — RAM, unidade local ou armazenamento remoto.
* `object_storage_type` ([String](/pt-BR/reference/data-types/index)) — Tipo de armazenamento de objetos, se o tipo de disk for object&#95;storage.
* `metadata_type` ([String](/pt-BR/reference/data-types/index)) — Tipo de armazenamento de metadados, se o tipo de disk for object&#95;storage.
* `is_encrypted` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que mostra se este disk criptografa os dados subjacentes.
* `is_read_only` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica que você só pode realizar operações de leitura com este disk.
* `is_write_once` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se o disk é write-once. Isso significa que ele oferece suporte a BACKUP nesse disk, mas não oferece suporte a INSERT em tabela MergeTree nesse disk.
* `is_remote` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica quais operações com este disk envolvem interação de rede.
* `is_broken` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se o disk está com defeito. Disks com defeito terão 0 bytes de espaço e não poderão ser usados.
* `cache_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para o diretório de cache na unidade local, caso o disk ofereça suporte a cache.

{/*AUTOGENERATED_END*/}
