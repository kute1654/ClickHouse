<div id="description">
  ## Descrição
</div>

Contém uma lista dos modelos CatBoost carregados na memória do LibraryBridge, junto com o momento em que foram carregados.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `model_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para o modelo treinado.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo do modelo. No momento, somente catboost.
* `loading_start_time` ([DateTime](/pt-BR/reference/data-types/index)) — Horário em que o carregamento do modelo foi iniciado.
* `loading_duration` ([Float32](/pt-BR/reference/data-types/index)) — Tempo necessário para carregar o modelo.

{/*AUTOGENERATED_END*/}
