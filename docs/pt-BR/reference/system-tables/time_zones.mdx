<div id="description">
  ## Descrição
</div>

Contém uma lista de fusos horários compatíveis com o servidor ClickHouse. Essa lista de fusos horários pode variar dependendo da versão do ClickHouse.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/pt-BR/reference/data-types/index)) — Lista de fusos horários suportados.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```
