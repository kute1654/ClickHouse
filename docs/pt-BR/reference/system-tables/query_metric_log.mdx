<div id="description">
  ## Descrição
</div>

Contém um histórico dos valores de memória e métricas da tabela `system.events` para consultas individuais, gravados periodicamente em disco.

Assim que uma consulta é iniciada, os dados são coletados em intervalos periódicos de `query_metric_log_interval` milissegundos (definido como 1000
por padrão). Os dados também são coletados quando a consulta termina, caso ela leve mais do que `query_metric_log_interval`.
