<div id="description">
  ## Descrição
</div>

Contém os valores históricos de `system.asynchronous_metrics`, que são salvos uma vez por intervalo de tempo (um segundo por padrão). É habilitado por padrão.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname do servidor que executa a consulta.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do evento.
* `metric` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome da métrica.
* `value` ([Float64](/pt-BR/reference/data-types/float)) — Valor da métrica.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.asynchronous_metric_log LIMIT 3 \G
```
