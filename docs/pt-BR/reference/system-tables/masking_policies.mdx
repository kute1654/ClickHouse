<div id="description">
  ## Descrição
</div>

Contém informações sobre todas as políticas de mascaramento definidas no sistema.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/string)) — Nome de uma política de mascaramento. O formato do nome completo é `short_name ON database.table`.
* `short_name` ([String](/pt-BR/reference/data-types/string)) — Nome curto de uma política de mascaramento. Por exemplo, se o nome completo for `mask_email ON mydb.mytable`, o nome curto é `mask_email`.
* `database` ([String](/pt-BR/reference/data-types/string)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/string)) — Nome da tabela.
* `id` ([UUID](/pt-BR/reference/data-types/uuid)) — ID da política de mascaramento.
* `storage` ([String](/pt-BR/reference/data-types/string)) — Nome do diretório em que a política de mascaramento é armazenada.
* `update_assignments` ([Nullable(String)](/pt-BR/reference/data-types/nullable)) — Atribuições de UPDATE que definem como os dados devem ser mascarados. Por exemplo: `email = '***masked***', phone = '***-***-****'`.
* `where_condition` ([Nullable(String)](/pt-BR/reference/data-types/nullable)) — Condição WHERE opcional que especifica quando o mascaramento deve ser aplicado.
* `priority` ([Int64](/pt-BR/reference/data-types/int-uint)) — Prioridade para aplicar várias políticas de mascaramento. As políticas com prioridade mais alta são aplicadas primeiro. O padrão é 0.
* `apply_to_all` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Indica se a política de mascaramento se aplica a todas as roles e/ou usuários. 1 se `true`, 0 caso contrário.
* `apply_to_list` ([Array(String)](/pt-BR/reference/data-types/array)) — Lista de roles e/ou usuários aos quais a política de mascaramento é aplicada.
* `apply_to_except` ([Array(String)](/pt-BR/reference/data-types/array)) — A política de mascaramento é aplicada a todas as roles e/ou usuários, exceto aos listados. Preenchido apenas quando `apply_to_all` é 1.

{/*AUTOGENERATED_END*/}
