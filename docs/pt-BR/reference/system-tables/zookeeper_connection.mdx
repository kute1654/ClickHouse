<div id="description">
  ## Descrição
</div>

Esta tabela não existe se o ZooKeeper não estiver configurado. A tabela &#39;system.zookeeper&#95;connection&#39; mostra as conexões atuais com o ZooKeeper (incluindo os ZooKeepers auxiliares). Cada linha exibe informações sobre uma conexão.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/string)) — Nome do cluster do ZooKeeper.
* `host` ([String](/pt-BR/reference/data-types/string)) — O hostname/IP do nó do ZooKeeper ao qual o ClickHouse se conectou.
* `port` ([UIn16](/pt-BR/reference/data-types/int-uint)) — A porta do nó do ZooKeeper ao qual o ClickHouse se conectou.
* `index` ([Nullable(UInt8)](/pt-BR/reference/data-types/int-uint)) — O índice do nó do ZooKeeper ao qual o ClickHouse se conectou. O índice vem da configuração do ZooKeeper. Se não houver conexão, esta coluna será NULL.
* `connected_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Quando a conexão foi estabelecida
* `session_uptime_elapsed_seconds` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Segundos decorridos desde que a conexão foi estabelecida.
* `is_expired` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Se a conexão atual expirou.
* `keeper_api_version` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Versão da API do Keeper.
* `client_id` ([Int64](/pt-BR/reference/data-types/int-uint)) — ID da sessão da conexão.
* `xid` ([Int64](/pt-BR/reference/data-types/int-uint)) — XID da sessão atual.
* `enabled_feature_flags` ([Array(Enum16)](/pt-BR/reference/data-types/array)) — Feature flags habilitadas. Aplicável somente ao ClickHouse Keeper. Os valores possíveis são `FILTERED_LIST`, `MULTI_READ`, `CHECK_NOT_EXISTS`, `CREATE_IF_NOT_EXISTS`, `REMOVE_RECURSIVE`.
* `availability_zone` ([String](/pt-BR/reference/data-types/string)) — Zona de disponibilidade.
* `session_timeout_ms` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Timeout da sessão (em milissegundos).
* `last_zxid_seen` ([Int64](/pt-BR/reference/data-types/int-uint)) — Último zxid visto pela sessão atual.

{/*AUTOGENERATED_END*/}
