<div id="description">
  ## Descrição
</div>

Exibe o conteúdo do [cache de condições de consulta](/pt-BR/concepts/features/performance/caches/query-condition-cache).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `key_hash` ([UInt128](/pt-BR/reference/data-types/index)) — Hash de (table&#95;uuid, part&#95;name, condition&#95;hash).
* `entry_size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho da entrada em bytes.
* `matching_marks` ([String](/pt-BR/reference/data-types/index)) — Marcas correspondentes.

{/*AUTOGENERATED_END*/}
