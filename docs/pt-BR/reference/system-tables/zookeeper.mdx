<div id="description">
  ## Descrição
</div>

A tabela não existe a menos que o ClickHouse Keeper ou o ZooKeeper esteja configurado. A tabela `system.zookeeper` expõe dados dos clusters do Keeper definidos na configuração.
A consulta deve ter uma condição `path =`   ou uma condição `path IN`  definida na cláusula `WHERE`, como mostrado abaixo. Isso corresponde ao caminho dos filhos dos quais você deseja obter dados.

A consulta `SELECT * FROM system.zookeeper WHERE path = '/clickhouse'` retorna dados de todos os filhos do nó `/clickhouse`.
Para retornar dados de todos os nós raiz, use path = &#39;/&#39;.
Se o caminho especificado em &#39;path&#39; não existir, uma exceção será lançada.

A consulta `SELECT * FROM system.zookeeper WHERE path IN ('/', '/clickhouse')` retorna dados de todos os filhos dos nós `/` e `/clickhouse`.
Se houver, na coleção &#39;path&#39; especificada, um caminho que não existe, uma exceção será lançada.
Isso pode ser usado para fazer um lote de consultas de caminhos do Keeper.

A consulta `SELECT * FROM system.zookeeper WHERE path = '/clickhouse' AND zookeeperName = 'auxiliary_cluster'` retorna dados no cluster ZooKeeper `auxiliary_cluster`.
Se o `auxiliary_cluster` especificado não existir, uma exceção será lançada.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` (String) — O nome do nó.
* `path` (String) — O caminho do nó.
* `value` (String) — Valor do nó.
* `zookeeperName` (String) — O nome do cluster ZooKeeper padrão ou de um dos clusters ZooKeeper auxiliares.
* `dataLength` (Int32) — Tamanho do valor.
* `numChildren` (Int32) — Número de descendentes.
* `czxid` (Int64) — ID da transaction que criou o nó.
* `mzxid` (Int64) — ID da transaction que alterou o nó pela última vez.
* `pzxid` (Int64) — ID da transaction que excluiu ou adicionou descendentes pela última vez.
* `ctime` (DateTime) — Horário de criação do nó.
* `mtime` (DateTime) — Horário da última modificação do nó.
* `version` (Int32) — Versão do nó: o número de vezes que o nó foi alterado.
* `cversion` (Int32) — Número de descendentes adicionados ou removidos.
* `aversion` (Int32) — Número de alterações na ACL.
* `ephemeralOwner` (Int64) — Para nós efêmeros, o ID da session proprietária deste nó.

{/*AUTOGENERATED_END*/}

```sql
SELECT *
FROM system.zookeeper
WHERE path = '/clickhouse/tables/01-08/visits/replicas'
FORMAT Vertical
```
