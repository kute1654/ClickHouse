<div id="description">
  ## Descrição
</div>

Contém informações sobre [quotas](/pt-BR/reference/system-tables/quotas).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da quota.
* `id` ([UUID](/pt-BR/reference/data-types/index)) — ID da quota.
* `storage` ([String](/pt-BR/reference/data-types/index)) — Armazenamento das quotas. Valor possível: &#39;users.xml&#39; se uma quota estiver configurada no arquivo users.xml, &#39;disk&#39; se uma quota estiver configurada por uma consulta SQL.
* `keys` ([Array(Enum8(&#39;user&#95;name&#39; = 1, &#39;ip&#95;address&#39; = 2, &#39;forwarded&#95;ip&#95;address&#39; = 3, &#39;client&#95;key&#39; = 4, &#39;normalized&#95;query&#95;hash&#39; = 7))](/pt-BR/reference/data-types/index)) — A chave especifica como a quota deve ser compartilhada. Se duas conexões usarem a mesma quota e a mesma chave, elas compartilharão as mesmas quantidades de recursos. Valores: [] — Todos os usuários compartilham a mesma quota, [&#39;user&#95;name&#39;] — Conexões com o mesmo nome de usuário compartilham a mesma quota, [&#39;ip&#95;address&#39;] — Conexões do mesmo IP compartilham a mesma quota, [&#39;client&#95;key&#39;] — Conexões com a mesma chave compartilham a mesma quota. Uma chave deve ser fornecida explicitamente por um cliente. Ao usar clickhouse-client, passe um valor de chave no parâmetro --quota&#95;key ou use o parâmetro quota&#95;key no arquivo de configuração do cliente. Ao usar a interface HTTP, use o cabeçalho X-ClickHouse-Quota, [&#39;user&#95;name&#39;, &#39;client&#95;key&#39;] — Conexões com a mesma client&#95;key compartilham a mesma quota. Se uma chave não for fornecida por um cliente, a quota será controlada por `user_name`, [&#39;client&#95;key&#39;, &#39;ip&#95;address&#39;] — Conexões com a mesma client&#95;key compartilham a mesma quota. Se uma chave não for fornecida por um cliente, a quota será controlada por ip&#95;address.
* `durations` ([Array(UInt32)](/pt-BR/reference/data-types/index)) — Duração dos intervalos de tempo em segundos.
* `apply_to_all` ([UInt8](/pt-BR/reference/data-types/index)) — Valor lógico. Mostra a quais usuários a quota é aplicada. Valores: 0 — A quota se aplica aos usuários especificados em apply&#95;to&#95;list. 1 — A quota se aplica a todos os usuários, exceto aos listados em apply&#95;to&#95;except.
* `apply_to_list` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de nomes de usuários/roles aos quais a quota deve ser aplicada.
* `apply_to_except` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de nomes de usuários/roles aos quais a quota não deve ser aplicada.
* `ipv4_prefix_bits` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Número de bits de prefixo para mascaramento de endereço IPv4. Conexões da mesma sub-rede compartilham a mesma quota. Usado somente quando a chave é `ip_address` ou `forwarded_ip_address`.
* `ipv6_prefix_bits` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Número de bits de prefixo para mascaramento de endereço IPv6. Conexões da mesma sub-rede compartilham a mesma quota. Usado somente quando a chave é `ip_address` ou `forwarded_ip_address`.
