<div id="description">
  ## Descrição
</div>

Contém informações sobre os colaboradores. A ordem é aleatória durante a execução da consulta.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do colaborador (author) no git log.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.contributors LIMIT 10
```

```text
┌─name─────────────┐

│ Max Vetrov       │
│ LiuYangkuan      │
│ svladykin        │
│ zamulla          │
│ Šimon Podlipský  │
│ BayoNet          │
│ Ilya Khomutov    │
│ Amy Krishnevsky  │
│ Loud_Scream      │
```

Para localizar seus próprios dados na tabela, use uma consulta:

```sql
SELECT * FROM system.contributors WHERE name = 'Olga Khvostikova'
```

```text
┌─name─────────────┐
│ Olga Khvostikova │
└──────────────────┘
```
