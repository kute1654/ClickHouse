<div id="description">
  ## Descrição
</div>

Contém uma lista de todos os combinadores de funções de agregação disponíveis, que podem ser aplicados às funções de agregação e alterar seu funcionamento.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — O nome do combinador.
* `is_internal` ([UInt8](/pt-BR/reference/data-types/index)) — Indica se este combinador é apenas para uso interno.
* `description` ([String](/pt-BR/reference/data-types/index)) — Uma descrição geral do que o combinador faz.
* `syntax` ([String](/pt-BR/reference/data-types/index)) — Como o combinador é aplicado ao nome de uma função de agregação.
* `examples` ([String](/pt-BR/reference/data-types/index)) — Exemplos de uso.
* `introduced_in` ([String](/pt-BR/reference/data-types/index)) — A versão do ClickHouse em que o combinador foi introduzido pela primeira vez, no formato major.minor.
* `related` ([Array(String)](/pt-BR/reference/data-types/index)) — Os nomes dos combinadores relacionados.

{/*AUTOGENERATED_END*/}
