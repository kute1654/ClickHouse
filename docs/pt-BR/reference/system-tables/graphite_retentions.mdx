<div id="description">
  ## Descrição
</div>

Contém informações sobre os parâmetros [graphite&#95;rollup](/pt-BR/reference/settings/server-settings/settings#graphite), usados em tabelas com motores [*GraphiteMergeTree](/pt-BR/reference/engines/table-engines/mergetree-family/graphitemergetree).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/pt-BR/reference/data-types/index)) — nome do parâmetro graphite&#95;rollup.
* `rule_type` ([String](/pt-BR/reference/data-types/index)) — O tipo de regra. Valores possíveis: RuleTypeAll = 0 - padrão, com regex, compatível com o formato antigo; RuleTypePlain = 1 - métricas simples, com regex, compatível com o formato antigo; RuleTypeTagged = 2 - métricas com tag, com regex, compatível com o formato antigo; RuleTypeTagList = 3 - métricas com tag, com regex (convertido para RuleTypeTagged a partir de uma string como &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](/pt-BR/reference/data-types/index)) — Um padrão para o nome da métrica.
* `function` ([String](/pt-BR/reference/data-types/index)) — O nome da função de agregação.
* `age` ([UInt64](/pt-BR/reference/data-types/index)) — A idade mínima dos dados em segundos.
* `precision` ([UInt64](/pt-BR/reference/data-types/index)) — Com que precisão definir a idade dos dados em segundos.
* `priority` ([UInt16](/pt-BR/reference/data-types/index)) — Prioridade do padrão.
* `is_default` ([UInt8](/pt-BR/reference/data-types/index)) — Se é o padrão.
* `Tables.database` ([Array(String)](/pt-BR/reference/data-types/index)) — Array com nomes de tabelas do banco de dados que usam o parâmetro `config_name`.
* `Tables.table` ([Array(String)](/pt-BR/reference/data-types/index)) — Array com nomes de tabelas que usam o parâmetro `config_name`.
