Contém informações sobre todas as transações executadas no servidor atual.

É seguro truncar ou excluir esta tabela a qualquer momento.

Colunas:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — O hostname em que a transação foi executada.
* `type` ([Enum8(&#39;Begin&#39; = 1, &#39;Commit&#39; = 2, &#39;Rollback&#39; = 3, &#39;AddPart&#39; = 10, &#39;LockPart&#39; = 11, &#39;UnlockPart&#39; = 12)](/pt-BR/reference/data-types/enum)) — O tipo da transação. Valores possíveis: Begin, Commit, Rollback, AddPart, LockPart, UnlockPart.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do registro.
* `event_time` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — Hora do registro
* `thread_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O identificador de uma thread.
* `query_id` ([String](/pt-BR/reference/data-types/string)) — O ID de uma consulta executada no escopo da transação.
* `tid` ([Tuple(UInt64, UInt64, UUID)](/pt-BR/reference/data-types/tuple)) — O identificador de uma transação.
* `tid_hash` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O hash do identificador.
* `csn` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de sequência de commit
* `database` ([String](/pt-BR/reference/data-types/string)) — O nome do banco de dados no qual a transação foi executada.
* `table` ([String](/pt-BR/reference/data-types/string)) — O nome da tabela na qual a transação foi executada.
* `uuid` ([UUID](/pt-BR/reference/data-types/uuid)) — O UUID da tabela na qual a transação foi executada.
* `part` ([String](/pt-BR/reference/data-types/string)) — O nome da part que participou da transação.

{/*AUTOGENERATED_END*/}
