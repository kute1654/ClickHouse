<div id="description">
  ## Descrição
</div>

Esta tabela de sistema contém os metadados de cada arquivo das tabelas Iceberg existentes no ClickHouse, com uma linha para cada arquivo de dados ou de exclusão referenciado pelo snapshot atual de cada tabela. Ela ficará vazia se você não tiver nenhuma tabela Iceberg no ClickHouse.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `snapshot_id` ([Int64](/pt-BR/reference/data-types/index)) — ID do snapshot em que o arquivo foi adicionado.
* `content` ([Enum8(&#39;DATA&#39; = 0, &#39;POSITION&#95;DELETE&#39; = 1, &#39;EQUALITY&#95;DELETE&#39; = 2)](/pt-BR/reference/data-types/index)) — Tipo de conteúdo do arquivo.
* `file_path` ([String](/pt-BR/reference/data-types/index)) — Caminho de armazenamento resolvido do arquivo.
* `file_format` ([String](/pt-BR/reference/data-types/index)) — Formato do arquivo, por exemplo, &#39;PARQUET&#39;.
* `record_count` ([Int64](/pt-BR/reference/data-types/index)) — Número de registros no arquivo.
* `file_size_in_bytes` ([Int64](/pt-BR/reference/data-types/index)) — Tamanho do arquivo em bytes.
* `partition` ([String](/pt-BR/reference/data-types/index)) — Representação textual da tupla de partição.
* `schema_id` ([Int32](/pt-BR/reference/data-types/index)) — ID do esquema resolvido para esta entrada do manifesto.
* `sequence_number` ([Int64](/pt-BR/reference/data-types/index)) — Número de sequência resolvido da entrada do manifesto (sempre 0 para o formato v1).
* `sort_order_id` ([Nullable(Int32)](/pt-BR/reference/data-types/index)) — ID da ordem de classificação do arquivo, se especificado.
* `null_value_counts` ([Map(Int32, Int64)](/pt-BR/reference/data-types/index)) — Contagem de valores NULL por coluna (ID da coluna -&gt; contagem).
* `column_sizes` ([Map(Int32, Int64)](/pt-BR/reference/data-types/index)) — Tamanho em bytes em disco por coluna (ID da coluna -&gt; bytes).
* `value_counts` ([Map(Int32, Int64)](/pt-BR/reference/data-types/index)) — Contagem total de valores por coluna (ID da coluna -&gt; contagem).
* `equality_ids` ([Array(Int32)](/pt-BR/reference/data-types/index)) — IDs dos campos de igualdade para arquivos de exclusão por igualdade (vazio para arquivos que não são de exclusão por igualdade).
