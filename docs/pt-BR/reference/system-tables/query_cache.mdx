<div id="description">
  ## Descrição
</div>

Exibe o conteúdo do [cache de consultas](/pt-BR/concepts/features/performance/caches/query-cache).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/pt-BR/reference/data-types/index)) — Texto da consulta.
* `query_id` ([String](/pt-BR/reference/data-types/index)) — ID da consulta.
* `is_subquery` ([UInt8](/pt-BR/reference/data-types/index)) — Se a entrada do cache de consultas é o resultado de uma subconsulta.
* `result_size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho da entrada do cache de consultas.
* `tag` ([LowCardinality(String)](/pt-BR/reference/data-types/index)) — Tag da entrada do cache de consultas.
* `stale` ([UInt8](/pt-BR/reference/data-types/index)) — Se a entrada do cache de consultas está obsoleta.
* `shared` ([UInt8](/pt-BR/reference/data-types/index)) — Se a entrada do cache de consultas é compartilhada entre vários usuários.
* `compressed` ([UInt8](/pt-BR/reference/data-types/index)) — Se a entrada do cache de consultas é comprimida.
* `expires_at` ([DateTime](/pt-BR/reference/data-types/index)) — Quando a entrada do cache de consultas se torna obsoleta.
* `key_hash` ([UInt64](/pt-BR/reference/data-types/index)) — Hash do texto da consulta, usado como chave para localizar entradas do cache de consultas.

{/*AUTOGENERATED_END*/}
