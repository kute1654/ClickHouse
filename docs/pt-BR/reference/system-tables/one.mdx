<div id="description">
  ## Descrição
</div>

Esta tabela contém uma única linha com uma única coluna `dummy` UInt8, que contém o valor 0.

Esta tabela é usada se uma consulta `SELECT` não especificar a cláusula `FROM`.

Ela é semelhante à tabela `DUAL` encontrada em outros SGBDs.
