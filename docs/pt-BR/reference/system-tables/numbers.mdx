<div id="description">
  ## Descrição
</div>

Esta tabela contém uma única coluna UInt64 chamada `number`, que contém quase todos os números naturais a partir de zero.

Você pode usar esta tabela para testes ou se precisar fazer uma busca exaustiva.

As leituras dessa tabela não são paralelizadas.
