<div id="description">
  ## Descrição
</div>

Contém informações sobre stack traces de erros fatais. A tabela não existe no banco de dados por padrão; ela é criada apenas quando ocorrem erros fatais.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — O hostname onde a falha ocorreu.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — A data da falha.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — O horário da falha.
* `timestamp_ns` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Timestamp do evento em nanossegundos.
* `signal` ([Int32](/pt-BR/reference/data-types/int-uint)) — Número do sinal.
* `signal_code` ([Int32](/pt-BR/reference/data-types/int-uint)) — Código do sinal. Fornece contexto adicional sobre a causa do sinal.
* `thread_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — ID da thread.
* `query_id` ([String](/pt-BR/reference/data-types/string)) — ID da consulta.
* `query` ([String](/pt-BR/reference/data-types/string)) — Texto da consulta que estava sendo executada quando a falha ocorreu.
* `trace` ([Array(UInt64)](/pt-BR/reference/data-types/array)) — Stack trace no momento da falha. Cada elemento é um endereço de memória virtual dentro do processo do servidor ClickHouse.
* `trace_full` ([Array(String)](/pt-BR/reference/data-types/array)) — Stack trace no momento da falha. Cada elemento contém um método invocado dentro do processo do servidor ClickHouse.
* `fault_address` ([Nullable(UInt64)](/pt-BR/reference/data-types/nullable)) — Endereço de memória que causou a falha.
* `fault_access_type` ([String](/pt-BR/reference/data-types/string)) — Tipo de acesso à memória que causou a falha (por exemplo, &#39;read&#39;, &#39;write&#39;).
* `signal_description` ([String](/pt-BR/reference/data-types/string)) — Descrição legível com base em signal&#95;code (por exemplo, &#39;Address not mapped to object&#39;).
* `current_exception_trace_full` ([Array(String)](/pt-BR/reference/data-types/array)) — Stack trace da exceção atual com frames simbolizados (somente em falhas de `std::terminate`).
* `version` ([String](/pt-BR/reference/data-types/string)) — Versão do servidor ClickHouse.
* `revision` ([UInt32](/pt-BR/reference/data-types/int-uint)) — Revisão do servidor ClickHouse.
* `build_id` ([String](/pt-BR/reference/data-types/string)) — BuildID gerado pelo compilador.
* `git_hash` ([String](/pt-BR/reference/data-types/string)) — Hash do commit Git do código-fonte do ClickHouse.
* `architecture` ([String](/pt-BR/reference/data-types/string)) — Arquitetura da CPU (por exemplo, x86&#95;64, aarch64).

{/*AUTOGENERATED_END*/}
