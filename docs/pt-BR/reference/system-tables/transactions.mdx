<div id="description">
  ## Descrição
</div>

Contém uma lista de transações e seus estados.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/pt-BR/reference/data-types/tuple)) — O identificador da transação.
* `tid_hash` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O hash do identificador.
* `elapsed` ([Float64](/pt-BR/reference/data-types/float)) — O tempo de processamento da transação.
* `is_readonly` ([UInt8](/pt-BR/reference/data-types/int-uint)) — O sinalizador que indica se a transação executou alguma operação de escrita.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/pt-BR/reference/data-types/enum)) — O estado da transação. Valores possíveis: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/}
