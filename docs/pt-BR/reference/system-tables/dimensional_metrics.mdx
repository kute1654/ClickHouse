<div id="description">
  ## Descrição
</div>

Esta tabela contém métricas dimensionais que podem ser calculadas instantaneamente e exportadas em formato Prometheus. Ela está sempre atualizada.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/pt-BR/reference/data-types/index)) — Nome da métrica.
* `value` ([Float64](/pt-BR/reference/data-types/index)) — Valor da métrica.
* `description` ([String](/pt-BR/reference/data-types/index)) — Descrição da métrica.
* `labels` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Labels da métrica.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

Você pode usar uma consulta como esta para exportar todas as métricas dimensionais no formato Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'gauge' AS type
FROM system.dimensional_metrics
FORMAT Prometheus
```

<div id="metric_descriptions">
  ## Descrição das métricas
</div>

<div id="merge_failures">
  ### merge_failures
</div>

Número total de merges com falha desde a inicialização.

<div id="startup_scripts_failure_reason">
  ### startup_scripts_failure_reason
</div>

Indica falhas em scripts de inicialização por tipo de erro. É definida como 1 quando um script de inicialização falha, com o nome do erro como rótulo.

<div id="merge_tree_parts">
  ### merge_tree_parts
</div>

Número de partes de dados do MergeTree, identificadas pelo estado da parte, pelo tipo da parte e por serem ou não partes de projeção.
