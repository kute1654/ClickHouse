Contém entradas de log com informações sobre arquivos processados pelo mecanismo S3Queue.

É seguro truncar ou remover esta tabela a qualquer momento.

Colunas:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome do host
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento de gravação desta linha de log
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do evento de gravação desta linha de log
* `database` ([String](/pt-BR/reference/data-types/string)) — O nome do banco de dados onde está a tabela de fila (`S3Queue` ou `AzureQueue`).
* `table` ([String](/pt-BR/reference/data-types/string)) — O nome da tabela de fila (`S3Queue` ou `AzureQueue`).
* `uuid` ([String](/pt-BR/reference/data-types/string)) — O UUID da tabela de fila (`S3Queue` ou `AzureQueue`).
* `file_name` ([String](/pt-BR/reference/data-types/string)) — Nome do arquivo em processamento.
* `rows_processed` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Número de linhas processadas.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/pt-BR/reference/data-types/enum)) — Status do arquivo em processamento.
* `processing_start_time` ([Nullable(DateTime)](/pt-BR/reference/data-types/nullable)) — Hora de início do processamento do arquivo.
* `processing_end_time` ([Nullable(DateTime)](/pt-BR/reference/data-types/nullable)) — Hora do fim do processamento do arquivo.
* `exception` ([String](/pt-BR/reference/data-types/string)) — Mensagem da exceção, se houver.
* `commit_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — ID da transação na qual este arquivo foi confirmado.
* `commit_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora em que o arquivo foi confirmado no Keeper (como falho ou processado).
* `transaction_start_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora em que toda a transação de processamento foi iniciada.
* `get_object_time_ms` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Tempo que levamos para encontrar o objeto no armazenamento de objetos.
