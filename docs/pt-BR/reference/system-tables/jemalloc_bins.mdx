<div id="description">
  ## Descrição
</div>

Contém informações sobre alocações de memória feitas por meio do alocador jemalloc em diferentes classes de tamanho (bins), agregadas de todas as arenas.
Essas estatísticas podem não ser totalmente precisas devido ao cache local de thread no jemalloc.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `index` ([UInt16](/pt-BR/reference/data-types/index)) — Índice do bin, ordenado por tamanho.
* `large` ([UInt8](/pt-BR/reference/data-types/index)) — true para alocações grandes e false para pequenas.
* `size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho das alocações neste bin.
* `allocations` ([Int64](/pt-BR/reference/data-types/index)) — Número de alocações.
* `deallocations` ([Int64](/pt-BR/reference/data-types/index)) — Número de desalocações.
* `nregs` ([Int64](/pt-BR/reference/data-types/index)) — Número de regiões por slab.
* `curslabs` ([Int64](/pt-BR/reference/data-types/index)) — Número atual de slabs.
* `curregs` ([Int64](/pt-BR/reference/data-types/index)) — Número atual de regiões para esta classe de tamanho.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

Identifique os tamanhos das alocações que mais contribuíram para o uso total atual de memória.

```sql
SELECT
    *,
    allocations - deallocations AS active_allocations,
    size * active_allocations AS allocated_bytes
FROM system.jemalloc_bins
WHERE allocated_bytes > 0
ORDER BY allocated_bytes DESC
LIMIT 10
```
