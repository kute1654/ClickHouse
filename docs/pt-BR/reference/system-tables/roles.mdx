<div id="description">
  ## Descrição
</div>

Contém informações sobre as [funções](/pt-BR/concepts/features/security/access-rights#role-management) configuradas.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da função.
* `id` ([UUID](/pt-BR/reference/data-types/index)) — ID da função.
* `storage` ([String](/pt-BR/reference/data-types/index)) — Caminho do armazenamento de funções. Configurado no parâmetro `access_control_path`.
