<div id="description">
  ## Descrição
</div>

Contém métricas que podem ser calculadas instantaneamente ou que têm um valor atual. Por exemplo, o número de consultas processadas simultaneamente ou o atraso atual da réplica. Esta tabela está sempre atualizada.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/pt-BR/reference/data-types/index)) — Nome da métrica.
* `value` ([Int64](/pt-BR/reference/data-types/index)) — Valor da métrica.
* `description` ([String](/pt-BR/reference/data-types/index)) — Descrição da métrica.

{/*AUTOGENERATED_END*/}

Você pode encontrar todas as métricas suportadas no arquivo-fonte [src/Common/CurrentMetrics.cpp](https://github.com/ClickHouse/ClickHouse/blob/master/src/Common/CurrentMetrics.cpp).

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.metrics LIMIT 10
```

```text
┌─metric───────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────┐
│ Query                                │     1 │ Number of executing queries                                            │
│ Merge                                │     0 │ Number of executing background merges                                  │
│ PartMutation                         │     0 │ Number of mutations (ALTER DELETE/UPDATE)                              │
│ ReplicatedFetch                      │     0 │ Number of data parts being fetched from replicas                       │
│ ReplicatedSend                       │     0 │ Number of data parts being sent to replicas                            │
│ ReplicatedChecks                     │     0 │ Number of data parts checking for consistency                          │
│ BackgroundMergesAndMutationsPoolTask │     0 │ Number of active merges and mutations in an associated background pool │
│ BackgroundFetchesPoolTask            │     0 │ Number of active fetches in an associated background pool              │
│ BackgroundCommonPoolTask             │     0 │ Number of active tasks in an associated background pool                │
│ BackgroundMovePoolTask               │     0 │ Number of active tasks in BackgroundProcessingPool for moves           │
└──────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────┘
```

<div id="compiledexpressioncachebytes">
  ### CompiledExpressionCacheBytes
</div>

Capacidade reservada de blocos de página (arredondada para páginas inteiras com um fator de superprovisionamento de 2x) mantida por `JITModuleMemoryManager` para seções executáveis/de dados de funções compiladas com JIT armazenadas em cache. NÃO corresponde aos bytes reais de código de máquina em uso (esse valor é menor). Alocada via `posix_memalign`, que é interceptado pelo jemalloc; portanto, isso é contabilizado na arena JIT dedicada e é um subconjunto de `jemalloc.jit_arena.active_bytes`.
