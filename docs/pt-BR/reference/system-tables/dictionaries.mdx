<div id="description">
  ## Descrição
</div>

Contém informações sobre [dicionários](/pt-BR/reference/statements/create/dictionary).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados que contém o dicionário criado pela consulta DDL. String vazia para outros dicionários.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do dicionário.
* `uuid` ([UUID](/pt-BR/reference/data-types/index)) — UUID do dicionário.
* `status` ([Enum8(&#39;NOT&#95;LOADED&#39; = 0, &#39;LOADED&#39; = 1, &#39;FAILED&#39; = 2, &#39;LOADING&#39; = 3, &#39;FAILED&#95;AND&#95;RELOADING&#39; = 4, &#39;LOADED&#95;AND&#95;RELOADING&#39; = 5, &#39;NOT&#95;EXIST&#39; = 6)](/pt-BR/reference/data-types/index)) — Status do dicionário. Valores possíveis:
  * **NOT&#95;LOADED** — O dicionário não foi carregado porque não foi utilizado
  * **LOADED** — Dicionário carregado com sucesso
  * **FAILED** — Não foi possível carregar o dicionário devido a um erro
  * **LOADING** — O dicionário está sendo carregado agora
  * **LOADED&#95;AND&#95;RELOADING** — O dicionário foi carregado com sucesso
  * **and is being reloaded right now (frequent reasons: SYSTEM RELOAD DICTIONARY query** — **e está sendo recarregado neste momento (motivos frequentes: consulta SYSTEM RELOAD DICTIONARY**
  * **timeout** — **tempo limite**
  * **dictionary config has changed)** — **a configuração do dicionário foi alterada)**
  * **FAILED&#95;AND&#95;RELOADING** — Não foi possível carregar o dicionário devido a um erro e ele está sendo carregado agora.
* `origin` ([String](/pt-BR/reference/data-types/index)) — Caminho para o arquivo de configuração que descreve o dicionário.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo de alocação do dicionário. Armazenamento de Dicionários na memória.
* `key.names` ([Array(String)](/pt-BR/reference/data-types/index)) — Array com os nomes das chaves fornecidas pelo dicionário.
* `key.types` ([Array(String)](/pt-BR/reference/data-types/index)) — Array correspondente com os tipos de chave fornecidos pelo dicionário.
* `attribute.names` ([Array(String)](/pt-BR/reference/data-types/index)) — Array com os nomes dos atributos fornecidos pelo dicionário.
* `attribute.types` ([Array(String)](/pt-BR/reference/data-types/index)) — Array correspondente com os tipos de atributo fornecidos pelo dicionário.
* `bytes_allocated` ([UInt64](/pt-BR/reference/data-types/index)) — Quantidade de RAM alocada para o dicionário.
* `hierarchical_index_bytes_allocated` ([UInt64](/pt-BR/reference/data-types/index)) — Quantidade de RAM alocada para o índice hierárquico.
* `query_count` ([UInt64](/pt-BR/reference/data-types/index)) — Número de consultas desde que o dicionário foi carregado ou desde a última reinicialização bem-sucedida.
* `hit_rate` ([Float64](/pt-BR/reference/data-types/index)) — Para dicionários do tipo cache, a porcentagem de usos em que o valor estava no cache.
* `found_rate` ([Float64](/pt-BR/reference/data-types/index)) — Porcentagem de usos em que o valor foi encontrado.
* `element_count` ([UInt64](/pt-BR/reference/data-types/index)) — Número de itens armazenados no dicionário.
* `load_factor` ([Float64](/pt-BR/reference/data-types/index)) — Percentual de preenchimento do dicionário (para um dicionário com hash, o percentual de preenchimento da tabela hash).
* `source` ([String](/pt-BR/reference/data-types/index)) — Texto que descreve a origem dos dados do dicionário.
* `lifetime_min` ([UInt64](/pt-BR/reference/data-types/index)) — Ciclo de vida mínimo do dicionário na memória, após o qual o ClickHouse tenta recarregá-lo (se `invalidate_query` estiver definido, somente se ele tiver sido alterado). Definido em segundos.
* `lifetime_max` ([UInt64](/pt-BR/reference/data-types/index)) — Ciclo de vida máximo do dicionário na memória, após o qual o ClickHouse tenta recarregá-lo (se `invalidate_query` estiver definido, somente se ele tiver sido alterado). Definido em segundos.
* `loading_start_time` ([DateTime](/pt-BR/reference/data-types/index)) — Hora de início do carregamento do dicionário.
* `last_successful_update_time` ([DateTime](/pt-BR/reference/data-types/index)) — Hora de término do carregamento ou da atualização do dicionário. Ajuda a monitorar problemas com origens de dicionário e investigar suas causas.
* `error_count` ([UInt64](/pt-BR/reference/data-types/index)) — Número de erros desde o último carregamento bem-sucedido. Ajuda a monitorar problemas com origens de dicionário e investigar suas causas.
* `loading_duration` ([Float32](/pt-BR/reference/data-types/index)) — Duração do carregamento de um dicionário.
* `last_exception` ([String](/pt-BR/reference/data-types/index)) — Texto do erro que ocorre ao criar ou recarregar o dicionário, caso ele não tenha podido ser criado.
* `comment` ([String](/pt-BR/reference/data-types/index)) — Texto do comentário do dicionário.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

Configure o dicionário:

```sql
CREATE DICTIONARY dictionary_with_comment
(
    id UInt64,
    value String
)
PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() TABLE 'source_table'))
LAYOUT(FLAT())
LIFETIME(MIN 0 MAX 1000)
COMMENT 'The temporary dictionary';
```

Certifique-se de que o dicionário esteja carregado.

```sql
SELECT * FROM system.dictionaries LIMIT 1 FORMAT Vertical;
```
