<div id="description">
  ## Descrição
</div>

Contém informações sobre os certificados disponíveis e suas origens.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/pt-BR/reference/data-types/index)) — Versão do certificado. Os valores são 0 para v1, 1 para v2 e 2 para v3.
* `serial_number` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Número de série do certificado atribuído pelo emissor.
* `signature_algo` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Algoritmo de assinatura — algoritmo usado pelo emissor para assinar este certificado.
* `issuer` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Emissor — identificador único da Autoridade Certificadora que emite este certificado.
* `not_before` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Início do período em que este certificado é válido.
* `not_after` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Fim do período em que este certificado é válido.
* `subject` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Titular — identifica o proprietário da chave pública.
* `pkey_algo` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Algoritmo de chave pública que define com qual algoritmo a chave pública pode ser usada.
* `path` ([String](/pt-BR/reference/data-types/index)) — Caminho para o arquivo ou diretório que contém este certificado.
* `default` ([UInt8](/pt-BR/reference/data-types/index)) — O certificado está no local padrão de certificados.
* `protocol` ([String](/pt-BR/reference/data-types/index)) — Nome do protocolo, se o certificado vier da configuração de TLS por protocolo; caso contrário, vazio.

{/*AUTOGENERATED_END*/}
