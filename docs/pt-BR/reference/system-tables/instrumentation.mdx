<div id="description">
  ## Descrição
</div>

Contém os pontos de instrumentação que usam o recurso XRay do LLVM.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `id` ([UInt32](/pt-BR/reference/data-types/int-uint)) — ID do ponto de instrumentação
* `function_id` ([Int32](/pt-BR/reference/data-types/int-uint)) — ID atribuído à função na seção xray&#95;instr&#95;map do binário ELF.
* `function_name` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome usado para instrumentar a função.
* `handler` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Handler aplicado por patch aos pontos de instrumentação da função.
* `entry_type` ([Enum8(&#39;Entry&#39; = 0, &#39;Exit&#39; = 1, &#39;EntryAndExit&#39; = 2)](/pt-BR/reference/data-types/enum)) — Tipo de entrada do patch.
* `symbol` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome completo e demanglado do símbolo.
* `arguments` ([Array(Dynamic)](/pt-BR/reference/data-types/array)) — Argumentos da chamada do handler.

{/*AUTOGENERATED_END*/}
