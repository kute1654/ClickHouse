<div id="description">
  ## Descrição
</div>

Contém informações sobre cada réplica de bancos de dados Replicated.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados Replicated.
* `is_readonly` ([UInt8](/pt-BR/reference/data-types/index)) — Indica se a réplica do banco de dados está em modo somente leitura.
* `max_log_ptr` ([Int32](/pt-BR/reference/data-types/index)) — Número máximo de entrada no log de atividade geral.
* `replica_name` ([String](/pt-BR/reference/data-types/index)) — Nome da réplica no ClickHouse Keeper.
* `replica_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para os dados da réplica no ClickHouse Keeper.
* `zookeeper_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para os dados do banco de dados no ClickHouse Keeper.
* `shard_name` ([String](/pt-BR/reference/data-types/index)) — Nome do shard no cluster.
* `log_ptr` ([Int32](/pt-BR/reference/data-types/index)) — Número máximo de entrada no log de atividade geral que a réplica copiou para sua fila de execução, mais um.
* `total_replicas` ([UInt32](/pt-BR/reference/data-types/index)) — Número total de réplicas conhecidas deste banco de dados.
* `zookeeper_exception` ([String](/pt-BR/reference/data-types/index)) — Mensagem da última exceção, obtida caso o erro tenha ocorrido ao buscar as informações do ClickHouse Keeper.
* `is_session_expired` ([UInt8](/pt-BR/reference/data-types/index)) — A sessão com o ClickHouse Keeper expirou. Basicamente, é o mesmo que `is_readonly`.

{/*AUTOGENERATED_END*/}
