<div id="description">
  ## Descrição
</div>

Retorna as estatísticas do jemalloc em uma única linha e uma única coluna. Equivalente ao comando SYSTEM JEMALLOC STATS.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `stats` ([String](/pt-BR/reference/data-types/index)) — saída das estatísticas do JEMalloc gerada por malloc&#95;stats&#95;print.
