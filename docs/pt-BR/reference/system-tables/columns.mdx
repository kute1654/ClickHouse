<div id="description">
  ## Descrição
</div>

Contém informações sobre as colunas de todas as tabelas.

Você pode usar esta tabela para obter informações semelhantes às da consulta [DESCRIBE TABLE](/pt-BR/reference/statements/describe-table), mas para várias tabelas ao mesmo tempo.

As colunas de [tabelas temporárias](/pt-BR/reference/statements/create/table#temporary-tables) ficam visíveis em `system.columns` apenas na sessão em que foram criadas. Elas são exibidas com o campo `database` vazio.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da coluna.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo da coluna.
* `position` ([UInt64](/pt-BR/reference/data-types/index)) — Posição ordinal de uma coluna em uma tabela, começando em 1.
* `default_kind` ([String](/pt-BR/reference/data-types/index)) — Tipo de expressão (DEFAULT, MATERIALIZED, ALIAS) para o valor padrão, ou uma string vazia se não estiver definido.
* `default_expression` ([String](/pt-BR/reference/data-types/index)) — Expressão do valor padrão, ou uma string vazia se não estiver definida.
* `data_compressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho dos dados comprimidos, em bytes.
* `data_uncompressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho dos dados descomprimidos, em bytes.
* `marks_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho das marcas, em bytes.
* `comment` ([String](/pt-BR/reference/data-types/index)) — Comentário da coluna, ou uma string vazia se não estiver definido.
* `is_in_partition_key` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a coluna está na expressão de partição.
* `is_in_sorting_key` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a coluna está na expressão da chave de ordenação.
* `is_in_primary_key` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a coluna está na expressão da chave primária.
* `is_in_sampling_key` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se a coluna está na expressão da chave de amostragem.
* `compression_codec` ([String](/pt-BR/reference/data-types/index)) — Nome do codec de compressão.
* `character_octet_length` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Comprimento máximo, em bytes, para dados binários, dados de caracteres, dados de texto e imagens. No ClickHouse, isso só faz sentido para o tipo de dado FixedString. Caso contrário, o valor NULL é retornado.
* `numeric_precision` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Precisão de dados numéricos aproximados, dados numéricos exatos, dados inteiros ou dados monetários. No ClickHouse, isso corresponde à largura em bits para tipos inteiros e à precisão decimal para tipos Decimal. Caso contrário, o valor NULL é retornado.
* `numeric_precision_radix` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — A base do sistema numérico da precisão de dados numéricos aproximados, dados numéricos exatos, dados inteiros ou dados monetários. No ClickHouse, é 2 para tipos inteiros e 10 para tipos Decimal. Caso contrário, o valor NULL é retornado.
* `numeric_scale` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — A escala de dados numéricos aproximados, dados numéricos exatos, dados inteiros ou dados monetários. No ClickHouse, isso só faz sentido para tipos Decimal. Caso contrário, o valor NULL é retornado.
* `datetime_precision` ([Nullable(UInt64)](/pt-BR/reference/data-types/index)) — Precisão decimal do tipo de dado DateTime64. Para outros tipos de dado, o valor NULL é retornado.
* `serialization_hint` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Uma indicação para a coluna escolher a serialização em inserts de acordo com as estatísticas.
* `statistics` ([String](/pt-BR/reference/data-types/index)) — Os tipos de estatísticas criados nessas colunas.

{/*AUTOGENERATED_END*/}
