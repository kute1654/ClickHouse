<div id="description">
  ## Descrição
</div>

Esta tabela do sistema pode ser usada para obter uma visão geral do uso de memória e dos ProfileEvents dos usuários.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `user` ([String](/pt-BR/reference/data-types/index)) — Nome do usuário.
* `memory_usage` ([Int64](/pt-BR/reference/data-types/index)) — Soma da RAM usada por todos os processos do usuário. Pode não incluir alguns tipos de memória dedicada. Consulte a configuração max&#95;memory&#95;usage.
* `peak_memory_usage` ([Int64](/pt-BR/reference/data-types/index)) — Pico de uso de memória do usuário. Ele pode ser redefinido quando nenhuma consulta é executada para esse usuário.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/pt-BR/reference/data-types/index)) — Resumo dos ProfileEvents que medem diferentes métricas do usuário. A descrição deles pode ser encontrada na tabela system.events

{/*AUTOGENERATED_END*/}
