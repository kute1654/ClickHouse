<div id="description">
  ## Descrição
</div>

Contém uma lista das [contas de usuário](/pt-BR/concepts/features/security/access-rights#user-account-management) configuradas no servidor.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do usuário.
* `id` ([UUID](/pt-BR/reference/data-types/index)) — ID do usuário.
* `storage` ([String](/pt-BR/reference/data-types/index)) — Caminho para o armazenamento dos usuários. Configurado no parâmetro access&#95;control&#95;path.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/pt-BR/reference/data-types/index)) — Mostra os tipos de autenticação. Há várias formas de identificar o usuário: sem senha, com senha em texto simples, com senha codificada em SHA256, com senha codificada em SHA-1 duplo ou com senha codificada em bcrypt.
* `auth_params` ([Array(String)](/pt-BR/reference/data-types/index)) — Parâmetros de autenticação em formato JSON, dependendo do auth&#95;type.
* `valid_until` ([Array(DateTime)](/pt-BR/reference/data-types/index)) — A data e hora de expiração das credenciais do usuário.
* `host_ip` ([Array(String)](/pt-BR/reference/data-types/index)) — Endereços IP dos hosts que têm permissão para se conectar ao servidor ClickHouse.
* `host_names` ([Array(String)](/pt-BR/reference/data-types/index)) — Nomes dos hosts que têm permissão para se conectar ao servidor ClickHouse.
* `host_names_regexp` ([Array(String)](/pt-BR/reference/data-types/index)) — Expressão regular para nomes de hosts que têm permissão para se conectar ao servidor ClickHouse.
* `host_names_like` ([Array(String)](/pt-BR/reference/data-types/index)) — Nomes dos hosts que têm permissão para se conectar ao servidor ClickHouse, definidos com o predicado LIKE.
* `default_roles_all` ([UInt8](/pt-BR/reference/data-types/index)) — Mostra que todos os roles concedidos são definidos como padrão para o usuário.
* `default_roles_list` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de roles concedidos definidos por padrão.
* `default_roles_except` ([Array(String)](/pt-BR/reference/data-types/index)) — Todos os roles concedidos definidos como padrão, exceto os listados.
* `grantees_any` ([UInt8](/pt-BR/reference/data-types/index)) — Sinalizador que indica se um usuário com qualquer opção de concessão pode concedê-la a qualquer pessoa.
* `grantees_list` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de usuários ou roles aos quais este usuário tem permissão para conceder opções.
* `grantees_except` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de usuários ou roles aos quais este usuário está proibido de conceder opções.
* `default_database` ([String](/pt-BR/reference/data-types/index)) — O nome do banco de dados padrão deste usuário.
