<div id="description">
  ## Descrição
</div>

Contém uma lista de todas as coleções nomeadas que foram criadas por meio de uma consulta SQL ou interpretadas a partir do arquivo de configuração.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome da coleção.
* `collection` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Estrutura interna da coleção.
* `source` ([String](/pt-BR/reference/data-types/index)) — Fonte da coleção nomeada.
* `create_query` ([String](/pt-BR/reference/data-types/index)) — Consulta CREATE da coleção nomeada.

{/*AUTOGENERATED_END*/}
