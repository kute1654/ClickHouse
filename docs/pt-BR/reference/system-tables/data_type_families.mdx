<div id="description">
  ## Descrição
</div>

Contém informações sobre os [tipos de dados](/pt-BR/reference/data-types/index).

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do tipo de dado.
* `case_insensitive` ([UInt8](/pt-BR/reference/data-types/index)) — Propriedade que indica se é possível usar um nome de tipo de dado em uma consulta sem diferenciar maiúsculas de minúsculas. Por exemplo, `Date` e `date` são ambos válidos.
* `alias_to` ([String](/pt-BR/reference/data-types/index)) — Nome do tipo de dado do qual `name` é um alias.
* `description` ([String](/pt-BR/reference/data-types/index)) — Uma descrição geral do tipo de dado.
* `syntax` ([String](/pt-BR/reference/data-types/index)) — Como o tipo de dado é escrito em uma consulta.
* `examples` ([String](/pt-BR/reference/data-types/index)) — Exemplos de uso.
* `introduced_in` ([String](/pt-BR/reference/data-types/index)) — A versão do ClickHouse em que o tipo de dado foi introduzido pela primeira vez, no formato major.minor.
* `related` ([Array(String)](/pt-BR/reference/data-types/index)) — Os nomes dos tipos de dados relacionados.

{/*AUTOGENERATED_END*/}

</div>
