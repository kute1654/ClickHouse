<div id="description">
  ## Descrição
</div>

Contém informações sobre os data skipping indexes existentes em todas as tabelas.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do índice.
* `type` ([String](/pt-BR/reference/data-types/index)) — Tipo de índice.
* `type_full` ([String](/pt-BR/reference/data-types/index)) — Expressão do tipo de índice da instrução CREATE.
* `expr` ([String](/pt-BR/reference/data-types/index)) — Expressão para o cálculo do índice.
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/pt-BR/reference/data-types/index)) — Se o índice foi criado implicitamente (via add&#95;minmax&#95;index&#95;for&#95;numeric&#95;columns ou semelhante)
* `granularity` ([UInt64](/pt-BR/reference/data-types/index)) — O número de grânulos no bloco.
* `data_compressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho dos dados comprimidos, em bytes.
* `data_uncompressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho dos dados descomprimidos, em bytes.
* `marks_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — O tamanho das marcas, em bytes.

{/*AUTOGENERATED_END*/}
