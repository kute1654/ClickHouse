<div id="description">
  ## Descrição
</div>

Contém as propriedades dos perfis de configuração.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Nome do perfil de configuração.
* `id` ([UUID](/pt-BR/reference/data-types/index)) — ID do perfil de configuração.
* `storage` ([String](/pt-BR/reference/data-types/index)) — Caminho para o armazenamento dos perfis de configuração. Configurado no parâmetro `access_control_path`.
* `num_elements` ([UInt64](/pt-BR/reference/data-types/index)) — Número de elementos deste perfil na tabela `system.settings_profile_elements`.
* `apply_to_all` ([UInt8](/pt-BR/reference/data-types/index)) — Indica que o perfil de configuração está definido para todas as funções e/ou usuários.
* `apply_to_list` ([Array(String)](/pt-BR/reference/data-types/index)) — Lista de funções e/ou usuários aos quais o perfil de configuração é aplicado.
* `apply_to_except` ([Array(String)](/pt-BR/reference/data-types/index)) — O perfil de configuração é aplicado a todas as funções e/ou usuários, exceto às listadas.

{/*AUTOGENERATED_END*/}
