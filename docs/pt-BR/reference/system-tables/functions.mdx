<div id="description">
  ## Descrição
</div>

Contém informações sobre funções comuns e funções de agregação.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — O nome da função.
* `is_aggregate` ([UInt8](/pt-BR/reference/data-types/index)) — Indica se a função é uma função de agregação.
* `case_insensitive` ([UInt8](/pt-BR/reference/data-types/index)) — Indica se o nome da função pode ser usado sem diferenciar maiúsculas de minúsculas.
* `alias_to` ([String](/pt-BR/reference/data-types/index)) — O nome original da função, caso o nome da função seja um alias.
* `create_query` ([String](/pt-BR/reference/data-types/index)) — Obsoleto.
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/pt-BR/reference/data-types/index)) — Obsoleto.
* `description` ([String](/pt-BR/reference/data-types/index)) — Uma descrição geral do que a função faz.
* `syntax` ([String](/pt-BR/reference/data-types/index)) — Assinatura da função.
* `arguments` ([String](/pt-BR/reference/data-types/index)) — Os argumentos da função.
* `parameters` ([String](/pt-BR/reference/data-types/index)) — Os parâmetros da função (somente para funções de agregação).
* `returned_value` ([String](/pt-BR/reference/data-types/index)) — O que a função retorna.
* `examples` ([String](/pt-BR/reference/data-types/index)) — Exemplo de uso.
* `introduced_in` ([String](/pt-BR/reference/data-types/index)) — A versão do ClickHouse em que a função foi introduzida pela primeira vez.
* `categories` ([String](/pt-BR/reference/data-types/index)) — A categoria da função.
* `deterministic` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Indica se a função retorna o mesmo resultado para os mesmos argumentos. NULL quando desconhecido (por exemplo, funções de agregação ou Funções Definidas pelo Usuário).
* `higher_order` ([Nullable(UInt8)](/pt-BR/reference/data-types/index)) — Indica se a função é de ordem superior — ou seja, aceita pelo menos uma expressão lambda como argumento (por exemplo, arrayMap, arrayFilter, mapApply). NULL quando desconhecido.

{/*AUTOGENERATED_END*/}
