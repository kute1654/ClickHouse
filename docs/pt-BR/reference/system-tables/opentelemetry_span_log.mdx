<div id="description">
  ## Descrição
</div>

Contém informações sobre [trace spans](https://opentracing.io/docs/overview/spans/) de consultas executadas.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — O hostname em que este span foi capturado.
* `trace_id` ([UUID](/pt-BR/reference/data-types/uuid)) — ID do trace da consulta executada.
* `span_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — ID do trace span.
* `parent_span_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — ID do trace span pai.
* `operation_name` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — O nome da operação.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/pt-BR/reference/data-types/enum)) — O SpanKind do span. INTERNAL — Indica que o span representa uma operação interna em um aplicativo. SERVER — Indica que o span abrange o processamento, no lado do servidor, de uma RPC síncrona ou outra solicitação remota. CLIENT — Indica que o span descreve uma solicitação a algum serviço remoto. PRODUCER — Indica que o span descreve os iniciadores de uma solicitação assíncrona. Esse span pai geralmente termina antes do span filho CONSUMER correspondente, possivelmente até mesmo antes de o span filho começar. CONSUMER - Indica que o span descreve um filho de uma solicitação assíncrona PRODUCER.
* `start_time_us` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O horário de início do trace span (em microssegundos).
* `finish_time_us` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O horário de término do trace span (em microssegundos).
* `finish_date` ([Date](/pt-BR/reference/data-types/date)) — A data de término do trace span.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/pt-BR/reference/data-types/enum)) — O código de status do span.
* `status_message` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Mensagem de erro.
* `attribute` ([Map(LowCardinality(String), String)](/pt-BR/reference/data-types/map)) — Atributo dependente do trace span. Eles são preenchidos de acordo com as recomendações do padrão OpenTelemetry.

**Aliases:**

* `attribute.names` — Alias para `mapKeys(attribute)`.
* `attribute.values` — Alias para `mapValues(attribute)`.

{/*AUTOGENERATED_END*/}
