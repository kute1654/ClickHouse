<div id="description">
  ## Descrição
</div>

Contém informações sobre os diretórios de usuário configurados — diretórios no sistema de arquivos dos quais o servidor ClickHouse tem permissão para ler dados fornecidos pelo usuário.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — O nome do diretório.
* `type` ([String](/pt-BR/reference/data-types/index)) — O tipo de armazenamento de acesso, por exemplo, users.xml, replicated ou memory.
* `params` ([String](/pt-BR/reference/data-types/index)) — JSON com os parâmetros do armazenamento de acesso.
* `precedence` ([UInt64](/pt-BR/reference/data-types/index)) — A ordem em que este diretório é declarado na configuração. Essa mesma ordem é usada quando o ClickHouse tenta encontrar um usuário ou role.

{/*AUTOGENERATED_END*/}
