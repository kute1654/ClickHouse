<div id="description">
  ## Descrição
</div>

Contém as licenças de bibliotecas de terceiros localizadas no diretório [contrib](https://github.com/ClickHouse/ClickHouse/tree/master/contrib) do código-fonte do ClickHouse.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `library_name` ([String](/pt-BR/reference/data-types/index)) — Nome da biblioteca.
* `license_type` ([String](/pt-BR/reference/data-types/index)) — Tipo de licença — por exemplo, Apache, MIT.
* `license_path` ([String](/pt-BR/reference/data-types/index)) — Caminho do arquivo que contém o texto da licença.
* `license_text` ([String](/pt-BR/reference/data-types/index)) — Texto da licença.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT library_name, license_type, license_path FROM system.licenses LIMIT 15
```

```text
┌─library_name───────┬─license_type─┬─license_path────────────────────────┐

│ brotli             │ MIT          │ /contrib/brotli/LICENSE             │
│ [...]              │ [...]        │ [...]                               │
└────────────────────┴──────────────┴─────────────────────────────────────┘
```
