<div id="description">
  ## Descrição
</div>

Contém as funções ativas do usuário atual. `SET ROLE` altera o conteúdo desta tabela.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/pt-BR/reference/data-types/index)) — Nome da função.
* `with_admin_option` ([UInt8](/pt-BR/reference/data-types/index)) — 1 se a função tiver o privilégio ADMIN OPTION.
* `is_default` ([UInt8](/pt-BR/reference/data-types/index)) — 1 se a função estiver definida como padrão.

{/*AUTOGENERATED_END*/}
