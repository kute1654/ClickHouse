Contém um histórico de todos os eventos ocorridos no cache do sistema de arquivos para objetos em um sistema de arquivos remoto.

É seguro truncar ou excluir esta tabela a qualquer momento.

Colunas:

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do evento
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do evento
* `query_id` ([String](/pt-BR/reference/data-types/string)) — ID da consulta
* `source_file_path` ([String](/pt-BR/reference/data-types/string)) — Caminho do segmento de arquivo no sistema de arquivos
* `file_segment_range` ([Tuple(UInt64, UInt64)](/pt-BR/reference/data-types/tuple)) — Intervalo do segmento de arquivo
* `total_requested_range` ([Tuple(UInt64, UInt64)](/pt-BR/reference/data-types/tuple)) — Intervalo completo de leitura
* `key` ([String](/pt-BR/reference/data-types/string)) — Chave do segmento de arquivo
* `offset` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Offset do segmento de arquivo
* `size` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Tamanho da leitura
* `read_type` ([String](/pt-BR/reference/data-types/string)) — Tipo de leitura: READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Indica se foi feita uma tentativa de leitura do cache
* `read_buffer_id` ([String](/pt-BR/reference/data-types/string)) — ID interno do buffer de leitura da implementação
* `user_id` ([String](/pt-BR/reference/data-types/string)) — ID do usuário que criou o segmento de arquivo
