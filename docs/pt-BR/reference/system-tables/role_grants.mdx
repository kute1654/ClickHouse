<div id="description">
  ## Descrição
</div>

Contém as concessões de função para usuários e funções. Para adicionar entradas a esta tabela, use `GRANT role TO user`.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `user_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome do usuário.
* `role_name` ([Nullable(String)](/pt-BR/reference/data-types/index)) — Nome da função.
* `granted_role_name` ([String](/pt-BR/reference/data-types/index)) — Nome da função concedida à função `role_name`. Para conceder uma função a outra, use `GRANT role1 TO role2`.
* `granted_role_id` ([UUID](/pt-BR/reference/data-types/index)) — ID da função.
* `granted_role_is_default` ([UInt8](/pt-BR/reference/data-types/index)) — Sinalizador que indica se `granted_role` é uma função padrão. Valores possíveis: • 1 — `granted_role` é uma função padrão, • 0 — `granted_role` não é uma função padrão.
* `with_admin_option` ([UInt8](/pt-BR/reference/data-types/index)) — Sinalizador que indica se `granted_role` é uma função com o privilégio `ADMIN OPTION`. Valores possíveis: • 1 — A função tem o privilégio `ADMIN OPTION`.• 0 — A função não tem o privilégio `ADMIN OPTION`.

{/*AUTOGENERATED_END*/}
