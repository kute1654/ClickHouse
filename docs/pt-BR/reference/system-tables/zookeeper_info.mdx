<div id="description">
  ## Descrição
</div>

Esta tabela exibe informações combinadas de introspecção sobre o ZooKeeper, e os nós são obtidos da configuração.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_cluster_name` ([String](/pt-BR/reference/data-types/string)) — nome do cluster do ZooKeeper.
* `host` ([String](/pt-BR/reference/data-types/string)) — O hostname/IP do nó do ZooKeeper ao qual o ClickHouse se conectou.
* `port` ([UInt16](/pt-BR/reference/data-types/int-uint)) — A porta do nó do ZooKeeper ao qual o ClickHouse se conectou.
* `index` ([Nullable(UInt8)](/pt-BR/reference/data-types/int-uint)) — O índice do nó do ZooKeeper ao qual o ClickHouse se conectou. O índice vem da configuração do ZooKeeper. Se não houver conexão, esta coluna será NULL.
* `is_connected` ([Nullable(UInt8)](/pt-BR/reference/data-types/int-uint)) — Indica se o ZooKeeper está conectado ou não.
* `is_readonly` ([UInt8](/pt-BR/reference/data-types/int-uint)) — É somente leitura.
* `version` ([String](/pt-BR/reference/data-types/string)) — A versão do ZooKeeper.
* `avg_latency` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A latência média.
* `max_latency` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Latência máxima.
* `min_latency` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A latência mínima.
* `packets_received` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de pacotes recebidos.
* `packets_sent` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de pacotes enviados.
* `outstanding_requests` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de requisições pendentes.
* `server_state` ([String](/pt-BR/reference/data-types/string)) — Estado do servidor.
* `is_leader` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Indica se este é o líder do ZooKeeper.
* `znode_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A quantidade de znodes.
* `watch_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A quantidade de watches.
* `ephemerals_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — A contagem de valores efêmeros.
* `approximate_data_size` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Tamanho aproximado dos dados.
* `followers` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Os seguidores do líder. Esse campo é exposto apenas pelo líder.
* `synced_followers` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Os seguidores sincronizados do líder. Este campo é exposto somente pelo líder.
* `pending_syncs` ([UInt64](/pt-BR/reference/data-types/int-uint)) — As sincronizações pendentes do líder. Este campo é exposto apenas pelo líder.
* `open_file_descriptor_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de descritores de arquivo abertos. Disponível apenas em plataformas Unix.
* `max_file_descriptor_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número máximo de descritores de arquivo. Disponível apenas em plataformas Unix.
* `connections` ([UInt64](/pt-BR/reference/data-types/int-uint)) — As conexões com o ZooKeeper.
* `outstanding` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O número de operações pendentes no ZooKeeper.
* `zxid` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O zxid do ZooKeeper.
* `node_count` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Número de nós do ZooKeeper.
* `snapshot_dir_size` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O tamanho do diretório de snapshots do ZooKeeper.
* `log_dir_size` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O tamanho do diretório de logs do ZooKeeper.
* `first_log_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O índice do primeiro log do ZooKeeper.
* `first_log_term` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O primeiro termo do log do ZooKeeper.
* `last_log_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O índice do último registro de log do ZooKeeper.
* `last_log_term` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O último termo do log do ZooKeeper.
* `last_committed_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O último índice confirmado do ZooKeeper.
* `leader_committed_log_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O índice do log confirmado pelo líder do ZooKeeper.
* `target_committed_log_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O índice do log confirmado de destino do ZooKeeper.
* `last_snapshot_idx` ([UInt64](/pt-BR/reference/data-types/int-uint)) — O índice do último snapshot do ZooKeeper.
