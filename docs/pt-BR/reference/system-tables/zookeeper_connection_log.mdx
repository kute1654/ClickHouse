<div id="description">
  ## Descrição
</div>

A tabela `system.zookeeper_connection_log` mostra o histórico das conexões com o ZooKeeper (incluindo ZooKeepers auxiliares). Cada linha mostra informações sobre um evento relacionado a essas conexões.

<Note>
  A tabela não contém eventos de desconexão causados pelo desligamento do servidor.
</Note>

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Hostname do servidor que se conectou ou se desconectou do ZooKeeper.
* `type` ([Enum8(&#39;Connected&#39; = 0, &#39;Disconnected&#39; = 1)](/pt-BR/reference/data-types/enum)) — O tipo do evento. Valores possíveis: Connected, Disconnected.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data da entrada.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora da entrada
* `event_time_microseconds` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — Hora da entrada com precisão de microssegundos.
* `name` ([String](/pt-BR/reference/data-types/string)) — Nome do cluster ZooKeeper.
* `host` ([String](/pt-BR/reference/data-types/string)) — O hostname/IP do nó do ZooKeeper ao qual o ClickHouse se conectou ou do qual se desconectou.
* `port` ([UInt16](/pt-BR/reference/data-types/int-uint)) — A porta do nó do ZooKeeper ao qual o ClickHouse se conectou ou do qual se desconectou.
* `index` ([UInt8](/pt-BR/reference/data-types/int-uint)) — O índice do nó do ZooKeeper ao qual o ClickHouse se conectou ou do qual se desconectou. O índice vem da configuração do ZooKeeper.
* `client_id` ([Int64](/pt-BR/reference/data-types/int-uint)) — ID da sessão da conexão.
* `keeper_api_version` ([UInt8](/pt-BR/reference/data-types/int-uint)) — Versão da API do Keeper.
* `enabled_feature_flags` ([Array(Enum16(&#39;FILTERED&#95;LIST&#39; = 0, &#39;MULTI&#95;READ&#39; = 1, &#39;CHECK&#95;NOT&#95;EXISTS&#39; = 2, &#39;CREATE&#95;IF&#95;NOT&#95;EXISTS&#39; = 3, &#39;REMOVE&#95;RECURSIVE&#39; = 4, &#39;MULTI&#95;WATCHES&#39; = 5, &#39;CHECK&#95;STAT&#39; = 6, &#39;PERSISTENT&#95;WATCHES&#39; = 7, &#39;CREATE&#95;WITH&#95;STATS&#39; = 8, &#39;TRY&#95;REMOVE&#39; = 9, &#39;LIST&#95;WITH&#95;STAT&#95;AND&#95;DATA&#39; = 10, &#39;GET&#95;CHILDREN&#95;RECURSIVE&#39; = 11))](/pt-BR/reference/data-types/array)) — Feature flags habilitadas. Aplicável apenas ao ClickHouse Keeper.
* `availability_zone` ([String](/pt-BR/reference/data-types/string)) — Zona de disponibilidade
* `reason` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Motivo da conexão ou desconexão.

{/*AUTOGENERATED_END*/}

SELECT * FROM system.zookeeper_connection_log;
