<div id="description">
  ## Descrição
</div>

Contém informações sobre arquivos locais que estão na fila para ser enviados aos shards. Esses arquivos locais contêm novas partes criadas ao inserir novos dados na tabela Distributed em modo assíncrono.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `data_path` ([String](/pt-BR/reference/data-types/index)) — Caminho para a pasta com arquivos locais.
* `is_blocked` ([UInt8](/pt-BR/reference/data-types/index)) — Indica se o envio de arquivos locais para o servidor está bloqueado.
* `error_count` ([UInt64](/pt-BR/reference/data-types/index)) — Número de erros.
* `data_files` ([UInt64](/pt-BR/reference/data-types/index)) — Número de arquivos locais em uma pasta.
* `data_compressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho dos dados comprimidos em arquivos locais, em bytes.
* `broken_data_files` ([UInt64](/pt-BR/reference/data-types/index)) — Número de arquivos marcados como corrompidos (devido a um erro).
* `broken_data_compressed_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho dos dados comprimidos em arquivos corrompidos, em bytes.
* `last_exception` ([String](/pt-BR/reference/data-types/index)) — Mensagem de texto sobre o último erro ocorrido (se houver).
* `last_exception_time` ([DateTime](/pt-BR/reference/data-types/index)) — Horário em que ocorreu a última exceção.

{/*AUTOGENERATED_END*/}
