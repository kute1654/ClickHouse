<div id="purpose">
  ## Descrição
</div>

A tabela `system.delta_lake_metadata_log` registra eventos de acesso e análise de metadados de tabelas Delta Lake lidas pelo ClickHouse. Ela fornece informações detalhadas sobre cada arquivo de metadados, o que é útil para depuração, auditoria e para entender a evolução da estrutura da tabela Delta.

Esta tabela registra cada arquivo de metadados lido de tabelas Delta Lake. Ela ajuda os usuários a rastrear como o ClickHouse interpreta os metadados da tabela Delta e a diagnosticar problemas relacionados à evolução do esquema, à resolução de snapshots ou ao planejamento de consultas.

<Note>
  Esta tabela se destina principalmente à depuração.
</Note>

<div id="controlling-log-verbosity">
  ### Controlando a verbosidade do log
</div>

Você pode controlar quais eventos de metadados são registrados por meio da configuração [`delta_lake_log_metadata`](/pt-BR/reference/settings/session-settings#delta_lake_log_metadata).

Para registrar todos os metadados usados na consulta atual:

```sql
SELECT * FROM my_delta_table SETTINGS delta_lake_log_metadata = 1;

SYSTEM FLUSH LOGS delta_lake_metadata_log;

SELECT *
FROM system.delta_lake_metadata_log
WHERE query_id = '{previous_query_id}';
```
