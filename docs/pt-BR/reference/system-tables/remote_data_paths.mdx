<div id="description">
  ## Descrição
</div>

Contém informações sobre arquivos de dados armazenados em discos remotos (por exemplo, S3 e Azure Blob Storage), incluindo o mapeamento entre caminhos de metadados locais e caminhos remotos de blobs.

Cada linha representa um objeto blob remoto associado a um arquivo de dados.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `disk_name` ([String](/pt-BR/reference/data-types/index)) — Nome do disco.
* `path` ([String](/pt-BR/reference/data-types/index)) — Caminho do disco.
* `cache_base_path` ([String](/pt-BR/reference/data-types/index)) — Diretório base dos arquivos de cache.
* `local_path` ([String](/pt-BR/reference/data-types/index)) — Caminho do arquivo do ClickHouse, também usado como caminho de metadados.
* `remote_path` ([String](/pt-BR/reference/data-types/index)) — Caminho do blob no armazenamento de objetos ao qual o arquivo do ClickHouse está associado.
* `size` ([UInt64](/pt-BR/reference/data-types/index)) — Tamanho do arquivo (comprimido).
* `common_prefix_for_blobs` ([String](/pt-BR/reference/data-types/index)) — Prefixo comum dos blobs no armazenamento de objetos.
* `cache_paths` ([Array(String)](/pt-BR/reference/data-types/index)) — Arquivos de cache do blob correspondente.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Exemplo
</div>

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
