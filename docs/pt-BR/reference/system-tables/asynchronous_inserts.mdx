<div id="description">
  ## Descrição
</div>

Contém informações sobre inserção assíncrona pendentes na fila.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/pt-BR/reference/data-types/index)) — Texto da consulta.
* `database` ([String](/pt-BR/reference/data-types/index)) — Nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `format` ([String](/pt-BR/reference/data-types/index)) — Nome do formato.
* `first_update` ([DateTime64(6)](/pt-BR/reference/data-types/index)) — Momento da primeira inserção com resolução de microssegundos.
* `total_bytes` ([UInt64](/pt-BR/reference/data-types/index)) — Número total de bytes aguardando na fila.
* `entries.query_id` ([Array(String)](/pt-BR/reference/data-types/index)) — Array de IDs das consultas das inserções aguardando na fila.
* `entries.bytes` ([Array(UInt64)](/pt-BR/reference/data-types/index)) — Array de bytes de cada consulta de inserção aguardando na fila.

{/*AUTOGENERATED_END*/}

SELECT * FROM system.asynchronous_inserts LIMIT 1 \G;
