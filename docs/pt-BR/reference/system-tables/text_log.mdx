<div id="description">
  ## Descrição
</div>

Contém registros de logging. O nível de logging enviado para esta tabela pode ser limitado pela configuração do servidor `text_log.level`.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome do host do servidor que executa a consulta.
* `event_date` ([Date](/pt-BR/reference/data-types/date)) — Data do registro.
* `event_time` ([DateTime](/pt-BR/reference/data-types/datetime)) — Hora do registro.
* `event_time_microseconds` ([DateTime64(6)](/pt-BR/reference/data-types/datetime64)) — Hora do registro com precisão de microssegundos.
* `thread_name` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome da thread em que o logging foi gerado.
* `thread_id` ([UInt64](/pt-BR/reference/data-types/int-uint)) — ID da thread do SO.
* `level` ([Enum8(&#39;Fatal&#39; = 1, &#39;Critical&#39; = 2, &#39;Error&#39; = 3, &#39;Warning&#39; = 4, &#39;Notice&#39; = 5, &#39;Information&#39; = 6, &#39;Debug&#39; = 7, &#39;Trace&#39; = 8, &#39;Test&#39; = 9)](/pt-BR/reference/data-types/enum)) — Nível do registro. Valores possíveis: 1 ou &#39;Fatal&#39;, 2 ou &#39;Critical&#39;, 3 ou &#39;Error&#39;, 4 ou &#39;Warning&#39;, 5 ou &#39;Notice&#39;, 6 ou &#39;Information&#39;, 7 ou &#39;Debug&#39;, 8 ou &#39;Trace&#39;.
* `query_id` ([String](/pt-BR/reference/data-types/string)) — ID da consulta.
* `logger_name` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Nome do logger (por exemplo, DDLWorker).
* `message` ([String](/pt-BR/reference/data-types/string)) — A própria mensagem.
* `revision` ([UInt32](/pt-BR/reference/data-types/int-uint)) — Revisão do ClickHouse.
* `source_file` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — Arquivo de origem em que o logging foi gerado.
* `source_line` ([UInt64](/pt-BR/reference/data-types/int-uint)) — Linha de origem em que o logging foi gerado.
* `message_format_string` ([LowCardinality(String)](/pt-BR/reference/data-types/lowcardinality)) — String de formato usada para formatar a mensagem.
* `value1` ([String](/pt-BR/reference/data-types/string)) — Argumento 1 usado para formatar a mensagem.
* `value2` ([String](/pt-BR/reference/data-types/string)) — Argumento 2 usado para formatar a mensagem.
* `value3` ([String](/pt-BR/reference/data-types/string)) — Argumento 3 usado para formatar a mensagem.
* `value4` ([String](/pt-BR/reference/data-types/string)) — Argumento 4 usado para formatar a mensagem.
* `value5` ([String](/pt-BR/reference/data-types/string)) — Argumento 5 usado para formatar a mensagem.
* `value6` ([String](/pt-BR/reference/data-types/string)) — Argumento 6 usado para formatar a mensagem.
* `value7` ([String](/pt-BR/reference/data-types/string)) — Argumento 7 usado para formatar a mensagem.
* `value8` ([String](/pt-BR/reference/data-types/string)) — Argumento 8 usado para formatar a mensagem.
* `value9` ([String](/pt-BR/reference/data-types/string)) — Argumento 9 usado para formatar a mensagem.
* `value10` ([String](/pt-BR/reference/data-types/string)) — Argumento 10 usado para formatar a mensagem.
