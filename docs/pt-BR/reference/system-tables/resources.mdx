<div id="description">
  ## Descrição
</div>

Contém informações sobre [recursos](/pt-BR/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage) armazenados no servidor local. A tabela contém uma linha para cada recurso.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — O nome do recurso.
* `read_disks` ([Array(String)](/pt-BR/reference/data-types/index)) — A lista dos nomes dos discos que usam este recurso para operações de leitura.
* `write_disks` ([Array(String)](/pt-BR/reference/data-types/index)) — A lista dos nomes dos discos que usam este recurso para operações de escrita.
* `unit` ([String](/pt-BR/reference/data-types/index)) — Unidade de recurso usada na medição de custo.
* `create_query` ([String](/pt-BR/reference/data-types/index)) — A consulta CREATE do recurso.

{/*AUTOGENERATED_END*/}

Exemplo:

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
Row 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

Row 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```
