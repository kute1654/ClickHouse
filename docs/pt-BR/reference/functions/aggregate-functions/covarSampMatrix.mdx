<div id="covarSampMatrix">
  ## covarSampMatrix
</div>

Introduzido em: v23.2.0

Retorna a matriz de covariância amostral de N variáveis.

**Sintaxe**

```sql
covarSampMatrix(x1[, x2, ...])
```

**Argumentos**

* `x1[, x2, ...]` — Um ou mais parâmetros para calcular a matriz de covariância amostral. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a matriz de covariância amostral. [`Array(Array(Float64))`](/pt-BR/reference/data-types/array)

**Exemplos**

**Cálculo básico da matriz de covariância amostral**

```sql title=Query
DROP TABLE IF EXISTS test;
CREATE TABLE test
(
    a UInt32,
    b Float64,
    c Float64,
    d Float64
)
ENGINE = Memory;
INSERT INTO test(a, b, c, d) VALUES (1, 5.6, -4.4, 2.6), (2, -9.6, 3, 3.3), (3, -1.3, -4, 1.2), (4, 5.3, 9.7, 2.3), (5, 4.4, 0.037, 1.222), (6, -8.6, -7.8, 2.1233), (7, 5.1, 9.3, 8.1222), (8, 7.9, -3.6, 9.837), (9, -8.2, 0.62, 8.43555), (10, -3, 7.3, 6.762);

SELECT arrayMap(x -> round(x, 3), arrayJoin(covarSampMatrix(a, b, c, d))) AS covarSampMatrix
FROM test
```
