## varPop {#varPop}

Introduzido em: v1.1.0

Calcula a variância populacional.

A variância populacional é calculada usando a fórmula:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

Onde:
- $x$ é cada valor na população
- $\bar{x}$ é a média populacional
- $n$ é o tamanho da população

<Note>
Esta função utiliza um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`varPopStable`](/reference/functions/aggregate-functions/varPopStable). Ela é mais lenta, mas apresenta menor erro computacional.
</Note>
    

**Sintaxe**

```sql
varPop(x)
```

**Aliases**: `VAR_POP`

**Argumentos**

- `x` — População de valores para a qual calcular a variância populacional. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valor retornado**

Retorna a variância populacional de `x`. [`Float64`](/reference/data-types/float)

**Exemplos**

**Calculando a variância populacional**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
