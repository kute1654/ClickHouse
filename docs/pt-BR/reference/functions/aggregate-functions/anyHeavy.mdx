<div id="anyHeavy">
  ## anyHeavy
</div>

Introduzido em: v1.1.0

Seleciona um valor que ocorre com frequência usando o algoritmo de [heavy hitters](https://doi.org/10.1145/762471.762473).
Se houver um valor que ocorra em mais da metade dos casos em cada uma das threads de execução da consulta, esse valor será retornado.
Normalmente, o resultado é não determinístico.

**Sintaxe**

```sql
anyHeavy(column)
```

**Argumentos**

* `column` — Nome da coluna. [`String`](/pt-BR/reference/data-types/string)

**Valor retornado**

Retorna um valor que ocorre com frequência. O resultado é não determinístico. [`Any`](/pt-BR/reference/data-types/index)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT anyHeavy(AirlineID) AS res
FROM ontime;
```

```response title=Response
┌───res─┐
│ 19690 │
└───────┘
```
