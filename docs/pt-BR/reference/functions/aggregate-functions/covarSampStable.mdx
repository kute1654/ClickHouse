## covarSampStable {#covarSampStable}

Introduzido em: v1.1.0

Calcula a covariância amostral:

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{n - 1}
$$

<br/>

É semelhante a [`covarSamp`](/reference/functions/aggregate-functions/covarSamp), mas utiliza um algoritmo numericamente estável.
Como resultado, `covarSampStable` é mais lento que `covarSamp`, porém apresenta menor erro computacional.
    

**Sintaxe**

```sql
covarSampStable(x, y)
```

**Argumentos**

- `x` — Primeira variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal`](/reference/data-types/decimal)
- `y` — Segunda variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal`](/reference/data-types/decimal)

**Valor retornado**

Retorna a covariância amostral entre `x` e `y`. Para `n <= 1`, retorna `inf`. [`Float64`](/reference/data-types/float)

**Exemplos**

**Cálculo básico de covariância amostral com algoritmo estável**

```sql title=Query
DROP TABLE IF EXISTS series;
CREATE TABLE series(i UInt32, x_value Float64, y_value Float64) ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6,-4.4),(2, -9.6,3),(3, -1.3,-4),(4, 5.3,9.7),(5, 4.4,0.037),(6, -8.6,-7.8),(7, 5.1,9.3),(8, 7.9,-3.6),(9, -8.2,0.62),(10, -3,7.3);

SELECT covarSampStable(x_value, y_value)
FROM
(
    SELECT
        x_value,
        y_value
    FROM series
);
```

```response title=Response
┌─covarSampStable(x_value, y_value)─┐
│                 7.206275555555556 │
└───────────────────────────────────┘
```

**Valor único retorna inf**

```sql title=Query
SELECT covarSampStable(x_value, y_value)
FROM
(
    SELECT
        x_value,
        y_value
    FROM series LIMIT 1
);
```

```response title=Response
┌─covarSampStable(x_value, y_value)─┐
│                               inf │
└───────────────────────────────────┘
```
