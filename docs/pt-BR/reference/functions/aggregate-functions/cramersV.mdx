<div id="cramersV">
  ## cramersV
</div>

Introduzido em: v22.1.0

[Cramer&#39;s V](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V) (às vezes também chamado de phi de Cramer) é uma medida de associação entre duas colunas de uma tabela.
O resultado da função `cramersV` varia de 0 (correspondendo à ausência de associação entre as variáveis) a 1, e só pode atingir 1 quando cada valor é completamente determinado pelo outro.
Ela pode ser vista como a associação entre duas variáveis expressa como uma porcentagem de sua variação máxima possível.

<Note>
  Para uma versão de Cramer&#39;s V com correção de viés, veja: [cramersVBiasCorrected](/pt-BR/reference/functions/aggregate-functions/cramersVBiasCorrected)
</Note>

**Sintaxe**

```sql
cramersV(column1, column2)
```

**Argumentos**

* `column1` — Primeira coluna a ser comparada. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)
* `column2` — Segunda coluna a ser comparada. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna um valor entre 0 (correspondente à ausência de associação entre os valores das colunas) e 1 (associação completa). [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Nenhuma associação entre as colunas**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 3 AS a,
            number % 5 AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─cramersV(a, b)─┐
│              0 │
└────────────────┘
```

**Forte associação entre colunas**

```sql title=Query
SELECT
    cramersV(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            if (number % 12 = 0, (number + 1) % 5, number % 5) AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─────cramersV(a, b)─┐
│ 0.9066801892162646 │
└────────────────────┘
```
