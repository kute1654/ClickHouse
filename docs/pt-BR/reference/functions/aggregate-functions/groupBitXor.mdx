<div id="groupBitXor">
  ## groupBitXor
</div>

Introduzido em: v1.1.0

Aplica XOR bit a bit a uma série de números.

**Sintaxe**

```sql
groupBitXor(expr)
```

**Aliases**: `BIT_XOR`

**Argumentos**

* `expr` — Expressão do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Valor retornado**

Retorna um valor do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Exemplos**

**Exemplo de XOR bit a bit**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Dados de teste:
-- binário    decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitXor(num) FROM t;
```

```response title=Response
-- Resultado:
-- binário     decimal
-- 01101000 = 104

┌─groupBitXor(num)─┐
│              104 │
└──────────────────┘
```
