## covarSamp {#covarSamp}

Introduzido em: v1.1.0

Calcula a covariância amostral:

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{n - 1}
$$

<Note>
Esta função utiliza um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`covarSampStable`](/reference/functions/aggregate-functions/covarSampStable).
Ela é mais lenta, mas produz um erro computacional menor.
</Note>
    

**Sintaxe**

```sql
covarSamp(x, y)
```

**Aliases**: `COVAR_SAMP`

**Argumentos**

- `x` — Primeira variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal`](/reference/data-types/decimal)
- `y` — Segunda variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal`](/reference/data-types/decimal)

**Valor retornado**

Retorna a covariância amostral entre `x` e `y`. Para `n <= 1`, é retornado `nan`. [`Float64`](/reference/data-types/float)

**Exemplos**

**Cálculo básico de covariância amostral**

```sql title=Query
DROP TABLE IF EXISTS series;
CREATE TABLE series(i UInt32, x_value Float64, y_value Float64) ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6,-4.4),(2, -9.6,3),(3, -1.3,-4),(4, 5.3,9.7),(5, 4.4,0.037),(6, -8.6,-7.8),(7, 5.1,9.3),(8, 7.9,-3.6),(9, -8.2,0.62),(10, -3,7.3);

SELECT covarSamp(x_value, y_value)
FROM series
```

```response title=Response
┌─covarSamp(x_value, y_value)─┐
│           7.206275555555556 │
└─────────────────────────────┘
```

**Valor único retorna NaN**

```sql title=Query
SELECT covarSamp(x_value, y_value)
FROM series LIMIT 1
```

```response title=Response
┌─covarSamp(x_value, y_value)─┐
│                         nan │
└─────────────────────────────┘
```
