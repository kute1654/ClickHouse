<div id="groupArrayMovingSum">
  ## groupArrayMovingSum
</div>

Introduzido em: v20.1.0

Calcula a soma móvel dos valores de entrada.

A função pode receber o tamanho da janela como parâmetro. Se não for especificado, a função usa um tamanho da janela igual ao número de linhas na coluna.

**Sintaxe**

```sql
groupArrayMovingSum(numbers_for_summing)
groupArrayMovingSum(window_size)(numbers_for_summing)
```

**Parâmetros**

* `window_size` — Tamanho da janela de cálculo. Se não for especificado, a função considera um tamanho de janela igual ao número de linhas da coluna. [`UInt64`](/pt-BR/reference/data-types/int-uint)

**Argumentos**

* `numbers_for_summing` — Expressão que resulta em um valor de tipo de dados numérico. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna um Array do mesmo tamanho e tipo dos dados de entrada. [`Array`](/pt-BR/reference/data-types/array)

**Exemplos**

**Exemplo de uso**

```sql title=Query
CREATE TABLE t
(
    `int` UInt8,
    `float` Float32,
    `dec` Decimal32(2)
)
ENGINE = Memory;

INSERT INTO t VALUES (1, 1.1, 1.10), (2, 2.2, 2.20), (4, 4.4, 4.40), (7, 7.77, 7.77);

SELECT
    groupArrayMovingSum(int) AS I,
    groupArrayMovingSum(float) AS F,
    groupArrayMovingSum(dec) AS D
FROM t;
```

```response title=Response
┌─I──────────┬─F───────────────────────────────┬─D──────────────────────┐
│ [1,3,7,14] │ [1.1,3.3000002,7.7000003,15.47] │ [1.10,3.30,7.70,15.47] │
└────────────┴─────────────────────────────────┴────────────────────────┘
```

**Com tamanho da janela**

```sql title=Query
SELECT
    groupArrayMovingSum(2)(int) AS I,
    groupArrayMovingSum(2)(float) AS F,
    groupArrayMovingSum(2)(dec) AS D
FROM t;
```

```response title=Response

└────────────┴─────────────────────────────────┴────────────────────────┘
```
