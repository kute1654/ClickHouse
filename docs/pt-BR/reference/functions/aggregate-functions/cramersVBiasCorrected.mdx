<div id="cramersVBiasCorrected">
  ## cramersVBiasCorrected
</div>

Introduzido em: v22.1.0

O V de Cramer é uma medida de associação entre duas colunas em uma tabela.
O resultado da [função `cramersV`](/pt-BR/reference/functions/aggregate-functions/cramersV) varia de 0 (o que corresponde à ausência de associação entre as variáveis) a 1 e só pode atingir 1 quando cada valor é completamente determinado pelo outro.
A função pode apresentar um viés significativo, por isso esta versão do V de Cramer usa a [correção de viés](https://en.wikipedia.org/wiki/Cram%C3%A9r%27s_V#Bias_correction).

**Sintaxe**

```sql
cramersVBiasCorrected(column1, column2)
```

**Argumentos**

* `column1` — Primeira coluna a ser comparada. [`Any`](/pt-BR/reference/data-types/index)
* `column2` — Segunda coluna a ser comparada. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um valor entre 0 (correspondente à ausência de associação entre os valores das colunas) e 1 (associação completa). [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Comparação com o cramersV regular**

```sql title=Query
SELECT
    cramersV(a, b),
    cramersVBiasCorrected(a, b)
FROM
    (
        SELECT
            number % 10 AS a,
            number % 4 AS b
        FROM
            numbers(150)
    );
```

```response title=Response
┌─────cramersV(a, b)─┬─cramersVBiasCorrected(a, b)─┐
│ 0.5798088336225178 │          0.5305112825189074 │
└────────────────────┴─────────────────────────────┘
```
