<div id="stddevSampStable">
  ## stddevSampStable
</div>

Introduzido em: v1.1.0

O resultado é igual à raiz quadrada de [varSamp](/pt-BR/reference/functions/aggregate-functions/varSamp). Ao contrário de [stddevSamp](/pt-BR/reference/functions/aggregate-functions/stddevSamp), esta função usa um algoritmo numericamente estável. Ela é mais lenta, mas produz um erro computacional menor.

**Sintaxe**

```sql
stddevSampStable(x)
```

**Argumentos**

* `x` — Valores para os quais calcular a raiz quadrada da variância amostral. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a raiz quadrada da variância amostral de `x`. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Uso básico**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSampStable(population)
FROM test_data;
```

```response title=Response
┌─stddevSampStable(population)─┐
│                            4 │
└──────────────────────────────┘
```
