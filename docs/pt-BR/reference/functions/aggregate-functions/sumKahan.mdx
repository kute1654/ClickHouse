<div id="sumKahan">
  ## sumKahan
</div>

Introduzido em: v1.1.0

Calcula a soma dos números usando o [algoritmo de soma compensada de Kahan](https://en.wikipedia.org/wiki/Kahan_summation_algorithm).
É mais lenta que a função [`sum`](/pt-BR/reference/functions/aggregate-functions/sum).
A compensação funciona apenas para tipos [Float](/pt-BR/reference/data-types/float).

**Sintaxe**

```sql
sumKahan(x)
```

**Argumentos**

* `x` — Valor de entrada. [`Integer`](/pt-BR/reference/data-types/int-uint) ou [`Float`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a soma dos números. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Exemplos**

**Demonstrando a melhoria na precisão com a soma de Kahan**

```sql title=Query
SELECT sum(0.1), sumKahan(0.1) FROM numbers(10);
```

```response title=Response
┌───────────sum(0.1)─┬─sumKahan(0.1)─┐
│ 0.9999999999999999 │             1 │
└────────────────────┴───────────────┘
```
