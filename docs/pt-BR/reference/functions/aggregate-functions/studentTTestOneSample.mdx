<div id="studentTTestOneSample">
  ## studentTTestOneSample
</div>

Introduzido em: v25.10.0

Aplica o teste t de Student para uma amostra para determinar se a média de uma amostra difere de uma média populacional conhecida.

Assume-se normalidade. A hipótese nula é que a média da amostra é igual à média populacional.

O `confidence_level` opcional permite calcular o intervalo de confiança.

**Notas:**

* São necessárias pelo menos 2 observações; caso contrário, o resultado será `(nan, nan)` (e os intervalos, se solicitados, serão `nan`).
* Uma entrada constante ou quase constante também retornará `nan` devido ao erro padrão zero (ou efetivamente zero).

**Veja também**

* [teste t de Student](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [função studentTTest](/pt-BR/reference/functions/aggregate-functions/studentTTest)

**Sintaxe**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**Parâmetros**

* `confidence_level` — Opcional. Nível de confiança dos intervalos de confiança. Float entre (0, 1). [`Float*`](/pt-BR/reference/data-types/float)

**Argumentos**

* `sample_data` — Dados da amostra. [`Integer`](/pt-BR/reference/data-types/int-uint) ou [`Float`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)
* `population_mean` — Média populacional conhecida usada no teste (geralmente uma constante). [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna uma tupla com dois ou quatro elementos (se `confidence_level` for especificado): estatística t calculada, valor p calculado (bilateral), [limite inferior calculado do intervalo de confiança], [limite superior calculado do intervalo de confiança]. Os intervalos de confiança são para a média da amostra no nível de confiança especificado. [`Tuple(Float64, Float64)`](/pt-BR/reference/data-types/tuple) ou [`Tuple(Float64, Float64, Float64, Float64)`](/pt-BR/reference/data-types/tuple)

**Exemplos**

**Sem intervalo de confiança**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
