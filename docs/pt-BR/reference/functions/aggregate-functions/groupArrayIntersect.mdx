<div id="groupArrayIntersect">
  ## groupArrayIntersect
</div>

Introduzido em: v24.2.0

Retorna a interseção dos arrays fornecidos (retorna todos os itens dos arrays que estão presentes em todos eles).

**Sintaxe**

```sql
groupArrayIntersect(x)
```

**Argumentos**

* `x` — Argumento (nome da coluna ou expressão). [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um array contendo os elementos presentes em todos os arrays. [`Array`](/pt-BR/reference/data-types/array)

**Exemplos**

**Exemplo de uso**

```sql title=Query
-- Create table with Memory engine
CREATE TABLE numbers (
    a Array(Int32)
) ENGINE = Memory;

-- Insert sample data
INSERT INTO numbers VALUES
    ([1,2,4]),
    ([1,5,2,8,-1,0]),
    ([1,5,7,5,8,2]);

SELECT groupArrayIntersect(a) AS intersection FROM numbers;
```

```response title=Response
┌─intersection──────┐
│ [1, 2]            │
└───────────────────┘
```
