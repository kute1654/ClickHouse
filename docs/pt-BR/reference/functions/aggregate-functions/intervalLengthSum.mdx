<div id="intervalLengthSum">
  ## intervalLengthSum
</div>

Introduzido em: v21.7.0

Recebe vários intervalos numéricos e calcula o comprimento total considerando a união de todas as partes sobrepostas em um único intervalo.

<Note>
  Os argumentos devem ser do mesmo tipo de dado.
  Caso contrário, uma exceção será lançada.
</Note>

**Sintaxe**

```sql
intervalLengthSum(start, end)
```

**Argumentos**

* `start` — O valor inicial do intervalo. [`(U)Int32/64`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`DateTime`](/pt-BR/reference/data-types/datetime) ou [`Date`](/pt-BR/reference/data-types/date)
* `end` — O valor final do intervalo. [`(U)Int32/64`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`DateTime`](/pt-BR/reference/data-types/datetime) ou [`Date`](/pt-BR/reference/data-types/date)

**Valor retornado**

Retorna o comprimento total da união de todos os intervalos (segmentos no eixo numérico). Dependendo do tipo do argumento, o valor retornado pode ser do tipo UInt64 ou Float64. [`UInt64`](/pt-BR/reference/data-types/int-uint) ou [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Exemplo com Float32**

```sql title=Query
CREATE TABLE fl_interval (id String, start Float32, end Float32) ENGINE = Memory;
INSERT INTO fl_interval VALUES ('a', 1.1, 2.9), ('a', 2.5, 3.2), ('a', 4, 5);

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM fl_interval GROUP BY id ORDER BY id;
```

```response title=Response
┌─id─┬─intervalLengthSum(start, end)─┬─toTypeName(intervalLengthSum(start, end))─┐
│ a  │                           3.1 │ Float64                                   │
└────┴───────────────────────────────┴───────────────────────────────────────────┘
```

**Exemplo de DateTime**

```sql title=Query
CREATE TABLE dt_interval (id String, start DateTime, end DateTime) ENGINE = Memory;
INSERT INTO dt_interval VALUES ('a', '2020-01-01 01:12:30', '2020-01-01 02:10:10'), ('a', '2020-01-01 02:05:30', '2020-01-01 02:50:31'), ('a', '2020-01-01 03:11:22', '2020-01-01 03:23:31');

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM dt_interval GROUP BY id ORDER BY id;
```

```response title=Response
┌─id─┬─intervalLengthSum(start, end)─┬─toTypeName(intervalLengthSum(start, end))─┐
│ a  │                          6610 │ UInt64                                    │
└────┴───────────────────────────────┴───────────────────────────────────────────┘
```

**Exemplo de data**

```sql title=Query
CREATE TABLE date_interval (id String, start Date, end Date) ENGINE = Memory;
INSERT INTO date_interval VALUES ('a', '2020-01-01', '2020-01-04'), ('a', '2020-01-12', '2020-01-18');

SELECT id, intervalLengthSum(start, end), toTypeName(intervalLengthSum(start, end)) FROM date_interval GROUP BY id ORDER BY id;
```

```response title=Response

└────┴───────────────────────────────┴───────────────────────────────────────────┘
```
