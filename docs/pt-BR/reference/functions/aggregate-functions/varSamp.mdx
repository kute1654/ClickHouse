## varSamp {#varSamp}

Introduzido em: v1.1.0

Calcula a variância amostral de um conjunto de dados.

A variância amostral é calculada usando a fórmula:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

Onde:
- $x$ é cada ponto de dado individual no conjunto de dados
- $\bar{x}$ é a média aritmética do conjunto de dados
- $n$ é o número de pontos de dados no conjunto de dados

A função assume que o conjunto de dados de entrada representa uma amostra de uma população maior. Se você quiser calcular a variância de toda a população (quando possui o conjunto de dados completo), use [`varPop`](/reference/functions/aggregate-functions/varPop).

<Note>
Esta função utiliza um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`varSampStable`](/reference/functions/aggregate-functions/varSampStable). Ela é mais lenta, mas apresenta menor erro computacional.
</Note>
    

**Sintaxe**

```sql
varSamp(x)
```

**Aliases**: `VAR_SAMP`

**Argumentos**

- `x` — A população para a qual você deseja calcular a variância amostral. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valor retornado**

Retorna a variância amostral do conjunto de dados de entrada `x`. [`Float64`](/reference/data-types/float)

**Exemplos**

**Calculando a variância amostral**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSamp(x),3) AS var_samp FROM test_data;
```

```response title=Response
┌─var_samp─┐
│    0.865 │
└──────────┘
```
