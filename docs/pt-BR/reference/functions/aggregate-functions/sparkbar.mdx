<div id="sparkbar">
  ## sparkbar
</div>

Introduzido em: v21.11.0

A função traça um histograma de frequência para os valores `x` e a taxa de repetição `y` desses valores no intervalo `[min_x, max_x]`.
As repetições de todos os `x` que caem no mesmo bucket são calculadas como média, portanto os dados devem ser pré-agregados.
Repetições negativas são ignoradas.

Se nenhum intervalo for especificado, o `x` mínimo será usado como início do intervalo, e o `x` máximo — como fim do intervalo.
Caso contrário, os valores fora do intervalo serão ignorados.

**Sintaxe**

```sql
sparkbar(buckets[, min_x, max_x])(x, y)
```

**Aliases**: `sparkBar`

**Parâmetros**

* `buckets` — O número de segmentos. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)
* `min_x` — Opcional. O início do intervalo. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)
* `max_x` — Opcional. O fim do intervalo. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Argumentos**

* `x` — O campo com valores. [`const String`](/pt-BR/reference/data-types/string)
* `y` — O campo com a frequência dos valores. [`const String`](/pt-BR/reference/data-types/string)

**Valor retornado**

Retorna o histograma de frequência. [`String`](/pt-BR/reference/data-types/string)

**Exemplos**

**Sem especificar o intervalo**

```sql title=Query
CREATE TABLE spark_bar_data (`value` Int64, `event_date` Date) ENGINE = MergeTree ORDER BY event_date;

INSERT INTO spark_bar_data VALUES (1,'2020-01-01'), (3,'2020-01-02'), (4,'2020-01-02'), (-3,'2020-01-02'), (5,'2020-01-03'), (2,'2020-01-04'), (3,'2020-01-05'), (7,'2020-01-06'), (6,'2020-01-07'), (8,'2020-01-08'), (2,'2020-01-11');

SELECT sparkbar(9)(event_date, cnt) FROM (SELECT sum(value) AS cnt, event_date FROM spark_bar_data GROUP BY event_date);
```

```response title=Response
┌─sparkbar(9)(event_date, cnt)─┐
│ ▂▅▂▃▆█  ▂                    │
└──────────────────────────────┘
```

**Com especificação de intervalo**

```sql title=Query
SELECT sparkbar(9, toDate('2020-01-01'), toDate('2020-01-10'))(event_date, cnt) FROM (SELECT sum(value) AS cnt, event_date FROM spark_bar_data GROUP BY event_date);
```

```response title=Response
┌─sparkbar(9, toDate('2020-01-01'), toDate('2020-01-10'))(event_date, cnt)─┐
│ ▂▅▂▃▇▆█                                                                  │
└──────────────────────────────────────────────────────────────────────────┘
```
