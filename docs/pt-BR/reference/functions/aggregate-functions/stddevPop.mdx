<div id="stddevPop">
  ## stddevPop
</div>

Introduzido em: v1.1.0

Retorna o desvio padrão populacional de uma sequência de dados numéricos.
O resultado é igual à raiz quadrada de [`varPop`](/pt-BR/reference/functions/aggregate-functions/varPop).

<Note>
  Esta função usa um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`stddevPopStable`](/pt-BR/reference/functions/aggregate-functions/stddevPopStable). Ela é mais lenta, mas apresenta menor erro computacional.
</Note>

**Sintaxe**

```sql
stddevPop(x)
```

**Aliases**: `STD`, `STDDEV_POP`

**Argumentos**

* `x` — Conjunto de valores para os quais se deseja calcular o desvio padrão populacional. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a raiz quadrada da variância populacional de `x`. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Calculando o desvio padrão populacional**

```sql title=Query
CREATE TABLE test_data (population UInt8) ENGINE = Log;
INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT stddevPop(population) AS stddev FROM test_data;
```

```response title=Response
┌────────────stddev─┐
│ 3.794733192202055 │
└───────────────────┘
```
