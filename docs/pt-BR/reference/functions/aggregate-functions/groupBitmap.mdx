<div id="groupBitmap">
  ## groupBitmap
</div>

Introduzido em: v20.1.0

Cria um bitmap (vetor de bits) a partir de uma coluna de inteiros sem sinal e, em seguida, retorna a quantidade de valores únicos (cardinalidade) nesse bitmap.
Ao acrescentar o sufixo combinador `-State`, em vez de retornar a contagem, retorna o [objeto bitmap](/pt-BR/reference/functions/regular-functions/bitmap-functions) propriamente dito.

**Sintaxe**

```sql
groupBitmap(expr)
groupBitmapState(expr)
```

**Argumentos**

* `expr` — Expressão que resulta em um tipo `UInt*`. [`UInt*`](/pt-BR/reference/data-types/int-uint)

**Valor retornado**

Retorna a contagem como `UInt64` ou um objeto bitmap ao usar `-State`. [`UInt64`](/pt-BR/reference/data-types/int-uint)

**Exemplos**

**Exemplo de uso**

```sql title=Query
CREATE TABLE t (UserID UInt32) ENGINE = Memory;
INSERT INTO t VALUES (1), (1), (2), (3);

SELECT groupBitmap(UserID) AS num FROM t;
```

```response title=Response
┌─num─┐
│   3 │
└─────┘
```
