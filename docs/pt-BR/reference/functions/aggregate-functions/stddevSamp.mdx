<div id="stddevSamp">
  ## stddevSamp
</div>

Introduzido em: v1.1.0

Retorna o desvio padrão amostral de uma sequência de dados numéricos.
O resultado é igual à raiz quadrada de [`varSamp`](/pt-BR/reference/functions/aggregate-functions/varSamp).

<Note>
  Esta função usa um algoritmo numericamente instável.
  Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`stddevSampStable`](/pt-BR/reference/functions/aggregate-functions/stddevSampStable). Ela é mais lenta, mas apresenta menor erro computacional.
</Note>

**Sintaxe**

```sql
stddevSamp(x)
```

**Aliases**: `STDDEV_SAMP`, `STDDEV`

**Argumentos**

* `x` — Valores dos quais se quer encontrar a raiz quadrada da variância amostral. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a raiz quadrada da variância amostral de `x`. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Cálculo do desvio padrão amostral**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population UInt8,
)
ENGINE = Log;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    stddevSamp(population)
FROM test_data;
```

```response title=Response
┌─stddevSamp(population)─┐
│                      4 │
└────────────────────────┘
```
