## varSampStable {#varSampStable}

Introduzido em: v1.1.0

Calcula a variância amostral de um conjunto de dados. Ao contrário de [`varSamp`](/reference/functions/aggregate-functions/varSamp), esta função utiliza um algoritmo [numericamente estável](https://en.wikipedia.org/wiki/Numerical_stability). Ela é mais lenta, mas apresenta um erro computacional menor.

A variância amostral é calculada com a mesma fórmula de [`varSamp`](/reference/functions/aggregate-functions/varSamp):

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

Onde:
- $x$ é cada ponto de dado individual no conjunto de dados
- $\bar{x}$ é a média aritmética do conjunto de dados
- $n$ é o número de pontos de dados no conjunto de dados
    

**Sintaxe**

```sql
varSampStable(x)
```

**Argumentos**

- `x` — A população para a qual você deseja calcular a variância amostral. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float) ou [`Decimal*`](/reference/data-types/decimal)

**Valor retornado**

Retorna a variância amostral do conjunto de dados de entrada. [`Float64`](/reference/data-types/float)

**Exemplos**

**Calculando a variância amostral estável**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSampStable(x),3) AS var_samp_stable FROM test_data;
```

```response title=Response
┌─var_samp_stable─┐
│           0.865 │
└─────────────────┘
```
