<div id="skewPop">
  ## skewPop
</div>

Introduzido em: v20.1.0

Calcula a [assimetria](https://en.wikipedia.org/wiki/Skewness) de uma sequência.

**Sintaxe**

```sql
skewPop(expr)
```

**Argumentos**

* `expr` — Uma expressão que retorna um número. [`Expressão`](/pt-BR/reference/data-types/special-data-types/expression)

**Valor retornado**

Retorna a assimetria da distribuição informada. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Distribuição simétrica**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**Distribuição com assimetria à direita**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
