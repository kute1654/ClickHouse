<div id="skewSamp">
  ## skewSamp
</div>

Introduzido em: v20.1.0

Calcula a [assimetria amostral](https://en.wikipedia.org/wiki/Skewness) de uma sequência.

Representa uma estimativa não enviesada da assimetria de uma variável aleatória se os valores fornecidos constituírem uma amostra dela.

**Sintaxe**

```sql
skewSamp(expr)
```

**Argumentos**

* `expr` — Uma expressão que resulta em um número. [`Expressão`](/pt-BR/reference/data-types/special-data-types/expression)

**Valor retornado**

Retorna a assimetria da distribuição informada. Se `n <= 1` (`n` é o tamanho da amostra), a função retorna `nan`. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Distribuição simétrica**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**Distribuição assimétrica à direita**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
