<div id="groupBitOr">
  ## groupBitOr
</div>

Introduzido na versão: v1.1.0

Aplica OR bit a bit a uma sequência de números.

**Sintaxe**

```sql
groupBitOr(expr)
```

**Aliases**: `BIT_OR`

**Argumentos**

* `expr` — Expressão do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Valor retornado**

Retorna um valor do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Exemplos**

**Exemplo de OR em nível de bit**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Dados de teste:
-- binário    decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Resultado:
-- binário    decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
