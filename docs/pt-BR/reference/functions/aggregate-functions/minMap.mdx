<div id="minMappedArrays">
  ## minMappedArrays
</div>

Introduzido em: v20.5.0

Calcula o valor mínimo do array `value` de acordo com as chaves especificadas no array `key`.

<Note>
  * Passar uma tupla de chaves e arrays de valores é o mesmo que passar um array de chaves e um array de valores.
  * O número de elementos em `key` e `value` deve ser o mesmo para cada linha totalizada.
</Note>

**Sintaxe**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**Argumentos**

* `key` — Array de chaves. [`Array(T)`](/pt-BR/reference/data-types/array)
* `value` — Array de valores. [`Array(T)`](/pt-BR/reference/data-types/array)

**Valor retornado**

Retorna uma tupla com dois arrays: chaves em ordem crescente e valores calculados para as chaves correspondentes. [`Tuple(Array(T), Array(T))`](/pt-BR/reference/data-types/tuple)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
