<div id="analysisOfVariance">
  ## analysisOfVariance
</div>

Introduzido em: v22.10.0

Fornece um teste estatístico de análise de variância de uma via (teste ANOVA). É um teste aplicado a vários grupos de observações normalmente distribuídas para determinar se todos os grupos têm a mesma média ou não.

<Note>
  Os grupos são enumerados a partir de 0, e deve haver pelo menos dois grupos para realizar o teste.
  Deve haver pelo menos um grupo com mais de uma observação.
</Note>

**Sintaxe**

```sql
analysisOfVariance(val, group_no)
```

**Aliases**: `anova`

**Argumentos**

* `val` — Valor. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)
* `group_no` — Número do grupo ao qual `val` pertence. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna uma tupla com a estatística F e o p-valor. [`Tuple(Float64, Float64)`](/pt-BR/reference/data-types/tuple)

**Exemplos**

**Uso básico**

```sql title=Query
SELECT analysisOfVariance(number, number % 2) FROM numbers(1048575);
```

```response title=Response
┌─analysisOfVariance(number, modulo(number, 2))─┐
│ (0,1)                                         │
└───────────────────────────────────────────────┘
```
