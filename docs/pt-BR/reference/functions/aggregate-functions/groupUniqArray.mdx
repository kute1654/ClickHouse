<div id="groupUniqArray">
  ## groupUniqArray
</div>

Introduzido em: v1.1.0

Cria um array a partir de valores distintos do argumento.
O consumo de memória desta função é o mesmo da função [`uniqExact`](/pt-BR/reference/functions/aggregate-functions/uniqExact).

**Sintaxe**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**Parâmetros**

* `max_size` — Limita o tamanho do array resultante a `max_size` elementos. `groupUniqArray(1)(x)` é equivalente a `[any(x)]`. [`UInt64`](/pt-BR/reference/data-types/int-uint)

**Argumentos**

* `x` — Expressão. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um array com valores únicos. [`Array`](/pt-BR/reference/data-types/array)

**Exemplos**

**Exemplo de uso**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**Usando o parâmetro max&#95;size**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
