<div id="distinctDynamicTypes">
  ## distinctDynamicTypes
</div>

Introduzido em: v24.9.0

Calcula a lista de tipos de dados distintos armazenados em uma coluna [Dynamic](/pt-BR/reference/data-types/dynamic).

**Sintaxe**

```sql
distinctDynamicTypes(dynamic)
```

**Argumentos**

* `dynamic` — coluna do tipo Dynamic. [`Dynamic`](/pt-BR/reference/data-types/dynamic)

**Valor retornado**

Retorna a lista ordenada de nomes de tipos de dados. [`Array(String)`](/pt-BR/reference/data-types/array)

**Exemplos**

**Uso básico com tipos mistos**

```sql title=Query
DROP TABLE IF EXISTS test_dynamic;
CREATE TABLE test_dynamic(d Dynamic) ENGINE = Memory;
INSERT INTO test_dynamic VALUES (42), (NULL), ('Hello'), ([1, 2, 3]), ('2020-01-01'), (map(1, 2)), (43), ([4, 5]), (NULL), ('World'), (map(3, 4));

SELECT distinctDynamicTypes(d) FROM test_dynamic;
```

```response title=Response
┌─distinctDynamicTypes(d)──────────────────────────────────────────┐
│ ['Array(Int64)', 'Date', 'Int64', 'Map(UInt8, UInt8)', 'String'] │
└──────────────────────────────────────────────────────────────────┘
```
