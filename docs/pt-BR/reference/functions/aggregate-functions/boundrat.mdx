<div id="boundingRatio">
  ## boundingRatio
</div>

Introduzido em: v20.1.0

Calcula a inclinação entre os pontos mais à esquerda e mais à direita de um conjunto de valores.

**Sintaxe**

```sql
boundingRatio(x, y)
```

**Argumentos**

* `x` — Valores da coordenada X. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)
* `y` — Valores da coordenada Y. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a inclinação da reta entre os pontos mais à esquerda e mais à direita, ou `NaN` se os dados estiverem vazios. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Dados de exemplo**

```sql title=Query
SELECT
    number,
    number * 1.5
FROM numbers(10)
```

```response title=Response
┌─number─┬─multiply(number, 1.5)─┐
│      0 │                     0 │
│      1 │                   1.5 │
│      2 │                     3 │
│      3 │                   4.5 │
│      4 │                     6 │
│      5 │                   7.5 │
│      6 │                     9 │
│      7 │                  10.5 │
│      8 │                    12 │
│      9 │                  13.5 │
└────────┴───────────────────────┘
```

**Exemplo de uso**

```sql title=Query
SELECT boundingRatio(number, number * 1.5)
FROM numbers(10)
```

```response title=Response
┌─boundingRatio(number, multiply(number, 1.5))─┐
│                                          1.5 │
└──────────────────────────────────────────────┘
```
