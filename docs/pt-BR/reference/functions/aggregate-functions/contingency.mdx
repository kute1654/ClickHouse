<div id="contingency">
  ## contingency
</div>

Introduzido em: v22.1.0

A função `contingency` calcula o [coeficiente de contingência](https://en.wikipedia.org/wiki/Contingency_table#Cram%C3%A9r's_V_and_the_contingency_coefficient_C), um valor que mede a associação entre duas colunas de uma tabela.
O cálculo é semelhante ao da função [`cramersV`](/pt-BR/reference/functions/aggregate-functions/cramersV), mas com um denominador diferente na raiz quadrada.

**Sintaxe**

```sql
contingency(column1, column2)
```

**Argumentos**

* `column1` — Primeira coluna para comparação. [`Any`](/pt-BR/reference/data-types/index)
* `column2` — Segunda coluna para comparação. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um valor entre 0 e 1. Quanto maior o resultado, mais forte é a associação entre as duas colunas. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Comparação com cramersV**

```sql title=Query
SELECT
    cramersV(a, b),
    contingency(a, b)
FROM
(
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌─────cramersV(a, b)─┬──contingency(a, b)─┐
│ 0.5798088336225178 │ 0.708607540104077  │
└────────────────────┴────────────────────┘
```
