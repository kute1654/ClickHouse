<div id="groupBitAnd">
  ## groupBitAnd
</div>

Introduzido em: v1.1.0

Aplica o AND bit a bit a séries de números.

**Sintaxe**

```sql
groupBitAnd(expr)
```

**Aliases**: `BIT_AND`

**Argumentos**

* `expr` — Expressão do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Valor retornado**

Retorna um valor do tipo `(U)Int*`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint)

**Exemplos**

**Exemplo de AND em nível de bit**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Dados de teste:
-- binário    decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Resultado:
-- binário    decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
