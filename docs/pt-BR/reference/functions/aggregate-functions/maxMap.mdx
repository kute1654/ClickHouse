<div id="maxMappedArrays">
  ## maxMappedArrays
</div>

Introduzido na versão: v20.5.0

Calcula o valor máximo do array `value` de acordo com as chaves especificadas no array `key`.

<Note>
  * Passar uma tupla de chaves e arrays de valores é idêntico a passar um array de chaves e um array de valores.
  * O número de elementos em `key` e `value` deve ser o mesmo para cada linha somada.
</Note>

**Sintaxe**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**Argumentos**

* `key` — Array de chaves. [`Array(T)`](/pt-BR/reference/data-types/array)
* `value` — Array de valores. [`Array(T)`](/pt-BR/reference/data-types/array)

**Valor retornado**

Retorna uma tupla com dois arrays: chaves em ordem crescente e valores calculados para as chaves correspondentes. [`Tuple(Array(T), Array(T))`](/pt-BR/reference/data-types/tuple)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)───┐
│ (['x','y','z'],[2,3,1]) │
└─────────────────────────┘
```
