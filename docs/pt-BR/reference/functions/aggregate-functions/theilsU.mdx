<div id="theilsU">
  ## theilsU
</div>

Introduzido em: v22.1.0

A função `theilsU` calcula o [coeficiente de incerteza U de Theil](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient), um valor que mede a associação entre duas colunas de uma tabela.
Seus valores variam de 0,0 (sem associação) a 1,0 (concordância perfeita).

**Sintaxe**

```sql
theilsU(column1, column2)
```

**Argumentos**

* `column1` — Primeira coluna a ser comparada. [`Any`](/pt-BR/reference/data-types/index)
* `column2` — Segunda coluna a ser comparada. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um valor de 0 a 1. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
