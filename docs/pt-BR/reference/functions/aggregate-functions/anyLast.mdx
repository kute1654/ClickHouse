<div id="anyLast">
  ## anyLast
</div>

Introduzido na versão: v1.1.0

Seleciona o último valor encontrado em uma coluna.

<Warning>
  Como uma consulta pode ser executada em ordem arbitrária, o resultado desta função é não determinístico.
  Se você precisar de um resultado arbitrário, mas determinístico, use as funções [min](/pt-BR/reference/functions/aggregate-functions/min) ou [max](/pt-BR/reference/functions/aggregate-functions/max).
</Warning>

Por padrão, a função nunca retorna NULL, ou seja, ignora valores NULL na coluna de entrada.
No entanto, se a função for usada com o modificador `RESPECT NULLS`, ela retornará o último valor lido, independentemente de ser NULL ou não.

**Sintaxe**

```sql
anyLast(column) [RESPECT NULLS]
```

**Aliases**: `last_value`

**Argumentos**

* `column` — O nome da coluna. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna o último valor encontrado. [`Any`](/pt-BR/reference/data-types/index)

**Exemplos**

**Exemplo de uso**

```sql title=Query
CREATE TABLE tab(city Nullable(String)) ENGINE=Memory;
INSERT INTO tab (city) VALUES ('Amsterdam'), (NULL), ('New York'), ('Tokyo'), ('Valencia'), (NULL);
SELECT anyLast(city), anyLastRespectNulls(city) FROM tab;
```

```response title=Response
┌─anyLast(city)─┬─anyLastRespectNulls(city)─┐
│ Valencia      │ ᴺᵁᴸᴸ                      │
└───────────────┴───────────────────────────┘
```
