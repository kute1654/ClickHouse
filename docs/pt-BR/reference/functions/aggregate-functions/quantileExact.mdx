<div id="quantileExact">
  ## quantileExact
</div>

Introduzido em: v1.1.0

Calcula exatamente o [quantil](https://en.wikipedia.org/wiki/Quantile) de uma sequência de dados numéricos.

Para obter o valor exato, todos os valores fornecidos são combinados em um array, que é então parcialmente ordenado.
Portanto, a função consome `O(n)` de memória, em que `n` é o número de valores fornecidos.
No entanto, para uma pequena quantidade de valores, a função é muito eficaz.

Ao usar várias funções `quantile*` com níveis diferentes em uma consulta, os estados internos não são combinados (ou seja, a consulta é executada com menos eficiência do que poderia).
Nesse caso, use a função [`quantiles`](/pt-BR/reference/functions/aggregate-functions/quantiles#quantiles).

**Sintaxe**

```sql
quantileExact(level)(expr)
```

**Aliases**: `medianExact`

**Parameters**

* `level` — Opcional. Nível do quantil. Número de ponto flutuante constante de 0 a 1. Recomendamos usar um valor de `level` no intervalo `[0.01, 0.99]`. Valor padrão: 0.5. Em `level=0.5`, a função calcula a mediana. [`Float*`](/pt-BR/reference/data-types/float)

**Arguments**

* `expr` — Expressão sobre os valores da coluna que resulta em tipos de dados numéricos, Date ou DateTime. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal) ou [`Date`](/pt-BR/reference/data-types/date) ou [`DateTime`](/pt-BR/reference/data-types/datetime)

**Returned value**

Quantil do nível especificado. Para tipos de dados numéricos, o formato de saída será o mesmo que o formato de entrada. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal) ou [`Date`](/pt-BR/reference/data-types/date) ou [`DateTime`](/pt-BR/reference/data-types/datetime)

**Examples**

**Cálculo do quantil exato**

```sql title=Query
SELECT quantileExact(number) FROM numbers(10);
```

```response title=Response
┌─quantileExact(number)─┐
│                     5 │
└───────────────────────┘
```
