## corr {#corr}

Introduzido em: v1.1.0

Calcula o [coeficiente de correlação de Pearson](https://en.wikipedia.org/wiki/Pearson_correlation_coefficient):

$$
\frac{\Sigma{(x - \bar{x})(y - \bar{y})}}{\sqrt{\Sigma{(x - \bar{x})^2} * \Sigma{(y - \bar{y})^2}}}
$$

<br/>

<Note>
Esta função utiliza um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`corrStable`](/reference/functions/aggregate-functions/corrStable). Ela é mais lenta, mas fornece um resultado mais preciso.
</Note>
    

**Sintaxe**

```sql
corr(x, y)
```

**Argumentos**

- `x` — Primeira variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float)
- `y` — Segunda variável. [`(U)Int*`](/reference/data-types/int-uint) ou [`Float*`](/reference/data-types/float)

**Valor retornado**

Retorna o coeficiente de correlação de Pearson. [`Float64`](/reference/data-types/float)

**Exemplos**

**Cálculo básico de correlação**

```sql title=Query
DROP TABLE IF EXISTS series;
CREATE TABLE series
(
    i UInt32,
    x_value Float64,
    y_value Float64
)
ENGINE = Memory;
INSERT INTO series(i, x_value, y_value) VALUES (1, 5.6, -4.4),(2, -9.6, 3),(3, -1.3, -4),(4, 5.3, 9.7),(5, 4.4, 0.037),(6, -8.6, -7.8),(7, 5.1, 9.3),(8, 7.9, -3.6),(9, -8.2, 0.62),(10, -3, 7.3);

SELECT corr(x_value, y_value)
FROM series
```

```response title=Response
┌─corr(x_value, y_value)─┐
│     0.1730265755453256 │
└────────────────────────┘
```
