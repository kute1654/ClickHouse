<div id="stddevPopStable">
  ## stddevPopStable
</div>

Introduzido na versão: v1.1.0

O resultado é igual à raiz quadrada de [varPop](/pt-BR/reference/functions/aggregate-functions/varPop). Diferentemente de [stddevPop](/pt-BR/reference/functions/aggregate-functions/stddevPop), esta função usa um algoritmo numericamente estável. Ela é mais lenta, mas apresenta menor erro computacional.

**Sintaxe**

```sql
stddevPopStable(x)
```

**Argumentos**

* `x` — Conjunto de valores para os quais se deseja calcular o desvio padrão. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal)

**Valor retornado**

Retorna a raiz quadrada da variância de `x`. [`Float64`](/pt-BR/reference/data-types/float)

**Exemplos**

**Uso básico**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population Float64,
)
ENGINE = Log;

INSERT INTO test_data SELECT randUniform(5.5, 10) FROM numbers(1000000);

SELECT
    stddevPopStable(population) AS stddev
FROM test_data;
```

```response title=Response
┌─────────────stddev─┐
│ 1.2999977786592576 │
└────────────────────┘
```
