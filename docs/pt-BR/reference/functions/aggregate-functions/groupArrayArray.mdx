Agrega arrays em um array maior com esses arrays.
Combina a funГ§ГЈo [`groupArray`](/pt-BR/reference/functions/aggregate-functions/groupArray) com o combinador [Array](/pt-BR/reference/functions/aggregate-functions/combinators#-array).

**Exemplo**

Temos dados que capturam sessГөes de navegaГ§ГЈo de usuГЎrios. Cada sessГЈo registra a sequГӘncia de pГЎginas visitadas por um usuГЎrio especГӯfico.
Podemos usar a funГ§ГЈo `groupArrayArray` para analisar os padrГөes de visita Г s pГЎginas de cada usuГЎrio.

```sql title="Setup"
CREATE TABLE website_visits (
    user_id UInt32,
    session_id UInt32,
    page_visits Array(String)
) ENGINE = Memory;

INSERT INTO website_visits VALUES
(101, 1, ['homepage', 'products', 'checkout']),
(101, 2, ['search', 'product_details', 'contact']),
(102, 1, ['homepage', 'about_us']),
(101, 3, ['blog', 'homepage']),
(102, 2, ['products', 'product_details', 'add_to_cart', 'checkout']);
```

```sql title="Query"
SELECT
    user_id,
    groupArrayArray(page_visits) AS user_session_page_sequences
FROM website_visits
GROUP BY user_id;
```
