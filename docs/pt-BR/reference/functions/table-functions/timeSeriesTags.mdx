`timeSeriesTags(db_name.time_series_table)` - Retorna a [tabela de tags](/pt-BR/reference/engines/table-engines/integrations/time-series#tags-table)
usada pela tabela `db_name.time_series_table` cujo motor de tabela é o [TimeSeries](/pt-BR/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS tags_table
```

A função também funciona se a tabela *tags* for a tabela interna:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries TAGS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

As consultas a seguir são equivalentes:

```sql
SELECT * FROM timeSeriesTags(db_name.time_series_table);
SELECT * FROM timeSeriesTags('db_name.time_series_table');
SELECT * FROM timeSeriesTags('db_name', 'time_series_table');
```
