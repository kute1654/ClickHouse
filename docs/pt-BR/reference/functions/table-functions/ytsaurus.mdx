<div id="ytsaurus-table-function">
  # Função de tabela ytsaurus
</div>

<ExperimentalBadge />

A função de tabela permite ler dados de um cluster YTsaurus.

<div id="syntax">
  ## Sintaxe
</div>

```sql
ytsaurus(http_proxy_url, cypress_path, oauth_token, format)
```

<Info>
  Este é um recurso experimental que pode mudar de forma incompatível com versões anteriores em versões futuras.
  Habilite o uso da função de tabela YTsaurus
  com a configuração [allow&#95;experimental&#95;ytsaurus&#95;table&#95;function](/pt-BR/reference/settings/session-settings#allow_experimental_ytsaurus_table_engine).
  Digite o comando `set allow_experimental_ytsaurus_table_function = 1`.
</Info>

<div id="arguments">
  ## Argumentos
</div>

* `http_proxy_url` — URL do proxy HTTP do YTsaurus.
* `cypress_path` — caminho do Cypress para a fonte de dados.
* `oauth_token` — token OAuth.
* `format` — O [formato](/pt-BR/reference/formats/index) da fonte de dados.

**Valor retornado**

Uma tabela com a estrutura especificada para ler dados no caminho do Cypress especificado no cluster YTsaurus.
