Representa o conteúdo dos arquivos de índice e de marcas das tabelas MergeTree. Pode ser usada para introspecção.

<div id="syntax">
  ## Sintaxe
</div>

```sql
mergeTreeIndex(database, table [, with_marks = true] [, with_minmax = true])
```

<div id="arguments">
  ## Argumentos
</div>

| Argumento     | Descrição                                                  |
| ------------- | ---------------------------------------------------------- |
| `database`    | O nome do banco de dados de onde ler o índice e as marcas. |
| `table`       | O nome da tabela de onde ler o índice e as marcas.         |
| `with_marks`  | Se deve incluir colunas com marcas no resultado.           |
| `with_minmax` | Se deve incluir o índice min-max no resultado.             |

<div id="returned_value">
  ## Valor retornado
</div>

Um objeto do tipo tabela com colunas contendo valores do índice primário e do índice min-max (se estiver habilitado) da tabela de origem, colunas com os valores das marcas (se estiverem habilitadas) para todos os arquivos possíveis nas partes de dados da tabela de origem e colunas virtuais:

* `part_name` - O nome da parte de dados.
* `mark_number` - O número da marca atual na parte de dados.
* `rows_in_granule` - O número de linhas no grânulo atual.

A coluna de marcas pode conter o valor `(NULL, NULL)` caso a coluna esteja ausente na parte de dados ou as marcas de um de seus substreams não tenham sido gravadas (por exemplo, em partes compactas).

<div id="usage-example">
  ## Exemplo de uso
</div>

```sql
CREATE TABLE test_table
(
    `id` UInt64,
    `n` UInt64,
    `arr` Array(UInt64)
)
ENGINE = MergeTree
ORDER BY id
SETTINGS index_granularity = 3, min_bytes_for_wide_part = 0, min_rows_for_wide_part = 8;

INSERT INTO test_table SELECT number, number, range(number % 5) FROM numbers(5);

INSERT INTO test_table SELECT number, number, range(number % 5) FROM numbers(10, 10);
```

```sql
SELECT * FROM mergeTreeIndex(currentDatabase(), test_table, with_marks = true);
```

```sql
DESCRIBE mergeTreeIndex(currentDatabase(), test_table, with_marks = true) SETTINGS describe_compact_output = 1;
```
