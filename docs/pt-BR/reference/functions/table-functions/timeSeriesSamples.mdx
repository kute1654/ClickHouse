`timeSeriesSamples(db_name.time_series_table)` - Retorna a [tabela de amostras](/pt-BR/reference/engines/table-engines/integrations/time-series#samples-table)
usada pela tabela `db_name.time_series_table` cujo motor de tabela é [TimeSeries](/pt-BR/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries SAMPLES samples_table
```

A função também funciona quando a tabela *samples* é interna:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries SAMPLES INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

As consultas a seguir são equivalentes:

```sql
SELECT * FROM timeSeriesSamples(db_name.time_series_table);
SELECT * FROM timeSeriesSamples('db_name.time_series_table');
SELECT * FROM timeSeriesSamples('db_name', 'time_series_table');
```

<Note>
  A função `timeSeriesSamples` tem um alias `timeSeriesData`, mantido por compatibilidade com versões anteriores.
</Note>
