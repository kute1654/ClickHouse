Retorna um valor avaliado na linha que estГЎ offset linhas apГіs a linha atual dentro do frame ordenado.

**Sintaxe**

```sql

```

**Valor retornado**

* valor avaliado na linha situada `offset` linhas apГіs a linha atual dentro do frame ordenado.

**Exemplo**

Este exemplo analisa [dados histГіricos](https://www.kaggle.com/datasets/sazidthe1/nobel-prize-data) de vencedores do PrГӘmio Nobel e usa a funГ§ГЈo `lead` para retornar uma lista de vencedores consecutivos na categoria de fГӯsica.


```sql title="Query"

    fullName,
    lead(year, 1, year) OVER (PARTITION BY category ORDER BY year ASC

FROM nobel_prize_laureates
WHERE category = 'physics'
ORDER BY year DESC
LIMIT 9
```
