Retorna um valor avaliado na linha que estГЎ a `offset` linhas apГіs a linha atual dentro da janela ordenada.

<Warning>
  O comportamento de `leadInFrame` difere do da funГ§ГЈo de janela padrГЈo SQL `lead`.
  A funГ§ГЈo de janela `leadInFrame` do ClickHouse respeita o frame da janela.
  Para obter um comportamento idГӘntico ao de `lead`, use `ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING`.
</Warning>

**Sintaxe**

```sql
leadInFrame(x[, offset[, default]])
  OVER ([[PARTITION BY grouping_column] [ORDER BY sorting_column]
        [ROWS or RANGE expression_to_bound_rows_withing_the_group]] | [window_name])
FROM table_name
WINDOW window_name as ([[PARTITION BY grouping_column] [ORDER BY sorting_column])
```

Para mais detalhes sobre a sintaxe de funГ§Гөes de janela, consulte: [FunГ§Гөes de janela - Sintaxe](/pt-BR/reference/functions/window-functions/index#syntax).

**ParГўmetros**

* `x` вҖ” Nome da coluna.
* `offset` вҖ” Deslocamento a ser aplicado. [(U)Int*](/pt-BR/reference/data-types/int-uint). (Opcional вҖ” `1` por padrГЈo).
* `default` вҖ” Valor a ser retornado se a linha calculada exceder os limites do frame da janela. (Opcional вҖ” valor padrГЈo do tipo da coluna quando omitido).

**Valor retornado**

* valor avaliado na linha que estГЎ a `offset` linhas apГіs a linha atual dentro do frame ordenado.

**Exemplo**

Este exemplo analisa [dados histГіricos](https://www.kaggle.com/datasets/sazidthe1/nobel-prize-data) de vencedores do PrГӘmio Nobel e usa a funГ§ГЈo `leadInFrame` para retornar uma lista de vencedores sucessivos na categoria de fГӯsica.

```sql title="Query"

AS SELECT *
FROM file('nobel_laureates_data.csv');
```

```sql title="Query"
SELECT
    fullName,
    leadInFrame(year, 1, year) OVER (PARTITION BY category ORDER BY year ASC
      ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
    ) AS year,
    category,
    motivation
FROM nobel_prize_laureates
WHERE category = 'physics'
ORDER BY year DESC
LIMIT 9
```
