<div id="geometry">
  ## Geometry
</div>

As funções de Geometry permitem calcular o perímetro e a área de tipos geométricos como POLYGON, LINESTRING, MULTIPOLYGON, MULTILINESTRING, RING e POINT. Use geometrias no tipo Geometry. Se o valor de entrada for `NULL`, todas as funções abaixo retornarão 0.

<div id="perimetercartesian">
  ## perimeterCartesian
</div>

Calcula o perímetro do objeto Geometry especificado no sistema de coordenadas cartesianas (plano).

**Sintaxe**

```sql
perimeterCartesian(geom)
```

**Argumentos**

* `geom` — Objeto do tipo Geometry. [Geometry](/pt-BR/reference/data-types/geo).

**Valores retornados**

* Número — Perímetro do objeto nas unidades do sistema de coordenadas. [Float64](/pt-BR/reference/data-types/float).

**Exemplo**

```sql title="Query"
CREATE TABLE IF NOT EXISTS geo_dst (geom Geometry) ENGINE = Memory();
INSERT INTO geo_dst SELECT readWKT('POLYGON((0 0,1 0,1 1,0 1,0 0))');
SELECT perimeterCartesian(geom) FROM geo_dst;
```

```response title="Response"
┌─perimeterCartesian(geom)─┐
│ 4.0                      │
└──────────────────────────┘
```

**Valores retornados**

```sql title="Query"
CREATE TABLE IF NOT EXISTS geo_dst (geom Geometry) ENGINE = Memory();

SELECT areaCartesian(geom) FROM geo_dst;
```
