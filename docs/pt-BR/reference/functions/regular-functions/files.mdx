<div id="file">
  ## file
</div>

Lê um arquivo como String e carrega os dados na coluna especificada. O conteúdo do arquivo não é interpretado.

Veja também a função de tabela [file](/pt-BR/reference/functions/table-functions/file).

**Sintaxe**

```sql
file(path[, default])
```

**Argumentos**

* `path` — O caminho do arquivo relativo a [user&#95;files&#95;path](/pt-BR/reference/settings/server-settings/settings#user_files_path). Suporta os curingas `*`, `**`, `?`, `{abc,def}` e `{N..M}`, em que `N` e `M` são números e `'abc'`, `'def'` são cadeias de caracteres.
* `default` — O valor retornado se o arquivo não existir ou não puder ser acessado. Tipos de dados compatíveis: [String](/pt-BR/reference/data-types/string) e [NULL](/pt-BR/reference/settings/formats#input_format_null_as_default).

**Exemplo**

Inserindo dados dos arquivos a.txt e b.txt em uma tabela como strings:

```sql
INSERT INTO table SELECT file('a.txt'), file('b.txt');
```
