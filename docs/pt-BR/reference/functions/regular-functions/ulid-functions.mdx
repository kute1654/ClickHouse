<div id="ULIDStringToDateTime">
  ## ULIDStringToDateTime
</div>

Introduzido em: v23.3.0

Esta função extrai o timestamp de um [ULID](https://github.com/ulid/spec).

**Sintaxe**

```sql
ULIDStringToDateTime(ulid[, timezone])
```

**Argumentos**

* `ulid` — ULID de entrada. [`String`](/pt-BR/reference/data-types/string) ou [`FixedString(26)`](/pt-BR/reference/data-types/fixedstring)
* `timezone` — Opcional. Nome do fuso horário do valor retornado. [`String`](/pt-BR/reference/data-types/string)

**Valor retornado**

Timestamp com precisão em milissegundos. [`DateTime64(3)`](/pt-BR/reference/data-types/datetime64)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')
```

```response title=Response
┌─ULIDStringToDateTime('01GNB2S2FGN2P93QPXDNB4EN2R')─┐
│                            2022-12-28 00:40:37.616 │
└────────────────────────────────────────────────────┘
```

<div id="generateULID">
  ## generateULID
</div>

Introduzido em: v23.2.0

Gera um [Identificador Universalmente Único Ordenável Lexicograficamente (ULID)](https://github.com/ulid/spec).

**Sintaxe**

```sql
generateULID([x])
```

**Argumentos**

* `x` — Opcional. Uma expressão que resulte em qualquer um dos tipos de dados compatíveis. O valor resultante é descartado, mas a própria expressão é usada para contornar a [eliminação de subexpressões comuns](/pt-BR/reference/functions/regular-functions/overview#common-subexpression-elimination) se a função for chamada várias vezes em uma consulta. [`Any`](/pt-BR/reference/data-types/index)

**Valor retornado**

Retorna um ULID. [`FixedString(26)`](/pt-BR/reference/data-types/fixedstring)

**Exemplos**

**Exemplo de uso**

```sql title=Query
SELECT generateULID()
```

```response title=Response
┌─generateULID()─────────────┐
│ 01GNB2S2FGN2P93QPXDNB4EN2R │
└────────────────────────────┘
```

**Exemplo de uso caso seja necessário gerar vários valores em uma linha**

```sql title=Query
SELECT generateULID(1), generateULID(2)
```

```response title=Response
┌─generateULID(1)────────────┬─generateULID(2)────────────┐
│ 01GNB2SGG4RHKVNT9ZGA4FFMNP │ 01GNB2SGG4V0HMQVH4VBVPSSRB │
└────────────────────────────┴────────────────────────────┘
```
