<div id="description">
  ## Descrição
</div>

Difere do formato [`TabSeparatedWithNames`](/pt-BR/reference/formats/TabSeparated/TabSeparatedWithNames)
porque as linhas são gravadas sem caracteres de escape.

<Note>
  Ao fazer o parsing com este formato, tabulações ou quebras de linha não são permitidas em nenhum campo.
</Note>

</div>

  ### Leitura de dados

```sql

FROM football
FORMAT TabSeparatedRawWithNames
```
