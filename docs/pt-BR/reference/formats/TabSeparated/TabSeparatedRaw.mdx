<div id="description">
  ## Descrição
</div>

Difere do formato [`TabSeparated`](/pt-BR/reference/formats/TabSeparated/TabSeparated) por gravar as linhas sem caracteres de escape.

<Note>
  Ao analisar este formato, tabulações ou quebras de linha não são permitidas em nenhum campo.
</Note>

Para comparar o formato `TabSeparatedRaw` com o formato `RawBlob`, veja: [Comparação entre formatos Raw](/pt-BR/reference/formats/RawBLOB#raw-formats-comparison)


```sql

FROM football
FORMAT TabSeparatedRaw
```
