<div id="description">
  ## Descrição
</div>

Difere do formato [`TabSeparatedWithNamesAndTypes`](/pt-BR/reference/formats/TabSeparated/TabSeparatedWithNamesAndTypes),
pois as linhas são escritas sem caracteres de escape.

<Note>
  Ao fazer o parsing com esse formato, tabulações ou quebras de linha não são permitidas em nenhum campo.
</Note>

  ### Leitura de dados

```sql

FROM football
FORMAT TabSeparatedRawWithNamesAndTypes
```
