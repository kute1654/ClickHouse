<div id="description">
  ## Descrição
</div>

Semelhante a [Template](/pt-BR/reference/formats/Template/Template), mas imprime ou lê todos os nomes e tipos de colunas e usa a regra de escape da configuração [format&#95;custom&#95;escaping&#95;rule](/pt-BR/reference/settings/formats#format_custom_escaping_rule), além dos delimitadores das seguintes configurações:

* [format&#95;custom&#95;field&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_field_delimiter)
* [format&#95;custom&#95;row&#95;before&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_row_before_delimiter)
* [format&#95;custom&#95;row&#95;after&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_row_after_delimiter)
* [format&#95;custom&#95;row&#95;between&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_row_between_delimiter)
* [format&#95;custom&#95;result&#95;before&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_result_before_delimiter)
* [format&#95;custom&#95;result&#95;after&#95;delimiter](/pt-BR/reference/settings/formats#format_custom_result_after_delimiter)

<Note>
  Ele não usa as configurações de regras de escape nem os delimitadores das strings de formato.
</Note>

Também existe o formato [`CustomSeparatedIgnoreSpaces`](/pt-BR/reference/formats/CustomSeparated/CustomSeparatedIgnoreSpaces), que é semelhante ao [TemplateIgnoreSpaces](/pt-BR/reference/formats/Template/TemplateIgnoreSpaces).

<div id="example-usage">
  ## Exemplo de uso
</div>

<div id="inserting-data">
  ### Inserindo dados
</div>

Use o seguinte arquivo txt, chamado `football.txt`:

```text
row('2022-04-30';2021;'Sutton United';'Bradford City';1;4),row('2022-04-30';2021;'Swindon Town';'Barrow';2;1),row('2022-04-30';2021;'Tranmere Rovers';'Oldham Athletic';2;0),row('2022-05-02';2021;'Salford City';'Mansfield Town';2;2),row('2022-05-02';2021;'Port Vale';'Newport County';1;2),row('2022-05-07';2021;'Barrow';'Northampton Town';1;3),row('2022-05-07';2021;'Bradford City';'Carlisle United';2;0),row('2022-05-07';2021;'Bristol Rovers';'Scunthorpe United';7;0),row('2022-05-07';2021;'Exeter City';'Port Vale';0;1),row('2022-05-07';2021;'Harrogate Town A.F.C.';'Sutton United';0;2),row('2022-05-07';2021;'Hartlepool United';'Colchester United';0;2),row('2022-05-07';2021;'Leyton Orient';'Tranmere Rovers';0;1),row('2022-05-07';2021;'Mansfield Town';'Forest Green Rovers';2;2),row('2022-05-07';2021;'Newport County';'Rochdale';0;2),row('2022-05-07';2021;'Oldham Athletic';'Crawley Town';3;3),row('2022-05-07';2021;'Stevenage Borough';'Salford City';4;2),row('2022-05-07';2021;'Walsall';'Swindon Town';0;3)
```

Configure as configurações personalizadas do delimitador:

```sql
SET format_custom_row_before_delimiter = 'row(';
SET format_custom_row_after_delimiter = ')';
SET format_custom_field_delimiter = ';';
SET format_custom_row_between_delimiter = ',';
SET format_custom_escaping_rule = 'Quoted';
```

Insira os dados:

```sql
INSERT INTO football FROM INFILE 'football.txt' FORMAT CustomSeparated;
```

<div id="reading-data">
  ### Leitura de dados
</div>

Configure as configurações personalizadas do delimitador:

```sql
SET format_custom_row_before_delimiter = 'row(';
SET format_custom_row_after_delimiter = ')';
SET format_custom_field_delimiter = ';';
SET format_custom_row_between_delimiter = ',';
SET format_custom_escaping_rule = 'Quoted';
```

Leia os dados usando o formato `CustomSeparated`:

```sql
SELECT *
FROM football
FORMAT CustomSeparated
```

A saída será exibida no formato personalizado configurado:

