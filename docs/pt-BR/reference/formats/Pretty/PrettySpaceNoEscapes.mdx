<div id="description">
  ## Descrição
</div>

Difere do formato [`PrettySpace`](/pt-BR/reference/formats/Pretty/PrettySpace) por não usar [sequências de escape ANSI](http://en.wikipedia.org/wiki/ANSI_escape_code).
Isso é necessário para exibir esse formato em um navegador, bem como para usar o utilitário de linha de comando `watch`.
