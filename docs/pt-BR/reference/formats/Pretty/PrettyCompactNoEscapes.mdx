<div id="description">
  ## Descrição
</div>

Difere do formato [`PrettyCompact`](/pt-BR/reference/formats/Pretty/PrettyCompact) por não usar [sequências de escape ANSI](http://en.wikipedia.org/wiki/ANSI_escape_code).
Isso é necessário para exibir o formato em um navegador, bem como para usar o utilitário de linha de comando &#39;watch&#39;.
