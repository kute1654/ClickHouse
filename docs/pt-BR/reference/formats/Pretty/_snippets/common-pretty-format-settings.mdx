---
title: Configurações comuns dos formatos Pretty
---

{/* Observação: este arquivo é usado como trecho em todos os arquivos que o importam */}

As configurações a seguir são comuns a todos os formatos `Pretty`:

| Configuração                                                                                                                                         | Descrição                                                                                                                                                                                                                            | Padrão  |
| ---------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------- |
| [`output_format_pretty_max_rows`](/pt-BR/reference/settings/formats#output_format_pretty_max_rows)                                                         | Limite de linhas para os formatos Pretty.                                                                                                                                                                                            | `10000` |
| [`output_format_pretty_max_column_pad_width`](/pt-BR/reference/settings/formats#output_format_pretty_max_column_pad_width)                                 | Largura máxima de preenchimento para todos os valores de uma coluna nos formatos Pretty.                                                                                                                                             | `250`   |
| [`output_format_pretty_max_value_width`](/pt-BR/reference/settings/formats#output_format_pretty_max_value_width)                                           | Largura máxima do valor exibido nos formatos Pretty. Se for maior, será truncado.                                                                                                                                                    | `10000` |
| [`output_format_pretty_color`](/pt-BR/reference/settings/formats#output_format_pretty_color)                                                               | Usa sequências de escape ANSI para aplicar cores nos formatos Pretty.                                                                                                                                                                | `true`  |
| [`output_format_pretty_grid_charset`](/pt-BR/reference/settings/formats#output_format_pretty_grid_charset)                                                 | Conjunto de caracteres usado para imprimir as bordas da grade. Conjuntos disponíveis: ASCII, UTF-8.                                                                                                                                  | `UTF-8` |
| [`output_format_pretty_row_numbers`](/pt-BR/reference/settings/formats#output_format_pretty_row_numbers)                                                   | Adiciona números de linha antes de cada linha no formato de saída Pretty.                                                                                                                                                            | `true`  |
| [`output_format_pretty_display_footer_column_names`](/pt-BR/reference/settings/formats#output_format_pretty_display_footer_column_names)                   | Exibe os nomes das colunas no rodapé se a tabela contiver muitas linhas.                                                                                                                                                             | `true`  |
| [`output_format_pretty_display_footer_column_names_min_rows`](/pt-BR/reference/settings/formats#output_format_pretty_display_footer_column_names_min_rows) | Define o número mínimo de linhas a partir do qual um rodapé será exibido, caso [`output_format_pretty_display_footer_column_names`](/pt-BR/reference/settings/formats#output_format_pretty_display_footer_column_names) esteja habilitado. | `50`    |
