<div id="description">
  ## Descrição
</div>

Difere do formato [`Pretty`](/pt-BR/reference/formats/Pretty/Pretty) por exibir a tabela com uma grade entre as linhas.
Por isso, o resultado é mais compacto.

<Note>
  Esse formato é usado por padrão no cliente de linha de comando no modo interativo.
</Note>
