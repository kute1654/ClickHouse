<div id="description">
  ## Descrição
</div>

No formato `Null`, nada é exibido.
À primeira vista, isso pode parecer estranho, mas é importante observar que, embora nada seja exibido, a consulta ainda é processada
e, ao usar o cliente de linha de comando, os dados são transmitidos ao cliente.

<Tip>
  O formato `Null` pode ser útil para testes de desempenho.
</Tip>
