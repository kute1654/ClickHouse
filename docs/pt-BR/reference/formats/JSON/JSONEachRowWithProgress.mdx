<div id="description">
  ## Descrição
</div>

Difere de [`JSONEachRow`](/pt-BR/reference/formats/JSON/JSONEachRow)/[`JSONStringsEachRow`](/pt-BR/reference/formats/JSON/JSONStringsEachRow) porque o ClickHouse também fornece informações de progresso como valores JSON.

<div id="example-usage">
  ## Exemplo de uso
</div>

```json
{"row":{"num":42,"str":"hello","arr":[0,1]}}
{"row":{"num":43,"str":"hello","arr":[0,1,2]}}
{"row":{"num":44,"str":"hello","arr":[0,1,2,3]}}
{"progress":{"read_rows":"3","read_bytes":"24","written_rows":"0","written_bytes":"0","total_rows_to_read":"3"}}
```
