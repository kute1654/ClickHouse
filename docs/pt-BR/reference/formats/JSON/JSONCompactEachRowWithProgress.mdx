<div id="description">
  ## Descrição
</div>

Este formato combina a saída compacta, linha por linha, de JSONCompactEachRow com informações de progresso transmitidas em streaming.
Ele produz dados como objetos JSON separados para metadados, linhas individuais, atualizações de progresso,
totais e exceções. Os valores são representados em seus tipos nativos.

Principais recursos:

* Produz primeiro os metadados, com nomes de colunas e tipos
* Cada linha é um objeto JSON separado, com uma chave &quot;row&quot; contendo um array de valores
* Inclui atualizações de progresso durante a execução da consulta (como objetos `{"progress":...}`)
* Oferece suporte a totais e extremos
* Os valores mantêm seus tipos nativos (números como números, strings como strings)

<div id="example-usage">
  ## Exemplo de uso
</div>

```sql title="Query"
SELECT *
FROM generateRandom('a Array(Int8), d Decimal32(4), c Tuple(DateTime64(3), UUID)', 1, 10, 2)
LIMIT 5
FORMAT JSONCompactEachRowWithProgress
```
