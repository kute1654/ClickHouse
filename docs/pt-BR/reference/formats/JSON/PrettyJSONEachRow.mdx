<div id="description">
  ## Descrição
</div>

Difere de [JSONEachRow](/pt-BR/reference/formats/JSON/JSONEachRow) apenas pelo fato de o JSON ser formatado de forma mais legível, com delimitadores de nova linha e recuos de 4 espaços.

<div id="example-usage">
  ## Exemplo de uso
</div>

<div id="reading-data">
  ### Lendo dados
</div>

Leia os dados usando o formato `PrettyJSONEachRow`:

```sql
SELECT *
FROM football
FORMAT PrettyJSONEachRow
```
