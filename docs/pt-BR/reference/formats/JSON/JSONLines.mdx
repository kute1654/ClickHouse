<div id="description">
  ## Descrição
</div>

Nesse formato, o ClickHouse gera cada linha como um objeto JSON separado por quebras de linha.

Esse formato também é conhecido como `JSONEachRow`, `JSONLines`, `NDJSON` (JSON delimitado por novas linhas) ou `JSONL`. Esses nomes são aliases do mesmo formato e podem ser usados indistintamente tanto para entrada quanto para saída.

<div id="reading-data">
  ### Leitura de dados
</div>

Leia os dados usando o formato `JSONLines`:

```sql
SELECT *
FROM football
FORMAT JSONLines
```
