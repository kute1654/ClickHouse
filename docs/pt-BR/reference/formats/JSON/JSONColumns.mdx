<div id="description">
  ## Descrição
</div>

<Tip>
  A saída dos formatos JSONColumns* fornece o nome do campo no ClickHouse e, em seguida, o conteúdo de cada linha da tabela para esse campo;
  visualmente, os dados são rotacionados 90 graus para a esquerda.
</Tip>

Neste formato, todos os dados são representados como um único objeto JSON.

<Note>
  O formato `JSONColumns` mantém todos os dados em memória e depois os gera como um único bloco, portanto isso pode levar a um alto consumo de memória.
</Note>

<div id="format-settings">
  ## Configurações de formato
</div>

Durante a importação, colunas com nomes desconhecidos serão ignoradas se a configuração [`input_format_skip_unknown_fields`](/pt-BR/reference/settings/formats#input_format_skip_unknown_fields) estiver definida como `1`.
As colunas que não estiverem presentes no bloco serão preenchidas com valores padrão (aqui, você pode usar a configuração [`input_format_defaults_for_omitted_fields`](/pt-BR/reference/settings/formats#input_format_defaults_for_omitted_fields))
