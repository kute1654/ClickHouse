<div id="description">
  ## Descrição
</div>

Neste formato, o ClickHouse gera cada linha como um objeto JSON separado, delimitado por quebras de linha.

<div id="example-usage">
  ## Exemplo de uso
</div>

Insira os dados:

```sql

```

<div id="reading-data">
  ### Leitura de dados
</div>

Leia os dados usando o formato `JSONEachRow`:

```sql
SELECT *
FROM football
FORMAT JSONEachRow
```
