<div id="description">
  ## Descrição
</div>

O formato `LineAsStringWithNames` é semelhante ao formato [`LineAsString`](/pt-BR/reference/formats/LineAsString/LineAsString), mas exibe a linha de cabeçalho com os nomes das colunas.

<div id="example-usage">
  ## Exemplo de uso
</div>

```sql title="Query"
CREATE TABLE example (
    name String,
    value Int32
)
ENGINE = Memory;

INSERT INTO example VALUES ('John', 30), ('Jane', 25), ('Peter', 35);

SELECT * FROM example FORMAT LineAsStringWithNames;
```

```response title="Response"
name    value
John    30
Jane    25
Peter    35
```
