Cancela a exclusão da tabela.

A partir da versão 23.3 do ClickHouse, é possível executar UNDROP em uma tabela de um banco de dados Atomic
até `database_atomic_delay_before_drop_table_sec` (8 minutos por padrão) após emitir a instrução DROP TABLE. As tabelas excluídas são listadas na
tabela de sistema `system.dropped_tables`.

Se você tiver uma visão materializada sem uma cláusula `TO` associada à tabela excluída, também precisará executar UNDROP na tabela interna dessa visão.

<Tip>
  Veja também [DROP TABLE](/pt-BR/reference/statements/drop)
</Tip>

Sintaxe:

```sql
UNDROP TABLE [db.]name [UUID '<uuid>'] [ON CLUSTER cluster]
```

**Exemplo**

```sql
CREATE TABLE tab
(
    `id` UInt8
)
ENGINE = MergeTree
ORDER BY id;

DROP TABLE tab;

SELECT *
FROM system.dropped_tables
FORMAT Vertical;
```

```response
Row 1:
──────

table:                 tab
uuid:                  aa696a1a-1d70-4e60-a841-4c80827706cc

table_dropped_time:    2023-04-05 14:12:12

1 row in set. Elapsed: 0.001 sec. 
```

````sql
UNDROP TABLE tab;

SELECT *
FROM system.dropped_tables
FORMAT Vertical;

```response
Ok.

0 rows in set. Elapsed: 0.001 sec. 
````
