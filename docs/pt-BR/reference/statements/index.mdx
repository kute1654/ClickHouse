Os usuários interagem com o ClickHouse usando instruções SQL. O ClickHouse oferece suporte a instruções SQL comuns, como [SELECT](/pt-BR/reference/statements/select/index) e [CREATE](/pt-BR/reference/statements/create/index), mas também fornece instruções especializadas, como [KILL](/pt-BR/reference/statements/kill) e [OPTIMIZE](/pt-BR/reference/statements/optimize).
