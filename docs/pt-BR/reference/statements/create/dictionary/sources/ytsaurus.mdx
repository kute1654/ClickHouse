<Info>
  Este é um recurso experimental que pode mudar de formas incompatíveis com versões anteriores em versões futuras.
  Habilite o uso da source de dicionário do YTsaurus
  usando a configuração [`allow_experimental_ytsaurus_dictionary_source`](/pt-BR/reference/settings/session-settings#allow_experimental_ytsaurus_dictionary_source).
</Info>

Exemplo de configurações:

<Tabs>
  <Tab title="DDL">
    ```sql
    SOURCE(YTSAURUS(
        http_proxy_urls 'http://localhost:8000'
        cypress_path '//tmp/test'
        oauth_token 'password'
    ))
    ```
  </Tab>

  <Tab title="Arquivo de configuração">
    ```xml
    <source>
        <ytsaurus>
            <http_proxy_urls>http://localhost:8000</http_proxy_urls>
            <cypress_path>//tmp/test</cypress_path>
            <oauth_token>password</oauth_token>
            <check_table_schema>1</check_table_schema>
        </ytsaurus>
    </source>
    ```
  </Tab>
</Tabs>

<br />

Campos de configuração:

| Configuração      | Descrição                                   |
| ----------------- | ------------------------------------------- |
| `http_proxy_urls` | URL do proxy HTTP do YTsaurus.              |
| `cypress_path`    | Caminho do Cypress para a tabela de origem. |
| `oauth_token`     | Token OAuth.                                |
