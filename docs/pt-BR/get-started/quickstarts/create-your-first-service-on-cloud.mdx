<div id="prerequisites">
  ## Pré-requisitos
</div>

Este guia de início rápido não tem pré-requisitos

<div id="what-youll-build">
  ## O que você vai criar
</div>

Neste guia de início rápido, você criará seu primeiro serviço do ClickHouse Cloud, vai se familiarizar com o console do ClickHouse Cloud e o console SQL integrado, e carregará um conjunto de dados de exemplo para ter dados reais para consultar.
Ao final, você terá um serviço do ClickHouse Cloud em funcionamento, com dados carregados, pronto para ser usado como base para o restante da série de guias de início rápido.

<Steps titleSize="h3">
  <Step>
    ### Cadastre-se no ClickHouse Cloud

    Acesse [console.clickhouse.cloud](https://console.clickhouse.cloud/). Você verá a página de acesso. Clique em **Register** ao lado de &quot;Don&#39;t have an account yet?&quot;, na parte inferior.

    <Frame caption="Clique em Register para criar uma nova conta">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="Página de acesso do ClickHouse Cloud com o link Register" />
    </Frame>

    Na página de cadastro, inscreva-se com Google, Microsoft ou informe seu e-mail corporativo. Não é necessário cartão de crédito — a avaliação gratuita oferece US$ 300 em créditos e 30 dias para explorar o ClickHouse Cloud.

    <Frame caption="Cadastre-se com Google, Microsoft ou seu e-mail corporativo">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="Página de cadastro do ClickHouse Cloud" />
    </Frame>

    Se você se cadastrar com e-mail, será solicitado a definir uma senha. Ela deve atender aos seguintes requisitos:

    * Pelo menos 12 caracteres
    * Pelo menos 3 destes itens: letras minúsculas, letras maiúsculas, números e caracteres especiais (por exemplo, `!@#$%^&*`)

    <Frame caption="Informe sua senha e aceite os Termos de Serviço">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="Cadastro por e-mail no ClickHouse Cloud com campo de senha" />
    </Frame>

    Aceite os Termos de Serviço e a Política de Privacidade e, em seguida, clique em **Sign up**. Verifique seu e-mail, abra o link de confirmação e clique em **Verify email** para ativar sua conta.

    Depois de concluir a verificação, faça login com seu novo e-mail e senha.
  </Step>

  <Step>
    ### Crie um novo serviço

    Depois de fazer login pela primeira vez, você será levado diretamente à página de configuração do serviço. Configure o seguinte:

    | Configuração                | Recomendação                                                                                             |
    | --------------------------- | -------------------------------------------------------------------------------------------------------- |
    | **Nome do serviço**         | Dê ao seu serviço um nome descritivo (por exemplo, `Meu primeiro serviço`)                               |
    | **Provedor de Cloud**       | Escolha AWS, GCP ou Azure de acordo com a sua preferência                                                |
    | **Região**                  | Selecione a região mais próxima de você para ter a menor latência                                        |
    | **Memória e escalonamento** | Selecione **Mini 12GB** (1 réplica) neste guia de início rápido - é a opção mais econômica para aprender |

    Clique em **Criar serviço** para provisionar seu serviço.

    <Frame caption="Configure seu serviço em nuvem">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="Página de configuração do serviço ClickHouse Cloud" />
    </Frame>

    Você será direcionado à página inicial do serviço, onde o status aparecerá inicialmente como **Provisioning**. Isso normalmente leva menos de um minuto. Quando for concluído, o status mudará para **Running**.

    <Frame caption="Seu serviço está em execução">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="Página inicial do serviço ClickHouse Cloud mostrando o status Running" />
    </Frame>
  </Step>

  <Step>
    ### Carregar um conjunto de dados de exemplo

    Clique em **Fontes de dados** na barra lateral esquerda. Você verá várias opções para adicionar dados ao seu serviço.

    <Frame caption="Página de Fontes de dados com opções para carregar dados">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="Página de Fontes de dados do ClickHouse Cloud" />
    </Frame>

    Clique em **Adicionar dados de exemplo** para ver os conjuntos de dados de exemplo disponíveis. Você verá três opções: Cell Towers, UK Property Price Paid e New York Taxi Data. Clique em **Começar** no conjunto de dados **UK Property Price Paid**.

    <Frame caption="Selecione o conjunto de dados UK Property Price Paid">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="Seleção de conjuntos de dados de exemplo do ClickHouse Cloud" />
    </Frame>

    Você verá uma prévia dos dados com o esquema da tabela. Clique em **Importar conjunto de dados** para carregar os dados no seu serviço.

    <Frame caption="Visualize os dados e clique em Importar conjunto de dados">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="Prévia da importação de conjunto de dados do ClickHouse Cloud com dados imobiliários do Reino Unido" />
    </Frame>

    A importação levará um ou dois minutos. Quando for concluída, você estará pronto para consultar os dados.
  </Step>

  <Step>
    ### Consultar os dados

    Clique em **SQL Console** na barra lateral esquerda para abrir o editor de consultas integrado. É nele que você vai escrever e executar consultas SQL diretamente no navegador, sem precisar de ferramentas externas.

    Experimente algumas consultas para explorar os dados de preços de imóveis do Reino Unido.

    Encontre os imóveis mais caros já vendidos:

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    Encontre o preço médio de venda por ano:

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    Essas consultas analisam milhões de linhas em segundos - é o armazenamento colunar e a compressão do ClickHouse em ação.
  </Step>
</Steps>
