<Image size="lg" img="/images/integrations/data-visualization/omni_01.webp" alt="Интерфейс администратора Omni с кнопкой Add Connection в разделе Connections" border />

<Image size="lg" img="/images/integrations/data-visualization/omni_02.webp" alt="Интерфейс настройки подключения Omni к ClickHouse с полями для ввода учетных данных" border />

<br />

Теперь вы можете запрашивать и визуализировать данные из ClickHouse в Omni.
