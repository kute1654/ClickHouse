<a href="https://www.zingdata.com/" target="_blank">Zing Data</a> — это платформа для анализа и визуализации данных. Zing Data подключается к ClickHouse с помощью JS-драйвера от ClickHouse.

<div id="how-to-connect">
  ## Как подключиться
</div>

1. Подготовьте сведения о подключении.

<ConnectionDetails />

2. Скачайте Zing Data или откройте его в браузере

   * Чтобы использовать ClickHouse с Zing Data на мобильном устройстве, скачайте приложение Zing Data в [Google Play Store](https://play.google.com/store/apps/details?id=com.getzingdata.android) или [Apple App Store](https://apps.apple.com/us/app/zing-data-collaborative-bi/id1563294091).

   * Чтобы использовать ClickHouse с Zing Data в веб-версии, откройте [веб-консоль Zing](https://console.getzingdata.com/) и создайте учетную запись.

3. Добавьте источник данных

   * Чтобы работать с данными ClickHouse в Zing Data, нужно настроить ***источник данных***. В меню мобильного приложения Zing Data выберите **Sources**, затем нажмите **Add a Datasource**.

   * Чтобы добавить источник данных в веб-версии, нажмите **Data Sources** в верхнем меню, затем **New Datasource** и выберите **ClickHouse** из раскрывающегося списка

   <Image size="md" img="/images/integrations/data-visualization/zing_01.webp" alt="Интерфейс Zing Data с кнопкой New Datasource и вариантом ClickHouse в раскрывающемся списке" border />

   <br />

4. Заполните сведения о подключении и нажмите **Check Connection**.

   <Image size="md" img="/images/integrations/data-visualization/zing_02.webp" alt="Форма настройки подключения ClickHouse в Zing Data с полями для сервера, порта, базы данных, имени пользователя и пароля" border />

   <br />

5. Если подключение прошло успешно, Zing предложит выбрать таблицы. Выберите нужные таблицы и нажмите **Save**. Если Zing не сможет подключиться к вашему источнику данных, появится сообщение с просьбой проверить учетные данные и повторить попытку. Если проблема сохраняется даже после проверки учетных данных и повторной попытки, <a id="contact_link" href="mailto:hello@getzingdata.com">обратитесь в поддержку Zing здесь.</a>

   <Image size="md" img="/images/integrations/data-visualization/zing_03.webp" alt="Интерфейс выбора таблиц в Zing Data, показывающий доступные таблицы ClickHouse с флажками" border />

   <br />

6. После добавления источника данных ClickHouse он будет доступен всем пользователям вашей организации Zing на вкладке **Data Sources** / **Sources**.

<div id="creating-charts-and-dashboards-in-zing-data">
  ## Создание диаграмм и панелей мониторинга в Zing Data
</div>

1. После добавления источника данных ClickHouse нажмите **Zing App** в веб-версии или источник данных на мобильном устройстве, чтобы начать создавать диаграммы.

2. Нажмите на таблицу в списке таблиц, чтобы создать диаграмму.

   <Image size="sm" img="/images/integrations/data-visualization/zing_04.webp" alt="Интерфейс Zing Data со списком таблиц и доступными таблицами ClickHouse" border />

   <br />

3. Используйте визуальный конструктор запросов, чтобы выбрать нужные поля, агрегации и т. д., затем нажмите **Run Question**.

   <Image size="md" img="/images/integrations/data-visualization/zing_05.webp" alt="Интерфейс визуального конструктора запросов в Zing Data с выбором полей и параметрами агрегации" border />

   <br />

4. Если вы знакомы с SQL, вы также можете написать собственный SQL для выполнения запросов и создания диаграммы.

   <Image size="md" img="/images/integrations/data-visualization/zing_06.webp" alt="Режим редактора SQL в Zing Data с интерфейсом для написания SQL-запроса" border />

   <Image size="md" img="/images/integrations/data-visualization/zing_07.webp" alt="Результаты SQL-запроса в Zing Data с данными в табличном формате" border />

5. Пример диаграммы будет выглядеть так. Запрос можно сохранить через меню с тремя точками. Вы можете добавить комментарий к диаграмме, отметить тегом участников команды, создать оповещения в реальном времени, изменить тип диаграммы и т. д.

   <Image size="md" img="/images/integrations/data-visualization/zing_08.webp" alt="Пример диаграммы в Zing Data с данными из ClickHouse и меню параметров" border />

   <br />

6. Панели мониторинга можно создавать с помощью значка &quot;+&quot; в разделе **Dashboards** на главном экране. В них можно перетаскивать существующие запросы, чтобы отображать их на панели мониторинга.

   <Image size="md" img="/images/integrations/data-visualization/zing_09.webp" alt="Панель мониторинга в Zing Data с несколькими визуализациями, размещенными на макете панели" border />

   <br />
