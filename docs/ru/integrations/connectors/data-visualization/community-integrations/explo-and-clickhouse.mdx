<div id="2--connect-explo-to-clickhouse">
  ## 2.  Подключите Explo к ClickHouse
</div>

1. Зарегистрируйтесь в Explo.

2. Нажмите вкладку Explo **Data** на левой боковой панели.

<Image img="/images/integrations/data-visualization/explo_01.webp" size="sm" alt="Вкладка Data" border />

3. Нажмите **Connect Data Source** в правом верхнем углу.

<Image img="/images/integrations/data-visualization/explo_02.webp" size="sm" alt="Connect Data Source" border />

4. Заполните информацию на странице **Getting Started**

<Image img="/images/integrations/data-visualization/explo_03.webp" size="md" alt="Getting Started" border />

5. Выберите **ClickHouse**

<Image img="/images/integrations/data-visualization/explo_04.webp" size="md" alt="ClickHouse" border />

6. Введите **учетные данные ClickHouse**.

<Image img="/images/integrations/data-visualization/explo_05.webp" size="md" alt="Учетные данные" border />

7. Настройте **безопасность**

<Image img="/images/integrations/data-visualization/explo_06.webp" size="md" alt="Безопасность" border />

8. В ClickHouse **добавьте IP-адреса Explo в белый список**.
   `54.211.43.19, 52.55.98.121, 3.214.169.94 и 54.156.141.148`

<div id="3-create-a-dashboard">
  ## 3. Создайте панель мониторинга
</div>

1. Перейдите на вкладку **Панель мониторинга** на левой навигационной панели.

<Image img="/images/integrations/data-visualization/explo_07.webp" size="sm" alt="Панель мониторинга" border />

2. Нажмите **Создать панель мониторинга** в правом верхнем углу и задайте имя панели мониторинга. Панель мониторинга создана!

<Image img="/images/integrations/data-visualization/explo_08.webp" size="sm" alt="Создать панель мониторинга" border />

3. Теперь вы увидите экран, похожий на этот:

<Image img="/images/integrations/data-visualization/explo_09.webp" size="md" alt="Панель мониторинга Explo" border />

<div id="4-run-a-sql-query">
  ## 4. Выполните SQL-запрос
</div>

1. Найдите имя таблицы на правой боковой панели под заголовком схемы. Затем вставьте следующую команду в редактор набора данных:
   `SELECT * FROM YOUR_TABLE_NAME
   LIMIT 100`

<Image img="/images/integrations/data-visualization/explo_10.webp" size="md" alt="Панель мониторинга Explo" border />

2. Теперь нажмите run и перейдите на вкладку preview, чтобы просмотреть данные.

<Image img="/images/integrations/data-visualization/explo_11.webp" size="md" alt="Панель мониторинга Explo" border />

<div id="5-build-a-chart">
  ## 5. Постройте диаграмму
</div>

1. Перетащите значок столбчатой диаграммы с левой панели на экран.

<Image img="/images/integrations/data-visualization/explo_16.webp" size="sm" alt="Explo Dashboard" border />

2. Выберите набор данных. Теперь вы увидите экран, похожий на следующий:

<Image img="/images/integrations/data-visualization/explo_12.webp" size="sm" alt="Explo Dashboard" border />

3. Заполните поле **county** для оси X и **Price** в разделе оси Y, как показано ниже:

<Image img="/images/integrations/data-visualization/explo_13.webp" size="sm" alt="Explo Dashboard" border />

4. Теперь измените агрегацию на **AVG**.

<Image img="/images/integrations/data-visualization/explo_14.webp" size="sm" alt="Explo Dashboard" border />

5. Теперь у нас есть средняя цена домов с разбивкой по округам!

<Image img="/images/integrations/data-visualization/explo_15.webp" size="md" alt="Explo Dashboard" />
