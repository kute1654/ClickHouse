QStudio — это бесплатный SQL GUI, который позволяет запускать SQL-скрипты, удобно просматривать таблицы, строить графики и экспортировать результаты. Он работает на любой операционной системе и с любой базой данных.

QStudio подключается к ClickHouse через JDBC.

<div id="1-gather-your-clickhouse-details">
  ## 1. Подготовьте данные для подключения к ClickHouse
</div>

QStudio использует JDBC через HTTP(S) для подключения к ClickHouse; вам понадобятся:

* конечная точка
* номер порта
* имя пользователя
* пароль

  ## 2. Скачайте QStudio
</div>

<div id="3-add-a-database">
  ## 3. Добавьте базу данных
</div>

* При первом запуске QStudio нажмите в меню **Server-&gt;Add Server** или кнопку добавления сервера на панели инструментов.
* Затем укажите следующие данные:

<Image img="/images/integrations/sql-clients/qstudio-add-connection.webp" size="lg" border alt="Экран настройки подключения к базе данных в QStudio с параметрами подключения ClickHouse" />

1. Server Type: Clickhouse.com
2. Обратите внимание: в поле Host обязательно нужно указать https://
   Host: https://abc.def.clickhouse.cloud
   Port: 8443
3. Username: default
   Password: `XXXXXXXXXXX`
4. Нажмите Add

Если QStudio обнаружит, что у вас не установлен ClickHouse JDBC driver, он предложит загрузить его автоматически:

<div id="4-query-clickhouse">
  ## 4. Выполнение запросов в ClickHouse
</div>

* Откройте редактор запросов и выполните запрос. Запросы можно выполнять с помощью:

* Ctrl + e — выполняет выделенный текст

* Ctrl + Enter — выполняет текущую строку
