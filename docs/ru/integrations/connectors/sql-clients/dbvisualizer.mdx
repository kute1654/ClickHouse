<div id="3-connect-to-clickhouse">
  ## 3. Подключение к ClickHouse
</div>

Чтобы подключить базу данных в DbVisualizer, сначала нужно создать и настроить подключение к базе данных.

1. Создайте новое подключение через **Database-&gt;Create Database Connection** и выберите драйвер для вашей базы данных во всплывающем меню.

2. Откроется вкладка **Object View** для нового подключения.

3. Введите имя подключения в поле **Name** и, при необходимости, описание подключения в поле **Notes**.

4. Оставьте значение **Database Type** как **Auto Detect**.

5. Если выбранный драйвер в **Driver Type** отмечен зелёной галочкой, он готов к использованию. Если зелёной галочки нет, возможно, вам потребуется настроить драйвер в **Driver Manager**.

6. В оставшихся полях укажите сведения о сервере базы данных.

7. Нажмите кнопку **Ping Server**, чтобы проверить, можно ли установить сетевое connection с указанным адресом и портом.

8. Если результат Ping Server показывает, что server доступен, нажмите **Connect**, чтобы подключиться к серверу базы данных.

:::tip
Если у вас возникают проблемы с подключением к базе данных, ознакомьтесь с [Fixing Connection Issues](https://www.dbvis.com/docs/ug/troubleshooting/fixing-connection-issues/) — там приведены полезные советы.
