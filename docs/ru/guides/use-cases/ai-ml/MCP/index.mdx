[Model Context Protocol](https://www.anthropic.com/news/model-context-protocol) (MCP) — это стандарт, разработанный Anthropic, который позволяет ИИ-ассистентам легко интегрироваться с внешними системами.
Этот протокол позволяет ИИ-ассистентам безопасно и единообразно подключаться к источникам данных, API, базам данных и другим системам.

MCP создает универсальный интерфейс между моделями ИИ и различными сервисами, устраняя необходимость разрабатывать отдельные инструменты для каждой интеграции.
Его можно рассматривать как универсальный стандарт API, разработанный специально для систем ИИ.

Ключевое преимущество MCP заключается в том, что библиотекам ИИ достаточно реализовать поддержку этого протокола только один раз.
После этого все сервисы, совместимые с MCP, сразу становятся доступными, что существенно экономит время сопровождающих библиотек ИИ.

<div id="mcp-architecture">
  ## Как устроена архитектура MCP?
</div>

MCP использует клиент-серверную архитектуру:

* Клиенты (например, Claude Desktop, Cursor или VS Code) устанавливают соединения с MCP-серверами. Подборку клиентов можно найти в GitHub-репозитории [awesome-mcp-clients](https://github.com/punkpeye/awesome-mcp-clients?tab=readme-ov-file#windsurf).
* Серверы предоставляют инструменты и возможности через стандартизированные интерфейсы. Подборку серверов можно найти в GitHub-репозитории [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers).
* Затем ИИ-модели могут использовать эти инструменты для доступа к внешним данным и функциональности по мере необходимости

<div id="clickhouse-mcp-server">
  ## Есть ли у ClickHouse MCP-сервер?
</div>

Да!
[MCP-сервер ClickHouse](https://github.com/ClickHouse/mcp-clickhouse) предоставляет следующие инструменты:

* `run_select_query` - Выполнение SQL-запросов к вашему кластеру ClickHouse.
* `list_databases` - Вывод списка всех баз данных в вашем кластере ClickHouse.
* `list_tables` - Вывод списка всех таблиц в базе данных.

Если вам нужен удалённый MCP-сервер в ClickHouse Cloud, см. страницу [&quot;Удалённый MCP-сервер в Cloud&quot;](/ru/products/cloud/features/ai-ml/remote-mcp)

<Tip>
  **Agent Skills**

  [Репозиторий ClickHouse Agent Skills](https://github.com/ClickHouse/agent-skills) содержит готовые инструкции, которые расширяют возможности ИИ-агентов для разработки (Claude Code, Cursor, Copilot и т. д.) за счёт экспертизы в предметной области. В этом репозитории собраны навыки для работы с базами данных ClickHouse, включая проектирование схем, оптимизацию запросов и шаблоны ингестии данных.
</Tip>
