  ### Архитектура демо
</div>

Демо включает микросервисы, написанные на разных языках программирования и взаимодействующие друг с другом по gRPC и HTTP, а также генератор нагрузки, использующий Locust для имитации пользовательского трафика. Исходный код этого демо был изменён для использования [инструментирования ClickStack](/ru/clickstack/ingesting-data/sdks/index).
