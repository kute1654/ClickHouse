ClickHouse Cloud отправляет уведомления о важных событиях, связанных с вашим сервисом или организацией. Чтобы понять, как отправляются и настраиваются уведомления, полезно учитывать несколько основных понятий:

1. **Категория уведомлений**: группы уведомлений, например уведомления о биллинге, уведомления, связанные с сервисом, и т. д. В каждой категории есть несколько типов уведомлений, для которых можно настроить способ доставки.
2. **Уровень важности уведомления**: может быть `info`, `warning` или `critical` в зависимости от значимости уведомления. Этот параметр не настраивается.
3. **Канал уведомлений**: способ получения уведомления, например через интерфейс, электронную почту, Slack и т. д. Для большинства уведомлений его можно настроить.

<div id="receiving-notifications">
  ## Получение уведомлений
</div>

Уведомления можно получать по разным каналам. ClickHouse Cloud поддерживает получение уведомлений по электронной почте, через интерфейс ClickHouse Cloud и в Slack. Чтобы просмотреть текущие уведомления, нажмите на значок колокольчика в меню в левом верхнем углу — откроется выдвижная панель. Если нажать кнопку **View All** в нижней части выдвижной панели, вы перейдёте на страницу с журналом активности по всем уведомлениям.

<div id="customizing-notifications">
  ## Настройка уведомлений
</div>

Для каждого уведомления можно настроить способ его получения. Открыть экран настроек можно из выдвижной панели уведомлений или на второй вкладке в журнале активности уведомлений.

Вы можете настраивать уведомления, доставляемые через интерфейс Cloud, и эти настройки применяются отдельно для каждого пользователя. Также можно настраивать уведомления, отправляемые на ваши собственные адреса электронной почты, однако только пользователи с правами администратора могут настраивать уведомления, отправляемые на дополнительные адреса электронной почты, а также уведомления, отправляемые в каналы Slack.

Чтобы настроить доставку для конкретного уведомления, нажмите значок карандаша и измените каналы доставки уведомлений.

<Image img="/images/cloud/manage/notifications-3.webp" size="md" alt="Экран настроек уведомлений ClickHouse Cloud" border />

<Image img="/images/cloud/manage/notifications-4.webp" size="md" alt="Настройки доставки уведомлений ClickHouse Cloud" border />

<Note>
  Некоторые **обязательные** уведомления, такие как **Payment failed**, настраивать нельзя.
</Note>

<div id="clickpipes-notifications">
  ## Уведомления ClickPipes
</div>

ClickHouse отправляет уведомления ClickPipes, когда в вашем ClickPipe возникают сбои или неполадки.

<div id="billing-notifications">
  ## Уведомления о биллинге
</div>

ClickHouse отправляет уведомления о биллинге при проблемах с оплатой, а также когда потребление по предоплаченным обязательствам достигает определённых пороговых значений.

<div id="supported-notifications">
  ## Поддерживаемые уведомления
</div>

В настоящее время мы отправляем уведомления, связанные с биллингом (сбой оплаты, превышение определенного порога потребления и т. п.), а также уведомления о событиях масштабирования (завершение масштабирования, блокировка масштабирования и т. д.).
