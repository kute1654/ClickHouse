ClickHouse Agents позволяет выполнять запросы к данным ClickHouse и изучать их через диалог, не записывая SQL и не создавая логику оркестрации вручную.
Агент интерпретирует ваше намерение, планирует шаги, вызывает настроенные вами инструменты и возвращает результаты.
Сейчас эта возможность находится в статусе бета; поведение и функциональность могут измениться до выхода в общую доступность.

<div id="what-you-can-do">
  ## Что можно делать
</div>

С ClickHouse Agents вы можете:

* Создавать собственных агентов без кода. Это позволяет писать инструкции, выбирать модель и подключать инструменты.
* Вести диалоги с вашим сервисом ClickHouse, а агент будет вызывать инструменты по мере необходимости.
* Делиться своими агентами с коллегами или публиковать их в маркетплейсе.
