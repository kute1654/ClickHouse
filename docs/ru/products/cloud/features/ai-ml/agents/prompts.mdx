Библиотека промптов — это место, где можно сохранять и повторно использовать промпты на естественном языке, которые вам приходится вводить снова и снова. По сути, это сниппеты для поля ввода чата — удобно, когда один и тот же аналитический вопрос или инструкция по форматированию повторяются в разных диалогах.

<div id="create-a-prompt">
  ## Создание промпта
</div>

Откройте панель **промптов** с помощью значка **промптов** в левой панели навигации и нажмите кнопку **+**, чтобы открыть форму **Create Prompt**. Заполните поля:

* **Prompt Name** (обязательно) — название, которое будет отображаться в списке выбора. Используйте понятные названия: *&quot;Weekly active users by region&quot;* лучше, чем *&quot;WAU&quot;*.
* **Text** (обязательно) — текст, который будет вставлен в поле ввода.
* **Special variables** — нажмите кнопку **Special variables**, чтобы вставить плейсхолдеры, или введите маркеры в формате `{{name}}` вручную. Перед вставкой список выбора предложит указать значения.
* **Category**, **Description**, **Command** (необязательно) — для организации библиотеки, текста предпросмотра в списке выбора и ярлыка для быстрого вызова.

Затем нажмите **Create Prompt** в правом нижнем углу.

  ## Использование промпта
</div>

На панели промптов откройте меню **...** на карточке промпта и выберите **Предпросмотр**:

В окне предпросмотра отображаются текст промпта, его автор и дата. Нажмите **Использовать промпт**, чтобы вставить содержимое в поле ввода. Если в промпте есть переменные, сначала заполните их.

<div id="share-prompts">
  ## Общий доступ к промптам
</div>

По умолчанию промпт доступен только тому, кто его создал. Владелец может изменить видимость промпта на:

* **Конкретные пользователи или группы** — любой, кого вы укажете, сможет найти и использовать промпт.
* **Вся организация** — любой пользователь в вашей организации ClickHouse Cloud сможет найти и использовать промпт.

Для промптов используется та же модель прав доступа, что и для агентов. Полную матрицу ролей и сведения о том, что может делать каждая из них, см. в разделе [Общий доступ и права доступа](/ru/products/cloud/features/ai-ml/agents/sharing-and-access).

<div id="prompts-vs-skills-vs-instructions">
  ## Промпты, навыки и инструкции: в чем разница
</div>

Промпты, навыки и инструкции агента добавляют текст в модель, но различаются тем, что именно их запускает и насколько долго они действуют.

* **Промпты** - текст, который вы сами вставляете в поле ввода и редактируете для каждого запроса.
* **[Навыки](/ru/products/cloud/features/ai-ml/agents/builder/skills)** - наборы инструкций, которые агент загружает сам, когда считает их релевантными для задачи.
* **Инструкции агента** - постоянный системный промпт агента, который применяется к каждому диалогу.

Используйте промпт, когда хотите повторно использовать формулировки, но при этом каждый раз самостоятельно задавать их точный вид. Используйте навык, когда хотите, чтобы агент
последовательно применял одни и те же указания к определенному типу задач без необходимости вводить их вручную. Используйте инструкции агента, когда нужное поведение должно сохраняться в течение
всего жизненного цикла агента.
