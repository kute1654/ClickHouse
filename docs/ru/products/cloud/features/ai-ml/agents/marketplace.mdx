Agent Marketplace — это раздел, где вы можете просматривать, искать и запускать агентов, которыми с вами поделились другие пользователи вашей организации. Здесь также отображаются агенты, опубликованные вашим администратором для всей организации. Откройте его через пункт **Agent Marketplace** на левой боковой панели.

<Image img="/images/cloud/agent-builder/marketplace/marketplace.webp" alt="Экран чата с выделенным пунктом Agent Marketplace на левой боковой панели" size="lg" />

<div id="browse">
  ## Обзор
</div>

Агенты сгруппированы по категориям — например, *Общее*, *Управление персоналом*, *Исследования и разработки*, *Финансы*, *IT*, *Продажи*. Переключаться между категориями можно с помощью вкладок в верхней части маркетплейса. На каждой карточке агента указаны название, значок и категория, к которой он относится.

Используйте строку поиска, чтобы найти агента по названию или ключевому слову во всех категориях.

<Image img="/images/cloud/agent-builder/marketplace/browse.webp" alt="Представление Agent Marketplace с заголовком, строкой поиска, вкладками категорий (General, Human Resources, Research and Development, Finance, IT, Sales, After Sales, All) и видимой карточкой агента" size="lg" />

<div id="open-an-agent">
  ## Откройте агента
</div>

Нажмите на карточку любого агента, чтобы открыть страницу с подробной информацией. Отсюда можно:

* **Start Chat** — открыть новый чат с агентом.
* **Pin** — добавить агента в избранное для быстрого доступа.
* **Copy link** — поделиться прямой ссылкой на агента.

<Image img="/images/cloud/agent-builder/marketplace/use-agent.webp" alt="Модальное окно с информацией об агенте, в котором показаны имя агента, значок и действия Pin, Copy link и Start Chat" size="md" />

<div id="publish-your-own">
  ## Опубликуйте своего агента
</div>

Чтобы агента можно было найти в маркетплейсе, откройте к нему нужный уровень доступа на панели [совместное использование и доступ](/ru/products/cloud/features/ai-ml/agents/sharing-and-access). Уровни видимости:

* **Private** - его видите только вы. В маркетплейсе не отображается.
* **Shared with users or groups** - виден этим субъектам. Появляется в их маркетплейсе.
* **Organization-wide** - виден всем в организации. Зависит от разрешений на маркетплейс, настроенных вашим администратором.

Перед публикацией выберите для агента категорию и напишите понятное описание — эти поля влияют и на поиск, и на просмотр по категориям.

<div id="admin-controls">
  ## Управление для роли Admin
</div>

Администраторы организации могут:

* Настраивать набор категорий, отображаемых в маркетплейсе.
* Продвигать отдельных агентов в представление для всей организации.
* Разрешать или ограничивать доступ к маркетплейсу для каждой Role.

См. раздел [совместное использование и доступ](/ru/products/cloud/features/ai-ml/agents/sharing-and-access), где описана модель разрешений.
