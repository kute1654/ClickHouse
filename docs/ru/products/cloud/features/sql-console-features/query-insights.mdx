Функция **Query Insights** упрощает работу со встроенным журналом запросов ClickHouse с помощью визуализаций и таблиц. Таблица `system.query_log` в ClickHouse — ключевой источник информации для оптимизации запросов, отладки, а также мониторинга общего состояния и производительности кластера.

<div id="query-overview">
  ## Обзор запросов
</div>

После выбора сервиса пункт **Monitoring** в левой боковой панели должен развернуться, и в нем появится новый подпункт **Query insights**. При нажатии на него открывается новая страница Query insights:

<Image img="/images/cloud/sqlconsole/insights_overview.webp" size="md" alt="Обзор интерфейса Query Insights" border />

<div id="top-level-metrics">
  ## Метрики верхнего уровня
</div>

Блоки статистики в верхней части показывают несколько основных метрик запросов за выбранный период времени. Ниже представлены три графика временных рядов, отражающие объём запросов, задержку и долю ошибок в разбивке по типу запроса (select, insert, other) в выбранном временном окне. График задержки можно дополнительно настроить для отображения значений p50, p90 и p99:

<Image img="/images/cloud/sqlconsole/insights_latency.webp" size="md" alt="График задержки в интерфейсе Query Insights" border />

<div id="recent-queries">
  ## Недавние запросы
</div>

Под сводными метриками верхнего уровня таблица показывает записи журнала запросов (сгруппированные по нормализованному хешу запроса и пользователю) за выбранный временной интервал:

<Image img="/images/cloud/sqlconsole/insights_recent.webp" size="md" alt="Интерфейс Query Insights: таблица недавних запросов" border />

В разделе недавних запросов можно фильтровать и сортировать данные по любому доступному полю. Таблицу также можно настроить так, чтобы отображать или скрывать дополнительные поля, например таблицы, задержки p90 и p99."}]}

<div id="query-drill-down">
  ## Детализация запроса
</div>

При выборе запроса в таблице недавних запросов откроется выдвижная панель с метриками и информацией по выбранному запросу:

<Image img="/images/cloud/sqlconsole/insights_drilldown.webp" size="md" alt="Интерфейс Query Insights: детализация запроса" border />

Как видно из выдвижной панели, этот запрос выполнялся более 3000 раз за последние 24 часа. Все метрики на вкладке **Информация о запросе** являются агрегированными, но метрики отдельных запусков также можно просмотреть, выбрав вкладку **История запроса**:

<Image img="/images/cloud/sqlconsole/insights_query_info.webp" size="sm" alt="Интерфейс Query Insights: информация о запросе" border />

<br />

На этой панели для каждого запуска запроса можно развернуть элементы `Settings` и `Profile Events`, чтобы увидеть дополнительную информацию.
