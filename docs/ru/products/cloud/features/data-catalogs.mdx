ClickHouse Cloud может напрямую подключаться к каталогам данных для открытых табличных форматов, предоставляя доступ к таблицам в озерах данных без дублирования данных.
Благодаря этой интеграции таблицы из вашего каталога будут отображаться в ClickHouse как базы данных, доступные для запросов.
Настройка доступна как через SQL-команду ([DataLakeCatalog](/ru/reference/engines/database-engines/datalake)), так и через интерфейс ClickHouse Cloud на вкладке Data Sources.

Через интерфейс:

* Упрощенная настройка с помощью формы с полями, соответствующими объектам вашего каталога данных
* Единый интерфейс для активных интеграций с каталогами данных
* Проверка подключений и учетных данных при сохранении
