<Note>
  Использование MySQL и MongoDB CDC ClickPipes бесплатно до достижения стадии General Availability (GA). Клиенты будут заранее уведомлены о выходе в GA, чтобы они могли проверить и оптимизировать использование ClickPipes.
</Note>
