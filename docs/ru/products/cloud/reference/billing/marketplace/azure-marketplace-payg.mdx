Начните работу с ClickHouse Cloud на [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps) в рамках публичного предложения PAYG (оплата по мере использования).

<div id="prerequisites">
  ## Предварительные требования
</div>

* Проект Azure, для которого ваш администратор по биллингу включил права на покупку.
* Чтобы подписаться на ClickHouse Cloud в Azure Marketplace, вы должны войти в аккаунт с правами на покупку и выбрать соответствующий проект.

1. Перейдите в [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps) и найдите ClickHouse Cloud. Убедитесь, что вы вошли в систему, чтобы иметь возможность приобрести предложение в маркетплейсе.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-1.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

2. На странице продукта нажмите **Get It Now**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-2.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

3. На следующем экране вам нужно будет указать имя, адрес электронной почты и данные о местоположении.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-3.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

4. На следующем экране нажмите **Subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-4.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

5. На следующем экране выберите подписку, группу ресурсов и расположение группы ресурсов. Расположение группы ресурсов не обязательно должно совпадать с местом, где вы планируете запускать свои сервисы в ClickHouse Cloud.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-5.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

6. Вам также нужно будет указать имя подписки, а также выбрать срок биллинга из доступных вариантов. Вы можете включить или отключить **Recurring billing**. Если установить значение &quot;off&quot;, ваш контракт завершится после окончания срока биллинга, а ваши ресурсы будут выведены из эксплуатации.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-6.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

7. Нажмите **&quot;Review + subscribe&quot;**.

8. На следующем экране проверьте, что все указано верно, и нажмите **Subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-7.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

9. Обратите внимание: на этом этапе вы оформите подписку Azure для ClickHouse Cloud, но ваш аккаунт в ClickHouse Cloud еще не будет настроен. Следующие шаги необходимы и критически важны, чтобы ClickHouse Cloud мог привязаться к вашей подписке Azure и биллинг через Azure Marketplace работал корректно.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-8.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

10. После завершения настройки Azure кнопка **Configure account now** должна стать активной.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-9.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

11. Нажмите **Configure account now**.

<br />

Вы получите электронное письмо, подобное показанному ниже, с информацией о настройке аккаунта:

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-10.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

12. Вы будете перенаправлены на страницу регистрации или входа в ClickHouse Cloud. После перехода в ClickHouse Cloud вы сможете либо войти с существующим аккаунтом, либо зарегистрировать новый. Этот шаг очень важен, чтобы мы могли привязать вашу организацию ClickHouse Cloud к биллингу Azure Marketplace.

13. Обратите внимание: если вы новый пользователь, вам также потребуется предоставить базовую информацию о вашей компании. См. скриншоты ниже.

<br />

<Image img="/images/cloud/manage/billing/marketplace/aws-marketplace-payg-8.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

После нажатия **Complete sign up** вы попадете в свою организацию в ClickHouse Cloud, где сможете открыть экран биллинга, убедиться, что оплата проходит через Azure Marketplace, и создавать сервисы.

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-11.webp" size="md" alt="Форма с информацией для регистрации в ClickHouse Cloud" border />

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-12.webp" size="md" alt="Форма регистрации в ClickHouse Cloud" border />

<br />

14. Если у вас возникнут какие-либо проблемы, пожалуйста, обращайтесь в [нашу службу поддержки](https://clickhouse.com/support/program).
