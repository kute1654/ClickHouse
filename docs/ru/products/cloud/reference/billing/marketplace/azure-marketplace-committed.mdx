Начните работу с ClickHouse Cloud на [Azure Marketplace](https://azuremarketplace.microsoft.com/en-us/marketplace/apps) по контракту с обязательством по расходам. Такой контракт, также известный как персональное предложение, позволяет клиентам взять на себя обязательство потратить определённую сумму на ClickHouse Cloud в течение определённого периода времени.

<div id="prerequisites">
  ## Необходимые условия
</div>

* Персональное предложение от ClickHouse на определённых договорных условиях.

<div id="steps-to-sign-up">
  ## Шаги по регистрации
</div>

1. Вы должны были получить электронное письмо со ссылкой, по которой можно просмотреть и принять ваше персональное предложение.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-1.webp" size="md" alt="Электронное письмо с персональным предложением Azure Marketplace" border />

<br />

2. Нажмите ссылку **Review Private Offer** в письме. Вы перейдете на страницу Azure Marketplace с подробной информацией о персональном предложении.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-2.webp" size="md" alt="Подробная информация о персональном предложении Azure Marketplace" border />

<br />

3. После принятия предложения вы попадете на экран **Private Offer Management**. Azure может потребоваться некоторое время, чтобы подготовить предложение к покупке.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-3.webp" size="md" alt="Страница Private Offer Management в Azure Marketplace" border />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-4.webp" size="md" alt="Загрузка страницы Private Offer Management в Azure Marketplace" border />

<br />

4. Через несколько минут обновите страницу. Предложение должно стать доступным для **Purchase**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-5.webp" size="md" alt="Страница Private Offer Management в Azure Marketplace, покупка доступна" border />

<br />

5. Нажмите **Purchase** — откроется выдвижная панель. Выполните следующие действия:

<br />

* Выберите подписку и группу ресурсов
* Укажите имя для SaaS-подписки
* Выберите тарифный план, для которого у вас есть персональное предложение. Сумма будет указана только для того срока, на который создано персональное предложение (например, 1 год). Для остальных вариантов срока сумма будет $0.
* Выберите, нужен ли вам регулярный биллинг. Если регулярный биллинг не выбран, контракт завершится в конце расчетного периода, а ресурсы будут выведены из эксплуатации.
* Нажмите **Review + subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-6.webp" size="md" alt="Форма подписки Azure Marketplace" border />

<br />

6. На следующем экране проверьте все данные и нажмите **Subscribe**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-7.webp" size="md" alt="Подтверждение подписки Azure Marketplace" border />

<br />

7. На следующем экране вы увидите **Your SaaS subscription in progress**.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-8.webp" size="md" alt="Страница отправки подписки Azure Marketplace" border />

<br />

8. Когда все будет готово, вы сможете нажать **Configure account now**. Обратите внимание: это критически важный шаг, который связывает подписку Azure с организацией ClickHouse Cloud для вашей учетной записи. Без этого шага ваша подписка через Marketplace не будет завершена.

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-committed-9.webp" size="md" alt="Кнопка Configure account now в Azure Marketplace" border />

<br />

9. Вы будете перенаправлены на страницу регистрации или входа в ClickHouse Cloud. Вы можете либо создать новую учетную запись, либо войти в существующую. После входа будет создана новая организация, готовая к использованию и оплате через Azure Marketplace.

10. Прежде чем продолжить, вам нужно будет ответить на несколько вопросов: указать адрес и сведения о компании.

<br />

<Image img="/images/cloud/manage/billing/marketplace/aws-marketplace-payg-8.webp" size="md" alt="Форма сведений для регистрации в ClickHouse Cloud" border />

<br />

11. После нажатия **Complete sign up** вы попадете в свою организацию в ClickHouse Cloud, где сможете открыть экран биллинга, чтобы убедиться, что оплата идет через Azure Marketplace, и создавать сервисы.

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-11.webp" size="sm" alt="Форма сведений для регистрации в ClickHouse Cloud" border />

<br />

<br />

<Image img="/images/cloud/manage/billing/marketplace/azure-marketplace-payg-12.webp" size="md" alt="Форма сведений для регистрации в ClickHouse Cloud" border />

<br />

Если у вас возникнут какие-либо проблемы, свяжитесь с [нашей службой поддержки](https://clickhouse.com/support/program).
