<div id="console-audit-log-events">
  ## События журнала аудита консоли
</div>

Различные типы событий, регистрируемых для организации, сгруппированы в 3 категории: **Организация**, **Сервис** и **Пользователь**. Подробнее о журнале аудита, а также о том, как его экспортировать или добавить API-интеграцию, см. в документации [журнал аудита консоли](/ru/products/cloud/guides/security/audit-logging/console-audit-log) в разделе Guides выше.

В журнал аудита записываются следующие события.

<div id="organization">
  ### Организация
</div>

* Организация создана
* Организация удалена
* Название организации изменено

<div id="service">
  ### Сервис
</div>

* Сервис создан
* Сервис удалён
* Сервис остановлен
* Сервис запущен
* Изменено имя сервиса
* Изменён IP Access List сервиса
* Пароль сервиса сброшен

<div id="user">
  ### Пользователь
</div>

* Изменена роль пользователя
* Пользователь удалён из организации
* Пользователь приглашён в организацию
* Пользователь присоединился к организации
* Приглашение пользователя удалено
* Пользователь покинул организацию
