  ## Дополнительные подключения к внешним источникам данных
</div>

<div id="clickpipes-for-data-ingestion">
  ### ClickPipes для ингестии данных
</div>

ClickPipes позволяют клиентам легко интегрировать данные из стриминговых источников. Дополнительные сведения см. в разделе [ClickPipes](/ru/integrations/clickpipes/home) нашей документации по интеграциям.

<div id="table-functions-as-external-data-sources">
  ### Табличные функции как внешние источники данных
</div>

ClickHouse поддерживает ряд табличных функций для работы с внешними источниками данных. Подробнее см. раздел [табличные функции](/ru/reference/functions/table-functions/index) в справочнике по SQL.
