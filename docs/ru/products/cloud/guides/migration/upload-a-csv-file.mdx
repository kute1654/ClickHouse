ClickHouse Cloud предоставляет простой способ импортировать файлы и поддерживает
следующие форматы:

| Формат                          |
| ------------------------------- |
| `CSV`                           |
| `CSVWithNamesAndTypes`          |
| `CSVWithNames`                  |
| `JSONEachRow`                   |
| `TabSeparated`                  |
| `TabSeparatedWithNames`         |
| `TabSeparatedWithNamesAndTypes` |

<Steps>
  <Step>
    ## Загрузка файла

    На главной странице Cloud выберите свой сервис, как показано ниже:

    <Image img="/images/cloud/migrate/csv_01.webp" alt="upload_file_02" />

    Если сервис бездействует, его нужно вывести из спящего режима.

    Выберите `Data sources` на левой панели, как показано ниже:

    <Image img="/images/cloud/migrate/csv_02.webp" alt="upload_file_03" />

    Затем выберите `Upload a file` в правой части страницы источников данных:

    <Image img="/images/cloud/migrate/csv_03.webp" alt="upload_file_04" />

    Появится диалоговое окно выбора файла, в котором вы сможете выбрать файл, который хотите
    использовать для вставки данных в таблицу вашего сервиса Cloud.

    <Image img="/images/cloud/migrate/csv_04.webp" alt="upload_file_05" />
  </Step>

  <Step>
    ## Настройка таблицы

    После загрузки файла вы сможете настроить таблицу, в которую хотите
    вставить данные. Будет показан предварительный просмотр таблицы с первыми тремя строками.

    <Image img="/images/cloud/migrate/csv_08.webp" alt="upload_file_08" />

    Теперь вы можете выбрать целевую таблицу. Варианты:

    * новая таблица
    * существующая таблица

    <br />

    Вы можете указать, в какую базу данных загрузить данные, а в случае
    новой таблицы — имя создаваемой таблицы. Вы также сможете выбрать ключ сортировки:

    <Image img="/images/cloud/migrate/csv_05.webp" alt="upload_file_05" />

    Столбцы, считанные из файла, отображаются как `Source field`, и для каждого поля
    можно изменить:

    * автоматически определённый тип
    * значение по умолчанию
    * должен ли столбец быть [Nullable](/ru/reference/data-types/nullable)

    <Image img="/images/cloud/migrate/csv_06.webp" alt="upload_file_06" />

    <Info>
      **Исключение полей**

      Вы также можете удалить поле, если не хотите включать его в импорт.
    </Info>

    Вы можете указать тип движка таблицы, который хотите использовать:

    * `MergeTree`
    * `ReplacingMergeTree`
    * `SummingMergeTree`
    * `Null`

    <br />

    Вы можете указать выражение ключа партиционирования и выражение первичного
    ключа.

    <Image img="/images/cloud/migrate/csv_07.webp" alt="upload_file_07" />

    Нажмите `Import to ClickHouse` (показано выше), чтобы импортировать данные. Импорт данных будет поставлен в очередь, что
    обозначается значком статуса `queued` в столбце `Status`, как показано ниже. Вы также можете нажать
    `Open as query` (показано выше), чтобы открыть запрос на вставку в SQL-консоли. Запрос выполнит вставку
    из файла, который был загружен в S3 бакет с использованием табличной функции `URL`.

    <Image img="/images/cloud/migrate/csv_09.webp" alt="upload_file_09" />

    Если задача завершится ошибкой, вы увидите значок статуса `failed` в столбце `Status` на вкладке
    `Data upload history`. Вы можете нажать `View Details`, чтобы получить более подробную информацию
    о причине сбоя загрузки. Возможно, вам потребуется изменить конфигурацию таблицы или очистить
    данные на основе сообщения об ошибке, полученного при неудачной вставке.

    <Image img="/images/cloud/migrate/csv_10.webp" alt="upload_file_11" />
  </Step>
</Steps>
