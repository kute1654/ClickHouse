MCP-сервер ClickHouse — это **слой доступа по открытому стандарту** для [Agentic Data Stack](/ru/products/agentic-data-stack/overview): он предоставляет интерфейсу чата доступ к ClickHouse через Model Context Protocol, чтобы агент мог изучать ваши данные и выполнять запросы к ним в режиме только для чтения.

В составе стека сервер с открытым исходным кодом работает рядом с ClickHouse и уже предварительно настроен в LibreChat. Он предоставляет агенту три инструмента:

* **Список баз данных** — перечислить базы данных в подключенном экземпляре ClickHouse.
* **Список таблиц** — просмотреть таблицы и их схемы в базе данных.
* **Выполнение SELECT-запросов** — выполнять запросы только для чтения и возвращать результаты.

Чтобы запустить его как часть стека, см. [руководство по настройке Docker](/ru/products/agentic-data-stack/docker-setup). Чтобы использовать MCP-сервер ClickHouse с другими клиентами и агентными фреймворками или ознакомиться с его исходным кодом и конфигурацией:

* [Каталог интеграций MCP](/ru/guides/use-cases/ai-ml/MCP): руководства по клиентам и библиотекам для агентов.
* [Репозиторий MCP](https://github.com/ClickHouse/mcp-clickhouse): исходный код и конфигурация.

<Info>
  **Предпочитаете управляемый MCP-сервер?** ClickHouse Cloud предлагает полностью управляемый [Remote MCP server](/ru/products/cloud/features/ai-ml/remote-mcp) — без установки, с OAuth-аутентификацией и более широким набором инструментов, который дополняет перечисленные выше инструменты только для чтения операциями управления (сервисы, биллинг, ClickPipes, резервные копии). См. [сравнение с сервером с открытым исходным кодом](/ru/products/cloud/features/ai-ml/remote-mcp#remote-vs-oss).
</Info>
