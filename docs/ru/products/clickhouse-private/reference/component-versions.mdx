<div id="notes">
  ## Примечания
</div>

* **Версия Helm-чарта оператора** (`1.18587.1`) отличается от **тега образа** оператора (`main-1.18587.1`). При установке через `helm install` используйте версию Helm-чарта, а при копировании образов контейнеров — тег образа.
* **FIPS-сборки** добавляют суффикс `-fips` к стандартным тегам образов как для server, так и для Keeper.
* Теги версий подставляются автоматически при сборке этой документации. Если вы видите необработанные переменные шаблона (например, `25.12.1.1497`), значит, этап сборки не выполнялся.
