title: Необходимое условие для Cloud

Чтобы успешно выполнить это руководство, вам потребуется следующее:

* Работающий сервис ClickHouse Cloud. Если у вас его ещё нет, сначала пройдите краткое руководство [Создайте свой первый сервис ClickHouse Cloud](/ru/get-started/quickstarts/create-your-first-service-on-cloud).
