<div id="what-youll-build">
  ## Что вы создадите
</div>

В этом быстром старте вы создадите свой первый сервис ClickHouse Cloud, познакомитесь с консолью Cloud и встроенной SQL-консолью, а также загрузите демонстрационный набор данных, чтобы у вас были реальные данные для запросов.
В итоге у вас будет работающий сервис ClickHouse Cloud с загруженными данными, готовый служить основой для остальной серии быстрых стартов.

<Steps titleSize="h3">
  <Step>
    ### Зарегистрируйтесь в ClickHouse Cloud

    Перейдите на [console.clickhouse.cloud](https://console.clickhouse.cloud/). Откроется страница входа. Внизу рядом с надписью &quot;Don&#39;t have an account yet?&quot; нажмите **Register**.

    <Frame caption="Нажмите Register, чтобы создать новую учетную запись">
      <img src="/images/get-started/quickstarts/signup-page.webp" alt="Страница входа в ClickHouse Cloud со ссылкой Register" />
    </Frame>

    На странице регистрации выберите вход через Google или Microsoft либо введите рабочий адрес электронной почты. Кредитная карта не требуется — бесплатный пробный период включает $300 кредитов и 30 дней на знакомство с ClickHouse Cloud.

    <Frame caption="Зарегистрируйтесь через Google, Microsoft или с помощью рабочего адреса электронной почты">
      <img src="/images/get-started/quickstarts/enter-details.webp" alt="Страница регистрации ClickHouse Cloud" />
    </Frame>

    Если вы регистрируетесь по электронной почте, вам будет предложено задать пароль. Он должен соответствовать следующим требованиям:

    * Не менее 12 символов
    * Как минимум 3 из следующих категорий: строчные буквы, заглавные буквы, цифры, специальные символы (например, `!@#$%^&*`)

    <Frame caption="Введите пароль и примите Terms of Service">
      <img src="/images/get-started/quickstarts/enter-email.webp" alt="Регистрация в ClickHouse Cloud по электронной почте с полем пароля" />
    </Frame>

    Примите Terms of Service и Privacy Policy, затем нажмите **Sign up**. Проверьте электронную почту: вам придет ссылка для подтверждения. Нажмите **Verify email**, чтобы активировать учетную запись.

    После подтверждения войдите с новым адресом электронной почты и паролем.
  </Step>

  <Step>
    ### Создайте новый сервис

    После первого входа вы сразу попадёте на страницу настройки сервиса. Укажите следующее:

    | Параметр                     | Рекомендация                                                                                                     |
    | ---------------------------- | ---------------------------------------------------------------------------------------------------------------- |
    | **Имя сервиса**              | Задайте сервису понятное имя (например, `My first service`)                                                      |
    | **Облачный провайдер**       | Выберите AWS, GCP или Azure в зависимости от ваших предпочтений                                                  |
    | **Регион**                   | Выберите ближайший к вам регион, чтобы обеспечить минимальную задержку                                           |
    | **Память и масштабирование** | Для этого краткого руководства выберите **Mini 12GB** (1 реплика) — это самый экономичный вариант для знакомства |

    Нажмите **Create service**, чтобы начать развертывание сервиса.

    <Frame caption="Настройка облачного сервиса">
      <img src="/images/get-started/quickstarts/service-setup.webp" alt="Страница конфигурации сервиса ClickHouse Cloud" />
    </Frame>

    Вы попадёте на главную страницу сервиса, где сначала будет отображаться статус **Provisioning**. Обычно это занимает меньше минуты. После завершения статус изменится на **Running**.

    <Frame caption="Ваш сервис запущен">
      <img src="/images/get-started/quickstarts/service-home.webp" alt="Главная страница сервиса ClickHouse Cloud со статусом Running" />
    </Frame>
  </Step>

  <Step>
    ### Загрузите пример датасета

    Нажмите **Data sources** на левой панели. Вы увидите несколько вариантов загрузки данных в ваш сервис.

    <Frame caption="Страница Data sources с вариантами загрузки данных">
      <img src="/images/get-started/quickstarts/data-sources.webp" alt="Страница Data sources в ClickHouse Cloud" />
    </Frame>

    Нажмите **Add sample data**, чтобы просмотреть доступные датасеты-примеры. Вы увидите три варианта: Cell Towers, UK Property Price Paid и New York Taxi Data. Нажмите **Get started** для датасета **UK Property Price Paid**.

    <Frame caption="Выберите датасет UK Property Price Paid">
      <img src="/images/get-started/quickstarts/example-datasets.webp" alt="Выбор датасетов-примеров в ClickHouse Cloud" />
    </Frame>

    Вы увидите предварительный просмотр данных и схему таблицы. Нажмите **Import dataset**, чтобы загрузить данные в ваш сервис.

    <Frame caption="Просмотрите данные и нажмите Import dataset">
      <img src="/images/get-started/quickstarts/import-dataset.webp" alt="Предварительный просмотр импорта датасета в ClickHouse Cloud с данными о недвижимости Великобритании" />
    </Frame>

    Импорт займет одну-две минуты. После завершения вы сможете выполнять запросы к данным.
  </Step>

  <Step>
    ### Выполните запросы к данным

    Нажмите **SQL Console** на левой боковой панели, чтобы открыть встроенный редактор запросов. Здесь вы сможете писать и выполнять SQL-запросы прямо в браузере — без каких-либо внешних инструментов.

    Попробуйте несколько запросов, чтобы изучить данные о ценах на недвижимость в Великобритании.

    Найдите самый дорогой объект недвижимости, когда-либо проданный:

    ```sql
    SELECT
        town,
        district,
        county,
        price,
        date
    FROM pp_complete
    ORDER BY price DESC
    LIMIT 5;
    ```

    Найдите среднюю цену продажи по годам:

    ```sql
    SELECT
        toYear(date) AS year,
        round(avg(price)) AS avg_price,
        formatReadableQuantity(count()) AS sales
    FROM pp_complete
    GROUP BY year
    ORDER BY year DESC;
    ```

    Эти запросы обрабатывают миллионы строк за секунды — так работают столбцовое хранение и сжатие в ClickHouse.
  </Step>
</Steps>
