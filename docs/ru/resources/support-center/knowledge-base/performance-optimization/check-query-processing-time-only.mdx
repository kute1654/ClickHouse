<div id="question">
  ## Вопрос
</div>

Мой запрос возвращает много строк, но меня интересует только выполнение запроса. Как отключить вывод результата запроса и посмотреть выполнение запроса?

<div id="answer">
  ## Ответ
</div>

Добавьте `FORMAT Null` к запросу, чтобы установить выходной формат `Null`. Это предотвратит передачу данных клиенту.

Например:

```sql
SELECT
    customer_id,
    count() AS total,
    any(review_headline)
FROM amazon_reviews
GROUP BY customer_id
ORDER BY total DESC
FORMAT Null
```

В ответе будет указано количество обработанных строк и затраченное время, но будет возвращено 0 строк:

```response
0 rows in set. Elapsed: 25.288 sec. Processed 222.04 million rows, 13.50 GB (8.78 million rows/s., 533.77 MB/s.)
```
