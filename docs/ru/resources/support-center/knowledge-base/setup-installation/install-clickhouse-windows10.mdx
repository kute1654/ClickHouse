<div id="how-to-install-and-test-clickhouse-on-microsoft-windows">
  ## Как установить и протестировать ClickHouse в Microsoft Windows
</div>

При установке ClickHouse в Windows 10 могут возникать ошибки при вставке данных, например:

```
DB::Exception: std::__1::__fs::filesystem::filesystem_error: filesystem error: in rename: Permission denied ["./store/711/71144174-d098-4056-8976-6ad1204205ec/tmp_insert_all_1_1_0/"] ["./store/711/71144174-d098-4056-8976-6ad1204205ec/all_1_1_0/"]. Stack trace:
```

В Windows 10 необходимо обновить WSL до версии WSL 2.

* Откройте PowerShell: щёлкните правой кнопкой мыши по значку PowerShell и выберите &quot;Запуск от имени администратора&quot;.

* Следуйте инструкциям Microsoft по обновлению до WSL 2:
  https://learn.microsoft.com/en-us/windows/wsl/install

* После обновления откройте WSL из PowerShell.

```
wsl
```

* Для тестирования следуйте этим инструкциям; у вас должен получиться похожий вывод:
  Поскольку это тестирование, я вошёл в систему как root, чтобы избежать проблем с правами доступа:

```
sudo -i
```

* Создайте каталог ClickHouse:

```
root@marspc2:~# mkdir /clickhouse
```
