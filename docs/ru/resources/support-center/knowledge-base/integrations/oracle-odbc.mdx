Если вы используете Oracle как источник внешних словарей ClickHouse через драйвер Oracle ODBC, необходимо задать корректное значение переменной окружения `NLS_LANG` в `/etc/default/clickhouse`. Дополнительные сведения см. в [FAQ по Oracle NLS&#95;LANG](https://www.oracle.com/technetwork/products/globalization/nls-lang-099431.html).

**Пример**

```sql
NLS_LANG=RUSSIAN_RUSSIA.UTF8
```
