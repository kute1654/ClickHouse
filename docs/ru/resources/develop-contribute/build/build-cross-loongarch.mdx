В ClickHouse есть экспериментальная поддержка LoongArch64

<div id="build-clickhouse">
  ## Сборка ClickHouse
</div>

Требуемая для сборки версия LLVM должна быть не ниже 21.1.0.

```bash
cd ClickHouse
mkdir build-loongarch64
cmake . -Bbuild-loongarch64 -DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-loongarch64.cmake
ninja -C build-loongarch64
```

Полученный бинарный файл будет работать только в Linux на процессорах с архитектурой LoongArch64.
