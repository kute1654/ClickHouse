Для сборки ClickHouse для Aarch64 на машине с архитектурой Aarch64 никаких специальных действий не требуется.

Чтобы выполнить кросс-компиляцию ClickHouse для AArch64 на x86-машине с Linux, передайте `cmake` следующий флаг: `-DCMAKE_TOOLCHAIN_FILE=cmake/linux/toolchain-aarch64.cmake`
