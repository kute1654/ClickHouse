<div id="file">
  ## file
</div>

Считывает файл как строку и загружает данные в указанный столбец. Содержимое файла не интерпретируется.

См. также табличную функцию [file](/ru/reference/functions/table-functions/file).

**Синтаксис**

```sql
file(path[, default])
```

**Аргументы**

* `path` — Путь к файлу, заданный относительно [user&#95;files&#95;path](/ru/reference/settings/server-settings/settings#user_files_path). Поддерживаются подстановочные шаблоны `*`, `**`, `?`, `{abc,def}` и `{N..M}`, где `N`, `M` — числа, а `'abc'`, `'def'` — строки.
* `default` — Значение, которое возвращается, если файл не существует или к нему нет доступа. Поддерживаемые типы данных: [String](/ru/reference/data-types/string) и [NULL](/ru/reference/settings/formats#input_format_null_as_default).

**Пример**

Вставка данных из файлов a.txt и b.txt в таблицу в виде строк:

```sql
INSERT INTO table SELECT file('a.txt'), file('b.txt');
```
