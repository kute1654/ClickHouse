`timeSeriesMetrics(db_name.time_series_table)` — Возвращает таблицу [метрик](/ru/reference/engines/table-engines/integrations/time-series#metrics-table),
которую использует таблица `db_name.time_series_table` с движком таблицы [TimeSeries](/ru/reference/engines/table-engines/integrations/time-series):

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS metrics_table
```

Функция также работает, если таблица *metrics* — внутренняя:

```sql
CREATE TABLE db_name.time_series_table ENGINE=TimeSeries METRICS INNER UUID '01234567-89ab-cdef-0123-456789abcdef'
```

Следующие запросы равнозначны:

```sql
SELECT * FROM timeSeriesMetrics(db_name.time_series_table);
SELECT * FROM timeSeriesMetrics('db_name.time_series_table');
SELECT * FROM timeSeriesMetrics('db_name', 'time_series_table');
```
