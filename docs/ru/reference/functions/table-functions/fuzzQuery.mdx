Вносит случайные изменения в указанную строку запроса.

<div id="syntax">
  ## Синтаксис
</div>

```sql
fuzzQuery(query[, max_query_length[, random_seed]])
```

<div id="arguments">
  ## Аргументы
</div>

| Аргумент           | Описание                                                                                       |
| ------------------ | ---------------------------------------------------------------------------------------------- |
| `query`            | (String) - Исходный запрос для фаззинга.                                                       |
| `max_query_length` | (UInt64) - Максимальная длина, которой может достичь запрос в процессе фаззинга.               |
| `random_seed`      | (UInt64) - Начальное значение генератора случайных чисел для получения стабильных результатов. |

<div id="returned_value">
  ## Возвращаемое значение
</div>

Табличный объект с одним столбцом, содержащим изменённые строки запросов.

<div id="usage-example">
  ## Пример использования
</div>

```sql
SELECT * FROM fuzzQuery('SELECT materialize(\'a\' AS key) GROUP BY key') LIMIT 2;
```

```response
   ┌─query──────────────────────────────────────────────────────────┐
1. │ SELECT 'a' AS key GROUP BY key                                 │
2. │ EXPLAIN PIPELINE compact = true SELECT 'a' AS key GROUP BY key │
   └────────────────────────────────────────────────────────────────┘
```
