Создаёт временную таблицу [Merge](/ru/reference/engines/table-engines/special/merge).
Схема таблицы выводится из базовых таблиц путём объединения их столбцов и определения общих типов.
Доступны те же виртуальные столбцы, что и у движка таблицы [Merge](/ru/reference/engines/table-engines/special/merge).

<div id="syntax">
  ## Синтаксис
</div>

```sql
merge(['db_name',] 'tables_regexp')
```

<div id="arguments">
  ## Аргументы
</div>

| Аргумент        | Описание                                                                                                                                                                                                                                                                                                               |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `db_name`       | Возможные значения (необязательно, по умолчанию — `currentDatabase()`):<br />    - имя базы данных,<br />    - константное выражение, возвращающее строку с именем базы данных, например `currentDatabase()`,<br />    - `REGEXP(expression)`, где `expression` — регулярное выражение для сопоставления с именами БД. |
| `tables_regexp` | Регулярное выражение для сопоставления с именами таблиц в указанной БД или БД.                                                                                                                                                                                                                                         |
