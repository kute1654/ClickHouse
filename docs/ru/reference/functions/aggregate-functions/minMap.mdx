<div id="minMappedArrays">
  ## minMappedArrays
</div>

Добавленный в: v20.5.0

Вычисляет минимальное значение из массива `value` по ключам, указанным в массиве `key`.

<Note>
  * Передача кортежа ключей и массивов значений эквивалентна передаче массива ключей и массива значений.
  * Количество элементов в `key` и `value` должно совпадать для каждой суммируемой строки.
</Note>

**Синтаксис**

```sql
minMappedArrays(key, value)
minMappedArrays(Tuple(key, value))
```

**Аргументы**

* `key` — Массив ключей. [`Array(T)`](/ru/reference/data-types/array)
* `value` — Массив значений. [`Array(T)`](/ru/reference/data-types/array)

**Возвращаемое значение**

Возвращает кортеж из двух массивов: ключи в отсортированном порядке и значения, вычисленные для соответствующих ключей. [`Tuple(Array(T), Array(T))`](/ru/reference/data-types/tuple)

**Примеры**

**Пример использования**

```sql title=Query
SELECT minMappedArrays(a, b)
FROM VALUES('a Array(Int32), b Array(Int64)', ([1, 2], [2, 2]), ([2, 3], [1, 1]));
```

```response title=Response
┌─minMappedArrays(a, b)───────────┐
│ ([1, 2, 3], [2, 1, 1])          │
└─────────────────────────────────┘
```
