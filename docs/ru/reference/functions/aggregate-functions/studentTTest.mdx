<div id="studentTTest">
  ## studentTTest
</div>

Добавленный в: v21.1.0

Применяет t-критерий Стьюдента к выборкам из двух генеральных совокупностей.

Значения обеих выборок находятся в столбце `sample_data`. Если `sample_index` равен 0, то значение в этой строке относится к выборке из первой генеральной совокупности. В противном случае оно относится к выборке из второй генеральной совокупности.
Нулевая гипотеза состоит в том, что средние значения генеральных совокупностей равны. Предполагается нормальное распределение с равными дисперсиями.

**См. также**

* [t-критерий Стьюдента](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [функция welchTTest](/ru/reference/functions/aggregate-functions/welchTTest)

**Синтаксис**

```sql
studentTTest([confidence_level])(sample_data, sample_index)
```

**Параметры**

* `confidence_level` — Необязательный. Уровень доверия для вычисления доверительных интервалов. [`Float`](/ru/reference/data-types/float)

**Аргументы**

* `sample_data` — Выборочные данные. [`Integer`](/ru/reference/data-types/int-uint) или [`Float`](/ru/reference/data-types/float) или [`Decimal`](/ru/reference/data-types/decimal)
* `sample_index` — Индекс выборки. [`Integer`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает кортеж из двух или четырёх элементов (если указан необязательный параметр `confidence_level`): вычисленная t-статистика, вычисленное p-значение, [вычисленная нижняя граница доверительного интервала], [вычисленная верхняя граница доверительного интервала]. [`Tuple(Float64, Float64)`](/ru/reference/data-types/tuple) или [`Tuple(Float64, Float64, Float64, Float64)`](/ru/reference/data-types/tuple)

**Примеры**

**Базовое использование**

```sql title=Query
SELECT studentTTest(sample_data, sample_index) FROM student_ttest;
```
