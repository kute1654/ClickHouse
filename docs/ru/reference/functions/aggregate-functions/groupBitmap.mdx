<div id="groupBitmap">
  ## groupBitmap
</div>

Добавленный в: v20.1.0

Создаёт битмап (битовый массив) из столбца беззнаковых целых чисел, а затем возвращает число уникальных значений (мощность) в этом битмапе.
Если добавить суффикс комбинатора `-State`, функция вместо этого вернёт не количество, а сам [объект битмапа](/ru/reference/functions/regular-functions/bitmap-functions).

**Синтаксис**

```sql
groupBitmap(expr)
groupBitmapState(expr)
```

**Аргументы**

* `expr` — выражение, результат которого имеет тип `UInt*`. [`UInt*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает количество типа `UInt64` или объект битмапа при использовании `-State`. [`UInt64`](/ru/reference/data-types/int-uint)

**Примеры**

**Пример использования**

```sql title=Query
CREATE TABLE t (UserID UInt32) ENGINE = Memory;
INSERT INTO t VALUES (1), (1), (2), (3);

SELECT groupBitmap(UserID) AS num FROM t;
```

```response title=Response
┌─num─┐
│   3 │
└─────┘
```
