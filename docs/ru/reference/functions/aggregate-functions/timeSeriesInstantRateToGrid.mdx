<div id="timeSeriesInstantRateToGrid">
  ## timeSeriesInstantRateToGrid
</div>

Добавленный в: v25.6.0

Агрегатная функция, которая принимает данные временных рядов в виде пар временных меток и значений и вычисляет на их основе [PromQL-подобный irate](https://prometheus.io/docs/prometheus/latest/querying/functions/#irate) на регулярной временной сетке, задаваемой начальной временной меткой, конечной временной меткой и шагом. Для каждой точки сетки значения для вычисления `irate` берутся в пределах указанного временного окна.

<Warning>
  Эта функция экспериментальная; чтобы включить её, установите `allow_experimental_ts_to_grid_aggregate_function=true`.
</Warning>

**Синтаксис**

```sql
timeSeriesInstantRateToGrid(start_timestamp, end_timestamp, grid_step, staleness)(timestamp, value)
```

**Параметры**

* `start_timestamp` — Задает начало сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `end_timestamp` — Задает конец сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `grid_step` — Задает шаг сетки в секундах. [`UInt32`](/ru/reference/data-types/int-uint)
* `staleness` — Задает максимально допустимую давность учитываемых измерений в секундах. Окно давности представляет собой полуоткрытый интервал: открытый слева и закрытый справа. [`UInt32`](/ru/reference/data-types/int-uint)

**Аргументы**

* `timestamp` — Временная метка измерения. Может быть отдельным значением или массивом. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime) или [`Array(UInt32)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array)
* `value` — Значение временного ряда, соответствующее временной метке. Может быть отдельным значением или массивом. [`Float*`](/ru/reference/data-types/float) или [`Array(Float*)`](/ru/reference/data-types/array)

**Возвращаемое значение**

Возвращает значения irate на указанной сетке. Возвращаемый массив содержит по одному значению для каждой точки временной сетки. Значение равно NULL, если в пределах окна недостаточно измерений, чтобы вычислить значение мгновенной скорости для конкретной точки сетки. [`Array(Nullable(Float64))`](/ru/reference/data-types/array)

**Примеры**

**Базовое использование с отдельными парами «временная метка — значение»**

```sql title=Query
WITH
    -- ПРИМЕЧАНИЕ: разрыв между 140 и 190 показывает, как заполняются значения для ts = 150, 165, 180 в соответствии с параметром window
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- массив значений, соответствующих временным меткам выше
    90 AS start_ts,       -- начало сетки временных меток
    90 + 120 AS end_ts,   -- конец сетки временных меток
    15 AS step_seconds,   -- шаг сетки временных меток
    45 AS window_seconds  -- окно "устаревания"
SELECT timeSeriesInstantRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- Этот подзапрос преобразует массивы временных меток и значений в строки `timestamp`, `value`
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesInstantRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,0,0.2,0.1,0.1,NULL,NULL,0.3]                                                      │
└──────────────────────────────────────────────────────────────────────────────────────────────┘
```

**Использование аргументов типа Array**

```sql title=Query
-- можно передать несколько образцов временных меток и значений в виде массивов одинакового размера
WITH
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values,
    90 AS start_ts,
    90 + 120 AS end_ts,
    15 AS step_seconds,
    45 AS window_seconds
SELECT timeSeriesInstantRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values);
```

```response title=Response
┌─timeSeriesInstantRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values)─┐
│ [NULL,NULL,0,0.2,0.1,0.1,NULL,NULL,0.3]                                                        │
└────────────────────────────────────────────────────────────────────────────────────────────────┘
```
