<div id="varPopStable">
  ## varPopStable
</div>

Добавленный в: v1.1.0

Возвращает дисперсию по генеральной совокупности.
В отличие от [`varPop`](/ru/reference/functions/aggregate-functions/varPop), эта функция использует [численно устойчивый](https://en.wikipedia.org/wiki/Numerical_stability) алгоритм.
Она работает медленнее, но обеспечивает меньшую погрешность вычислений.

**Синтаксис**

```sql
varPopStable(x)
```

**Аргументы**

* `x` — Набор значений, для которого вычисляется дисперсия генеральной совокупности. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal)

**Возвращаемое значение**

Возвращает дисперсию генеральной совокупности для `x`. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Вычисление устойчивой дисперсии генеральной совокупности**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT
    varPopStable(x) AS var_pop_stable
FROM test_data;
```

```response title=Response
┌─var_pop_stable─┐
│           14.4 │
└────────────────┘
```
