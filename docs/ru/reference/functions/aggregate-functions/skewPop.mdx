<div id="skewPop">
  ## skewPop
</div>

Добавленный в: v20.1.0

Вычисляет [коэффициент асимметрии](https://en.wikipedia.org/wiki/Skewness) последовательности.

**Синтаксис**

```sql
skewPop(expr)
```

**Аргументы**

* `expr` — Выражение, возвращающее число. [`Expression`](/ru/reference/data-types/special-data-types/expression)

**Возвращаемое значение**

Возвращает коэффициент асимметрии заданного распределения. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Симметричное распределение**

```sql title=Query
SELECT skewPop(number) FROM numbers(100);
```

```response title=Response
┌─skewPop(number)─┐
│               0 │
└─────────────────┘
```

**Правосторонне скошенное распределение**

```sql title=Query
SELECT skewPop(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌─────────skewPop(x)─┐
│ 0.6735701055423582 │
└────────────────────┘
```
