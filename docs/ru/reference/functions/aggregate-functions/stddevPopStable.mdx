<div id="stddevPopStable">
  ## stddevPopStable
</div>

Добавленный в: v1.1.0

Результат равен квадратному корню из [varPop](/ru/reference/functions/aggregate-functions/varPop). В отличие от [stddevPop](/ru/reference/functions/aggregate-functions/stddevPop), эта функция использует численно устойчивый алгоритм. Она работает медленнее, но дает меньшую вычислительную погрешность.

**Синтаксис**

```sql
stddevPopStable(x)
```

**Аргументы**

* `x` — Набор значений, для которых вычисляется стандартное отклонение. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal)

**Возвращаемое значение**

Возвращает квадратный корень из дисперсии `x`. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Базовое использование**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    population Float64,
)
ENGINE = Log;

INSERT INTO test_data SELECT randUniform(5.5, 10) FROM numbers(1000000);

SELECT
    stddevPopStable(population) AS stddev
FROM test_data;
```

```response title=Response
┌─────────────stddev─┐
│ 1.2999977786592576 │
└────────────────────┘
```
