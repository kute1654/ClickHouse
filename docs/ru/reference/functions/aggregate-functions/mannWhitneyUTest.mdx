<div id="mannWhitneyUTest">
  ## mannWhitneyUTest
</div>

Добавленный в: v21.1.0

Применяет ранговый критерий Манна — Уитни к выборкам из двух генеральных совокупностей.

Значения обеих выборок находятся в столбце `sample_data`.
Если `sample_index` равен 0, то значение в этой строке относится к выборке из первой генеральной совокупности.
В противном случае оно относится к выборке из второй генеральной совокупности.
Нулевая гипотеза заключается в том, что две генеральные совокупности стохастически равны.
Также можно проверять односторонние гипотезы.
Этот критерий не предполагает, что данные имеют нормальное распределение.

**Синтаксис**

```sql
mannWhitneyUTest[(alternative[, continuity_correction])](sample_data, sample_index)
```

**Параметры**

* `alternative` — Необязательный. Альтернативная гипотеза. &#39;two-sided&#39; (по умолчанию): две совокупности стохастически не равны. &#39;greater&#39;: значения в первой выборке стохастически больше, чем во второй. &#39;less&#39;: значения в первой выборке стохастически меньше, чем во второй. [`String`](/ru/reference/data-types/string)
* `continuity_correction` — Необязательный. Если значение не равно 0, применяется поправка на непрерывность в нормальном приближении для p-значения. Значение по умолчанию — 1. [`UInt64`](/ru/reference/data-types/int-uint)

**Аргументы**

* `sample_data` — Данные выборки. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal)
* `sample_index` — Индекс выборки. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает кортеж из двух элементов: вычисленные U-статистика и p-значение. [`Tuple(Float64, Float64)`](/ru/reference/data-types/tuple)

**Примеры**

**Пример теста Манна — Уитни**

```sql title=Query
CREATE TABLE mww_ttest (sample_data Float64, sample_index UInt8) ENGINE = Memory;
INSERT INTO mww_ttest VALUES (10, 0), (11, 0), (12, 0), (1, 1), (2, 1), (3, 1);

SELECT mannWhitneyUTest('greater')(sample_data, sample_index) FROM mww_ttest;
```

```response title=Response
┌─mannWhitneyUTest('greater')(sample_data, sample_index)─┐
│ (9,0.04042779918503192)                                │
└────────────────────────────────────────────────────────┘
```
