<div id="groupBitXor">
  ## groupBitXor
</div>

Добавленный в: v1.1.0

Выполняет побитовую операцию XOR для серии чисел.

**Синтаксис**

```sql
groupBitXor(expr)
```

**Псевдонимы**: `BIT_XOR`

**Аргументы**

* `expr` — выражение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает значение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Примеры**

**Пример побитовой операции XOR**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Тестовые данные:
-- двоичное     десятичное
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitXor(num) FROM t;
```

```response title=Response
-- Результат:
-- двоичное  десятичное
-- 01101000 = 104

┌─groupBitXor(num)─┐
│              104 │
└──────────────────┘
```
