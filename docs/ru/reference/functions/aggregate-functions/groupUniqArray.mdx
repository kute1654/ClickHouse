<div id="groupUniqArray">
  ## groupUniqArray
</div>

Добавленный в: v1.1.0

Создаёт массив из различных значений аргумента.
Объём памяти, потребляемой этой функцией, такой же, как у функции [`uniqExact`](/ru/reference/functions/aggregate-functions/uniqExact).

**Синтаксис**

```sql
groupUniqArray(x)
groupUniqArray(max_size)(x)
```

**Параметры**

* `max_size` — Ограничивает размер результирующего массива значением `max_size` элементов. `groupUniqArray(1)(x)` эквивалентно `[any(x)]`. [`UInt64`](/ru/reference/data-types/int-uint)

**Аргументы**

* `x` — Выражение. [`Any`](/ru/reference/data-types/index)

**Возвращаемое значение**

Возвращает массив уникальных значений. [`Array`](/ru/reference/data-types/array)

**Примеры**

**Пример использования**

```sql title=Query
CREATE TABLE t (x UInt8) ENGINE = Memory;
INSERT INTO t VALUES (1), (2), (1), (3), (2), (4);

SELECT groupUniqArray(x) FROM t;
```

```response title=Response
┌─groupUniqArray(x)─┐
│ [1,2,3,4]         │
└───────────────────┘
```

**С параметром max&#95;size**

```sql title=Query
SELECT groupUniqArray(2)(x) FROM t;
```

```response title=Response
┌─groupUniqArray(2)(x)─┐
│ [1,2]                │
└──────────────────────┘
```
