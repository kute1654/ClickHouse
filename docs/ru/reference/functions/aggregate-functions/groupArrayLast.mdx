<div id="groupArrayLast">
  ## groupArrayLast
</div>

Добавленный в: v23.1.0

Создает массив из последних значений аргументов.
Например, `groupArrayLast(1)(x)` эквивалентно `[anyLast(x)]`.
В некоторых случаях на порядок выполнения всё же можно полагаться.
Это относится к случаям, когда SELECT выполняется из подзапроса с ORDER BY, если результат подзапроса достаточно мал.

**Синтаксис**

```sql
groupArrayLast(max_size)(x)
```

**Параметры**

* `max_size` — Максимальный размер результирующего массива. [`UInt64`](/ru/reference/data-types/int-uint)

**Аргументы**

* `max_size` — Максимальный размер результирующего массива. [`UInt64`](/ru/reference/data-types/int-uint)
* `x` — Аргумент (имя столбца или выражение). [`Any`](/ru/reference/data-types/index)

**Возвращаемое значение**

Возвращает массив последних значений аргументов. [`Array(T)`](/ru/reference/data-types/array)

**Примеры**

**Пример использования**

```sql title=Query
SELECT groupArrayLast(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [9,10]  │
└─────────┘
```

**Сравнение с groupArray**

```sql title=Query
-- Compare with groupArray (first values)
SELECT groupArray(2)(number+1) numbers FROM numbers(10);
```

```response title=Response
┌─numbers─┐
│ [1,2]   │
└─────────┘
```
