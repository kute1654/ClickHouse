## varPop {#varPop}

Добавленный в: v1.1.0

Вычисляет дисперсию генеральной совокупности.

Дисперсия генеральной совокупности вычисляется по формуле:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

Где:
- $x$ — каждое значение в генеральной совокупности
- $\bar{x}$ — среднее значение генеральной совокупности
- $n$ — размер генеральной совокупности

<Note>
Эта функция использует численно нестабильный алгоритм. Если в вычислениях требуется [численная устойчивость](https://en.wikipedia.org/wiki/Numerical_stability), используйте функцию [`varPopStable`](/reference/functions/aggregate-functions/varPopStable). Она работает медленнее, но обеспечивает меньшую вычислительную погрешность.
</Note>
    

**Синтаксис**

```sql
varPop(x)
```

**Псевдонимы**: `VAR_POP`

**Аргументы**

- `x` — Совокупность значений, для которой вычисляется дисперсия генеральной совокупности. [`(U)Int*`](/reference/data-types/int-uint) или [`Float*`](/reference/data-types/float) или [`Decimal*`](/reference/data-types/decimal)

**Возвращаемое значение**

Возвращает дисперсию генеральной совокупности для `x`. [`Float64`](/reference/data-types/float)

**Примеры**

**Вычисление дисперсии генеральной совокупности**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
