<div id="max">
  ## max
</div>

Добавленный в: v1.1.0

Агрегатная функция, вычисляющая максимальное значение в группе значений.

**Синтаксис**

```sql
max(column)
```

**Аргументы**

* `column` — Имя столбца или выражение. [`Any`](/ru/reference/data-types/index)

**Возвращаемое значение**

Максимальное значение в группе; тип совпадает с типом входного значения. [`Any`](/ru/reference/data-types/index)

**Примеры**

**Простой пример max**

```sql title=Query
CREATE TABLE employees (name String, salary UInt32) ENGINE = Memory;
INSERT INTO employees VALUES ('Alice', 3000), ('Bob', 4000), ('Charlie', 3500);

SELECT max(salary) FROM employees;
```

```response title=Response
┌─max(salary)─┐
│        4000 │
└─────────────┘
```

**Максимум с GROUP BY**

```sql title=Query
CREATE TABLE sales (department String, revenue UInt32) ENGINE = Memory;
INSERT INTO sales VALUES ('Engineering', 100000), ('Engineering', 120000), ('Marketing', 80000), ('Marketing', 90000);

SELECT department, max(revenue) FROM sales GROUP BY department ORDER BY department;
```

```response title=Response
┌─department──┬─max(revenue)─┐
│ Engineering │       120000 │
│ Marketing   │        90000 │
└─────────────┴──────────────┘
```

**Примечание о максимуме без агрегации**

```sql title=Query
-- If you need non-aggregate function to choose a maximum of two values, see greatest():
SELECT greatest(a, b) FROM table;
```
