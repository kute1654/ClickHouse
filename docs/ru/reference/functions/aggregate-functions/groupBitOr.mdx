<div id="groupBitOr">
  ## groupBitOr
</div>

Добавленный в: v1.1.0

Применяет операцию побитового ИЛИ к последовательности чисел.

**Синтаксис**

```sql
groupBitOr(expr)
```

**Псевдонимы**: `BIT_OR`

**Аргументы**

* `expr` — выражение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает значение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Примеры**

**Пример побитового ИЛИ**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Тестовые данные:
-- двоичное     десятичное
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Результат:
-- двоичное  десятичное
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
