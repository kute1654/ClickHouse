<div id="timeSeriesRateToGrid">
  ## timeSeriesRateToGrid
</div>

Добавленный в: v25.6.0

Агрегатная функция, которая принимает данные временных рядов в виде пар временных меток и значений и вычисляет [rate в стиле PromQL](https://prometheus.io/docs/prometheus/latest/querying/functions/#rate) на основе этих данных на регулярной временной сетке, задаваемой начальной временной меткой, конечной временной меткой и шагом. Для каждой точки сетки выборки для вычисления `rate` берутся в пределах указанного временного окна.

<Warning>
  Эта функция экспериментальная; чтобы включить её, задайте `allow_experimental_ts_to_grid_aggregate_function=true`.
</Warning>

**Синтаксис**

```sql
timeSeriesRateToGrid(start_timestamp, end_timestamp, grid_step, staleness)(timestamp, value)
```

**Параметры**

* `start_timestamp` — Задаёт начало сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `end_timestamp` — Задаёт конец сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `grid_step` — Задаёт шаг сетки в секундах. [`UInt32`](/ru/reference/data-types/int-uint)
* `staleness` — Задаёт максимально допустимую давность рассматриваемых отсчётов в секундах. Окно устаревания представляет собой интервал, открытый слева и закрытый справа. [`UInt32`](/ru/reference/data-types/int-uint)

**Аргументы**

* `timestamp` — Временная метка отсчёта. Может быть отдельным значением или массивом. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime) или [`Array(UInt32)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array)
* `value` — Значение временного ряда, соответствующее временной метке. Может быть отдельным значением или массивом. [`Float*`](/ru/reference/data-types/float) или [`Array(Float*)`](/ru/reference/data-types/array)

**Возвращаемое значение**

Возвращает значения скорости на указанной сетке. Возвращаемый массив содержит по одному значению для каждой точки временной сетки. Значение равно NULL, если в пределах окна недостаточно отсчётов, чтобы вычислить значение скорости для конкретной точки сетки. [`Array(Nullable(Float64))`](/ru/reference/data-types/array)

**Примеры**

**Базовое использование с отдельными парами временная метка — значение**

```sql title=Query
WITH
    -- ПРИМЕЧАНИЕ: пропуск между 140 и 190 показывает, как заполняются значения для ts = 150, 165, 180 согласно параметру window
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- массив значений, соответствующих приведённым выше временным меткам
    90 AS start_ts,       -- начало сетки временных меток
    90 + 120 AS end_ts,   -- конец сетки временных меток
    15 AS step_seconds,   -- шаг сетки временных меток
    45 AS window_seconds  -- окно устаревания ("staleness")
SELECT timeSeriesRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- Этот подзапрос преобразует массивы временных меток и значений в строки `timestamp`, `value`
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,0,0.06666667,0.1,0.083333336,NULL,NULL,0.083333336]                        │
└───────────────────────────────────────────────────────────────────────────────────────┘
```

**Использование аргументов типа Array**

```sql title=Query
WITH
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values,
    90 AS start_ts,
    90 + 120 AS end_ts,
    15 AS step_seconds,
    45 AS window_seconds
SELECT timeSeriesRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values);
```

```response title=Response
┌─timeSeriesRateToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values)─┐
│ [NULL,NULL,0,0.06666667,0.1,0.083333336,NULL,NULL,0.083333336]                          │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
