<div id="maxIntersectionsPosition">
  ## maxIntersectionsPosition
</div>

Добавленный в: v1.1.0

Агрегатная функция, вычисляющая позиции вхождений для функции [`maxIntersections`](/ru/reference/functions/aggregate-functions/maxIntersections).

**Синтаксис**

```sql
maxIntersectionsPosition(start_column, end_column)
```

**Аргументы**

* `start_column` — Числовой столбец, обозначающий начало каждого интервала. Если `start_column` равен `NULL` или 0, интервал будет пропущен. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float)
* `end_column` — Числовой столбец, обозначающий конец каждого интервала. Если `end_column` равен `NULL` или 0, интервал будет пропущен. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float)

**Возвращаемое значение**

Возвращает начальные позиции для максимального числа пересекающихся интервалов. [`Any`](/ru/reference/data-types/index)

**Примеры**

**Поиск позиции для максимального числа пересечений**

```sql title=Query
CREATE TABLE my_events (
    start UInt32,
    end UInt32
)
ENGINE = MergeTree
ORDER BY tuple();

INSERT INTO my_events VALUES
(1, 3),
(1, 6),
(2, 5),
(3, 7);

SELECT maxIntersectionsPosition(start, end) FROM my_events;
```

```response title=Response
┌─maxIntersectionsPosition(start, end)─┐
│                                    2 │
└──────────────────────────────────────┘
```
