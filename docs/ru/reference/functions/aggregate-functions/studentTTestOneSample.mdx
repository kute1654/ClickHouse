<div id="studentTTestOneSample">
  ## studentTTestOneSample
</div>

Добавленный в: v25.10.0

Применяет одновыборочный t-критерий Стьюдента, чтобы определить, отличается ли среднее значение выборки от известного среднего значения генеральной совокупности.

Предполагается нормальное распределение. Нулевая гипотеза состоит в том, что среднее значение выборки равно среднему значению генеральной совокупности.

Необязательный параметр `confidence_level` позволяет вычислить доверительный интервал.

**Примечания:**

* Требуется как минимум 2 наблюдения; в противном случае результатом будет `(nan, nan)` (а интервалы, если они запрошены, будут `nan`).
* Постоянный или почти постоянный входной набор данных также вернёт `nan` из-за нулевой (или фактически нулевой) стандартной ошибки.

**См. также**

* [t-критерий Стьюдента](https://en.wikipedia.org/wiki/Student%27s_t-test)
* [функция studentTTest](/ru/reference/functions/aggregate-functions/studentTTest)

**Синтаксис**

```sql
studentTTestOneSample([confidence_level])(sample_data, population_mean)
```

**Параметры**

* `confidence_level` — Необязательный параметр. Уровень доверия для доверительных интервалов. Float в диапазоне (0, 1). [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `sample_data` — Выборочные данные. [`Integer`](/ru/reference/data-types/int-uint) или [`Float`](/ru/reference/data-types/float) или [`Decimal`](/ru/reference/data-types/decimal)
* `population_mean` — Известное среднее генеральной совокупности, относительно которого выполняется проверка (обычно константа). [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal`](/ru/reference/data-types/decimal)

**Возвращаемое значение**

Возвращает кортеж из двух или четырех элементов (если указан `confidence_level`): вычисленная t-статистика, вычисленное p-значение (двустороннее), [вычисленная нижняя граница доверительного интервала], [вычисленная верхняя граница доверительного интервала]. Доверительные интервалы приводятся для выборочного среднего при заданном уровне доверия. [`Tuple(Float64, Float64)`](/ru/reference/data-types/tuple) или [`Tuple(Float64, Float64, Float64, Float64)`](/ru/reference/data-types/tuple)

**Примеры**

**Без доверительного интервала**

```sql title=Query
SELECT studentTTestOneSample()(value, 20.0) FROM t;
```
