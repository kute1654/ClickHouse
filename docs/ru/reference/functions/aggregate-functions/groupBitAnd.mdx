<div id="groupBitAnd">
  ## groupBitAnd
</div>

Добавленный в: v1.1.0

Вычисляет побитовое И для ряда чисел.

**Синтаксис**

```sql
groupBitAnd(expr)
```

**Псевдонимы**: `BIT_AND`

**Аргументы**

* `expr` — выражение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Возвращает значение типа `(U)Int*`. [`(U)Int*`](/ru/reference/data-types/int-uint)

**Примеры**

**Пример побитового И**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Тестовые данные:
-- двоичное     десятичное
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Результат:
-- двоичное  десятичное
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
