<div id="timeSeriesDeltaToGrid">
  ## timeSeriesDeltaToGrid
</div>

Добавленный в: v25.6.0

Агрегатная функция, которая принимает данные временных рядов в виде пар временных меток и значений и вычисляет [дельта в стиле PromQL](https://prometheus.io/docs/prometheus/latest/querying/functions/#delta) по этим данным на регулярной временной сетке, задаваемой начальной временной меткой, конечной временной меткой и шагом.
Для каждой точки сетки выборки для вычисления `delta` берутся в пределах указанного временного окна.

<Warning>
  Эта функция экспериментальная. Чтобы включить её, установите `allow_experimental_ts_to_grid_aggregate_function=true`.
</Warning>

**Синтаксис**

```sql
timeSeriesDeltaToGrid(start_timestamp, end_timestamp, grid_step, staleness)(timestamp, value)
```

**Параметры**

* `start_timestamp` — Задает начало сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `end_timestamp` — Задает конец сетки. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime)
* `grid_step` — Задает шаг сетки в секундах. [`UInt32`](/ru/reference/data-types/int-uint)
* `staleness` — Задает максимально допустимую давность учитываемых отсчетов в секундах. Окно устаревания представляет собой интервал, открытый слева и закрытый справа. [`UInt32`](/ru/reference/data-types/int-uint)

**Аргументы**

* `timestamp` — Временная метка отсчета. Может быть отдельным значением или массивом. [`UInt32`](/ru/reference/data-types/int-uint) или [`DateTime`](/ru/reference/data-types/datetime) или [`Array(UInt32)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array)
* `value` — Значение временного ряда, соответствующее временной метке. Может быть отдельным значением или массивом. [`Float*`](/ru/reference/data-types/float) или [`Array(Float*)`](/ru/reference/data-types/array)

**Возвращаемое значение**

Возвращает значения дельты на указанной сетке. Возвращаемый массив содержит по одному значению для каждой точки временной сетки. Значение равно NULL, если в пределах окна недостаточно отсчетов, чтобы вычислить значение дельты для конкретной точки сетки. [`Array(Nullable(Float64))`](/ru/reference/data-types/array)

**Примеры**

**Базовое использование с отдельными парами «временная метка — значение»**

```sql title=Query
WITH
    -- ПРИМЕЧАНИЕ: пропуск между 140 и 190 демонстрирует, как значения заполняются для ts = 150, 165, 180 в соответствии с параметром window
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values, -- массив значений, соответствующих временным меткам выше
    90 AS start_ts,      -- начало сетки временных меток
    90 + 120 AS end_ts,  -- конец сетки временных меток
    15 AS step_seconds,  -- шаг сетки временных меток
    45 AS window_seconds -- окно устаревания ("staleness")
SELECT timeSeriesDeltaToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)
FROM
(
    -- Этот подзапрос преобразует массивы временных меток и значений в строки `timestamp`, `value`
    SELECT
        arrayJoin(arrayZip(timestamps, values)) AS ts_and_val,
        ts_and_val.1 AS timestamp,
        ts_and_val.2 AS value
);
```

```response title=Response
┌─timeSeriesDeltaToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamp, value)─┐
│ [NULL,NULL,0,3,4.5,3.75,NULL,NULL,3.75]                                               │
└───────────────────────────────────────────────────────────────────────────────────────┘
```

**Использование аргументов типа Array**

```sql title=Query
-- можно передавать несколько выборок временных меток и значений в виде массивов одинакового размера
WITH
    [110, 120, 130, 140, 190, 200, 210, 220, 230]::Array(DateTime) AS timestamps,
    [1, 1, 3, 4, 5, 5, 8, 12, 13]::Array(Float32) AS values,
    90 AS start_ts,
    90 + 120 AS end_ts,
    15 AS step_seconds,
    45 AS window_seconds
SELECT timeSeriesDeltaToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values);
```

```response title=Response
┌─timeSeriesDeltaToGrid(start_ts, end_ts, step_seconds, window_seconds)(timestamps, values)─┐
│ [NULL,NULL,0,3,4.5,3.75,NULL,NULL,3.75]                                                 │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
