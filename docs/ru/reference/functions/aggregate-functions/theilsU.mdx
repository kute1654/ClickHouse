<div id="theilsU">
  ## theilsU
</div>

Добавленный в: v22.1.0

Функция `theilsU` вычисляет [коэффициент неопределённости Тейлса U](https://en.wikipedia.org/wiki/Contingency_table#Uncertainty_coefficient) — величину, измеряющую связь между двумя столбцами в таблице.
Её значения находятся в диапазоне от 0.0 (связь отсутствует) до 1.0 (полное соответствие).

**Синтаксис**

```sql
theilsU(column1, column2)
```

**Аргументы**

* `column1` — Первый сравниваемый столбец. [`Any`](/ru/reference/data-types/index)
* `column2` — Второй сравниваемый столбец. [`Any`](/ru/reference/data-types/index)

**Возвращаемое значение**

Возвращает значение в диапазоне от 0 до 1. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Пример использования**

```sql title=Query
SELECT theilsU(a, b)
FROM (
    SELECT
        number % 10 AS a,
        number % 4 AS b
    FROM
        numbers(150)
);
```

```response title=Response
┌────────theilsU(a, b)─┐
│  0.30195720557678846 │
└──────────────────────┘
```
