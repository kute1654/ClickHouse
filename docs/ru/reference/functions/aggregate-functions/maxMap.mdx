<div id="maxMappedArrays">
  ## maxMappedArrays
</div>

Добавленный в: v20.5.0

Вычисляет максимальное значение из массива `value` в соответствии с ключами, указанными в массиве `key`.

<Note>
  * Передача кортежа ключей и массивов значений эквивалентна передаче массива ключей и массива значений.
  * Количество элементов в `key` и `value` должно быть одинаковым для каждой суммируемой строки.
</Note>

**Синтаксис**

```sql
maxMappedArrays(key, value)
maxMappedArrays(Tuple(key, value))
```

**Аргументы**

* `key` — Массив ключей. [`Array(T)`](/ru/reference/data-types/array)
* `value` — Массив значений. [`Array(T)`](/ru/reference/data-types/array)

**Возвращаемое значение**

Возвращает кортеж из двух массивов: ключи в отсортированном порядке и значения, вычисленные для соответствующих ключей. [`Tuple(Array(T), Array(T))`](/ru/reference/data-types/tuple)

**Примеры**

**Пример использования**

```sql title=Query
SELECT maxMappedArrays(a, b)
FROM VALUES('a Array(Char), b Array(Int64)', (['x', 'y'], [2, 2]), (['y', 'z'], [3, 1]));
```

```response title=Response
┌─maxMappedArrays(a, b)───┐
│ (['x','y','z'],[2,3,1]) │
└─────────────────────────┘
```
