<div id="skewSamp">
  ## skewSamp
</div>

Добавленный в: v20.1.0

Вычисляет [выборочную асимметрию](https://en.wikipedia.org/wiki/Skewness) последовательности значений.

Представляет собой несмещённую оценку асимметрии случайной величины, если переданные значения образуют её выборку.

**Синтаксис**

```sql
skewSamp(expr)
```

**Аргументы**

* `expr` — Выражение, возвращающее число. [`Выражение`](/ru/reference/data-types/special-data-types/expression)

**Возвращаемое значение**

Возвращает коэффициент асимметрии заданного распределения. Если `n <= 1` (`n` — размер выборки), функция возвращает `nan`. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Симметричное распределение**

```sql title=Query
SELECT skewSamp(number) FROM numbers(100);
```

```response title=Response
┌─skewSamp(number)─┐
│                0 │
└──────────────────┘
```

**Правосторонне-асимметричное распределение**

```sql title=Query
SELECT skewSamp(x) FROM (SELECT pow(number, 2) AS x FROM numbers(10));
```

```response title=Response
┌────────skewSamp(x)─┐
│ 0.5751042382747413 │
└────────────────────┘
```
