<div id="stddevPop">
  ## stddevPop
</div>

Добавленный в: v1.1.0

Возвращает стандартное отклонение генеральной совокупности для последовательности числовых данных.
Результат равен квадратному корню из [`varPop`](/ru/reference/functions/aggregate-functions/varPop).

<Note>
  Эта функция использует численно неустойчивый алгоритм. Если вам нужна [численная устойчивость](https://en.wikipedia.org/wiki/Numerical_stability) при вычислениях, используйте функцию [`stddevPopStable`](/ru/reference/functions/aggregate-functions/stddevPopStable). Она работает медленнее, но обеспечивает меньшую вычислительную погрешность.
</Note>

**Синтаксис**

```sql
stddevPop(x)
```

**Псевдонимы**: `STD`, `STDDEV_POP`

**Аргументы**

* `x` — Набор значений, для которого вычисляется стандартное отклонение генеральной совокупности. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal)

**Возвращаемое значение**

Возвращает квадратный корень из дисперсии генеральной совокупности `x`. [`Float64`](/ru/reference/data-types/float)

**Примеры**

**Вычисление стандартного отклонения генеральной совокупности**

```sql title=Query
CREATE TABLE test_data (population UInt8) ENGINE = Log;
INSERT INTO test_data VALUES (3),(3),(3),(4),(4),(5),(5),(7),(11),(15);

SELECT stddevPop(population) AS stddev FROM test_data;
```

```response title=Response
┌────────────stddev─┐
│ 3.794733192202055 │
└───────────────────┘
```
