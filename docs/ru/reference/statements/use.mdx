```sql
USE [DATABASE] db
```

Позволяет задать текущую базу данных для сеанса.

Текущая база данных используется для поиска таблиц, если база данных не указана в запросе явно через точку перед именем таблицы.

Этот запрос нельзя выполнить при использовании HTTP-протокола, так как в нём нет понятия сеанса.
