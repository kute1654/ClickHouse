```sql
EXISTS [TEMPORARY] [TABLE|DICTIONARY|DATABASE] [db.]name [INTO OUTFILE filename] [FORMAT format]
```

Возвращает один столбец типа `UInt8`, содержащий значение `0`, если таблица или база данных не существуют, или `1`, если таблица существует в указанной базе данных.
