Пример настроек:

<Tabs>
  <Tab title="DDL">
    ```sql
    SOURCE(REDIS(
        host 'localhost'
        port 6379
        storage_type 'simple'
        db_index 0
    ))
    ```
  </Tab>

  <Tab title="Файл конфигурации">
    ```xml
    <source>
        <redis>
            <host>localhost</host>
            <port>6379</port>
            <storage_type>simple</storage_type>
            <db_index>0</db_index>
        </redis>
    </source>
    ```
  </Tab>
</Tabs>

<br />

Поля настроек:

| Настройка      | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `host`         | Хост Redis.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `port`         | Порт сервера Redis.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `storage_type` | Структура внутреннего хранилища Redis, используемая для работы с ключами. `simple` использует плоское отображение ключ-значение и поддерживает структуры с простым ключом, а также структуры со сложным ключом из одного столбца (например, `complex_key_cache` и `complex_key_direct`). `hash_map` использует хеш Redis и требуется для составных сложных ключей; предполагается наличие ровно двух столбцов ключа. Столбцы ключа должны быть целочисленного или строкового типа. Структуры с диапазонами не поддерживаются. Значение по умолчанию — `simple`. Необязательный параметр. |
| `db_index`     | Числовой индекс логической базы данных Redis. Значение по умолчанию — `0`. Необязательный параметр.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
