<div id="direct">
  ## direct
</div>

Словарь не хранится в памяти и при обработке запроса обращается напрямую к источнику.

Ключ словаря имеет тип [UInt64](/ru/reference/data-types/int-uint).

Поддерживаются все типы [источников](/ru/reference/statements/create/dictionary/sources/overview#dictionary-sources), кроме локальных файлов.

Пример конфигурации:

<Tabs>
  <Tab title="DDL">
    ```sql
    LAYOUT(DIRECT())
    ```
  </Tab>

  <Tab title="Файл конфигурации">
    ```xml
    <layout>
      <direct />
    </layout>
    ```
  </Tab>
</Tabs>

<br />

  ## complex_key_direct
</div>
