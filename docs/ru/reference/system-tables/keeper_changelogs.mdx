<div id="description">
  ## Описание
</div>

Эта таблица отсутствует, если на этом узле не настроен запуск внутрипроцессного ClickHouse Keeper. Она содержит по одной строке для каждого файла журнала изменений Raft (`changelog_<from>_<to>.bin[.zstd]`), отслеживаемого внутрипроцессным хранилищем журналов Keeper, включая активный файл, в который сейчас дописываются данные.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `from_log_index` ([UInt64](/ru/reference/data-types/int-uint)) — Первый индекс лога Raft в файле (включительно).
* `to_log_index` ([UInt64](/ru/reference/data-types/int-uint)) — Последний индекс лога Raft, охватываемый именем файла (включительно). Для активного файла это целевой индекс ротации, и он может быть больше `last_entry_index`.
* `last_entry_index` ([Nullable(UInt64)](/ru/reference/data-types/int-uint)) — Наибольший индекс лога, фактически записанный в этот файл. `NULL`, если активный файл ещё не получил ни одной записи или файл повреждён.
* `entries` ([UInt64](/ru/reference/data-types/int-uint)) — Количество записей, добавленных в этот файл. `ALIAS` для `ifNull(last_entry_index - from_log_index + 1, 0)`.
* `path` ([String](/ru/reference/data-types/string)) — Путь к файлу на диске.
* `disk_name` ([String](/ru/reference/data-types/string)) — Имя диска, на котором находится файл.
* `size_bytes` ([UInt64](/ru/reference/data-types/int-uint)) — Размер файла на диске.
* `modification_time` ([DateTime](/ru/reference/data-types/datetime)) — Время последнего изменения файла.
* `is_compressed` ([Bool](/ru/reference/data-types/boolean)) — Полезная нагрузка файла сжата с помощью zstd.
* `active` ([Bool](/ru/reference/data-types/boolean)) — В этот файл сейчас добавляются записи.
* `is_broken` ([Bool](/ru/reference/data-types/boolean)) — При запуске обнаружена повреждённая последняя запись.

{/*AUTOGENERATED_END*/}

Пример:

```sql
SELECT from_log_index, to_log_index, entries, path, active FROM system.keeper_changelogs ORDER BY from_log_index;
```

```text
┌─from_log_index─┬─to_log_index─┬─entries─┬─path───────────────────────────┬─active─┐
│              1 │         1000 │    1000 │ changelog_1_1000.bin.zstd      │ false  │
│           1001 │         2000 │     537 │ changelog_1001_2000.bin.zstd   │ true   │
└────────────────┴──────────────┴─────────┴────────────────────────────────┴────────┘
```
