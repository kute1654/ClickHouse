<div id="description">
  ## Описание
</div>

Содержит сведения о [словарях](/ru/reference/statements/create/dictionary).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных, содержащей словарь, созданный DDL-запросом. Для других словарей — пустая строка.
* `name` ([String](/ru/reference/data-types/index)) — Имя словаря.
* `uuid` ([UUID](/ru/reference/data-types/index)) — UUID словаря.
* `status` ([Enum8(&#39;NOT&#95;LOADED&#39; = 0, &#39;LOADED&#39; = 1, &#39;FAILED&#39; = 2, &#39;LOADING&#39; = 3, &#39;FAILED&#95;AND&#95;RELOADING&#39; = 4, &#39;LOADED&#95;AND&#95;RELOADING&#39; = 5, &#39;NOT&#95;EXIST&#39; = 6)](/ru/reference/data-types/index)) — Статус словаря. Возможные значения:
  * **NOT&#95;LOADED** — Словарь не был загружен, поскольку не использовался
  * **LOADED** — Словарь успешно загружен
  * **FAILED** — Не удалось загрузить словарь из-за ошибки
  * **LOADING** — Словарь загружается в данный момент
  * **LOADED&#95;AND&#95;RELOADING** — Словарь успешно загружен
  * **and is being reloaded right now (frequent reasons: SYSTEM RELOAD DICTIONARY query** — и сейчас перезагружается (частые причины: запрос SYSTEM RELOAD DICTIONARY
  * **timeout** — тайм-аут
  * **dictionary config has changed)** — конфигурация словаря изменилась)
  * **FAILED&#95;AND&#95;RELOADING** — Не удалось загрузить словарь из-за ошибки, и сейчас он загружается.
* `origin` ([String](/ru/reference/data-types/index)) — Путь к файлу конфигурации, описывающему словарь.
* `type` ([String](/ru/reference/data-types/index)) — Тип выделения памяти для словаря. Хранение словарей в памяти.
* `key.names` ([Array(String)](/ru/reference/data-types/index)) — Массив имен ключей, предоставляемых словарем.
* `key.types` ([Array(String)](/ru/reference/data-types/index)) — Соответствующий массив типов ключей, предоставляемых словарем.
* `attribute.names` ([Array(String)](/ru/reference/data-types/index)) — Массив имен атрибутов, предоставляемых словарем.
* `attribute.types` ([Array(String)](/ru/reference/data-types/index)) — Соответствующий массив типов атрибутов, предоставляемых словарем.
* `bytes_allocated` ([UInt64](/ru/reference/data-types/index)) — Объем оперативной памяти, выделенной для словаря.
* `hierarchical_index_bytes_allocated` ([UInt64](/ru/reference/data-types/index)) — Объем оперативной памяти, выделенной для иерархического индекса.
* `query_count` ([UInt64](/ru/reference/data-types/index)) — Количество запросов с момента загрузки словаря или с момента последней успешной перезагрузки.
* `hit_rate` ([Float64](/ru/reference/data-types/index)) — Для cache-словарей — процент использований, при которых значение находилось в кэше.
* `found_rate` ([Float64](/ru/reference/data-types/index)) — Процент использований, при которых значение было найдено.
* `element_count` ([UInt64](/ru/reference/data-types/index)) — Количество элементов, хранящихся в словаре.
* `load_factor` ([Float64](/ru/reference/data-types/index)) — Степень заполнения словаря в процентах (для хешированного словаря — степень заполнения хеш-таблицы).
* `source` ([String](/ru/reference/data-types/index)) — Текст, описывающий источник данных для словаря.
* `lifetime_min` ([UInt64](/ru/reference/data-types/index)) — Минимальное время жизни словаря в памяти, по истечении которого ClickHouse пытается перезагрузить словарь (если задан `invalidate&#95;query`, то только если он изменился). Задается в секундах.
* `lifetime_max` ([UInt64](/ru/reference/data-types/index)) — Максимальное время жизни словаря в памяти, по истечении которого ClickHouse пытается перезагрузить словарь (если задан `invalidate&#95;query`, то только если он изменился). Задается в секундах.
* `loading_start_time` ([DateTime](/ru/reference/data-types/index)) — Время начала загрузки словаря.
* `last_successful_update_time` ([DateTime](/ru/reference/data-types/index)) — Время завершения загрузки или обновления словаря. Помогает отслеживать проблемы с источниками словарей и выяснять их причины.
* `error_count` ([UInt64](/ru/reference/data-types/index)) — Количество ошибок с момента последней успешной загрузки. Помогает отслеживать проблемы с источниками словарей и выяснять их причины.
* `loading_duration` ([Float32](/ru/reference/data-types/index)) — Длительность загрузки словаря.
* `last_exception` ([String](/ru/reference/data-types/index)) — Текст ошибки, возникающей при создании или перезагрузке словаря, если словарь не удалось создать.
* `comment` ([String](/ru/reference/data-types/index)) — Текст комментария к словарю.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

Настройте словарь:

```sql
CREATE DICTIONARY dictionary_with_comment
(
    id UInt64,
    value String
)
PRIMARY KEY id
SOURCE(CLICKHOUSE(HOST 'localhost' PORT tcpPort() TABLE 'source_table'))
LAYOUT(FLAT())
LIFETIME(MIN 0 MAX 1000)
COMMENT 'The temporary dictionary';
```

Убедитесь, что словарь загружен.

```sql
SELECT * FROM system.dictionaries LIMIT 1 FORMAT Vertical;
```
