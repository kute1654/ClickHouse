<div id="description">
  ## Описание
</div>

Эта таблица содержит информацию о столбцах в частях проекций таблиц семейства MergeTree.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `partition` ([String](/ru/reference/data-types/index)) — Имя партиции.
* `name` ([String](/ru/reference/data-types/index)) — Имя части данных.
* `part_type` ([String](/ru/reference/data-types/index)) — Формат хранения части данных.
* `parent_name` ([String](/ru/reference/data-types/index)) — Имя исходной (родительской) части данных.
* `parent_uuid` ([UUID](/ru/reference/data-types/index)) — UUID исходной (родительской) части данных.
* `parent_part_type` ([String](/ru/reference/data-types/index)) — Формат хранения исходной (родительской) части данных.
* `active` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, активна ли часть данных
* `marks` ([UInt64](/ru/reference/data-types/index)) — Количество отметок.
* `rows` ([UInt64](/ru/reference/data-types/index)) — Количество строк.
* `bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер всех файлов части данных в байтах.
* `data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в части данных. Вспомогательные файлы (например, файлы с отметками) не включаются.
* `data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в части данных. Вспомогательные файлы (например, файлы с отметками) не включаются.
* `marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер файла с отметками.
* `parent_marks` ([UInt64](/ru/reference/data-types/index)) — Количество отметок в исходной (родительской) части.
* `parent_rows` ([UInt64](/ru/reference/data-types/index)) — Количество строк в исходной (родительской) части.
* `parent_bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер всех файлов исходной (родительской) части данных в байтах.
* `parent_data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в исходной (родительской) части данных.
* `parent_data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в исходной (родительской) части данных.
* `parent_marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер файла с отметками в исходной (родительской) части данных.
* `modification_time` ([DateTime](/ru/reference/data-types/index)) — Время изменения каталога с частью данных. Обычно оно соответствует времени создания части данных.
* `remove_time` ([DateTime](/ru/reference/data-types/index)) — Время, когда часть данных стала неактивной.
* `refcount` ([UInt32](/ru/reference/data-types/index)) — Количество мест, где используется часть данных. Значение больше 2 указывает на то, что часть данных используется в запросах или слияниях.
* `min_date` ([Date](/ru/reference/data-types/index)) — Минимальное значение для столбца Date, если он включён в ключ партиционирования.
* `max_date` ([Date](/ru/reference/data-types/index)) — Максимальное значение для столбца Date, если он включён в ключ партиционирования.
* `min_time` ([DateTime](/ru/reference/data-types/index)) — Минимальное значение для столбца DateTime, если он включён в ключ партиционирования.
* `max_time` ([DateTime](/ru/reference/data-types/index)) — Максимальное значение для столбца типа DateTime, если он включён в ключ партиционирования.
* `partition_id` ([String](/ru/reference/data-types/index)) — Идентификатор партиции.
* `min_block_number` ([Int64](/ru/reference/data-types/index)) — Минимальное число частей данных, из которых состоит текущая часть после слияния.
* `max_block_number` ([Int64](/ru/reference/data-types/index)) — Максимальное число частей данных, из которых состоит текущая часть после слияния.
* `level` ([UInt32](/ru/reference/data-types/index)) — Глубина дерева слияния. Ноль означает, что текущая часть была создана вставкой, а не слиянием других частей.
* `data_version` ([UInt64](/ru/reference/data-types/index)) — Число, используемое для определения того, какие мутации должны быть применены к части данных (мутации с версией выше, чем data&#95;version).
* `primary_key_bytes_in_memory` ([UInt64](/ru/reference/data-types/index)) — Объём памяти (в байтах), занимаемый значениями первичного ключа.
* `primary_key_bytes_in_memory_allocated` ([UInt64](/ru/reference/data-types/index)) — Объём памяти (в байтах), зарезервированный для значений первичного ключа.
* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `engine` ([String](/ru/reference/data-types/index)) — Имя движка таблицы без параметров.
* `disk_name` ([String](/ru/reference/data-types/index)) — Имя диска, на котором хранится часть данных.
* `path` ([String](/ru/reference/data-types/index)) — Абсолютный путь к папке с файлами части данных.
* `column` ([String](/ru/reference/data-types/index)) — Имя столбца.
* `type` ([String](/ru/reference/data-types/index)) — Тип столбца.
* `column_position` ([UInt64](/ru/reference/data-types/index)) — Порядковая позиция столбца в таблице, начиная с 1.
* `default_kind` ([String](/ru/reference/data-types/index)) — Тип выражения (DEFAULT, MATERIALIZED, ALIAS) для значения по умолчанию или пустая строка, если он не определён.
* `default_expression` ([String](/ru/reference/data-types/index)) — Выражение для значения по умолчанию или пустая строка, если оно не определено.
* `column_bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер столбца в байтах.
* `column_data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в столбце в байтах.
* `column_data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в столбце в байтах.
* `column_marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер столбца вместе с marks, в байтах.
* `column_modification_time` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Время последнего изменения столбца.
