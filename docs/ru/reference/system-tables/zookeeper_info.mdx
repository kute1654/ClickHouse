<div id="description">
  ## Описание
</div>

Эта таблица выводит сводную диагностическую информацию о ZooKeeper, а список узлов берётся из конфигурации.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `zookeeper_cluster_name` ([String](/ru/reference/data-types/string)) — название кластера ZooKeeper.
* `host` ([String](/ru/reference/data-types/string)) — Имя хоста или IP-адрес узла ZooKeeper, к которому подключён ClickHouse.
* `port` ([UInt16](/ru/reference/data-types/int-uint)) — Порт узла ZooKeeper, к которому был подключен ClickHouse.
* `index` ([Nullable(UInt8)](/ru/reference/data-types/int-uint)) — Индекс узла ZooKeeper, к которому подключён ClickHouse. Индекс берётся из конфигурации ZooKeeper. Если подключение отсутствует, в этом столбце значение NULL.
* `is_connected` ([Nullable(UInt8)](/ru/reference/data-types/int-uint)) — Есть ли подключение к zookeeper.
* `is_readonly` ([UInt8](/ru/reference/data-types/int-uint)) — Только для чтения.
* `version` ([String](/ru/reference/data-types/string)) — версия ZooKeeper.
* `avg_latency` ([UInt64](/ru/reference/data-types/int-uint)) — Средняя задержка.
* `max_latency` ([UInt64](/ru/reference/data-types/int-uint)) — Максимальная задержка.
* `min_latency` ([UInt64](/ru/reference/data-types/int-uint)) — Минимальная задержка.
* `packets_received` ([UInt64](/ru/reference/data-types/int-uint)) — Количество принятых пакетов.
* `packets_sent` ([UInt64](/ru/reference/data-types/int-uint)) — количество отправленных пакетов.
* `outstanding_requests` ([UInt64](/ru/reference/data-types/int-uint)) — Количество незавершённых запросов.
* `server_state` ([String](/ru/reference/data-types/string)) — Состояние сервера.
* `is_leader` ([UInt8](/ru/reference/data-types/int-uint)) — Является ли этот узел лидером ZooKeeper.
* `znode_count` ([UInt64](/ru/reference/data-types/int-uint)) — Число znode.
* `watch_count` ([UInt64](/ru/reference/data-types/int-uint)) — Число наблюдений.
* `ephemerals_count` ([UInt64](/ru/reference/data-types/int-uint)) — Количество эфемерных значений.
* `approximate_data_size` ([UInt64](/ru/reference/data-types/int-uint)) — Примерный размер данных.
* `followers` ([UInt64](/ru/reference/data-types/int-uint)) — Последователи лидера. Это поле доступно только на лидере.
* `synced_followers` ([UInt64](/ru/reference/data-types/int-uint)) — Синхронизированные узлы-последователи лидера. Это поле доступно только на лидере.
* `pending_syncs` ([UInt64](/ru/reference/data-types/int-uint)) — Ожидающие синхронизации на лидере. Это поле доступно только на лидере.
* `open_file_descriptor_count` ([UInt64](/ru/reference/data-types/int-uint)) — Количество открытых файловых дескрипторов. Доступно только на Unix-платформах.
* `max_file_descriptor_count` ([UInt64](/ru/reference/data-types/int-uint)) — Максимально допустимое количество файловых дескрипторов. Доступно только на платформах Unix.
* `connections` ([UInt64](/ru/reference/data-types/int-uint)) — Подключения к ZooKeeper.
* `outstanding` ([UInt64](/ru/reference/data-types/int-uint)) — Показатель outstanding в ZooKeeper.
* `zxid` ([UInt64](/ru/reference/data-types/int-uint)) — `zxid` в ZooKeeper.
* `node_count` ([UInt64](/ru/reference/data-types/int-uint)) — Число узлов ZooKeeper.
* `snapshot_dir_size` ([UInt64](/ru/reference/data-types/int-uint)) — Размер каталога со снимками ZooKeeper.
* `log_dir_size` ([UInt64](/ru/reference/data-types/int-uint)) — Размер каталога логов ZooKeeper.
* `first_log_idx` ([UInt64](/ru/reference/data-types/int-uint)) — Первый индекс журнала ZooKeeper.
* `first_log_term` ([UInt64](/ru/reference/data-types/int-uint)) — Первый терм журнала ZooKeeper.
* `last_log_idx` ([UInt64](/ru/reference/data-types/int-uint)) — Индекс последней записи журнала ZooKeeper.
* `last_log_term` ([UInt64](/ru/reference/data-types/int-uint)) — Последний терм журнала ZooKeeper.
* `last_committed_idx` ([UInt64](/ru/reference/data-types/int-uint)) — Последний зафиксированный индекс в ZooKeeper.
* `leader_committed_log_idx` ([UInt64](/ru/reference/data-types/int-uint)) — Индекс журнала, который зафиксировал лидер ZooKeeper.
* `target_committed_log_idx` ([UInt64](/ru/reference/data-types/int-uint)) — Целевой индекс подтверждённого журнала ZooKeeper.
* `last_snapshot_idx` ([UInt64](/ru/reference/data-types/int-uint)) — индекс последнего снимка в ZooKeeper.
