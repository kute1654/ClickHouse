<div id="description">
  ## Описание
</div>

Содержит описание движков таблиц, поддерживаемых сервером, а также сведения о поддерживаемых ими возможностях.

Эта таблица содержит следующие столбцы (тип столбца указан в скобках):

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя движка таблицы.
* `supports_settings` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы секцию SETTINGS.
* `supports_skipping_indices` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы индексы пропуска.
* `supports_projections` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы проекции.
* `supports_sort_order` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы секции PARTITION&#95;BY, PRIMARY&#95;KEY, ORDER&#95;BY и SAMPLE&#95;BY.
* `supports_ttl` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы TTL.
* `supports_replication` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы репликацию данных.
* `supports_deduplication` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы дедупликацию данных.
* `supports_parallel_insert` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, поддерживает ли движок таблицы параллельную вставку (см. настройку max&#95;insert&#95;threads).
* `description` ([String](/ru/reference/data-types/index)) — Описание назначения движка таблицы. Для движков с отдельной страницей документации здесь содержится полный Markdown-текст этой страницы; для остальных движков это краткое описание.
* `syntax` ([String](/ru/reference/data-types/index)) — Как движок таблицы используется при создании таблицы. Для большинства движков это секция ENGINE в запросе CREATE TABLE, но некоторые движки (например, различные виды представлений или движок Loop) используются в других формах, таких как CREATE VIEW или SELECT из табличной функции.
* `examples` ([String](/ru/reference/data-types/index)) — Примеры использования.
* `introduced_in` ([String](/ru/reference/data-types/index)) — Версия ClickHouse, в которой движок таблицы впервые появился, в формате major.minor.
* `related` ([Array(String)](/ru/reference/data-types/index)) — Имена связанных движков таблиц.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql title="Query"
SELECT
    name,
    supports_settings,
    supports_skipping_indices,
    supports_sort_order,
    supports_ttl,
    supports_replication,
    supports_deduplication,
    supports_parallel_insert
FROM system.table_engines
WHERE name IN ('Kafka', 'MergeTree', 'ReplicatedCollapsingMergeTree')
```
