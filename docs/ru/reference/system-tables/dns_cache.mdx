<div id="description">
  ## Описание
</div>

Содержит сведения о кэшированных DNS-записях.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([String](/ru/reference/data-types/index)) — Имя хоста.
* `ip_address` ([String](/ru/reference/data-types/index)) — IP-адрес.
* `ip_family` ([Enum8(&#39;IPv4&#39; = 0, &#39;IPv6&#39; = 1, &#39;UNIX&#95;LOCAL&#39; = 2)](/ru/reference/data-types/index)) — Семейство IP-адресов.
* `cached_at` ([DateTime](/ru/reference/data-types/index)) — Временная метка кэширования записи.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname  | ip&#95;address | ip&#95;family | cached&#95;at       |
| :-------- | :------------- | :------------ | :------------------ |
| localhost | ::1            | IPv6          | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1      | IPv4          | 2024-02-11 17:04:40 |
