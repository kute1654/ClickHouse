<div id="description">
  ## Описание
</div>

Содержит список всех именованных коллекций, созданных SQL-запросом или разобранных из файла конфигурации.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя коллекции.
* `collection` ([Map(String, String)](/ru/reference/data-types/index)) — Внутренние компоненты коллекции.
* `source` ([String](/ru/reference/data-types/index)) — Источник именованной коллекции.
* `create_query` ([String](/ru/reference/data-types/index)) — Запрос CREATE для именованной коллекции.
