<div id="description">
  ## Описание
</div>

Содержит историю значений ошибок из таблицы `system.errors`, которые периодически сбрасываются на диск.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события.
* `code` ([Int32](/ru/reference/data-types/int-uint)) — Код ошибки.
* `error` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя ошибки.
* `value` ([UInt64](/ru/reference/data-types/int-uint)) — Число ошибок, произошедших за интервал времени.
* `remote` ([UInt8](/ru/reference/data-types/int-uint)) — Удалённое исключение (то есть полученное в ходе одного из распределённых запросов).
* `last_error_time` ([DateTime](/ru/reference/data-types/datetime)) — Время, когда произошла последняя ошибка.
* `last_error_message` ([String](/ru/reference/data-types/string)) — Сообщение о последней ошибке.
* `last_error_query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса, вызвавшего последнюю ошибку (если доступен).
* `last_error_trace` ([Array(UInt64)](/ru/reference/data-types/array)) — Трассировка стека, представляющая собой список физических адресов, по которым расположены вызванные методы.

{/*AUTOGENERATED_END*/}
