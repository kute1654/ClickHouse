<div id="description">
  ## Описание
</div>

Содержит сведения о назначенных ролях пользователей и ролей. Чтобы добавить записи в эту таблицу, используйте `GRANT role TO user`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `user_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя пользователя.
* `role_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя роли.
* `granted_role_name` ([String](/ru/reference/data-types/index)) — Имя роли, предоставленной роли `role_name`. Чтобы предоставить одну роль другой, используйте `GRANT role1 TO role2`.
* `granted_role_id` ([UUID](/ru/reference/data-types/index)) — Идентификатор роли.
* `granted_role_is_default` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли `granted_role` ролью по умолчанию. Возможные значения: • 1 — `granted_role` является ролью по умолчанию, • 0 — `granted_role` не является ролью по умолчанию.
* `with_admin_option` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, имеет ли `granted_role` привилегию `ADMIN OPTION`. Возможные значения: • 1 — у роли есть привилегия `ADMIN OPTION`. • 0 — у роли нет привилегии `ADMIN OPTION`.

{/*AUTOGENERATED_END*/}
