<div id="description">
  ## Описание
</div>

Содержит сведения о [рабочих нагрузках](/ru/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage), размещённых на локальном сервере. Таблица содержит по одной строке для каждой рабочей нагрузки.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Название рабочей нагрузки.
* `parent` ([String](/ru/reference/data-types/index)) — Название родительской рабочей нагрузки.
* `create_query` ([String](/ru/reference/data-types/index)) — запрос CREATE для рабочей нагрузки.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql
SELECT *
FROM system.workloads
FORMAT Vertical
```

```text
Строка 1:
──────
name:         production
parent:       all
create_query: CREATE WORKLOAD production IN `all` SETTINGS weight = 9

Строка 2:
──────
name:         development
parent:       all
create_query: CREATE WORKLOAD development IN `all`

Строка 3:
──────
name:         all
parent:
create_query: CREATE WORKLOAD `all`
```
