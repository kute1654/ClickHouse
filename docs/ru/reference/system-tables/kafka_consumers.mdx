<div id="description">
  ## Описание
</div>

Содержит информацию о потребителях Kafka.
Относится к [движку таблицы Kafka](/ru/reference/engines/table-engines/integrations/kafka) (встроенная интеграция ClickHouse)

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — База данных таблицы с движком Kafka.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы с движком Kafka.
* `consumer_id` ([String](/ru/reference/data-types/index)) — Идентификатор потребителя Kafka. Обратите внимание: у таблицы может быть несколько потребителей Kafka. Задаётся параметром `kafka_num_consumers`.
* `assignments.topic` ([Array(String)](/ru/reference/data-types/index)) — Kafka topic.
* `assignments.partition_id` ([Array(Int32)](/ru/reference/data-types/index)) — Идентификатор Kafka partition. Обратите внимание: партиция может быть назначена только одному потребителю Kafka.
* `assignments.current_offset` ([Array(Int64)](/ru/reference/data-types/index)) — Текущее смещение.
* `assignments.intent_size` ([Array(Nullable(Int64))](/ru/reference/data-types/index)) — Количество отправленных, но ещё не подтверждённых сообщений в новом StorageKafka.
* `exceptions.time` ([Array(DateTime)](/ru/reference/data-types/index)) — Временная метка, когда были сгенерированы 10 последних исключений.
* `exceptions.text` ([Array(String)](/ru/reference/data-types/index)) — Текст 10 последних исключений.
* `last_poll_time` ([DateTime](/ru/reference/data-types/index)) — Временная метка последнего опроса.
* `num_messages_read` ([UInt64](/ru/reference/data-types/index)) — Количество сообщений, прочитанных потребителем Kafka.
* `last_commit_time` ([DateTime](/ru/reference/data-types/index)) — Временная метка последнего опроса.
* `num_commits` ([UInt64](/ru/reference/data-types/index)) — Общее количество commit для потребителя Kafka.
* `last_rebalance_time` ([DateTime](/ru/reference/data-types/index)) — Временная метка последней перебалансировки Kafka.
* `num_rebalance_revocations` ([UInt64](/ru/reference/data-types/index)) — Количество случаев, когда у потребителя Kafka отзывали его партиции.
* `num_rebalance_assignments` ([UInt64](/ru/reference/data-types/index)) — Количество случаев, когда потребитель Kafka назначался кластеру Kafka.
* `is_currently_used` ([UInt8](/ru/reference/data-types/index)) — Флаг, показывающий, используется ли потребитель Kafka в данный момент.
* `last_used` ([DateTime64(6)](/ru/reference/data-types/index)) — Время последнего использования этого потребителя Kafka.
* `rdkafka_stat` ([String](/ru/reference/data-types/index)) — Внутренняя статистика библиотеки. Чтобы отключить, установите `statistics_interval_ms` в 0; значение по умолчанию — 3000 (раз в три секунды).
* `dependencies` ([Array(Array(String))](/ru/reference/data-types/index)) — Транзитивные зависимости базы данных.
* `missing_dependencies` ([Array(Array(String))](/ru/reference/data-types/index)) — Отсутствующие транзитивные зависимости базы данных.

{/*AUTOGENERATED_END*/}
