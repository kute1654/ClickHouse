<div id="description">
  ## Описание
</div>

Содержит сведения о существующих проекциях во всех таблицах.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `name` ([String](/ru/reference/data-types/index)) — Имя проекции.
* `type` ([Enum8(&#39;Normal&#39; = 0, &#39;Aggregate&#39; = 1)](/ru/reference/data-types/index)) — Тип проекции.
* `sorting_key` ([Array(String)](/ru/reference/data-types/index)) — Ключ сортировки проекции.
* `query` ([String](/ru/reference/data-types/index)) — Запрос для проекции.
* `settings` ([Map(String, String)](/ru/reference/data-types/index)) — Настройки проекции.

{/*AUTOGENERATED_END*/}

SELECT * FROM system.projections LIMIT 2 FORMAT Vertical;
