<div id="description">
  ## Описание
</div>

Содержит сведения о задачах в фоновых пулах планировщика. Фоновые пулы планировщика используются для выполнения периодических задач, таких как распределённая отправка данных, сброс буферов и операции брокера сообщений.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `pool` ([String](/ru/reference/data-types/index)) — Имя пула. Возможные значения: `schedule` — пул расписания общего назначения, `buffer_flush` — пул для сброса данных таблицы Buffer, `distributed` — пул для операций distributed таблицы, `message_broker` — пул для операций брокера сообщений.
* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `table_uuid` ([UUID](/ru/reference/data-types/index)) — UUID таблицы.
* `query_id` ([String](/ru/reference/data-types/index)) — Query ID (если выполняется в данный момент). Примечание: это не настоящий запрос, а случайно сгенерированный ID для сопоставления с записями журнала в `system.text_log`.
* `elapsed_ms` ([UInt64](/ru/reference/data-types/index)) — Время выполнения задачи (если выполняется в данный момент).
* `log_name` ([String](/ru/reference/data-types/index)) — Имя журнала для задачи.
* `deactivated` ([UInt8](/ru/reference/data-types/index)) — Деактивирована ли задача (всегда false, так как деактивированные задачи удаляются из пула).
* `scheduled` ([UInt8](/ru/reference/data-types/index)) — Запланирована ли задача к выполнению.
* `delayed` ([UInt8](/ru/reference/data-types/index)) — Запланирована ли задача с задержкой.
* `executing` ([UInt8](/ru/reference/data-types/index)) — Выполняется ли задача в данный момент.

{/*AUTOGENERATED_END*/}
