<div id="description">
  ## Описание
</div>

Содержит информацию о локальных файлах, находящихся в очереди на отправку в сегменты. Эти локальные файлы содержат новые части, которые создаются при вставке новых данных в таблицу Distributed в асинхронном режиме.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `data_path` ([String](/ru/reference/data-types/index)) — Путь к папке с локальными файлами.
* `is_blocked` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, заблокирована ли отправка локальных файлов на сервер.
* `error_count` ([UInt64](/ru/reference/data-types/index)) — Количество ошибок.
* `data_files` ([UInt64](/ru/reference/data-types/index)) — Количество локальных файлов в папке.
* `data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер сжатых данных в локальных файлах, в байтах.
* `broken_data_files` ([UInt64](/ru/reference/data-types/index)) — Количество файлов, помеченных как повреждённые (из-за ошибки).
* `broken_data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер сжатых данных в повреждённых файлах, в байтах.
* `last_exception` ([String](/ru/reference/data-types/index)) — Текстовое сообщение о последнем возникшем исключении (если оно было).
* `last_exception_time` ([DateTime](/ru/reference/data-types/index)) — Время возникновения последнего исключения.

{/*AUTOGENERATED_END*/}
