<div id="description">
  ## Описание
</div>

Содержит информацию о параметрах сборки сервера ClickHouse.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Название параметра сборки.
* `value` ([String](/ru/reference/data-types/index)) — Значение параметра сборки.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql
SELECT * FROM system.build_options LIMIT 5
```

```text
┌─name─────────────┬─value─┐
│ USE_BROTLI       │ 1     │
│ USE_BZIP2        │ 1     │
│ USE_CAPNP        │ 1     │
│ USE_CASSANDRA    │ 1     │
│ USE_DATASKETCHES │ 1     │
└──────────────────┴───────┘
```
