<div id="description">
  ## Описание
</div>

Эта таблица содержит размерные метрики, которые можно вычислить мгновенно и экспортировать в формате Prometheus. Она всегда актуальна.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ru/reference/data-types/index)) — Имя метрики.
* `value` ([Float64](/ru/reference/data-types/index)) — Значение метрики.
* `description` ([String](/ru/reference/data-types/index)) — Описание метрики.
* `labels` ([Map(String, String)](/ru/reference/data-types/index)) — Метки метрики.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

С помощью такого запроса можно экспортировать все размерные метрики в формате Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'gauge' AS type
FROM system.dimensional_metrics
FORMAT Prometheus
```

<div id="metric_descriptions">
  ## Описания метрик
</div>

<div id="merge_failures">
  ### merge_failures
</div>

Общее количество неудачных слияний с момента запуска.

<div id="startup_scripts_failure_reason">
  ### startup_scripts_failure_reason
</div>

Указывает на сбои скриптов запуска по типу ошибки. При сбое скрипта запуска принимает значение 1 и помечается именем ошибки.

<div id="merge_tree_parts">
  ### merge_tree_parts
</div>

Количество частей данных MergeTree с метками состояния части, типа части и признака того, является ли она частью проекции.
