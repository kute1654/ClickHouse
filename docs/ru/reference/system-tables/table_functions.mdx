  ## Описание
</div>

Содержит список всех доступных табличных функций с их описанием.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя табличной функции.
* `description` ([String](/ru/reference/data-types/index)) — Краткое описание табличной функции.
* `allow_readonly` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, может ли пользователь с правами только для чтения использовать эту функцию.
