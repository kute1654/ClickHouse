<div id="description">
  ## Описание
</div>

Содержит записи журнала с информацией о различных операциях объектного хранилища, таких как загрузка и удаление.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события.
* `event_time_microseconds` ([DateTime64(6)](/ru/reference/data-types/datetime64)) — Время события с точностью до микросекунд.
* `event_type` ([Enum8(&#39;Upload&#39; = 1, &#39;Delete&#39; = 2, &#39;MultiPartUploadCreate&#39; = 3, &#39;MultiPartUploadWrite&#39; = 4, &#39;MultiPartUploadComplete&#39; = 5, &#39;MultiPartUploadAbort&#39; = 6, &#39;Read&#39; = 7)](/ru/reference/data-types/enum)) — Тип события. Возможные значения: &#39;Upload&#39;, &#39;Delete&#39;, &#39;MultiPartUploadCreate&#39;, &#39;MultiPartUploadWrite&#39;, &#39;MultiPartUploadComplete&#39;, &#39;MultiPartUploadAbort&#39;, &#39;Read&#39;
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса, связанного с событием, если применимо.
* `thread_id` ([UInt64](/ru/reference/data-types/int-uint)) — Идентификатор потока, выполняющего операцию.
* `thread_name` ([String](/ru/reference/data-types/string)) — Имя потока, выполняющего операцию.
* `disk_name` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя соответствующего диска.
* `bucket` ([String](/ru/reference/data-types/string)) — Имя бакета.
* `remote_path` ([String](/ru/reference/data-types/string)) — Путь к удалённому ресурсу.
* `local_path` ([String](/ru/reference/data-types/string)) — Путь к файлу метаданных в локальной системе, который ссылается на удалённый ресурс.
* `data_size` ([UInt64](/ru/reference/data-types/int-uint)) — Размер данных, связанных с событием загрузки.
* `elapsed_microseconds` ([UInt64](/ru/reference/data-types/int-uint)) — Время, затраченное на операцию, в микросекундах.
* `error_code` ([Int32](/ru/reference/data-types/int-uint)) — Код ошибки операции. 0, если ошибки не было.
* `error` ([String](/ru/reference/data-types/string)) — Сообщение об ошибке, связанное с событием, если применимо.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

Предположим, что в объектное хранилище загружается файл и при этом записывается событие:

```sql
SELECT * FROM system.blob_storage_log WHERE query_id = '7afe0450-504d-4e4b-9a80-cd9826047972' ORDER BY event_date, event_time_microseconds \G
```

```text
Row 1:
──────
hostname:                clickhouse.eu-central1.internal
event_date:              2023-10-31
event_time:              2023-10-31 16:03:40
event_time_microseconds: 2023-10-31 16:03:40.481437
event_type:              Upload
query_id:                7afe0450-504d-4e4b-9a80-cd9826047972
thread_id:               2381740
disk_name:               disk_s3
bucket:                  bucket1
remote_path:             rrr/kxo/tbnqtrghgtnxkzgtcrlutwuslgawe
local_path:              store/654/6549e8b3-d753-4447-8047-d462df6e6dbe/tmp_insert_all_1_1_0/checksums.txt
data_size:               259
error:
```

В этом примере операция загрузки была связана с запросом `INSERT` с идентификатором `7afe0450-504d-4e4b-9a80-cd9826047972`. Локальный файл метаданных `store/654/6549e8b3-d753-4447-8047-d462df6e6dbe/tmp_insert_all_1_1_0/checksums.txt` указывает на удалённый путь `rrr/kxo/tbnqtrghgtnxkzgtcrlutwuslgawe` в бакете `bucket1` на диске `disk_s3`; его размер составляет 259 байт.
