<div id="description">
  ## Описание
</div>

Содержит сведения о столбцах всех таблиц.

Эту таблицу можно использовать, чтобы получить сведения, аналогичные результату запроса [DESCRIBE TABLE](/ru/reference/statements/describe-table), но сразу для нескольких таблиц.

Столбцы из [временных таблиц](/ru/reference/statements/create/table#temporary-tables) видны в `system.columns` только в том сеансе, в котором они были созданы. Они отображаются с пустым полем `database`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `name` ([String](/ru/reference/data-types/index)) — Имя столбца.
* `type` ([String](/ru/reference/data-types/index)) — Тип столбца.
* `position` ([UInt64](/ru/reference/data-types/index)) — Порядковый номер столбца в таблице, начиная с 1.
* `default_kind` ([String](/ru/reference/data-types/index)) — Тип выражения (DEFAULT, MATERIALIZED, ALIAS) для значения по умолчанию или пустая строка, если он не задан.
* `default_expression` ([String](/ru/reference/data-types/index)) — Выражение для значения по умолчанию или пустая строка, если оно не задано.
* `data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер сжатых данных в байтах.
* `data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер несжатых данных в байтах.
* `marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер marks в байтах.
* `comment` ([String](/ru/reference/data-types/index)) — Комментарий к столбцу или пустая строка, если он не задан.
* `is_in_partition_key` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, входит ли столбец в выражение партиционирования.
* `is_in_sorting_key` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, входит ли столбец в выражение ключа сортировки.
* `is_in_primary_key` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, входит ли столбец в выражение первичного ключа.
* `is_in_sampling_key` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, входит ли столбец в выражение ключа выборки.
* `compression_codec` ([String](/ru/reference/data-types/index)) — Имя кодека сжатия.
* `character_octet_length` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимальная длина в байтах для двоичных, символьных, текстовых данных и изображений. В ClickHouse имеет смысл только для типа данных FixedString. В противном случае возвращается значение NULL.
* `numeric_precision` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Точность приближённых числовых данных, точных числовых данных, целочисленных данных или денежных данных. В ClickHouse это разрядность в битах для целочисленных типов и точность Decimal для Decimal types. В противном случае возвращается значение NULL.
* `numeric_precision_radix` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Основание системы счисления для точности приближённых числовых данных, точных числовых данных, целочисленных данных или денежных данных. В ClickHouse это 2 для целочисленных типов и 10 для Decimal types. В противном случае возвращается значение NULL.
* `numeric_scale` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Масштаб приближённых числовых данных, точных числовых данных, целочисленных данных или денежных данных. В ClickHouse имеет смысл только для Decimal types. В противном случае возвращается значение NULL.
* `datetime_precision` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Десятичная точность типа данных DateTime64. Для других типов данных возвращается значение NULL.
* `serialization_hint` ([Nullable(String)](/ru/reference/data-types/index)) — Подсказка для столбца по выбору сериализации при вставках на основе статистики.
* `statistics` ([String](/ru/reference/data-types/index)) — Типы статистики, созданные для этого столбца.

{/*AUTOGENERATED_END*/}
