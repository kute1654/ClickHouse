<div id="description">
  ## Описание
</div>

Содержит все активные роли в данный момент, включая текущую роль текущего пользователя и роли, предоставленные текущей роли.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `role_name` ([String](/ru/reference/data-types/index)) — Имя роли.
* `with_admin_option` ([UInt8](/ru/reference/data-types/index)) — 1, если у роли есть привилегия ADMIN OPTION.
* `is_current` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли `enabled_role` текущей ролью текущего пользователя.
* `is_default` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли `enabled_role` ролью по умолчанию.

{/*AUTOGENERATED_END*/}
