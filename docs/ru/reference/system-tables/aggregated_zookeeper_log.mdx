<div id="description">
  ## Описание
</div>

Эта таблица содержит агрегированную статистику по операциям ZooKeeper (например, количество операций, среднюю задержку, ошибки), сгруппированную по `(session_id, parent_path, operation, component, is_subrequest)` и периодически сбрасываемую на диск.

В отличие от [system.zookeeper&#95;log](/ru/reference/system-tables/zookeeper_log), которая фиксирует в журнале каждый отдельный запрос и ответ, эта таблица агрегирует операции по группам, что делает её значительно менее ресурсоёмкой и, следовательно, более подходящей для промышленных рабочих нагрузок.

Операции, входящие в батч `Multi` или `MultiRead`, отслеживаются отдельно в столбце `is_subrequest`. У подзапросов задержка равна нулю, поскольку суммарная задержка относится к охватывающей операции `Multi`/`MultiRead`.
