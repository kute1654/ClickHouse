<div id="description">
  ## Описание
</div>

Таблица `system.background_schedule_pool_log` создаётся только в том случае, если указана настройка сервера [background&#95;schedule&#95;pool&#95;log](/ru/reference/settings/server-settings/settings#background_schedule_pool_log).

Эта таблица содержит историю выполнения задач в фоновом пуле планировщика. Фоновые пулы планировщика используются для выполнения периодических задач, таких как распределённые отправки, flush буфера и операции брокера сообщений.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события.
* `event_time_microseconds` ([DateTime64(6)](/ru/reference/data-types/datetime64)) — Время события с точностью до микросекунд.
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса, связанного с фоновой задачей.
* `database` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя базы данных.
* `table` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя таблицы.
* `table_uuid` ([UUID](/ru/reference/data-types/uuid)) — UUID таблицы, к которой относится фоновая задача.
* `log_name` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя фоновой задачи.
* `duration_ms` ([UInt64](/ru/reference/data-types/int-uint)) — Продолжительность выполнения задачи в миллисекундах.
* `error` ([UInt16](/ru/reference/data-types/int-uint)) — Код возникшего исключения.
* `exception` ([String](/ru/reference/data-types/string)) — Текст сообщения об ошибке.

{/*AUTOGENERATED_END*/}

Таблица `system.background_schedule_pool_log` создаётся после первого выполнения фоновой задачи.
