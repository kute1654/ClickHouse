<div id="description">
  ## Описание
</div>

Содержит информацию о доступных сертификатах и источниках их получения.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `version` ([Int32](/ru/reference/data-types/index)) — Версия сертификата. Значения: 0 для v1, 1 для v2, 2 для v3.
* `serial_number` ([Nullable(String)](/ru/reference/data-types/index)) — Серийный номер сертификата, присвоенный центром сертификации.
* `signature_algo` ([Nullable(String)](/ru/reference/data-types/index)) — Алгоритм подписи — алгоритм, который центр сертификации использует для подписи этого сертификата.
* `issuer` ([Nullable(String)](/ru/reference/data-types/index)) — Издатель — уникальный идентификатор центра сертификации, выдавшего этот сертификат.
* `not_before` ([Nullable(String)](/ru/reference/data-types/index)) — Начало временного интервала, в течение которого этот сертификат действителен.
* `not_after` ([Nullable(String)](/ru/reference/data-types/index)) — Конец временного интервала, в течение которого этот сертификат действителен.
* `subject` ([Nullable(String)](/ru/reference/data-types/index)) — Субъект — идентификатор владельца открытого ключа.
* `pkey_algo` ([Nullable(String)](/ru/reference/data-types/index)) — Алгоритм открытого ключа определяет, с каким алгоритмом можно использовать открытый ключ.
* `path` ([String](/ru/reference/data-types/index)) — Путь к файлу или каталогу, содержащему этот сертификат.
* `default` ([UInt8](/ru/reference/data-types/index)) — Сертификат находится в расположении сертификатов по умолчанию.
* `protocol` ([String](/ru/reference/data-types/index)) — Имя протокола, если сертификат получен из TLS-конфигурации для этого протокола, иначе пусто.

{/*AUTOGENERATED_END*/}
