<div id="description">
  ## Описание
</div>

Содержит информацию о существующих индексах пропуска данных во всех таблицах.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `name` ([String](/ru/reference/data-types/index)) — Имя индекса.
* `type` ([String](/ru/reference/data-types/index)) — Тип индекса.
* `type_full` ([String](/ru/reference/data-types/index)) — Выражение типа индекса из оператора CREATE.
* `expr` ([String](/ru/reference/data-types/index)) — Выражение для вычисления индекса.
* `creation` ([Enum8(&#39;Explicit&#39; = 0, &#39;Implicit&#39; = 1)](/ru/reference/data-types/index)) — Был ли индекс создан неявно (через add&#95;minmax&#95;index&#95;for&#95;numeric&#95;columns или аналогичный механизм)
* `granularity` ([UInt64](/ru/reference/data-types/index)) — Количество гранул в блоке.
* `data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер сжатых данных в байтах.
* `data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер распакованных данных в байтах.
* `marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер меток в байтах.

{/*AUTOGENERATED_END*/}
