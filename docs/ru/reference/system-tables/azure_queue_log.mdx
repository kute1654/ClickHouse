Содержит записи журнала с информацией о файлах, обработанных движком AzureQueue.

Эту таблицу можно в любой момент безопасно очистить командой TRUNCATE или удалить командой DROP.

Столбцы:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события при записи этой строки Log
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события при записи этой строки Log
* `database` ([String](/ru/reference/data-types/string)) — Имя базы данных, в которой находится таблица очереди (`S3Queue` или `AzureQueue`).
* `table` ([String](/ru/reference/data-types/string)) — Имя таблицы очереди (`S3Queue` или `AzureQueue`).
* `uuid` ([String](/ru/reference/data-types/string)) — UUID таблицы очереди (`S3Queue` или `AzureQueue`).
* `file_name` ([String](/ru/reference/data-types/string)) — Имя обрабатываемого файла.
* `rows_processed` ([UInt64](/ru/reference/data-types/int-uint)) — Количество обработанных строк.
* `status` ([Enum8(&#39;Processed&#39; = 0, &#39;Failed&#39; = 1)](/ru/reference/data-types/enum)) — Статус обрабатываемого файла.
* `processing_start_time` ([Nullable(DateTime)](/ru/reference/data-types/nullable)) — Время начала обработки файла.
* `processing_end_time` ([Nullable(DateTime)](/ru/reference/data-types/nullable)) — Время окончания обработки файла.
* `exception` ([String](/ru/reference/data-types/string)) — Сообщение об исключении, если оно возникло.
* `commit_id` ([UInt64](/ru/reference/data-types/int-uint)) — Идентификатор транзакции, в которой был зафиксирован этот файл.
* `commit_time` ([DateTime](/ru/reference/data-types/datetime)) — Время фиксации файла в Keeper (как ошибочного или обработанного).
* `transaction_start_time` ([DateTime](/ru/reference/data-types/datetime)) — Время начала всей транзакции обработки.
* `get_object_time_ms` ([UInt64](/ru/reference/data-types/int-uint)) — Время, потребовавшееся для поиска объекта в Объектном хранилище.
