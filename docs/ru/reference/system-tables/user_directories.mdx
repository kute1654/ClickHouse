<div id="description">
  ## Описание
</div>

Содержит сведения о настроенных пользовательских каталогах — каталогах в файловой системе, из которых сервер ClickHouse может читать предоставленные пользователем данные.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя каталога.
* `type` ([String](/ru/reference/data-types/index)) — Тип хранилища доступа, например `users.xml`, `replicated`, `memory` и т. д.
* `params` ([String](/ru/reference/data-types/index)) — JSON с параметрами хранилища доступа.
* `precedence` ([UInt64](/ru/reference/data-types/index)) — Порядок, в котором этот каталог объявлен в конфигурации. Этот же порядок используется, когда ClickHouse пытается найти пользователя или роль.

{/*AUTOGENERATED_END*/}
