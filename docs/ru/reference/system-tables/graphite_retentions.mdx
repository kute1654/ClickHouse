<div id="description">
  ## Описание
</div>

Содержит сведения о параметрах [graphite&#95;rollup](/ru/reference/settings/server-settings/settings#graphite), используемых в таблицах с движками [*GraphiteMergeTree](/ru/reference/engines/table-engines/mergetree-family/graphitemergetree).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `config_name` ([String](/ru/reference/data-types/index)) — Имя параметра graphite&#95;rollup.
* `rule_type` ([String](/ru/reference/data-types/index)) — Тип правила. Возможные значения: RuleTypeAll = 0 — по умолчанию, с regex, совместимо со старой scheme; RuleTypePlain = 1 — обычные метрики, с regex, совместимо со старой scheme; RuleTypeTagged = 2 — метрики с tag, с regex, совместимо со старой scheme; RuleTypeTagList = 3 — метрики с tag, с regex (преобразуется в RuleTypeTagged из строки вида &#39;retention=10min ; env=(staging|prod)&#39;)
* `regexp` ([String](/ru/reference/data-types/index)) — Шаблон регулярного выражения для имени метрики.
* `function` ([String](/ru/reference/data-types/index)) — Имя агрегирующей функции.
* `age` ([UInt64](/ru/reference/data-types/index)) — Минимальный возраст данных в секундах.
* `precision` ([UInt64](/ru/reference/data-types/index)) — Точность определения возраста данных в секундах.
* `priority` ([UInt16](/ru/reference/data-types/index)) — Приоритет шаблона.
* `is_default` ([UInt8](/ru/reference/data-types/index)) — Используется ли шаблон по умолчанию.
* `Tables.database` ([Array(String)](/ru/reference/data-types/index)) — Массив имен таблиц базы данных, использующих параметр `config_name`.
* `Tables.table` ([Array(String)](/ru/reference/data-types/index)) — Массив имен таблиц, использующих параметр `config_name`.
