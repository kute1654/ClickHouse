<div id="description">
  ## Описание
</div>

Содержит сведения о задачах из очередей репликации, хранящихся в ClickHouse Keeper или ZooKeeper, для таблиц семейства `ReplicatedMergeTree`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `replica_name` ([String](/ru/reference/data-types/index)) — Имя реплики в ClickHouse Keeper. У разных реплик одной и той же таблицы разные имена.
* `position` ([UInt32](/ru/reference/data-types/index)) — Позиция задачи в очереди.
* `node_name` ([String](/ru/reference/data-types/index)) — Имя узла в ClickHouse Keeper.
* `type` ([String](/ru/reference/data-types/index)) — Тип задачи в очереди, один из: • GET&#95;PART — Получить часть с другой реплики, • ATTACH&#95;PART — Присоединить часть, возможно, из собственной реплики (если она найдена в папке detached). Это можно рассматривать как GET&#95;PART с некоторыми оптимизациями, поскольку они почти идентичны, • MERGE&#95;PARTS — Объединить части, • DROP&#95;RANGE — Удалить части в указанной партиции в указанном числовом диапазоне. • CLEAR&#95;COLUMN — ПРИМЕЧАНИЕ: Устарело. Удалить конкретный столбец из указанной партиции. • CLEAR&#95;INDEX — ПРИМЕЧАНИЕ: Устарело. Удалить конкретный индекс из указанной партиции. • REPLACE&#95;RANGE — Удалить определённый диапазон частей и заменить его новыми. • MUTATE&#95;PART — Применить к части одну или несколько мутаций. • ALTER&#95;METADATA — Применить изменение ALTER в соответствии с глобальными путями /metadata и /columns.
* `create_time` ([DateTime](/ru/reference/data-types/index)) — Дата и время отправки задачи на выполнение.
* `required_quorum` ([UInt32](/ru/reference/data-types/index)) — Количество реплик, ожидающих завершения задачи с подтверждением выполнения. Этот столбец актуален только для задачи GET&#95;PARTS.
* `source_replica` ([String](/ru/reference/data-types/index)) — Имя исходной реплики.
* `new_part_name` ([String](/ru/reference/data-types/index)) — Имя новой части.
* `parts_to_merge` ([Array(String)](/ru/reference/data-types/index)) — Имена частей для слияния или обновления.
* `is_detach` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, находится ли задача DETACH&#95;PARTS в очереди.
* `is_currently_executing` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, выполняется ли конкретная задача в данный момент.
* `num_tries` ([UInt32](/ru/reference/data-types/index)) — Количество неудачных попыток выполнить задачу.
* `last_exception` ([String](/ru/reference/data-types/index)) — Текстовое сообщение о последней возникшей ошибке (если она была).
* `last_exception_time` ([DateTime](/ru/reference/data-types/index)) — Дата и время возникновения последней ошибки.
* `last_attempt_time` ([DateTime](/ru/reference/data-types/index)) — Дата и время последней попытки выполнить задачу.
* `num_postponed` ([UInt32](/ru/reference/data-types/index)) — Количество случаев, когда действие откладывалось.
* `postpone_reason` ([String](/ru/reference/data-types/index)) — Причина, по которой задача была отложена.
* `last_postpone_time` ([DateTime](/ru/reference/data-types/index)) — Дата и время, когда задача была отложена в последний раз.
* `merge_type` ([String](/ru/reference/data-types/index)) — Тип текущего слияния. Пусто, если это мутация.

{/*AUTOGENERATED_END*/}
