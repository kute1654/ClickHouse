<div id="description">
  ## Описание
</div>

Эту системную таблицу можно использовать, чтобы получить общее представление об использовании памяти и ProfileEvents пользователей.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `user` ([String](/ru/reference/data-types/index)) — Имя пользователя.
* `memory_usage` ([Int64](/ru/reference/data-types/index)) — Суммарный объём оперативной памяти, используемой всеми процессами пользователя. Может не включать некоторые типы выделенной памяти. См. настройку max&#95;memory&#95;usage.
* `peak_memory_usage` ([Int64](/ru/reference/data-types/index)) — Пиковое использование памяти пользователем. Может быть сброшено, если для этого пользователя не выполняются запросы.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/ru/reference/data-types/index)) — Сводка по ProfileEvents, измеряющим различные метрики, относящиеся к пользователю. Их описание можно найти в таблице system.events

{/*AUTOGENERATED_END*/}
