<div id="description">
  ## Описание
</div>

Содержит список поддерживаемых сервером движков баз данных.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Название движка базы данных.
* `description` ([String](/ru/reference/data-types/index)) — Общее описание того, что делает этот движок базы данных.
* `syntax` ([String](/ru/reference/data-types/index)) — Как движок базы данных указывается в предложении ENGINE запроса CREATE DATABASE.
* `examples` ([String](/ru/reference/data-types/index)) — Примеры использования.
* `introduced_in` ([String](/ru/reference/data-types/index)) — Версия ClickHouse, в которой этот движок базы данных был впервые представлен, в формате major.minor.
* `related` ([Array(String)](/ru/reference/data-types/index)) — Названия связанных движков баз данных.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql title="Query"
SELECT name
FROM system.database_engines
WHERE name IN ('Atomic', 'Ordinary')
ORDER BY name
```

│ Atomic   │
