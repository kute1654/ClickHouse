<div id="description">
  ## Описание
</div>

Содержит сведения о трассировках стека при фатальных ошибках. По умолчанию эта таблица в базе данных отсутствует и создается только при возникновении фатальных ошибок.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста, на котором произошёл сбой.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата сбоя.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время сбоя.
* `timestamp_ns` ([UInt64](/ru/reference/data-types/int-uint)) — Временная метка события в наносекундах.
* `signal` ([Int32](/ru/reference/data-types/int-uint)) — Номер сигнала.
* `signal_code` ([Int32](/ru/reference/data-types/int-uint)) — Код сигнала. Даёт дополнительную информацию о причине сигнала.
* `thread_id` ([UInt64](/ru/reference/data-types/int-uint)) — Идентификатор потока.
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса.
* `query` ([String](/ru/reference/data-types/string)) — Текст запроса, который выполнялся в момент сбоя.
* `trace` ([Array(UInt64)](/ru/reference/data-types/array)) — Трассировка стека в момент сбоя. Каждый элемент — адрес виртуальной памяти внутри процесса сервера ClickHouse.
* `trace_full` ([Array(String)](/ru/reference/data-types/array)) — Трассировка стека в момент сбоя. Каждый элемент содержит вызванный метод внутри процесса сервера ClickHouse.
* `fault_address` ([Nullable(UInt64)](/ru/reference/data-types/nullable)) — Адрес памяти, вызвавший сбой.
* `fault_access_type` ([String](/ru/reference/data-types/string)) — Тип обращения к памяти, вызвавший сбой (например, &#39;read&#39;, &#39;write&#39;).
* `signal_description` ([String](/ru/reference/data-types/string)) — Понятное человеку описание на основе signal&#95;code (например, &#39;Address not mapped to object&#39;).
* `current_exception_trace_full` ([Array(String)](/ru/reference/data-types/array)) — Текущая трассировка стека исключения с символизированными кадрами (только для сбоев std::terminate).
* `version` ([String](/ru/reference/data-types/string)) — Версия сервера ClickHouse.
* `revision` ([UInt32](/ru/reference/data-types/int-uint)) — Ревизия сервера ClickHouse.
* `build_id` ([String](/ru/reference/data-types/string)) — BuildID, сгенерированный компилятором.
* `git_hash` ([String](/ru/reference/data-types/string)) — Хеш коммита Git в исходном коде ClickHouse.
* `architecture` ([String](/ru/reference/data-types/string)) — Архитектура CPU (например, x86&#95;64, aarch64).

{/*AUTOGENERATED_END*/}
