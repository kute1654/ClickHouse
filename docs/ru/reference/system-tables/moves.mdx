<div id="description">
  ## Описание
</div>

Таблица содержит сведения о выполняемых [перемещениях частей данных](/ru/reference/statements/alter/partition#move-partitionpart) таблиц [MergeTree](/ru/reference/engines/table-engines/mergetree-family/mergetree). Каждое такое перемещение представлено одной строкой.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы, содержащей перемещаемую часть данных.
* `elapsed` ([Float64](/ru/reference/data-types/index)) — Время (в секундах), прошедшее с начала перемещения части данных.
* `target_disk_name` ([String](/ru/reference/data-types/index)) — Имя диска, на который перемещается часть данных.
* `target_disk_path` ([String](/ru/reference/data-types/index)) — Путь к точке монтирования диска в файловой системе.
* `part_name` ([String](/ru/reference/data-types/index)) — Имя перемещаемой части данных.
* `part_size` ([UInt64](/ru/reference/data-types/index)) — Размер части данных.
* `thread_id` ([UInt64](/ru/reference/data-types/index)) — Идентификатор потока, выполняющего перемещение.

{/*AUTOGENERATED_END*/}
