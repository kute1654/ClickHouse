<div id="description">
  ## Описание
</div>

Таблица `system.part_log` создается только в том случае, если указана настройка сервера [part&#95;log](/ru/reference/settings/server-settings/settings#part_log).

Эта таблица содержит информацию о событиях, происходивших с [частями данных](/ru/reference/engines/table-engines/mergetree-family/custom-partitioning-key) в таблицах семейства [MergeTree](/ru/reference/engines/table-engines/mergetree-family/mergetree), например о добавлении данных или их слиянии.

Таблица `system.part_log` содержит следующие столбцы:

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — имя хоста сервера, выполняющего запрос.
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса INSERT, в результате которого была создана эта часть данных.
* `event_type` ([Enum8(&#39;NewPart&#39; = 1, &#39;MergeParts&#39; = 2, &#39;DownloadPart&#39; = 3, &#39;RemovePart&#39; = 4, &#39;MutatePart&#39; = 5, &#39;MovePart&#39; = 6, &#39;MergePartsStart&#39; = 7, &#39;MutatePartStart&#39; = 8)](/ru/reference/data-types/enum)) — Тип события, произошедшего с частью данных. Может принимать одно из следующих значений: NewPart — вставка новой части данных, MergePartsStart — слияние частей данных началось, MergeParts — слияние частей данных завершилось, DownloadPart — загрузка части данных, RemovePart — удаление или отсоединение части данных с помощью [DETACH PARTITION](/ru/reference/statements/alter/partition#detach-partitionpart). MutatePartStart — мутация части данных началась, MutatePart — мутация части данных завершилась, MovePart — перемещение части данных с одного диска на другой.
* `merge_reason` ([Enum8(&#39;NotAMerge&#39; = 1, &#39;RegularMerge&#39; = 2, &#39;TTLDeleteMerge&#39; = 3, &#39;TTLRecompressMerge&#39; = 4, &#39;TTLDropMerge&#39; = 5)](/ru/reference/data-types/enum)) — Причина события типа MERGE&#95;PARTS. Может принимать одно из следующих значений: NotAMerge — текущее событие имеет тип, отличный от MERGE&#95;PARTS, RegularMerge — обычное слияние, TTLDeleteMerge, TTLDropMerge — очистка устаревших данных. TTLRecompressMerge — повторное сжатие части данных с помощью.
* `merge_algorithm` ([Enum8(&#39;Undecided&#39; = 0, &#39;Vertical&#39; = 1, &#39;Horizontal&#39; = 2)](/ru/reference/data-types/enum)) — Алгоритм слияния для события типа MERGE&#95;PARTS. Может принимать одно из следующих значений: Undecided, Horizontal, Vertical
* `event_date` ([Date](/ru/reference/data-types/date)) — дата события.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события.
* `event_time_microseconds` ([DateTime64(6)](/ru/reference/data-types/datetime64)) — Время события с точностью до микросекунд.
* `duration_ms` ([UInt64](/ru/reference/data-types/int-uint)) — Продолжительность этой операции.
* `database` ([String](/ru/reference/data-types/string)) — Имя базы данных, в которой находится эта часть данных.
* `table` ([String](/ru/reference/data-types/string)) — Имя таблицы, которой принадлежит часть данных.
* `table_uuid` ([UUID](/ru/reference/data-types/uuid)) — UUID таблицы, которой принадлежит часть данных.
* `part_name` ([String](/ru/reference/data-types/string)) — Название части данных.
* `partition_id` ([String](/ru/reference/data-types/string)) — идентификатор партиции, в которую была вставлена часть данных. Столбец принимает значение `all`, если партиционирование выполняется по `tuple()`.
* `partition` ([String](/ru/reference/data-types/string)) — Имя партиции.
* `part_type` ([String](/ru/reference/data-types/string)) — Тип части данных. Возможные значения: Wide и Compact.
* `part_storage_type` ([String](/ru/reference/data-types/string)) — Тип DataPartStorage. Возможные значения: Packed — все файлы хранятся в одном blob, Full — отдельный blob для каждого файла.
* `disk_name` ([String](/ru/reference/data-types/string)) — Имя диска, на котором расположена часть данных.
* `path_on_disk` ([String](/ru/reference/data-types/string)) — Абсолютный путь к папке с файлами части данных.
* `rows` ([UInt64](/ru/reference/data-types/int-uint)) — количество строк в части данных.
* `size_in_bytes` ([UInt64](/ru/reference/data-types/int-uint)) — Размер части данных на диске, в байтах.
* `merged_from` ([Array(String)](/ru/reference/data-types/array)) — Массив имён частей-источников, из которых была сформирована текущая часть.
* `bytes_uncompressed` ([UInt64](/ru/reference/data-types/int-uint)) — Размер результирующей части без сжатия в байтах.
* `read_rows` ([UInt64](/ru/reference/data-types/int-uint)) — Количество строк, прочитанных в ходе слияния.
* `read_bytes` ([UInt64](/ru/reference/data-types/int-uint)) — Количество байтов, считанных при слиянии.
* `peak_memory_usage` ([UInt64](/ru/reference/data-types/int-uint)) — Максимальный объём оперативной памяти, использованный во время слияния
* `deduplication_block_ids` ([Array(String)](/ru/reference/data-types/array)) — Массив идентификаторов блоков, используемый для дедупликации при вставке данной части.
* `error` ([UInt16](/ru/reference/data-types/int-uint)) — Код ошибки, соответствующей возникшему исключению.
* `exception` ([String](/ru/reference/data-types/string)) — Текст сообщения об ошибке.
* `mutation_ids` ([Array(String)](/ru/reference/data-types/array)) — массив идентификаторов мутаций, применённых к исходной части (merged&#95;from), для события типа MUTATE&#95;PART&#95;START и MUTATE&#95;PART.
* `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/ru/reference/data-types/map)) — Все события профилирования, собранные во время этой операции.
* `projections_duration_ms` ([Map(LowCardinality(String), UInt64)](/ru/reference/data-types/map)) — Длительность слияния/перестроения каждой проекции в миллисекундах.

**Псевдонимы:**

* `ProfileEvents.Names` — псевдоним `mapKeys(ProfileEvents)`.
* `ProfileEvents.Values` — псевдоним `mapValues(ProfileEvents)`.
* `name` — псевдоним `part_name`.

{/*AUTOGENERATED_END*/}

Таблица `system.part_log` создается после первой вставки данных в таблицу `MergeTree`.
