<div id="description">
  ## Описание
</div>

Содержит информацию о дисках, определённых в [конфигурации сервера](/ru/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя диска в конфигурации сервера.
* `path` ([String](/ru/reference/data-types/index)) — Путь к точке монтирования в файловой системе.
* `free_space` ([UInt64](/ru/reference/data-types/index)) — Объём свободного места на диске в байтах.
* `total_space` ([UInt64](/ru/reference/data-types/index)) — Объём диска в байтах.
* `unreserved_space` ([UInt64](/ru/reference/data-types/index)) — Свободное пространство, не занятое резервированиями (`free&#95;space` минус объём резервирований, используемых слияниями, вставками и другими выполняющимися в данный момент операциями записи на диск).
* `keep_free_space` ([UInt64](/ru/reference/data-types/index)) — Объём дискового пространства в байтах, который должен оставаться свободным. Задаётся параметром `keep&#95;free&#95;space&#95;bytes` в конфигурации диска.
* `type` ([String](/ru/reference/data-types/index)) — Тип диска, указывающий, где хранятся данные: в оперативной памяти, на локальном диске или в удалённом хранилище.
* `object_storage_type` ([String](/ru/reference/data-types/index)) — Тип объектного хранилища, если тип диска — `object&#95;storage`
* `metadata_type` ([String](/ru/reference/data-types/index)) — Тип хранилища метаданных, если тип диска — `object&#95;storage`
* `is_encrypted` ([UInt8](/ru/reference/data-types/index)) — Флаг, показывающий, шифрует ли этот диск хранимые данные.
* `is_read_only` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, что с этим диском можно выполнять только операции чтения.
* `is_write_once` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли диск диском однократной записи. Это означает, что он поддерживает BACKUP на этот диск, но не поддерживает INSERT в таблицу семейства MergeTree на этом диске.
* `is_remote` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, какие операции с этим диском требуют сетевого взаимодействия.
* `is_broken` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, повреждён ли диск. Повреждённые диски имеют 0 свободного места и не могут использоваться.
* `cache_path` ([String](/ru/reference/data-types/index)) — Путь к каталогу кэша на локальном диске, если диск поддерживает кэширование.

{/*AUTOGENERATED_END*/}
