<div id="description">
  ## Описание
</div>

Содержит историю значений памяти и метрик из таблицы `system.events` для отдельных запросов; эти данные периодически сбрасываются на диск.

После запуска запроса данные собираются через каждые `query_metric_log_interval` миллисекунд (по умолчанию — 1000
). Данные также собираются при завершении запроса, если он выполняется дольше, чем `query_metric_log_interval`.
