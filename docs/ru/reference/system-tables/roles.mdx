  ## Описание
</div>

Содержит сведения о настроенных [ролях](/ru/concepts/features/security/access-rights#role-management).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя роли.
* `id` ([UUID](/ru/reference/data-types/index)) — Идентификатор роли.
* `storage` ([String](/ru/reference/data-types/index)) — Путь к хранилищу ролей. Задаётся в параметре `access_control_path`.
