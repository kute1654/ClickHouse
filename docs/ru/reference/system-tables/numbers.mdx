<div id="description">
  ## Описание
</div>

Эта таблица содержит единственный столбец UInt64 с именем `number`, в котором содержатся почти все натуральные числа, начиная с нуля.

Эту таблицу можно использовать для тестирования или когда нужен полный перебор.

Чтение из этой таблицы не выполняется параллельно.
