<div id="description">
  ## Описание
</div>

Содержит фильтры для конкретной таблицы, а также список ролей и/или пользователей, для которых должна применяться эта ROW POLICY.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя ROW POLICY.
* `short_name` ([String](/ru/reference/data-types/index)) — Краткое имя ROW POLICY. Имена ROW POLICY являются составными, например: myfilter ON mydb.mytable. Здесь &#39;myfilter ON mydb.mytable&#39; — имя ROW POLICY, а &#39;myfilter&#39; — её краткое имя.
* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы. Пусто, если ROW POLICY задана для базы данных.
* `id` ([UUID](/ru/reference/data-types/index)) — Идентификатор ROW POLICY.
* `storage` ([String](/ru/reference/data-types/index)) — Имя каталога, в котором хранится ROW POLICY.
* `select_filter` ([Nullable(String)](/ru/reference/data-types/index)) — Выражение, используемое для фильтрации в запросах SELECT.
* `is_restrictive` ([UInt8](/ru/reference/data-types/index)) — Показывает, ограничивает ли ROW POLICY доступ к строкам. Значение: • 0 — ROW POLICY определена с предложением `AS PERMISSIVE`, • 1 — ROW POLICY определена с предложением AS RESTRICTIVE.
* `apply_to_all` ([UInt8](/ru/reference/data-types/index)) — Показывает, что ROW POLICY заданы для всех ролей и/или пользователей.
* `apply_to_list` ([Array(String)](/ru/reference/data-types/index)) — Список ролей и/или пользователей, к которым применяется ROW POLICY.
* `apply_to_except` ([Array(String)](/ru/reference/data-types/index)) — ROW POLICY применяется ко всем ролям и/или пользователям, кроме перечисленных.

{/*AUTOGENERATED_END*/}
