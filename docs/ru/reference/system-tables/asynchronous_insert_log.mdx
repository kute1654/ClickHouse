<div id="description">
  ## Описание
</div>

Содержит информацию об асинхронных вставках. Каждая запись представляет собой запрос на вставку, помещённый в буфер асинхронных вставок.

Чтобы включить логирование, настройте параметры в разделе [asynchronous&#95;insert&#95;log](/ru/reference/settings/server-settings/settings#asynchronous_insert_log).

Период сброса данных задаётся параметром `flush_interval_milliseconds` в разделе настроек сервера [asynchronous&#95;insert&#95;log](/ru/reference/settings/server-settings/settings#asynchronous_insert_log). Чтобы принудительно выполнить сброс, используйте запрос [SYSTEM FLUSH LOGS](/ru/reference/statements/system#flush-logs).

ClickHouse не удаляет данные из таблицы автоматически. Подробнее см. в разделе [Введение](/ru/reference/system-tables/overview#system-tables-introduction).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата, когда произошла асинхронная вставка.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Дата и время, когда асинхронная вставка завершилась.
* `event_time_microseconds` ([DateTime64(6)](/ru/reference/data-types/datetime64)) — Дата и время, когда асинхронная вставка завершилась, с точностью до микросекунд.
* `query` ([String](/ru/reference/data-types/string)) — Строка запроса.
* `database` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя базы данных, в которой находится таблица.
* `table` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя таблицы.
* `format` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя формата.
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор исходного запроса.
* `bytes` ([UInt64](/ru/reference/data-types/int-uint)) — Количество вставленных байтов.
* `rows` ([UInt64](/ru/reference/data-types/int-uint)) — Количество вставленных строк.
* `exception` ([String](/ru/reference/data-types/string)) — Сообщение об исключении.
* `status` ([Enum8(&#39;Ok&#39; = 0, &#39;ParsingError&#39; = 1, &#39;FlushError&#39; = 2)](/ru/reference/data-types/enum)) — Статус вставки. Значения: &#39;Ok&#39; = 0 — успешная вставка, &#39;ParsingError&#39; = 1 — исключение при разборе данных, &#39;FlushError&#39; = 2 — исключение при сбросе данных.
* `data_kind` ([Enum8(&#39;Parsed&#39; = 0, &#39;Preprocessed&#39; = 1)](/ru/reference/data-types/enum)) — Состояние данных. Значения: &#39;Parsed&#39; и &#39;Preprocessed&#39;.
* `flush_time` ([DateTime](/ru/reference/data-types/datetime)) — Дата и время, когда произошел сброс.
* `flush_time_microseconds` ([DateTime64(6)](/ru/reference/data-types/datetime64)) — Дата и время, когда произошел сброс, с точностью до микросекунд.
* `flush_query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса сброса.
* `timeout_milliseconds` ([UInt64](/ru/reference/data-types/int-uint)) — Адаптивный тайм-аут, рассчитанный для этой записи.

{/*AUTOGENERATED_END*/}
