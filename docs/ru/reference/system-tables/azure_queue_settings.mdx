<div id="description">
  ## Описание
</div>

Содержит сведения о настройках таблиц [AzureQueue](/ru/reference/engines/table-engines/integrations/azure-queue).
Доступно начиная с версии сервера `24.10`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — База данных таблицы с движком AzureQueue.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы с движком AzureQueue.
* `name` ([String](/ru/reference/data-types/index)) — Имя настройки.
* `value` ([String](/ru/reference/data-types/index)) — Значение настройки.
* `type` ([String](/ru/reference/data-types/index)) — Тип настройки (строковое значение, зависящее от реализации).
* `changed` ([UInt8](/ru/reference/data-types/index)) — 1, если настройка была явно задана в конфигурации или явно изменена.
* `description` ([String](/ru/reference/data-types/index)) — Описание настройки.
* `alterable` ([UInt8](/ru/reference/data-types/index)) — Показывает, может ли текущий пользователь изменить настройку с помощью ALTER TABLE MODIFY SETTING: 0 — текущий пользователь не может изменить настройку, 1 — текущий пользователь может изменить настройку.

{/*AUTOGENERATED_END*/}
