<div id="description">
  ## Описание
</div>

Содержит список [учётных записей пользователей](/ru/concepts/features/security/access-rights#user-account-management), настроенных на сервере.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя пользователя.
* `id` ([UUID](/ru/reference/data-types/index)) — Идентификатор пользователя.
* `storage` ([String](/ru/reference/data-types/index)) — Путь к хранилищу пользователей. Настраивается параметром access&#95;control&#95;path.
* `auth_type` ([Array(Enum8(&#39;no&#95;password&#39; = 0, &#39;plaintext&#95;password&#39; = 1, &#39;sha256&#95;password&#39; = 2, &#39;double&#95;sha1&#95;password&#39; = 3, &#39;ldap&#39; = 4, &#39;kerberos&#39; = 5, &#39;ssl&#95;certificate&#39; = 6, &#39;bcrypt&#95;password&#39; = 7, &#39;ssh&#95;key&#39; = 8, &#39;http&#39; = 9, &#39;jwt&#39; = 10, &#39;scram&#95;sha256&#95;password&#39; = 11, &#39;no&#95;authentication&#39; = 12))](/ru/reference/data-types/index)) — Показывает типы аутентификации. Существует несколько способов идентификации пользователя: без пароля, с паролем в открытом виде, с паролем, закодированным с помощью SHA256, с паролем, дважды закодированным с помощью SHA-1, или с паролем, закодированным с помощью bcrypt.
* `auth_params` ([Array(String)](/ru/reference/data-types/index)) — Параметры аутентификации в формате JSON в зависимости от auth&#95;type.
* `valid_until` ([Array(DateTime)](/ru/reference/data-types/index)) — Дата и время истечения срока действия учетных данных пользователя.
* `host_ip` ([Array(String)](/ru/reference/data-types/index)) — IP-адреса хостов, которым разрешено подключаться к серверу ClickHouse.
* `host_names` ([Array(String)](/ru/reference/data-types/index)) — Имена хостов, которым разрешено подключаться к серверу ClickHouse.
* `host_names_regexp` ([Array(String)](/ru/reference/data-types/index)) — Регулярное выражение для имен хостов, которым разрешено подключаться к серверу ClickHouse.
* `host_names_like` ([Array(String)](/ru/reference/data-types/index)) — Имена хостов, которым разрешено подключаться к серверу ClickHouse, заданные с помощью предиката LIKE.
* `default_roles_all` ([UInt8](/ru/reference/data-types/index)) — Показывает, что все предоставленные роли заданы для пользователя по умолчанию.
* `default_roles_list` ([Array(String)](/ru/reference/data-types/index)) — Список предоставленных ролей, заданных по умолчанию.
* `default_roles_except` ([Array(String)](/ru/reference/data-types/index)) — Все предоставленные роли заданы по умолчанию, кроме перечисленных.
* `grantees_any` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, может ли пользователь с любой grant option выдавать ее кому угодно.
* `grantees_list` ([Array(String)](/ru/reference/data-types/index)) — Список пользователей или ролей, которым этот пользователь может выдавать grant options.
* `grantees_except` ([Array(String)](/ru/reference/data-types/index)) — Список пользователей или ролей, которым этому пользователю запрещено выдавать grant options.
* `default_database` ([String](/ru/reference/data-types/index)) — Имя базы данных по умолчанию для этого пользователя.
