<div id="description">
  ## Описание
</div>

Содержит список транзакций и сведения об их состоянии.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `tid` ([Tuple(UInt64, UInt64, UUID)](/ru/reference/data-types/tuple)) — Идентификатор транзакции.
* `tid_hash` ([UInt64](/ru/reference/data-types/int-uint)) — Хэш идентификатора.
* `elapsed` ([Float64](/ru/reference/data-types/float)) — Время обработки транзакции.
* `is_readonly` ([UInt8](/ru/reference/data-types/int-uint)) — Флаг, показывающий, выполняла ли транзакция какие-либо операции записи.
* `state` ([Enum8(&#39;RUNNING&#39; = 0, &#39;COMMITTING&#39; = 1, &#39;COMMITTED&#39; = 2, &#39;ROLLED&#95;BACK&#39; = 3)](/ru/reference/data-types/enum)) — Состояние транзакции. Возможные значения: RUNNING, COMMITTING, COMMITTED, ROLLED&#95;BACK.

{/*AUTOGENERATED_END*/}
