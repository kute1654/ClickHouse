<div id="description">
  ## Описание
</div>

Содержит сведения о частях, которые в данный момент перемещаются между сегментами, и о ходе этого перемещения.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных, в которой выполняется перемещение.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы, в которой выполняется перемещение.
* `task_name` ([String](/ru/reference/data-types/index)) — Имя задачи перемещения.
* `task_uuid` ([UUID](/ru/reference/data-types/index)) — Идентификатор задачи перемещения.
* `create_time` ([дата и время](/ru/reference/data-types/index)) — Время создания задачи.
* `part_name` ([String](/ru/reference/data-types/index)) — Имя части, находящейся в процессе перемещения.
* `part_uuid` ([UUID](/ru/reference/data-types/index)) — UUID части, находящейся в процессе перемещения.
* `to_shard` ([String](/ru/reference/data-types/index)) — Имя сегмента назначения.
* `dst_part_name` ([String](/ru/reference/data-types/index)) — Имя результирующей части.
* `update_time` ([дата и время](/ru/reference/data-types/index)) — Время последнего обновления.
* `state` ([String](/ru/reference/data-types/index)) — Текущее состояние перемещения.
* `rollback` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, был ли выполнен откат операции.
* `num_tries` ([UInt32](/ru/reference/data-types/index)) — Количество попыток завершить операцию.
* `last_exception` ([String](/ru/reference/data-types/index)) — Имя последнего исключения, если оно есть.

{/*AUTOGENERATED_END*/}
