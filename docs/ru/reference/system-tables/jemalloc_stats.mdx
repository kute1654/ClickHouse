<div id="description">
  ## Описание
</div>

Возвращает статистику jemalloc в виде одной строки с одним столбцом. Соответствует команде SYSTEM JEMALLOC STATS.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `stats` ([String](/ru/reference/data-types/index)) — вывод статистики JEMalloc, полученный из malloc&#95;stats&#95;print.
