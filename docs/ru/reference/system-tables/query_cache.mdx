<div id="description">
  ## Описание
</div>

Показывает содержимое [кэша запросов](/ru/concepts/features/performance/caches/query-cache).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ru/reference/data-types/index)) — Строка запроса.
* `query_id` ([String](/ru/reference/data-types/index)) — Идентификатор запроса.
* `is_subquery` ([UInt8](/ru/reference/data-types/index)) — Является ли запись кэша запросов результатом подзапроса.
* `result_size` ([UInt64](/ru/reference/data-types/index)) — Размер записи кэша запросов.
* `tag` ([LowCardinality(String)](/ru/reference/data-types/index)) — Тег записи кэша запросов.
* `stale` ([UInt8](/ru/reference/data-types/index)) — Является ли запись кэша запросов устаревшей.
* `shared` ([UInt8](/ru/reference/data-types/index)) — Является ли запись кэша запросов общей для нескольких пользователей.
* `compressed` ([UInt8](/ru/reference/data-types/index)) — Сжата ли запись кэша запросов.
* `expires_at` ([DateTime](/ru/reference/data-types/index)) — Когда запись кэша запросов становится устаревшей.
* `key_hash` ([UInt64](/ru/reference/data-types/index)) — Хэш строки запроса, используемый в качестве ключа для поиска записей кэша запросов.

{/*AUTOGENERATED_END*/}
