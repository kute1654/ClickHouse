<div id="description">
  ## Описание
</div>

Содержит информацию о [ресурсах](/ru/concepts/features/configuration/server-config/workload-scheduling#workload_entity_storage), расположенных на локальном сервере. Таблица содержит по одной строке для каждого ресурса.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя ресурса.
* `read_disks` ([Array(String)](/ru/reference/data-types/index)) — Список имён дисков, использующих этот ресурс для операций чтения.
* `write_disks` ([Array(String)](/ru/reference/data-types/index)) — Список имён дисков, использующих этот ресурс для операций записи.
* `unit` ([String](/ru/reference/data-types/index)) — Единица измерения стоимости ресурса.
* `create_query` ([String](/ru/reference/data-types/index)) — CREATE-запрос для ресурса.

{/*AUTOGENERATED_END*/}

Пример:

```sql
SELECT *
FROM system.resources
FORMAT Vertical
```

```text
Row 1:
──────
name:         io_read
read_disks:   ['s3']
write_disks:  []
create_query: CREATE RESOURCE io_read (READ DISK s3)

Row 2:
──────
name:         io_write
read_disks:   []
write_disks:  ['s3']
create_query: CREATE RESOURCE io_write (WRITE DISK s3)
```
