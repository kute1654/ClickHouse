<div id="description">
  ## Описание
</div>

Содержит сведения об обычных и агрегатных функциях.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя функции.
* `is_aggregate` ([UInt8](/ru/reference/data-types/index)) — Является ли функция агрегатной.
* `case_insensitive` ([UInt8](/ru/reference/data-types/index)) — Можно ли использовать имя функции без учета регистра.
* `alias_to` ([String](/ru/reference/data-types/index)) — Исходное имя функции, если данное имя является псевдонимом.
* `create_query` ([String](/ru/reference/data-types/index)) — Устарело.
* `origin` ([Enum8(&#39;System&#39; = 0, &#39;SQLUserDefined&#39; = 1, &#39;ExecutableUserDefined&#39; = 2, &#39;WasmUserDefined&#39; = 3)](/ru/reference/data-types/index)) — Устарело.
* `description` ([String](/ru/reference/data-types/index)) — Общее описание назначения функции.
* `syntax` ([String](/ru/reference/data-types/index)) — Сигнатура функции.
* `arguments` ([String](/ru/reference/data-types/index)) — Аргументы функции.
* `parameters` ([String](/ru/reference/data-types/index)) — Параметры функции (только для агрегатной функции).
* `returned_value` ([String](/ru/reference/data-types/index)) — Что возвращает функция.
* `examples` ([String](/ru/reference/data-types/index)) — Пример использования.
* `introduced_in` ([String](/ru/reference/data-types/index)) — Версия ClickHouse, в которой функция была впервые добавлена.
* `categories` ([String](/ru/reference/data-types/index)) — Категория функции.
* `deterministic` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Возвращает ли функция один и тот же результат для одних и тех же аргументов. NULL, если это неизвестно (например, для агрегатных или пользовательских функций).
* `higher_order` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Является ли функция функцией высшего порядка, то есть принимает ли она как минимум одно лямбда-выражение в качестве аргумента (например, arrayMap, arrayFilter, mapApply). NULL, если это неизвестно.

{/*AUTOGENERATED_END*/}
