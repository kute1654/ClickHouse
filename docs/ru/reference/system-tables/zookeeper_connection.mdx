<div id="description">
  ## Описание
</div>

Эта таблица отсутствует, если ZooKeeper не настроен. Таблица &#39;system.zookeeper&#95;connection&#39; показывает текущие соединения с ZooKeeper (включая дополнительные экземпляры ZooKeeper). Каждая строка содержит информацию об одном соединении.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/string)) — Имя кластера ZooKeeper.
* `host` ([String](/ru/reference/data-types/string)) — Имя хоста/IP-адрес узла ZooKeeper, к которому подключился ClickHouse.
* `port` ([UIn16](/ru/reference/data-types/int-uint)) — Порт узла ZooKeeper, к которому подключился ClickHouse.
* `index` ([Nullable(UInt8)](/ru/reference/data-types/int-uint)) — Индекс узла ZooKeeper, к которому подключился ClickHouse. Индекс берется из конфигурации ZooKeeper. Если соединение не установлено, этот столбец имеет значение NULL.
* `connected_time` ([DateTime](/ru/reference/data-types/datetime)) — Время установления соединения.
* `session_uptime_elapsed_seconds` ([UInt64](/ru/reference/data-types/int-uint)) — Количество секунд, прошедших с момента установления соединения.
* `is_expired` ([UInt8](/ru/reference/data-types/int-uint)) — Истек ли срок действия текущего соединения.
* `keeper_api_version` ([UInt8](/ru/reference/data-types/int-uint)) — Версия API ClickHouse Keeper.
* `client_id` ([Int64](/ru/reference/data-types/int-uint)) — Идентификатор сеанса соединения.
* `xid` ([Int64](/ru/reference/data-types/int-uint)) — XID текущего сеанса.
* `enabled_feature_flags` ([Array(Enum16)](/ru/reference/data-types/array)) — Включенные feature flags. Применимо только к ClickHouse Keeper. Возможные значения: `FILTERED_LIST`, `MULTI_READ`, `CHECK_NOT_EXISTS`, `CREATE_IF_NOT_EXISTS`, `REMOVE_RECURSIVE`.
* `availability_zone` ([String](/ru/reference/data-types/string)) — Зона доступности.
* `session_timeout_ms` ([UInt64](/ru/reference/data-types/int-uint)) — Тайм-аут сеанса (в миллисекундах).
* `last_zxid_seen` ([Int64](/ru/reference/data-types/int-uint)) — Последний zxid, который видел текущий сеанс.

{/*AUTOGENERATED_END*/}
