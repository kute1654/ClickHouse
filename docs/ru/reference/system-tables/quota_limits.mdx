<div id="description">
  ## Описание
</div>

Содержит информацию о максимальных значениях для всех интервалов всех квот. Одной квоте может соответствовать любое число строк или ни одной.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ru/reference/data-types/index)) — Имя квоты.
* `duration` ([UInt32](/ru/reference/data-types/index)) — Длительность временного интервала для расчета потребления ресурсов, в секундах.
* `is_randomized_interval` ([UInt8](/ru/reference/data-types/index)) — Булево значение. Показывает, является ли интервал случайным. Если интервал не случайный, он всегда начинается в одно и то же время. Например, интервал в 1 минуту всегда начинается на целой минуте (то есть он может начаться в 11:20:00, но никогда не начинается в 11:20:01), а интервал в один день всегда начинается в полночь UTC. Если интервал случайный, самый первый интервал начинается в случайный момент времени, а последующие начинаются один за другим. Значения: 0 — интервал не случайный, 1 — интервал случайный.
* `max_queries` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов всех типов, которое можно выполнить.
* `max_query_selects` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов SELECT, которое можно выполнить.
* `max_query_inserts` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов INSERT, которое можно выполнить.
* `max_errors` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов, завершившихся ошибкой, в течение указанного периода времени.
* `max_result_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое общее количество строк в результирующем наборе всех запросов в течение указанного периода времени.
* `max_result_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое общее количество байтов в результирующем наборе всех запросов в течение указанного периода времени.
* `max_read_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимальное количество строк, которое можно прочитать при выполнении всех запросов в течение указанного периода времени.
* `max_read_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимальное количество байтов, которое можно прочитать при выполнении всех запросов в течение указанного периода времени.
* `max_execution_time` ([Nullable(Float64)](/ru/reference/data-types/index)) — Максимально допустимое время выполнения всех запросов в течение указанного периода времени (в наносекундах)
* `max_written_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимальное количество байтов, которое можно записать при выполнении всех запросов в течение указанного периода времени.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество последовательных неудачных попыток аутентификации в течение указанного периода времени.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество выполнений любого отдельного нормализованного запроса в течение указанного периода времени.
