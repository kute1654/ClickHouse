<div id="description">
  ## Описание
</div>

Содержит список часовых поясов, поддерживаемых сервером ClickHouse. Этот список может различаться в зависимости от версии ClickHouse.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `time_zone` ([String](/ru/reference/data-types/index)) — Список поддерживаемых часовых поясов.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql
SELECT * FROM system.time_zones LIMIT 10
```
