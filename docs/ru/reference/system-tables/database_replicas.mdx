<div id="description">
  ## Описание
</div>

Содержит информацию о репликах каждой базы данных Replicated.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных Replicated.
* `is_readonly` ([UInt8](/ru/reference/data-types/index)) — Находится ли реплика базы данных в режиме только для чтения.
* `max_log_ptr` ([Int32](/ru/reference/data-types/index)) — Максимальный номер записи в журнале общей активности.
* `replica_name` ([String](/ru/reference/data-types/index)) — Имя реплики в ClickHouse Keeper.
* `replica_path` ([String](/ru/reference/data-types/index)) — Путь к данным реплики в ClickHouse Keeper.
* `zookeeper_path` ([String](/ru/reference/data-types/index)) — Путь к данным базы данных в ClickHouse Keeper.
* `shard_name` ([String](/ru/reference/data-types/index)) — Имя сегмента в кластере.
* `log_ptr` ([Int32](/ru/reference/data-types/index)) — Максимальный номер записи в журнале общей активности, которую реплика скопировала в свою очередь выполнения, плюс один.
* `total_replicas` ([UInt32](/ru/reference/data-types/index)) — Общее число известных реплик этой базы данных.
* `zookeeper_exception` ([String](/ru/reference/data-types/index)) — Сообщение о последнем исключении, полученное, если ошибка возникла при получении информации из ClickHouse Keeper.
* `is_session_expired` ([UInt8](/ru/reference/data-types/index)) — Сеанс с ClickHouse Keeper истёк. По сути это то же самое, что и `is_readonly`.

{/*AUTOGENERATED_END*/}
