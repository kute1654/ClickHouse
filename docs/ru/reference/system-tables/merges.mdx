<div id="description">
  ## Описание
</div>

Содержит информацию о выполняющихся в данный момент слияниях и мутациях частей в таблицах семейства MergeTree.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных, в которой находится таблица.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `elapsed` ([Float64](/ru/reference/data-types/index)) — Время (в секундах), прошедшее с начала слияния.
* `progress` ([Float64](/ru/reference/data-types/index)) — Доля выполненной работы, от 0 до 1.
* `num_parts` ([UInt64](/ru/reference/data-types/index)) — Количество частей, которые будут слиты.
* `source_part_names` ([Array(String)](/ru/reference/data-types/index)) — Список имён исходных частей.
* `result_part_name` ([String](/ru/reference/data-types/index)) — Имя части, которая будет сформирована в результате слияния.
* `source_part_paths` ([Array(String)](/ru/reference/data-types/index)) — Список путей для каждой исходной части.
* `result_part_path` ([String](/ru/reference/data-types/index)) — Путь части, которая будет сформирована в результате слияния.
* `partition_id` ([String](/ru/reference/data-types/index)) — Идентификатор партиции, в которой происходит слияние.
* `partition` ([String](/ru/reference/data-types/index)) — Имя партиции
* `is_mutation` ([UInt8](/ru/reference/data-types/index)) — 1, если этот процесс является мутацией части.
* `total_size_bytes_compressed` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в слитых фрагментах.
* `total_size_bytes_uncompressed` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в слитых фрагментах.
* `total_size_marks` ([UInt64](/ru/reference/data-types/index)) — Общее количество меток в слитых частях.
* `bytes_read_uncompressed` ([UInt64](/ru/reference/data-types/index)) — Количество прочитанных байтов в несжатом виде.
* `rows_read` ([UInt64](/ru/reference/data-types/index)) — Количество прочитанных строк.
* `bytes_written_uncompressed` ([UInt64](/ru/reference/data-types/index)) — Количество записанных байтов в несжатом виде.
* `rows_written` ([UInt64](/ru/reference/data-types/index)) — Количество записанных строк.
* `columns_written` ([UInt64](/ru/reference/data-types/index)) — Количество записанных столбцов (для алгоритма Вертикального слияния).
* `memory_usage` ([UInt64](/ru/reference/data-types/index)) — Потребление памяти в процессе слияния.
* `thread_id` ([UInt64](/ru/reference/data-types/index)) — Идентификатор потока процесса слияния.
* `merge_type` ([String](/ru/reference/data-types/index)) — Тип текущего слияния. Пусто, если это мутация.
* `merge_algorithm` ([String](/ru/reference/data-types/index)) — Алгоритм, используемый в текущем слиянии. Пусто, если это мутация.
* `current_projection` ([String](/ru/reference/data-types/index)) — Имя проекции, которая в данный момент сливается или перестраивается. Пусто, если это не этап слияния проекций.
* `current_projection_progress` ([Float64](/ru/reference/data-types/index)) — Прогресс текущего слияния проекции от 0 до 1.
* `current_projection_parts_merging` ([UInt64](/ru/reference/data-types/index)) — Количество частей проекции, которые сейчас сливаются.
* `current_projection_parts_remaining` ([UInt64](/ru/reference/data-types/index)) — Количество частей проекции, которые осталось слить для текущей проекции.
* `projections_completed` ([Array(String)](/ru/reference/data-types/index)) — Список проекций, которые уже были слиты или перестроены.
* `projections_remaining` ([Array(String)](/ru/reference/data-types/index)) — Список проекций, которые ещё нужно слить или перестроить.
