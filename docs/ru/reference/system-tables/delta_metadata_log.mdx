<div id="purpose">
  ## Описание
</div>

Таблица `system.delta_lake_metadata_log` регистрирует события доступа к метаданным и их парсинга для таблиц Delta Lake, которые читает ClickHouse. Она содержит подробную информацию о каждом файле метаданных, что полезно для отладки, аудита и понимания эволюции структуры таблиц Delta.

Эта таблица регистрирует каждый файл метаданных, прочитанный из таблиц Delta Lake. Она помогает пользователям отслеживать, как ClickHouse интерпретирует метаданные таблиц Delta, и диагностировать проблемы, связанные с эволюцией схемы, разрешением снимков или планированием запросов.

<Note>
  Эта таблица в первую очередь предназначена для отладки.
</Note>

<div id="controlling-log-verbosity">
  ### Управление уровнем подробности журналирования
</div>

Вы можете указать, какие события метаданных записывать в журнал, с помощью настройки [`delta_lake_log_metadata`](/ru/reference/settings/session-settings#delta_lake_log_metadata).

Чтобы записывать в журнал все метаданные, используемые в текущем запросе:

```sql
SELECT * FROM my_delta_table SETTINGS delta_lake_log_metadata = 1;

SYSTEM FLUSH LOGS delta_lake_metadata_log;

SELECT *
FROM system.delta_lake_metadata_log
WHERE query_id = '{previous_query_id}';
```
