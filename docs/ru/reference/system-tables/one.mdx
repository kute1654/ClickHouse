<div id="description">
  ## Описание
</div>

Эта таблица содержит одну строку с одним столбцом `dummy` типа UInt8 со значением 0.

Эта таблица используется, если в запросе `SELECT` не указано предложение `FROM`.

Это аналог таблицы `DUAL`, которая есть в других СУБД.
