<div id="description">
  ## Описание
</div>

Эта таблица содержит метрики гистограммы, которые можно мгновенно вычислять и экспортировать в формате Prometheus. Данные в ней всегда актуальны. Заменяет устаревшую `system.latency_log`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ru/reference/data-types/index)) — Название метрики.
* `value` ([Float64](/ru/reference/data-types/index)) — Значение метрики.
* `description` ([String](/ru/reference/data-types/index)) — Описание метрики.
* `labels` ([Map(String, String)](/ru/reference/data-types/index)) — Метки метрик.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

Вы можете использовать такой запрос, чтобы экспортировать все метрики гистограммы в формате Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```
