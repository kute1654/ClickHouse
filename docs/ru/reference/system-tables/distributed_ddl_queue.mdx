<div id="description">
  ## Описание
</div>

Содержит информацию о [распределённых DDL-запросах (предложение ON CLUSTER)](/ru/reference/statements/distributed-ddl), выполненных в кластере.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `entry` ([String](/ru/reference/data-types/index)) — Query id.
* `entry_version` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Версия записи.
* `initiator_host` ([Nullable(String)](/ru/reference/data-types/index)) — Хост, с которого была инициирована операция DDL.
* `initiator_port` ([Nullable(UInt16)](/ru/reference/data-types/index)) — Порт, используемый инициатором.
* `cluster` ([String](/ru/reference/data-types/index)) — Имя кластера; пусто, если не определено.
* `query` ([String](/ru/reference/data-types/index)) — Выполненный запрос.
* `settings` ([Map(String, String)](/ru/reference/data-types/index)) — Настройки, использованные в операции DDL.
* `query_create_time` ([DateTime](/ru/reference/data-types/index)) — Время создания запроса.
* `host` ([Nullable(String)](/ru/reference/data-types/index)) — Имя хоста.
* `port` ([Nullable(UInt16)](/ru/reference/data-types/index)) — Порт хоста.
* `status` ([Nullable(Enum8(&#39;Inactive&#39; = 0, &#39;Active&#39; = 1, &#39;Finished&#39; = 2, &#39;Removing&#39; = 3, &#39;Unknown&#39; = 4))](/ru/reference/data-types/index)) — Статус запроса.
* `exception_code` ([Nullable(UInt16)](/ru/reference/data-types/index)) — Код исключения.
* `exception_text` ([Nullable(String)](/ru/reference/data-types/index)) — Текст исключения.
* `query_finish_time` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Время завершения запроса.
* `query_duration_ms` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Длительность выполнения запроса (в миллисекундах).

{/*AUTOGENERATED_END*/}
