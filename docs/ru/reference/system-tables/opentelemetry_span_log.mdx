<div id="description">
  ## Описание
</div>

Содержит информацию о [спанах трассировки](https://opentracing.io/docs/overview/spans/) выполненных запросов.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста, на котором был зафиксирован этот спан.
* `trace_id` ([UUID](/ru/reference/data-types/uuid)) — Идентификатор трассы для выполненного запроса.
* `span_id` ([UInt64](/ru/reference/data-types/int-uint)) — Идентификатор спана трассировки.
* `parent_span_id` ([UInt64](/ru/reference/data-types/int-uint)) — Идентификатор родительского спана трассировки.
* `operation_name` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя операции.
* `kind` ([Enum8(&#39;INTERNAL&#39; = 0, &#39;SERVER&#39; = 1, &#39;CLIENT&#39; = 2, &#39;PRODUCER&#39; = 3, &#39;CONSUMER&#39; = 4)](/ru/reference/data-types/enum)) — SpanKind спана. INTERNAL — указывает, что спан представляет внутреннюю операцию в приложении. SERVER — указывает, что спан охватывает обработку на стороне сервера для синхронного RPC или другого удалённого запроса. CLIENT — указывает, что спан описывает запрос к удалённому сервису. PRODUCER — указывает, что спан описывает инициатора асинхронного запроса. Этот родительский спан часто завершается раньше соответствующего дочернего спана CONSUMER, возможно, даже до того, как дочерний спан начнётся. CONSUMER - указывает, что спан описывает дочерний спан асинхронного запроса PRODUCER.
* `start_time_us` ([UInt64](/ru/reference/data-types/int-uint)) — Время начала спана трассировки (в микросекундах).
* `finish_time_us` ([UInt64](/ru/reference/data-types/int-uint)) — Время завершения спана трассировки (в микросекундах).
* `finish_date` ([Date](/ru/reference/data-types/date)) — Дата завершения спана трассировки.
* `status_code` ([Enum8(&#39;UNSET&#39; = 0, &#39;OK&#39; = 1, &#39;ERROR&#39; = 2)](/ru/reference/data-types/enum)) — Код состояния спана.
* `status_message` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Сообщение об ошибке.
* `attribute` ([Map(LowCardinality(String), String)](/ru/reference/data-types/map)) — Атрибуты, зависящие от спана трассировки. Они заполняются в соответствии с рекомендациями стандарта OpenTelemetry.

**Псевдонимы:**

* `attribute.names` — Псевдоним для `mapKeys(attribute)`.
* `attribute.values` — Псевдоним для `mapValues(attribute)`.
