<div id="description">
  ## Описание
</div>

Содержит информацию об асинхронных вставках, ожидающих выполнения, в очереди.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `query` ([String](/ru/reference/data-types/index)) — Текст запроса.
* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `format` ([String](/ru/reference/data-types/index)) — Имя формата.
* `first_update` ([DateTime64(6)](/ru/reference/data-types/index)) — Время первой вставки с точностью до микросекунд.
* `total_bytes` ([UInt64](/ru/reference/data-types/index)) — Общее количество байтов, ожидающих в очереди.
* `entries.query_id` ([Array(String)](/ru/reference/data-types/index)) — Массив идентификаторов запросов на вставку, ожидающих в очереди.
* `entries.bytes` ([Array(UInt64)](/ru/reference/data-types/index)) — Массив значений в байтах для каждого запроса на вставку, ожидающего в очереди.

{/*AUTOGENERATED_END*/}
