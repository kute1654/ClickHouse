<div id="purpose">
  ## Описание
</div>

Содержит информацию об отсоединённых частях таблиц [MergeTree](/ru/reference/engines/table-engines/mergetree-family/mergetree). Столбец `reason` указывает, по какой причине часть была отсоединена.

Для частей, отсоединённых пользователем, причина не указана. Такие части можно подключить командой [ALTER TABLE ATTACH PARTITION|PART](/ru/reference/statements/alter/partition#attach-partitionpart).

Описание остальных столбцов см. в [system.parts](/ru/reference/system-tables/parts).

Если имя части некорректно, значения некоторых столбцов могут быть `NULL`. Такие части можно удалить командой [ALTER TABLE DROP DETACHED PART](/ru/reference/statements/alter/partition#drop-detached-partitionpart).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных, к которой относится эта часть.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы, к которой относится эта часть.
* `partition_id` ([Nullable(String)](/ru/reference/data-types/index)) — Идентификатор партиции, к которой относится эта часть.
* `name` ([String](/ru/reference/data-types/index)) — Имя части.
* `bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер всех файлов части данных в байтах.
* `modification_time` ([DateTime](/ru/reference/data-types/index)) — Время изменения каталога с частью данных. Обычно оно соответствует моменту выполнения detach.
* `disk` ([String](/ru/reference/data-types/index)) — Имя диска, на котором хранится эта часть данных.
* `path` ([String](/ru/reference/data-types/index)) — Путь на диске к файлу этой части данных.
* `reason` ([Nullable(String)](/ru/reference/data-types/index)) — Причина, по которой эта часть была отсоединена.
* `min_block_number` ([Nullable(Int64)](/ru/reference/data-types/index)) — Минимальный номер частей данных, из которых после слияния состоит текущая часть.
* `max_block_number` ([Nullable(Int64)](/ru/reference/data-types/index)) — Максимальный номер частей данных, из которых после слияния состоит текущая часть.
* `level` ([Nullable(UInt32)](/ru/reference/data-types/index)) — Глубина дерева слияния. Ноль означает, что текущая часть была создана вставкой, а не слиянием других частей.
