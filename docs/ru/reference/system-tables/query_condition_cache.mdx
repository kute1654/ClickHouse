<div id="description">
  ## Описание
</div>

Показывает содержимое [кэша условий запроса](/ru/concepts/features/performance/caches/query-condition-cache).

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `key_hash` ([UInt128](/ru/reference/data-types/index)) — Хеш (table&#95;uuid, part&#95;name, condition&#95;hash).
* `entry_size` ([UInt64](/ru/reference/data-types/index)) — Размер записи в байтах.
* `matching_marks` ([String](/ru/reference/data-types/index)) — Соответствующие метки.

{/*AUTOGENERATED_END*/}
