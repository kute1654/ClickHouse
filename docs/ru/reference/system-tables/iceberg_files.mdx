<div id="description">
  ## Описание
</div>

Эта системная таблица содержит метаданные отдельных файлов для таблиц Iceberg в ClickHouse: по одной строке для каждого файла данных или файла удаления, на который ссылается текущий снимок каждой таблицы. Она будет пустой, если в ClickHouse нет ни одной таблицы Iceberg.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `snapshot_id` ([Int64](/ru/reference/data-types/index)) — ID снимка, в котором был добавлен файл.
* `content` ([Enum8(&#39;DATA&#39; = 0, &#39;POSITION&#95;DELETE&#39; = 1, &#39;EQUALITY&#95;DELETE&#39; = 2)](/ru/reference/data-types/index)) — Тип содержимого файла.
* `file_path` ([String](/ru/reference/data-types/index)) — Итоговый путь к файлу в хранилище.
* `file_format` ([String](/ru/reference/data-types/index)) — Формат файла, например &#39;PARQUET&#39;.
* `record_count` ([Int64](/ru/reference/data-types/index)) — Количество записей в файле.
* `file_size_in_bytes` ([Int64](/ru/reference/data-types/index)) — Размер файла в байтах.
* `partition` ([String](/ru/reference/data-types/index)) — Текстовое представление кортежа партиции.
* `schema_id` ([Int32](/ru/reference/data-types/index)) — ID схемы, определённый для этой записи манифеста.
* `sequence_number` ([Int64](/ru/reference/data-types/index)) — Номер последовательности, определённый для записи манифеста (всегда 0 для формата v1).
* `sort_order_id` ([Nullable(Int32)](/ru/reference/data-types/index)) — ID порядка сортировки файла, если указан.
* `null_value_counts` ([Map(Int32, Int64)](/ru/reference/data-types/index)) — Количество NULL-значений по каждому столбцу (id столбца -&gt; количество).
* `column_sizes` ([Map(Int32, Int64)](/ru/reference/data-types/index)) — Размер на диске в байтах для каждого столбца (id столбца -&gt; байты).
* `value_counts` ([Map(Int32, Int64)](/ru/reference/data-types/index)) — Общее количество значений по каждому столбцу (id столбца -&gt; количество).
* `equality_ids` ([Array(Int32)](/ru/reference/data-types/index)) — ID полей равенства для файлов удаления по равенству (пусто для файлов, не являющихся файлами удаления по равенству).
