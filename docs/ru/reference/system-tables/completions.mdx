<div id="description">
  ## Описание
</div>

Содержит список токенов автодополнения.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `word` ([String](/ru/reference/data-types/index)) — токен автодополнения.
* `context` ([String](/ru/reference/data-types/index)) — тип сущности токена (например, table).
* `belongs` ([Nullable(String)](/ru/reference/data-types/index)) — сущность, к которой относится этот токен (например, имя базы данных-владельца).
