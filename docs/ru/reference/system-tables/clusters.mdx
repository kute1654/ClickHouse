<div id="description">
  ## Описание
</div>

Содержит информацию о кластерах, доступных в файле конфигурации, и входящих в них серверах.

<div id="Columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `cluster` ([String](/ru/reference/data-types/index)) — Имя кластера.
* `shard_num` ([UInt32](/ru/reference/data-types/index)) — Номер сегмента в кластере, начиная с 1.
* `shard_name` ([String](/ru/reference/data-types/index)) — Имя сегмента в кластере.
* `shard_weight` ([UInt32](/ru/reference/data-types/index)) — Относительный вес сегмента при записи данных.
* `internal_replication` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли этот хост частью группы узлов, способной самостоятельно реплицировать данные.
* `replica_num` ([UInt32](/ru/reference/data-types/index)) — Номер реплики в сегменте, начиная с 1.
* `host_name` ([String](/ru/reference/data-types/index)) — Имя хоста, как указано в конфигурации.
* `host_address` ([String](/ru/reference/data-types/index)) — IP-адрес хоста, полученный через DNS.
* `port` ([UInt16](/ru/reference/data-types/index)) — Порт для подключения к серверу.
* `is_local` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, является ли хост локальным.
* `user` ([String](/ru/reference/data-types/index)) — Имя пользователя для подключения к серверу.
* `default_database` ([String](/ru/reference/data-types/index)) — Имя базы данных по умолчанию.
* `errors_count` ([UInt32](/ru/reference/data-types/index)) — Количество случаев, когда этому хосту не удалось обратиться к реплике.
* `slowdowns_count` ([UInt32](/ru/reference/data-types/index)) — Количество замедлений, которые привели к смене реплики при установлении соединения с хеджированными запросами.
* `estimated_recovery_time` ([UInt32](/ru/reference/data-types/index)) — Количество секунд, оставшихся до обнуления счётчика ошибок реплики, после чего она снова будет считаться работающей нормально.
* `database_shard_name` ([String](/ru/reference/data-types/index)) — Имя сегмента базы данных `Replicated` (для кластеров, принадлежащих базе данных `Replicated`).
* `database_replica_name` ([String](/ru/reference/data-types/index)) — Имя реплики базы данных `Replicated` (для кластеров, принадлежащих базе данных `Replicated`).
* `is_shared_catalog_cluster` ([UInt8](/ru/reference/data-types/index)) — Булево значение, указывающее, принадлежит ли кластер Shared Catalog.
* `is_active` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Статус реплики базы данных Replicated (для кластеров, принадлежащих базе данных Replicated): 1 означает &#39;реплика в сети&#39;, 0 означает &#39;реплика не в сети&#39;, NULL означает &#39;неизвестно&#39;.
* `unsynced_after_recovery` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Указывает, превышает ли отставание репликации у реплики базы данных Replicated значение `max&#95;replication&#95;lag&#95;to&#95;enqueue` после создания или восстановления реплики.
* `replication_lag` ([Nullable(UInt32)](/ru/reference/data-types/index)) — Отставание репликации реплики базы данных `Replicated` (для кластеров, принадлежащих базе данных Replicated).
* `recovery_time` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Время восстановления реплики базы данных `Replicated` (для кластеров, принадлежащих базе данных Replicated), в миллисекундах.

{/*AUTOGENERATED_END*/}
