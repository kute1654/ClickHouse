<div id="description">
  ## Описание
</div>

Содержит информацию о текущих фоновых загрузках.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `elapsed` ([Float64](/ru/reference/data-types/index)) — Время (в секундах), прошедшее с начала выполнения отображаемых в данный момент фоновых загрузок.
* `progress` ([Float64](/ru/reference/data-types/index)) — Доля выполненной работы в диапазоне от 0 до 1.
* `result_part_name` ([String](/ru/reference/data-types/index)) — Имя части, которая будет сформирована в результате выполняемых в данный момент фоновых загрузок.
* `result_part_path` ([String](/ru/reference/data-types/index)) — Абсолютный путь к части, которая будет сформирована в результате выполняемых в данный момент фоновых загрузок.
* `partition_id` ([String](/ru/reference/data-types/index)) — ID партиции.
* `total_size_bytes_compressed` ([UInt64](/ru/reference/data-types/index)) — Общий размер (в байтах) сжатых данных в результирующей части.
* `bytes_read_compressed` ([UInt64](/ru/reference/data-types/index)) — Количество сжатых байтов, прочитанных из результирующей части.
* `source_replica_path` ([String](/ru/reference/data-types/index)) — Абсолютный путь к исходной реплике.
* `source_replica_hostname` ([String](/ru/reference/data-types/index)) — Имя хоста исходной реплики.
* `source_replica_port` ([UInt16](/ru/reference/data-types/index)) — Номер порта исходной реплики.
* `interserver_scheme` ([String](/ru/reference/data-types/index)) — Название межсерверной схемы.
* `URI` ([String](/ru/reference/data-types/index)) — Унифицированный идентификатор ресурса.
* `to_detached` ([UInt8](/ru/reference/data-types/index)) — Этот флаг указывает, выполняется ли текущая фоновая загрузка с использованием выражения TO DETACHED.
* `thread_id` ([UInt64](/ru/reference/data-types/index)) — Идентификатор потока.

{/*AUTOGENERATED_END*/}
