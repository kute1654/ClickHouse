<div id="description">
  ## Описание
</div>

Содержит информацию о кодеках сжатия и шифрования.

Эту таблицу можно использовать для получения сведений о доступных кодеках сжатия и шифрования

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Название кодека.
* `method_byte` ([UInt8](/ru/reference/data-types/index)) — Байт, указывающий кодек в сжатом файле.
* `is_compression` ([UInt8](/ru/reference/data-types/index)) — Значение true, если этот кодек выполняет сжатие. В противном случае это может быть просто преобразование, помогающее сжатию.
* `is_generic_compression` ([UInt8](/ru/reference/data-types/index)) — Кодек представляет собой универсальный алгоритм сжатия, например lz4 или zstd.
* `is_encryption` ([UInt8](/ru/reference/data-types/index)) — Кодек выполняет шифрование.
* `is_timeseries_codec` ([UInt8](/ru/reference/data-types/index)) — Кодек предназначен для временных рядов с числами с плавающей запятой.
* `is_experimental` ([UInt8](/ru/reference/data-types/index)) — Кодек является экспериментальным.
* `description` ([String](/ru/reference/data-types/index)) — Общее описание кодека.

{/*AUTOGENERATED_END*/}

  ## Пример
</div>

SELECT * FROM system.codecs WHERE name='LZ4'
