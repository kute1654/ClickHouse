<div id="description">
  ## Описание
</div>

Содержит список всех доступных комбинаторов агрегатных функций, которые можно применять к агрегатным функциям, чтобы изменять их поведение.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Название комбинатора.
* `is_internal` ([UInt8](/ru/reference/data-types/index)) — Указывает, предназначен ли этот комбинатор только для внутреннего использования.
* `description` ([String](/ru/reference/data-types/index)) — Краткое описание назначения комбинатора.
* `syntax` ([String](/ru/reference/data-types/index)) — Как комбинатор применяется к имени агрегатной функции.
* `examples` ([String](/ru/reference/data-types/index)) — Примеры использования.
* `introduced_in` ([String](/ru/reference/data-types/index)) — Версия ClickHouse, в которой комбинатор впервые появился, в формате major.minor.
* `related` ([Array(String)](/ru/reference/data-types/index)) — Названия связанных комбинаторов.

{/*AUTOGENERATED_END*/}
