<div id="description">
  ## Описание
</div>

Содержит список всех настроек, специфичных для движка ReplicatedMergeTree, их текущих и значений по умолчанию, а также описания. Любую из них можно изменить в разделе SETTINGS в запросе CREATE.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя настройки.
* `value` ([String](/ru/reference/data-types/index)) — Значение настройки.
* `default` ([String](/ru/reference/data-types/index)) — Значение настройки по умолчанию.
* `changed` ([UInt8](/ru/reference/data-types/index)) — 1, если настройка была явно задана в конфигурации или явно изменена.
* `description` ([String](/ru/reference/data-types/index)) — Описание настройки.
* `min` ([Nullable(String)](/ru/reference/data-types/index)) — Минимальное значение настройки, если оно задано через ограничения. Если у настройки нет минимального значения, содержит NULL.
* `max` ([Nullable(String)](/ru/reference/data-types/index)) — Максимальное значение настройки, если оно задано через ограничения. Если у настройки нет максимального значения, содержит NULL.
* `disallowed_values` ([Array(String)](/ru/reference/data-types/index)) — Список недопустимых значений
* `readonly` ([UInt8](/ru/reference/data-types/index)) — Показывает, может ли текущий пользователь изменить настройку: 0 — текущий пользователь может изменить настройку, 1 — текущий пользователь не может изменить настройку.
* `type` ([String](/ru/reference/data-types/index)) — Тип настройки (строковое значение, зависящее от реализации).
* `is_obsolete` ([UInt8](/ru/reference/data-types/index)) — Показывает, является ли настройка устаревшей.
* `tier` ([Enum8(&#39;Production&#39; = 0, &#39;Obsolete&#39; = 4, &#39;Experimental&#39; = 8, &#39;Beta&#39; = 12)](/ru/reference/data-types/index)) —
  Уровень поддержки этой возможности. Возможности ClickHouse распределены по уровням, которые различаются в зависимости от текущего состояния их
  разработки и ожиданий от их использования:

- PRODUCTION: Возможность стабильна, безопасна в использовании и не вызывает проблем при взаимодействии с другими возможностями уровня PRODUCTION.
- BETA: Возможность стабильна и безопасна. Результат её совместного использования с другими возможностями неизвестен, и корректность не гарантируется. Приветствуются тестирование и отчёты.
- EXPERIMENTAL: Возможность находится в разработке. Предназначена только для разработчиков и энтузиастов ClickHouse. Возможность может как работать, так и не работать, и может быть удалена в любой момент.
- OBSOLETE: Больше не поддерживается. Либо уже удалена, либо будет удалена в будущих выпусках.
