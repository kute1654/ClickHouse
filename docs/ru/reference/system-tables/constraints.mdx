<div id="description">
  ## Описание
</div>

Содержит сведения о существующих ограничениях во всех таблицах.

Ограничения, заданные для [временных таблиц](/ru/reference/statements/create/table#temporary-tables), видны в `system.constraints` только в том сеансе, в котором они были созданы. Они отображаются с пустым полем `database`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `name` ([String](/ru/reference/data-types/index)) — Имя ограничения.
* `type` ([Enum8(&#39;CHECK&#39; = 0, &#39;ASSUME&#39; = 1)](/ru/reference/data-types/index)) — Тип ограничения.
* `expression` ([String](/ru/reference/data-types/index)) — Выражение ограничения.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql
SELECT * FROM system.constraints LIMIT 2 FORMAT Vertical;
```

```text
Row 1:
──────
database:    default
table:       hits
name:        check_hits
type:        CHECK
expression:  CounterID > 0

Row 2:
──────
database:    default
table:       hits
name:        assume_positive
type:        ASSUME
expression:  WatchID > 0
```
