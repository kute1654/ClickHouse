<div id="description">
  ## Описание
</div>

Содержит список всех доступных failpoint, зарегистрированных на сервере, а также их тип и сведения о том, включены ли они в данный момент.

Failpoint можно включать и отключать во время выполнения с помощью команд `SYSTEM ENABLE FAILPOINT` и `SYSTEM DISABLE FAILPOINT`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — Имя failpoint.
* `type` ([Enum8(&#39;once&#39; = 0, &#39;regular&#39; = 1, &#39;pauseable&#95;once&#39; = 2, &#39;pauseable&#39; = 3)](/ru/reference/data-types/index)) — Тип failpoint: &#39;once&#39; срабатывает один раз, после чего автоматически отключается; &#39;regular&#39; срабатывает каждый раз; &#39;pauseable&#95;once&#39; один раз приостанавливает выполнение; &#39;pauseable&#39; приостанавливает выполнение каждый раз, пока оно не будет возобновлено.
* `enabled` ([UInt8](/ru/reference/data-types/index)) — Указывает, включен ли failpoint в данный момент (1) или отключен (0).

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql title="Query"

SELECT * FROM system.fail_points WHERE enabled = 1
```
