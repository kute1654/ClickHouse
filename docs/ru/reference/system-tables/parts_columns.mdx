<div id="description">
  ## Описание
</div>

Содержит сведения о частях данных и столбцах таблиц [MergeTree](/ru/reference/engines/table-engines/mergetree-family/mergetree).
Каждая строка описывает одну часть данных.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `partition` ([String](/ru/reference/data-types/index)) — Имя партиции.
* `name` ([String](/ru/reference/data-types/index)) — Имя части данных.
* `uuid` ([UUID](/ru/reference/data-types/index)) — UUID части.
* `part_type` ([String](/ru/reference/data-types/index)) — Формат хранения части данных. Возможные значения: Wide — каждый столбец хранится в отдельном файле в файловой системе, Compact — все столбцы хранятся в одном файле в файловой системе.
* `active` ([UInt8](/ru/reference/data-types/index)) — Флаг, указывающий, активна ли часть данных. Если часть данных активна, она используется в таблице. В противном случае она удаляется. Неактивные части данных остаются после слияния.
* `marks` ([UInt64](/ru/reference/data-types/index)) — Количество отметок. Чтобы получить приблизительное количество строк в части данных, умножьте число отметок на гранулярность индекса (обычно 8192) (эта подсказка не работает для адаптивной гранулярности).
* `rows` ([UInt64](/ru/reference/data-types/index)) — Количество строк.
* `bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер всех файлов части данных в байтах.
* `data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в части данных. Все вспомогательные файлы (например, файлы с отметками) не включаются.
* `data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в части данных. Все вспомогательные файлы (например, файлы с отметками) не включаются.
* `marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер файла с отметками.
* `modification_time` ([DateTime](/ru/reference/data-types/index)) — Время изменения каталога с частью данных. Обычно это соответствует времени создания части данных.
* `remove_time` ([DateTime](/ru/reference/data-types/index)) — Время, когда часть данных стала неактивной.
* `refcount` ([UInt32](/ru/reference/data-types/index)) — Количество мест, где используется часть данных. Значение больше 2 указывает на то, что часть данных используется в запросах или слияниях.
* `min_date` ([Date](/ru/reference/data-types/index)) — Минимальное значение для столбца Date, если он включён в ключ партиционирования.
* `max_date` ([Date](/ru/reference/data-types/index)) — Максимальное значение для столбца Date, если он включён в ключ партиционирования.
* `min_time` ([DateTime](/ru/reference/data-types/index)) — Минимальное значение для столбца DateTime, если он включён в ключ партиционирования.
* `max_time` ([DateTime](/ru/reference/data-types/index)) — Максимальное значение для столбца DateTime, если он включён в ключ партиционирования.
* `partition_id` ([String](/ru/reference/data-types/index)) — Идентификатор партиции.
* `min_block_number` ([Int64](/ru/reference/data-types/index)) — Минимальный номер частей данных, образующих текущую часть после слияния.
* `max_block_number` ([Int64](/ru/reference/data-types/index)) — Максимальный номер частей данных, образующих текущую часть после слияния.
* `level` ([UInt32](/ru/reference/data-types/index)) — Глубина дерева слияния. Ноль означает, что текущая часть была создана вставкой, а не слиянием других частей.
* `data_version` ([UInt64](/ru/reference/data-types/index)) — Номер, используемый для определения того, какие мутации должны быть применены к части данных (мутации с версией выше, чем data&#95;version).
* `primary_key_bytes_in_memory` ([UInt64](/ru/reference/data-types/index)) — Объём памяти (в байтах), используемой значениями первичного ключа.
* `primary_key_bytes_in_memory_allocated` ([UInt64](/ru/reference/data-types/index)) — Объём памяти (в байтах), зарезервированной для значений первичного ключа.
* `database` ([String](/ru/reference/data-types/index)) — Имя базы данных.
* `table` ([String](/ru/reference/data-types/index)) — Имя таблицы.
* `engine` ([String](/ru/reference/data-types/index)) — Имя движка таблицы без параметров.
* `disk_name` ([String](/ru/reference/data-types/index)) — Имя диска, на котором хранится data part.
* `path` ([String](/ru/reference/data-types/index)) — Абсолютный путь к папке с файлами data part.
* `column` ([String](/ru/reference/data-types/index)) — Имя столбца.
* `type` ([String](/ru/reference/data-types/index)) — Тип столбца.
* `column_position` ([UInt64](/ru/reference/data-types/index)) — Порядковая позиция столбца в таблице, начиная с 1.
* `default_kind` ([String](/ru/reference/data-types/index)) — Тип выражения (DEFAULT, MATERIALIZED, ALIAS) для значения по умолчанию или пустая строка, если оно не определено.
* `default_expression` ([String](/ru/reference/data-types/index)) — Выражение для значения по умолчанию или пустая строка, если оно не определено.
* `column_bytes_on_disk` ([UInt64](/ru/reference/data-types/index)) — Общий размер столбца в байтах.
* `column_data_compressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер сжатых данных в столбце, в байтах.
* `column_data_uncompressed_bytes` ([UInt64](/ru/reference/data-types/index)) — Общий размер несжатых данных в столбце, в байтах.
* `column_marks_bytes` ([UInt64](/ru/reference/data-types/index)) — Размер меток столбца, в байтах.
* `column_modification_time` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Время последнего изменения столбца.
* `column_ttl_min` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Минимальное значение вычисленного TTL-выражения столбца.
* `column_ttl_max` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Максимальное значение вычисленного TTL-выражения столбца.
* `statistics` ([Array(String)](/ru/reference/data-types/index)) — Статистика столбца.
* `estimates.min` ([Nullable(String)](/ru/reference/data-types/index)) — Оценочное минимальное значение столбца.
* `estimates.max` ([Nullable(String)](/ru/reference/data-types/index)) — Оценочное максимальное значение столбца.
* `estimates.cardinality` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Оценочная мощность столбца.
* `estimates.null_count` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Оценочное количество значений NULL в столбце.
* `serialization_kind` ([String](/ru/reference/data-types/index)) — Способ сериализации столбца
* `substreams` ([Array(String)](/ru/reference/data-types/index)) — Имена подпотоков, в которые сериализуется столбец
* `filenames` ([Array(String)](/ru/reference/data-types/index)) — Имена файлов, соответствующих каждому подпотоку столбца
* `subcolumns.names` ([Array(String)](/ru/reference/data-types/index)) — Имена подстолбцов столбца
* `subcolumns.types` ([Array(String)](/ru/reference/data-types/index)) — Типы подстолбцов столбца
* `subcolumns.serializations` ([Array(String)](/ru/reference/data-types/index)) — Типы сериализации подстолбцов столбца
* `subcolumns.bytes_on_disk` ([Array(UInt64)](/ru/reference/data-types/index)) — Размеры в байтах для каждого подстолбца
* `subcolumns.data_compressed_bytes` ([Array(UInt64)](/ru/reference/data-types/index)) — Размеры сжатых данных для каждого подстолбца в байтах
* `subcolumns.data_uncompressed_bytes` ([Array(UInt64)](/ru/reference/data-types/index)) — Размеры несжатых данных для каждого подстолбца в байтах
* `subcolumns.marks_bytes` ([Array(UInt64)](/ru/reference/data-types/index)) — Размеры меток для каждого подстолбца столбца в байтах

{/*AUTOGENERATED_END*/}
