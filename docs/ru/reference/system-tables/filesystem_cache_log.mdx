Содержит историю всех событий, связанных с файловым кэшем объектов в удалённой файловой системе.

Эту таблицу можно в любой момент безопасно очистить с помощью TRUNCATE или удалить.

Столбцы:

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события
* `query_id` ([String](/ru/reference/data-types/string)) — Идентификатор запроса
* `source_file_path` ([String](/ru/reference/data-types/string)) — Путь к сегменту файла в файловой системе
* `file_segment_range` ([Tuple(UInt64, UInt64)](/ru/reference/data-types/tuple)) — Диапазон сегмента файла
* `total_requested_range` ([Tuple(UInt64, UInt64)](/ru/reference/data-types/tuple)) — Полный диапазон чтения
* `key` ([String](/ru/reference/data-types/string)) — Ключ сегмента файла
* `offset` ([UInt64](/ru/reference/data-types/int-uint)) — Смещение сегмента файла
* `size` ([UInt64](/ru/reference/data-types/int-uint)) — Размер чтения
* `read_type` ([String](/ru/reference/data-types/string)) — Тип чтения: READ&#95;FROM&#95;CACHE, READ&#95;FROM&#95;FS&#95;AND&#95;DOWNLOADED&#95;TO&#95;CACHE, READ&#95;FROM&#95;FS&#95;BYPASSING&#95;CACHE
* `read_from_cache_attempted` ([UInt8](/ru/reference/data-types/int-uint)) — Была ли предпринята попытка чтения из кэша
* `read_buffer_id` ([String](/ru/reference/data-types/string)) — Идентификатор буфера чтения во внутренней реализации
* `user_id` ([String](/ru/reference/data-types/string)) — Идентификатор пользователя, создавшего сегмент файла
