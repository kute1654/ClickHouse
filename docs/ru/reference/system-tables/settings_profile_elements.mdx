<div id="description">
  ## Описание
</div>

Описывает содержимое профиля настроек:

* Ограничения.
* Роли и пользователи, к которым применяются настройки.
* Родительские профили настроек.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `profile_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя профиля настроек.
* `user_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя пользователя.
* `role_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя роли.
* `index` ([UInt64](/ru/reference/data-types/index)) — Порядковый номер элемента профиля настроек.
* `setting_name` ([Nullable(String)](/ru/reference/data-types/index)) — Имя настройки.
* `value` ([Nullable(String)](/ru/reference/data-types/index)) — Значение настройки.
* `min` ([Nullable(String)](/ru/reference/data-types/index)) — Минимальное значение настройки. NULL, если не задано.
* `max` ([Nullable(String)](/ru/reference/data-types/index)) — Максимальное значение настройки. NULL, если не задано.
* `writability` ([Nullable(Enum8(&#39;WRITABLE&#39; = 0, &#39;CONST&#39; = 1, &#39;CHANGEABLE&#95;IN&#95;READONLY&#39; = 2))](/ru/reference/data-types/index)) — Свойство, показывающее, можно ли изменить настройку.
* `inherit_profile` ([Nullable(String)](/ru/reference/data-types/index)) — Родительский профиль для данного профиля настроек. NULL, если не задано. Профиль настроек наследует все значения настроек и ограничения (min, max, readonly) от родительских профилей.
