<div id="description">
  ## Описание
</div>

Содержит исторические значения `system.asynchronous_metrics`, сохраняемые один раз за интервал времени (по умолчанию — раз в секунду). Включено по умолчанию.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `hostname` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Имя хоста сервера, выполняющего запрос.
* `event_date` ([Date](/ru/reference/data-types/date)) — Дата события.
* `event_time` ([DateTime](/ru/reference/data-types/datetime)) — Время события.
* `metric` ([LowCardinality(String)](/ru/reference/data-types/lowcardinality)) — Название метрики.
* `value` ([Float64](/ru/reference/data-types/float)) — Значение метрики.

{/*AUTOGENERATED_END*/}
