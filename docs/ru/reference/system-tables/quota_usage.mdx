<div id="description">
  ## Описание
</div>

Использование квоты текущим пользователем: сколько использовано и сколько осталось.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `quota_name` ([String](/ru/reference/data-types/index)) — Имя квоты.
* `quota_key` ([String](/ru/reference/data-types/index)) — Значение ключа.
* `start_time` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Время начала расчета потребления ресурсов.
* `end_time` ([Nullable(DateTime)](/ru/reference/data-types/index)) — Время окончания расчета потребления ресурсов.
* `duration` ([Nullable(UInt32)](/ru/reference/data-types/index)) — Длительность временного интервала для расчета потребления ресурсов, в секундах.
* `queries` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее количество выполненных запросов.
* `max_queries` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов всех типов.
* `query_selects` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее количество выполненных запросов SELECT.
* `max_query_selects` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов SELECT.
* `query_inserts` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее количество выполненных запросов INSERT.
* `max_query_inserts` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов INSERT.
* `errors` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее количество запросов, завершившихся ошибкой.
* `max_errors` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество запросов, завершившихся ошибкой, за указанный период времени.
* `result_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее общее количество строк в результирующем наборе всех запросов за текущий период времени.
* `max_result_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое общее количество строк в результирующем наборе всех запросов за указанный период времени.
* `result_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее общее количество байтов в результирующем наборе всех запросов за текущий период времени.
* `max_result_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое общее количество байтов в результирующем наборе всех запросов за указанный период времени.
* `read_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее общее количество строк, прочитанных при выполнении всех запросов за текущий период времени.
* `max_read_rows` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество строк, считываемых при выполнении всех запросов за указанный период времени.
* `read_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее общее количество байтов, прочитанных при выполнении всех запросов за текущий период времени.
* `max_read_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество байтов, считываемых при выполнении всех запросов за указанный период времени.
* `execution_time` ([Nullable(Float64)](/ru/reference/data-types/index)) — Текущее суммарное время (в наносекундах), затраченное на выполнение запросов за текущий период времени
* `max_execution_time` ([Nullable(Float64)](/ru/reference/data-types/index)) — Максимально допустимое суммарное время (в наносекундах) на выполнение всех запросов за указанный период времени
* `written_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее общее количество байтов, записанных при выполнении всех запросов за текущий период времени.
* `max_written_bytes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество байтов, записываемых при выполнении всех запросов за указанный период времени.
* `failed_sequential_authentications` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее количество последовательных неудачных попыток аутентификации за текущий период времени.
* `max_failed_sequential_authentications` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество последовательных неудачных попыток аутентификации за указанный период времени.
* `queries_per_normalized_hash` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Текущее максимальное количество выполнений любого отдельного нормализованного запроса за текущий период времени.
* `max_queries_per_normalized_hash` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Максимально допустимое количество выполнений любого отдельного нормализованного запроса за указанный период времени.
