<div id="description">
  ## Описание
</div>

Содержит сведения о файлах данных, хранящихся на удалённых дисках (например, S3, Azure Blob Storage), включая сопоставление локальных путей метаданных с удалёнными путями blob-объектов.

Каждая строка соответствует одному удалённому blob-объекту, связанному с файлом данных.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `disk_name` ([String](/ru/reference/data-types/index)) — Имя диска.
* `path` ([String](/ru/reference/data-types/index)) — Путь к диску.
* `cache_base_path` ([String](/ru/reference/data-types/index)) — Базовый каталог файлов кэша.
* `local_path` ([String](/ru/reference/data-types/index)) — Путь к файлу ClickHouse; также используется как путь к метаданным.
* `remote_path` ([String](/ru/reference/data-types/index)) — Путь к blob-объекту в объектном хранилище, с которым связан файл ClickHouse.
* `size` ([UInt64](/ru/reference/data-types/index)) — Размер файла (в сжатом виде).
* `common_prefix_for_blobs` ([String](/ru/reference/data-types/index)) — Общий префикс для blob-объектов в объектном хранилище.
* `cache_paths` ([Array(String)](/ru/reference/data-types/index)) — Файлы кэша для соответствующего blob-объекта.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

```sql
SELECT * FROM system.remote_data_paths LIMIT 1 FORMAT Vertical;
```

```text
Row 1:
──────
disk_name:              s3
path:                   /var/lib/clickhouse/disks/s3/
cache_base_path:        /var/lib/clickhouse/disks/s3_cache/
local_path:             store/123/1234abcd-1234-1234-1234-1234abcd1234/all_0_0_0/data.bin
remote_path:            abc123/all_0_0_0/data.bin
size:                   1048576
common_prefix_for_blobs:
cache_paths:            ['/var/lib/clickhouse/disks/s3_cache/a1/b2/c3d4e5f6']
```
