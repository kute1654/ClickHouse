<div id="description">
  ## Описание
</div>

Содержит метрики, которые можно вычислить мгновенно или у которых есть текущее значение. Например, число одновременно обрабатываемых запросов или текущую задержку реплики. Эта таблица всегда содержит актуальные данные.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ru/reference/data-types/index)) — Имя метрики.
* `value` ([Int64](/ru/reference/data-types/index)) — Значение метрики.
* `description` ([String](/ru/reference/data-types/index)) — Описание метрики.

{/*AUTOGENERATED_END*/}

Все поддерживаемые метрики перечислены в исходном файле [src/Common/CurrentMetrics.cpp](https://github.com/ClickHouse/ClickHouse/blob/master/src/Common/CurrentMetrics.cpp).

<div id="example">
  ## Пример
</div>

```sql
SELECT * FROM system.metrics LIMIT 10
```

```text
┌─metric───────────────────────────────┬─value─┬─description────────────────────────────────────────────────────────────┐
│ Query                                │     1 │ Количество выполняемых запросов                                        │
│ Merge                                │     0 │ Количество выполняемых фоновых слияний                                 │
│ PartMutation                         │     0 │ Количество мутаций (ALTER DELETE/UPDATE)                               │
│ ReplicatedFetch                      │     0 │ Количество частей данных, загружаемых с реплик                         │
│ ReplicatedSend                       │     0 │ Количество частей данных, отправляемых на реплики                      │
│ ReplicatedChecks                     │     0 │ Количество частей данных, проверяемых на согласованность               │
│ BackgroundMergesAndMutationsPoolTask │     0 │ Количество активных слияний и мутаций в связанном фоновом пуле         │
│ BackgroundFetchesPoolTask            │     0 │ Количество активных загрузок в связанном фоновом пуле                  │
│ BackgroundCommonPoolTask             │     0 │ Количество активных задач в связанном фоновом пуле                     │
│ BackgroundMovePoolTask               │     0 │ Количество активных задач в BackgroundProcessingPool для перемещений   │
└──────────────────────────────────────┴───────┴────────────────────────────────────────────────────────────────────────┘
```

  ### CompiledExpressionCacheBytes
</div>
