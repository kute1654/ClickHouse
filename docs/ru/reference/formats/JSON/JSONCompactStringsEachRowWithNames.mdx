<div id="description">
  ## Описание
</div>

Отличается от формата [`JSONCompactEachRow`](/ru/reference/formats/JSON/JSONCompactEachRow) тем, что также выводит строку заголовка с именами столбцов — аналогично формату [TabSeparatedWithNames](/ru/reference/formats/TabSeparated/TabSeparatedWithNames).

<div id="example-usage">
  ## Пример использования
</div>

<div id="inserting-data">

</div>

Вставьте данные:

```sql

```

<div id="format-settings">
  ## Настройки формата
</div>

<Note>
  Если настройка [`input_format_with_names_use_header`](/ru/reference/settings/formats#input_format_with_names_use_header) установлена в `1`,
  то столбцы из входных данных будут сопоставлены со столбцами таблицы по именам, а столбцы с неизвестными именами будут пропущены, если настройка [`input_format_skip_unknown_fields`](/ru/reference/settings/formats#input_format_skip_unknown_fields) установлена в `1`.
  В противном случае первая строка будет пропущена.
</Note>
