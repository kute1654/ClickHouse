<div id="description">
  ## Описание
</div>

Отличается от формата [`JSONCompactEachRow`](/ru/reference/formats/JSON/JSONCompactEachRow) тем, что также выводит две строки заголовка с именами и типами столбцов, аналогично формату [TabSeparatedWithNamesAndTypes](/ru/reference/formats/TabSeparated/TabSeparatedWithNamesAndTypes).

<div id="example-usage">
  ## Пример использования
</div>

</div>

```sql

```

<div id="format-settings">
  ## Настройки формата
</div>

<Note>
  Если настройка [`input_format_with_names_use_header`](/ru/reference/settings/formats#input_format_with_names_use_header) установлена в `1`,
  столбцы из входных данных будут сопоставляться со столбцами таблицы по именам, а столбцы с неизвестными именами будут пропускаться, если настройка [input&#95;format&#95;skip&#95;unknown&#95;fields](/ru/reference/settings/formats#input_format_skip_unknown_fields) установлена в `1`.
  В противном случае первая строка будет пропущена.
  Если настройка [`input_format_with_types_use_header`](/ru/reference/settings/formats#input_format_with_types_use_header) установлена в `1`,
  типы из входных данных будут сравниваться с типами соответствующих столбцов таблицы. В противном случае вторая строка будет пропущена.
</Note>
