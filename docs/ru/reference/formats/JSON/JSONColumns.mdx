<div id="description">
  ## Описание
</div>

<Tip>
  Вывод форматов JSONColumns* содержит имя поля ClickHouse, а затем — содержимое каждой строки таблицы для этого поля;
  визуально данные повёрнуты на 90 градусов влево.
</Tip>

В этом формате все данные представлены в виде единого объекта JSON.

<Note>
  Формат `JSONColumns` буферизует все данные в памяти, а затем выводит их одним блоком, поэтому это может привести к высокому потреблению памяти.
</Note>

</div>

```sql
SELECT *
FROM football
FORMAT JSONColumns
```

<div id="format-settings">
  ## Настройки формата
</div>

При импорте столбцы с неизвестными именами будут пропущены, если для настройки [`input_format_skip_unknown_fields`](/ru/reference/settings/formats#input_format_skip_unknown_fields) установлено значение `1`.
Столбцы, отсутствующие в блоке, будут заполнены значениями по умолчанию (здесь можно использовать настройку [`input_format_defaults_for_omitted_fields`](/ru/reference/settings/formats#input_format_defaults_for_omitted_fields))
