<div id="description">
  ## Описание
</div>

Выводит данные в виде последовательности SQL-операторов `INSERT INTO table (columns...) VALUES (...), (...) ...;`.

<div id="example-usage">
  ## Пример использования
</div>

Пример:

```sql
SELECT number AS x, number + 1 AS y, 'Hello' AS z FROM numbers(10) FORMAT SQLInsert SETTINGS output_format_sql_insert_max_batch_size = 2
```

```sql
INSERT INTO table (x, y, z) VALUES (0, 1, 'Hello'), (1, 2, 'Hello');
INSERT INTO table (x, y, z) VALUES (2, 3, 'Hello'), (3, 4, 'Hello');
INSERT INTO table (x, y, z) VALUES (4, 5, 'Hello'), (5, 6, 'Hello');
INSERT INTO table (x, y, z) VALUES (6, 7, 'Hello'), (7, 8, 'Hello');
INSERT INTO table (x, y, z) VALUES (8, 9, 'Hello'), (9, 10, 'Hello');
```

Для чтения данных в этом формате можно использовать входной формат [MySQLDump](/ru/reference/formats/MySQLDump).

<div id="format-settings">
  ## Настройки формата
</div>

| Настройка                                                                                                                    | Описание                                                | По умолчанию |
| ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- | ------------ |
| [`output_format_sql_insert_max_batch_size`](/ru/reference/settings/formats#output_format_sql_insert_max_batch_size)             | Максимальное количество строк в одном операторе INSERT. | `65505`      |
| [`output_format_sql_insert_table_name`](/ru/reference/settings/formats#output_format_sql_insert_table_name)                     | Имя таблицы в выходном INSERT-запросе.                  | `'table'`    |
| [`output_format_sql_insert_include_column_names`](/ru/reference/settings/formats#output_format_sql_insert_include_column_names) | Включать имена столбцов в INSERT-запрос.                | `true`       |
| [`output_format_sql_insert_use_replace`](/ru/reference/settings/formats#output_format_sql_insert_use_replace)                   | Использовать оператор REPLACE вместо INSERT.            | `false`      |
| [`output_format_sql_insert_quote_names`](/ru/reference/settings/formats#output_format_sql_insert_quote_names)                   | Заключать имена столбцов в символы &#96;&#96;&#96;.     | `true`       |
