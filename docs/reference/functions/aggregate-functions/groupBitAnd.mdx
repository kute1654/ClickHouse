## groupBitAnd {#groupBitAnd}

Introduced in: v1.1.0

Applies bitwise AND for series of numbers.
    

**Syntax**

```sql
groupBitAnd(expr)
```

**Aliases**: `BIT_AND`

**Arguments**

- `expr` — Expression of `(U)Int*` type. [`(U)Int*`](/reference/data-types/int-uint)

**Returned value**

Returns a value of `(U)Int*` type. [`(U)Int*`](/reference/data-types/int-uint)

**Examples**

**Bitwise AND example**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitAnd(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 00000100 = 4

┌─groupBitAnd(num)─┐
│                4 │
└──────────────────┘
```
