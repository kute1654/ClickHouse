## groupBitOr {#groupBitOr}

Introduced in: v1.1.0

Applies bitwise OR for series of numbers.
    

**Syntax**

```sql
groupBitOr(expr)
```

**Aliases**: `BIT_OR`

**Arguments**

- `expr` — Expression of `(U)Int*` type. [`(U)Int*`](/reference/data-types/int-uint)

**Returned value**

Returns a value of `(U)Int*` type. [`(U)Int*`](/reference/data-types/int-uint)

**Examples**

**Bitwise OR example**

```sql title=Query
CREATE TABLE t (num UInt32) ENGINE = Memory;
INSERT INTO t VALUES (44), (28), (13), (85);

-- Test data:
-- binary     decimal
-- 00101100 = 44
-- 00011100 = 28
-- 00001101 = 13
-- 01010101 = 85

SELECT groupBitOr(num) FROM t;
```

```response title=Response
-- Result:
-- binary     decimal
-- 01111101 = 125

┌─groupBitOr(num)─┐
│             125 │
└─────────────────┘
```
