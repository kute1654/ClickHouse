## groupArraySample {#groupArraySample}

Introduced in: v20.3.0

Creates an array of sample argument values.
The size of the resulting array is limited to `max_size` elements.
Argument values are selected and added to the array randomly.
    

**Syntax**

```sql
groupArraySample(max_size[, seed])(x)
```

**Parameters**

- `max_size` вҖ” Maximum size of the resulting array. [`UInt64`](/reference/data-types/int-uint)
- `seed` вҖ” Optional. Seed for the random number generator. Default value: 123456. [`UInt64`](/reference/data-types/int-uint)
- `x` вҖ” Argument (column name or expression). [`Any`](/reference/data-types/index)

**Arguments**

- `array_column` вҖ” Column containing arrays to be aggregated. [`Array`](/reference/data-types/array)

**Returned value**

Array of randomly selected x arguments. [`Array(T)`](/reference/data-types/array)

**Examples**

**Usage example**

```sql title=Query
CREATE TABLE default.colors (
    id Int32,
    color String
) ENGINE = Memory;

INSERT INTO default.colors VALUES
(1, 'red'),
(2, 'blue'),
(3, 'green'),
(4, 'white'),
(5, 'orange');

SELECT groupArraySample(3)(color) as newcolors FROM default.colors;
```
