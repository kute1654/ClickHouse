## Description {#description}

Contains information about normal and aggregate functions.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) вҖ” The name of the function.
- `is_aggregate` ([UInt8](/reference/data-types/index)) вҖ” Whether the function is an aggregate function.
- `case_insensitive` ([UInt8](/reference/data-types/index)) вҖ” Whether the function name can be used case-insensitively.
- `alias_to` ([String](/reference/data-types/index)) вҖ” The original function name, if the function name is an alias.
- `create_query` ([String](/reference/data-types/index)) вҖ” Obsolete.
- `origin` ([Enum8('System' = 0, 'SQLUserDefined' = 1, 'ExecutableUserDefined' = 2, 'WasmUserDefined' = 3)](/reference/data-types/index)) вҖ” Obsolete.
- `description` ([String](/reference/data-types/index)) вҖ” A high-level description what the function does.
- `syntax` ([String](/reference/data-types/index)) вҖ” Signature of the function.
- `arguments` ([String](/reference/data-types/index)) вҖ” The function arguments.
- `parameters` ([String](/reference/data-types/index)) вҖ” The function parameters (only for aggregate function).
- `returned_value` ([String](/reference/data-types/index)) вҖ” What does the function return.
- `examples` ([String](/reference/data-types/index)) вҖ” Usage example.
- `introduced_in` ([String](/reference/data-types/index)) вҖ” ClickHouse version in which the function was first introduced.
- `categories` ([String](/reference/data-types/index)) вҖ” The category of the function.
- `deterministic` ([Nullable(UInt8)](/reference/data-types/index)) вҖ” Whether the function returns the same result for the same arguments. NULL when unknown (e.g. aggregate or user-defined functions).
- `higher_order` ([Nullable(UInt8)](/reference/data-types/index)) вҖ” Whether the function is higher-order вҖ” i.e. accepts at least one lambda expression as an argument (e.g. arrayMap, arrayFilter, mapApply). NULL when unknown.
{/*AUTOGENERATED_END*/}

## Example {#example}

```sql title="Query"
 SELECT name, is_aggregate, deterministic, case_insensitive, alias_to FROM system.functions LIMIT 5;
```
