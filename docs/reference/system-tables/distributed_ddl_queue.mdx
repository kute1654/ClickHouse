## Description {#description}

Contains information about [distributed ddl queries (ON CLUSTER clause)](/reference/statements/distributed-ddl) that were executed on a cluster.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `entry` ([String](/reference/data-types/index)) — Query id.
- `entry_version` ([Nullable(UInt8)](/reference/data-types/index)) — Version of the entry.
- `initiator_host` ([Nullable(String)](/reference/data-types/index)) — Host that initiated the DDL operation.
- `initiator_port` ([Nullable(UInt16)](/reference/data-types/index)) — Port used by the initiator.
- `cluster` ([String](/reference/data-types/index)) — Cluster name, empty if not determined.
- `query` ([String](/reference/data-types/index)) — Query executed.
- `settings` ([Map(String, String)](/reference/data-types/index)) — Settings used in the DDL operation.
- `query_create_time` ([DateTime](/reference/data-types/index)) — Query created time.
- `host` ([Nullable(String)](/reference/data-types/index)) — Hostname.
- `port` ([Nullable(UInt16)](/reference/data-types/index)) — Host Port.
- `status` ([Nullable(Enum8('Inactive' = 0, 'Active' = 1, 'Finished' = 2, 'Removing' = 3, 'Unknown' = 4))](/reference/data-types/index)) — Status of the query.
- `exception_code` ([Nullable(UInt16)](/reference/data-types/index)) — Exception code.
- `exception_text` ([Nullable(String)](/reference/data-types/index)) — Exception message.
- `query_finish_time` ([Nullable(DateTime)](/reference/data-types/index)) — Query finish time.
- `query_duration_ms` ([Nullable(UInt64)](/reference/data-types/index)) — Duration of query execution (in milliseconds).
{/*AUTOGENERATED_END*/}
