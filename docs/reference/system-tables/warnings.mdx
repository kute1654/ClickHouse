## Description {#description}

Contains warnings about server configuration to be displayed by clickhouse-client right after it connects to the server.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `message` ([String](/reference/data-types/index)) — A warning message issued by ClickHouse server.
- `message_format_string` ([LowCardinality(String)](/reference/data-types/index)) — A format string that was used to format the message.
{/*AUTOGENERATED_END*/}
