## Description {#description}

Contains information about configured [roles](/concepts/features/security/access-rights#role-management).

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — Role name.
- `id` ([UUID](/reference/data-types/index)) — Role ID.
- `storage` ([String](/reference/data-types/index)) — Path to the storage of roles. Configured in the `access_control_path` parameter.
{/*AUTOGENERATED_END*/}
