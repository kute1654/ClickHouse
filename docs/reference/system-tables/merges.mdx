## Description {#description}

Contains information about merges and part mutations currently in process for tables in the MergeTree family.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `database` ([String](/reference/data-types/index)) — The name of the database the table is in.
- `table` ([String](/reference/data-types/index)) — Table name.
- `elapsed` ([Float64](/reference/data-types/index)) — The time elapsed (in seconds) since the merge started.
- `progress` ([Float64](/reference/data-types/index)) — The percentage of completed work from 0 to 1.
- `num_parts` ([UInt64](/reference/data-types/index)) — The number of parts to be merged.
- `source_part_names` ([Array(String)](/reference/data-types/index)) — The list of source parts names.
- `result_part_name` ([String](/reference/data-types/index)) — The name of the part that will be formed as the result of merging.
- `source_part_paths` ([Array(String)](/reference/data-types/index)) — The list of paths for each source part.
- `result_part_path` ([String](/reference/data-types/index)) — The path of the part that will be formed as the result of merging.
- `partition_id` ([String](/reference/data-types/index)) — The identifier of the partition where the merge is happening.
- `partition` ([String](/reference/data-types/index)) — The name of the partition
- `is_mutation` ([UInt8](/reference/data-types/index)) — 1 if this process is a part mutation.
- `total_size_bytes_compressed` ([UInt64](/reference/data-types/index)) — The total size of the compressed data in the merged chunks.
- `total_size_bytes_uncompressed` ([UInt64](/reference/data-types/index)) — The total size of compressed data in the merged chunks.
- `total_size_marks` ([UInt64](/reference/data-types/index)) — The total number of marks in the merged parts.
- `bytes_read_uncompressed` ([UInt64](/reference/data-types/index)) — Number of bytes read, uncompressed.
- `rows_read` ([UInt64](/reference/data-types/index)) — Number of rows read.
- `bytes_written_uncompressed` ([UInt64](/reference/data-types/index)) — Number of bytes written, uncompressed.
- `rows_written` ([UInt64](/reference/data-types/index)) — Number of rows written.
- `columns_written` ([UInt64](/reference/data-types/index)) — Number of columns written (for Vertical merge algorithm).
- `memory_usage` ([UInt64](/reference/data-types/index)) — Memory consumption of the merge process.
- `thread_id` ([UInt64](/reference/data-types/index)) — Thread ID of the merge process.
- `merge_type` ([String](/reference/data-types/index)) — The type of current merge. Empty if it's an mutation.
- `merge_algorithm` ([String](/reference/data-types/index)) — The algorithm used in current merge. Empty if it's an mutation.
- `current_projection` ([String](/reference/data-types/index)) — The name of the projection currently being merged or rebuilt. Empty if not in the projection merge stage.
- `current_projection_progress` ([Float64](/reference/data-types/index)) — The progress of the current projection merge from 0 to 1.
- `current_projection_parts_merging` ([UInt64](/reference/data-types/index)) — The number of projection parts currently being merged.
- `current_projection_parts_remaining` ([UInt64](/reference/data-types/index)) — The number of projection parts remaining to be merged for the current projection.
- `projections_completed` ([Array(String)](/reference/data-types/index)) — The list of projections that have been merged or rebuilt so far.
- `projections_remaining` ([Array(String)](/reference/data-types/index)) — The list of projections that still need to be merged or rebuilt.
{/*AUTOGENERATED_END*/}
