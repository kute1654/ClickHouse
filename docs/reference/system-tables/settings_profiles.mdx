## Description {#description}

Contains properties of configured setting profiles.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — Setting profile name.
- `id` ([UUID](/reference/data-types/index)) — Setting profile ID.
- `storage` ([String](/reference/data-types/index)) — Path to the storage of setting profiles. Configured in the `access_control_path` parameter.
- `num_elements` ([UInt64](/reference/data-types/index)) — Number of elements for this profile in the `system.settings_profile_elements` table.
- `apply_to_all` ([UInt8](/reference/data-types/index)) — Shows that the settings profile set for all roles and/or users.
- `apply_to_list` ([Array(String)](/reference/data-types/index)) — List of the roles and/or users to which the setting profile is applied.
- `apply_to_except` ([Array(String)](/reference/data-types/index)) — The setting profile is applied to all roles and/or users excepting of the listed ones.
{/*AUTOGENERATED_END*/}
