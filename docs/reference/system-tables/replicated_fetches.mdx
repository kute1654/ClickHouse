## Description {#description}

Contains information about currently running background fetches.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `database` ([String](/reference/data-types/index)) — Name of the database.
- `table` ([String](/reference/data-types/index)) — Name of the table.
- `elapsed` ([Float64](/reference/data-types/index)) — The time elapsed (in seconds) since showing currently running background fetches started.
- `progress` ([Float64](/reference/data-types/index)) — The percentage of completed work from 0 to 1.
- `result_part_name` ([String](/reference/data-types/index)) — The name of the part that will be formed as the result of showing currently running background fetches.
- `result_part_path` ([String](/reference/data-types/index)) — Absolute path to the part that will be formed as the result of showing currently running background fetches.
- `partition_id` ([String](/reference/data-types/index)) — ID of the partition.
- `total_size_bytes_compressed` ([UInt64](/reference/data-types/index)) — The total size (in bytes) of the compressed data in the result part.
- `bytes_read_compressed` ([UInt64](/reference/data-types/index)) — The number of compressed bytes read from the result part.
- `source_replica_path` ([String](/reference/data-types/index)) — Absolute path to the source replica.
- `source_replica_hostname` ([String](/reference/data-types/index)) — Hostname of the source replica.
- `source_replica_port` ([UInt16](/reference/data-types/index)) — Port number of the source replica.
- `interserver_scheme` ([String](/reference/data-types/index)) — Name of the interserver scheme.
- `URI` ([String](/reference/data-types/index)) — Uniform resource identifier.
- `to_detached` ([UInt8](/reference/data-types/index)) — The flag indicates whether the currently running background fetch is being performed using the TO DETACHED expression.
- `thread_id` ([UInt64](/reference/data-types/index)) — Thread identifier.
{/*AUTOGENERATED_END*/}
