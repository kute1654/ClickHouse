## Description {#description}

Contains a list of CatBoost models loaded into a LibraryBridge's memory along with time when it was loaded.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `model_path` ([String](/reference/data-types/index)) — Path to trained model.
- `type` ([String](/reference/data-types/index)) — Model type. Now catboost only.
- `loading_start_time` ([DateTime](/reference/data-types/index)) — The time when the loading of the model started.
- `loading_duration` ([Float32](/reference/data-types/index)) — How much time did it take to load the model.
{/*AUTOGENERATED_END*/}
