## Description {#description}

Contains information of each Replicated database replicas.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `database` ([String](/reference/data-types/index)) — The name of the Replicated database is in.
- `is_readonly` ([UInt8](/reference/data-types/index)) — Whether the database replica is in read-only mode.
- `max_log_ptr` ([Int32](/reference/data-types/index)) — Maximum entry number in the log of general activity.
- `replica_name` ([String](/reference/data-types/index)) — Replica name in ClickHouse Keeper.
- `replica_path` ([String](/reference/data-types/index)) — Path to replica data in ClickHouse Keeper.
- `zookeeper_path` ([String](/reference/data-types/index)) — Path to database data in ClickHouse Keeper.
- `shard_name` ([String](/reference/data-types/index)) — The name of the shard in the cluster.
- `log_ptr` ([Int32](/reference/data-types/index)) — Maximum entry number in the log of general activity that the replica copied to its execution queue, plus one.
- `total_replicas` ([UInt32](/reference/data-types/index)) — The total number of known replicas of this database.
- `zookeeper_exception` ([String](/reference/data-types/index)) — The last exception message, got if the error happened when fetching the info from ClickHouse Keeper.
- `is_session_expired` ([UInt8](/reference/data-types/index)) — The session with ClickHouse Keeper has expired. Basically the same as `is_readonly`.
{/*AUTOGENERATED_END*/}
