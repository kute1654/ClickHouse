## Description {#description}

Contains information about cached DNS records.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `hostname` ([String](/reference/data-types/index)) — Hostname.
- `ip_address` ([String](/reference/data-types/index)) — IP address.
- `ip_family` ([Enum8('IPv4' = 0, 'IPv6' = 1, 'UNIX_LOCAL' = 2)](/reference/data-types/index)) — IP address family.
- `cached_at` ([DateTime](/reference/data-types/index)) — Record cached timestamp.
{/*AUTOGENERATED_END*/}

## Example {#example}

```sql title="Query"
SELECT * FROM system.dns_cache;
```

| hostname | ip\_address | ip\_family | cached\_at |
| :--- | :--- | :--- | :--- |
| localhost | ::1 | IPv6 | 2024-02-11 17:04:40 |
| localhost | 127.0.0.1 | IPv4 | 2024-02-11 17:04:40 |
