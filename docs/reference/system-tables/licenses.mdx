## Description {#description}

Contains licenses of third-party libraries that are located in the [contrib](https://github.com/ClickHouse/ClickHouse/tree/master/contrib) directory of ClickHouse sources.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `library_name` ([String](/reference/data-types/index)) ‚Äî Name of the library.
- `license_type` ([String](/reference/data-types/index)) ‚Äî License type ‚Äî e.g. Apache, MIT.
- `license_path` ([String](/reference/data-types/index)) ‚Äî Path to the file with the license text.
- `license_text` ([String](/reference/data-types/index)) ‚Äî License text.
{/*AUTOGENERATED_END*/}

## Example {#example}

```sql
SELECT library_name, license_type, license_path FROM system.licenses LIMIT 15
```
