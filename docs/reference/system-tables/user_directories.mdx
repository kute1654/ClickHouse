## Description {#description}

Contains the information about configured user directories - directories on the file system from which ClickHouse server is allowed to read user provided data.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — The name of the directory.
- `type` ([String](/reference/data-types/index)) — The type of the access storage e.g. users.xml or replicated or memory etc.
- `params` ([String](/reference/data-types/index)) — JSON with the parameters of the access storage.
- `precedence` ([UInt64](/reference/data-types/index)) — The order in which this directory is declared in the config. The same order is used when ClickHouse tries to find a user or role.
{/*AUTOGENERATED_END*/}
