## Description {#description}

Contains information about compression and encryption codecs.

You can use this table to get information about the available compression and encryption codecs

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — Codec name.
- `method_byte` ([UInt8](/reference/data-types/index)) — Byte which indicates codec in compressed file.
- `is_compression` ([UInt8](/reference/data-types/index)) — True if this codec compresses something. Otherwise it can be just a transformation that helps compression.
- `is_generic_compression` ([UInt8](/reference/data-types/index)) — The codec is a generic compression algorithm like lz4, zstd.
- `is_encryption` ([UInt8](/reference/data-types/index)) — The codec encrypts.
- `is_timeseries_codec` ([UInt8](/reference/data-types/index)) — The codec is for floating point timeseries codec.
- `is_experimental` ([UInt8](/reference/data-types/index)) — The codec is experimental.
- `description` ([String](/reference/data-types/index)) — A high-level description of the codec.
{/*AUTOGENERATED_END*/}

## Example {#example}

```sql title="Query"
SELECT * FROM system.codecs WHERE name='LZ4'
```
