## Description {#description}

Contains a list of [user accounts](/concepts/features/security/access-rights#user-account-management) configured on the server.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — User name.
- `id` ([UUID](/reference/data-types/index)) — User ID.
- `storage` ([String](/reference/data-types/index)) — Path to the storage of users. Configured in the access_control_path parameter.
- `auth_type` ([Array(Enum8('no_password' = 0, 'plaintext_password' = 1, 'sha256_password' = 2, 'double_sha1_password' = 3, 'ldap' = 4, 'kerberos' = 5, 'ssl_certificate' = 6, 'bcrypt_password' = 7, 'ssh_key' = 8, 'http' = 9, 'jwt' = 10, 'scram_sha256_password' = 11, 'no_authentication' = 12))](/reference/data-types/index)) — Shows the authentication types. There are multiple ways of user identification: with no password, with plain text password, with SHA256-encoded password, with double SHA-1-encoded password or with bcrypt-encoded password.
- `auth_params` ([Array(String)](/reference/data-types/index)) — Authentication parameters in the JSON format depending on the auth_type.
- `valid_until` ([Array(DateTime)](/reference/data-types/index)) — The expiration date and time for user credentials.
- `host_ip` ([Array(String)](/reference/data-types/index)) — IP addresses of hosts that are allowed to connect to the ClickHouse server.
- `host_names` ([Array(String)](/reference/data-types/index)) — Names of hosts that are allowed to connect to the ClickHouse server.
- `host_names_regexp` ([Array(String)](/reference/data-types/index)) — Regular expression for host names that are allowed to connect to the ClickHouse server.
- `host_names_like` ([Array(String)](/reference/data-types/index)) — Names of hosts that are allowed to connect to the ClickHouse server, set using the LIKE predicate.
- `default_roles_all` ([UInt8](/reference/data-types/index)) — Shows that all granted roles set for user by default.
- `default_roles_list` ([Array(String)](/reference/data-types/index)) — List of granted roles provided by default.
- `default_roles_except` ([Array(String)](/reference/data-types/index)) — All the granted roles set as default excepting of the listed ones.
- `grantees_any` ([UInt8](/reference/data-types/index)) — The flag that indicates whether a user with any grant option can grant it to anyone.
- `grantees_list` ([Array(String)](/reference/data-types/index)) — The list of users or roles to which this user is allowed to grant options to.
- `grantees_except` ([Array(String)](/reference/data-types/index)) — The list of users or roles to which this user is forbidden from grant options to.
- `default_database` ([String](/reference/data-types/index)) — The name of the default database for this user.
{/*AUTOGENERATED_END*/}
