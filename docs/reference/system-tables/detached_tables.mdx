## Description {#description}

Contains information about each detached table.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `database` ([String](/reference/data-types/index)) — The name of the database the table is in.
- `table` ([String](/reference/data-types/index)) — Table name.
- `uuid` ([UUID](/reference/data-types/index)) — Table uuid (Atomic database).
- `metadata_path` ([String](/reference/data-types/index)) — Path to the table metadata in the file system.
- `is_permanently` ([UInt8](/reference/data-types/index)) — Table was detached permanently.
{/*AUTOGENERATED_END*/}

## Example {#example}

```sql
SELECT * FROM system.detached_tables FORMAT Vertical;
```

```text
is_permanently:             1
```
