## Description {#description}

Contains information about disks defined in the [server configuration](/reference/engines/table-engines/mergetree-family/mergetree#table_engine-mergetree-multiple-volumes_configure).

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — Name of a disk in the server configuration.
- `path` ([String](/reference/data-types/index)) — Path to the mount point in the file system.
- `free_space` ([UInt64](/reference/data-types/index)) — Free space on disk in bytes.
- `total_space` ([UInt64](/reference/data-types/index)) — Disk volume in bytes.
- `unreserved_space` ([UInt64](/reference/data-types/index)) — Free space which is not taken by reservations (free_space minus the size of reservations taken by merges, inserts, and other disk write operations currently running).
- `keep_free_space` ([UInt64](/reference/data-types/index)) — Amount of disk space that should stay free on disk in bytes. Defined in the keep_free_space_bytes parameter of disk configuration.
- `type` ([String](/reference/data-types/index)) — The disk type which tells where this disk stores the data - RAM, local drive or remote storage.
- `object_storage_type` ([String](/reference/data-types/index)) — Type of object storage if disk type is object_storage
- `metadata_type` ([String](/reference/data-types/index)) — Type of metadata storage if disk type is object_storage
- `is_encrypted` ([UInt8](/reference/data-types/index)) — Flag which shows whether this disk encrypts the underlying data.
- `is_read_only` ([UInt8](/reference/data-types/index)) — Flag which indicates that you can only perform read operations with this disk.
- `is_write_once` ([UInt8](/reference/data-types/index)) — Flag which indicates if disk is write-once. Which means that it does support BACKUP to this disk, but does not support INSERT into MergeTree table on this disk.
- `is_remote` ([UInt8](/reference/data-types/index)) — Flag which indicated what operations with this disk involve network interaction.
- `is_broken` ([UInt8](/reference/data-types/index)) — Flag which indicates if disk is broken. Broken disks will have 0 space and cannot be used.
- `cache_path` ([String](/reference/data-types/index)) — The path to the cache directory on local drive in case when the disk supports caching.
{/*AUTOGENERATED_END*/}
