## Description {#description}

This system table is used for implementing the `SHOW PROCESSLIST` query.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `is_initial_query` ([UInt8](/reference/data-types/index)) — Whether this query comes directly from user or was issues by ClickHouse server in a scope of distributed query execution.
- `user` ([String](/reference/data-types/index)) — The user who made the query. Keep in mind that for distributed processing, queries are sent to remote servers under the default user. The field contains the username for a specific query, not for a query that this query initiated.
- `query_id` ([String](/reference/data-types/index)) — Query ID, if defined.
- `address` ([IPv6](/reference/data-types/index)) — The IP address the query was made from. The same for distributed processing. To track where a distributed query was originally made from, look at system.processes on the query requestor server.
- `port` ([UInt16](/reference/data-types/index)) — The client port the query was made from.
- `initial_user` ([String](/reference/data-types/index)) — Name of the user who ran the initial query (for distributed query execution).
- `initial_query_id` ([String](/reference/data-types/index)) — ID of the initial query (for distributed query execution).
- `initial_address` ([IPv6](/reference/data-types/index)) — IP address that the parent query was launched from.
- `initial_port` ([UInt16](/reference/data-types/index)) — The client port that was used to make the parent query.
- `interface` ([UInt8](/reference/data-types/index)) — The interface which was used to send the query. TCP = 1, HTTP = 2, GRPC = 3, MYSQL = 4, POSTGRESQL = 5, LOCAL = 6, TCP_INTERSERVER = 7.
- `os_user` ([String](/reference/data-types/index)) — Operating system username who runs clickhouse-client.
- `client_hostname` ([String](/reference/data-types/index)) — Hostname of the client machine where the clickhouse-client or another TCP client is run.
- `client_name` ([String](/reference/data-types/index)) — The clickhouse-client or another TCP client name.
- `client_agent` ([LowCardinality(String)](/reference/data-types/index)) — The AI coding agent that invoked the client (e.g. `claude-code`, `cursor`), detected from environment variables. Empty if no agent was detected.
- `client_revision` ([UInt64](/reference/data-types/index)) — Revision of the clickhouse-client or another TCP client.
- `client_version_major` ([UInt64](/reference/data-types/index)) — Major version of the clickhouse-client or another TCP client.
- `client_version_minor` ([UInt64](/reference/data-types/index)) — Minor version of the clickhouse-client or another TCP client.
- `client_version_patch` ([UInt64](/reference/data-types/index)) — Patch component of the clickhouse-client or another TCP client version.
- `http_method` ([UInt8](/reference/data-types/index)) — HTTP method that initiated the query. Possible values: 0 — The query was launched from the TCP interface. 1 — GET method was used. 2 — POST method was used.
- `http_user_agent` ([String](/reference/data-types/index)) — HTTP header UserAgent passed in the HTTP query.
- `http_referer` ([String](/reference/data-types/index)) — HTTP header Referer passed in the HTTP query (contains an absolute or partial address of the page making the query).
- `forwarded_for` ([String](/reference/data-types/index)) — HTTP header X-Forwarded-For passed in the HTTP query.
- `quota_key` ([String](/reference/data-types/index)) — The quota key specified in the quotas setting (see keyed).
- `distributed_depth` ([UInt64](/reference/data-types/index)) — The number of times query was retransmitted between server nodes internally.
- `elapsed` ([Float64](/reference/data-types/index)) — The time in seconds since request execution started.
- `is_cancelled` ([UInt8](/reference/data-types/index)) — Was query cancelled.
- `is_all_data_sent` ([UInt8](/reference/data-types/index)) — Was all data sent to the client (in other words query had been finished on the server).
- `read_rows` ([UInt64](/reference/data-types/index)) — The number of rows read from the table. For distributed processing, on the requestor server, this is the total for all remote servers.
- `read_bytes` ([UInt64](/reference/data-types/index)) — The number of uncompressed bytes read from the table. For distributed processing, on the requestor server, this is the total for all remote servers.
- `total_rows_approx` ([UInt64](/reference/data-types/index)) — The approximation of the total number of rows that should be read. For distributed processing, on the requestor server, this is the total for all remote servers. It can be updated during request processing, when new sources to process become known.
- `written_rows` ([UInt64](/reference/data-types/index)) — The amount of rows written to the storage.
- `written_bytes` ([UInt64](/reference/data-types/index)) — The amount of bytes written to the storage.
- `memory_usage` ([Int64](/reference/data-types/index)) — Amount of RAM the query uses. It might not include some types of dedicated memory
- `peak_memory_usage` ([Int64](/reference/data-types/index)) — The current peak of memory usage.
- `query` ([String](/reference/data-types/index)) — The query text. For INSERT, it does not include the data to insert.
- `normalized_query_hash` ([UInt64](/reference/data-types/index)) — A numeric hash value, such as it is identical for queries differ only by values of literals.
- `query_kind` ([String](/reference/data-types/index)) — The type of the query - SELECT, INSERT, etc.
- `thread_ids` ([Array(UInt64)](/reference/data-types/index)) — The list of identifiers of all threads which participated in this query.
- `peak_threads_usage` ([UInt64](/reference/data-types/index)) — Maximum count of simultaneous threads executing the query.
- `ProfileEvents` ([Map(LowCardinality(String), UInt64)](/reference/data-types/index)) — ProfileEvents calculated for this query.
- `Settings` ([Map(LowCardinality(String), LowCardinality(String))](/reference/data-types/index)) — The list of modified user-level settings.
- `current_database` ([String](/reference/data-types/index)) — The name of the current database.
- `is_internal` ([UInt8](/reference/data-types/index)) — Indicates whether it is an auxiliary query executed internally.
{/*AUTOGENERATED_END*/}
