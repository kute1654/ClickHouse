## Description {#description}

Contains information about the databases that are available to the current user.

## Columns {#columns}

{/*AUTOGENERATED_START*/}
- `name` ([String](/reference/data-types/index)) — Database name.
- `engine` ([String](/reference/data-types/index)) — Database engine.
- `data_path` ([String](/reference/data-types/index)) — Data path.
- `metadata_path` ([String](/reference/data-types/index)) — Metadata path.
- `uuid` ([UUID](/reference/data-types/index)) — Database UUID.
- `engine_full` ([String](/reference/data-types/index)) — Parameters of the database engine.
- `comment` ([String](/reference/data-types/index)) — Database comment.
- `is_external` ([UInt8](/reference/data-types/index)) — Database is external (i.e. PostgreSQL/DataLakeCatalog).
{/*AUTOGENERATED_END*/}

The `name` column from this system table is used for implementing the `SHOW DATABASES` query.

## Example {#example}

Create a database.

```sql title="Query"
CREATE DATABASE test;
```

Check all of the available databases to the user.

```sql title="Query"
SELECT * FROM system.databases;
```
